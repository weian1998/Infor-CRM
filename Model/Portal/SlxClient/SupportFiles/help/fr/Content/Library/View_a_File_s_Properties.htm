Propriétés des documents de la bibliothèque

La boîte de dialogue Propriétés du document permet d'ajouter ou de modifier les propriétés des fichiers de la bibliothèque. En plus des champs modifiables, cette boîte de dialogue offre les informations en lecture seule suivantes :

  • Répertoire : emplacement où le document est enregistré.
  • Etat : ne concerne que les utilisateurs distants. L'état peut être : Disponible, Commandé, Distribué, Non lu, Lu ou Révisé.
  • Créé : date à laquelle le fichier a été ajouté à la bibliothèque.
  • Taille : taille du fichier.

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Pour afficher ou définir les propriétés des documents de bibliothèque

  1. Dans la barre de navigation, cliquez sur Bibliothèque.
  2. Dans la liste, sélectionnez le document de bibliothèque que vous souhaitez modifier, et cliquez sur le bouton Propriétés de fichier.
  3. Utilisez la zone Révisé pour modifier la date et l'heure auxquelles le fichier a été modifié pour la dernière fois afin que la synchronisation envoie ce fichier aux clients distants.
    Ce champ n'est pas automatiquement mis à jour.
  4. Désélectionnez l'option Toujours valide pour activer la zone Expire le.
    Cette case est cochée par défaut.
  5. Définissez une date de validation dans la zone de texte Expire le. Cette option peut être utilisée pour les documents dont la date importe, comme les listes de tarifs.
  6. La zone de texte Description permet d'inclure un commentaire sur le fichier (255 caractères maximum).
    La description du fichier s'affiche dans la vue des détails de la bibliothèque.
  7. Le champ Résumé permet d'inclure un commentaire sur le fichier (1 024 caractères maximum).
    Ce Résumé n'est visible que dans cette boîte de dialogue.
  8. Cochez l'option Forcer la distribution de ce fichier pour que les fichiers et les révisions de fichiers soient automatiquement envoyés à tous les utilisateurs distants.
  9. Cliquez sur OK.
