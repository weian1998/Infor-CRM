Ajout ou modification d'un actif de compte

Vous pouvez ajouter un nouvel actif à un compte ou modifier la liste des actifs associés à un compte.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour ajouter

  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Actifs. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  3. Cliquez sur le bouton Ajouter un actif.  
     
  4. Renseignez les zones d'informations sur l'actif requises.
  5. Cliquez sur OK. 
     L'actif apparaît dans la liste des produits de ce compte.

Pour modifier

  1. Dans la vue Détails du compte, sélectionnez l'onglet Actifs.
  2. Sur l'onglet Actifs, effectuez l'une des opérations suivantes :
    • cliquez sur Modifier, modifiez les zones d'informations sur les actifs, puis cliquez sur OK.

    • Sélectionnez l'enregistrement à modifier, faites vos modifications dans la grille, puis cliquez sur Enregistrer. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur Enregistrer.

      Les colonnes ne peuvent pas toutes être modifiées dans la grille. Pour savoir si une colonne peut être modifiée, cliquez dans la grille. Si une zone blanche apparaît, la colonne est modifiable.
