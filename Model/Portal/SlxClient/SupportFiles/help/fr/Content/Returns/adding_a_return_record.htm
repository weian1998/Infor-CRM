Ajout d'un retour

Utilisez la vue Insérer un retour pour créer l'enregistrement d'un retour.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour ajouter un retour

  1. Ouvrez la vue Insérer un nouveau retour.
    • Dans le menu Nouveau, cliquez sur Retour.
  2. Renseignez les zones d'informations sur le retour requises.
  3.  Cliquez sur Enregistrer. 
    La vue Détails du retour s'ouvre.
  4. Utilisez l'onglet Détails de l'expédition pour ajouter des informations sur les produits de remplacement expédiés au client, comme partie intégrante du retour sélectionné.
  5. Utilisez l' onglet Détails de retour pour ajouter les actifs retournés.
