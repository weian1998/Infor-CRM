Utilisation de la vue Détails du retour

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Procédure

Ouvrez la vue Détails des retours
  • Dans la barre de navigation, cliquez sur Retours pour ouvrir la vue Liste des retours. Si nécessaire, utilisez la Recherche pour localiser un retour, puis cliquez sur le N° de Retour.

Utilisez la Recherche pour rechercher les enregistrements

Afficher un groupe

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.
Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • E-mail
  • Publipostage
  • Ajouter une note
  • Nouveau rendez-vous
  • Nouvel appel téléphonique
  • Nouvelle tâche à faire

zones d'informations

Les zones d'informations sur le retour contiennent des informations sur le retour sélectionné. Les utilisateurs disposant des autorisations appropriées peuvent modifier ces zones.

onglets des vues Détails

Chaque onglet affiche des informations spécifiques utiles dans le cadre des activités relatives au retour. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
