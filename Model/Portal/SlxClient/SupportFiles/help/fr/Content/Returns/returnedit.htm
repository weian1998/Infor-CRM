Modification d'un retour

Utilisez la vue Détails du retour pour modifier les informations sur un retour existant.

Pour modifier

  1. Ouvrez la vue Détails des retours.
  2. Modifiez les zones d'informations du retour selon les besoins.
  3. Cliquez sur Enregistrer..
