Utilisation de l'onglet Commentaires sur les retours

Utilisez cet onglet pour toutes les notes internes associées à un retour.

Pour ajouter des commentaires

  1. Ouvrez la vue Détails des retours.
  2. Cliquez sur l'onglet Commentaires. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  3. Saisissez les informations dans la zone Commentaires.
