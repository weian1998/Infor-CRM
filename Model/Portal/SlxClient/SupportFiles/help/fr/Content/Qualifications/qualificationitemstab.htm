Onglet Éléments de qualification

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez cette vue pour ajouter, modifier ou supprimer des éléments de qualification.

Procédure

Ouvrez cette vue
  • Dans la vue Détails de la qualification, cliquez sur l'onglet Éléments.

Ajout d'un élément de qualification

Modification d'un élément de qualification

Suppression d'un élément de qualification
  1. Cliquez sur Supprimer en regard de l'élément de qualification que vous souhaitez supprimer.
  2. Cliquez sur OK.

Cette vue contient les colonnes suivantes :

  • Visible - Indique si l'élément s'affiche ou non dans la vue Détails du lead.
  • Position de tri - Indique dans quel ordre l'élément s'affiche dans la vue Détails du lead.
  • Description - Indique la description qui s'affiche dans la vue Détails du lead.
  • Afficher les notes - Indique si un utilisateur peut ou non saisir des informations dans une zone Notes dans la vue Détails du lead.
