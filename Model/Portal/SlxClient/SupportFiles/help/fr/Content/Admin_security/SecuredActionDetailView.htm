Utilisation de la vue Détails des actions sécurisées

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Procédure

Créer une action sécurisée

Ouvrez la vue Détails des actions sécurisées

  1. Sur la barre de navigation, cliquez sur Administration, puis sur actions sécurisées pour ouvrir la vue Liste des actions sécurisées.
  2. Cliquez sur une action sécurisée dans la liste ou utilisez la fonction Rechercher pour trouver une action sécurisée.

Les informations sur l'action sécurisée s'ouvrent dans la vue Détails.

Utilisez la fonction Rechercher pour rechercher des enregistrements.

Ouvrir un groupe

Passer en vue Liste

  • Dans la barre d'outils, cliquez sur le bouton Vue Liste.

Zones d'informations

Les zones d'informations sur l'action sécurisée contiennent les informations essentielles sur l'action sécurisée.

  • Action sécurisée : le nom de l'action sécurisée.
  • Description : description de l'action sécurisée.

Onglets de la vue Détails

Chaque onglet affiche des informations spécifiques concernant l'action sécurisée. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
