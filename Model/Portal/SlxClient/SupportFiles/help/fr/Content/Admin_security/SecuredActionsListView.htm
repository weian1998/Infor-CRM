Utilisation de la vue Liste des actions sécurisées

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

La vue Liste des actions sécurisées affiche les actions sécurisées dans une grille. Le tableau fournit des informations sur chaque action sécurisée, comme le nom de l'action sécurisée et sa description.

  • Pour trier la grille selon un champ particulier, cliquez sur l'en-tête de colonne correspondant.
  • Pour afficher toutes les actions sécurisées d'un type particulier, cliquez sur un onglet dans la grille.

Ouvrez la vue Liste des actions sécurisées
  • Dans la barre de navigation, cliquez sur Administration, puis sur Actions sécurisées.

Ouvrez la vue Détails des actions sécurisées
  • Cliquez sur le nom de l'action sécurisée pour accéder à la vue Détails de l'action sécurisée.

Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.
