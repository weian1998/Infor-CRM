Affectation d'utilisateurs au rôle d'Utilisateur standard

Le rôle Utilisateur standard est assigné par défaut à tous les utilisateurs créés dans le client Web. Ce rôle permet l'ajout, la modification et la suppression d'entités non administratives du client Web, telles que les comptes, les contacts et les opportunités.

Aucun rôle n'est assigné aux utilisateurs créés dans l'Administrateur et la sécurité des rôles ne peut pas être définie dans l'Administrateur. Si vous disposez déjà d'utilisateurs, ou si vous créez de nouveaux utilisateurs dans l'Administrateur, vous devez attribuer aux utilisateurs le rôle d'Utilisateur standard.

Pour attribuer

  1. Ouvrez la vue Liste des utilisateurs.
  2. Sélectionnez les utilisateurs qui auront les autorisations d'ajouter, de modifier et de supprimer aux entités non administratives du client Web.
  3. Dans le volet Tâches utilisateurs, cliquez sur Ajouter au rôle.
  4. Dans la boîte de dialogue Sélectionner un rôle, recherchez et sélectionnez le rôle Utilisateur standard, puis cliquez sur OK.

  5. Cliquez sur OK.
