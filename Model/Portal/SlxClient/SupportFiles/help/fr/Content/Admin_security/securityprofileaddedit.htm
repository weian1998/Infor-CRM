Ajout ou modification d'un profil de sécurité

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez cette boîte de dialogue pour ajouter des informations relatives à un nouveau profil de sécurité ou pour modifier les informations relatives à un profil de sécurité existant.

Pour ajouter ou modifier un profil de sécurité.

  1. Ouvrez la vue Gestionnaire de sécurité.
  2. Depuis le volet des tâches, sous Tâches courantes, cliquez sur Ajouter un profil ou sur Modifier un profil.
  3. Dans le champ Nom, entrez un nom court et descriptif pour le profil de sécurité.
  4. Dans la liste Type de profil, sélectionnez Utilisateur ou Système.
    Ces deux types sont fournis à des fins organisationnelles uniquement ; ils ne présentent aucune différence d'ordre fonctionnel. Les profils système sont destinés aux équipes, tandis que les profils utilisateur ont été conçus pour des utilisateurs individuels.

  5. Cliquez sur OK.

Pour les étapes de mise à jour des droits d'accès à l'intérieur d'un profil, reportez-vous aux étapes décrites dans « Mettre à jour l'accès aux champs à l'intérieur d'un profil de sécurité », sous la rubrique d'aide Utilisation de la vue Gestionnaire de sécurité.
