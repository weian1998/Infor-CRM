Création d'un rôle

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Les autorisations d'accès aux fonctions et fonctionnalités dans le client Web sont déterminées par les rôles. Vous pouvez créer autant de rôles que vous le souhaitez et les affecter à un nombre illimité d'utilisateurs. Vous pouvez également utiliser un rôle par défaut.

Un utilisateur peut être affecté à un ou plusieurs rôles. Si un utilisateur est affecté à plus d'un rôle, il aura accès à toutes les actions sécurisées contenues dans ces rôles, même si chaque rôle ne contient pas les mêmes actions. Vous ne pouvez pas attribuer un rôle à un service ou à une équipe.

Pour créer

  1. Dans le menu Administration, cliquez sur Nouveau rôle.
  2. Dans le champ Nom, tapez le nom du rôle.
  3. Dans le champ Description, entrez des informations descriptives sur le rôle.
  4. Cliquez sur Enregistrer..
  5. Utilisez l'onglet Actions pour ajouter des actions au rôle.
  6. Utilisez l'onglet Utilisateurs pour ajouter des utilisateurs au rôle ou supprimer des utilisateurs du rôle.
