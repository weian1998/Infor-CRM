Utilisation de la vue Liste des rôles

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

La vue Liste des rôles affiche les rôles dans une grille. Le tableau fournit des informations sur chaque rôle, comme le nom du rôle et sa description.

  • Pour trier la grille selon un champ particulier, cliquez sur l'en-tête de colonne correspondant.
  • Pour afficher tous les rôles d'un type particulier, cliquez sur un onglet dans le tableau.
  • Pour n'afficher que les rôles correspondant à des critères spécifiques, configurez les filtres.

Procédure

Créer un rôle

Modifier ou supprimer un rôle

Ouvrez la vue Liste des rôles
  • Dans la barre de navigation, cliquez sur Administration, puis sur Rôles.
Ouvrez la vue Détails du rôle
  • Cliquez sur le nom du rôle pour ouvrir la vue Détails du rôle.

Travailler avec des enregistrements dans une grille

Filtrer ma liste

Utilisez la fonction Rechercher pour rechercher les enregistrements du rôle

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe
  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
