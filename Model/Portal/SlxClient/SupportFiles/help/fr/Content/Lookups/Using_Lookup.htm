Utilisation de la Recherche

La fonction Recherche permet de choisir un critère dans une liste de sélection et de saisir du texte. Pour que la recherche aboutisse, vous devez saisir au moins un paramètre de recherche. Vous pouvez restreindre votre recherche en incluant plusieurs conditions.  Vous pouvez aussi enregistrer une recherche en tant que groupe.

Pour réaliser une recherche

  1. Effectuez l’une des opérations suivantes :
    • Ouvrez la vue Détails ou Liste du type d'enregistrement que vous recherchez et cliquez sur l'onglet Recherche.  (Par exemple : contacts)
    • Dans la barre de navigation, cliquez avec le bouton droit sur l'enregistrement à rechercher, puis sur Recherche.... Par exemple, Recherche de comptes...
  2. Dans la zone Recherche par, sélectionnez une condition de recherche dans la liste déroulante.

    		3.

    Les options de recherche dépendent des colonnes incluses dans la présentation de la recherche, qui sont déterminées par le groupe sélectionné comme Présentation par défaut des recherches dans vos options de groupe. Pour inclure un autre élément de recherche dans la liste, vous devez mettre à jour le groupe Présentation par défaut des recherches pour inclure la colonne sur laquelle effectuer la recherche, ou remplacer la Présentation par défaut des recherches par un groupe qui inclut cette colonne dans la présentation.
  3. Sélectionnez un opérateur de condition de recherche dans la liste déroulante. Par exemple, contient ou commence par.
  4. Dans la zone suivante, indiquez vos critères de recherche en fonction de la condition sélectionnée.
     Par exemple, pour rechercher un compte intitulé Abbott SA, tapez « ab » dans cette zone.
  5. Pour ajouter un autre critère de recherche, cliquez sur le bouton Ajouter une condition et répétez les étapes 2 et -4 chaque fois que vous ajoutez un critère de recherche.
  6. Quand vous avez saisi toutes les conditions de recherche, cliquez sur Rechercher.
    Les enregistrements correspondant à vos critères s'afficheront directement dans l'onglet Résultats de la recherche.

Les recherches sont exécutées sur les enregistrements auxquels vous avez accès. Selon les options choisies par votre administrateur, vous pourrez également rechercher les enregistrements en fonction du groupe auquel ils appartiennent.
