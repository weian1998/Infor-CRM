Pièces jointes ouvertes en tant que dossier compressé

Problème :

Lorsque j'ouvre une pièce jointe, je reçois un message disant que le fichier joint est un « dossier compressé (zip) ».

Solution :

Contactez votre administrateur avec les détails de ce message, y compris le type de fichier que vous essayez d'ouvrir.
