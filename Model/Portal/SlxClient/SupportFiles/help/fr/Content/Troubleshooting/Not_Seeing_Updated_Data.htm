Je ne vois pas les données mises à jour

Problème :

Ma vue de liste n'est pas mise à jour après l'importation, la suppression ou la mise à jour de divers enregistrements.

Solution :

Pour actualiser les données de la vue de liste

  • Dans la vue de liste, cliquez sur le bouton Actualiser.

Problème :

Mes listes de sélection ne comprennent pas les valeurs mises à jour.

Solution :

Pour actualiser les données de la liste de sélection

  1. Dans le menu Outils, cliquez sur Options.

  2. Cliquez sur l'onglet Général.
  3. Cliquez sur le bouton Actualiser les données de la liste de sélection.
