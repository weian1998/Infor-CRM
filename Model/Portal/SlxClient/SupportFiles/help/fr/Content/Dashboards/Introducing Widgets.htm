Présentation des widgets

  Les widgets sont des plugins que vous pouvez ajouter à un tableau de bord. Vous pouvez utiliser plusieurs types de widgets pour afficher vos données. Déplacez votre souris sur le graphique dans un widget pour afficher les dimensions et les mesures sélectionnées pour ce widget.

 Les dimensions sont les données associées à une entité à afficher.

Par exemple, pour l'entité Compte, vous pouvez afficher les dimensions suivantes :

  • Gestionnaire de compte
  • Ville
  • Secteur
  • Propriétaire
  • SIC
  • Etat
  • Statut
  • Sous-type
  • Type

Les métriques sont la façon dont vous calculez les données à afficher.

Par exemple, pour l'entité Compte, vous pouvez utiliser les métriques suivantes :

  • Jours moyens depuis la dernière activité
  • Revenu moyen
  • Temps moyen en tant que client
  • Nombre de comptes
  • Revenu total

Cliquez sur le lien au bas de chaque widget pour afficher des informations plus détaillées pour ces données. Par exemple, si vous avez configuré le widget Liste des groupes de façon à ce qu'il affiche Toutes les opportunités ouvertes, le lien Afficher le groupe ouvre la vue Liste des opportunités. Cliquez sur Accueil dans la barre de navigation pour revenir au tableau de bord.

Il existe plusieurs types de widgets :

La liste des widgets disponible est déterminée par votre implémentation. Il est possible que vous n'ayez pas accès à tous les types de widget.

 Widgets graphiques

  • Barres : affiche les données sous forme de barres horizontales. ExempleSi vous sélectionnez les données pour Taille de commande moyenne par propriétaire, les équipes sont affichées sur l'axe Y et le nombre de commandes sur l'axe X. Chaque barre s'étend horizontalement jusqu'à atteindre le nombre de l'axe X qui correspond à la valeur la plus proche du total.
  • Colonne : affiche les données sous forme de colonnes verticales. Exemple Si vous sélectionnez les données pour Taille de commande moyenne par propriétaire, les équipes sont affichées sur l'axe Y et le nombre de commandes sur l'axe Y. Chaque barre s'étend verticalement jusqu'à atteindre le nombre de l'axe Y qui correspond à la valeur la plus proche du total. Lors de la configuration, vous pouvez ajouter un objectif qui s'affiche sous forme de ligne horizontale en travers du graphique, indiquant où se situent les colonnes par rapport à cet objectif.
  •  Entonnoir - Affiche les données dans des proportions progressives dans un entonnoir codé par couleur. Utilisez les graphiques en entonnoir pour illustrer les informations d'un processus par phase, par exemple le revenu potentiel ou le nombre d'opportunités dans votre pipeline d'opportunités. Exemples

Opportunités ouvertes par potentiel de ventes

Utilisez un graphique en entonnoir pour représenter le revenu potentiel des opportunités de votre pipeline en fonction des niveaux de revenu.

  1. Cliquez avec le bouton droit de la souris sur l'onglet Tableau de bord, puis cliquez sur Ajouter du contenu.
  2. Cliquez sur le bouton Ajouter sous Graphique en entonnoir.
  3. Donnez un titre au graphique en entonnoir, par exemple Opportunités ouvertes par potentiel de ventes.
  4. Dans la liste Entité, sélectionnez Opportunité.
  5. Dans la liste Groupe, sélectionnez Toutes.
  6. Dans la liste Dimension, sélectionnez Etape.
  7. Dans la liste Mesure, sélectionnez Somme du potentiel de ventes.
  8. Décidez si vous voulez Afficher les étiquettes, Tronquer les étiquettes et quelle est la longueur maximale de l'étiquette.
  9. Cliquez sur OK.
    Une plage de montants de ventes potentielles s'affiche, chaque plage de ventes potentielles étant représentée par un niveau de couleur différente. La taille de chaque niveau représente le nombre d'oportunités au sein d'une plage de ventes potentielles spécifique.

Opportunités ouvertes par étape

Utilisez un graphique en entonnoir pour afficher le nombre d'opportunités dans chaque étape du pipeline d'opportunité.

  1. Cliquez avec le bouton droit de la souris sur l'onglet Tableau de bord, puis cliquez sur Ajouter du contenu.
  2. Cliquez sur le bouton Ajouter sous Graphique en entonnoir.
  3. Donnez un titre au graphique en entonnoir, par exemple Mes opportunités par étape.
  4. Dans la liste Entité, sélectionnez Opportunité.
  5. Dans la liste Groupe, sélectionnez Toutes les opportunités.
  6. Dans la liste Dimension, sélectionnez Etape.
  7. Dans la liste Mesure, sélectionnez Nombre d'opportunités.
  8. Décidez si vous voulez Afficher les étiquettes, Tronquer les étiquettes et quelle est la longueur maximale de l'étiquette.
  9. Cliquez sur OK. 
    Chaque étape est représentée dans l'ordre de traitement, par un niveau de couleur différente. La taille de chaque niveau représente le nombre d'oportunités au sein d'une étape spécifique.
  •  Courbe : affiche les données sous la forme d'une série de points connectés par une courbe. Exemple Si vous sélectionnez les données relatives aux Campagnes marketing actives par région, les équipes s'affichent sur l'axe des X, le nombre de campagnes sur l'axe des Y et les effectifs sont représentés par des points de données sur le graphique.
  •  Graphique circulaire : affiche les données sous la forme d'un graphique circulaire dont les secteurs sont séparés par couleur et d'une légende. Exemple Si vous sélectionnez Comptes par région, avec cinq secteurs, le graphique s'affiche sous forme d'un graphique circulaire à cinq secteurs colorés et d'une légende affichant les propriétaires (équipes) ainsi que le nombre total de comptes par équipe. Vous pouvez configurer le graphique circulaire de façon à ce qu'il affiche cinq ou dix secteurs et choisir d'afficher ou non un secteur Autre qui inclut toutes les données non affichées.

Widgets de liste

  • Groupe - Affiche les informations de groupe spécifiées. Exemple Si vous sélectionnez Mes opportunités clôturées. Vous pouvez trier les colonnes dans la liste et naviguer jusqu'au compte ou à l'entité pour des informations plus détaillées avec le lien Afficher le groupe.

    		•
    • Si vous partagez un onglet Tableau de bord contenant un groupe, le groupe doit aussi être partagé avec les utilisateurs afin que le groupe s'affiche dans l'onglet Tableau de bord.
    • Le lien Afficher le groupe n'est pas disponible pour l'historique

  • Liens - Affiche les liens spécifiés. Exemple Ajouter les liens que vous utilisez fréquemment dans Infor CRM ou sur Internet.
  • Affiché récemment - Affiche les détails affichés récemment sous forme de liens. Si vous avez affiché les détails d'un compte et d'un contact, les liens vers ces vues Détails s'afficheront dans la liste.

Autres widgets

  • Activités d'aujourd'hui - Affiche une liste de toutes vos activités planifiées pour aujourd'hui. Pour plus d'informations, cliquez sur le lien pour ouvrir la vue Détails de l'activité.
  • Accueil - Affiche une présentation du Client Web Infor CRM, avec un lien vers l'Aide Client Web Infor CRM. Une brève description de toutes les fonctionnalités que Infor CRM propose pour vous aider à gérer votre travail.
  •  SData - Affiche un flux Web utilisant des flux SData (Sage Data). Vous pouvez faire votre sélection sur la liste des flux disponibles dans votre base de donnés Infor CRM ou dans l'intégration de la comptabilité.
  •  Site Web - Affiche un site Web dans un widget sur le tableau de bord. Vous pouvez coller ou taper une URL et définir la hauteur du widget pour déterminer l'espace qu'il occupera sur le tableau de bord.

    		Il est possible que certains sites ne soient pas optimisés pour être ajoutés dans le cadre d'un widget. Vous devez tout d'abord créer un onglet « test » et y ajouter le widget du site web pour vous assurer que le site web se comporte comme prévu avant d'ajouter un widget de site web à un onglet existant.
  •  Flux Web - Affiche un flux Web RSS ou Atom dans un widget sur le tableau de bord. Vous pouvez coller ou taper une adresse de flux Web, déterminer comment s'afficheront les articles du flux et et définir la hauteur du widget pour déterminer l'espace qu'il occupera sur le tableau de bord.
