Utilisation des widgets

 Infor CRM fournit plusieurs widgets prêts à l'emploi. Vous pouvez ajouter, modifier et supprimer des widgets dans le tableau de bord et réorganiser les widgets sur la page via l'outil glisser-déposer.

À l'aide des dimensions prédéfinies qui sont un type de filtre, vous pouvez afficher les métriques pour vos données sur des plages ou à partir de valeurs distinctes et afficher ces données dans l'un des widgets graphiques. Par exemple, vous pouvez configurer un widget graphique de colonne pour afficher toutes les opportunités cloturées et gagnées, par propriétaire et le montant total des ventes, ou un graphique circulaire représentant les ventes d'une équipe.

La liste des widgets disponible est déterminée par votre implémentation. Il est possible que vous n'ayez pas accès à tous les types de widget.

Procédure

Ajout d'un widget
  1. Cliquez avec le bouton droit sur l'onglet Mon tableau de bord, puis cliquez sur Ajouter du contenu.
  2. Dans la boîte de dialogue Ajouter du contenu, cliquez sur le bouton Ajouter correspondant au widget que vous souhaitez ajouter.
    Par exemple, pour ajouter un widget graphique circulaire, cliquez sur Ajouter sous Graphique circulaire.
    Le widget vide apparaît sur le tableau de bord et l'éditeur de widget s'ouvre.
  3. Configurez les paramètres du widget et cliquez sur OK.
Modification d'un widget existant
  1. Sur la barre d'outils du widget, cliquez sur le bouton Paramètres .
  2. Effectuez les modifications nécessaires.
  3. Cliquez sur OK.
    Le widget affiche les modifications.

      • Légende : le texte qui s'affiche sous le titre et donne des informations supplémentaires.

      • Entité - L'entité dont vous souhaitez extraire les données. Par exemple, Opportunités. Toutes les entités avec des vues Liste peuvent être visualisées sous forme de graphiques et de listes.

      • Groupe - Le groupe pour lequel vous voulez afficher les données. Par exemple, Toutes les opportunités. Tous les groupes accessibles à un utilisateur peuvent être utilisés dans des graphiques et des listes.

      • Dimension - Les données que vous souhaitez afficher. Par exemple, Probabilité. Tous les filtres Plage et Distinct actuels sont disponibles sous forme de dimensions. 
         

        Les dimensions disponibles dépendent des filtres d'entité disponibles et d'autres paramètres de configuration. Pour plus d'informations, adressez-vous à votre administrateur système.

      • Métrique - La façon dont vous calculez les données. Par exemple, Montant réel total. Toutes les entités avec des vues Liste ont une métrique Count associée, par exemple, CountLeads et CountAccounts. Il existe des métriques supplémentaires pour les opportunités et l'historique.

      • Axe X - L'étiquette utilisée pour décrire les données sur l'axe X du graphique.

      • Axe Y - L'étiquette utilisée pour décrire les données sur l'axe Y du graphique.

      • Tronquer les étiquettes - Sélectionnez Oui ou Non pour tronquer les étiquettes afin qu'elles apparaissent sur le graphique.

      • Longueur d'étiquette - La longueur maximale d'une étiquette avant qu'elle soit tronquée.

Configurer le widget Liste des groupes
  1. Dans la boîte de dialogue Paramètres de la liste Groupe, saisissez un titre pour le widget.
  2. Sélectionnez une Entité et un Groupe dans les listes déroulantes. Par exemple, Opportunités et Clôturées et gagnées.
  3. Saisissez le Nombre d'enregistrements que vous souhaitez dans la liste. Par exemple, 10, 25 ou 50.
  4. Saisissez le nombre d'enregistrements que vous voulez rendre visibles dans la zone Enregistrements visibles (10, 25 ou 50).
    Par exemple, si vous avez saisi 50 enregistrements à l'étape 3, mais ne souhaitez en voir que 10 à la fois, tapez 10. Le widget se redimensionne en fonction du nombre d'enregistrements visibles.

    Vous pouvez utiliser n'importe quel nombre entre 1 et le nombre maximum d'enregistrements devant s'afficher. Le maximum par défaut est 50.

  5. Cliquez sur OK. 
    Le widget affiche la liste.

    Pour afficher ou masquer des colonnes dans une liste de groupes, cliquez avec le bouton droit de la souris sur un en-tête de colonne, sélectionnez les colonnes que vous voulez ajouter et effacez celles que vous voulez supprimer.

Configurer le widget Liste des liens
  1. Dans la boîte de dialogue Paramètres de lien, saisissez un titre pour le widget.
  2. Cliquez sur Ajouter un lien.
  3. Double-cliquez sur la zone URL de l'icône et saisissez le chemin d'accès de l'icône que vous souhaitez afficher pour le lien. Par exemple, /google.jpg.

    		4.

    Cette URL doit être un chemin relatif vers une image sur le serveur Web.
  4. Double-cliquez sur la zone URL du lien et saisissez le chemin d'accès du lien que vous souhaitez afficher. Par exemple, http://localhost/SlxClient/Account.aspx.

    		5.

    Ce chemin peut être un chemin relatif à l'intérieur du portail SLXClient, ou un lien absolu vers un site externe. L'URL d'un site externe doit inclure l'indicatif http://. Par exemple, pour accéder à www.google.com, l'URL doit être http://www.google.com. Sinon, le navigateur recherchera http://<SLXClient site>/www.google.com.
  5. Double-cliquez sur la zone Texte du lien et saisissez le texte que vous souhaitez afficher comme lien. Par exemple, Page de recherche Google.
  6. Double-cliquez sur la zone Zone cible et saisissez un identifiant pour la cible. Par exemple, si vous souhaitez que le lien s'ouvre dans une nouvelle fenêtre de navigateur, saisissez Nouvelle fenêtre. Lorsque vous cliquerez sur le lien, une nouvelle fenêtre/un nouvel onglet de navigateur s'ouvrira.

    		7.

    Si vous laissez ce champ vide, le lien s'ouvre dans la fenêtre de navigateur active / dans l'onglet actif lorsque vous cliquez dessus. Si plusieurs liens ont la même désignation cible, chaque lien remplace l'autre au fur et à mesure que vous cliquez dessus.
  7. Cliquez sur Ajouter un lien pour ajouter un autre lien ou sur OK pour fermer l'éditeur.
    Lorsque vous cliquez sur OK, le widget est actualisé et affiche les liens.
Configurer le widget SData
  1. Dans la boîte de dialogue Modifier les paramètres du widget, tapez un titre pour le widget.
  2. Cliquez sur la flèche de la liste déroulante Flux SData et sélectionnez un flux sur la liste. Les options incluent :
       
    • Infor CRMEntité dynamique - donne accès à des types de ressources d'entité standard comme les contacts, les comptes et les tickets.
    • Infor CRM Système - donne accès à des types de ressources non standard comme les activités et les pièces jointes.
    • Tous les flux d'intégration de la comptabilité qui sont à votre disposition.
  3. Cliquez sur Suivant.
  4. Si vous recevez un message indiquant que le flux ne contient pas de ressources, vous devez sélectionner un autre flux. Si un flux ne contient pas au minimum une ressource, il n'y aura aucune donnée à afficher.
  5. Cliquez sur la flèche de la liste déroulante Ressource et sélectionnez le type d'information à afficher sur la liste. Par exemple, Compte ou Activité.

    Le contenu de la liste Ressource est fonction du flux que vous avez sélectionné ou entré.

  6. Dans la zone Clause Where, entrez l'instruction "where" qui vous a été fournie. Si une clause Where n'a pas été fournie, contactez votre administrateur.
    Si une clause Where n'est pas définie, toutes les données auxquelles vous avez accès seront retournées pour la ressource SData. Ce peut donner un ensemble très volumineux de données.

    La création d'une clause Where est recommandée aux utilisateurs expérimentés seulement. Pour plus d'informations sur la création d'une clause Where, reportez-vous à Création d'une clause Where.

  7. Cliquez sur Suivant.
  8. Sélectionnez dans votre ressource SData les colonnes que vous voulez afficher dans le widget.
    • Sélectionnez une ou plusieurs colonnes dans la première liste, puis cliquez sur le bouton >. Pour ajouter toutes les colonnes, il est inutile de sélectionner chaque colonne ; il suffit de cliquez sur le bouton >>.
    • Utilisez les boutons < et << pour supprimer les colonnes que vous ne voulez pas afficher.
  9. Dans la zone Enregistrements visibles, tapez une valeur entre 1 et 100 pour déterminer le nombre maximum d'enregistrements à afficher sur votre tableau de bord.
  10. Cliquez sur Terminer.
    Une fois les paramètres de widget enregistrés, le widget affiche les enregistrements du flux. Les informations issues d'un flux SData sont soumises à la sécurité ; par conséquent, vous voyez seulement les enregistrements que vous avez la permission de voir. Si le nombre d'enregistrement est supérieur aux Enregistrements visibles, utilisez la barre de défilement.
Configurer le widget Site Web
  1. Dans la boîte de dialogue Modifier les paramètres du widget, tapez un titre pour le widget.
  2. Dans la zone Adresse de site Web, tapez ou collez l'URL du site Web que vous voulez ajouter au tableau de bord.
    L'URL doit être au format http://website.com

  3. Effectuez l’une des opérations suivantes :

    • Acceptez la Hauteur par défaut de 250 pixels.
      C'est à peu près la même taille qu'un widget Liste de groupes avec 10 enregistrements visibles.
    • Modifiez la Hauteur en utilisant un nombre compris entre 100 et 1000.
      Par exemple, la modification de la hauteur à 500 pixels rendrait le widget deux fois plus haut ; inversement, une hauteur de 125 pixels donnerait un widget deux fois moins haut.
  4. Cliquez sur OK. 
     Le site Web est ajouté au tableau de bord. Si le site Web semble vide, l'Adresse du site Web est incorrecte ou le site Web est peut-être indisponible.
Configurer le widget Flux Web
  1. Dans la boîte de dialogue Modifier les paramètres du widget, tapez un titre pour le widget.
  2. Dans la zone Adresse de flux, tapez ou collez l'URL du flux que vous voulez ajouter au tableau de bord.
  3. Cliquez sur la flèche de la liste déroulante Afficher et sélectionnez une des options suivantes :
    • Titre seulement - affiche seulement le titre de chaque article de flux.
    • Titre et résumé - affiche le titre et un résumé de chaque article de flux.
  4. Dans la zone Maximum de, tapez un nombre entre 1 et 25 pour déterminer le nombre maximum d'éléments de flux à afficher sur le tableau de bord.
    Selon le flux Web, il est possible que moins d'éléments s'affichent sur le tableau de bord. Le nombre par défaut est 10, qui constitue une limite courante sur de nombreux flux Web.

  5. Effectuez l’une des opérations suivantes :

    • Acceptez la Hauteur par défaut de 250 pixels.
      C'est à peu près la même taille qu'un widget Liste de groupes avec 10 enregistrements visibles.
    • Modifiez la Hauteur en utilisant un nombre compris entre 100 et 1000.
      Par exemple, la modification de la hauteur à 500 pixels rendrait le widget deux fois plus haut ; inversement, une hauteur de 125 pixels donnerait un widget deux fois moins haut.
  6. Cliquez sur OK. 
     Le flux Web est ajouté au tableau de bord.
Réduire un widget
  • Sur la barre d'outils du widget, cliquez sur le bouton Réduire .
    Le contenu du widget est réduit dans le tableau de bord. Cliquez de nouveau sur le bouton agrandir le widget.
Supprimer un widget
  • Sur la barre d'outils du widget, cliquez sur le bouton Fermer .
    Le widget est définitivement supprimé du tableau de bord.
