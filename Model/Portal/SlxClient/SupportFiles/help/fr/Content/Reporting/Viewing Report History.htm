Affichage, impression et stockage de rapports

Lorsque vous exécutez un rapport, il est stocké dans l'onglet Historique des rapports jusqu'à ce que vous décidiez de le supprimer. Dans cet onglet, vous pouvez afficher l'historique du rapport, ouvrir le rapport, l'imprimer, l'enregistrer dans un autre dossier ou le stocker pour le consulter ultérieurement.

 

Pour accéder à l'onglet Historique des rapports

  1. Dans la barre de navigation, cliquez sur Rapports.
  2. Dans la vue Liste des rapports, cliquez sur l'onglet Historique.

Pour ouvrir un rapport

  • Dans l'onglet Historique des rapports, cliquez sur le nom du rapport dans la colonne Rapport.
    Le rapport s'ouvre dans l'application native au type de fichier.

Pour imprimer ou enregistrer le rapport

  • Utilisez les outils disponibles dans l'application native.

Pour supprimer un rapport

  1. Dans la grille de l'onglet Historique des rapports, sélectionnez le rapport.
  2. Cliquez avec le bouton droit de la souris, puis cliquez sur Supprimer.
