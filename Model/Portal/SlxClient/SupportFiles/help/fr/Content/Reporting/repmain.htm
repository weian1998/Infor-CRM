Aperçu du rapport

Vos rapports terminés sont stockés sur l'onglet Historique des rapports.

Vous pouvez uniquement visualiser les rapports ayant été diffusés à votre attention ou à celle d'une équipe à laquelle vous appartenez. Vous ne pouvez pas enregistrer de rapports dans le Client Web Infor CRM.

Vous devez autoriser les pop-up avant de pouvoir exporter ou imprimer un rapport.

Procédure

Afficher ou imprimer un rapport détaillé depuis une vue Détails

Afficher ou imprimer un rapport à partir de l'onglet Historique des rapports
  1. Sur la barre de navigation, cliquez sur Rapports, puis sur l'onglet Historique.
  2. Localisez le rapport de votre choix. Vous pouvez trier par titres de colonne de grille.
  3. Cliquez sur le rapport.
  4. Le rapport s'ouvre dans le visualiseur natif au type de fichier. Par exemple, Acrobat Reader pour formats PDF.
Prévisualiser un rapport dans le Visualiseur
  1. Sélectionnez le rapport à visualiser, puis cliquez sur Afficher le rapport. Le rapport apparaît dans le Visualiseur de rapports. Selon le rapport sélectionné et le nombre de résultats obtenus, une arborescence peut s'afficher sur la partie gauche de la fenêtre. Elle donne la liste des pages du rapport par ordre alphanumérique ou chronologique.
  2. Pour afficher une page spécifique du rapport, cliquez sur le signe plus pour développer la liste, puis cliquez sur un nom dans l'arborescence.
