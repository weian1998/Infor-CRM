Assistant de planification des rapports

L'assistant de planification des rapports permet de modifier les paramètres des rapports afin d'ajuster le contenu du rapport sélectionné selon vos besoins.

Pour modifier les paramètres

  1. Dans la boîte de dialogue Sélectionner des enregistrements, dans le champ Afficher les enregistrements correspondants, sélectionnez l'une des options suivantes :
  • Tous : inclut tous les enregistrements pour le type de rapport sélectionné.
  • Specific Conditions (Conditions particulières) : permet d'ajouter des paramètres de filtre au rapport.
  • Current User (Utilisateur actuel) : inclut tous les enregistrements de l'utilisateur connecté.
  1. Si cette fonctionnalité est disponible, dans la liste de sélection Faire correspondre, vous pouvez choisir de filtrer avec tous les paramètres sélectionnés ou l'un de ces paramètres.
  • Tous affiche uniquement les enregistrements correspondant exactement à tous les paramètres.
  • Any (N'importe lequel) affiche tous les enregistrements correspondant à l'un des paramètres de filtre.
  1. Si cette fonctionnalité est disponible, cliquez sur l'icône Plus pour ajouter des paramètres de filtre à l'aide de la boîte de dialogue Ajouter/modifier les conditions, puis cliquez sur OK.
  2. Cliquez sur Suivant.
  3. Si cette fonctionnalité est disponible, dans la boîte de dialogue Enter Values (Entrer les valeurs), sélectionnez l'une des options disponibles pour personnaliser le rapport de tâches selon vos besoins.
  4. Dans la boîte de dialogue Export Options/Export and Scheduling Options (Options d'exportation/Options d'exportation et de planification), procédez comme suit :
    1. Dans le champ Description, sélectionnez le texte par défaut ou personnalisez le.
      Ce texte apparaît dans l'onglet Reports Schedules (Programmes des rapports) de la colonne Description.
    2. Sélectionnez le format de fichier de votre rapport. En fonction du type de tâche exécutée, les options disponibles sont PDF, Word, Excel, CSV et XML.
    3. (Administrateur uniquement) Exécutez d'autres rapports d'utilisateur à partir du champ Run As (Exécuter en tant que). Utilisez la fonction de recherche pour sélectionner un autre utilisateur.
    4. Si vous avez sélectionné Programme à l'étape 1, vous pouvez créer un programme de tâches pour exécuter votre rapport. Faites votre sélection à partir des options de programmation disponibles.
    5. Cliquez sur Terminer.
