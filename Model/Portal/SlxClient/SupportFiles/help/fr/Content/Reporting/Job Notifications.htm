Notifications

Une alerte de notification de tâches s'affiche dans la barre de menus pour vous informer que vous pouvez consulter les résultats. Cliquez sur la flèche de la liste déroulante pour afficher les détails de la notification et accéder aux résultats de la tâche. Le chiffre à côté de l'icône indique le nombre de tâches pour lesquelles vous pouvez consulter les résultats.

Le champ Notifications de tâches contient le type de rapport, la progression et l'état. Un lien dans la colonne Résultat vous permet d'ouvrir le fichier généré par la tâche. Les taches s'ouvrent dans l'application native du type de fichier généré.

Tous les types de tâches prédéfinis suivants apparaissent dans Notifications de tâches :

  • Exécutez les rapports à partir de la vue Liste des rapports ou les vues Détails du contact, Détails du compte, Détails de l'opportunité, Détails du ticket, Détails du défaut ou Détails des bons de commande.
  • Rapports planifiés
  • Exporter vers Excel
  • Le report des activités incomplètes
  • Mise à jour des journées restantes pour les contrats
  • La mise à jour des opportunités
  • Mise à jour ou suppression de leads via le volet des tâches.

Les tâches sont visibles dans la boîte de dialogue Notifications de tâches pendant une heure. Passé ce délai, vous pouvez trouver le fichier généré par la tâche dans l'onglet Historique des rapports.
