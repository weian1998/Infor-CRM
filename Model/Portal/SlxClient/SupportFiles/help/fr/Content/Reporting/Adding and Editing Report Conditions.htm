Ajout et modification de conditions de rapport

Les conditions de rapport vous permettent de modifier le contenu d'un rapport prédéfini pour générer les données les plus utiles. Les options à votre disposition varient selon les sélections que vous faites.

Faire correspondre par  
Intervalle de dates

Sur la liste de sélection, sélectionnez une des périodes par défaut ou utilisez Dates spécifiques pour sélectionner une période personnalisée.

Groupe

Les groupes sont des ensembles d'enregistrements. Vous pouvez utiliser des groupes pour travailler sur un sous-ensemble d'enregistrements.

Utilise la Recherche pour faire votre sélection sur une liste de groupes.

Requête

Sur les listes de sélection, sélectionnez ce qui suit :

  • Table,
  • Champ
  • Opérateur
  • Valeur : tapez une valeur ou sélectionnez une date.

Par exemple,

Table : Opportunity

Champ : CREATEDATE

Opérateur : Est postérieur

Date : 1er octobre.
Utilisateur

Utilisez la Recherche pour faire votre sélection sur une liste d'utilisateurs.
