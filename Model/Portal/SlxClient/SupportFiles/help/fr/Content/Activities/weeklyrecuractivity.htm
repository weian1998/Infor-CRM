Activités périodiques, Toutes les semaines

Vous pouvez choisir de planifier une activité pour qu'elle ait lieu une fois, tous les jours, toutes les semaines, tous les mois ou tous les ans.

Pour planifier une activité avec une périodicité hebdomadaire

  1. Planifiez ou modifiez une activité.
  2. Cliquez sur l'onglet Périodicité.
  3. Sous Fréquence, sélectionnez Hebdomadaire.
  4. Sélectionnez le type de fréquence de l'activité.
    • Si vous souhaitez que l'activité se répète un jour particulier de la semaine, sélectionnez Chaque X Semaine(s), saisissez le nombre de semaines entre les répétitions, puis sélectionnez les jours de la semaine où vous souhaitez que l'activité se répète.
    • Si vous souhaitez que l'activité ne se répète qu'une fois les occurrences précédentes terminées, sélectionnez l'option Chaque X Semaine(s) après la fin, puis saisissez le nombre de semaines entre les répétitions.
  5. Dans la zone Démarrer la périodicité, cliquez sur le BoutonAgenda, sélectionnez une date puis cliquez sur OK.

  6. Effectuez l’une des opérations suivantes :

    • Dans la zone Terminer après, saisissez le nombre de fois que l'activité a lieu.
    • Cliquez sur Terminer la périodicité BoutonAgenda, sélectionnez une autre date puis cliquez sur OK. 
  7. Lorsque vous avez terminé toutes les modifications d'activité, cliquez sur OK.
