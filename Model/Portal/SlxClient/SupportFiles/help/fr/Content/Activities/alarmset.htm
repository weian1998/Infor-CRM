Configuration des alarmes

Les alarmes vous avertissent qu'une activité est sur le point de se terminer. Vous pouvez paramétrer une alarme pour une activité existante, ou lorsque vous planifiez une activité. Vous devez avoir paramétré les alertes pour inclure les alarmes afin que vos alarmes vous soient signalées.

Pour régler l'alarme

  1. Effectuez l’une des opérations suivantes :
  • Planifier une nouevlle activité.
  • Rechercher l'activité pour laquelle vous souhaitez paramétrer une alarme, puis ouvrez la vue Détails de l'activité.
  1. Sélectionnez la case Alarme, sélectionnez le moment auquel vous souhaitez que l'activité vous soit rappelée.
  2. Lorsque vous avez terminé, cliquez sur OK.
