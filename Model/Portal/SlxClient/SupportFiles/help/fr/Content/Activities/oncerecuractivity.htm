Activités périodiques, Une fois

Vous pouvez choisir de planifier une activité pour qu'elle ait lieu une fois, tous les jours, toutes les semaines, tous les mois ou tous les ans.

Pour planifier une activité isolée qui se répète une fois

  1. Planifiez ou modifiez une activité.
  2. Cliquez sur l'onglet Périodicité.
  3. Sous Fréquence, sélectionnez Une fois. Cette option est sélectionnée par défaut.
  4. Lorsque vous avez terminé toutes les modifications d'activité, cliquez sur OK.
