Onglet Documentation

L'onglet Documentation de la vue Activités répertorie les demandes de documentation planifiées par un ou des utilisateurs sélectionnés. La documentation peut comprendre des catalogues, des brochures, des publicités, etc.

Procédure

Afficher des demandes de documentation
  1. Ouvrez la vue Activités.
  2. Cliquez sur l'onglet Documentation.
    La liste de toutes les demandes de documentation auxquelles vous avez accès apparaît.
  3. Si nécessaire, utilisez les filtres pour restreindre la liste.
  4. Cliquez sur Afficher pour ouvrir une demande de documentation ou faites un clic droit sur une demande et cliquez sur Ouvrir une demande de documentation.
Afficher ou masquer le volet des détails
  1. Cliquez sur Détails.
  2. Sélectionnez un enregistrement.
    Le volet des détails apparaît en bas de la vue Liste et affiche des informations détaillées sur l'enregistrement sélectionné.
  1. Pour masquer le volet des détails, cliquez sur Masquer les détails.
Afficher le volet de résumé
  1. Cliquez sur Résumé.
  2. Pour retourner vers la vue Liste, cliquez sur Liste.
Planifier une demande de documentation

  1. Effectuez l’une des opérations suivantes :

    • Dans la zone Tâches communes du volet Tâches sous Planifier, cliquez sur Demande.
    • Cliquez avec le bouton droit de la souris dans la grille et cliquez sur Planifier la demande.
  2. Suivez les étapes permettant de planifier une demande de documentation.
