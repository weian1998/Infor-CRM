Présentation de la requête

Utilisez la boîte de dialogue Affecter une présentation à la requête pour indiquer l'apparence de chaque colonne de données (ou champ) dans la requête.

Pour définir les options de présentation

  1. Ouvrez Query Builder et cliquez sur l'onglet Présentation.
  2. Ajouter ou modifier un champ Présentation.
  3. Dans la zone Légende, saisissez le nom qui s'affichera dans l'en-tête de la colonne.
  4. Dans la zone Largeur, sélectionnez la largeur de la colonne. La valeur par défaut est 100.

Les modifications de la largeur des groupes ne sont actives que pour la session en cours.

  1. Dans la zone Alignement de légende, sélectionnez l'alignement de l'en-tête de la colonne.
  2. Dans la zone Alignement texte, sélectionnez l'alignement des données.
  3. Dans la zone Type de format, sélectionnez le format adapté au type de données affiché dans ce champ.
     

    Le format de numéro de téléphone formate la valeur d'un champ sous la forme d'un numéro de téléphone standard à 10 chiffres propre aux Etats-Unis. Si votre société n'utilise pas ce format standard à 10 chiffres, il est conseillé de paramétrer le format de numéro de téléphone sur Aucun pour empêcher l'application d'un masque de numéro de téléphone.

  4. Si besoin est, saisissez une chaîne de formatage dans la zone Format pour modifier le type de format par défaut que vous venez de sélectionner.
  5. Pour lier les informations de la colonne à une vue de portail d'entité, sélectionnez Lier aux entités.
  6. Pour appliquer un format de feuilles de styles en cascade (CSS), saisissez la classe à appliquer dans la zone Classe Css.
