Gestionnaire de jointures globales

 Administrateur système uniquement

  • Faites très attention au moment de créer des jointures globales car vous pourriez en créer une qui provoquerait de sérieux problèmes de performances.
  • Par défaut, la table ADDRESS est liée à la table ACCOUNT grâce à une jointure interne. Ne modifiez pas les propriétés de cette jointure.

Le Gestionnaire de jointures globales affiche toutes les jointures globales de votre base de données Infor CRM. Il permet à l'administrateur système ou à l'utilisateur disposant de droits d'administration d'ajouter, de modifier et de supprimer des jointures globales en fonction des besoins. Une jointure globale est permanente et existe de session en session dans la structure de l'arborescence des données de Query Builder.

Infor CRM contient une série de jointures globales prédéfinies (ou jointures système) qui ne peuvent pas être supprimées. Tout utilisateur peut employer ces jointures globales prédéfinies en sélectionnant un champ dans Query Builder et en cliquant sur le bouton Jointure globale. S'il existe une jointure prédéfinie pour ce champ, l'option qui permet de la visualiser figure dans le menu contextuel sous l'intitulé « Jointure globale à » suivi d'un nom de table.

Le Gestionnaire de jointures globales n'est disponible que si vous êtes connecté en tant qu'administrateur.

Procédure

Ouvrez le Gestionnaire de jointure globale
  1. Ouvrir Query Builder
  1. Ouvrez Query Builder, puis cliquez sur le bouton Jointures globales.

Ce bouton n'apparaît que si vous êtes connecté en tant qu'administrateur.

Le Gestionnaire de jointures globales affiche les champs suivants :

Champ Description
Syst. Jointure système ne pouvant être supprimée.
1:1 Indique une relation un-à-un.
Table parent, Champ parent Table et champ principaux dans la relation de jointure.
Table enfant, Champ enfant Table et champ joints à la Table parent et au Champ parent.
Type Type de jointure (interne, gauche ou droite).
Cascade Façon dont les enregistrements sont supprimés dans une jointure.
Visibilité Affichage de la jointure dans Query Builder.
