Types de champs calculés

Il existe deux types de calculs : chaîne et numérique. Lorsque vous générez un champ calculé à partir de chaînes (valeurs alphanumériques), vous créez une concaténation, ou une connexion, des valeurs pour qu'elles n'en forment qu'une. Les champs numériques ne peuvent pas être utilisés dans une chaîne. Lorsque vous générez un champ calculé à l'aide d'un ou de plusieurs champs numériques, vous effectuez une opération mathématique (addition, soustraction, multiplication ou division) sur les valeurs de chaque champ afin de créer une nouvelle valeur. Vous pouvez également utiliser des constantes dans les calculs numériques. Les champs non numériques (chaînes) ne peuvent pas être appliqués aux calculs numériques.
