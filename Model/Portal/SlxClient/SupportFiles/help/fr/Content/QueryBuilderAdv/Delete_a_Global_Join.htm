Suppression d'une jointure globale

 Administrateur système uniquement

Une jointure globale est permanente et existe de session en session dans la structure de l'arborescence des données de Query Builder. Vous ne pouvez pas supprimer les jointures globales (jointures système) prédéfinies.

Pour supprimer une jointure globale

  1. Ouvrez Query Builder, puis cliquez sur le bouton Jointures globales.

Ce bouton n'apparaît que si vous êtes connecté en tant qu'administrateur.

  1. Sélectionnez la jointure à supprimer.
  2. Cliquez sur Supprimer.
  3. Cliquez sur OK.
