Utiliser la valeur comme une valeur littérale

La fonction Utiliser la valeur comme une valeur littérale de la boîte de dialogue Assigner une condition place le texte dans la requête exactement comme il apparaît dans la zone La valeur est. Dans la plupart des cas, vous ne devez pas sélectionner cette option afin que Query Builder puisse déterminer la bonne instruction SQL. En revanche, les utilisateurs avancés peuvent parfois vouloir saisir un texte spécifique directement dans l'instruction SQL.

Par exemple, vous pouvez saisir une fonction système propre à la base de données (GETDATE dans une base de données de Microsoft SQL Server, par exemple) et sélectionner Utiliser la valeur comme un littéral. Ceci vous assure que la commande GETDATE est insérée dans l'instruction SQL sans guillemet.
