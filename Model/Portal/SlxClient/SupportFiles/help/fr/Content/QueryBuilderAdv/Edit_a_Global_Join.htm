Modification d'une jointure globale

 Administrateur système uniquement

Une jointure globale est permanente et existe de session en session dans la structure de l'arborescence des données de Query Builder. Concernant les jointures globales (jointures système) prédéfinies, vous ne pouvez modifier que les options Type de jointure, Visibilité et Type de cascade.

Pour modifier

  1. Ouvrez Query Builder, puis cliquez sur le bouton Jointures globales.

Ce bouton n'apparaît que si vous êtes connecté en tant qu'administrateur.

  1. Dans la boîte de dialogue Gestionnaire de jointures globales, sélectionnez sur la jointure à modifier puis cliquez sur Modifier.
  2. Modifiez le Type de jointure, la Visibilité ou le Type de cascade.
  3. Cliquez sur OK..
