Ajout d'une jointure locale

Une jointure locale est temporaire et n'existe que dans une seule session de Query Builder. L'objet utilisant la requête obtenue conservera les propriétés de cette jointure locale pendant toute son utilisation.

Pour ajouter

  1. Ouvrir Query Builder
  2. Sélectionner une table.
  3. Sélectionnez un champ.
  4. Cliquez sur le lien Créer une jointure locale.

Dans la boîte de dialogue jointure locale, la table et le champ sélectionnés dans Query Builder apparaissent sous la section Table parent et champ de jointure.

  1. Sous Table enfant et champ de jointure, sélectionnez la table et le champ de la base de données vers lesquels vous souhaitez créer une jointure locale.
  2. Sous Type de jointure, sélectionnez le type de jointure.

L'option par défaut est Jointure interne.

  1. Cliquez sur OK.
