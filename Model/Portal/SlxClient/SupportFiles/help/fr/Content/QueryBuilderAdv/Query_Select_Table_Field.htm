Sélection des tables et des champs

Vous pouvez ajouter un champ à afficher dans une requête.

Pour sélectionner une table

  • Sélectionnez une table dans le côté gauche de Query Builder. Si besoin, cliquez sur le signe plus pour développer la liste des tables.

Pour sélectionner un champ

  • Sélectionnez une table du côté droit de Query Builder. Cette zone affiche tous les champs appartenant à la table choisie. Double-cliquez sur le champ pour l'ajouter à la grille au bas de la boîte de dialogue.
