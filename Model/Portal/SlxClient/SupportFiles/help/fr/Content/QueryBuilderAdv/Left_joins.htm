Jointures gauches

Les jointures gauches sont le type de jointure globale par défaut. Utilisez une jointure gauche pour inclure tous les enregistrements de la table parent qui répondent aux critères et ceux de la table enfant dont les valeurs des champs que vous êtes en train de lier sont égales.

Exemple : Vous souhaitez obtenir une liste de toutes les opportunités, que des activités y soient associées ou non. Si l'opportunité est associée à une activité, sélectionnez-la également. Les enregistrements affichés dans la colonne Résultats de jointure sont renvoyés par cette requête. L'opportunité spécifique, associée à une activité, est indiquée entre parenthèses, à droite de l'activité.
