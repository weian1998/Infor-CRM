Affichage de la version SQL d'une requête

Lorsque vous créez une requête, elle est convertie en instruction SQL (Structured Query Language). Cette instruction est ensuite appliquée sur la base de données. Vous pouvez consulter la version SQL d'une requête, mais vous ne pouvez pas modifier une instruction SQL.

Afficher l'instruction SQL pour une requête peut se révéler très utile pour comprendre le mode de fonctionnement de la requête. Cette option peut également être utile si vous tentez d'utiliser le langage SQL dans d'autres parties du programme et si vous souhaitez voir comment le code SQL fonctionne.

Pour afficher la version SQL d'une requête

  1. Ouvrez Query Builder.
  2. Cliquez sur Afficher SQL.
  3. Après avoir vu l'instruction SQL, fermez la boîte de dialogue.
