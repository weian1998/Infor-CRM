Champs masqués

Certains champs sont définis comme champs masqués dans le Gestionnaire de base de données. Les champs masqués peuvent s'afficher mais sont invisibles par défaut dans Query Builder.

Pour masquer des champs masqués

  • Dans Query Builder, cliquez sur Masquer les champs masqués.

Pour afficher les champs masqués

  • Dans Query Builder, cliquez sur Afficher les champs masqués.
    Cette option est uniquement disponible après avoir masqué les champs masqués.
