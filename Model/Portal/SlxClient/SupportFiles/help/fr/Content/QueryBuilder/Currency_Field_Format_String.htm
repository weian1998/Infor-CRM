Chaînes de format de champ Devise et Téléphone

Le tableau ci-après dresse la liste des options de chaîne de formatage pour les champs Devise et Numéro de téléphone.

Type de champ Option Description
Devise m Argent. Le format de ce champ doit être de type valeur à virgule flottante. La valeur est convertie en une chaîne qui représente la quantité en devise.
Numéro de téléphone # Utilisez le signe # pour indiquer comment doit s'afficher un numéro. Vous pouvez utiliser tous les caractères en tant que séparateur, à l'exception des lettres et des chiffres. Le format de numéro de téléphone par défaut est (###) ###-####.

Une chaîne de formatage doit commencer par un signe % et se terminer par un caractère de type de format :

[%] [type]
