Séparateurs Date/Heure

Le tableau suivant dresse la liste des séparateurs Date/Heure.

Séparateur Description
/ Affiche le caractère de séparation de la date fourni par la variable globale SéparateurDate.
: Affiche le caractère de séparation de l'heure fourni par la variable globale SéparateurHeure.
\x Les caractères précédés d'une barre oblique inverse « \ » sont affichés tels quels et n'affectent pas le format.
