Conditions de Query Builder

Utilisez l'onglet Conditions de Query Builder pour ajouter, modifier ou supprimer des instructions conditionnelles qui sont appliquées à la base de données pour sélectionner des informations. Les conditions s'appliquent dans l'ordre dans lequel elles sont répertoriées dans la grille ; cependant, vous pouvez modifier cet ordre à l'aide des boutons Monter et Descendre. Vous pouvez également regrouper les instructions conditionnelles.
  • Le Query Builder prend en charge de nombreuses fonctionnalités. Les onglets visibles dépendent de la fonctionnalité que vous êtes en train d'utiliser.
  • Il est recommandé d'insérer au moins une condition dans chaque groupe que vous créez. Les groupes incluant tous les enregistrements peuvent ralentir la performance du système, et un avertissement risque d'apparaître lorsque les utilisateurs tentent de visionner ces groupes.
Pour ajouter une instruction conditionnelle
  1. Ouvrez Query Builder.
  2. Dans Query Builder, cliquez sur l'onglet Conditions.
  3. Sélectionnez une table dans le coin supérieur gauche de Query Builder. Si besoin, cliquez sur le signe plus pour développer la liste des tables.
  4. Sélectionnez un champ dans le coin supérieur droit de Query Builder. Cette zone affiche tous les champs appartenant à la table choisie. Double-cliquez sur le champ pour l'ajouter à la grille au bas de la boîte de dialogue.

La boîte de dialogue Affecter une condition s'ouvre automatiquement. Vous pouvez ouvrir cette boîte de dialogue en double-cliquant sur une instruction de la grille.

  1. Sélectionnez un opérateur.
  2. Dans la zone Valeur, saisissez une valeur pour ce champ ou cliquez sur Parcourir pour la sélectionner dans une liste d'entrées existantes.

Si vous sélectionnez « In » en tant qu'opérateur, les valeurs doivent être entourées de guillemets à chaque entrée.

  1. Pour que la casse soit prise en compte dans l'évaluation de la condition, sélectionnez Comparaison sensible à la casse. Cette option est sélectionnée par défaut.

Si vous utilisez MS SQL Server, cette option n'est pas disponible. Les requêtes ne sont pas sensibles à la casse.

  1. Si besoin est, sélectionnez Utiliser la valeur comme une valeur littérale.
  2. Cliquez sur OK. 
  3. Pour ajouter une autre instruction de condition, répétez les étapes 1 à 9, puis sélectionnez le connecteur approprié.
Pour modifier une instruction conditionnelle
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Conditions.
  3. Sélectionnez une instruction dans la grille (au bas de la boîte de dialogue Query Builder).
  4. Cliquez sur Modifier.
  5. Dans la boîte de dialogue Affecter des conditions, effectuez vos modifications.
  6. Cliquez sur OK. 
Pour supprimer une instruction conditionnelle
  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Conditions.
  3. Sélectionnez une instruction dans la grille.
  4. Cliquez sur Supprimer.
