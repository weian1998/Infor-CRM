Chaînes de format de champ Entier

Le tableau suivant dresse la liste des options de chaîne de formatage pour les champs de nombres entiers. La validité d'un format de chaîne dépend du type de format sélectionné.

Une chaîne de formatage doit commencer par un signe % et se terminer par un caractère de type de format :

[%] [type]
Option Description
d Décimal. La valeur doit être un nombre entier. La valeur est affichée en tant que nombre décimal.
x Hexadécimal. Le format du champ doit être une valeur entière. La valeur est convertie en une chaîne de chiffres hexadécimaux. Si la chaîne de formatage contient un spécificateur de précision, la chaîne résultante doit alors contenir au moins le nombre spécifié de chiffres ; si ce n'est pas le cas, la chaîne est remplie de zéros à gauche.
