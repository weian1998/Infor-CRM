Utilisation de « Not » dans une instruction conditionnelle

Vous pouvez insérer NOT devant une instruction conditionnelle ou un groupe d'instructions conditionnelles. Si NOT est inséré devant un groupe d'instructions, il s'appliquera à toutes les instructions conditionnelles de ce groupe.

Pour insérer NOT devant une instruction conditionnelle ou un groupe

  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Conditions et ajoutez toutes les instructions conditionnelles.
  3. Si vous le souhaitez, vous pouvez regrouper les instructions conditionnelles.
  4. Cochez la case Not d'une instruction conditionnelle ou la première instruction d'un groupe.
  5. Cliquez sur OK pour exécuter la requête.

Exemple

Query Builder sélectionnera les contacts vivant partout en Californie, sauf à Los Angeles ou San Francisco.

Voici comment la grille de l'onglet Conditions apparaîtra selon cet exemple :

Non ( Champ Opérateur Valeur Respecter la casse ) Et/ou
NON ( Contact.Address.City égal à Los Angeles Vrai   OR
    Contact.Address.City égal à San Francisco Vrai ) AND
    Contact.Address.State égal à CA Vrai   END

Le groupe d'une requête peut contenir des sous-groupes. Si NOT est utilisé devant un groupe, il s'appliquera à tous les sous-groupes de ce dernier.
