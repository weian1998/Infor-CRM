Propriétés de Query Builder

Utilisez l'onglet Propriétés de Query Builder pour saisir le nomet l'objectif d'une requête.

Le Query Builder prend en charge de nombreuses fonctionnalités. Les onglets visibles dépendent de la fonctionnalité que vous êtes en train d'utiliser.

Pour définir les propriétés de Query Builder

  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Propriétés.
  3. Dans le champ Nom, entrez le nom de votre requête.

Si vous créez un groupe, le nom de la requête apparaît dans le gestionnaire de groupes. 

  1. Dans le champ Nom d'affichage, entrez le nom tel que vous souhaitez qu'il apparaisse.

Si vous créez un groupe, ce nom s'affiche sur l'onglet Groupe dans les vues de listes. Si aucun Nom d'affichage n'est saisi, le texte saisi dans le champ Nom sera afficher sur l'onglet.

  1. Dans la zone Description, saisissez une description pour identifier l'objectif de la requête.
