Réordonner une instruction conditionnelle 

Si une requête contient plusieurs instructions conditionnelles, vous pouvez en déplacer une vers le haut ou vers le bas pour changer l'ordre dans lequel elles seront appliquées à la requête.

Pour réorganiser une instruction conditionnelle

  1. Ouvrez Query Builder.
  2. Cliquez sur l'onglet Conditions.
  3. Sélectionnez une instruction dans la grille.
  4. Cliquez sur Monter ou Descendre.
