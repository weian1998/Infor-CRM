Chaînes de format de champ booléen

La chaîne de formatage permettant de personnaliser une chaîne booléenne dépend du résultat souhaité. Renseignez les valeurs Vrai/ Faux. Les valeurs utilisées doivent être composées de caractères alphanumériques et inclure un caractère / entre les entrées Vrai et Faux. Vrai l'emporte toujours sur Faux. Le format par défaut du FormatType ftBoolean est Oui/Non.

Par exemple

FormatType:ftBoolean

FormatString:Achat produit/Pas intéressé

Données (non formatées):T

Résultat:Achat produit
