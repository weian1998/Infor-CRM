Mise en route du Client Web Infor CRM

Le Client Web Infor CRM vous aide à gérer tous les aspects du cycle de vente par Internet. Cette rubrique inclut des liens vers les tâches courantes et une présentation des rubriques pour vous aider à vous familiariser avec le Client Web Infor CRM.

Votre espace de travail peut être différent selon vos droits d'accès ou si votre installation a été personnalisée.
