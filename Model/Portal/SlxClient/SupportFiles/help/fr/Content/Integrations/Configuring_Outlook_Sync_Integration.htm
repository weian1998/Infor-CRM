Configuration de l'intégration d'Outlook Sync

Procédez comme suit pour configurer l'intégration d'Outlook Sync.

  1. Exécutez le fichier SLXConversionUtility.exe.
    Une fois suffit pour la prise en charge de toutes les intégrations.
  2. Configurer laInfor CRM synchronisation SData.
    Une fois suffit pour la prise en charge de toutes les intégrations. .
  3. Configurez le service de synchronisation d'intégration.
  4. Arrêtez le service de synchronisation (service) xsSDatade Infor CRM et le service de tâches (service) de Infor CRM sur le serveur Web si l'un deux est en cours d'exécution.
  5. Dans la vue Liste des intégrations, sélectionnez OutlookSync.
    Une fois suffit pour la prise en charge de toutes les intégrations.
  6. Examinez les zones d'informations.
  7. Sous Type de lien, sélectionnez Utilisateur.
  8. Cochez la case Activé.
  9. Définissez les options de synchronisation.
    1. Dans le champ Synchroniser toutes les, tapez le nombre de minutes qui doit s'écouler entre le début des cycles de synchronisation.
    2. Dans les champs Synchroniser entre, cliquez sur le BoutonAgenda et sélectionnez l'heure de début et l'heure de fin marquant la période durant laquelle la synchronisation doit avoir lieu. Infor CRM recommande que les heures sélectionnées ne chevauchent pas minuit.

      Lorsque vous activez ou désactivez une intégration, vous devez vous déconnecter puis vous reconnecter à Infor CRM pour que ces modifications prennent effet.

  10. Cliquez sur Enregistrer.

  11. Accordez les droits d'accès appropriés :

    1. Configurez le rôle Intégration pour chaque utilisateur auquel vous voulez accorder l'accès administrateur aux intégrations autres que celles du système de comptabilité.
    2. Pour donner à des utilisateus individuels l'accès à des tâches d'intégration non administratives, ajoutez les actions sécurisées au rôle Utilisateur standard :
      • Integration/View
      • Entities/SyncHistory/View
      • Entities/SyncJob/View
