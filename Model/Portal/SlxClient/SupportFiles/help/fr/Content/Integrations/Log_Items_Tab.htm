Onglet Eléments du journal

L'onglet Eléments du journal affiche les messages générés lorsqu'une synchronisation est effectuée. Pour définir les informations de consignation que vous voulez afficher, utilisez les filtres proposés sur cet onglet.

Pour afficher

  1. Ouvrez la vue Détails de tâche de synchronisation de la tâche qui contient les éléments que vous voulez afficher.
  2. Dans la liste Filtrer par type de journal, sélectionnez le type de message d'erreur à afficher.
  • Débogage : affiche les erreurs détaillées pour vous aider à résoudre les problèmes.
  • Erreurs : affiche des informations si un problème significatif survient, tel que la perte de données ou d'une fonctionnalité.
  • Informationnel : affiche les actions effectuées par le service.
  • Avertissements : affiche un événement survenu non nécessairement significatif, mais qui peut indiquer un éventuel problème à venir.
  1. Dans la liste Filtrer par type de ressource, sélectionnez l'entité du type de journal sélectionné.
  2. Cliquez sur le bouton Actualiser les éléments du journal.
  3. Cliquez sur le lien Sélectionner d'un journal dans la grille. Les informations détaillées s'affichent dans la zone Pile d'appels.
