Onglet Intégrations de l'utilisateur

Utilisez l'onglet Intégrations pour afficher toutes les intégrations associées à l'utilisateur.

Procédure

Ouvrir l'onglet Intégrations
  • Dans la vue Détails de l'utilisateur, cliquez sur l'onglet Intégrations. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Activer la synchronisation pour une intégration
  • Sélectionnez une ou plusieurs intégrations sur la liste, puis cliquez sur le bouton Activer la synchronisation.
Désactiver la synchronisation pour une intégration
  • Sélectionnez une ou plusieurs intégrations sur la liste, puis cliquez sur le bouton Désactiver la synchronisation.

La grille de l'onglet Intégrations comporte les colonnes suivantes :

  • Activer la synchronisation - Cochez la case pour activer la synchronisation d'un utilisateur ou supprimez-la pour la désactiver.
  • Intégration - Affiche le nom de l'intégration et des liens à la vue Détails de l'intégration.
  • Nom d'affichage - Nom d'affichage du lien d'intégration qui synchronise les informations de l'utilisateur.
  • Dernière synchronisation - Affiche la date et l'heure de dernière synchronisation de l'intégration.
  • Mappage de l'intégration - Affiche sur quoi l'utilisateur est mappé dans le système d'intégration. Cliquez dans la grille pour modifier le mappage d'intégration.
