Onglet Liens d'intégration

Utilisez l'onglet Liens pour afficher et gérer les liens qui transfèrent les informations entre Infor CRM et un système externe. Selon le type de lien, vous pourrez voir un ou plusieurs liens sur l'onglet.

L'onglet Liens est disponible uniquement pour les intégrations avec une méthode de synchronisation côté client ou côté serveur.

Procédure

Ouvrir l'onglet Liens
  • Dans la vue Détails de l'intégration, cliquez sur l'onglet Liens.
Activer ou désactiver un lien pour synchronisation
  1. Cochez la case Activé à côté d'un lien pour activer ou désactiver la synchronisation. Cochez la case pour activer la synchronisation d'un lien ou supprimez-la pour la désactiver.
  2. Cliquez sur Enregistrer..
Synchroniser tous les liens d'un coup
  • Cliquez sur le bouton Synchroniser tous.

Ajouter ou modifier un lien si le Type d'autorisation est Basique
