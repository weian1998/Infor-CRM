Présentation de la synchronisation d'agendas avec Outlook

L'intégration de l'agenda vous permet de synchroniser les éléments d'agenda entre Infor CRM et Outlook. Votre administrateur doit configurer et accorder une autorisation pour vous permettre de synchroniser des agendas avec Outlook.

Il est possible que cette fonction ne soit pas disponible pour tous les utilisateurs. Les intégrations externes doivent être implémentées par votre administrateur.

Pendant un cycle de synchronisation, Infor CRM compare les champs dans les deux applications et apporte des modifications de sorte que les informations soient les mêmes partout. Seules les données des champs mappés entre les applications sont transférées. Les données sont comparées et des modifications sont apportées afin de s'assurer que les informations correspondent. Par exemple, si un enregistrement a été ajouté dans une application et pas dans l'autre, il va être ajouté dans l'application incomplète.

 La synchronisation est affectée par la sécurité intégrée des données associée à l'utilisateur connecté à Infor CRM. La quantité de données pouvant être transférée est limitée par les droits de sécurité de l'utilisateur connecté.

Comportement de synchronisation des agendas

  • Si le même enregistrement est mis à jour dans les deux applications durant le même cycle de synchronisation, la dernière modification a préséance.
  • La suppression d'une activité dans l'une ou l'autre application supprime l'activité des deux applications.
  • Responsables, membres et ressources d'activité :
    • Les informations concernant les membres et les ressources ne sont pas synchronisées.
    • Si l'utilisateur est membre ou responsable d'une activité, lorsque l'activité est synchronisée, l'utilisateur obtient son propre enregistrement d'activité.
  • Les activités confirmées et non confirmées seront synchronisées.
  • Activités refusées :
    • Les activités refusées avant un cycle de synchronisation ne sont pas synchronisées.
    • Les activités qui sont refusées après un cycle de synchronisation sont supprimées de l'agenda de l'utilisateur dans les deux applications.
  • Les types d'activité dans Infor CRM peuvent être synchronisés avec différents types d'activité dans Outlook. Pour des détails, voir le tableau Présentation de l'échange de données d'agenda.
  • Les pièces jointes de l'activité ne sont pas synchronisées entre Infor CRM et Outlook.
  • (Synchronisation d'Outlook uniquement) Outlook comporte des options de synchronisation Outlook qui empêchent la synchronisation des rendez-vous et des tâches avant une date sélectionnée. En revanche, les activités qui sont modifiées (y compris la mise en veille ou la désactivation d'une alarme) après la date sélectionnée et toute occurrence passée d'une activité récurrente en cours seront synchronisées.

Pour des scénarios spécifiques, voir Scénarios de synchronisation d'agendas.

Le tableau suivant répertorie comment les données de l'agenda Infor CRM sont synchronisées avec Outlook :

*Evénement ne désigne pas les événements Infor CRM, qui ne sont pas synchronisé avec Outlook.

Procédure

  • Définition des options de l'agenda
  • Synchronisation manuelle de l'agenda et des contacts (synchronisation Outlook uniquement)
    • Dans Outlook, cliquez sur le bouton Synchroniser maintenant.

Il est déconseillé de synchroniser simultanément le client distant avec la base de données du bureau principal et Microsoft Outlook. Cela réduit considérablement la vitesse de la synchronisation et provoque des problèmes de mémoire. Infor CRM recommande de vous synchroniser avec MS Outlook avant ou après avoir synchronisé votre base de données distante.
