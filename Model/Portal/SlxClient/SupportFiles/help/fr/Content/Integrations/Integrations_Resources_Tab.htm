Onglet Ressources d'intégration

Utilisez l'onglet Ressources pour afficher des définitions et le comportement des ressources propres à l'intégration. Les ressources définissent quelles informations sont mappées entre Infor CRM et le contrat d'intégration. À partir de cet onglet, vous pouvez voir si une ressource est activée pour synchroniser l'intégration et dans quelle direction les informations sont synchronisées.

L'onglet Ressources est disponible uniquement pour les intégrations avec une méthode de synchronisation côté client ou côté serveur.

Procédure

Ouvrir l'onglet Ressources
  1. Ouvrez la vue Détails de l'intégration.
  2. Cliquez sur l'onglet Ressources.
Comparer les champs
  • Sur l'onglet Ressources, cliquez sur le bouton Gérer schéma.
    La boîte de dialogue Gérer schéma s'ouvre.

Cette vue contient les colonnes suivantes :

  • Ressource - Nom de la ressource dans le contrat d'intégration.
  • Infor CRM  - Nom de la ressource correspondante dans Infor CRM
  • Activé - Case cochée si la synchronisation est activée pour la ressource.
  • Direction de synchronisation - Direction dans laquelle les informations sont synchronisées : Bidirectionnel, VersSlxSeulement ou DeSlxSeulement.
