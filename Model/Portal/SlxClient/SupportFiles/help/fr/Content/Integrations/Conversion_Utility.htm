Exécution de l'utilitaire SLXConversion

Avant de configurer vos points d'extrémité de l'intégration, vous devez exécuter l'utilitaire SLXConversion pour mettre à jour des enregistrements du numéro de téléphone et e-mail du compte. Cet utilitaire prend en charge l'intégration en ajoutant des colonnes à la base de données Infor CRM pour pouvoir utiliser les relations de un à plusieurs pour les numéros de téléphone et les adresses e-mail dans Comptes et Contacts.

Ceci crée un enregistrement dans la table ERPPHONENUMBER ou ERPEMAILADDRESS pour chaque colonne du numéro de téléphone et de l'adresse e-mail dans la table ACCOUNT ou CONTACT. Ces enregistrements sont associés à la colonne de l'entité originale de laquelle ils proviennent.

 

L'authentification Windows doit être configurée pour l'utilisateur administratif avant d'exécuter l'utilitaire SLXConversion. Pour les étapes détaillées, consultez la rubrique « Utilisation de l'authentification Windows » dans l'aide du module Administrateur.

Pour exécuter

  1. Sur votre poste de travail administratif, naviguez jusqu'aux fichiers SLXConversionUtility.exe et SLXConversionUtility.config.
    Par défaut, les fichiers de configuration se trouvent dans ...\Program Files\Saleslogix\ ou ...\Program Files (x86)\Saleslogix\ 
  2. Ouvrez SLXConversionUtility.config dans l'éditeur de texte de votre choix.
  3. Accédez à la section <connectionStrings>, modifiez la valeur du catalogue initial de SALESLOGIX_EVAL à votre alias de base de données et enregistrez la modification.
    Par exemple :
    <connectionStrings> <clear/> <add name="Default" connectionString="Provider=SLXOLEDB.1;Persist Security Info=True;Initial Catalog=MyDatabaseAlias;Data Source=localhost;Extended Properties=&quot;PORT=1706;LOG=ON;CASEINSENSITIVEFIND=ON;AUTOINCBATCHSIZE=1;SVRCERT=12345;ACTIVITYSECURITY=OFF;TIMEZONE=NONE&quot;"/> </connectionStrings>
  4. Double-cliquez sur SLXConversionUtility.exe pour exécuter l'utilitaire.
  5. Cochez toutes les cases.
  6. Cliquez sur Mettre à jour.
  7. Cliquez sur Fermer.
