Onglet Portées d'intégration

Utilisez l'onglet Portées pour afficher les portées définies pour une intégration. Les portées déterminent quelles informations sont disponibles de l'intégration à Infor CRM.

L'onglet Portées est uniquement disponible pour les intégrations avec un type d'authentification OAuth1 ou OAuth2.

Pour ouvrir

  1. Ouvrez la vue Détails de l'intégration.
  2. Cliquez sur l'onglet Portées.
