Ajout ou modification d'un lien d'intégration de base

Utilisez la boîte de dialogue Ajouter un lien ou Modifier un lien pour ajouter ou modifier des liens d'intégration. Si le type d'autorisation d'une intégration est Basique, utilisez cette boîte de dialogue pour ajouter des liens.

Procédure

Ajouter un lien
  1. Dans la vue Détails de l'intégration, cliquez sur l'onglet Liens.

    L'onglet Liens est disponible uniquement pour les intégrations avec une méthode de synchronisation côté client ou côté serveur.

  2. Cliquez sur le bouton Ajouter un lien.
    Ce bouton est disponible uniquement si le type de lien d'intégration est Système ou Admin.
  3. Dans la zone URL, tapez l'URL SData utilisée par Infor CRM pour partager les informations entre les systèmes.
    • Utilisez un des formats suivants :
      • Si votre implémentation n'utilise par le port par défaut (port 80), utilisez le format http://server:port/sdatavirtualdir/application/contract/dataset/
        •  
      • Si votre implémentation utilise le port 80, omettez le numéro de port de l'URL.
        Le service de synchronisation expurge automatiquement le port par défaut de l'URL qui cause les erreurs et les problèmes correspondants. Evitez ceci en utilisant le format http://server/sdatavirtualdir/application/contract/dataset/
    • Remplacez par les valeurs fournies par votre partenaire d'intégration :
      •  Sdatavirtualdir - Par exemple, sdata.

      •  application - le nom de votre application de comptabilité. Par exemple, comptes50.

      •  contract - le nom du contrat. Par exemple, GCRM.
      •  dataset - fait la distinction entre un jeu de données et une société d'exploitation. Par exemple, /-/
  4. Dans la zone Nom d'affichage, tapez le nom que vous voulez afficher lorsque des utilisateurs associent des enregistrements entre Infor CRM et le système intégré.
  5. Dans la zone Nom d'utilisateur, tapez le nom que vous utilisez pour vous connecter au système d'intégration.
  6. Dans la zone Mot de passe, tapez le mot de passe correspondant qui permet à l'utilisateur de se connecter au système d'intégration.
  7. Cliquez sur le bouton Tester le lien.
    Le nom du bouton dépend du type d'autorisation et du type de lien.
  8. Cliquez sur OK.
Modifier un lien
  1. Dans la vue Détails de l'intégration, cliquez sur l'onglet Liens.

    		2.

    L'onglet Liens est disponible uniquement pour les intégrations avec une méthode de synchronisation côté client ou côté serveur.
  2. Cliquez sur le lien hypertexte Modifier correspondant au lien que vous voulez modifier.
    Ce lien est disponible uniquement si le type de lien d'intégration est Système ou Admin.
  3. Apportez vos modifications.
  4. Cliquez sur le bouton Tester le lien.
  5. Cliquez sur OK.
