Utilisation de la vue Configuration de synchronisation SData

Utilisez la vue Configuration de synchronisation SData pour configurer le flux Infor CRM utilisé pour synchroniser Infor CRM avec les systèmes intégrés.

Pour configurer SData

  1. Sur l'hôte Web, mettez fin au service de synchronisation SData (Service) Infor CRM.
  2. Dans la barre de navigation, cliquez sur Configuration de synchronisation SData.
  3. Configurez le flux Infor CRM en complétant les zones suivantes. Le flux se met automatiquement à jour à mesure que vous entrez les données.
    1. Si votre URL SData utilise Secure Socket Layer, cochez la case Utiliser SSL (Secure Socket Layer).
    2. Dans la zone Serveur, saisissez le nom du serveur pour votre URL SData.
    3. Dans la zone Port, saisissez le numéro de port pour votre URL SData.
  4. Dans la zone Nom d'utilisateur, laissez « ADMIN » ou saisissez le nom d'utilisateur correspondant à la base de données Infor CRM.

    L'authentification Windows doit être configurée pour l'utilisateur administratif. Pour les étapes détaillées, consultez la rubrique « Utilisation de l'authentification Windows » dans l'aide du module Administrateur.

  5. Dans le champ Mot de passe, saisissez le mot de passe correspondant à ce nom d'utilisateur.
  6. Dans la zone Nom d'affichage, laissez « Infor CRM » ou saisissez le nom d'affichage que vous voulez voir représenter le flux SData.
  7. Cliquez sur le bouton Tester le lien et affichez les informations dans la zone État.
    Si le flux échoue au test, examinez les zones d'informations, apportez vos modifications et testez une nouvelle fois le lien.
  8. Cliquez sur Enregistrer..
