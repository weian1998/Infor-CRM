Scénarios de synchronisation d'agendas pour Outlook

Les scénarios suivants sont des scénarios courants qui permettent d'expliquer le fonctionnement de la synchronisation des agendas avec Outlook.

 La synchronisation est affectée par la sécurité intégrée des données associée à l'utilisateur connecté à Infor CRM. La quantité de données pouvant être transférée est limitée par les droits de sécurité de l'utilisateur connecté.

Scénarios de synchronisation

Les types d'activité dans Infor CRM peuvent être synchronisés avec différents types d'activité dans Outlook. Pour des détails, voir le tableau Présentation de l'échange de données d'agenda.

Pour les scénarios suivants :

  • « élément d'agenda » représentera un type d'activité d'agenda dans Outlook, et des appels téléphoniques, des réunions ou des activités personnelles dans Infor CRM.
  • « tâche » représente un type d'activité hors agenda dans Outlook et une action à exécuter dans Infor CRM.
Action Résultat
Planification, modification, exécution ou suppression d'éléments d'agenda
Planifiez un élément d'agenda dans l'une ou l'autre application. L'élément d'agenda se synchronise.
Modifiez un élément d'agenda dans l'une ou l'autre application. La modification de l'élément d'agenda se synchronise.
Modifiez le même élément d'agenda dans les deux applications durant le même cycle de synchronisation.

L'élément d'agenda qui a été modifié en dernier se synchronise avec l'autre application et met à jour toutes les valeurs avec des différences.
Exécutez un élément d'agenda dans Infor CRM. L'élément d'agenda est exécuté dans Infor CRM et le mot "Terminé" s'affiche dans la ligne Objet de l'élément dans Outlook.
Supprimez un élément d'agenda de l'une ou l'autre application. L'élément d'agenda est supprimé de l'agenda de l'utilisateur dans les deux applications.

Planification, modification, exécution ou suppression de tâches
Planifiez une tâche dans l'une ou l'autre application. La tâche se synchronise.
Modifiez une tâche dans l'une ou l'autre application. La modification de la tâche se synchronise.
Modifiez la même tâche dans les deux applications durant le même cycle de synchronisation.

La tâche qui a été modifiée en dernier se synchronise avec Outlook et met à jour toutes les valeurs avec des différences.
Exécutez une tâche à faire dans Infor CRM. La modification se synchronise et la tâche est marquée comme exécutée dans Outlook.
Terminez une tâche dans Outlook La modification se synchronise et la tâche à faire est exécutée et consignée dans l'historique de Infor CRM.
Planifiez une tâche sans date ou heure de début dans Outlook. La tâche se synchronise comme une tâche à faire dans Infor CRM avec la date de création comme date de début. A la synchronisation suivante, la nouvelle date de début se synchronisera et mettra la date à jour dans Outlook.

Planification d'éléments d'agenda et de tâches à occurrences multiples
Planifiez un élément d'agenda récurrent dans Infor CRM. L'élément d'agenda et toutes les occurrences sont synchronisés avec Outlook.
Planifiez un élément d'agenda avec de multiples occurrences dans Outlook.

L'élément d'agenda est synchronisé avec Infor CRM, avec le nombre maximum d'occurrences permises dans Infor CRM :

  • Tous les jours - 999
  • Toutes les semaines - 260
  • Tous les mois - 60
  • Tous les ans - 20

Cette limite est à nouveau synchronisée avec Outlook.
Planifiez une tâche récurrente dans Infor CRM. La tâche récurrente est synchronisée avec une tâche récurrente dans Outlook.
Planifiez une tâche récurrente dans Outlook.

La tâche est synchronisée avec Infor CRM comme une tâche à faire avec le nombre maximum d'occurrences autorisé dans Infor CRM :

  • Tous les jours - 999
  • Toutes les semaines - 260
  • Tous les mois - 60
  • Tous les ans - 20

Cette limite est à nouveau synchronisée avec Outlook.

Planification d'éléments d'agenda avec membres
Planifiez un élément d'agenda dans l'une ou l'autre application, où je suis le responsable et qui comporte 3 autres membres.

Dans l'application où l'élément d'agenda a été programmé, l'élément d'agenda inclut les informations sur le responsable et les membres, comme prévu. Lorsqu'il se synchronise avec l'autre application, vous restez responsable, mais il ne contient aucune information sur le membres.

Les informations sur les membres resteront dans l'élément d'agenda de l'application source, même si l'élément d'agenda est mis à jour.
Planifiez un élément d'agenda dans l'une ou l'autre application, où je suis un membre, mais pas le responsable.

Dans l'application où l'élément d'agenda a été programmé, l'élément d'agenda inclut les informations sur le responsable et les membres, comme prévu. Lorsqu'il se synchronise avec l'autre application, vous devenez le responsable, mais il ne contient aucune information sur le membres.

Les informations sur les membres resteront dans l'élément d'agenda de l'application source, même si l'élément d'agenda est mis à jour.
Remplacez-moi comme responsable par un autre utilisateur dans Infor CRM après la synchronisation de l'élément d'agenda.

Supprime l'élément d'agenda dans Outlook.*

*Vous ne pouvez pas modifier le responsable d'un élément d'agenda dans Outlook.

Acceptation et suppression d'invitations à des éléments d'agenda
Acceptez une invitation à un élément d'agenda dans l'une ou l'autre application avant de synchroniser cet élément. L'élément d'agenda est synchronisé et indique qu'il est accepté.
Acceptez une invitation à un élément d'agenda dans l'une ou l'autre application après la synchronisation de cet élément. L'élément d'agenda se synchronise lorsqu'il est planifié et il se synchronise à nouveau apès sa mise à jour pour indiquer son acceptation.
Acceptez de manière provisoire une invitation à un élément d'agenda dans Outlook. L'élément d'agenda se synchronise lorsqu'il est planifié et il se synchronise à nouveau apès sa mise à jour pour indiquer son acceptation. Infor CRM ne comporte pas d'état Provisoire pour les activités.
Refusez une invitation à un élément d'agenda dans l'une ou l'autre application avant de synchroniser cet élément.

L'élément d'agenda est supprimé de l'agenda de l'utilisateur dans les deux applications.
Refusez une invitation à un élément d'agenda dans Infor CRM après la synchronisation de cet élément.

L'élément d'agenda est supprimé de l'agenda de l'utilisateur dans les deux applications.
Refusez une invitation à un élément d'agenda dans Infor CRM après la synchronisation de cet élément. L'élément d'agenda est supprimé de l'agenda de l'utilisateur dans les deux applications.

Utilisation d'éléments d'agenda personnels et privés
Planifiez une activité personnelle dans Infor CRM. L'activité personnelle se synchronise comme un élément d'agenda configuré comme étant Privé dans Outlook.
Planifiez un élément d'agenda privé dans Outlook. L'élément d'agenda privé se synchronise comme une activité personnelle dans Infor CRM.
Dans Outlook, effacez l'indicateur Privé après la synchronisation de l'élément d'agenda. L'élément d'agenda se synchronise et modifie l'activité personnelle en une réunion dans Infor CRM.
Dans Outlook, sélectionnez l'indicateur Privé après la synchronisation de l'élément d'agenda. L'élément d'agenda se synchronise et modifie l'activité personnelle en une réunion dansInfor CRM.

Utilisation d'élément d'agenda et de tâches sans horaire défini
Planifiez un élément d'agenda sans horaire défini dans Infor CRM. L'élément d'agenda sans horaire défini se synchronise comme un événement de toute une journée avec Outlook.
Planifiez un événement de toute une journée dans Outlook. L'événement de toute une journée se synchronise comme une réunion sans horaire défini dans Infor CRM.
Modifiez un élément d'agenda comme un élément sans horaire défini dans Infor CRM. L'élément d'agenda est modifié en un événement de toute une journée dans Outlook.
Passez le curseur de la souris sur un élément d'agenda sans horaire défini dans Infor CRM. La modification se synchronise avec Outlook et remplace la date de début par la date actuelle sur laquelle le curseur passe.

Mise en veille des alarmes
Mettez une alarme en sourdine pour un élément d'agenda dans l'une ou l'autre application.

La modification de l'élément d'agenda se synchronise.

Selon le moment du cycle de synchronisation où l'élément d'agenda est mis en sourdine, l'alarme pourra se déclencher avant la synchronisation de la mise en sourdine.

Infor CRM recommande la mise en sourdine d'alarmes dans une seule application.
Mettez en sourdine une alarme pour une tâche dans l'une ou l'autre application.

La modification de la tâche se synchronise.

Selon le moment du cycle de synchronisation où l'élément d'agenda est mis en sourdine, l'alarme pourra se déclencher avant la synchronisation de la mise en sourdine.

Infor CRM recommande la mise en sourdine d'alarmes dans une seule application.
