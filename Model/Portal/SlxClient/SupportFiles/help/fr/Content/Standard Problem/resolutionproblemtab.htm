Onglet Résolution standard Problèmes standard

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez cette vue pour afficher ou associer les problèmes standard à une résolution.

Procédure

Ouvrez cette vue
  • Dans la vue Détails de la résolution standard, cliquez sur l'onglet Problèmes standard.
Associer une résolution
  1. Cliquez sur le bouton Associer un problème.
  2. Utilisez la fonction Rechercher pour trouver un problème standard.
Supprimer une résolution
  1. Sélectionnez l'enregistrement à retirer, puis cliquez sur le bouton Supprimer.
  2. Cliquez sur OK.

Cette vue contient les colonnes suivantes :

  • Description - Indique la description du problème.
  • Zone - Indique où le problème se produit.
  • Catégorie - Indique le type de problème.
  • Problème - Indique le problème.
  • Notes - Indique de brèves notes à propos du problème.
