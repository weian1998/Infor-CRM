Ajout ou modification d'une résolution d'un problème standard

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Les résolutions de problèmes standard fournissent des informations pour résoudre un problème courant.

Procédure

Ajouter une résolution
  1. Dans le menu Nouveau, cliquez sur Résolution standard.
  2. Ajoutez les informations suivantes :
  • Description - Saisissez une description pour la résolution.
  • Notes - Saisissez des notes pour décrire comment résoudre le problème.

  1. Cliquez sur Enregistrer.

Modifier une résolution
  1. Dans la barre de navigation, cliquez sur Service, puis sur Résolutions standard pour ouvrir la vueListe des résolutions standard. Si nécessaire, utilisez la Recherche pour localiser une résolution, puis cliquez sur le N° de résolution standard.
  2. Modifiez l'une des informations suivantes :
  • Description - Saisissez une description pour la résolution.
  • Notes - Saisissez des notes pour décrire comment résoudre le problème.

  1. Cliquez sur Enregistrer..

Supprimer une résolution standard
  1. Dans la barre de navigation, cliquez sur Service, puis sur Résolutions standard pour ouvrir la vueListe des résolutions standard. Si nécessaire, utilisez la Recherche pour localiser une résolution, puis cliquez sur le N° de résolution standard.
  2. Dans la vue Détails de la résolution standard, cliquez sur le bouton Supprimer.
  3. Cliquez sur OK.
