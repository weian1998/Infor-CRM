Utilisation de la vue Détails de la résolution

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné.

Procédure

Ouvrez la vue Détails de la résolution
  • Dans la barre de navigation, cliquez sur Service, puis sur Résolutions standard pour ouvrir la vueListe des résolutions. Si nécessaire, utilisez la Recherche pour localiser une résolution, puis cliquez sur le N° de résolution.

Utilisez la Recherche pour rechercher les enregistrements

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser la liste des groupes du Volet des tâches ou les tâches courantes

Afficher un groupe

Zones d'informations

La zone Description fournit une brève description de la résolution et la zone Notes est destinée aux informations supplémentaires.

Onglets de la vue Détails

L'onglet Éléments affiche tous les éléments associés à une résolution. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

  • Problèmes standard

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
