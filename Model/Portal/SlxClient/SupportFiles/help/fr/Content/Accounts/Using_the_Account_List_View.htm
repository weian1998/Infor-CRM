Utilisation de la vue Liste des comptes

La vue de la liste Compte affiche tous les comptes auxquels vous avez accès.  Utilisez les options de groupes et de recherche pour restreindre la liste des enregistrements qui s'affiche..

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Procédure

Ouvrez la vue Liste des comptes
  • Cliquez sur Comptes dans la barre de navigation.

Travailler avec des enregistrements dans une grille

Utiliser la vue Résumé

Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Filtrer ma liste

Utilisez la Recherche pour rechercher les enregistrements

Ouvrez la vue Détails du compte
  • Cliquez sur le nom du compte pour accéder à la vue Détails du compte.

Utilisation du menu contextuel des groupes

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe
  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
  • Promouvoir vers le tableau de bord
