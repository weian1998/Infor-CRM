Configuration de la synchronisation et de l'intégration de MS Outlook

Les préférences de synchronisation et d'intégration doivent être configurées dans Microsoft Outlook.

 Il est possible que les options Connexion et Envoyer à CRM/Enregistrer dans l'historique soient déjà configurées pour les utilisateurs Client Web qui définissent leurs options dans le Gestionnaire de Bureau.

Vous pouvez configurer différents paramètres pour différents types d'enregistrements. Par exemple, vous pouvez activer la synchronisation unidirectionnelle pour les contacts et la synchronisation bidirectionnelle pour les éléments d'agenda. Les étapes suivantes indiquent les paramètres de synchronisation recommandés entre Microsoft Outlook et Infor CRM.

Avant de configurer MS Outlook :

  • Microsoft Outlook doit être installé et s'exécuter sur la machine locale.
    • Web - Desktop Integration ne s'installera pas si Microsoft Office, y compris Outlook, n'est pas déjà installé.
    • Windows - Si Microsoft Outlook n'a pas été installé avant d'installer le client Infor CRM, vous devez effectuer des étapes de configuration supplémentaires.
  • Vous devez connaître le serveur de connexion et le numéro de port, de même que votre nom d'utilisateur et votre mot de passe.
    Contactez l'administrateur système pour en savoir plus.
  • Vous devez créer un groupe de synchronisation des contacts pour les contacts que vous voulez synchroniser entre MS Outlook et Infor CRM.

Pour configurer

  1. Ouvrez Microsoft Outlook.
  2. Si vous n'y êtes pas invité, cliquez sur le bouton Options.
    Si le bouton Options n'apparaît pas, Synchronisation d'Outlook a été mal installé.
  3. Utilisez les onglets pour définir les options suivantes :
    • Connexion - connexion au serveur Infor CRM. Obligatoire pour la synchronisation des contacts et des agendas et l'intégration de la messagerie.
    • Synchronisation - options de synchronisation pour les contacts, rendez-vous (appels téléphoniques, réunions et activités personnelles) et tâches (à faire). Obligatoire pour la synchronisation des contacts et des agendas.
    • Envoyer à CRM/Enregistrer dans l'historique - options pour les invites, l'affichage d'informations et le comportement des pièces jointes lors de l'utilisation de Envoyer à CRM ou Enregistrer dans l'historique. Utilisé pour l'intégration de la messagerie.
    • Carnet d'adresses - affiche des options pour votre carnet d'adresses Infor CRM. Utilisé pour l'intégration de la messagerie.
    • Consignation - possibilité d'activer ou de désactiver et d'afficher les informations de consignation pour dépannage. Utilisé pour la synchronisation des contacts et des agendas 
    • Toutes les options - possibilité d'afficher des options sous forme de paires de valeurs. Recommandé aux utilisateurs avancés seulement.
       

  4. Cliquez sur OK.
