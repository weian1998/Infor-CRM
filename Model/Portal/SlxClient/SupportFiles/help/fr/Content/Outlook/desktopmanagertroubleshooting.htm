Dépannage du module d'intégration du bureau

Vous risquez de rencontrer les problèmes suivants avec le module d'intégration du bureau.

Problème :

L'icône du gestionnaire de bureau Infor CRM n'apparaît pas dans ma barre des tâches système.

Causes possibles et solutions : 

  • Le module d'intégration du bureau n'est pas installé.
  • Vous devez installer le module d'intégration du bureau.
  • Le module d'intégration du bureau a été installé avant votre navigateur.
  • Si vous avez installé votre navigateur actuel après avoir installé le module d'intégration du bureau, vous devez réinstaller le module d'intégration du bureau en utilisant votre navigateur actuel.

    		•

    Cela peut survenir si vous avez installé Firefox après avoir installé le module d'intégration du bureau en utilisant Internet Explorer.
  • Naviguez jusqu'à SLXDesktopManager.exe et ouvrez-le manuellement. L'emplacement de ce fichier dépend de votre système d'exploitation :

L'emplacement par défaut est :

\\Documents and Settings\[nom d'utilisateur]\Local Settings\Application Data\ (Windows XP/2000/2003) ou

\\Users\<user.domain>\appdata\local (Windows Vista/7/2008).

Problème :

Comment désactiver le gestionnaire de bureau ?

Solution :

  • Dans la barre des tâches système Windows, cliquez avec le bouton droit de la souris sur l'icône du Infor CRMgestionnaire de bureau, puis cliquez sur Quitter.

Vous devez d'abord quitter le gestionnaire de bureau Infor CRM avant de pouvoir désinstaller le module d'intégration du bureau Infor CRM.

Problème :

Comment désinstaller le module d'intégration du bureau ?

Solution :

  1. Ouvrez Ajout ou suppression de programmes dans Windows.
  2. Sélectionnez Infor CRM Module d'intégration du bureau dans la liste et cliquez sur Supprimer.
  3. Cliquez sur Oui.

    		4.

    Si vous utilisez Firefox comme navigateur, vous devez d'abord désinstaller le module complémentaire du module d'intégration du bureau Infor CRM dans Firefox.

    Pour désinstaller le module complémentaire du module d'intégration du bureau, ouvrez Firefox. Sur le menu Outils, cliquez sur Modules complémentaires, sélectionnez Module d'intégration du bureau Infor CRM et cliquez sur Désinstaller. Cliquez sur Désinstallez pour confirmer. Cliquez sur Redémarrer Firefox. Cliquez sur Redémarrer.
