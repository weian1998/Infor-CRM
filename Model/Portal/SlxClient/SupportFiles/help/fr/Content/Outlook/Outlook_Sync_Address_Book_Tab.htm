Définition des options du carnet d’adresses du connecteur Infor CRM

Utilisez les options du carnet d'adresses pour configurer l'apparence de votre carnet d'adresses Infor CRM.

Cet onglet est requis pour l'Intégration d'Outlook seulement.

Pour définir les options

  1. Cliquez sur le bouton Infor CRM Options., et, si nécessaire, cliquez sur Carnet d'adresses.
  2. Sous Afficher les noms de contacts comme, sélectionnez comment vous voulez voir apparaître les noms dans votre carnet d'adresses.
  3. Sous Lorsque le carnet d'adresses s'affiche, sélectionnez ou effacez les options pour personnaliser votre carnet d'adresses.
  4. Cliquez sur OK si vous avez fini de définir toutes vos options, ou sélectionnez un autre onglet Options.
