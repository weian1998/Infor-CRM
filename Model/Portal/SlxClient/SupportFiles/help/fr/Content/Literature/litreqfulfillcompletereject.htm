Traitement, clôture ou rejet des demandes de documentations

Utilisez les Tâches courantes de la vue Liste des demandes de documentation pour traiter, clôturer ou rejeter les demandes de documentation.

 Les demandes de documentation qui incluent une page de garde peuvent être satisfaites uniquement dans Internet Explorer, car la page de garde exige la fonction de publipostage, qui est uniquement prise en charge dans Internet Explorer versions 9 et ultérieures. Pour en savoir plus, voir « Fonctions d'intégration de bureau prises en charge par le navigateur ».

Procédure

Traiter les demandes de documentation.

Pendant la demande, un enregistrement de la demande est ajouté à l'onglet Notes/Historique pour chaque contact et chaque compte associé à la demande.

  1. Dans le menu Outils, cliquez sur Gestion des demandes de documentation.
  2. Sélectionnez une ou plusieurs demandes de documentation dans la liste.

Pour sélectionner plusieurs enregistrements, maintenez la touche [Ctrl] ou [Maj] enfoncée en mettant les enregistrements en surbrillance.

  1. Pour imprimer des étiquettes de publipostage ou des enveloppes, sélectionnez un format d'étiquette ou d'enveloppe dans la liste Imprimer les étiquettes.
  2. Dans le Volet des tâches, sous Tâches de documentation, cliquez sur Traiter.

Une liste de la documentation demandée est imprimée sur l'imprimante sélectionnée, en même temps qu'une copie de la lettre d'accompagnement incluse dans la demande et toute étiquette ou enveloppe.

  1. Rassemblez la documentation demandée afin qu'elle soit prête à être envoyée. Une fois la documentation envoyée, clôturez la demande de documentation.

Clôturer des demandes de documentation

Pendant le traitement, un enregistrement d'exécution est ajouté à la demande de documentation d'origine afin que l'expéditeur puisse suivre l'état de la demande.

  1. Dans le menu Outils, cliquez sur Gestion des demandes de documentation.
  2. Sélectionnez une ou plusieurs demandes de documentation dans la liste.

Pour sélectionner plusieurs enregistrements, maintenez la touche [Ctrl] ou [Maj] enfoncée en mettant les enregistrements en surbrillance.

  1. Dans le Volet des tâches, sous Tâches de documentation, cliquez sur Clôturer.
Rejeter des demandes de documentation

Rejeter une demande de documentation supprime l'enregistrement de demande de documentation de la liste et de tous les contacts et comptes associés.

  1. Dans le menu Outils, cliquez sur Gestion des demandes de documentation.
  2. Sélectionnez une ou plusieurs demandes de documentation dans la liste.

Pour sélectionner plusieurs enregistrements, maintenez la touche [Ctrl] ou [Maj] enfoncée en mettant les enregistrements en surbrillance.

  1. Dans le Volet des tâches, sous Tâches de documentation, cliquez sur Clôturer.
Actualiser la liste
  • Dans le Volet des tâches, sous Tâches de documentation, cliquez sur Actualiser.
