Suppression de documentation de la liste

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Vous pouvez supprimer un élément de documentation. Supprimer un élément de documentation le supprime de la liste des éléments disponibles lors de la création d'une demande de documentation, mais ne le supprime pas des demandes de documentation qui ont déjà été planifiées.

Pour supprimer

  1. Dans la barre de navigation, cliquez sur Eléments de documentation pour ouvrir la vue Liste des éléments de documentation. Si nécessaire, utilisez la fonction Rechercher pour localiser un élément de documentation, puis cliquez sur le numéro de l'élément.
  2. Dans la vue Détails de la documentation, cliquez sur le bouton Supprimer.

  3. Cliquez sur OK.
