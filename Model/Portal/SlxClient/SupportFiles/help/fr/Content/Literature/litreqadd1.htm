Planification d'une demande de documentation

 Vous pouvez planifier une demande de documentation afin d'envoyer des brochures marketing ou autres documents à un ou plusieurs contacts.  Une fois la demande planifiée, la personne ou le groupe de votre société responsable des envois de documentation peut traiter et imprimer la demande.

Si la demande concerne un seul contact, sélectionnez ce dernier avant d'effectuer la procédure décrite ci-après.

Pour planifer

  1. Sur le menu Planification, cliquez sur Demande de documentation.
  2. Dans la zone Description, saisissez une courte description de la demande de documentation.
  3. Dans le champ Objet de la demande, effectuez l'une des opérations suivantes :
    • Afin de créer une demande associée à un seul contact, sélectionnez Contact, cliquez sur le bouton Rechercher le contact  et sélectionnez un contact.
      La sélection a peut-être déjà eu lieu si vous avez effectué votre planification dans la vue Détail du contact.
    • Afin de créer une demande pour un groupe de contacts, sélectionnez Groupe, puis cliquez sur la flèche du menu déroulant et effectuez votre sélection dans la liste.
    • Afin de créer une demande pour une opportunité, sélectionnez Opportunité, cliquez sur le bouton Rechercher une opportunité , puis à l'aide de la recherche, sélectionnez une opportunité, puis cliquez sur Fermer.
  4. Cliquez sur le BoutonAgenda Envoyer avant le, sélectionnez une date puis cliquez sur OK.
  5. Activez la case à cocher Gérer la demande localement pour diriger la demande sur votre imprimante par défaut (elle s'imprime une fois la demande de documentation planifiée).
  6. Dans les zones Envoyer via (méthode de livraison) et Priorité, cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
  7. Dans la zone Page de garde, cliquez sur le dossier puis choisissez dans la liste le modèle qui convient pour votre lettre d'accompagnement puis cliquez sur OK.
  8. Dans la section Imprimer la liste de documentation, choisissez d'imprimer la liste et la lettre d'accompagnement sur des pages séparées, ou d'imprimer uniquement la liste des pièces jointes ou la lettre d'accompagnement (sans la liste).
  9. Pour utiliser un modèle de lettre d'accompagnement, cliquez sur le bouton Rechercher de Page de garde, puis cliquez sur le modèle à utiliser.

    La sélection d'un modèle est uniquement disponible dans Internet Explorer, car la page de garde exige la fonction de publipostage, qui est uniquement prise en charge dans Internet Explorer versions 9 et ultérieures. Pour en savoir plus, voir « Fonctions d'intégration de bureau prises en charge par le navigateur ».

    Pour utiliser un modèle dans Internet Explorer, vous devez télécharger l'intégration du bureau Infor CRM. Vous pouvez installer l'intégration du bureau Infor CRM depuis l'écran de connexion du client Web. 
  10. Dans la section Eléments disponibles,  sélectionnez les éléments à inclure dans la demande.
     
    1. Au besoin, cliquez sur le bouton Eléments de filtre disponiblesEléments de filtre disponibles et saisissez tout caractère contenu dans la famille, le nombre ou le nom de l'élément à ajouter.
    2. Double-cliquez dans la colonne Quantité et saisissez le nombre d'exemplaires de l'élément que vous souhaitez demander.
  11.  Cliquez sur le bouton Planifier une demande de documentation.
  12. Si vous avez sélectionné Gérer la demande localement et qu'il vous est demandé de choisir une imprimante, sélectionnez une imprimante dans la liste et cliquez sur OK.

La demande de documentation est enregistrée sur l'onglet Documentation de la vue Activités, et sur l'onglet Demandes de documentation des vues Détails du contact et Détails du compte.

L'employé désigné par l'administrateur pour gérer les demandes de documentation recevra votre demande et la traitera à la date indiquée.
  • Le coût total est déterminé par le coût d'un document et le nombre de documents sélectionnés.
  • Si vous disposez des droits d'accès appropriés, vous pouvez ajouter, supprimer ou modifier des documents dans la liste de documentation.
