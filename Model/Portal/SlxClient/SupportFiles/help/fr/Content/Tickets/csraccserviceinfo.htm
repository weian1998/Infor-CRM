La boîte de dialogue Informations sur les services du compte fournit des informations sur le compte, le ticket, le contact, les actifs et le contrat pour vous aider à comprendre la relation entre la personne demandant une assistance et son compte. Vous pouvez également utiliser cette boîte de dialogue pour déterminer le type d'assistance que vous pouvez lui fournir en fonction du contrat de service du compte.

Pour ouvrir la boîte de dialogue des informations sur les services du compte et obtenir un ticket

  • Dans la vue Détails du ticket, cliquez sur le bouton Infos de service du compte.

Pour utiliser les informations sur les services du compte lors de l'ajout d'un nouveau ticket

  1. Dans le menu Nouveau, cliquez sur Ticket.
  2. Dans la vue Insérer un nouveau ticket, dans la barre d'outils, cliquez sur le bouton Infos de service du compte. pour ouvrir la vue correspondante.
  3. Dans la zone Compte de la vue Informations sur les services du compte, si le nom du compte ne s'affiche pas, utilisez la fonction de recherche pour localiser le compte approprié.

Une fois le compte sélectionné, toutes les informations de service liées s'affichent dans la vue.

  1. Dans la zone Contact, sélectionnez le contact autorisé à rendre compte du problème.

Si le contact est autorisé à effectuer une demande de service, « Oui » s'affiche dans la colonne Autorisé.

  1. Dans la zone Contrats, sélectionnez le contrat sur lequel le ticket sera facturé.

Si vous ne savez pas quel contrat sélectionner,

  • vérifiez que la colonne Actif soit sur 'Oui'
  • Sélectionnez le contrat dans la liste : tout commentaire sur ce contrat apparaît dans la zone Commentaires.

Si aucun contrat ne s'affiche, vous devez utiliser le Client Infor CRM pour ajouter un contrat au compte.
  1. Cliquez sur OK pour enregistrer vos sélections et fermer la vue Informations sur les services du compte.
  2. Suivez les étapes restantes de l'ajout d'un ticket. Les informations de contact et de contrat sélectionnées dans la vue Informations sur les services du compte s'affichent dans la vue Insérer un nouveau ticket.
