Zones d'informations sur le ticket

Le tableau suivant décrit les zones d'informations par défaut.

Zone Description/Action
Nom du compte Le compte associé au ticket. Pour modifier le compte, cliquez sur le bouton Rechercher, recherchez le compte adéquat, puis cliquez sur OK.
Nom du contact Le contact associé au ticket. Pour modifier le contact, cliquez sur le bouton Rechercher, recherchez le contact adéquat, puis cliquez sur OK.
Téléphone Le numéro de téléphone du contact.
E-mail L'adresse e-mail du contact.
Contrat Le contrat sélectionné lors de la création du ticket. Pour modifier le contrat, cliquez sur le bouton Rechercher, recherchez le contrat adéquat, puis cliquez sur OK.
Zone - Catégorie - Problème

Ces trois champs décrivent brièvement la nature du problème signalé. Cliquez sur l'un des boutons Rechercher en regard de chaque zone, sélectionnez les éléments appropriés dans les listes puis cliquez sur OK.

Les éléments Catégorie affichés dépendent de l'option de Zone sélectionnée et les éléments Problème disponibles dépendent de l'option de Catégorie sélectionnée.
Source Contient une liste des différentes façons de signaler un ticket. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste, puis cliquez sur OK.
Statut L'état par défaut est Ouvert. Pour sélectionner un autre statut, cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste, puis cliquez sur OK.
Priorité Indique l'importance du problème. Pour modifier la priorité, cliquez sur la flèche du menu déroulant et sélectionnez l'élément qui correspond le mieux à la priorité du client.
Echéance La date à laquelle le ticket doit être résolu. La date par défaut dépend de la priorité sélectionnée. Pour effectuer une modification, cliquez sur le BoutonAgenda, sélectionnez une date.  
Date de l'affectation La date et l'heure auxquelles le ticket a été affecté à l'utilisateur actuel. Pour effectuer une modification, cliquez sur le BoutonAgenda, sélectionnez une date.  
Affecté à L'utilisateur du service client affecté à la Zone sélectionnée pour ce ticket. Pour réaffecter le ticket, cliquez sur le bouton Rechercher, recherchez l'utilisateur concerné, puis cliquez sur OK.
Résolu au 1er appel ? Sélectionnez cette case si vous pouvez résoudre immédiatement le problème du demandeur sans devoir retéléphoner au client ou que celui-ci rappelle.
Soumettre à SpeedSearch

Sélectionnez cette case si vous souhaitez que les informations sur le ticket soient disponibles pour les recherches effectuées par les autres utilisateurs.

En fonction de la manière dont votre administrateur a configuré SpeedSearch, la sélection de cette option peut automatiquement rendre les informations disponibles pour les recherches ou nécessiter que les informations soient approuvées pour la recherche rapide par un autre utilisateur.
Approuvé pour SpeedSearch Sélectionnez cette case pour approuver un ticket qui a été soumis à SpeedSearch. Cette option est disponible si votre administrateur vous a désigné comme une personne disposant de l'autorité pour approuver des tickets pour SpeedSearch.
Accès public Cliquez sur le bouton flèche déroulante vers le bas  pour sélectionner l'option d'accès public de ce ticket. Par défaut, l'accès public est défini par votre administrateur. Les désignations d'accès public déterminent les utilisateurs (internes ou externes) qui peuvent afficher certains groupes de tickets (par exemple, uniquement les nouveaux tickets, tous les tickets, etc.). Cette option est disponible si votre administrateur vous a désigné comme une personne disposant de l'autorité pour approuver des tickets pour SpeedSearch.
