Suppression d'un ticket

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour supprimer un ticket

  1. Recherchez le ticket que vous souhaitez supprimer.
  2. Dans la vue Détails du ticket, cliquez sur le bouton Supprimer.
  3. Cliquez sur OK pour supprimer l'enregistrement de façon définitive ou sur Annuler pour annuler cette suppression.
