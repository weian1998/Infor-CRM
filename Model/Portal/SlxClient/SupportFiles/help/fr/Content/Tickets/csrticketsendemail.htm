Envoi d'un message e-mail depuis un ticket

Vous pouvez envoyer un message e-mail à un client ou à un autre Infor CRM utilisateur tout en ajoutant un nouveau ticket ou depuis la vue Détails du ticket. L'e-mail peut fournir des informations sur le ticket, informer d'autres utilisateurs dont le client, de l'ouverture du ticket ou poser des questions pour faciliter la résolution du problème par exemple.

Pour envoyer un e-mail depuis un ticket

  1. Dans la barre de navigation, cliquez sur Tickets.
  2. Dans la vue de liste Tickets, utilisez la fonction Recherche pour localiser le ticket et ouvrez la vue Détails du ticket.
  3. Dans la vue Détails du Ticket, dans la barre d'outils, cliquez sur le bouton Détails de l'e-mail.
  4. Dans la zone A de la boîte de dialogue Envoyer un e-mail du ticket, cliquez sur une ou plusieurs des options de destinataire.
  5. Sous Type d'e-mail, sélectionnez l'un des types d'informations à envoyer. Les options sont les suivantes :
    OptionAction
    Informations sur le ticketLe message comprend les valeurs suivantes : Date de réception, Description, Résolution (si disponible), tous les commentaires internes et N° de Ticket.
    AucunLe N° de Ticket renseigne la ligne objet de l'e-mail. Aucune autre information concernant le ticket n'est comprise dans le message.

    Les commentaires internes ne sont disponibles qu'aux autres utilisateurs Infor CRM. Si l'un des destinataires de l'e-mail est un contact, les commentaires internes ne seront alors pas inclus dans l'e-mail.

  6. Cliquez sur OK.
  7. Envoyez votre e-mail en suivant l'une des méthodes ci-dessous :
    • Cliquez sur le bouton Envoyer. Le message ne sera pas enregistré dans l'historique.
    • Si vous utilisez Microsoft Outlook, cliquez sur le bouton Envoyer à CRM. Si vous avez joint un fichier et souhaitez l'enregistrer en tant que pièce jointe, cliquez sur Oui lorsque vous y êtes invité. L'e-mail est enregistré dans l'historique et le fichier apparaît dans l'onglet Pièces jointes.
