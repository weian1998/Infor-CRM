Utilisation du Gestionnaire de processus

Le gestionnaire de processus de contact sur le Web vous permet d'afficher les processus affectés à vos contacts ou d'ajouter de nouveaux processus. Toute modification ultérieure apportée aux processus doit s'effectuer dans le Client réseau Infor CRM ou Client distant Infor CRM. Utilisez l'onglet Processus de la vue Détails du contact pour afficher les processus liés à un contact donné.

Vos préférences utilisateur déterminent l'état par défaut des processus. Vous pouvez toutefois afficher d'autres états en cliquant sur l'onglet approprié dans le gestionnaire de processus. Les états de processus comprennent : Tous les processus, Lancé, En cours, En attente, Suspendu, Terminé et Abandonné. Seuls les processus liés aux contacts auxquels vous pouvez accéder s'affichent.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour afficher un processus

  1. Dans la barre de navigation, cliquez sur Processus.
  2. Dans la vue Gestionnaire de processus, cliquez sur un onglet pour afficher tous les processus ayant un état spécifique. Les options disponibles sont les suivantes : Lancé, En cours, En attente, Suspendu, Terminé, Abandonné ou Tous les processus. Vos préférences utilisateur déterminent l'état par défaut des processus. Vous pouvez afficher d'autres états en cliquant sur l'onglet approprié dans le gestionnaire de processus.

Pour afficher les informations détaillées du contact

  • Cliquez sur le nom du contact que vous souhaitez afficher dans la liste.
