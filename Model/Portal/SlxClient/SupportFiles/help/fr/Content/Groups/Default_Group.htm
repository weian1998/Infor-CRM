Définition d'un groupe par défaut

Vous pouvez définir un groupe par défaut pour chaque vue de liste. Le groupe par défaut est le groupe qui s'ouvrira lors de votre premier accès à la vue de liste. Le groupe par défaut détermine aussi la présentation des nouveaux groupes. Il ne peut y avoir qu'un groupe par défaut par vue de liste.

Pour définir par défaut

  • Effectuez l’une des opérations suivantes :

    • Dans une vue de liste, cliquez avec le bouton droit de la souris sur l'onglet du groupe que vous voulez définir comme groupe par défaut, puis cliquez sur Définir par défaut.
    • Dans le menu Outils, cliquez sur Options puis sur l'onglet Groupes. Utilisez les listes Vue principale et Groupe par défaut pour définir le groupe par défaut pour chaque vue principale.
