Présentation des groupes

Les groupes sont des ensembles d'enregistrements. Ils peuvent vous aider à gérer votre flux de travail. Vous pouvez utiliser des groupes pour travailler avec un sous-ensemble d'enregistrements, sélectionner des données à imprimer sur des rapports et envoyer des courriers, des e-mails et des fax à l'aide de la fonction de publipostage. Un enregistrement peut être membre de plusieurs groupes. Vous pouvez exporter les enregistrements d'un groupe vers MS Excel ou partager vos groupes avec d'autres utilisateurs.

Pour afficher d'autres onglets de groupes, cliquez sur les flèches.

Les trois types de groupes sont :

  • Groupes Ad Hoc - groupes créés à partir des enregistrements sélectionnés
  • Groupes de recherche - groupes basés sur une recherche
  • Groupes de requêtes avancées- groupes basés sur les conditions de Query Builder

Pour ouvrir un groupe

  • Procédez d'une des manières suivantes :
    • Dans le menu Groupes, sélectionnez le groupe que vous souhaitez afficher.
    • Cliquez sur l'onglet du groupe que vous souhaitez ouvrir. Par exemple, Tous les comptes.

Pour afficher les conditions d'un groupe

  1. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  2. Cliquez sur Modifier.
    Le Infor CRM Query Builder s'ouvre.
  3. Utilisez l'onglet Conditions de Query Builder pour afficher les conditions que doivent remplir les enregistrements du groupe. Cet onglet n'est pas disponible pour les groupes ad hoc.
