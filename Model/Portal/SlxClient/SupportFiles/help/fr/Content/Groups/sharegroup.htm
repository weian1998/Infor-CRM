Partage de groupes avec d'autres utilisateurs

Vous pouvez partager vos groupes avec d'autres utilisateurs de Infor CRM, équipes ou services.

  • Si vous diffusez un groupe à d'autres utilisateurs, puis que vous le supprimez, ce groupe n'est plus disponible pour les autres utilisateurs.
  • Vous ne pouvez pas partager des groupes qui ont été partagés avec vous par un autre utilisateur.
  • Lorsque vous apportez des modifications à un groupe diffusé auprès d'autres utilisateurs, vous devez mettre à jour le groupe pour que les autres utilisateurs puissent recevoir la nouvelle version.
  • Vous pouvez afficher les informations d'accès des groupes existants afin de déterminer quel en est le propriétaire et quels sont les utilisateurs autorisés à le visualiser.
  • Les utilisateurs peuvent uniquement visualiser les comptes, les contacts, les opportunités et les tickets pour lesquels ils disposent de droits d'accès.
  • Si un groupe ad hoc est diffusé à d'autres utilisateurs, ces derniers peuvent y ajouter des enregistrements et en supprimer.
  • Si vous partagez un onglet Tableau de bord qui contient un groupe, ce groupe doit être également partagé avec les utilisateurs pour pouvoir être affiché sur l'onglet Tableau de bord.
Pour partager un groupe avec d’autres utilisateurs
  1. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  2. Cliquez sur Partager.
     Si vous n'avez pas l'autorisation de partager le groupe, cette option n'est pas disponible.
  3. Cliquez sur le bouton Ajouter.
  4. Dans la boîte de dialogue Affecter un propriétaire, cliquez sur l'onglet Utilisateurs.
  5. Pour limiter le nombre d'utilisateurs répertoriés, sélectionnez un type d'utilisateur dans la liste déroulante Type d'utilisateur (A distance, par exemple).
  6. Sélectionnez les utilisateurs avec lesquels vous souhaitez partager votre groupe, puis cliquez sur OK.
Pour partager un groupe avec plusieurs équipes
  1. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  2. Cliquez sur Partager.
     Si vous n'avez pas l'autorisation de partager le groupe, cette option n'est pas disponible.
  3. Cliquez sur le bouton Ajouter.
  4. Dans la boîte de dialogue Affecter un propriétaire, cliquez sur l'onglet Équipes.
Pour partager un groupe avec d'autres services
  1. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  2. Cliquez sur Partager.
     Si vous n'avez pas l'autorisation de partager le groupe, cette option n'est pas disponible.
  3. Cliquez sur le bouton Ajouter.
  4. Dans la boîte de dialogue Affecter un propriétaire, cliquez sur l'onglet Services.
  5. Sélectionnez les services avec lesquels vous souhaitez partager votre groupe, puis cliquez sur OK.
Pour supprimer l'accès à un groupe partagé
  1. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  2. Cliquez sur Partager.
     Si vous n'avez pas l'autorisation de partager le groupe, cette option n'est pas disponible.
  3. Sélectionnez l'utilisateur, le service ou l'équipe avec qui vous ne souhaitez plus partager votre groupe.
  4. Cliquez sur le bouton Supprimer, puis sur OK.
Pour mettre à jour un groupe partagé

Si vous modifiez un groupe et souhaitez partager les modifications, vous devez mettre à jour le groupe.

  1. Cliquez avec le bouton droit de la souris sur un onglet de groupe.
  2. Cliquez sur Partager.
     Si vous n'avez pas l'autorisation de partager le groupe, cette option n'est pas disponible.
  3. Dans la boîte de dialogue Partager le groupe, cliquez sur OK pour diffuser les modifications aux utilisateurs répertoriés.
