Ajout d'un groupe de requêtes avancées

Utilisez Query Builder pour créer un groupe Les requêtes avancées permettent de définir les conditions du groupe. À chaque fois que vous ouvrez le groupe, la requête évalue la base de données pour déterminer les enregistrements à afficher. Si vous voulez vous assurer que le groupe ne change pas, vous pouvez créer un groupe ad hoc.

 

Pour créer un groupe de requêtes avancées

  1. Ouvrez une vue Liste ou Détails.
  2. Cliquez sur le bouton Ajouter un groupe.
    Le Infor CRM Query Builder s'ouvre.
  3. Utilisez l'onglet Conditions pour définir la ou les instruction(s) conditionnelle(s) utilisées pour sélectionner les enregistrements.
  4. Le cas échéant, utilisez les onglets Query Builder suivants :
    • Onglet Propriétés - Utilisez cet onglet pour définir le nom et la description du groupe.
      •  
    • Onglet Présentation - Utilisez cet onglet pour définir les données qui apparaîtront dans la vue de liste.

      La présentation par défaut est définie d'après le groupe que vous avez sélectionné comme groupe par défaut dans vos options de groupe.

    • OngletTri : définit le mode de tri des données.
  5. Une fois toutes les modifications effectuées, cliquez sur OK.
