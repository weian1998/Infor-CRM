Suppression de groupes

Vous pouvez supprimer une macro que vous n'utilisez plus. Vous pouvez uniquement supprimer les groupes que vous avez ajoutés. Cependant, l'administrateur système peut supprimer tous les groupes.

 

Pour supprimer un groupe

  1. Dans une vue Liste, cliquez avec le bouton droit sur l'onglet d'un groupe.
  2. Cliquez sur Supprimer. 
     Si vous n'avez pas l'autorisation de supprimer le groupe, cette option n'est pas disponible.
  3. Cliquez sur OK.
