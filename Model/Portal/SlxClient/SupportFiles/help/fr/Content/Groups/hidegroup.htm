Masquer des groupes

Vous pouvez masquer les groupes que vous ne souhaitez plus visualiser sans pour autant les supprimer.

Pour masquer un groupe

  1. Dans une vue Liste, cliquez avec le bouton droit sur l'onglet d'un groupe.
  2. Cliquez sur Masquer.

Vous ne pouvez pas masquer le groupe par défaut. Si le groupe que vous souhaitez masquer est le groupe par défaut, vous devez tout d'abord accéder aux options et sélectionner un nouveau groupe par défaut.
