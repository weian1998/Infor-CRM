Ajout d'un groupe ad hoc

Vous pouvez créer un groupe ad hoc pour y stocker la liste des enregistrements sélectionnés à partir de l'une des vuesListe.

Pour ajouter un groupe ad hoc

  1. Ouvrez une vue Liste.
  2. Ouvrez un groupe ou effectuez une Recherche pour trouver les enregistrements que vous souhaitez ajouter au groupe.
  3. Sélectionnez un ou plusieurs enregistrements en appuyant sur la touche CTRL ou Maj pendant que vous cliquez sur chaque élément, puis effectuez l'une des tâches suivantes :
    • dans le volet des tâches, sélectionnez Enregistrer les enregistrements en tant que groupe.
    • Cliquez avec le bouton droit dans la grille de la vue Liste et sélectionnez Enregistrer les enregistrements en tant que groupe.

    		4.

    Si vous ne sélectionnez aucun enregistrement, on vous demandera si tous les enregistrements du groupe doivent être utilisés. Si vous cliquez sur OK, tous les enregistrements du groupe seront sélectionnés.
  4. Dans la boîte de dialogue Ajouter des enregistrements au nouveau groupe, entrez un nom descriptif dans la zone Nom du groupe.

  5. Cliquez sur OK.
