Utilisation de la vue Détails de la campagne

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Procédure

Ouvrir la vue Détails de la campagne
  • Dans la barre de navigation, cliquez sur Campagnes pour ouvrir la vue Liste des campagnes. Si nécessaire, utilisez la fonction Rechercher pour localiser une campagne, puis cliquez sur le nom de la campagne.

Utilisez la Recherche pour rechercher les enregistrements

Afficher un groupe

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste.
Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • E-mail
  • Publipostage
  • Ajouter une note
  • Nouveau rendez-vous
  • Nouvel appel téléphonique
  • Nouvelle tâche à faire

zones d'informations

Les zones d'informations sur la campagne contiennent des informations sur la campagne sélectionnée. Les utilisateurs disposant des autorisations appropriées peuvent modifier ces zones. Vous pouvez également lancer une campagne vers les cibles de campagne.

onglets des vues Détails

Chaque onglet affiche des informations spécifiques que vous pouvez utiliser dans le cadre des activités relatives à la campagne. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.
