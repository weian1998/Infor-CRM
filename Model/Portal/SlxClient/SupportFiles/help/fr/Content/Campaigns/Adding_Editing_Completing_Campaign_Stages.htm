Ajout/Modification/Réalisation des phases de campagne

Les phases de campagne sont des groupes de tâches qui permettent d'aider les utilisateurs du composant Marketing de Infor CRM à gérer leurs campagnes. Vous pouvez ajouter et modifier des informations concernant les phases ou définir une phase comme terminée une fois qu'elle a été effectuée.

Procédure

Ajouter une phase
  1. Dans la vue Détails de la campagne, sélectionnez l'onglet Phases/Tâches, puis cliquez sur Ajouter une phase. 
  2. Saisissez une description de la phase. 
  3. Dans la zone État, cliquez sur le flèche déroulante vers le bas , et sélectionnez un élément dans la liste.
  4. Dans la zone Source du lead, cliquez sur le bouton Rechercher pour sélectionner une source de lead.
    Votre sélection s'ajoute à la boîte de dialogue.
  5. Si nécessaire, modifiez le campagne. Cliquez sur OK dans la boîte de dialogue.

    La modification du code de la campagne signifie que le code de campagne de cette phase ne correspondra plus à la campagne ou à toute autre phase de la campagne.

  6. Dans la zone Commentaires, saisissez les informations supplémentaires éventuelles. 
  7. Si nécessaire, cliquez sur la Date de début BoutonAgenda pour sélectionner la date, puis cliquez sur OK.   
  8. Cliquez sur la Date de fin  BoutonAgenda pour sélectionner la date, puis cliquez sur OK.  
  9. Ajoutez des tâches à la grille Tâches.
  10. Cliquez sur OK.
Modifier une phase de campagne
  1. Dans la vue Détails de la campagne, sélectionnez l'onglet Phases/Tâches, puis cliquez sur Modifier une phase.
    • Dans les boîtes de dialogue Insérer une campagne ou Copier une campagne, sélectionnez la phase à modifier, puis cliquez sur Modifier.
    • Dans la vue Détails de la campagne, sélectionnez l'onglet Phases/Tâches, cliquez avec le bouton droit de la souris sur la phase à modifier, puis pointez sur Modifier une phase.
  2. Effectuez les modifications nécessaires, puis cliquez sur OK.
Terminer une phase de campagne
  1. Dans la vue Détails de la campagne, sélectionnez l'onglet Phases/Tâches, sélectionnez la phase à terminer, puis cliquez sur Terminer une phase.
    La boîte de dialogue Terminer la phase s'ouvre avec l'état défini sur « Terminé ».
  2. Effectuez les modifications nécessaires, puis cliquez sur OK.
