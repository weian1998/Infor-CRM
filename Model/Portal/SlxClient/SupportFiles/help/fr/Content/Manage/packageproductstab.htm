Onglet Lots de produit

Utilisez cette vue pour ajouter, modifier ou supprimer des produits associés au lot.

Procédure

Ouvrez cette vue
  • Dans la vue Détails du lot, cliquez sur l'onglet Produits.
Ajouter des produits à un lot
  1. Cliquez sur Associer des produits.
  2. Utilisez la fonction Rechercher pour localiser le produit que vous souhaitez ajouter.
  3. Après avoir sélectionné un ou plusieurs produits, cliquez sur Ajouter la sélection.
Modifier la quantité de produits

  • Sélectionnez l'enregistrement à modifier, faites vos modifications dans la grille, puis cliquez sur Enregistrer. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur Enregistrer.
    Les colonnes ne peuvent pas toutes être modifiées dans la grille. Pour savoir si une colonne peut être modifiée, cliquez dans la grille. Si une zone blanche apparaît, la colonne est modifiable.

Supprimer un produit d'un lot
  1. Sélectionnez l'enregistrement à retirer, puis cliquez sur le bouton Supprimer.

  2. Cliquez sur OK.

Cette vue contient les colonnes suivantes :

  • Produit - Affiche le nom du produit.
  • Code article - Affiche l'unité de gestion des stocks (code article) du produit.
  • Quantité - Affiche la quantité numérique du produit incluse dans le lot.
  • Etat - Affiche l'état du produit. Par exemple Disponible.
  • Famille - Affiche la famille à laquelle appartient le produit.
