Modification d'un lot de produits

Utilisez la vue Détails du lot pour modifier les informations du lot.

Pour modifier

  1. Sur la barre de navigation, cliquez sur Ventes , Support, ou Administration, puis sur Progiciels pour ouvrir la vue Liste des lots. Si nécessaire, utilisez la fonction Rechercher pour localiser un lot, puis cliquez sur le nom du lot.
  2. Si nécessaire, modifiez les informations du lot.
  3. Cliquez sur Enregistrer..
