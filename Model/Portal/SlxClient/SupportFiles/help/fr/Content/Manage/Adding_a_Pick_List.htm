Ajout d'une liste de sélection

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Pour ajouter

  1. Dans la barre de navigation, cliquez sur Administration, puis sur Listes de sélection.
  2. Dans le volet Tâches courantes, cliquez sur Ajouter une liste de sélection.
  3. Dans la boîte de dialogue Nom de la liste de sélection, saisissez le nom de la liste et cliquez sur OK.
  4. Dans la vue Détails de la liste de sélection, définissez les attributs de la liste de sélection.
  5. Ajoutez des éléments à la liste de sélection via l'onglet Eléments.
  6. Si nécessaire, cliquez sur le bouton Tester cette liste flèche déroulante vers le bas  pour afficher un aperçu de la liste de sélection.
  7. Cliquez sur Enregistrer..
