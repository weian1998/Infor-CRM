Vue Détails de l'historique de la recherche des doublons

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

La vue Détails de l'historique de la recherche des doublons affiche les détails de l'exécution d'une tâche dans l'Assistant de recherche des doublons. Vous pouvez y consulter les informations relatives à la tâche et résoudre les doublons potentiels.

Procédure

Ouvrir la vue Détails de l'historique de la recherche des doublons
  1. Dans le menu Outils, cliquez sur Historique de tâche de duplication.
  2. Cliquez sur un numéro de tâche dans la liste pour ouvrir la vue de détails associée.

Utilisez la Recherche pour rechercher les enregistrements

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser la liste des groupes du Volet des tâches ou les tâches courantes

Afficher un groupe

Zones d'informations

Les informations de tâche suivantes sont en lecture seule.

Zone Description
Numéro de tâche Le numéro de tâche de l'assistant de recherche des doublons.
Description de la tâche L'entité sur laquelle la tâche a été exécutée.
Type d'entité L'entité sur laquelle la tâche a été exécutée.
Description de la source Le groupe sur lequel la tâche a été exécutée.
Nombre de sources Le nombre d'enregistrements dans le groupe source.
Nombre traités Le nombre d'enregistrements traités dans la tâche.
Nombre de doublons Le nombre de doublons potentiels.
Démarré le La date et l'heure auxquelles la tâche a démarré.
Date d'achèvement La date et l'heure auxquelles la tâche s'est terminée.
Démarré par : L'utilisateur ayant soumis la tâche.
Statut L'état de la tâche.

Onglets de la vue Détails

Utilisez l'onglet Résultats de la tâche pour consulter et résoudre les doublons.
