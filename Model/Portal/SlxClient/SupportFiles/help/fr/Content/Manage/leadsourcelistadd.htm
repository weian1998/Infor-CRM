Ajout d'une source de lead à la liste

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Vous pouvez ajouter des sources, telles que salons professionnels et publicités, à la liste des sources des leads disponibles.

Pour ajouter

  1. Dans le menu Administration, cliquez sur Nouvelle source de lead.
  2. Remplissez les zones de sources de lead :
  • Description - Saisissez une description pour la source de lead.
  • Description abrégée - Saisissez une description abrégée pour la source de lead.
  • Type - Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
  • Etat - Cliquez sur le flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
  • Coût - Saisissez le coût associé à la production de la source de lead.
  • Date - Pour modifier, cliquez sur le bouton Agenda, sélectionnez la date puis cliquez sur OK.
  1. Cliquez sur Enregistrer..
