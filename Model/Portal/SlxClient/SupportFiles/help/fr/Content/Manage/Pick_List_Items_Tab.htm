Onglet Eléments

Utilisez l'onglet Éléments pour gérer les éléments d'une liste de sélection. Cet onglet affiche les éléments que voient les utilisateurs lorsqu'ils ouvrent une liste de sélection.

Si vous utilisez une autre application qui partage les informations de base de données avec Infor CRM, vous devez utiliser les mêmes codes et éléments de liste de sélection dans les deux applications.

 

Pour gérer les éléments

  1. Dans la vue Détails de la liste de sélection, cliquez sur l'onglet Eléments.
  2. Procédez d'une des manières suivantes :
    • Pour ajouter un élément, cliquez sur Ajouter un élément. Renseignez la boîte de dialogue Ajouter un nouvel élément, puis cliquez sur Enregistrer et nouveau pour continuer à ajouter des éléments, ou sur OK lorsque vous avez fini d'ajouter des éléments.
    • Pour modifier un élément existant, cliquez sur Modifier sous l'élément que vous souhaitez modifier. Renseignez la boîte de dialogue Modifier un élément, puis cliquez sur OK.
    • Pour supprimer un élément, cliquez sur Supprimer sous l'élément que vous souhaitez supprimer.

  3. Cliquez sur OK.
