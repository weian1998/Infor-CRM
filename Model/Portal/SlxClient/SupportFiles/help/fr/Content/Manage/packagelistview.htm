Utilisation de la vue Liste des lots

La vue Liste des lots affiche tous les lots auxquels vous avez accès. Utilisez les options de groupes et de recherche pour restreindre la liste des enregistrements qui s'affiche.

Procédure

Ouvrez la vue Liste des lots
  • Sur la barre de navigation, cliquez sur Ventes , Support, ou Administration, puis sur Progiciels.
Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Travailler avec des enregistrements dans une grille

Filtrer ma liste

Utilisez la Recherche pour rechercher les enregistrements

Ouvrez la vue Détails des lots
  • Cliquez sur le nom du lot pour ouvrir la vue Détails du lot.

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe
  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
