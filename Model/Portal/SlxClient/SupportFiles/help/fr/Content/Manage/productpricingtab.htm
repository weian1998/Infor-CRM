Onglet Niveaux de prix de produit

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Vous pouvez utiliser l'onglet Niveaux de prix de la vue Détails du produit pour afficher ou gérer les informations de prix de produit.

Si l'onglet Niveaux de prix n'apparaît pas dans la vue Détails, cliquez sur Autres onglets pour accéder à la liste des onglets disponibles. Sélectionnez l'onglet Niveaux de prix puis faites-le glisser vers la zone des onglets ou dans la section utilisateur de la vue Détails.

Procédure

Ajouter un prix de produit

Modifier un prix de produit

Supprimer un prix de produit
  1. Cliquez sur Supprimer en regard du prix que vous souhaitez supprimer.

  2. Cliquez sur OK.

L'onglet Niveaux de prix contient les informations suivantes :

  • Description - Affiche la description du prix.
  • Prix - Affiche le prix du produit. Cette colonne repose sur le type de devise par défaut défini par l'administrateur.
  • Défaut - Affiche le prix par défaut du produit.
