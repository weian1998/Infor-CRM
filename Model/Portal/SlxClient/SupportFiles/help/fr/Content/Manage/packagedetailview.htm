Utilisation de la vue Détails du lot

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Votre espace de travail peut être différent selon vos droits d'accès ou si votre installation a été personnalisée.

Procédure

Ouvrez la vue Détails du lot
  • Sur la barre de navigation, cliquez sur Ventes , Support, ou Administration, puis sur Progiciels pour ouvrir la vue Liste des lots. Si nécessaire, utilisez la fonction Rechercher pour localiser un lot, puis cliquez sur le nom du lot.

Utilisez la Recherche pour rechercher les enregistrements

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser la liste des groupes du Volet des tâches

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe

Zones d'informations

  • Nom - Le nom du lot de produits.
  • Description - La description du lot de produits.
  • État - L'état du lot. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
  • Date de création - Date à laquelle le lot de produits a été créé.
  • Créé par - Affiche le nom de l'utilisateur qui a créé le lot de produits.
  • Date de modification - Affiche la date à laquelle le lot de produits a été modifié pour la dernière fois.
  • Modifié par - Affiche le nom de l'utilisateur qui a modifié pour la dernière fois les informations du lot de produits.

onglets des vues Détails

Chaque onglet affiche des informations spécifiques que vous pouvez utiliser dans le cadre des activités relatives au lot. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

Produits

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
