Utilisation de la vue Détails du concurrent

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné.

Votre espace de travail peut être différent selon vos droits d'accès ou si votre installation a été personnalisée.

Procédure

Ouvrez la vue Détails du concurrent
  • Dans la barre de navigation, cliquez sur Administration, puis cliquez sur Concurrents pour ouvrir la vue Liste des concurrents. Si nécessaire, utilisez la Recherche pour localiser un concurrent, puis cliquez sur son nom.

Utilisez la Recherche pour rechercher les enregistrements

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser la liste des groupes du Volet des tâches ou les tâches courantes

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe

Zones d'informations

  • Concurrent : affiche le nom du concurrent.
  • Compte - Cliquez sur le bouton Rechercher et sélectionnez un compte à l'aide de la fonction Rechercher.
  • Contact -Cliquez sur le bouton Rechercher
  • Téléphone : saisissez ou modifier le numéro de téléphone du concurrent.
  • Adresse Internet : tapez ou modifiez l'adresse Internet du concurrent.
  • Evaluation : indiquez ou modifiez la cote du concurrent.
  • Points faibles : indiquez ou modifiez les points forts du concurrent.
  • Points forts : indiquez ou modifiez les points forts du concurrent.
  • Stratégie : définissez ou modifiez la stratégie commerciale suivie par le concurrent.
  • Notes  : saisissez ou modifiez des remarques sur le concurrent.
