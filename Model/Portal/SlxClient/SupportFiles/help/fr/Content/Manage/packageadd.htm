Ajout d'un lot de produits

Vous pouvez ajouter des lots à la liste des lots. Cette liste est disponible lors de l'ajout ou de la modification de produits associés à une opportunité ou de bon de commande.

Pour ajouter

  1. Dans le menu Administration, cliquez sur Nouveau lot.
    La boîte de dialogue Insérer un lot s'ouvre.
  2. Renseignez les zones du lot de produits :
    • Nom - Saisissez le nom du lot de produits.
    • Description - Saisissez une description pour le lot de produits.
    • Etat - Cliquez sur le flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
  3. Cliquez sur Enregistrer.
  4. Pour ajouter des produits au lot, procédez comme suit :
    1.  Sur la vue Détails du lot, ouvrez l'onglet Produits et cliquez sur Associer un produit.
    2. Après avoir sélectionné un ou plusieurs produits, cliquez sur Ajouter la sélection.
    3. Si nécessaire, modifiez la quantité. Sélectionnez l'enregistrement à modifier, faites vos modifications dans la grille, puis cliquez sur Enregistrer. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur Enregistrer.
      Les colonnes ne peuvent pas toutes être modifiées dans la grille. Pour savoir si une colonne peut être modifiée, cliquez dans la grille. Si une zone blanche apparaît, la colonne est modifiable.
