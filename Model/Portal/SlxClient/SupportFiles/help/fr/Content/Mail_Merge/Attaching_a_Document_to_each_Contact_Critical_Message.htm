Ajout d'un document à chaque message critique de contact ou de lead

Pour les utilisateurs de Web Infor CRM, ce message critique est applicable lors d'une fusion pour un grand nombre de contacts.

Si vous cliquez sur « Oui », un document est joint pour chaque contact et si le nombre de contacts est élevé, le processus de fusion est anormalement lent. Cela est dû au fait que chaque document doit être transféré par HTTP au serveur Web ; il est ensuite copié vers le chemin d'accès de pièce jointe et journalisé dans le chemin d'accès de connexion. Ce processus peut être long et ralentir le serveur Web. Si vous fusionnez pour un grand nombre de contacts, il est recommandé de disposer d'une connexion à haut débit. Idéalement, il est également recommandé de ne pas fusionner plus de 100 contacts si le chemin d'accès à la pièce jointe et le chemin d'accès de connexion se situent aussi sur le serveur Web (conditions optimales).

Si vous cliquez sur « Non » l'option reste désélectionnée et aucun document n'est joint.
