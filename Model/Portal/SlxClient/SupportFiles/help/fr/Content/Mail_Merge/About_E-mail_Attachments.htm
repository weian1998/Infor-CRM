A propos des pièces jointes

Lorsque vous envoyez un e-mail à un contact, vous pouvez joindre un fichier ou un lead. Si les options appropriées ont été sélectionnées, le fichier est stocké dans l'onglet Pièces jointes pour le contact et le compte ou le lead, ainsi que dans le dossier Documents par défaut. Pour les utilisateurs de Web Infor CRM, le fichier se trouve dans le dossier Documents du serveur Web. Sur l'onglet Pièces jointes, le nom de fichier est le nom exact du fichier ou du document. Dans le dossier Documents, le nom du fichier contient le nom du document ainsi qu'un code de référence interne attribué par Infor CRM.

Vous pouvez joindre des fichiers Windows ou de bibliothèque Infor CRM à un modèle. Lorsque vous envoyez par e-mail le modèle à l'occasion d'un publipostage, les fichiers joints sont envoyés aux contacts ou aux leads. Si vous copiez un modèle, toutes les pièces jointes sont aussi copiées dans le nouveau modèle. Vous pouvez supprimer une pièce jointe d'un modèle.

Pour joindre un fichier à un modèle
  1. Ouvrez l'Editeur de modèles.
  2. Effectuez l’une des opérations suivantes :
  • Pour joindre un fichier Windows, cliquez sur le bouton Insérer un fichier, naviguez jusqu'à l'emplacement du fichier, puis cliquez sur Insérer un fichier.
  • Pour joindre un fichier de bibliothèque Infor CRM, sur le menu ou le ruban Insérer, cliquez sur Bibliothèque. Dans la boîte de dialogue Bibliothèque, cliquez sur le(s) fichier(s) puis sur Insérer.
  1. Vous pouvez joindre un autre fichier ou cliquer sur le bouton Enregistrer et fermer.
Pour supprimer un fichier d'un modèle
  1. Ouvrez l'Editeur de modèles.
  2. Dans la zone Pièce(s) jointe(s), sélectionnez la pièce jointe que vous souhaitez supprimer.
  3. Effectuez l’une des opérations suivantes :
  • Cliquez sur le bouton droit de la souris, puis cliquez sur Supprimer.
  • Appuyez sur Supprimer.
  1. Vous pouvez supprimer un autre fichier ou cliquer sur le bouton Enregistrer et fermer.
