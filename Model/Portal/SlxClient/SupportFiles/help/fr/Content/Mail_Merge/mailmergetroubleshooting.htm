Dépannage de l'intégration du bureau - Publipostage

Vous risquez de rencontrer les problèmes suivants avec l'intégration du bureau et le publipostage.

Problème :

« Impossible de charger le service de publipostage. Cette erreur peut survenir si l'intégration du bureau Infor CRM n'est pas installée ou si elle est mal installée. »

Causes possibles et solutions :

  • Le module d'intégration du bureau n'est pas installé.
  • Vous devez installer le module d'intégration du bureau.
  • Le module d'intégration du bureau a été installé avant votre navigateur.
  • Si vous avez installé votre navigateur actuel après avoir installé le module d'intégration du bureau, vous devez réinstaller le module d'intégration du bureau en utilisant votre navigateur actuel.

    		•

    Cela peut survenir si vous avez installé Firefox après avoir installé le module d'intégration du bureau en utilisant Internet Explorer.

Problème :

« Une erreur s'est produite dans GetMailMergeService(). Cette erreur peut survenir si l'intégration du bureau Infor CRM n'est pas installée ou si elle est mal installée. Le nom de classe spécifié ne peut pas être mappé sur un objet COM. Détails : nom=Erreur; ligne=257; source=sage-mailmerge-loader-debug.js. »

Causes possibles et solutions :

  • L'un des .dlls du publipostage ne s'est pas installé correctement
  • Vous devez réinstaller le module d'intégration du bureau.

Problème :

« Une erreur est survenue dans Sage.MailMergeService.PopulateWriteMenu(). Cette erreur peut survenir si l'intégration du bureau Infor CRM n'est pas installée ou si elle est mal installée. Le nom de classe spécifié ne peut pas être mappé sur un objet COM. Détails : nom=SyntaxError; numéro=-2146827287. »

Causes possibles et solutions :

  • L'un des .dlls du publipostage ne s'est pas installé correctement
  • Vous devez réinstaller le module d'intégration du bureau.

Problème :

« Une erreur s'est produite dans ExecuteWriteAction(). L'objet ne prend pas en charge cette propriété ou méthode. »

Causes possibles et solutions :

  • Votre site Client Web Infor CRM n'est pas un site de confiance.
  • Ajoutez le site Client Web Infor CRM à la liste des sites de confiance de votre navigateur Internet. Pour plus d'informations, consultez le système d'aide de votre navigateur Internet.
  • Votre navigateur s'exécute en mode protégé.
  • Si vous utilisez Internet Explorer v. 8.0, vous devez désactiver le mode protégé. Pour plus d'informations, consultez le système d'aide de votre navigateur Internet.

Causes possibles et solutions :

  • Votre site Client Web Infor CRM n'est pas un site de confiance.
  • Ajoutez le site Client Web Infor CRM à la liste des sites de confiance de votre navigateur Internet. Pour plus d'informations, consultez le système d'aide de votre navigateur Internet.

  • Si vous utilisez Internet Explorer v. 8.0, vous devez désactiver le mode protégé. Pour plus d'informations, consultez le système d'aide de votre navigateur Internet.
