Utilisation du menu Rédiger

Le menu Rédiger Infor CRM pour effectuer un publipostage ou envoyer un e-mail. Vous pouvez communiquer avec un seul contact ou lead ou avec un groupe de contacts ou de leads.

Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Infor CRM. Vous pouvez installer l'intégration du bureau Infor CRM depuis l'écran de connexion du client Web. 

Utiliser le menu Rédiger pour :

  • rédiger un e-mail à un contact ou à un lead ;
  • rédiger un e-mail, une lettre ou un fax à l'aide d'un modèle ;
  • utiliser le moteur de publipostage pour envoyer des messages à un groupe de contacts, de prospects, de comptes, etc.
  • créer et imprimer des étiquettes d'adresse ou des enveloppes ;
  • créer, modifier et supprimer des modèles.

Les options de menu Rédiger un E-mail/Lettre/Fax à partir d'un modèle contiennent les modèles utilisés le plus récemment . Le nombre d'éléments à afficher est défini par vous ou votre administrateur système lors de la configuration de vos options Infor CRM.

Si une liste de modèles est grisée, vous risquez de ne pas pouvoir accéder à l'élément de menu. Si vous avez besoin d'y accéder, contactez votre administrateur système.

Remarques sur le publipostage

  • Microsoft Word est déjà ouvert : si une autre instance de Word est déjà ouverte, vous serez invité à fermer Word pour que la fusion puisse se poursuivre.
  • Sortie e-mail HTML : la fonction de publipostage utilise le formatage HTML de Microsoft Internet Explorer HTML pour les e-mails. Lorsque certains destinataires ne peuvent pas lire les messages HTML, les messages doivent être envoyés sous forme de message MIME. Quant aux graphiques et aux informations d'en-tête et de pied de page, MS Word ne les affiche pas dans une page HTML, mais il enregistre les données correspondantes.
  • Fusion dans la table des matières : la fusion dans un modèle qui contient une table des matières n'est pas prise en charge. Il est recommandé de créer la table des matières après l'exécution de la fusion. Utilisez la fonction Modifier la sortie de publipostage avant l'envoi, puis insérez la table des matières.
  • Placer des champs de fusion dans des zones de texte : ll est impossible de placer des champs de fusion Infor CRM dans une zone de texte. La fusion ne sera pas exécutée correctement.
