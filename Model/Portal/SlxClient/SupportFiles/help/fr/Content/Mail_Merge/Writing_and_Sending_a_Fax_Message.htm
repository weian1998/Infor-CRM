Rédaction et envoi d'un message par fax

La procédure de rédaction et d'envoi d'un fax à un seul contact ou lead est décrite ci-après. Si vous souhaitez envoyer un fax à plusieurs contacts ou leads, utilisez l'option Publipostage.

Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Infor CRM. Vous pouvez installer l'intégration du bureau Infor CRM depuis l'écran de connexion du client Web. 

Pour rédiger et envoyer un message par fax

  1. Dans le menu Rédiger, pointez sur Fax à partir d'un modèle, puis sélectionnez un modèle dans la liste de modèles récemment utilisés ou cliquez sur Autres modèles pour choisir un autre modèle. Si vous êtes invité à ajouter le modèle à la liste des modèles les plus récemment utilisés, cliquez sur Oui pour ajouter le modèle à la liste, sinon cliquez sur Non.

    Si un service de fax compatible n'a pas été installé sur votre ordinateur, l'option Autres modèles est désactivée.

  2. Votre prochaine étape dépend de votre cible dans Client Web Infor CRM.

    • Si la cible est un contact ou un lead particulier, la fusion se poursuit.
    • Si aucun contact ou lead particulier n'est activé, utilisez la boîte de dialogue Sélectionner le nom ou Sélectionner le lead pour sélectionner un contact ou un lead.
    • Si votre cible est un compte ou une opportunité, la boîte de dialogue Sélectionner un contact apparaît et vous demande de choisir un contact.
    • Si votre cible est un ticketle document est fusionné au contact associé.

  3. La boîte de dialogue Sélectionner le type d'adresse apparaît si le contact a plusieurs adresses et que le modèle contient des champs de fusion de la table d'adresses.
  4. Si nécessaire, apportez des modifications au modèle dans la boîte de dialogue Editeur de fax.
  5. Fermez l'Editeur de fax.
  6. Dans la boîte de dialogue Options de fax, saisissez un objet dans la zone Objet.
  7. Dans la liste Page de garde, effectuez l'une des opérations suivantes :
    • Conservez la page de garde s'affichant par défaut.
    • Cliquez sur la flèche, puis cliquez sur une autre page de garde.
    • Cliquez sur la flèche, puis cliquez sur Aucune page de garde.
  1. Si vous souhaitez inclure un message sur la page de garde, effectuez l'une des opérations suivantes :
    • S'il s'agit d'un message court, saisissez-le dans la zone Message.
    • Pour un message long, cliquez sur Message. Dans la boîte de dialogue Modifier le message, saisissez votre message, puis cliquez sur OK.
  1. (Facultatif) Pour changer la planification de distribution par défaut du fax, cliquez sur Options d'envoi. Après avoir défini les options d'envoi, cliquez sur OK.
  2. Cliquez sur Envoyer.

  3. Utilisez la boîte de dialogue Terminer une activité pour modifier les informations de l'historique, saisir des notes ou planifier une activité de suivi, puis cliquez sur Terminer. 

  4. Si vous avez planifié une activité de suivi, complétez la boîte de dialogue Planifier une activité, puis cliquez sur cliquez sur Enregistrer.
