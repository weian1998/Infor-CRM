Utilisation du publipostage

Vous pouvez utiliser la fonction de publipostage lorsque vous voulez correspondre avec un groupe de contacts, de comptes, d'opportunités ou de leads. Le publipostage vous aide à simplifier le processus d'envoi des mêmes informations à un grand nombre de personnes, en fusionnant les informations dans un modèle. Vous pouvez effectuer une fusion vers un fichier, envoyer un e-mail, un fax ou imprimer le document fusionné.

  • Si le menu Rédiger est indisponible, vous devez télécharger l'intégration du bureau Infor CRM. Vous pouvez installer l'intégration du bureau Infor CRM depuis l'écran de connexion du client Web. 

  • si le modèle choisi pour la fusion comporte une table des matières avec des champs de fusion MS Word, les champs ne fusionneront pas correctement. Il est recommandé d'utiliser la fonction Modifier les différents <documents> avant de les envoyer et de créer la table des matières une fois que la fusion est terminée.
  • Lorsque vous sélectionnez une option de sortie dans l'onglet Options de fusion, seules les options disponibles apparaissent dans la sélection. Par exemple, si vous n'avez pas configuré de système de messagerie électronique, l'option E-mail ne figure pas parmi les options de sortie disponibles.

Pour utiliser le publipostage

  1.  Effectuez l’une des opérations suivantes :
    • Dans le menu Rédiger, cliquez sur Publipostage.
    • Dans certaines vues du Volet des tâches, cliquez sur Publipostage.
  2. Sélectionnez un modèle dans la boîte de dialogue Publipostage. 
  3. Cliquez sur l'onglet Options de fusion pour définir les options de fusion.
  4. Cliquez sur l'onglet Options de l'historique pour définir les options de l'historique.
  5. Si vous avez sélectionné Imprimante en tant qu'option de sortie dans l'onglet Options de fusionet que vous imprimez des étiquettes d'adresse, veillez à ce que du papier et des étiquettes soient chargés dans l'imprimante.
  6. Cliquez sur l'option Fusionner.
     
  7. Si vous avez sélectionné Modifier le modèle pour tous les contacts dans Options de fusion, dans la boîte de dialogue d'éditeur, effectuez vos modifications du modèle, puis cliquez sur Enregistrer et fermer.
  8. Si le modèle requiert une sélection d'adresse ou que l'enregistrement a plusieurs adresses, sélectionnez une adresse dans la boîte de dialogue Sélectionner le type d'adresse.
  9. Si vous avez sélectionné Modifier les différents <documents> avant de les envoyer dans Options de fusion, utilisez la boîte de dialogue Modification des documents fusionnés pour sélectionner un élément et effectuer les modifications. Une fois toutes les modifications effectuées, cliquez sur Terminé.
  10. Si vous recevez un message indiquant qu'un certain nombre de fusions vont échouer, choisissez de continuer ou d'annuler la fusion. Si vous choisissez de poursuivre, un message d'erreur s'affiche à la fin de la fusion, vous demandant si vous souhaitez afficher le groupe Problèmes de publipostage. Si vous cliquez sur Oui, le groupe Problèmes du publipostage s'ouvre. Si vous cliquez sur Non, la boîte de dialogue se referme et vous revenez à la vue sur laquelle vous vous trouviez au début du publipostage.
