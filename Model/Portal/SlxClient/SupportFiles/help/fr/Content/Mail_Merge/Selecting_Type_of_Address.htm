Sélection du type d'adresse

La boîte de dialogue Sélectionner le type d'adresse apparaît lorsqu'un publipostage est effectué vers un contact ayant plusieurs adresses ou un groupe de contacts et lorsque vous utilisez un modèle qui contient des champs de fusion Infor CRM issus de la table Adresse.

Pour sélectionner un type d'adresse

  1. Dans la boîte de dialogue Sélection du type d'adresse, sélectionnez l'une des options suivantes :
    • Adresse principale du contact : cette option utilise les adresses définies comme « principales » pour tous les contacts traités dans le publipostage.
    • Adresse d'expédition du contact : cette option utilise les adresses définies comme « expédition » pour tous les contacts traités dans le publipostage.
    • Autre adresse : utilisez le flèche déroulante vers le bas  pour effectuer une sélection dans la liste des autres types d'adresses. L'adresse par défaut de tous les contacts traités dans le publipostage pour lesquels aucune adresse n'a été sélectionnée, est l'adresse principale.
  2. Cliquez sur OK.

Si vous ne souhaitez pas que la boîte de dialogue Sélectionner le type d'adresse s'affiche mais que vous avez besoin d'informations sur les adresses dans le modèle, choisissez un champ d'adresse associé aux tables Contact ou UserInfo. Quel que soit le type de fusion, l'utilisation des champs d'adresse associés aux tables Contact ou UserInfo force le publipostage à utiliser l'adresse principale des contacts et empêche la boîte de dialogue Sélectionner le type d'adresse d'apparaître.
