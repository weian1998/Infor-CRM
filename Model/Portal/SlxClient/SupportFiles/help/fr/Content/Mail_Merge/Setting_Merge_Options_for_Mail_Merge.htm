Définition des options de fusion du Publipostage

L'onglet Options de fusion vous permet de définir les options de fusion du publipostage.

Pour définir les options de fusion

  1. Dans l'onglet Options de fusion dans la zone Modèle, cliquez sur le dossier approprié et sélectionnez un modèle.
    Lors de la sélection de modèles de leads, sélectionnez uniquement ceux qui se terminent par "(Lead)".
  2. Dans la zone Fusionner avec, sélectionnez une des Infor CRMoptions de source.
      • Enregistrement actif : sélectionnez cette option pour utiliser l'enregistrement actif. L'information source peut contenir un contact pour l'enregistrement de compte, de contact, ou d'opportunité actif, ou pour un lead.
      • Groupe actif : sélectionnez cette option pour utiliser le groupe de contacts, de comptes, de leads ou d'opportunités actuellement actif. Les informations sur la source sont les noms des contacts, les noms des comptes; les noms des leads et les adresses.
      • Groupe spécifique : sélectionnez un groupe dans la liste. Les informations de source sont les noms de contacts, les noms de comptes, les noms de leads et les adresses pour le groupe.
      • Spécifique : sélectionnez Compte ou Opportunité dans la liste puis utilisez la recherche pour trouver un enregistrement. Compte - Les informations sur la source sont les noms des contacts, le nom du compte et les adresses. Opportunité - Les informations sur la source sont les noms des contacts, le nom du compte et les adresses associées à l'opportunité.
  3. Dans la liste Sortie, sélectionnez une des options et renseignez les cases d'informations pour l'option sélectionnée. La sortie Imprimante vous permet d'imprimer les étiquettes d'adresse ou les enveloppes.

    • E-mail

      1. Dans la liste Sortie, sélectionnez E-mail.
        Dans la zone De, l'adresse électronique de l'utilisateur connecté s'affiche.
      2. (Facultatif) Cliquez sur Carnet d'adresses pour sélectionner une autre adresse électronique de l'utilisateur comme expéditeur.
         
      3. (Facultatif) Cliquez sur Carnet d'adresses dans les zones CC et CCI pour sélectionner d'autres destinataires.
      4. Dans la zone Objet, saisissez les informations que vous souhaitez faire figurer dans la ligne objet de l'e-mail.
      5. (Facultatif) Dans la case Joindre, cliquez sur la flèche adjacente au dossier, puis sélectionnez le type de pièce jointe devant accompagner le document fusionné.
      6. Dans la zone Format, le format par défaut du modèle sélectionné est affiché, mais vous pouvez sélectionner un autre format.
      7. L'option Enregistrer une copie dans Eléments envoyés est cochée par défaut, mais vous pouvez la désactiver.
        Cette option enregistre une copie de tous les messages de publipostage dans le dossier Outlook ou dans le dossier Eléments envoyés du client de messagerie.

    • Fax

      1. Dans la liste Sortie, sélectionnez Fax.
      2. Dans la zone Objet, saisissez un objet.
      3. Dans la liste Page de garde, effectuez l'une des opérations suivantes :
        • Conservez la page de garde s'affichant par défaut.
        • Cliquez sur la flèche et sélectionnez une page de garde différente.
        • Cliquez sur la flèche et sélectionnez Aucune page de garde.
      4. Pour faire figurer un message sur la page de garde, effectuez l'une des opérations suivantes :
        • Pour un message court, tapez le message dans la case.
        • Pour un message long, cliquez sur Message. Dans la boîte de dialogue Modifier le message, saisissez votre message, puis cliquez sur OK.

    • Imprimante

      1. Dans la liste Sortie, sélectionnez Imprimante. Le nom de l'imprimante par défaut apparaît dans la zone, mais vous pouvez en sélectionner une autre.
      2. (Facultatif) Pour configurer l'imprimante, cliquez sur Propriétés de l'imprimante.
      3. Pour imprimer les étiquettes d'adresse ou les enveloppes, cliquez sur Créer les étiquettes d'adresse.
      4. Pour formater des étiquettes d'adresse ou des enveloppes, cliquez sur Configurer les étiquettes.

    • Fichier

      1. Dans la liste Sortie, sélectionnez Fichier.
      2. Dans la zone Répertoire, cliquez sur un dossier pour sélectionner l'emplacement du fichier.
        Le fichier de publipostage sera nommé selon la convention suivante : Nom du Modèle (NOM DE L'ENTITÉ)DATEHEURE.DOC. Si des contacts ou des leads ont le même nom, Infor CRM ajoute un numéro après le nom lorsqu'un fichier portant ce nom existe déjà.
        Exemple :
        Lettre par défaut (Keith Abbott) 13-07-2005 14.07.17.doc
        Lettre par défaut (Keith Abbott2) 09-14-2005 12.11.43.doc
  4. Pour estimer la quantité de papier ou le nombre d'étiquettes à charger dans l'imprimante, cliquez sur Compter pour afficher les informations concernant le compte du publipostage.
    Les informations de comptage vous permettent également de déterminer s'il manque des adresses électroniques et des numéros de fax, avant la fusion.
  5. Dans la case Options de fusion, sélectionnez une ou plusieurs des options. Les options disponibles dépendent du type de sortie que vous avez sélectionné.
    • Modifier des <e-mails/lettres/fax> individuels avant de les envoyer
      • Vous souhaitez modifier des éléments individuels de publipostage avant l'envoi. Par exemple, vous pouvez ajouter un mot personnel à une des lettres. Si cette option est sélectionnée, la boîte de dialogue Modifier les documents fusionnés s'affiche pendant le processus de fusion pour vous permettre de choisir des documents et d'en modifier le contenu.
    • Modifier le modèle pour tous les enregistrements
      • Vous souhaitez modifier le modèle pour la fusion en cours. Par exemple, vous avez sélectionné un modèle de lettre de base, mais vous souhaitez ajouter des champs de fusion supplémentaires au modèle. Si cette option est sélectionnée, l'Éditeur de modèle s'ouvre pendant le processus de fusion pour vous permettre d'apporter des modifications au modèle.
    • Ne PAS envoyer aux contacts portant la mention « Ne pas démarcher »
      • Cette option est sélectionnée par défaut pour vous permettre de ne pas envoyer des éléments de publipostage à des contacts qui ont spécifiquement demandé de ne pas recevoir de courriers. La case à cocher Ne pas démarcher apparaît dans les vues Détails du contact et Détails du lead.
      • Vous pouvez désélectionner l'option pour envoyer les éléments fusionnés à tous les contacts
  6. Une fois que vous avez configuré vos options de fusion et d'historique, cliquez sur Fusionner.
