Définition des options d'historique du publipostage

L'onglet Options d'historique vous permet de créer un élément d'historique pour le publipostage et de planifier une activité de suivi.

Pour définir les options d'Historique

  1. Dans la zone Historique de l'onglet Options de l'historique, effectuez l'une des opérations suivantes :
    1. Pour joindre la correspondance à l'onglet Pièces jointes de chaque enregistrement, sélectionnez Joindre<e-mail/fax/lettre>à chaque<Contact/Lead>.
      Le fichier de fusion est joint à chaque lead ou contact, compte et opportunité selon l'option Fusionner avec sélectionnée dans les Options de fusion.
      Pour les utilisateurs Web Infor CRM, cette option entraîne l'affichage d'un message d'avertissement durant la fusion.
    2. Pour ajouter un élément à l'historique de l'enregistrement, sélectionnez Ajouter un enregistrement d'historique pour chaque <Contact/Lead>.
    3. Dans la zone Résultat, cliquez sur les points de suspension pour sélectionner un type de résultat dans la liste de Codes des résultats des tâches à faire disponibles.
    4. Dans la zone Objet, cliquez sur les points de suspension pour sélectionner un type d'objet dans la liste.
    5. Dans la zone Catégorie, cliquez sur les points de suspension pour sélectionner un type de catégorie dans la liste.
    6. Dans la case Notes, saisissez des notes ou des éléments importants à inclure dans l'historique.
  2. Pour planifier une activité de suivi, sélectionnez Planifier une activité de suivi et procédez comme suit :
    1. Si vous recevez le message de confirmation d'activité, cliquez sur OK.
    2. Dans la liste, sélectionnez le type d'activité de suivi.
    3. Pour recopier les notes dans l'activité de suivi, sélectionnez Reporter les notes.
    4. Dans la zone Date de début, cliquez sur le bouton Agenda pour sélectionner une date de début de l'activité.
    5. Effectuez l’une des opérations suivantes :
      • Sélectionnez la case Sans horaire pour créer une activité qui a lieu à une date spécifique mais qui n'a pas d'heure spécifique et sans durée particulière.
      • Cliquez sur Durée la flèche du menu déroulant et effectuez votre sélection dans la liste.
    6. Pour définir une alarme d'activité, sélectionnez Rappel. Utilisez les zones de la liste déroulante pour sélectionner un nombre de minutes, d'heures ou de jours. Vous devez saisir une plage horaire entre 1 et 240.
    7. Dans la zone Objet, cliquez sur les points de suspension pour sélectionner un type d'objet.
    8. Dans la zone Notes, effectuez l'une des opérations suivantes :
      • Saisissez des notes dans la zone.
      • Cliquez sur Notes pour ouvrir une zone de saisie plus grande, tapez vos notes, puis cliquez sur OK.
    9. Dans la zone Priorité, sélectionnez l'importance de l'activité.
    10. Dans la zone Catégorie, cliquez sur les points de suspension pour sélectionner un type de catégorie.
    11. Dans la zone Responsable, sélectionnez un responsable d'activité à l'aide de la Recherche.
  3. Une fois que vous avez configuré vos options de fusion et d'historique, cliquez sur Fusionner.
