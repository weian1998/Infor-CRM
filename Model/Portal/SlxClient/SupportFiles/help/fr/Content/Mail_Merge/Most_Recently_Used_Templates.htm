Modèles utilisés le plus récemment

Dans la liste des modèles utilisés le plus récemment, vous pouvez sélectionner un modèle utilisé récemment sans avoir à ouvrir la boîte de dialogue Sélectionner un modèle. C'est lors de la configuration des options Infor CRM que vous-même ou l'administrateur système déterminez le nombre de modèles récemment utilisés affichés dans cette liste.

  • Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Infor CRM. Vous pouvez installer l'intégration du bureau Infor CRM depuis l'écran de connexion du client Web. 

  • La liste des modèles utilisés récemment n'est pas mise à jour automatiquement lorsqu'un modèle est supprimé. Si vous sélectionnez un modèle dans la liste et que le modèle a été supprimé ou que vous n'y avez plus accès, un message vous informe que le modèle sera supprimé du menu. Contactez votre administrateur si vous souhaitez avoir accès au modèle non disponible.

Pour sélectionner un modèle dans la liste des modèles utilisés le plus récemment

  1. Depuis le menu Rédiger, sélectionnez l'option de rédiger un e-mail, une lettre, ou un fax à l'aide d'un modèle.

Une liste des modèles utilisés le plus récemment s'affiche.

  1. Cliquez sur le modèle que vous souhaitez utiliser.
