Rédaction et envoi de lettres ou de mémos

Les instructions suivantes indiquent comment rédiger et envoyer une lettre à un seul contact ou lead. Si vous souhaitez envoyer une lettre ou un mémo à plusieurs contacts ou leads, utilisez l'option Publipostage.

Si le menu Rédiger n'est pas disponible, vous devez télécharger l'intégration du bureau Infor CRM. Vous pouvez installer l'intégration du bureau Infor CRM depuis l'écran de connexion du client Web. 

Pour rédiger une lettre :

  1. Dans le menu Rédiger, effectuez l'une des opérations suivantes :
    • Pointez sur Lettre à partir d'un modèle, puis sélectionnez un modèle dans la liste des modèles récemment utilisés.
    • Pour sélectionner un autre modèle, cliquez sur Autres modèles. Si vous sélectionnez un modèle particulier pour la première fois, la boîte de dialogue Ajouter le modèle au menu apparaît. (Facultatif) Sélectionnez Oui pour ajouter le modèle au menu « Lettre à partir d'un modèle ... » existant.

  2. Votre prochaine étape dépend de votre cible dans Client Web Infor CRM.

    • Si la cible est un contact ou un lead particulier, la fusion se poursuit.
    • Si aucun contact ou lead particulier n'est activé, utilisez la boîte de dialogue Sélectionner le nom ou Sélectionner le lead pour sélectionner un contact ou un lead.
    • Si votre cible est un compte ou une opportunité, la boîte de dialogue Sélectionner un contact apparaît et vous demande de choisir un contact.
    • Si votre cible est un ticketle document est fusionné au contact associé.

  3. La boîte de dialogue Sélectionner le type d'adresse apparaît si le contact a plusieurs adresses et que le modèle contient des champs de fusion de la table d'adresses.
  4. (Facultatif) Vous pouvez entrer des informations supplémentaires ou apporter des changements au modèle dans la boîte de dialogue Editeur de lettre.
  5. Choisissez entre :
    • Enregistrer la lettre ou le mémo dans un fichier. Dans ce cas, choisissez l'une des options Enregistrer dans le menu Fichier.
    • Imprimer la lettre ou le mémo. Dans ce cas, cliquez sur le bouton Imprimer.
  6. Cliquez sur Enregistrer et fermer pour fermer l'Editeur de lettre.
  7. A l'invite Créer une étiquette d'adresse, cliquez sur Oui si vous souhaitez imprimer une étiquette d'adresse ou une enveloppe pour la lettre, sinon cliquez sur Non.

  8. Utilisez la boîte de dialogue Terminer une activité pour modifier les informations de l'historique, saisir des notes ou planifier une activité de suivi, puis cliquez sur Terminer. 

  9. Si vous avez planifié une activité de suivi, complétez la boîte de dialogue Planifier une activité, puis cliquez sur cliquez sur Enregistrer.
