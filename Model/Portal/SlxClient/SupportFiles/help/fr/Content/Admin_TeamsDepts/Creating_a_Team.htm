Création d'une équipe

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Une équipe peut être constituée d'utilisateurs, d'équipes et de services. Lorsque vous créez une équipe, vous devez désigner une personne comme propriétaire de cette équipe.

Lorsque vous accordez à un utilisateur le droit de créer et d'assurer le suivi des équipes, vous devez tenir compte des règles suivantes.

Un utilisateur ne peut jamais :

  • s'auto-ajouter à une équipe existante ;
  • s'auto-supprimer d'une équipe ;
  • supprimer l'administrateur système d'une équipe ;
  • supprimer une équipe.

En savoir plus sur la sécurité d'équipe...

Lorsqu'un utilisateur est ajouté à une équipe, sa sécurité au sein de cette équipe est définie par défaut sur le profil défini sur l'onglet Sécurité. Le profil de sécurité peut être modifié sur chaque équipe dont l'utilisateur est membre. Cette opération n'a aucune incidence sur le profil par défaut de l'utilisateur, pas plus que sur le profil de toute autre équipe dont cet utilisateur est membre.

Si un utilisateur possède plusieurs profils de sécurité (par exemple, l'utilisateur est ajouté à une équipe en tant qu'équipe ou service imbriqué(e), mais il est également un membre direct de l'équipe parent), les profils de sécurité partagent le même code de sécurité (propriétaire). Lorsque les droits d'accès de sécurité d'un champ sont déterminés, plusieurs profils peuvent être renvoyés. Dans le client Web, l'accès est déterminé en combinant les droits d'accès maximum de tous les profils attribués à cet utilisateur et propriétaire.

Pour créer

  1. Dans le menu Administration, cliquez sur Nouvelle équipe.
  2. Dans le champ Nom, tapez le nom de l'équipe.
  3. Dans la zone Propriétaire, sélectionnez un propriétaire d'équipe. Une équipe peut avoir plusieurs propriétaires.
  4. Dans la zone Profil de sécurité par défaut, sélectionnez un profil par défaut pour l'équipe. Ce profil est appliqué lorsque l'équipe est ajoutée à une autre équipe.
  5. Cochez la case Ajouter automatiquement le responsable de l'utilisateur à l'équipe si vous souhaitez ajouter le responsable de l'utilisateur à l'équipe lorsque vous ajoutez un nouveau membre à une équipe.
  6. Cliquez sur Enregistrer..
  7. Ajouter des membres à une équipe.
  8. Si nécessaire, modifiez les droits de sécurité de chaque membre de l'équipe.
  9. Cliquez sur Enregistrer..
