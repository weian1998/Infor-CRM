Supprimer ou renommer une équipe

Utilisez la vue Détails de l'équipe pour supprimer ou renommer une équipe.

Procédure

Supprimer une équipe

  1. Ouvrez la vue Détails de l'équipe de l'équipe que vous souhaitez supprimer.
  2. Cliquez sur Supprimer.

Renommer une équipe

  1. Ouvrez la vue Détails de l'équipe de l'équipe que vous souhaitez renommer.
  2. Dans le champ Nom, tapez le nouveau nom de l'équipe.
  3. Cliquez sur Enregistrer..
