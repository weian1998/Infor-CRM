Vue Détails des services

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Utilisez la vue Détails pour afficher, ajouter ou modifier des informations sur un enregistrement sélectionné. La vue Détails est composée de zones d'informations, d'onglets et d'une section centrale définie par l'utilisateur.

Procédure

Ouvrez la vue Détails du service
  1. Sur la barre de navigation, cliquez sur Administration, puis sur Services pour ouvrir la vue Liste des services.
  2. Cliquez sur un service dans la liste, ou utilisez la fonction Rechercher pour localiser un service.

Les informations sur le service s'ouvrent dans la vue Détails.

Utilisez la fonction Rechercher pour rechercher des enregistrements de service

Ouvrir un groupe

Passer en vue Liste
  • Dans la barre d'outils, cliquez sur le bouton Vue Liste, ou cliquez sur l'onglet du groupe actif.

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe
  • Remplacer sur toutes les équipes
  • Copier le service

Zones d'informations

Les zones d'informations sur le service contiennent des informations essentielles sur le service.

Onglets de la vue Détails

Chaque onglet affiche des informations spécifiques sur le profil du service. Cliquez sur un lien pour voir davantage d'informations sur chaque onglet.

Membres
Equipes

section définie par l'utilisateur

Vous pouvez faire glisser les onglets dans la section utilisateur dans la partie centrale. L'onglet y restera jusqu'à ce qu'il soit ramené vers la section inférieure.
