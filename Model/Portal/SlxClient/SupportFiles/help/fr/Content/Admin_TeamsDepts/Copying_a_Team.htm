Copie d'une équipe

Lorsque vous copiez une équipe existante, tous ses membres sont inclus dans la nouvelle équipe.

Pour copier

  1. Ouvrez la vue Détails de l'équipe de l'équipe que vous souhaitez copier.
  2. Dans le volet Tâches courantes, cliquez sur Copier l'équipe.
  3. Pour confirmer, cliquez sur OK.
  4. Dans le champ Nom, tapez le nouveau nom de l'équipe.
  5. Si cela est nécessaire, modifiez les membres.
  6. Cliquez sur Enregistrer..
