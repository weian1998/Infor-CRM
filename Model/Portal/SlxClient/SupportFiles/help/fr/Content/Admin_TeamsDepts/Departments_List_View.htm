Utilisation de la vue Liste des services

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

La vue Liste des services affiche les services dans une grille. La grille fournit des informations sur chaque service.

  • Pour trier la grille selon un champ particulier, cliquez sur l'en-tête de colonne correspondant.
  • Pour afficher tous les services d'un type particulier, cliquez sur un onglet dans la grille.

Procédure

Créer un service

Ouvrez la vue Liste des services
  • Dans la barre de navigation, cliquez sur Administration, puis sur Services.

Ouvrez la vue Détails du service
  • Cliquez sur le nom du service pour ouvrir la vue Détails du service.

Utilisez la fonction Rechercher pour rechercher des enregistrements de service

Travailler avec des enregistrements dans une grille

Actualiser les informations de la liste
  • Cliquez sur le bouton Actualiser.

Filtrer ma liste

Afficher un groupe

Utiliser les tâches courantes

Dans le Volet des tâches, sous Tâches courantes, vous pouvez :

  • Ajouter au groupe
  • Supprimer du groupe
  • Enregistrer les enregistrements en tant que groupe
  • Exporter vers un fichier
  • Ajouter à l'équipe
  • Supprimer de toutes les équipes
