Onglet Sources de leads

L'onglet Sources des leads apparaît dans la vue Détails du contact. Lorsque vous ajoutez un nouveau contact ou une nouvelle opportunité, vous pouvez lui associer une source de lead pour indiquer comment votre entreprise a eu connaissance de ce contact ou de cette opportunité (salons, publicités dans la presse, télémarketing, etc.). Cliquez sur une source de lead pour en afficher les informations détaillées. Les colonnes de l'onglet sont : date, description et code source.

Procédure

Ouvrir l'onglet Source des leads
  • Dans la vue Détails du contact, cliquez sur l'onglet Sources de leads. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Ajout d'une source de lead

Modification d'une source de lead

Supprimer une source de lead

  1. Sélectionnez l'enregistrement à retirer, puis cliquez sur le bouton Supprimer.

  2. Cliquez sur OK.
