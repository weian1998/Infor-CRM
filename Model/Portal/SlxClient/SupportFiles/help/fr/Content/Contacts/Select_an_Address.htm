Sélection d'une adresse alternative

Vous pouvez utiliser la boîte de dialogue Sélectionner une adresse pour sélectionner une adresse existante pour un nouveau contact. Le bouton Sélectionner une autre adresse n'est disponible que si des adresses existent déjà. Par exemple, si vous ajoutez un contact à un compte possédant plusieurs adresses, les adresses du compte sont disponibles.

Pour sélectionner une autre adresse

  1. À partir de la boîte de dialogue Informations du contact/compte, cliquez sur le bouton Sélectionner une autre adresse, s'il est disponible.

La boîte de dialogue Sélectionner une adresse s'ouvre.

  1. Cliquez sur l'adresse que vous souhaitez utiliser.
  2. Cliquez sur Enregistrer..
