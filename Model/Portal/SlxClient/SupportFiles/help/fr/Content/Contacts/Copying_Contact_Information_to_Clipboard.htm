Copie d'informations vers le Presse-papiers

Les informations sont copiées vers le Presse-papiers, afin de permettre leur utilisation dans d'autres applications. Ceci est utile lorsque vous avez besoin de copier des informations dans un e-mail ou dans un document Microsoft Word.

Cette fonctionnalité n'est pas disponible si vous utilisez le navigateur Firefox.

Pour copier des informations de contact à partir d'un contact

  1. Ouvrez la vue Détails du contact.
  2. S'il est disponible, cliquez sur le bouton Copier les informations dans le Presse-papiers.
  3. Lorsque la boîte de dialogue Copier du texte vers le Presse-papiers s'ouvre, sélectionnez le texte que vous souhaitez copier, appuyez sur Ctrl + C, puis cliquez sur Fermer.

Pour copier des informations de contact à partir d'un compte

  1. Ouvrez la vue Détails du compte.
  2. S'il est disponible, cliquez sur le bouton Copier les informations dans le Presse-papiers.
