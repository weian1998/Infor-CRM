Modification d'un contact

Utilisez la vue Détails du contact pour modifier les informations d'un contact. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur le bouton Enregistrer.

Pour modifier

  1. Dans la barre de navigation, cliquez sur Contacts.
  2.  Recherchez le contact à modifier, puis cliquez sur le nom du contact.
  3. Dans la vue Détails du contact, modifiez les informations sur le contact.
  4. Cliquez sur Enregistrer..
