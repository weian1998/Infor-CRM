Ajouter ou modifier un produit personnalisé

Vous pouvez ajouter ou modifier un produit personnalisé dans l'onglet Produit de bon de commande.

Vous ne pouvez pas ajouter un produit personnalisé à partir de la vue Insérer un bon de commande. Utilisez la vue Détails du bon de commande pour ajouter des produits personnalisés.

Pour ajouter ou modifier

  1. Ouvrez l'onglet Produits de bon de commande.
  2. Procédez d'une des manières suivantes :
    • Cliquez sur le bouton Ajouter un produit personnalisé.
    •  Cliquez sur le bouton Modifier pour le produit personnalisé à modifier.
  3. Dans la boîte de dialogue Produit personnalisé, définissez les éléments suivants :
    • Produit - Tapez un nom pour le produit personnalisé.
    • Description - Tapez une description du produit.
    • Code article - Tapez le code article du produit.
    • Famille - Sélectionnez la famille à laquelle le produit sera associé.
    • Prix - Tapez le prix du produit.
    • Remise - Tapez le pourcentage de remise sur le produit.
    • Prix ajusté - Affiche le prix de base du produit moins la remise.
    • Quantité - Tapez le nombre de produits pour ce bon de commande.
    • Prix étendu - Affiche le total calculé du prix du produit multiplié par la quantité, moins toute remise éventuelle. Ce calcul est basé sur la devise par défaut définie par votre administrateur.
    • Prix verrouillé - Sélectionnez pour verrouiller le prix du produit. Vous devez avoir le rôle approprié pour verrouiller les prix. (Intégration de la comptabilité uniquement)

  4. Cliquez sur OK.
