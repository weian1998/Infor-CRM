Onglet Facturation et expédition du bon de commande

L'onglet Adresse de facturation/expédition apparaît sur la vue Détails du bon de commande. Utilisez cet onglet pour gérer les adresses de facturation et d'expédition d'un bon de commande.

Pour ajouter ou modifier

  1. Dans la vue Détails du bon de commande, cliquez sur l'onglet Adresse de facturation/expédition. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Si nécessaire, modifiez les informations de l'Adresse de facturation.
  1. Pour sélectionner un Contact, cliquez sur le bouton Rechercher.
  2. Cliquez sur la la flèche du menu déroulant et effectuez votre sélection dans la liste Adresse.
  3. Pour modifier l'adresse, cliquez sur le bouton Modifier. Pour afficher une carte de l'adresse, cliquez sur le bouton MapQuest.

L'adresse du contact associé ne sera pas mise à jour avec ces modifications. Pour modifier les informations de l'adresse du contact, vous devez modifier l'adresse dans la vue Détails du contact associée.
  1. Ajoutez ou modifiez les informations de l'Adresse d'expédition.
  1. Pour sélectionner un Contact, cliquez sur le bouton Rechercher.
  2. Cliquez sur la la flèche du menu déroulant et effectuez votre sélection dans la liste Adresse.
  3. Pour modifier l'adresse, cliquez sur le bouton Modifier. Pour afficher une carte de l'adresse, cliquez sur le bouton MapQuest.

L'adresse du contact associé ne sera pas mise à jour avec ces modifications. Pour modifier les informations de l'adresse du contact, vous devez modifier l'adresse dans la vue Détails du contact associée.
  1. Cliquez sur Enregistrer..
