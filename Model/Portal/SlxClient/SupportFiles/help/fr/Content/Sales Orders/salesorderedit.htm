Modification d'un bon de commande

Utilisez la vue Détails du bon de commande pour modifier les informations d'un bon de commande. Pour annuler vos modifications, cliquez sur Réinitialiser avant de cliquer sur le bouton Enregistrer.

Pour modifier

  1. Dans la barre de navigation, cliquez sur Bons de commande pour ouvrir la vue Liste des bons de commande. Si nécessaire, utilisez la Recherche pour localiser un bon de commande, puis cliquez sur le numéro du bon de commande. Utilisez la fonction Rechercher pour rechercher des enregistrements.
  2. Modifiez les informations sur le bon de commande.
  3. Cliquez sur Enregistrer..
