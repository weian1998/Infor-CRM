Onglet Système client

Utilisez l'onglet Système client de la vue Détails de l'utilisateur pour définir les modèles de base d'un utilisateur et le propriétaire par défaut des comptes créés par cet utilisateur. Les utilisateurs peuvent modifier ces paramètres sur l'onglet Général dans Options.

Pour définir les options

  1. Dans la vue Détails de l'utilisateur, cliquez sur l'onglet Système client.
  2. Utilisez la section Propriétaire par défaut pour définir un propriétaire par défaut pour les nouveaux comptes.
    1. Dans la zone Comptes, sélectionnez le propriétaire par défaut des nouveaux comptes créés par cet utilisateur.
    2. Pour empêcher l'utilisateur de modifier le paramètre de propriétaire par défaut, décochez la case Permettre à l'utilisateur de modifier ces paramètres. En savoir plus...

    3. Si vous autorisez l'utilisateur à modifier ce paramètre, vous risquez d'avoir un propriétaire de compte défini par défaut sur un propriétaire qui n'adhère pas aux règles métier établies. Ceci risque de laisser des comptes en dehors des rapports ou des requêtes établis.

  3. Sélectionnez les modèles de publipostage de base à utiliser lors de la création d'un modèle d'e-mail, de fax ou de lettre pour un nouveau contact.
    1. Cliquez sur la flèche de la liste déroulante Type, et sélectionnez Contact.
    2. Cliquez sur le bouton Modèle d'e-mail de base Rechercher.

    3. Sélectionnez un modèle et cliquez sur OK.

    4. Répétez les étapes b et c pour le Modèle de lettre de base et le Modèle de fax de base.

  4. Sélectionnez les modèles de publipostage de base à utiliser lors de la création d'un modèle d'e-mail, de fax ou de lettre pour un nouveau lead.

    Lors de la sélection de modèles de leads, sélectionnez uniquement ceux qui se terminent par "(Lead)".
    1. Cliquez sur la flèche de la liste déroulante Type, et sélectionnez Lead.
    2. Cliquez sur le bouton Modèle d'e-mail de base Rechercher.

    3. Sélectionnez un modèle et cliquez sur OK.

    4. Répétez les étapes b et c pour le Modèle de lettre de base et le Modèle de fax de base.
  5. (Facultatif) Pour empêcher l'utilisateur de modifier les modèles par défaut, décochez la case Permettre à l'utilisateur de modifier ces paramètres.
  6. Cliquez sur Enregistrer..

Lors de la configuration des utilisateurs WebViewer, seule la section Propriétaire par défaut est disponible.
