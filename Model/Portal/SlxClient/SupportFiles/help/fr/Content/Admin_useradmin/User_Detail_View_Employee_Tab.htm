Onglet Employé

Utilisez l'onglet Employé de la vue Détails de l'utilisateur pour ajouter des informations de société et des informations personnelles chez un utilisateur. L'onglet Employé contient également des informations utilisateur qui permettent à Infor CRM de communiquer avec les applications de comptabilité tierces prises en charge.

Pour définir les options

  1. Dans la vue Détails de l'utilisateur, cliquez sur l'onglet Employé.
  2. Dans la section Travail, saisissez les informations de contact de l'utilisateur.
  3. Au besoin, dans le champ ID de comptabilité, saisissez le code d'identification comptable de l'utilisateur

L'ID de comptabilité est nécessaire pour le partage des données entre Infor CRM et les applications de comptabilité.

  1. Si votre société attribue des numéros à ses employés et à leur ordinateur, complétez les champs N° de salarié, Actif n° et Description de l'actif.
  2. Dans la section Personnel, saisissez les informations personnelles de l'utilisateur.
  3. Cliquez sur Enregistrer..
