Création d'un modèle utilisateur

Les modèles d'utilisateurs définissent les paramètres à utiliser lorsque vous créez de nouveaux utilisateurs. Nous recommandons d'utiliser les modèles car ils réduisent considérablement le temps nécessaire à l'administration.

Infor CRM comprend plusieurs modèles d'utilisateurs par défaut. Vous pouvez personnaliser les modèles d'utilisateurs par défaut ou en créer de nouveaux. En savoir plus...

Par exemple, vous pouvez créer un modèle d'utilisateurs nommé Responsable qui peut servir de profil générique d'utilisateur pour tous les responsables de votre société. Ainsi, lorsque vous créez des utilisateurs et leur appliquez ce modèle, leur profil est déjà configuré avec les paramètres appropriés.

Les modèles d'utilisateur n'utilisent pas de licence utilisateur Infor CRM et ne permettent pas de se connecter à Infor CRM.

Si le modèle n'est pas attribué au Rôle Utilisateur Standard, les nouveaux utilisateurs créés à partir du modèle n'auront pas accès à certaines fonctionnalités du client Web.

Pour créer

  1. Dans le menu Administration, cliquez sur Ajouter des utilisateurs.
  2. Dans la zone Ajouter, sélectionnez Modèle.
  3. Dans la zone Quantité, sélectionnez le nombre de modèles d'utilisateurs à ajouter.
  4. Si vous souhaitez modifier manuellement chaque onglet du profil du modèle pour configurer les paramètres, passez à l'étape 7.
  5. Pour créer le modèle à partir d'un modèle existant ou à l'aide du profil d'un utilisateur existant, cochez la case Créer à partir d'un profil. Sélectionnez ensuite l'une des options suivantes :
  • L'option Modèle ajoute un modèle basé sur un modèle existant. Dans la zone Modèles, sélectionnez le nom du modèle.
  • L'option Utilisateur existant ajoute un nouvel utilisateur à partir du profil d'un utilisateur existant. Dans le champ Utilisateur, sélectionnez le nom de cet utilisateur existant.
  1. Sélectionnez les onglets à copier du modèle ou du profil utilisateur. En savoir plus...
  • Général - les paramètres généraux et les paramètres du service sont copiés. les champs NomUtilisateur, Nom, Fonction et E-mail ne s'appliquent pas lorsque vous copiez des informations.
  • Employé - les champs Fax, Portable, Pager, Direct, N° ID, Comptabilité et tous les champs de données personnelles ne s'appliquent pas lorsque vous copiez des informations.
  • Sécurité - tous les paramètres de sécurité sont copiés.
  • Equipes -tous les paramètres d'équipes sont copiés.
  • Agenda - tous les paramètres d'agenda sont copiés.
  • Service/Assistance technique - tous les paramètres service et support technique sont copiés.
  • Options client (système client) - les paramètres du propriétaire du compte par défaut et du modèle de base sont copiés.
  1. Cliquez sur OK.
  2. Une fois le modèle créé, utilisez la vue Détails de l'utilisateur pour modifier le profil de l'utilisateur.

Certains champs du profil utilisateur ne s'appliquent pas aux modèles d'utilisateurs et vous ne pourrez pas les modifier.
