Modification du type d'un utilisateur

Utilisez la vue Détails de l'utilisateur pour convertir le type d'un utilisateur en un autre type. Par exemple, vous pouvez convertir un utilisateur Web uniquement en utilisateur simultané. Vous pouvez également désactiver un utilisateur ou réactiver un utilisateur désactivé.

Une licence doit être disponible pour le nouveau type d'utilisateur, sauf lorsque vous désactivez un utilisateur. La désactivation d'un utilisateur libère une instance de la licence précédemment associée à cet utilisateur.

Règles

Avant de convertir un utilisateur en utilisateur désactivé, procédez comme suit :

  • Si l'utilisateur détient des enregistrements, effectuez un réalignement de territoire (dans l'Administrateur).
  • Si l'utilisateur est responsable de compte, effectuez un réalignement de territoire (dans l'Administrateur) pour désigner un nouveau responsable de compte.
  • Si l'utilisateur est un responsable, réattribuez tout utilisateur dont il dépend à un nouveau responsable ou à l'administrateur.

Avant de modifier un utilisateur en utilisateur Web uniquement, procédez comme suit :

  • Si l'utilisateur est un responsable, supprimez ce paramètre dans la vue Détails de l'utilisateur.
  • Si d'autres utilisateurs sont répertoriés comme membre de l'équipe d'utilisateurs de cet utilisateur, supprimez-les.

Pour modifier

  1. Ouvrez la vue Détails de l'utilisateur de l'utilisateur que vous souhaitez convertir.
  2. Dans la zone Type d'utilisateur, sélectionnez un nouveau type d'utilisateur.
  3. Cliquez sur Enregistrer..
