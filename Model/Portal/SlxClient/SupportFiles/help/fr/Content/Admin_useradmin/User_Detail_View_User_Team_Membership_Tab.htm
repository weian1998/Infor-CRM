Onglet Appartenance à une équipe d'utilisateurs

Grâce aux équipes d'utilisateurs, les utilisateurs accèdent aux comptes détenus par un autre utilisateur qu'eux-mêmes. Par exemple, si vous voulez que Laurent ait accès à tous les comptes de Daniel, vous pouvez ajouter Laurent à l'équipe d'utilisateurs de Daniel.

L'utilisateur dont vous modifiez le profil aura accès aux comptes détenus par les utilisateurs répertoriés sur cet onglet.

Procédure

Ouvrir l'onglet Appartenance à une équipe d'utilisateurs
  • Dans la vue Détails de l'utilisateur, cliquez sur l'onglet Appartenance à une équipe d'utilisateurs. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Ajouter l'utilisateur à une équipe
  1. Pour ajouter un utilisateur, cliquez sur Ajouter cet utilisateur à une autre équipe d'utilisateurs.
  2. Utilisez la fonction Rechercher pour localiser et sélectionner l'utilisateur que vous souhaitez ajouter.

  3. Cliquez sur OK.

Modifier le profil de sécurité de l'utilisateur principal dans l'équipe d'utilisateurs sélectionnée
  1. Cliquez sur le lien Profil de sécurité et définissez les autorisations dans la boîte de dialogue.
  2. Cliquez sur OK.
    Ce profil de sécurité ne s'applique que lorsque l'utilisateur principal (l'utilisateur dont vous modifiez le profil) accède aux comptes détenus par l'utilisateur sélectionné. Cela ne s'applique pas lorsque l'utilisateur sélectionné accède à des comptes dont il est le propriétaire.
Supprimer l'utilisateur d'une équipe
  1. Sélectionnez l'enregistrement à retirer, puis cliquez sur le bouton Supprimer.
  2. Cliquez sur OK.
