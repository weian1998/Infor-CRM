Copie des informations d'un profil

Vous pouvez copier les informations de configuration d'un utilisateur ou d'un modèle existant vers un ou plusieurs utilisateurs. La copie des informations de profil d'un utilisateur vers un autre écrase les paramètres existants. En savoir plus...

Lorsque vous copiez les informations de l'agenda d'un utilisateur Web vers un autre, l'utilisateur vers lequel vous copiez ces informations peut avoir des restrictions inattendues par rapport à son propre agenda. Par exemple, lorsque l'agenda de l'utilisateur A est copié à partir de l'utilisateur B, la sécurité d'agenda que l'utilisateur B a dans son enregistrement de sécurité d'agenda pour l'utilisateur A est copié dans la sécurité de l'utilisateur A. Par conséquent, les droits d'accès de l'utilisateur A seront limités si l'utilisateur B avait des restrictions appliquées à l'agenda de l'utilisateur A. Pour éviter tout problème, ne copiez pas les informations d'agenda. Définissez l'accès aux agendas sur les onglets Autres agendas et Agenda de l'utilisateur après avoir copié les informations de profil.

Pour copier

  1. Ouvrez la vue Détails de l'utilisateur de l'utilisateur vers le profil duquel vous souhaitez copier des informations.
  2. Dans le volet Tâches courantes, cliquez sur Copier le profil.
  3. Dans la boîte de dialogue Copier le profil, choisissez de copier le profil à partir d'un Utilisateur existant ou d'un Modèle.
  4. Sélectionnez l'Utilisateur ou le Modèle à partir duquel vous souhaitez copier le profil.
  5. Sélectionnez les onglets à copier. En savoir plus...
  • Général - les paramètres généraux et les paramètres du service sont copiés. les champs NomUtilisateur, Nom, Fonction et E-mail ne s'appliquent pas lorsque vous copiez des informations.
  • Employé - les champs Fax, Portable, Pager, Direct, N° ID, Comptabilité et tous les champs de données personnelles ne s'appliquent pas lorsque vous copiez des informations.
  • Sécurité - tous les paramètres de sécurité sont copiés.
  • Equipes -tous les paramètres d'équipes sont copiés.
  • Agenda - tous les paramètres d'agenda sont copiés.
  • Options client (système client) - les paramètres du propriétaire du compte par défaut et du modèle de base sont copiés.
  • Service/Assistance technique - tous les paramètres service et support technique sont copiés.
  1. Dans la section Utilisateur cible, vérifiez qu'il s'agit de l'utilisateur dont vous souhaitez écraser le profil.
  2. Cliquez sur OK.
  3. Pour confirmer, cliquez sur OK.
