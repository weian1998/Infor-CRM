Zones d'information sur l'utilisateur

Le tableau suivant décrit les zones d'informations par défaut.

Zone Description/Action
Connexion activée Détermine si l'utilisateur peut se connecter à Infor CRM. Désélectionnez cette option si l'utilisateur devient inactif et qu'il ne doit pas accéder à Infor CRM.
Nom d'utilisateur

Le nom dont l'utilisateur se servira pour se connecter à Infor CRM.

Le nom d'utilisateur est impératif et ne doit pas contenir d'espace ni d'apostrophe.
Nom Le nom de l'utilisateur. Cliquez sur le bouton Modifierpour ouvrir la boîte de dialogue Modifier le nom.
Title La fonction de l'utilisateur.
E-mail L'adresse e-mail de l'utilisateur.
Type d'utilisateur Le type d'utilisateur de Infor CRM. Pour modifier le type d'utilisateur, sélectionnez un nouveau type dans la liste. Si vous passez en utilisateur WebViewer, vous devez réinitialiser IIS.
Modifier le mot de passe Modifie le mot de passe de l'utilisateur.
Mot de passe mis à jour Affiche la date et l'heure de la dernière modification du mot de passe.
Modèle utilisé Le dernier modèle d'utilisateur appliqué à cet utilisateur.
Région L'emplacement de l'utilisateur.
Division La division de l'utilisateur.
Gestionnaire Le responsable de l'utilisateur. Seuls les responsables pouvant être à juste titre responsables de cet utilisateur s’affichent dans la liste. Les responsables sont automatiquement ajoutés à la liste, sauf s'ils sont inactifs.
L'utilisateur est un responsable

Détermine si l'utilisateur est un responsable et si son nom doit apparaître dans la liste des responsables. Cette option n'est pas disponible pour l'utilisateur administrateur.
