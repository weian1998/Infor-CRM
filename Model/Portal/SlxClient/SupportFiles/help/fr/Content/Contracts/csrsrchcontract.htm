Sélection d'un contrat

Vous pouvez utiliser la vue Sélectionner un contrat lorsque le compte dispose de plusieurs contrats de service et que vous devez sélectionner le contrat sur lequel facturer le ticket.

Vos droits d'accès de sécurité déterminent les fonctionnalités disponibles. Contactez votre administrateur pour toute modification des droits d'accès. Les utilisateurs WebViewer risquent de ne pas pouvoir utiliser l'ensemble des fonctionnalités.

Pour sélectionner un contrat

  1. Dans la vue Détails du Ticket, cliquez sur le bouton Rechercher de Contrat.
  2. Utilisez la vue Rechercher un contrat pour trouver un contrat, sélectionnez le contrat dans la liste, puis cliquez sur OK.

Les tickets et les activités de ticket seront enregistrés sur le contrat que vous aurez sélectionné.
