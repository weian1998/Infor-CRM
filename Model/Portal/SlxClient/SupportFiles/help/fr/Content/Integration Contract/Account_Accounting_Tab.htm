Onglet Comptabilité

 (Intégration de la comptabilité uniquement) L'onglet Comptabilité de la vue Détails du compte affiche les informations de lien pour tous les systèmes de comptabilité actuellement associés à Infor CRM. Vous pouvez utiliser cet onglet pour gérer les liens entre les systèmes. Seuls les systèmes de comptabilité configurés dans la vue Détails d'intégration apparaissent dans la liste. Cette vue contient les colonnes suivantes :

  • Système de comptabilité : le système de comptabilité auquel est associé le compte.
  • Etat : l'état du lien.
  • Partagé par : le nom du système de comptabilité qui partage les informations de compte d'exploitation.
  • Promu par : l'utilisateur qui a créé le lien.
  • Date de promotion : la date à laquelle le compte a été associé.

Procédure

Ouvrez l'onglet Comptabilité.

  1. Ouvrez la vue Détails du compte.
  2. Cliquez sur l'onglet Comptabilité. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Comprendre l'état du lien

La colonne État affiche l'état du lien entre le compte Infor CRM et le système de comptabilité.

  • Associé - Le compte est associé à un compte d'exploitation dans le système de comptabilité.
  • Partagé - Le système de comptabilité partage les informations du compte d'exploitation avec un système de comptabilité associé différent. Si l'état d'un système de comptabilité est Partagé, le système ne peut pas être associé. Vous ne pouvez associer que le compte parent des systèmes partagés. Par exemple, le système de comptabilité peut être une filiale d'un autre système de comptabilité associé et tous deux partagent le même compte d'exploitation. Les informations du compte partagé s'affichent dans la colonne Partagé par.
