Volet des tâches Comptabilité

Cette fonctionnalité est uniquement disponible si votre administrateur a configuré l'intégration de la comptabilité.

Utilisez le volet des Tâches Comptabilité pour accomplir votre travail. Le volet Tâches de comptabilité apparaît dans les vues Liste et détails des comptes. Les options disponibles dépendent du type de vue dans lequel vous travaillez.

À partir de la vue Liste de comptes, vous pouvez :

  • Promouvoir un compte

À partir de la vue Détails de comptes, vous pouvez :

  • Promouvoir un compte
  • Promouvoir vers un autre Système de comptabilité

Le volet des tâches affiche également l'état des liens :

  • Associé - Le compte est associé à un compte d'exploitation dans le système de comptabilité.
  • Dissocié : un système de comptabilité est disponible pour l'association mais le compte Infor CRM n'est pas associé au système de comptabilité.
