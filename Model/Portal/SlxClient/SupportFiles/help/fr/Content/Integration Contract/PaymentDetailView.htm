Vue Détails de paiement

 (Intégration de la comptabilité uniquement) La vue Détails de paiement affiche des informations en lecture seule provenant de votre système de comptabilité.

Pour afficher

  1. Ouvrez la vue Détails du compte et cliquez sur l'onglet Paiements des clients. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur le numéro du reçu que vous souhaitez afficher. Les informations suivantes s'affichent :
    3. Zone Description
    Numéro de reçu Le numéro de reçu.
    Date de paiement La date du reçu.
    Date de traitement La date à laquelle le reçu a été traité.
    Nom Le nom du reçu.
    Type Type du reçu. Par exemple, pré-paiement, acompte, etc.
    Statut L'état du reçu. Par exemple, réglé, en attente, autorisé, etc.
    Devise La devise du reçu.
    Type de réglement La méthode de réglement utilisée pour le reçu. Par exemple, liquide, chèque, carte de crédit, etc.
    Référence du réglement La référence du reçu associée avec le type de réglement. Par exemple, un numéro de chèque, de carte de crédit, etc.
    Facture La référence à un autre document si le reçu est pour ce document. Par exemple : une facture.
    Total net La valeur totale de la facture, incluant les remises, taxes exclues.
    Remises Toute remise pouvant être appliquée à la facture.
    Charges Le montant total des frais appliqués à la ligne de facture (par exemple, les charges financières ou les intérêts).
    Taxes Les taxes qui s'appliquent à la ligne de facture.
    Le montant du paiement La valeur totale du reçu, incluant les taxes, les remises et les frais.
    Source La source du reçu. Par exemple, chèque, liquide, prélèvement automatique, etc.
    Compte Le compte (client) depuis lequel le reçu a été obtenu.
    Code de taxe Le code de taxe du reçu.
  3. Cliquez sur Fermer.
