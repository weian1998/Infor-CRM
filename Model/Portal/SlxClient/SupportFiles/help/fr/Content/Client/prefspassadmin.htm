Modification d'un mot de passe (Administrateur)

Vous pouvez modifier le mot de passe d'un utilisateur dans la vue Utilisateurs et via les Options utilisateur.

En savoir plus...

Les nouveaux utilisateurs sont créés avec le mot de passe par défaut « password ». L'administrateur système, l'utilisateur disposant de droits d'administration ou un utilisateur de Infor CRM peut modifier ce mot de passe à n'importe quel moment.
  • Les mots de passe doivent commencer et se terminer par un chiffre (0-9) ou une lettre (A-Z ou a-z).
  • Le point-virgule n'est pas valide.

Pour modifier un mot de passe dans Options utilisateur

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet Mot de passe, puis sélectionnez l'utilisateur souhaité dans la liste fournie.
  3. Entrez le nouveau mot de passe.
  4. Saisissez une nouvelle fois ce mot de passe dans la zone Confirmer le mot de passe.
  5. Cliquez sur Enregistrer.. Un message s'affiche vous indiquant que le mot de passe a bien été modifié.

    		6.

    Vous devez vous déconnecter du Client Web Infor CRM, puis vous reconnecter avant que les modifications apportées aux options prennent effet.

Pour modifier un mot de passe dans la vue Utilisateurs

  1. Dans la barre de navigation, cliquez sur Administration.
  2. Cliquez sur Utilisateurs puis sur le nom de l'utilisateur concerné.
  3. Dans la vue Détails de l'utilisateur, cliquez sur le bouton Modifier le mot de passe.

  2. Dans la zone Utilisateur, sélectionnez l'utilisateur dont vous souhaitez modifier le mot de passe.

  1. Saisissez le nouveau mot de passe dans les zones Nouveau mot de passe et Confirmer le mot de passe.
  2. Cliquez sur OK.
