Modification d'un mot de passe

Vous pouvez modifier votre mot de passe via les options dans le Client Web Infor CRM.

Pour modifier votre mot de passe

  1. Dans le menu Outils, cliquez sur Options.
  2. Cliquez sur l'onglet Mot de passe, saisissez votre mot de passe dans la zone Mot de passe actuel.
  3. Dans la zone Nouveau mot de passe, entrez votre nouveau mot de passe.
  4. Saisissez une nouvelle fois ce mot de passe dans la zone Confirmer le mot de passe.
  5. Cliquez sur Enregistrer.
