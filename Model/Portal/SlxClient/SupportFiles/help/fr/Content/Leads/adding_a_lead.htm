Ajout d'un lead

La boîte de dialogue Insérer un lead permet d'ajouter un nouveau lead dans le Client Web Infor CRM.

Pour ajouter

  1. Dans le menu Nouveau, cliquez sur Lead. 
  2. Cliquez sur le bouton Nom Modifier. 
    La boîte de dialogue Modifier le nom s'ouvre.
  3. Renseignez les autres zones d'informations sur le lead.
  4. Cliquez sur Rechercher des enregistrements correspondants pour rechercher dans la base de données les leads, les contacts et les comptes dont les informations correspondent à celles que vous avez indiquées.
      
    1. Si aucune correspondance potentielle n'est trouvée, cliquez sur OK pour fermer la boîte de dialogue et poursuivre la saisie d'informations sur le lead.
    2. Si des correspondances sont trouvées, la boîte de dialogue Recherche des doublons s'ouvre.
    3. Si vous souhaitez que le système recherche des correspondances potentielles sans devoir cliquer sur le bouton Rechercher des enregistrements correspondants, cochez la case Rechercher automatiquement une fois le numéro de téléphone saisi.
      4. La boîte de dialogue Leads correspondants s'affiche. Si un enregistrement correspondant s'affiche dans cette boîte de dialogue, vous pouvez l'ouvrir afin de voir si le lead, le contact ou le compte est le même que le lead que vous entrez.
  5. Effectuez l’une des opérations suivantes :
    • Pour enregistrer le lead et fermer la vue, cliquez sur Enregistrer.
    • Pour enregistrer le lead puis en entrer un nouveau, cliquez sur Enregistrer & Nouveau.
