Révision des sélections d'importation

Utilisez cette vue pour revoir vos sélections avant d'importer des leads dans Infor CRM.

Pour sélectionner

  1. Lors de l'importation de leads, après avoir sélectionné des actions d'importation de groupe, cliquez sur Suivant.
  2. Revoyez les points suivants :
    • Fichier de liste de leads : le fichier contenant les leads à importer.
    • Propriétaire par défaut : le propriétaire de tous les leads importés.
    • Source des leads par défaut : la source attribuée à tous les leads importés.
    • Ajouter au groupe de leads : indique si tous les leads importés seront ajoutés à un groupe ou non.
    • Nom du groupe : tous les leads importés seront ajoutés à ce groupe.
    • Recherche de doublons : indique si l'importation va inclure une recherche des doublons dans Infor CRM.
    • Actions de groupe : liste des actions d'importation à effectuer pour les leads importés.
  3. Cliquez sur Envoyer pour traiter la demande.
