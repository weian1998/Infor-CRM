Utilisation des options de correspondance avancée

Utilisez les options de correspondance avancée pour définir comment trouver les correspondances potentielles.

Pour définir les options avancées

  1. Utilisez les cases Définition de doublons par score pour déterminer le type de correspondance. Cette option n'est disponible que dans l'étape Gestion des doublons lors de l'importation de leads.
  1. Dans la zone Doublon, entrez le pourcentage minimum qui déterminera si un mot est un doublon.
  2. Dans la zone Doublon possible, entrez le pourcentage minimum qui déterminera si un mot est un doublon potentiel.
  1. Sélectionnez Utiliser la lemmatisation pour inclure d'autres formes d'un mot.
  2. Sélectionnez Utiliser la phonétique pour inclure des mots ayant une consonance similaire.
  3. Sélectionnez Utiliser la recherche synonymique pour inclure des mots ayant un sens similaire.
  4. Sélectionnez Recherche par correspondance approximative pour inclure des mots similaires ou mal orthographiés. Sélectionnez ensuite le Niveau d'approximation pour déterminer à quel point l'orthographe peut être différente pour qu'un mot soit considéré comme une correspondance. Chaque chiffre représente le nombre de lettres pouvant être différentes.

Pour être une correspondance approximative :

  • Chaque correspondance doit commencer par la première lettre du critère de correspondance.
  • Après la première lettre, chaque correspondance ne doit pas comporter plus de différences de caractères que le niveau d'approximation n'autorise. Par exemple, un niveau d'approximation de 3 donnera tous les noms avec 3 caractères différents au plus.
    Exemple

    Si « Jon » est votre critère de recherche, un niveau d'approximation de 1 trouverait que Jon et John sont une correspondance approximative, mais pas Jon et Johnn. Pour que Johnny soit une correspondance, vous devez augmenter le niveau d'approximation à 3. Toutefois, vous obtiendrez alors des noms tels que Jack ou Jeff.
