Historique d'importation des leads - Onglet Historique

Utilisez cet onglet pour afficher une liste des enregistrements qui ont rencontré des problèmes lors de l'importation. Les enregistrements apparaissent dans l'onglet dans les cas suivants :

  • Erreur : l'enregistrement a rencontré une erreur et n'a pas été importé.
  • Doublon : l'enregistrement a été considéré comme un doublon devant être résolu avant de pouvoir être importé.
  • Fusionné : l'enregistrement a été importé et fusionné avec un autre enregistrement.

Pour ouvrir

  • Dans la vue Détails de l'historique d'importation des leads, cliquez sur l'onglet Historique. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.

Cette vue contient les colonnes suivantes :

  • Horodatage : date et heure auxquelles le problème est survenu.
  • Type : le problème qui est survenu.
  • Description : description de l'enregistrement, du problème rencontré et de toute action entreprise.
  • Résolu : indique si le doublon a été résolu.
  • Date de résolution : date à laquelle le doublon a été résolu.
  • Description de la résolution : description de la façon dont le doublon a été résolu. Par exemple, Fusionné avec.
