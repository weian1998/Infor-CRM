Définition d'un séparateur

Utilisez cette vue pour sélectionner le séparateur approprié pour votre liste et obtenir un aperçu des enregistrements avant de les importer dans Infor CRM. Un séparateur permet de fractionner les champs d'une liste. Infor CRM analyse la liste pour déterminer quel est le séparateur utilisé dans celle-ci, mais vous pouvez sélectionner un autre séparateur si nécessaire.

Pour définir un séparateur et obtenir un aperçu

  1. Lors de l'importation de leads, après avoir sélectionné un fichier, cliquez sur Suivant.
  2. Si vous devez modifier le séparateur, cliquez sur une des options suivantes :
    • Tabulations
    • Point-virgule
    • Virgule
    • Espace
    • Autre
  3. Si vous avez sélectionné Autres, vous devez préciser le séparateur utilisé pour séparer les champs de la liste. Saisissez le caractère de séparation dans la zone Autre.
  4. Dans la liste Identificateur de texte, cliquez sur la flèche du menu déroulant, puis sélectionnez un identificateur dans la liste.
    Un identificateur de texte identifie le texte qui doit être importé exactement tel qu'il apparaît.
  5. Si la liste contient des noms de champ, cochez la case La première ligne contient les noms de champs pour afficher ces noms dans la zone Aperçu.

    Si des caractères étendus sont présents dans le fichier importé et qu'ils ne s'affichent pas correctement dans la zone Aperçu, vous devez enregistrer à nouveau le fichier importé et définir de manière explicite le type de codage par exemple UTF-8, Unicode, ou ANSI. Une fois le fichier enregistré, cliquez sur le bouton Précédent pour revenir à l'étapeSélectionner un fichier, et sélectionnez de nouveau le fichier mis à jour avant de poursuivre l'importation.

  6. Cliquez sur Suivant pour passer à l'étape suivante.
