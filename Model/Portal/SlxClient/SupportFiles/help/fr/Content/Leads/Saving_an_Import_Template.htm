Enregistrer un modèle d'importation

Vous pouvez enregistrer des informations d'importation, comme les champs correspondants, le propriétaire par défaut du lead et la source par défaut du lead pour les utiliser lors d'importations ultérieures.

Pour enregistrer un modèle d'importation

  1. Après la définition des champs du mappage d’importation, cliquer sur Enregistrer en tant que modèle d'importation.
    La boîte de dialogue Modèle d'importation de leads s'affiche.
  2. Dans la zone Description, saisissez une brève description du modèle d'importation, puis cliquez sur Enregistrer en tant que modèle d'importation.
    La description du nouveau modèle d'importation est ajoutée à la liste des modèles d'importation affichée dans la boîte de dialogue Importer des leads.
