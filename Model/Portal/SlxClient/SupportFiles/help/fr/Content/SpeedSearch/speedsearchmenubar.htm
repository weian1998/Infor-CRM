Utilisation de SpeedSearch depuis la barre de menus

Vous pouvez désormais saisir vos critères de recherche et effectuer une recherche SpeedSearch depuis la barre de menus.

Pour effectuer une recherche SpeedSearch depuis la barre de menus

  • Dans la barre de menu, saisissez vos critères de recherche dans la zone SpeedSearch puis cliquez sur le bouton SpeedSearch.
     SpeedSearch s'ouvre avec les résultats de votre recherche ou vous permet de sélectionner les options avancées.
