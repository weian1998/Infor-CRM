SpeedSearch

SpeedSearch est un outil puissant qui permet aux utilisateurs de rechercher des informations stockées dans la base de données Infor CRM ou dans des fichiers externes. Par exemple, vous pouvez rechercher parmi des tickets ou des procédures existantes pour aider un client à résoudre son problème ou rechercher un document spécifique, tel qu'un argumentaire de vente. Le type d'informations que vous pouvez rechercher dépend des index que votre administrateur système a définis dans le module Administrateur.

Pour réaliser une recherche SpeedSearch

  • Effectuez l’une des opérations suivantes :

    • Entrez les informations relatives à la recherche dans la barre de menus.
    • Cliquez sur le bouton SpeedSearch dans l'onglet Détails du ticket ou la vue Insérer un ticket.
    • Dans le menu Outils, cliquez sur Advanced SpeedSearch.

Si vous rencontrez des problèmes pour ouvrir ou utiliser SpeedSearch, contactez votre administrateur Infor CRM. Les informations de configuration et de dépannage de SpeedSearch figure dans l'aide du module Administrateur.
