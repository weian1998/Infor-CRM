Ajout d'un élément d'historique

Les éléments d'historique correspondent aux activités effectuées, e-mails envoyés, questions posées et modifications insérées dans la base de données.

Pour ajouter

  1. Dans la vue Détails du contact, du compte, du lead ou de l'opportunité, cliquez sur Onglet Notes/Historique, puis sur le bouton Terminer l'activité.
  2. Choisissez de terminer une activité planifiée ou une activité non planifiée.
  3. Suivez les étapes pour terminer une activité.
    Si vous avez planifié une activité de suivi, la boîte de dialogue Planifier une activité s'ouvre.
