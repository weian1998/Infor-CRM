Onglet Notes/Historique 

L'onglet Notes/Historique affiche les vues Détails du compte, du contact, de l'opportunité et du Lead. L'onglet contient des options de filtrage qui vous permettent d'accéder rapidement aux informations importantes. Cet onglet vous permet également d'envoyer par e-mail un enregistrement d'historique, de créer un document Microsoft Word, d'afficher une pièce jointe ou de modifier des notes.

L'onglet Notes/Historique contient toutes les interactions, y compris les notes intégrales et proposent des options de filtrage permettant d'afficher, d'envoyer par e-mail et d'imprimer les éléments d'historique et les notes.

 Si l'option Afficher l'historique pour tous les participants n'est pas sélectionnée, vous verrez un élément de l'historique uniquement pour le participant principal. Pour plus d'informations, consultez la rubrique « Définition des options des activités ».

Procédure

Ouvrez l'onglet Notes/Historique
  • Dans la vue Détails du contact, Détails du compte, Détails de l'opportunité ou Détails du lead, cliquez sur l'onglet Notes/Historique. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Application de filtres
  1. Cliquez sur Filtre pour afficher les options du filtre.

Vous pouvez sélectionner plusieurs options de filtrage. Vous pouvez, par exemple, sélectionner les options Mon historique et Période du afin d'afficher tous vos éléments d'historique pour une période déterminée.

  1. Sélectionnez un ou plusieurs des filtres suivants :
    • Pour filtrer la liste par type d'enregistrement, dans la liste Type, sélectionnez le type d'enregistrements que vous souhaitez afficher.
    • Les modifications de la base de données sont masquées par défaut. Pour afficher les modifications de la base de données, sélectionnez l'option Afficher les modifications de la base de données.
    • Utilisez les zones Plage de dates pour sélectionner une plage de dates pendant laquelle la pièce jointe a été mise à jour pour la dernière fois.
    • Cliquez sur Utilisateur la flèche du menu déroulant et effectuez votre sélection dans la liste.
    • Dans la zone Objet, saisissez un ou plusieurs caractères contenus dans les informations sur l'objet. Plus vous saisirez de caractères, plus la liste renvoyée sera courte.
  2. Cliquez sur Appliquer.
  3. Pour effacer les filtres, cliquez sur Réinitialiser.
    Cliquez de nouveau sur le bouton Filtrer pour masquer les filtres.
Aperçu de la note ou des détails de l'historique
  • Lorsque vous sélectionnez un enregistrement dans la liste, les détails de celui-ci s'affichent dans le volet aperçu, à droite de la liste.
Ouvrez une note ou un enregistrement de l'historique
  • Cliquez sur le lien Type de l'élément que vous souhaitez afficher.
Envoi d'enregistrements d'historique par e-mail
  1. Sélectionnez le ou les enregistrement(s) à envoyer par e-mail.
  2. Cliquez sur Envoyer par e-mail.
    Une boîte d'envoi de message e-mail s'ouvre, dans laquelle figurent les informations sur le contact; l'objet et les informations sur les enregistrements d'historique. Si vous sélectionnez plusieurs enregistrements, les informations relatives à chacun d'eux sont indiquées.
  3. Cliquez sur Envoyer.
Envoi d'éléments d'historique vers Microsoft Word
  1. Sélectionnez le ou les enregistrements à copier dans un document Word.
  2. Cliquez sur Envoyer à Word.
    Microsoft Word s'ouvre et affiche un document contenant les informations sur la note ou l'enregistrement de l'historique.
  3. Vous pouvez imprimer le document ou enregistrer le fichier.
Ajouter une note
  • Dans l'onglet Notes/Historique, cliquez sur Ajouter une note effectuez toutes les étapes pour ajouter une note.
