Suppression d'une note

Vous pouvez supprimer uniquement les notes que vous avez créées. Vous devez supprimer la totalité de l'élément d'historique associé à la note.

Pour supprimer une note

  1. Ouvrez la vue Détails du Contact, du Lead, du Compte,du Lead ou de l'Opportunité avec la note associée.
  2. Cliquez sur l'onglet Notes/Historique puis cliquez sur le lien Note que vous souhaitez ouvrir.Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  3. Dans la vue Détails de la note, cliquez sur le bouton Supprimer.
  4. Cliquez sur OK pour supprimer l'enregistrement de façon définitive ou sur Annuler pour annuler cette suppression.
