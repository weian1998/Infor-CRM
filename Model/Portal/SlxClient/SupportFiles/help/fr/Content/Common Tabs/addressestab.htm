Onglet Adresses

L'onglet Adresses est situé dans les vues Détails du compte et Détails du contact. Cet onglet affiche toutes les adresses du contact ou du compte sélectionné. Seule l'adresse principale est affichée dans la vue détaillée.

Les colonnes de l'onglet sont : Principale, Facturation principale, Expédition principale, Description, A l'attention de, Adresse, Ville, Département, Code postal et Pays.

Pour consulter ou modifier les détails de l'adresse

  1. Dans les vues Détails du contact ou Détails du compte, cliquez sur Adresses .Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur Modifier.

Pour ajouter une nouvelle adresse

  1. Cliquez sur le bouton Ajouter une adresse pour ajouter une nouvelle adresse.
  2. Suivez les étapes d'ajout des informations sur l'adresse.
