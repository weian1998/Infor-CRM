Suppression d'un élément d'historique

Vous pouvez supprimer uniquement les éléments d'historique que vous avez créés.

Pour supprimer

  1. Ouvrez les vues Détails du contact, du compte, du lead ou de l'opportunité avec l'élément d'historique associé, puis cliquez sur Onglet Notes/Historique. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Cliquez sur le lien Type de l'élément d'historique que vous souhaitez supprimer.
  3. Cliquez sur le bouton Supprimer.
  4. Cliquez sur OK pour supprimer l'enregistrement de façon définitive ou sur Annuler pour annuler cette suppression.
