Affichage d'un plan pour une adresse

Cette fonction n'est disponible que pour la première ligne de l'adresse principale d'un compte, d'un contact ou d'un lead. La deuxième ligne d'adresse n'est pas utilisée. Si l'adresse comporte un signe dièse (#), placez celui-ci dans la deuxième ligne d'adresse pour permettre à MapQuest de mieux localiser l'adresse.

MapQuest risque de ne pas pouvoir localiser toutes les adresses ou vous devrez éventuellement effectuer une recherche plus précise sur les résultats obtenus afin d'avoir l'emplacement exact d'une adresse donnée.

Pour afficher le plan d'accès d'une adresse

  • Dans la vue Détails du Contact, du Compte ou du Lead, cliquez sur le bouton MapQuest.

Une autre page de navigateur s'ouvre. Le site MapQuest affiche un plan de la zone dans laquelle se trouve l'adresse indiquée. L'adresse est saisie de façon automatique.
