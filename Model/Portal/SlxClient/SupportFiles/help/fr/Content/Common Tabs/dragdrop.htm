Glisser-déposer

Utilisez la fonction glisser-déposer pour réorganiser des parties de l'interface Infor CRM. Si vous avez installé l'intégration du bureau, vous pouvez également ajouter des pièces jointes ou des messages e-mail Microsoft Outlook à Infor CRM.

Procédure

Réorganiser les éléments dans l'interface
  1. Positionnez votre souris sur l'élément à déplacer. Si l'onglet peut être déplacé, le curseur se transforme.
  2. Cliquez sur l'élément et, tout en maintenant le bouton de la souris enfoncé, faites-le glisser vers l'emplacement souhaité.

Par exemple, ailleurs dans une grille, sur la barre des onglets, dans la zone Autres onglets ou dans la section utilisateur dans la partie centrale.

  1. Relâchez le bouton de la souris.
Joindre un fichier à l'aide du copier-coller

  • Si vous utilisez Internet Explorer, cette fonctionnalité nécessite l'installation du module d'intégration du bureau.
  • L'outil glisser-déposer n'est pas disponible sur les onglets Pièces jointes de l'historique ou de l'activité.
  1. Naviguez jusqu'au fichier que vous souhaitez ajouter.
  2. Faites glisser le fichier vers la vue Détails de l'enregistrement approprié.

La boîte de dialogue Ajouter des pièces jointes s'ouvre.

    Si vous êtes invité à installer l'intégration du bureau Infor CRM et que vous souhaitez utiliser cette fonction, cliquez sur Oui et suivez les étapes d'installation de l'intégration du bureau Infor CRM. Pour effectuer l'installation, vous serez invité à fermer toutes les fenêtres de navigateur et ainsi que Microsoft Outlook.
  1. Dans la zone Description, saisissez une description du fichier si la description par défaut n'est pas claire.

  2. Cliquez sur OK.

Enregistrer un message e-mail dans l'historique
  1. Ouvrez la vue des détails du contact, compte, lead, opportunité ou ticket approprié.
  2. Dans Microsoft Outlook, sélectionnez le message e-mail que vous souhaitez ajouter à Infor CRM.
  3. Cliquez sur le message et maintenez appuyé le bouton de la souris sur l'élément et faites-le glisser vers la vue des détails ouverte dans Infor CRM.
    En fonction de vos options, il peut vous être demandé de saisir des informations de l'historique.
