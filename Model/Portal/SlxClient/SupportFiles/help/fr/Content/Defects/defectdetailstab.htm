Onglet Détails du défaut

L'onglet Détails apparaît dans la vue Détails du défaut. Il contient des informations spécifiques sur le défaut.

Pour modifier l'onglet Détails

  1. Dans la vue Détails du défaut, cliquez sur l'onglet Détails.Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  2. Dans la zone Objet, saisissez une brève description du problème.
  3. Utilisez la zone Description pour saisir une description détaillée du problème.
  4. Utilisez la zone Résolution pour saisir les mesures prises pour résoudre le défaut.
  5. Cliquez sur Enregistrer..
