Utilisation de l'onglet Commentaires

Utilisez cet onglet pour toutes les notes internes associées à un défaut ou à un contrat.

Pour ajouter

  1. Effectuez l’une des opérations suivantes :

    • Ouvrez la vue Détails du défaut.
    • Ouvrez la vue Détails du contrat.
  2. Cliquez sur l'onglet Commentaires. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
  3. Saisissez les informations dans la zone Commentaires.
