Zones d'informations sur un défaut

Le tableau suivant décrit les zones d'informations par défaut.

Zone Description/Action

N° de Défaut

 Affiche automatiquement le prochain ID disponible lors de la création du défaut. Vous ne pouvez pas modifier l'ID.
Projet Le projet auquel est associé le défaut. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Version identifiée Sélectionnez la version du produit ou du projet dans laquelle le défaut a été identifié. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste
Version cible Sélectionnez la version dans laquelle le défaut sera probablement résolu. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Progression actuelle Sélectionnez l'option Progression pour le défaut, par exemple En attente ou Non identifié. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Corrigé dans la version Si le défaut a été corrigé, sélectionnez la version appropriée.
Zone/Catégorie/Problème

Ces trois champs décrivent brièvement la nature du problème signalé. Cliquez sur l'un des boutons Rechercher en regard de chaque zone, sélectionnez les éléments appropriés dans les listes puis cliquez sur OK.

Les éléments Catégorie affichés dépendent de l'option de Zone sélectionnée et les éléments Problème disponibles dépendent de l'option de Catégorie sélectionnée.

Objet

(Insérer une défaut uniquement)

 Saisissez une brève description du défaut.

Description

(Insérer une défaut uniquement)

 Saisissez la description détaillée du défaut.

Résolution

(Insérer une défaut uniquement)

 Sélectionnez les étapes suivies pour résoudre le défaut.
Statut Par défaut, lors de l'ajout d'un nouveau défaut, l'état est configuré sur Ouvert. Pour modifier, cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Source La source par laquelle les informations sur le défaut ont été reçues, comme un ticket ou un rappel. Pour modifier, cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Type Sélectionner le type de défaut Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Priorité Priorité détermine l'ordre dans lequel les défauts sont traités et les ressources attribuées pour la résolution des problèmes. Les défauts avec une haute priorité sont généralement les premiers traités. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Gravité Affecter un niveau de gravité au défaut en sélectionnant un élément dans la liste. Cliquez sur flèche déroulante vers le bas  et sélectionnez un élément dans la liste.
Frequency Sélectionnez une fréquence dans la liste de sélection.
Soumettre à SpeedSearch

Sélectionnez cette zone si vous souhaitez que les informations sur le défaut soient disponibles pour les recherches effectuées par les autres utilisateurs.

En fonction de la manière dont votre administrateur a configuré SpeedSearch, la sélection de cette option peut automatiquement rendre les informations disponibles pour les recherches ou nécessiter que les informations soient approuvées pour la recherche rapide par un autre utilisateur.
Approuvé pour SpeedSearch Sélectionnez cette zone pour approuver un défaut qui a été soumis à SpeedSearch. Cette option est disponible si votre administrateur vous a désigné comme une personne pouvant approuver des défauts pour SpeedSearch.
Accès public

Cette zone apparaît si votre administrateur vous a donné l'autorisation d'approuver des soumissions de SpeedSearch. Sélectionnez l'une des options suivantes :

  • 5-Interne : les défauts sont inclus dans les index Speed Search auxquels seuls les utilisateurs internes ont accès.
  • 1-Client - les défauts sont inclus dans les index de SpeedSearch auxquels les utilisateurs internes et les clients utilisant Portail client Infor CRM ont accès.
Affecté à Le système insère le nom d'utilisateur en fonction des paramètres d'affectation automatique pour l'élément Domaine sélectionné. Pour affecter un autre utilisateur, cliquez sur Bouton  Rechercher  , recherchez l'utilisateur concerné, puis cliquez sur OK.
Téléphone Le champ est renseigné avec le numéro de téléphone assigné à l'utilisateur.
Date de l'affectation La date et l'heure auxquelles le défaut a été affecté à l'utilisateur actuel. Pour modifier la date d'affectation, cliquez sur le bouton BoutonAgenda, sélectionnez la date puis cliquez sur OK.
Propriétaire Utilisateur ou équipe d'utilisateurs disposant de droits d'accès au défaut Utilisez la fonction Recherche pour sélectionner un utilisateur ou une équipe.
