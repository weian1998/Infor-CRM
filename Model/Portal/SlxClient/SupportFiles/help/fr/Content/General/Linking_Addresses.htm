Liaison d'adresses

Utilisez l'étape de liaison d'adresses pour déterminer quelles informations d'adresse seront incluses lors de la fusion d'enregistrements.

L'administrateur système et les utilisateurs dotés d'un rôle approprié peuvent accéder à ces fonctionnalités.

Pour lier une adresse pour des enregistrements fusionnés

  1. Lors de la fusion d'enregistrements, dans une vue Liste, cliquez sur Suivant.
  2. Remplissez l'écran Lier des adresses.
    ZoneDescription/action
    Adresses sources non liées issues de (Source)

    Affiche les adresses dans la source qui seront remplacées dans les enregistrements fusionnés, sauf si l'adresse est liée à une adresse cible.

    Cliquez sur Sélectionner pour sélectionner les détails de l'adresse.

    Adresses cibles non liées issues de (cible)

    Affiche les adresses dans la cible qui resteront associées aux enregistrements fusionnés, sauf si l'adresse est liée à une adresse cible.

    Cliquez sur Sélectionner pour sélectionner les détails de l'adresse.

    Bouton Lier

    Sélectionnez une adresse non liée parmi les Adresses sources non liées et les Adresses cibles non liées sur les listes,puis cliquez sur Lier pour créer une connexion.

    Adresses liées

    Affiche la liste des adresses liées.

    Cliquez sur Dissocier pour supprimer le lien proposé.

  3. Cliquez sur Suivant pour passer à l'étape suivante.
