Utilisation des Filtres du Volet des tâches

Utilisez les filtres du Volet des tâches pour limiter la liste des enregistrements s'affichant dans un groupe de vue Liste. Les filtres disponibles sont basés sur les colonnes de la vue Liste sélectionnée et sur les informations contenues dans ces colonnes. Les filtres sont enregistrés sur votre ordinateur jusqu'à la suppression du cache de votre navigateur.

À côté de chaque élément de filtre se trouve le nombre d'enregistrements qui répondent aux critères des filtres appliqués et le nombre total d'enregistrements dans le groupe qui correspondent à l'élément de filtre. Exemple Si (12/27) apparaît à côté d'un élément, alors il y a un total de 27 enregistrements dans le groupe qui correspondent à l'élément de filtre, mais seuls 12 de ces enregistrements répondent également aux critères des filtres appliqués.

Si la colonne correspondant à un filtre sélectionné est supprimée du groupe de la vue Liste, le filtre n'est pas appliqué à la liste des enregistrements.

Chaque filtre est composé des critères que vous choisissez. Lorsque vous appliquez un filtre, seuls les enregistrements répondant aux critères du filtre s'affichent. Par exemple :

Procédure

Appliquer un filtre
  1. A partir d'une vue Liste, dans le Volet des tâches, sous Filtres, cliquez sur un nom de filtre. Par exemple, cliquez sur Propriétaire.
  2. Sélectionnez les éléments de filtre que vous voulez appliquer à la liste d'enregistrements. Les éléments de filtre disponibles peuvent être personnalisés.
Effacer un filtre
  • Cliquez sur le filtre à effacer, puis cliquez sur Effacer.
Effacer un élément de filtre sélectionné
  1. Cliquez sur le filtre incluant l'élément de filtre que vous voulez effacer.
  2. Cliquez sur l'élément de filtre pour décocher la case.
Effacer tous les filtres
  • A partir d'une vue Liste, dans le Volet des tâches, cliquez sur Effacer tout.
    L'effacement de tous les filtres fera disparaître tous les filtres et affichera tous les enregistrements du groupe.
