Utilisation de la vue Détails de liste des groupes

La vue Détails de liste des groupes affiche la liste de tous les enregistrements d'un groupe ou d'une recherche dans la vue Détails. Vous pouvez pouvoir en un clin d'œil d'autres enregistrements présents dans le groupe ou la recherche, ou cliquez sur un enregistrement de la liste pour accéder à la vue Détails de l'enregistrement. Il existe deux modes différents pour la liste des groupes : le mode normal affiche uniquement le nom de l'enregistrement, le mode étendu affiche des informations plus détaillées dans des colonnes supplémentaires. Vous pouvez définir des options pour déterminer le comportement par défaut de la liste des groupes.

Pour afficher des détails supplémentaires

  • Cliquez sur le bouton Développer la liste des groupes.

Pour masquer les détails supplémentaires

  • Cliquez sur le bouton Réduire la liste des groupes.

Pour définir des options de liste de groupe

  1. Cliquez sur le bouton Options de liste de groupes étendues.
  2. Dans la zone Colonnes, saisissez le nombre de colonnes à afficher dans le mode de détails supplémentaires. Pour voir toutes les colonnes affichée dans la vue Liste, saisissez 0.
    Ces colonnes sont déterminées par les colonnes qui s'affichent dans la vue Liste.
  3. Sélectionnez ou désélectionnez l'option Masquer la liste de groupe étendue après sélection de l'enregistrement.
    • Sélectionnez cette option pour maquer la liste développée après avoir sélectionné un enregistrement dans la liste.
    • Désélectionnez cette option pour conserver l'état actuel de la liste Groupe après la sélection d'un enregistrement dans la liste.
  4. Sélectionnez ou désélectionnez l'option Rester dans la vue Détails après une recherche.
    • Sélectionnez cette option pour rester dans la vue Détails lorsque vous naviguez parmi les enregistrements dans Résultats de la recherche.
    • Désélectionnez cette option pour toujours afficher Résultats de la recherche dans la vue Liste.
  5. Sélectionnez ou désélectionnez l'option Afficher la liste Groupe développée après une recherche.
    • Sélectionnez cette option pour afficher la liste développée lorsque vous consultez la vue Détails des enregistrements dans Résultats de la recherche.
    • Désélectionnez cette option pour masquer automatiquement les détails de liste développée lors de la consultation de la vue Détails dans Résultats de la recherche.

  6. Cliquez sur OK.
