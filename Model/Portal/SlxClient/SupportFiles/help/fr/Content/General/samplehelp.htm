Rubrique d'aide modèle

Il s'agit d'un exemple de rubrique d'aide.

Dans cette rubrique, remplacez le texte selon les instructions à suivre pour utiliser votre vue personnalisée, puis enregistrez la rubrique en utilisant le nom conventionnel suggéré.

Pour obtenir plus d'informations concernant les ajouts de rubriques d'aides personnalisées, consultez la rubrique Présentation de l'aide Web dans l'aide de l'application Architect.

Conventions à suivre pour les noms

En créant un aide personnalisée pour les vues Liste, vous devez utiliser les conventions de noms comme indiqué ci-dessous, afin que l'aide puisse s'afficher correctement.

 

vuelisteentité.htm - Exemple : vuelistecompte.htm

Avec entité qui est le nom de l'entité associée à la vue Liste personnalisée

Conseils pour écrire une rubrique d'aide

Lors de la création d'une rubrique d'aide personnalisée pour une vue Liste, il convient d'inclure un paragraphe général ou d'introduction décrivant brièvement les types d'enregistrements qui apparaîtront sur la vue Liste. Vous pouvez aussi choisir d'inclure les étapes à suivre pour ouvrir la vue Liste, travailler avec des groupes, ou toute autre tâche pouvant être réalisée à partir de la vue Liste.
