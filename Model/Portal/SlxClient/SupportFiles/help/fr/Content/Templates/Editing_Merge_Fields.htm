Modification des champs de fusion

L'éditeur des champs de publipostage vous permet de modifier les champs de fusion, qu'ils soient nouveaux ou non. Vous pouvez afficher les champs de fusion du modèle dans la vue Liste et vos options de modification dans le volet de modification. Vous pouvez également supprimer des champs de fusion du modèle en cliquant sur le bouton Supprimer du modèle de la barre d'outils.

Pour modifier des champs de fusion

  1. Ouvrez l'Editeur de modèles.
  2. Cliquez sur Modifier les champs.
    La boîte de dialogue Editeur des champs de publipostage Infor CRM s'ouvre. Les champs de fusion du modèle sont affichés dans la vue Liste. La vue Liste comporte les colonnes suivantes :
    • Champ : affiche le nom du champ de fusion tel qu'il apparaît dans le modèle, ainsi que l'icône associée au type du champ.
    • Type : affiche le type du champ de fusion. Les types valides sont : Ordinaire, Signature MS Outlook, Table des produits de l'opportunité, SQL et Personnalisé.
    • Position : affiche la position numérique du champ tel qu'il apparaît dans le modèle.
  3. Mettez en surbrillance le champ de fusion à modifier dans la vue Liste.
    Les options de modification affichées varient en fonction du type de champ de fusion.
  • Modification d'un champ de publipostage standard
  • Identificateur de champ : permet de modifier le texte affiché dans le modèle. L'identificateur de champ d'un champ de publipostage standard comprend la table et le nom d'affichage correspondants. Vous ne pouvez modifier que le nom d'affichage.
  • Type de format : permet de modifier le type de formatage appliqué au champ de fusion sélectionné. Par exemple : fixe, nombre entier, pourcentage ou devise.
  • Chaîne de formatage : permet de formater la valeur affichée dans le champ de fusion. Avant d'ajouter une chaîne de formatage, vous devez définir le type de format. La validité d'un format de chaîne dépend du type de format sélectionné.
  • Texte avant : permet d'entrer le texte affiché avant le champ de fusion sélectionné.
  • Texte après : permet d'entrer le texte affiché après le champ de fusion sélectionné.
  • Format spécial : permet d'appliquer un formatage de caractère spécial au champ de fusion (y compris au texte affiché avant et après le champ de fusion).
  • Table : affiche la table associée au champ de la base de données. Si le champ de la base de données est le résultat d'une jointure, la légende pour cette option sera « Table principale ».
  • Champ : affiche le champ de la base de données. Si le champ de la base de données est le résultat d'une jointure, la légende pour cette option sera « Champ de clé ».
  • Calculé : indique si le champ est une valeur calculée.
  • Jointure : indique si le champ est le résultat d'une jointure de base de données.
  • Protéger le format des caractères : permet de conserver le formatage initial des caractères au cours d'une fusion.
