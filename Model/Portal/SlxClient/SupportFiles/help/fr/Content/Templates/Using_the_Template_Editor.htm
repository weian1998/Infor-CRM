Utilisation de l'éditeur de modèle

L'éditeur de modèle vous permet de créer un nouveau modèle de fax ou de modifier un modèle existant. L'éditeur contient une Infor CRMbarre d'outils une qui vous permet d'accéder aux fonctions spéciales de modèle ainsi qu'à de nombreuses fonctions Microsoft Word. Les fonctions disponibles dépendent du type du modèle modifié : Lettre, e-mail ou fax.

Afin de vous aider à identifier les boutons de barre d'outils, Infor CRM propose une info-bulle pour chaque outil. Pour afficher une info-bulle, placez le curseur sur l'outil.

Microsoft Word étant incorporé dans l'éditeur, les barres d'outils Word sont également affichées. Cliquez sur le bouton Afficher le menu Word pour afficher des barres d'outils Microsoft Word supplémentaires actuellement invisibles.

Pour ouvrir l'éditeur de modèle

  1. Dans le menu Rédiger, sélectionnez Modèles.

  2. Effectuez l’une des opérations suivantes :

    • Sélectionnez un modèle privé, puis cliquez sur le bouton Modifier.
    • Cliquez sur le bouton Nouveau etdéfinissez les propriétés du modèle, puis cliquez sur OK.

Pour ajouter des champs de fusion SLX à un modèle

  1. Dans le modèle, placez le curseur à l'endroit où vous voulez insérer le champ de fusion.
  2. Cliquez sur Insérer le champ SLX.
    Le menu s'ouvre avec les champs de fusion Infor CRM le plus souvent utilisés.
  3. Sélectionnez un nom de champ.
  4. Cliquez sur Autres champs SLX pour accéder à des champs de fusion Infor CRM supplémentaires.
  5. Pour prévisualiser le modèle fusionné, dans la barre d'outils cliquez sur Aperçu.
    Cette action fusionne les informations du contact en cours à l'intérieur du modèle.
    Vous ne pouvez pas modifier ou enregistrer un modèle dans la fenêtre Aperçu. Fermez la fenêtre d'aperçu lorsque vous avez terminé de visualiser celui-ci.
  6. Dans la barre d'outils, cliquez sur Enregistrer ou Enregistrer et fermer.
    Le modèle est enregistré dans la base de données Infor CRM. Si l'option Enregistrer et fermer est sélectionnée, l'éditeur se ferme. La boîte de dialogue Publipostage s'ouvre avec le nom du modèle actuel dans la zone Modèle.
  • Les champs de fusion Infor CRM ne peuvent pas être ajoutés au champ Objet d'un e-mail.
  • Les champs de fusion Infor CRM sont au format XML, qui est sensible à la casse. Si la casse est modifiée dans un champ de fusion, il ne sera pas fusionné.
  • Vous ne pouvez pas placer des champs de fusion Infor CRM dans une zone de texte lorsque vous créez ou modifiez un modèle.
  • Le modèle doit contenir un champ de fusion de la table Address pour pouvoir afficher la boîte de dialogue Sélectionner le type d'adresse durant un publipostage. Pour obtenir des informations supplémentaires, consultez la rubrique Sélection du type d'adresse.
