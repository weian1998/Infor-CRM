Sélection d'un utilisateur Infor CRM

Utilisez la boîte de dialogue Sélectionner un utilisateur dans les cas suivants :

  • vous créez ou copiez un modèle et devez sélectionner un propriétaire pour le modèle.
  • vous définissez des options d'historique pour publipostage et devez sélectionner un responsable pour une activité de suivi.

Procédure

Sélection d'un propriétaire pour un modèle
  1. Créez ou copiez un modèle.
  2. Dans la boîte de dialogue de propriétés du modèle, dans la zone Propriétaire, cliquez sur Rechercher.
  3. Dans la boîte de dialogue Sélectionner un utilisateur, effectuez l'une des opérations suivantes pour sélectionner un propriétaire :
    • Sélectionnez un utilisateur dans la liste et cliquez sur OK.
    • Double-cliquez sur un nom d'utilisateur.
Sélectionnez un responsable pour une activité de suivi
  1. Dans le menu Rédiger, cliquez sur Publipostage.
  2. Sélectionnez un modèle pour la fusion.
  3. Sous l'onglet Options d'historique, dans la zone Planifier une activité de suivi, complétez les informations pour planifier une activité de suivi.
  4. Dans la zone Responsable, cliquez sur Rechercher.
  5. Dans la boîte de dialogue Sélectionner un utilisateur, effectuez l'une des opérations suivantes pour sélectionner un propriétaire :
    • Sélectionnez un utilisateur dans la liste et cliquez sur OK.
    • Double-cliquez sur un nom d'utilisateur.
