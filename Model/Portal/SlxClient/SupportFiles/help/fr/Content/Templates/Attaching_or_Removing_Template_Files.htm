Mise en pièce jointe ou suppression de fichiers de modèle

Vous pouvez joindre des fichiers Windows ou de bibliothèque à un modèle. Lorsque ce modèle est utilisé dans un publipostage, ces fichiers joints sont automatiquement envoyés au contact ou au lead. Si vous copiez un modèle contenant une pièce jointe, cette dernière est également copiée. Vous pouvez également supprimer une pièce jointe d'un modèle.

Pour joindre un fichier à un modèle

  1. Ouvrez l'Editeur de modèles.
  2. Selon le type de fichier à joindre, effectuez l'une des opérations suivantes :
     
    • Fichier Windows : cliquez sur Insérer un fichier. Localisez l'emplacement du fichier, puis cliquez sur Insérer le fichier.
    • Pour joindre un fichier de bibliothèque : sur le menu ou le ruban Insérer, cliquez sur Bibliothèque. Dans la boîte de dialogue Bibliothèque, cliquez sur le(s) fichier(s), puis sur Insérer.
      Le nom du fichier s'affiche dans la zone Pièce(s) jointe(s).
  3. Vous pouvez ajouter un autre fichier ou cliquer sur Enregistrer et fermer.

Pour supprimer une pièce jointe d'un modèle

  1. Ouvrez l'Editeur de modèles.
  2. Dans la zone Pièce(s) jointe(s), sélectionnez la pièce jointe que vous souhaitez supprimer.
  3. Effectuez l'une des opérations suivantes pour supprimer la pièce jointe :
    • Cliquez avec le bouton droit de la souris, puis sélectionnez Supprimer.
    • Appuyez sur Supprimer.
  4. Vous pouvez supprimer un autre fichier ou cliquer sur Enregistrer et fermer pour fermer l'éditeur de modèle.
