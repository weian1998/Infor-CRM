Diffusion d'un modèle privé

Si vous diffusez un modèle privé à un autre utilisateur, il est recommandé de copier le modèle dans un premier temps, puis de diffuser le modèle copié à l'autre utilisateur. Cela permet de maintenir le modèle privé dans votre liste.
  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Dans la liste Modèles privés, cliquez sur le dossier Lettre, Fax ou E-mail pour le développer.
  3. Cliquez avec le bouton droit de la souris sur le modèle que vous souhaitez diffuser à un autre utilisateur et dans le menu contextuel, pointez sur Propriétés.
  4. Dans la boîte de dialogue Propriétés pour, dans la zone Propriétaire, cliquez sur Rechercher pour sélectionner un utilisateur Infor CRM dans la liste. Double-cliquez sur le nom de l'utilisateur.
  5. Cliquez sur OK. Le modèle s'affiche dans le dossier Modèles privés pour l'utilisateur sélectionné dans l'étape 3 et est supprimé de la liste Modèles privés de l'utilisateur actif.
