Création d'une proposition

La fonction de publipostage permet de créer des propositions commerciales.

Pour créer une proposition

  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Dans la boîte de dialogue Gestion des modèles, cliquez sur Nouveau.
  3. Dans la boîte de dialogue des propriétés, définissez les propriétés du modèle, puis cliquez sur OK.
    Un exemple de modèle type correspondant à votre sélection apparaît dans l'éditeur. Par exemple, si vous créez un modèle de lettre, un exemple de lettre type apparaît dans l'éditeur.
    Vous pouvez taper ou supprimer un contenu, ou encore coller du texte provenant d'un autre document.
  4. Pour ajouter des champs de fusion Infor CRM au modèle, placez le curseur dans le modèle à l'endroit voulu pour le champ de fusion, cliquez sur Insérer le champ SLX, sélectionnez ensuite un champ dans la liste des champs de fusion.
    Le nouveau champ de fusion apparaît dans le modèle.
  5. Vous pouvez ajouter ou supprimer des champs de fusion selon vos besoins.
  6. Pour ajouter la table des produits d'une opportunité, placez le curseur dans le modèle à l'endroit voulu pour cette table ; puis, dans la barre d'outils, cliquez sur Insérer un champ spécial.
    1. Dans le menu, sélectionnez Table des produits de l'opportunité.
      La boîte de dialogue Editeur des champs de publipostage Infor CRM s'ouvre.
    2. Dans la liste, mettez en surbrillance le champ de la table à modifier.
    3. Utilisez Modifier les champs pour effectuer vos modifications, puis cliquez sur Appliquer.
      • Identificateur de champ : permet de modifier le texte affiché dans le modèle.
      • Format de la table : permet d'appliquer un formatage au champ d'une table.
      • Options : permet d'appliquer des propriétés au format d'une table.
      • Placement automatique : permet de définir des propriétés de placement automatique dans la table.
      • Texte résumé : permet d'entrer le texte à utiliser comme libellé du prix global des produits de la table.
      • Modifier les colonnes de la table : permet de modifier les légendes, la largeur et l'alignement des colonnes de la table.

  7. Cliquez sur OK.
