Modification des colonnes

L'option permettant de modifier les colonnes est réservée aux utilisateurs expérimentés.

Lorsque vous modifiez des colonnes dans la boîte de dialogue Modifier les colonnes, vous devez entrer manuellement les informations sur ces colonnes en fonction des informations SQL utilisées. Il n'existe pas de synchronisation automatique entre les informations SQL et celles des colonnes. Si vous ne fournissez aucune information sur les colonnes, ces dernières sont formatées à l'aide des paramètres par défaut et les noms de champ sont utilisés pour leur légende. Si le nombre de colonnes définies est supérieur au nombre de colonnes SQL, les informations sur les colonnes supplémentaires sont ignorées. Si le nombre de colonnes définies est inférieur au nombre de colonnes SQL, les colonnes restantes sont formatées à l'aide des paramètres par défaut, mais aucune légende ne leur est affectée.

La boîte de dialogue Modifier les colonnes propose les options suivantes :

  • Cliquez sur le bouton Ajouter une colonne pour ajouter une nouvelle colonne en fin de liste, avec les valeurs par défaut suivantes :
    • Légende = « Colonne%d », où %d est le numéro de la colonne
    • Colonne = %d, où %d est le numéro de la colonne
    • Largeur = 1,00
    • Alignement = (par défaut)
    • Type de format = (aucun)
    • Chaîne de formatage = vide
  • Le bouton Supprimer la colonne supprime la colonne sélectionnée de la liste.
  • Le bouton Monter la colonne déplace la colonne sélectionnée d'un rang vers le haut dans la liste. Si la colonne est déjà en première position, cette action n'a aucun effet.
  • Le bouton Descendre la colonne déplace la colonne sélectionnée d'un rang vers le bas dans la liste. Si la colonne est déjà en dernière position, cette action n'a aucun effet.
  • La grille des colonnes permet d'afficher et de modifier les colonnes. Vous pouvez modifier les colonnes Légende, Largeur, Alignement, Type de format et Chaîne de formatage en cliquant dans la colonne et en appuyant sur la touche F2.
    • Légende : représente le texte affiché dans la première cellule d'une colonne.
    • Colonne : affiche la position ordinale de la colonne. Vous ne pouvez pas modifier cette colonne.
    • Largeur : représente la largeur de la colonne (en pouces). La largeur par défaut est 1,00.
    • Alignement : représente l'alignement du texte à l'intérieur de la cellule.
    • Type de formatage : représente le type de formatage à appliquer à ce champ.
    • Vous pouvez choisir l'un des éléments suivants dans la liste : (aucun), Fixe, Entier, Dateheure,
    • Pourcentage, Devise, Utilisateur, Téléphone, Propriétaire et Valeur booléenne.
    • Chaîne de formatage : représente la chaîne à utiliser pour le type de format sélectionné.

    		•

    Le champ de fusion Table des produits de l'opportunité est fixe ; vous ne pouvez pas déplacer, ajouter ou supprimer de colonnes.
