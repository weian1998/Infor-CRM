Personnalisation du menu Insérer le champ SLX

Infor CRM vous permet de personnaliser les champs du menu Insérer le champ SLX. Vous pouvez définir le type de format et la chaîne pour le champ, insérer un séparateur, supprimer un champ ou déplacer un champ plus haut ou plus bas dans la liste.

Pour personnaliser le menu Insérer le champ SLX

  1. Ouvrez l'Editeur de modèles.
  2. Dans la barre d'outils de l'éditeur de modèle, cliquez sur Insérer le champ SLX, puis sélectionnez Personnaliser.
    La boîte de dialogue Personnaliser le menu s'affiche.
  3. Effectuez l'une des opérations suivantes pour personnaliser le menu :
    • Pour changer le format d'un champ, sélectionnez celui-ci puis cliquez sur Formater un champ.
    • Pour ajouter un champ au menu Insérer le champ SLX, cliquez sur Insérer un champ.
    • Pour insérer un séparateur dans le menu, positionnez le curseur dans le menu, puis cliquez sur Insérer un séparateur.
    • Pour supprimer un champ du menu, sélectionnez le nom de champ, puis cliquez sur Supprimer.
    • Pour modifier l'emplacement du champ dans le menu, sélectionnez le nom du champ, puis cliquez sur Monter ou Descendre.
    • Pour restaurer les éléments du menu de champ de fusion Infor CRM par défaut, cliquez sur Rétablir les valeurs par défaut. Dans la fenêtre de confirmation du rétablissement des valeurs par défaut, cliquez sur Oui pour supprimer toutes les personnalisations ou sur Non pour les conserver.
