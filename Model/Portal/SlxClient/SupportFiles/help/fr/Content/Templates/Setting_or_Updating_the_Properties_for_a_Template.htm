Définition ou mise à jour des propriétés d'un modèle

Vous pouvez définir ou modifier les propriétés d'un modèle privé ou public lorsque vous créez, modifiez ou copiez ce modèle. Si vous modifiez ou copiez un modèle, certaines propriétés sont paramétrées par défaut avec les paramètres du modèle sélectionné ; cependant, vous pouvez modifier ces paramètres, si nécessaire.

Pour ouvrir la boîte de dialogue Propriétés, effectuez l'une des opérations suivantes :
  • Dans la boîte de dialogue Gestion des modèles, cliquez sur Propriétés.
  • Dans la boîte de dialogue Gestion des modèles, cliquez sur Nouveau.
  • Dans la boîte de dialogue Gestion des modèles, sélectionnez un modèle, puis cliquez sur Modifier ou sur Copier.

Pour définir ou modifier les propriétés pour un modèle

  1. Dans la zone Type de modèle, sélectionnez un des types de modèle. Les valeurs par défaut sont E-mail, FAX ou Lettre, mais votre société peut créer d'autres types.
  2. Dans la zone Nom, entrez le nom du nouveau modèle.

    Le nom ne doit pas contenir d'apostrophes (‘), de barre oblique inverse (\) ni de tiret (-).

  3. Dans la zone Famille, sélectionnez une famille pour le modèle. Les valeurs par défaut sont Lettre, FAX et E-mail, mais votre société peut créer d'autres types de famille.
  4. Dans la zone Table principale, sélectionnez Contact ou Lead. Pour créer un modèle à utiliser pour des contacts, sélectionnez Contact. Pour créer un modèle à utiliser pour des leads, sélectionnez Lead.
  5. Dans la zone Propriétaire, sélectionnez un utilisateur Infor CRM en tant que propriétaire du modèle.

    Si vous diffusez un modèle privé à un autre utilisateur, il est recommandé de commencer par copier ce modèle, puis de diffuser la copie du modèle à l'autre utilisateur. Cela permet de maintenir le modèle privé dans votre liste.

  6. Cliquez sur OK.
