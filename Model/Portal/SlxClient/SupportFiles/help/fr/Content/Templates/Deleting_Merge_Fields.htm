Suppression des champs de fusion

Vous pouvez supprimer des champs de fusion de modèles à partir de l'éditeur de modèle d'e-mail, de lettre ou de fax.

Pour supprimer un champ de fusion

  1. Ouvrez l'Editeur de modèles.
  2. Cliquez sur Modifier les champs dans la barre d'outils.
    La boîte de dialogue Editeur des champs de publipostage Infor CRM s'ouvre.
  3. Dans la barre d'outils, cliquez sur Supprimer du modèle.
  4. A l'invite, cliquez sur Oui pour supprimer le champ.
  5. Cliquez sur OK pour fermer l'éditeur des champs de publipostage de Infor CRM.
