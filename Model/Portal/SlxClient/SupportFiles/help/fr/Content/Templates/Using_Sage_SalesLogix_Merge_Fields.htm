Utilisation des champs de fusion  Infor CRM 

Le menu Insérer le champ SLX contient les champs de fusion Infor CRM les plus couramment utilisés. Cependant, lorsque vous créez ou modifiez un modèle et recherchez un champ Infor CRM qui n'est pas dans le menu, vous pouvez utiliser Sélectionner plus de champs SLX pour accéder à tous les champs de fusion Infor CRM. De plus, vous pouvez ajouter un champ au menu et personnaliser celui-ci.
  • Les champs de fusion Infor CRM sont au format XML, qui est sensible à la casse. Si la casse est modifiée dans un champ de fusion, il ne sera pas fusionné.
  • Lorsque vous créez ou modifiez un modèle, vous ne pouvez pas placer des champs de fusion Infor CRM dans une zone de texte étant donné que cette fonction n'est pas prise en charge par Microsoft Word.

La boîte de dialogue Insérer le champ Publipostage Infor CRM comporte deux volets :

  • Le volet gauche affiche les onglets avec les tables Contact ou Lead, Adresse et Infos utilisateur. Cliquez sur le signe plus pour développer l'arborescence des tables. Sélectionnez Afficher toutes les jointures pour afficher toutes les tables jointes.
  • Le volet droit affiche tous les champs pour la table sélectionnée. Vous pouvez cliquez avec le bouton droit de la souris pour créer une jointure locale.
    • Tabulation Champ
    Contact ou lead Contient des informations sur le destinataire de la lettre, notamment son nom et celui de sa société. Si vous avez sélectionné Contact dans les propriétés du modèle de la Table principale, les champs de la table Contact s'affichent. Si vous avez sélectionné Lead comme table principale, les champs de la table Lead s'affichent.
    Adresse Contient des informations relatives à l'adresse du destinataire. Utilisez cet onglet lorsque vous sélectionnez les champs d'adresse du contact ou les champs d'adresse du lead.
    Infos utilisateur Contient des champs relatifs à l'expéditeur, y compris les informations sur l'adresse.

Pour accéder à des champs de fusion supplémentaires

  1. Dans le menu Insérer le champ SLX, cliquez sur Autres champs SLX.
  2. Sélectionnez un des onglets.
  3. Dans le volet gauche, sélectionnez un nom de table.
  4. Dans le volet droit, sélectionnez un champ.puis cliquez sur Insérer.
  5. Ajoutez des champs au modèle, puis cliquez sur Terminé.

Ajout de champs au menu Insérer le champ SLX

  1. Dans le menu Insérer le champ SLX, cliquez sur Personnaliser.
  2. Cliquez sur Insérer un champ.
  3. Cliquez sur l'onglet qui contient les tables et les champs que vous souhaitez insérer.
  4. Dans le volet droit, sélectionnez un champ.
  5. S'il vous faut changer le type de format du champ, dans la liste Type de format, sélectionnez un type.
  6. Dans la zone Chaîne de formatage, tapez la chaîne de formatage.
  7. Dans la zone Affichage sous forme de, le champ dans le menu apparaît comme décrit, mais vous pouvez tapez un nom d'affichage pour le champ.
  8. Cliquez sur Ajouter au menu.
  9. Ajoutez des champs au menu ou cliquez sur Terminé.
