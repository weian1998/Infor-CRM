Suppression d'un modèle

Vous ne pouvez supprimer que les modèles privés dont vous êtes propriétaire. Cependant, si le modèle a été diffusé à d'autres et est partagé, l'administrateur Infor CRM doit le supprimer.

Pour supprimer un modèle

  1. Dans le menu Rédiger, sélectionnez Modèles.
  2. Ouvrez la boîte de dialogue de gestion des modèles ou de sélection.
    • Utilisateurs de dans le menu Rédiger, sélectionnez Modèles.
  3. Dans la boîte de dialogue Gestion des modèles, sélectionnez un modèle dans la liste Modèles privés, puis cliquez sur Supprimer.
  4. Cliquez sur OK pour fermer la fenêtre de message.
  5. Dans la boîte de dialogue Gestion des modèles, effectuez l'une des opérations suivantes :
    • Sélectionnez un autre modèle privé à supprimer.
    • Cliquez sur Fermer pour fermer la boîte de dialogue.
