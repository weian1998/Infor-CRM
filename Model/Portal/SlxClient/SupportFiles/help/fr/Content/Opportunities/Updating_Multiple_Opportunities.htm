Mise à jour de plusieurs opportunités

Vous pouvez mettre à jour simultanément plusieurs opportunités à l'aide de la boîte de dialogue Mettre à jour les opportunités.

Pour mettre à jour plusieurs opportunités avec des informations différentes

  • Utilisez la grille Opportunité pour afficher et modifier les informations sur l'opportunité, puis cliquez sur OK pour enregistrer les modifications et fermer la boîte de dialogue Mettre à jour les opportunités.

Pour mettre à jour plusieurs opportunités

  1. Dans la barre de navigation, cliquez sur le bouton Opportunités.
  2. Ouvrez le groupe qui contient les opportunités que vous souhaitez mettre à jour.
  3. Sélectionnez les enregistrements à mettre à jour et effectuez l'une des opérations suivantes :
    • Cliquez avec le bouton droit et sélectionnez l'option Mettre à jour les opportunités.
    • Dans le Volet des tâches, sous Tâches courantes, cliquez sur Mettre à jour les opportunités.
  4. Utilisez la boîte de dialogue Mettre à jour les opportunités afin de mettre à jour les informations suivantes :

    • Gestionnaire de compte

      1. Sélectionnez Responsable du compte dans la liste déroulante Mettre à jour.
      2. Cliquez sur À : Bouton  Rechercher  , sélectionnez un nouveau responsable de compte dans la liste puis cliquez sur OK.
    • Ajouter à la prévision
      1. Dans la liste déroulante Mettre à jour, sélectionnez Ajouter à la prévision ?.
      2. Sélectionnez Oui pour inclure les opportunités dans les rapports prévisionnels, ou Non pour en exclure les opportunités.
    • % de probabilité de clôture
      1. Sélectionnez % de probabilité de clôture dans la liste déroulante Mettre à jour.
      2. Cliquez sur les flèche déroulante vers le bas  et sélectionnez une nouvelle probabilité de clôture dans la liste.

    • Clôture approx.

      1. Dans la liste déroulante Mettre à jour, sélectionnez Clôture approx., .
      2. Sélectionner le mode de mise à jour de la date estimée de la clôture :
           
        • Pour modifier la date estimée de la clôture et indiquer une date spécifique, sélectionnez l'option À : et cliquez sur bouton Agenda, sélectionnez une nouvelle date, puis cliquez sur OK.
        • Pour déplacer la date de clôture estimée de quelques jours dans le passé, sélectionnez l'option Déplacer, sélectionnez En arrière dans la liste déroulante, puis saisissez le nombre de jours duquel la date sera reculée.
        • Pour reculer d'un certain nombre de jours la date estimée de clôture, cliquez sur l'option Déplacer, puis sélectionnez Avancer dans la liste déroulante, et indiquez de combien de jours vous souhaitez reculer la date.
    • Commentaires
      1. Dans la liste déroulante Mettre à jour, sélectionnez Commentaires.
      2. Saisissez vos remarques dans le champ À : en regard de la liste déroulante.
  5. Cliquez sur OK.
