Concurrents sur l'opportunité

Utilisez cette vue pour ajouter, modifier ou supprimer des concurrents associés à une opportunité et pour vous aider à établir votre stratégie.

Procédure

Ouvrez cette vue
  • Cette vue apparaît dans la vue Insérer une opportunité et sous forme d'onglet dans la vue Détails de l'opportunité. Si l'onglet n'apparaît pas, cliquez sur l'onglet Autres onglets.
Ajouter un concurrent à une opportunité
  1. Cliquez sur Ajouter un concurrent.
  2. Localisez et sélectionnez le concurrent à l'aide de la fonction Recherche.
  3. Cliquez sur OK.

Modifier les informations relatives aux concurrents d'une opportunité

Supprimer un concurrent d'une opportunité

Gérer les concurrents (cette option n'est pas disponible pour tous les utilisateurs)

Cette vue contient les colonnes suivantes :

  • Concurrent : affiche le nom du concurrent.
  • Evaluation - Contient l'évaluation du concurrent pour l'opportunité en cours.
  • Faiblesses - Contient les points faibles du concurrent.
