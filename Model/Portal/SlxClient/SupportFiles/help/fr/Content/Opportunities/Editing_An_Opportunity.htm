Modification d'une opportunité

Utilisez la vue Détails des opportunités pour modifier les informations d'une opportunité.

Pour modifier une opportunité

  1. Dans la barre de navigation, cliquez sur Opportunités.
  2.  Recherchez l'opportunité à modifier, puis cliquez sur la description de l'opportunité.
  3. Dans la vue Détails de l'opportunité, modifiez les informations sur l'opportunité.
  4. Cliquez sur Enregistrer.
