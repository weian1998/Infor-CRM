Свойства Элемента управления Валюта

Вы можете просматривать и менять значения следующих свойств данного элемента управления:

Основное Свойство Описание

Заголовок

Данное свойство служит для отображения подписи данного элемента управления.

Выравнивание Заголовка

Выравнивание текста метки. Существуют следующие доступные параметры:

  • По левому краю
  • По правому краю
  • По центру

Всплывающая Подсказка

Текст, который отображается как всплывающая подсказка, когда курсор мыши находится над элементом управления.

Только Чтение

Запрещает редактирование в элементе управления.
Расширенное Свойство Описание

Режим Отображения

Режим визуального отображения элемента управления. Доступные параметры меняются в зависимости от типа элемента управления:

  • Как Элемент Управления - выглядит как текстовое окно.
  • Как Текст - отображается как простой текст.
  • Как Гиперссылка - отображается как ссылка.
  • Нормальный - выглядит как метка.

Расположение Метки

Расположение метки относительно элемента управления.

Видимый

Показать или скрыть данный элемент управления в этой форме.

Десятичные цифры

Количество цифр после десятичного для отображения и учитывания ввода данных.

Автоматически -1 обозначает используйте "Нет. из цифр после десятичного", оговоренное в региональных настройках пользователя. Также наберите целое от 0 до 4. Если вы хотите ввести число большее 4, то зарегистрируйтесь а Приложении Архитектор, чтобы удостовериться, что поле, к которому привязан элемент управления, был определен по крайней мере такими же цифрами после десятичного.

Когда пользователь вводит несколько цифр, чем оговореннaя точность, элемент управления или столбец Валюта заполняет цифры нулями. Если вы хотите, чтобы элемент управления Числовой заполнился цифрами с оговоренной точностью, настройте Ограниченный=Верный.

Включенный

Позволяет пользователю взаимодействовать с данным элементом управления.

Максимальная Длина

Максимальное количество символов, которое может ввести пользователь.

Обязательный

Запрашивает значение при сохранении данных.

Привязка Данных

Определяет откуда происходят данные элемента управления. Данная информация только для чтения в Проектировщике Web Форм.

Тип Обменного Курса

Определяет тип обменного курса, который вы хотите использовать в валютных полях, отображаемых в форме. Вы должны настроить Код Обменного Курса и Обменный Курс с помощью привязки данных, если Тип Обменного Курса = Тип Сущности. Available options are:

  • Базовый Курс - Базовый валютный курс.
  • Мой Курс - Приоритет обменного курса текущего пользователя.
  • Курс Сущности - Обменный курс хранится в сущности. Данный параметр требует настройки Обменного Курса и Кода Обменного Курса. Используйте Приложение Архитектор.

Идентификатор Элемента Управления

Идентификатор для данного элемента управления. Данная информация только для чтения в Проектировщике Web Форм.

Тип Элемента Управления

Тип элемента управления Saleslogix такой, как Текстовое Окно. Данная информация только для чтения в Проектировщике Web Форм.
Свойство Макета Описание

Строка

Номер строки элемента управления.

Столбец

Номер столбца данного элемента управления.

Интервал Строк

Количество ячеек, которые элемент управления занимает по вертикали.

Интервал Столбца

Количество ячеек, которые элемент управления занимает горизонтально.
