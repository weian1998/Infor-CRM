Что такое Формы, Представления, Элементы Управления и Поля Данных?

Формы и Представления

Формы это многоразовые части собраные в различных сочетаниях для создания различных представлений; формы являются строительными блоками всех представлений в Saleslogix. Например, Детальное представление Субъекта создается из одной формы для детальной панели и других форм для таблиц.

Форма может использоваться во многих представлениях. Например, форма Добавить Контаткты используется в Детальном представлении Контактов и в представлении Добавить Констакты.

Существует два типа форм: пользовательские формы и Quick Формы.

Вы можете сделать простые изменения представления с помощью редактирования его составных форм по средством Проектировщика Web Форм. Например, вы можете изменить заголовки и переместить элементы по форме. Когда вы сохраняете изменения формы, все представления, содержащие форму, также меняются.Изменения, которые вы производителе, влияют на каждый элемент в системе, потому что определения формы и представления схожи. элементы управления

Каждый элемент в форме, например, кнопка или поле, называется "элемент управления". Существует около 30 разных видов элементов управления, использующихся в Saleslogix формах.

Некоторые типы элементов управления способствуют выполнению действия.Примеры являются кнопками, которые сохраняют данные, и ссылками, открывающими представление.

Многие типы элементов управления показывают данные из базы данных Saleslogix, так что пользователи могут просматривать или редактировать данные. Например, данный список выбора дает возможность пользователям просматривать и изменять статус субъекта:

Все типы элементов управления, о которых мы до сих пор говорили, являются видимыми элементами управления; таким образом, они предназначены для отображения в формах. Некоторые типы элементов управления невидимы.

Проектировщик Web Форм показывает макет элементов управления в формах и позволяет вам перетаскивать и помещать элементы управления в различные позиции.

Свойства элементов управления

Свойства элементов управления - это настройки, которые определяют поведение и вид элемента управления. Например, свойство Изображения кнопки определяет иконку для кнопки. Элементы управления Список Выбора имеют свойство, называемое Позволить Многочисленные Выборки. Два свойства - Заголовок и Всплывающая Подсказка - являются общими почти для всех элементов управления.

При выборе элемента управления в области редактирования формы в Проектировщике Web Форм, свойства элемента управления отображаются на вкладках, где вы можете изменить многие из них.

Дополнительные свойства элемента управления могут быть доступны из Архитектора приложений.

Поля данных

Поля данных - это индивидуальными видами информации о субъектах, контактах и т.д., которую пользователи просматривают и редактируют. Имя Субъекта и Статус являются примерами плей данных. Элемент управления, показывающий данные из базы данных Saleslogix, возможен, так как он связан с полем данных в базе данных.

Проектировщик Web Форм показывает поле данных, к которому привязан элемен управления.
