Добавление Поля Данных к Форме

Вы можете добавить существующие поля данных к форме, используя панель Добавить Поле Проектировщика Web Форм. Она покажет дерево полей данных, которое может добавляться к текущей форме. Вы можете увеличить пункты в дереве, имеющие знак плюс, чтобы использовать их состовляющие поля, например, Адрес.Город.

Если поле, которое вы хотите добавить на форму, еще не существует в базе данных, то оно должно быть сначала добавлено в базу данных. Для получения дополнительной информации смотрите "Добавление поля в таблицу" в справке к Администратору или «Изменение сущности" и "Вставка нового свойства сущности» в справке Архитектора приложений.

Чтобы добавить

  1. При необходимости, сначала нажмите Сохранить в случае необходимости отменить следующие шаги.
  2. Щелкните на пункт в панеле Добавить Поле и перетащите его.

  3. Бросьте его на любое место, которое выделяется при наведении курсора на него.

    Будет создан элемент управления и привязан к полю данных в базе данных. Тип элемента управления будет установлен в зависимости от типа данных. Например, при переносе номера телефона на форму создастся элемент упаравления Номер телефона.

  4. По выбору редактировать значения характеристик нового элемента контроля.
  5. Если вы уверены, что вы хотите сохранить поле, которое вы добавили, нажмите Сохранить. Иначе, нажмите F5, чтобы вернуться к последней сохраненной версии формы.
