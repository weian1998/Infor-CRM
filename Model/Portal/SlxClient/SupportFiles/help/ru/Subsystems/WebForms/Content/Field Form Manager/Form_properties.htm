Свойства Формы

Вы можете просматривать и менять значения следующих свойств данной формы:

Основное Свойство Описание
Активный Элемент Управления Идентификатор элемента управления, который вы хотите, чтобы курсор пользователя запустился при первом отображении страницы. Например, Имя Субъекта в форме Добавить Субъект.
Название Название формы, например, Данные Субъекта. Данная информация только для чтения в Проектировщике Web Форм.
Описание Описание формы.
Основное Свойство Описание
Использовать Имя Сущности в качестве Заголовка При настройке на Верный название формы определяется по типу сущности, на которой основывается форма. Например, если форма основывается на Субъекте, то заглавием формы будет текущие название субъекта.
Основное Свойство Описание
Портал Название веб-сайта, частью которого яввляется представление. Примеры порталов находятся в Sales Saleslogix Web Клиенте и Портале Клиента. Данная информация только для чтения в Проектировщике Web Форм.
Представление Название представления, которое содержит форму. Например, Детали Субъекта. Данная информация только для чтения в Проектировщике Web Форм.
Режимы Для данного представления режимы, в которых представлена форма: Деталь, Вставка и Список. Данная информация только для чтения в Проектировщике Web Форм.
Описание Описание представления. Данная информация только для чтения в Проектировщике Web Форм.
