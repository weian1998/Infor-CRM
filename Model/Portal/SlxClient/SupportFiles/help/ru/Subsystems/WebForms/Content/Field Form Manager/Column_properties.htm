Свойства Колонки

Вы можете отображать и менять значения следующих свойств данной колонки:

Основное свойство Описание

Индекс

Положение строки и столбца в форме. Начинается с нуля.

Данная информация только для чтения в Проектировщике Web Форм.

Ширина

Определяет назначение ширины каждого столбца.

Тип Размера

Способ назначение ширины является одним из следующих :

 

  • Абсолютный: Фиксированная ширина столбца вне зависимости от размера окна.
  • Автоматический Размер: Система автоматически определяет оптимальную ширину для каждого столбца, учтывая минимальную ширину ширину для каждого столбца перед прокруткой. Данная минимальная ширина определяется системой и не может настраиваться конструктором формы.
  • Процентный: Процентное назначение, основанное на доступной ширине окна. Процентное отношение назначений ко всем столбцам должно доходить до 100%. Если окно падает ниже определенного размера, то столбцы не сократятся ниже минимальной ширины. Вместо этого форма прокрутится. Данная минимальная ширина определяется системой и не может настраиваться редактором quick формы.
