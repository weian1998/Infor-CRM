Редактирование Свойств Элемента Управления

Редактор форм показывает значения свойств для текущего выбранного элемента управления и дает вам возможность редактировать их.

Для редактирования

  1. Откройте форму в Редактор форм.
  2. Щелкните на элемент управления в макете.
    Свойства выбранного элемента управления отображаются в закладках ниже панели макета элемента управления.
    Дополнительные свойства элемента управления могут быть доступны из Архитектора приложений.
  3. Щелкните на закладку и отредактируйте настройки для данного элемента управления, который вы конфигурируете. Нажмите на ссылку ниже, чтобы увидить подробную информацию о свойствах элемента управления:

Визуальные Элементы Управления

  • Флажок
  • Комбинированное Поле
  • Валюта
  • Таблица Данных
  • Календарь
  • Email
  • Числовое поле
  • Владелец
  • Номер телефона
  • Справочник
  • Текстовое Поле
  • URL
  • Пользователь
  • *Все другие визуальные элементы управления

Невизуальные Элементы Управления

  • Источник Данных
  • Скрытый Текст
