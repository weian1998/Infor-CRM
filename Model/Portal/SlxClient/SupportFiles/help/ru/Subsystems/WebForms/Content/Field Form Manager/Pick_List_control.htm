Свойства Элемента управления Справочник

Вы можете просмотреть и изменить значения следующих свойств данного элемента управления:

Основное Свойство Описание

Заголовок

Данное свойство служит для отображения подписи данного элемента управления.

Выравнивание Заголовка

Выравнивание текста метки. Существуют следующие доступные параметры:

  • По левому краю
  • По правому краю
  • По центру

Всплывающая подсказка

Текст, который отображается как всплывающая подсказка, когда курсор мыши находится над элементом управления.

Только Чтение

Запрещает редактирование в элементе управления.
Расширенное Свойство Описание

Режим визуального отображения

Режим визуального отображения элемента управления. Доступные параметры меняются в зависимости от типа элемента управления:

  • Как Элемент Управления - выглядит как текстовое окно.
  • Как Текст - отображается как простой текст.
  • Как Гиперссылка - отображается как ссылка.
  • Нормальный - выглядит как метка.

Расположение Метки

Расположение метки относительно элемента управления.

Видимое

Показать или скрыть данный элемент управления в этой форме.

Позволить Многочисленные Выборы

Позволяет пользователю выбирать более одного пункта для каждого списка выбора.

Поведения списка поисков Сети и Web Клиента могут определятся в Детальном представлении Списка Выбора в Web Клиенте. Например, вы можете включить или выключить Запрашиваемый Ввод для индивидуального списка выбора. Для применения поведений к спискам выбора Web Клиента, длжны быть выбраны параметры Применить к свойству списка выбора. Если не так, то поведения, определенные элементом управления Quick Формы, которая вместо использует применить список выбора.

До настройки параметров списка выбора, убедитесь, что вы понимаете свойства элемента управля Список Выбора и каким образом данные списка выбора сохраняются в базе данных. Понимание как привязки данных из списка выбора к базе данных определит поддерживается ли параметр. Например, если вы редактируете Приоритет списка выбора в отношении "Позволить многочисленные выборки", то список выбора не будет работать надлежащим способом, потому что поле базы данных слишком мало, чтобы сохранить большое количество текстаt.

Подключенный

Позволяет пользователю взаимодействовать с данным элементом управления.

Требует, чтобы был набрано значение в списке.

Максимальная Длина

Максимальное количество символов, которое может ввести пользователь.

Запрашиваемый

Запрашивает значение при сохранении данных.

Привязки Данных

Определяет откуда происходят данные элемента управления. Данная информация только для чтения в Проектировщике Web Форм.

Название Списка Поиска

Название списка значений, из которого пользователь может выбирать, при вводе данных. Данная информация только для чтения в Проектировщике Web Форм.

Режим Сохранения

Каким образом значение сохраняется. Доступные параметры:

  • Текст- Если значение сохранилось как строка, отображает строку.
  • ID - Если значение сохранилось как Идентификатор, ищет соответствующую строку из списка выбора и отображет её.
  • Код- Если код сохранился, отображает соответствующую строку. Нечасто используется.

Данная информация только для чтения в Проектировщике Web Форм.

Идентификатор Элемента Управления

Идентификатор для данного элемента управления. Данная информация только для чтения в Проектировщике Web Форм.

Тип Элемента Управления

Тип элемента управления Saleslogix такой, как Текстовое Окно. Данная информация только для чтения в Проектировщике Web Форм.
Свойство Макета Описание

Строка

Номер строки элемента управления.

Столбец

Номер столбца данного элемента управления.

Интервал Строк

Количество ячеек, которые элемент управления занимает по вертикали.

Интервал Столбца

Количество ячеек, которые элемент управления занимает горизонтально.
