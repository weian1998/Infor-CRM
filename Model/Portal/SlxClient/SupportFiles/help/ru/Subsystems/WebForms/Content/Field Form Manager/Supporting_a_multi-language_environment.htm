Поддержка многоязычного окружения

Текстовые строки, используемые для элементов управления, такие как заголовки и всплывающие подсказки, собраны в ресурсном файле для облегчения локализации. Установки, поддерживающие различные языки, имеют ресурсный файл переведенных строк для каждой языковой культуры. Например, установка Saleslogix, поддерживающая основную культуру и немецкий (в Германии), имеет один ресурсный файл для Инвариант (базовая культура) и один 'нем-Нем'. Пользователь Web Клиента может переключить для отображения немецких строк с помощью настройки приоритета языка браузера на 'нем-Нем'.

Если ваш сайт сконфигурирован для поддержки различных языков, то вы можете редактировать заголовки и другие строки на каждом языке.

Для локализации текстовых строк

  1. Редактируйте форму с помощью Проектировщика Web Форм.
  2. Выберите Язык из выпадающего списка.
    Языки могут быть добавлены в этот список. Смотрите тему "Добавление Культуры в Список Проектировщика Web Форм" в Справке Приложения Архитектор для большей информации.
  3. Редактируйте значения свойств локализуемого элемента управления, как Заголовок и Всплывающая подсказка.
  4. Нажмите Сохранить.
    Заголовки и другие локализуемые строки сохраняются в ресурсном файле для текущей культуры.
  5. Просматрите форму. Чтобы увидить локализованные заголовки, измените настройку языкового приоритета вашего броузера.
