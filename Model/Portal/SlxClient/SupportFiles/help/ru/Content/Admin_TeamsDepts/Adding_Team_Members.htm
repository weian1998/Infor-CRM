Добавление участников команды

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Можно добавить пользователей, подразделения или другую команду к команде. Можно добавить членов команды по одному или группой. При добавлении пользователя к команде, можно автоматически включить непосредственного менеджера пользователя. Дальнейшего наследования не происходит (например, менеджер менеджера). Как выполнить?

Добавить участника команды с помощью Детального представления Команды

  1. На Детальном представлении Команды перейдите к закладке Участники.
  2. Отметьте флажок Автоматическое добавление менеджера пользователя в команду если вы хотите добавить менеджера пользователя к команде при добавлении нового участника команды.
  3. Нажмите кнопку Добавить.
  4. Используйте Поиск чтобы найти и выбрать участника команды, которого вы хотите добавить.
  5. При необходимости, редактируйте права доступа нового пользователя, добавленного в команду.
  6. Нажмите Сохранить.

Добавить несколько участников команды используя Списочное представление

  1. Откройте Списочное представление Пользователей, Команд, или Департаментов.

  2. Выберите участников команды, которых вы хотите добавить.

  3. На Панели Общие задачи, нажмите Добавить к Команде.
  4. В поле Команда используйте Поиск , чтобы найти и выбрать команду, к которой вы хотите добавить выбранных участников.
  5. Нажмите ОК.
  6. При необходимости, редактируйте права доступа нового пользователя, добавленного в команду.
