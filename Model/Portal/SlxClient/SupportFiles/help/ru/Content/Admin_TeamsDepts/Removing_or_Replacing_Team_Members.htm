Удаление или Замена Участников команды

Можно удалить участников команды или заменить одного пользователя другим в команде.

Как выполнить?

Удалить участника команды

  1. Откройте Детальное представление команды, в которую вы хотите добавить участников.
  2. Выберите закладку Участники.
  3. Щелкните ссылку Удалить для участника команды, которого вы хотите удалить.
  4. Нажмите ОК для подтверждения.
  5. Нажмите Сохранить.

Удалить пользователя из всех команд

  1. Откройте Список пользователей и выберите пользователя, которого вы хотите удалить из всех команд.
  2. На Панели Общие задачи , нажмите Удалить из всех Команд.
  3. Нажмите ОК для подтверждения.

Заменить одного участника команды другим во всех командах

  1. Откройте Детальное представление пользователя, детальное представление департамента, или детальное представление команды для участника команды, которого вы хотите заменить.
  2. На Панели Общие задачи, нажмите Заменить во всех Командах.
  3. В поле Заменить проверьте пользователя, департамент или команду, которую вы хотите удалить из всех команд.
  4. В поле На используйте поиск, чтобы найти пользователя, департамент или команду, чтобы заменить удаляемого пользователя, департамент или команду.
  5. Нажмите ОК.
