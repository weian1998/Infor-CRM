Изменение Профиля безопасности в Команде

После добавления участника команды, вы можете изменить права доступа. Изначальные права доступа устанавливаются в соответствии с стандартным профилем безопасности пользователя. Нельзя изменять профили безопасности Веб просмотра. Узнать больше о безопасности...

Для того чтобы изменить

  1. На Детальном представлении Команды перейдите к закладке Участники.
  2. Щелкните ссылку Профиль безопасности для участника, которого вы хотите изменить.

  3. Выберите новый профиль безопасности.

Стандартные Профили:

  • Стандартный Только Чтение - предоставляет доступ Только чтение для всех полей и таблиц.
  • Стандартный Чтение/Запись - предоставляет доступ на чтение и изменение информации во всех таблицах и полях за исключением поля SecCodeID таблицы Субъектов. Это ограничивает права пользователя по переназначению владельцев субъектов.
  • Профиль руководителя команды - предоставляет доступ на чтение и изменение информации для всех полей и таблиц. Это позволяет пользователю переназначить владельца субъекта. Нельзя приписать профиль руководителя команды пользователям Веб-Просмотра.
  1. Нажмите ОК.
  2. Нажмите Сохранить.
