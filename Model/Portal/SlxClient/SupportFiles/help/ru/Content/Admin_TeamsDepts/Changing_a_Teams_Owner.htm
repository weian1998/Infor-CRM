Изменение Руководителя команды

Можно использовать представления Команд или Пользователей, чтобы изменить руководителя команды. Команды могут иметь несколько руководителей.

Как выполнить?

Использовать Представление Команды для управления руководителем команды

  1. Откройте Детальную карточку Команды.
  2. На закладке Участники, нажмите Профиль безопасности для пользователя, которого хотите удалить или добавить в качестве руководителя команды.
  3. В поле профиль безопасности выберите новый профиль безопасности для пользователя.
  4. Нажмите ОК.
  5. Нажмите Сохранить.

Использовать Представление Пользователя для управления куратором команды

  1. Откройте Детальное представление пользователя для пользователя, которого хотите удалить или добавить в качестве куратора команды.
  2. Выберите закладку Состав команды .
  3. Нажмите Профиль безопасности для нужной команды.
  4. В поле Профиль безопасности выберите новый профиль безопасности для пользователя.
  5. Нажмите ОК.
  6. Нажмите Сохранить.
