Копирование Команды

При копировании существующей команды, все участники включаются в новую команду.

Для того чтобы скопировать

  1. Откройте Детальное представление Команды , которую необходимо копировать.
  2. На Панели Общие задачи, нажмите Копировать команду.
  3. Нажмите ОК для подтверждения.
  4. В поле Имя введите новое название команды.
  5. При необходимости, измените Участников команды.
  6. Нажмите Сохранить.
