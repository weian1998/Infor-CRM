Изменение участников Департамента

Чтобы редактировать участников департамента, можно редактировать департамент или изменить профиль пользователя.

Как выполнить?

Редактирование участников департаментов с помощью Детального представления Пользователя

  1. Откройте Детальное представление пользователя для пользователя, которого хотите удалить или добавить в отдел.
  2. Нажмите закладку Участники департамента и сделайте следующее:
  • Нажмите Добавить , чтобы добавить пользователя в департамент. Для того чтобы выбрать необходимый департамент используйте форму поиска.
  • Нажмите Удалить для департамента, из которого вы хотите удалить пользователя.
  1. Нажмите Сохранить.

Редактирование участников отдела с помощью Детального представления Отдела

  1. Откройте Детальное представление Департамента, который необходимо изменить.
  2. Откройте закладку Участники и сделайте следующее:
  • Нажмите Добавить, чтобы добавить пользователя в департамент. Для того чтобы выбрать необходимого пользователя используйте форму поиска.
  • Щелкните ссылку Удалить для участника департамента, которого вы хотите удалить.
  1. Нажмите Сохранить.
