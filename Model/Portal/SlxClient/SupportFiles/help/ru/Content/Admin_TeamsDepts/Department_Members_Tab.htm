Закладка Участники

Закладка Участники отображает пользователей внутри департамента. Используйте эту закладку для управления участниками департамента.

Чтобы управлять

  1. На Детальном представлении Департамента перейдите к закладке Участники.

  2. Выполните одно из следующих действий:

    • Нажмите Добавить, чтобы добавить пользователя к отделу. Для того чтобы выбрать необходимого участника используйте форму поиска.
    • Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.

  3. Нажмите ОК.
