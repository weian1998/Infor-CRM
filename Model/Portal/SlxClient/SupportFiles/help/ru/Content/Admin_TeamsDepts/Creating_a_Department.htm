Создание Департамента

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Департамент - это специализированная команда. По умолчанию, пользователи не приписаны к департаменту.

Правила Департаментов

  • Департаменты содержат пользователей, но не могут содержать команды. Команды могут содержать отделы.

  • Пользователь может принадлежать более чем одному отделу.

  • Отделы не могут иметь субъекты.

Чтобы создать

  1. В меню Администрирование выберите Новый Департамент.
  2. В поле Имя введите название отдела.
  3. В поле Стандартный профиль безопасности выберите стандартный профиль для департамента. Этот профиль применяется, когда департамент добавляется в команду.
  4. Нажмите Сохранить.
  5. Добавьте пользователей к отделу с помощью Закладки Участники.
  6. Добавьте отдел к командам с помощью Закладки Команды.
