Закладка Заметки

Используйте закладку Заметки на детальном представлении пользователя, чтобы добавить заметки о пользователе. Эта закладка позволяет добавлять информацию, не охваченную другими закладками.

Для управления

  1. На Детальном представлении пользователя перейдите к закладке Заметки.
  2. Чтобы добавить отметку даты/времени, нажмите .
  3. Введите заметки о пользователе.
  4. Нажмите Сохранить.
