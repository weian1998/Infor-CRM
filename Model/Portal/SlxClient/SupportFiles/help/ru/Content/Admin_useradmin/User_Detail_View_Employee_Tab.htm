Закладка Сотрудник

Используйте закладку Сотрудник, чтобы добавить компанию и личную информацию пользователя. Закладка Сотрудник также содержит информацию о пользователе, которая позволяет Infor связываться со сторонними приложениями.

Чтобы настроить параметры

  1. На Детальном представлении пользователей перейдите на закладку Сотрудник.
  2. В секции Рабочий введите контактную информацию пользователя.
  3. При необходимости, в поле ИД субъекта введите идентификационный код субъекта.

ИД субъекта необходим для обмена данными между Infor и приложениями субъекта.

  1. Если ваша компания приписывает номера сотрудникам и компьютерам, заполните поля Сотрудник №, № Проданного продукта#и Описание проданного продукта.
  2. В секции Личные введите личную информацию пользователя.
  3. Нажмите Сохранить.
