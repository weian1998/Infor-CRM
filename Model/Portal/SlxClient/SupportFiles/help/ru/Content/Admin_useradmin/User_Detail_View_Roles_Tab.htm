Закладка Роли

Используйте закладку Роли на детальном представлении Пользователя, чтобы приписать пользователю роль или удалить пользователя из роли. Когда пользователь приписан к роли, у него есть доступ к действиям, определяемым этой ролью. Если пользователь приписан к более чем одной роли, он/она будет иметь доступ ко всем действиям, связанным с ролями, даже если каждая роль содержит различные действия.

Как выполнить?

Открыть закладку Р

  1. Откройте Детальное представление пользователя.
  2. Перейдите на закладку Роли.

Назначить пользователю роль

  1. Нажмите Добавить роль.
  2. Используйте поиск для поиска и выбора роли, которую вы хотите назначить пользователю.
  3. Нажмите Добавить выбранные.
    Вы не можете добавить одну и ту же роль несколько раз.

Удалить роль у пользователя

  1. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.
  2. Нажмите Да.
