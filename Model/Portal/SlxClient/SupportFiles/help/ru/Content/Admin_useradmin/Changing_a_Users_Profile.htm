Изменение профиля Пользователя

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Пользователи не могут быть переведены в пользовательские шаблоны, пользовательские шаблоны не могут быть переведены в другие типы пользователей.

Для изменения

  1. Откройте Детальное представление пользователя которого необходимо изменить.
  2. Измените профиль пользователя.

Заполните эти закладки для всех пользователей:

  • Используйте информационные поля, чтобы изменить имя пользователя, e-mail, должность, подразделение, менеджера и пароль. Имя пользователя - то, что пользователь будет вводить при входе в Infor, не должно содержать пробелы или апостроф.
  • Используйте закладку Сотрудник, чтобы внести информацию по компании, рабочий телефон и номера факсов, а также личную информацию пользователя.
  • Используйте закладку Состав команды, чтобы добавить пользователя к командам.
  • Используйте закладку Члены команды пользователей, чтобы предоставить доступ другим пользователям к субъектам этого пользователя.
  • Используйте закладку Состав команды пользователей, чтобы предоставить этому пользователю доступ к субъектам других пользователей.
  • Используйте закладку Другой календарь, чтобы установить доступ к календарям других пользователей.
  • Используйте закладку Календарь пользователя, чтобы предоставить доступ другим пользователям к календарю этого пользователя.
  • Используйте закладку Безопасность, чтобы установить профиль безопасности на уровне полей.
  • Используйте закладку Участники отдела, чтобы добавить пользователя к подразделениям.
  • Используйте закладку Клиентская система, чтобы установить куратора по умолчанию и основной шаблон.

Заполните эту закладку для сервиса клиентов и/или пользователей поддержки:

  • Используйте закладку Сервис/Поддержка, чтобы установить настройки, связанные с опциями сервиса клиентов, опциями поддержки и Быстрого Поиска.

Заполните эти закладки при необходимости:

  • Используйте закладку Заметки для хранения дополнительной пользовательской информации. (Это опциональная информация, находящаяся в персональном файле пользователя).
  1. Нажмите Сохранить .
