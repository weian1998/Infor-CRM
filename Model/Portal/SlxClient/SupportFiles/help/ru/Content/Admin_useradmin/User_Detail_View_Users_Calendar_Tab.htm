Закладка Календарь пользователя

Используйте закладку Календарь пользователя на детальном представлении Пользователя, чтобы установить права доступа для других пользователей по просмотру и изменению календаря этого пользователя. По умолчанию, пользователи не имеют доступа к другим календарям пользователей. Когда календарь пользователя добавляется к профилю другого пользователя, доступ на просмотр устанавливается автоматически.

При определении доступа к календарю придерживайтесь следующих правил:

  • Пользователь появляется в своем списке календарей. Не удаляйте пользователя из его собственного календаря. Удаление пользователя приводит к удалению доступа к его собственному календарю.
  • Пользователь Администратор появляется в списке календаря пользователя и не может быть удален.
  • Если пользователь не появляется в списке календарей, этот пользователь не имеет доступа к календарю выбранного пользователя.
  • Если пользователь появляется в списке, он может просматривать календарь выбранного пользователя (когда Добавить, Изменить, Удалить и Синхронизировать не выбраны, пользователь имеет доступ только на чтение).
  • Если пользователь организовывает дело, другой пользователь с доступом к его календарю изменяет его, человек, который организовал дело, получает автоматическое уведомление об изменении.

Как выполнить?

Открыть закладку Календарь пользователя
  • В Детальном представлении пользователя откройте закладку Календарь пользователя. Если закладка не отображается, то откройте закладку Все закладки.
Разрешить другим пользователям доступ к календарю этого пользователя
  1. Для добавления пользователя нажмите Добавить пользователя.
  2. Используйте поиск для поиска и выбора пользователя, которого вы хотите добавить.

  3. Нажмите ОК.

  4. Используйте диалоговое окно Доступ к календарю для установки прав доступа.

  5. Нажмите ОК.

Изменение доступа к календарю этого пользователя
  1. Нажмите Изменить рядом с пользователем, которого вы хотите изменить.
  2. Используйте Установка прав доступа.
  3. Используйте диалоговое окно Доступ к календарю для установки прав доступа.
Удалить доступ к календарю этого пользователя
  1. Нажмите Удалить рядом с пользователем, которого вы хотите удалить.
    Если вы удаляете пользователя, выбранного в качестве календаря по умолчанию для этого пользователя в Веб клиенте, календарь этого пользователя все еще доступен до тех пор пока новый пользователь по умолчанию не будет выбран.
  2. Нажмите ОК.
