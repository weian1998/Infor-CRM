Поля Информации о Пользователе

В таблице ниже описаны основные поля.

Поле Описание/Действие
Логин доступен Определяет, может ли пользователь войти в Infor. Снимите флаг, если пользователь становится неактивным и не должен иметь доступ к Infor.
Имя пользователя

Имя, которое будет использовать пользователь при входе в систему.

Имя пользователя - необходимое поле и не может содержать пробелы и апостроф.

Имя Имя пользователя. Нажмите кнопку Изменить для открытия диалогового окна Изменить имя.
Должность Должность пользователя.
E-mail E-mail адрес пользователя.
Тип пользователя Тип пользователя Infor. Чтобы изменить тип пользователя, выберите новый тип в списке. При изменении на пользователя веб-просмотрщика, нужно перегрузить IIS.
Изменить пароль Изменение пароля пользователя.
Пароль изменен Отображает дату и время, когда пароль был изменен в последний раз.
Используемый шаблон Последний  шаблон пользователя, примененный к этому пользователю.
Регион Местоположение пользователя.
Подразделение Подразделение пользователя.
Менеджер Менеджер пользователя. Только менеджеры, которые могут быть менеджером пользователя, отображаются в списке. Менеджеры автоматически добавляются в список.
Пользователь является менеджером

Определяет, является ли пользователь менеджером и должно ли его имя отображаться в списке менеджеров. Этот параметр не доступен для пользователя admin.
