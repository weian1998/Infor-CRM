Закладка Безопасность

Эти функции доступны, только если установлена соответствующая лицензия. Для получения подробной информации о лицензировании обратитесь к Руководству по Планированию Infor.

Используйте закладку Безопасность на детальном представлении Пользователя, чтобы определять профиль безопасности на уровне полей. Системный администратор имеет доступ на все, права доступа системного администратора не могут быть изменены.

Для настройки параметров

  1. В Детальном представлении пользователя откройте закладку Безопасность.
  2. В поле Профиль безопасности уровня поля выберите стандартный профиль, который будет использоваться при добавлении пользователя к команде. Профили безопасности создаются и управляются в Администраторе.
    3. Узнать больше...

    Когда пользователь А вручную добавляется к команде Пользователя B (используя закладку лены команды пользователя), пользователь А может иметь доступ ко всем записям пользователя B. Доступ уровня поля пользователя А будет определять, имеет ли пользователь А возможность изменить записи пользователя B. Пользователи всегда имеют доступ на чтение/запись ко всем субъектам, которыми владеют, или ко всем субъектам с владельцем Каждый.

  3. Нажмите Сохранить.
