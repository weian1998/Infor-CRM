Закладка Члены команды пользователя

Команды пользователя позволяют пользователям иметь доступ к субъектам, принадлежащим другому пользователю. Например, если вы хотите, чтобы Василий имел доступ ко всем субъектам Петра, вы можете добавить Василия в команду Петра.

Пользователи, отображаемые на этой вкладке, могут иметь доступ к субъектам, принадлежащим основному пользователю (пользователю, чей профиль вы изменяете).

Как выполнить?

Открыть закладку Члены команды пользователя
  • В Детальном представлении пользователя откройте закладку Члены команды пользователя. Если закладка не отображается, то откройте закладку Все закладки.
Разрешить другим пользователем доступ к субъектам этого пользователя
  1. Нажмите кнопку Добавить еще одного пользователя к команде этого пользователя.
  2. Используйте поиск для поиска и выбора пользователя, которого вы хотите добавить.

  3. Нажмите ОК.

Изменить профиль безопасности пользователя
  1. Нажмите ссылку Профиль безопасности и установите права доступа.
  2. Нажмите ОК.
    Профиль безопасности применяется только когда выбранный пользователь работает с субъектами, принадлежащими основному пользователю (пользователю, чей профиль вы редактируете). Он не применяется, когда выбранный пользователь работает со своими субъектами.
Удалить пользователя из команды пользователей
  1. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.

  2. Нажмите ОК.
