Управлению связью с Контактом

Если Ваша Infor реализация включает в себя создание контакт для каждого пользователя Infor, Вы можете легко связать пользователя и контакт в списочном представлении пользователей.

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Как выполнить?

Для установления связи с Контактом
  1. На навигационной панели нажмите Администрирование, и затем нажмите Пользователи.

  2. Выберите пользователя, которого необходимо связать с Контактом.

  3. На панели задач нажмите Связать с Контактом.
  4. В диалоге Связь пользователя с Контактомнажмите рядом с полем Контакт кнопку Найти.
  5. Воспользуйтесь Поиском, чтобы найти Контакт, с которым необходимо связать пользователя и нажмите OK.
  6. Нажмите ОК.
Удаление связи пользователя с Контактом
  1. На навигационной панели нажмите Администрирование, и затем нажмите Пользователи.

  2. Выберите одного или нескольких пользователей для удаления связи с контактами.

  3. На панели задач нажмите Удалить взаимосвязь.
  4. Нажмите ОК.
