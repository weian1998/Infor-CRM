Строки Форматирования Числового Поля

Приведенная ниже таблица описывает параметры строки форматирования для числовых полей. Допустимые форматы строк зависят от выбранного вами типа формата.

Параметр Описание
e Научный. Форматом поля будет число с плавающей запятой. Значение преобразуется в строку вида "-d.ddd...E+ddd". Результирующая строка начинается со знака минуса, если число отрицательное. Разделителю дробной части всегда предшествует одна цифра. Конечное количество цифр в результирующей строке (включая одну перед разделителем дробной части) задается параметром точности в строке форматирования; если не задано, то по умолчанию подразумевается точность 15. После символа показателя степени "E" в результирующей строке всегда следует знак плюс или минус и, как минимум, три цифры.
f Фиксированное. Значение показывается как фиксированное поле. Формат поля должен быть значением с плавающей запятой. Значение преобразуется в строку вида "-ddd.ddd...". Результирующая строка начинается со знака минуса, если число отрицательное. Количество цифр после разделителя дробной части задается параметром точности в строке форматирования; если не задано, то по умолчанию подразумевается точность 2.
g Основной. Формат поля должен быть значением с плавающей запятой. Значение преобразуется в наикратчайшую десятичную строку, использующую научный или числовой формат. Количество значащих цифр в результирующей строке задается параметром точности в строке форматирования; если не задано, то по умолчанию подразумевается точность 15. Завершающие нули удаляются из результирующей строки, а разделитель дробной части отображается только при необходимости. Результирующая строка использует формат с фиксированной запятой, если количество цифр слева от разделителя дробной части в числе меньше или равно заданной точности, или если число больше или равно 0.00001. В противном случае, результирующая строка использует научный формат.
n Число. Формат поля должен быть значением с плавающей запятой. Значение преобразуется в строку вида "-d,ddd,ddd.ddd...". Формат "n" соответствует формату "f", за тем лишь исключением, что результирующая строка содержит разделители тысяч.
