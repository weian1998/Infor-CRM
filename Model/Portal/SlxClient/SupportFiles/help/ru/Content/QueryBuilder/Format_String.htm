Строка Форматирования

Используйте строку форматирования для форматирования данных, отображаемых в запросе. Во-первых, выберите тип форматирования. Затем, ввыедите строку форматирования для изменения выбранного типа форматирования. Допустимые параметры зависят от выбранного вами типа форматрования. Строка форматирования должна начинаться со знака % и заканчиваться символом типа форматирования:

[%] [тип]

При необходимости, вы также можете добавить:

  • указатель номера аргумента
  • признак выравнивания слева
  • указатель ширины
  • показатель точности

[%] [номер:] [-] [ширина] [.точность] [тип]

Для указания, сколько десятичных знаков использовать в строке форматирования, задайте стандартный формат, который вы будете использовать. Затем добавьте показатель точности - десятичный разделитель и число цифр после него - перед символом формата строки, но после начального %. Например:

  • Тип Поля: Float
  • Тип Формата: ftCurrency.
  • Строка Форматирования: %.4f
  • Данные (Неформатированные): 3.1415926
  • Результат: 3.1416

Вы можете использовать любые из следующих типов форматирования строк:

  • Строки Форматирования Логического Поля
  • Строка Форматирования Полей Валюта и Телефон
  • Строки Форматирования Поля Дата/Время
  • Строки Форматирования Числового Поля
  • Строки Форматирования Целочисленного Поля
