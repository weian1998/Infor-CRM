Использование "Не" в Условии Запроса

Вы можете добавить НЕ перед условием запроса или группы условий запроса. Если перед группой условий запроса вставлен оператор НЕ, он применяется для всех условий запроса внутри группы.

Для добавления НЕ перед условием или группой

  1. Откройте Конструктор Запросов.
  2. Выберите закладку Условия и добавьте все необходимые условия.
  3. При необходимости, сгруппируйте условия запроса.
  4. Установите флажок Не для условия запроса или первого условия в группе.
  5. Нажмите ОК для выполнения запроса.

Пример

Построиель Запросов выберет контакты, которые живут где угодно в Калифорнии, кроме Лос Анджелеса и Сан Франциско.

Так будет выглядеть таблица на закладке Условия для данного примера:

Не ( Поле Оператор Значение Учитывать регистр ) И/Или
НЕ ( Контакт.Адрес.Город (=) равно Лос Анджелес Верно   ИЛИ
    Контакт.Адрес.Город (=) равно Сан Франциско Верно ) И
    Контакт.Адрес.Штат (=) равно CA Верно   КОНЕЦ

Запрос может содержать подгруппы внутри групп. Если НЕ используется перед группой, он применяется для всех входящих в нее подгрупп.
