Тип Форматирования

Вы можете выбрать любой из следующих типов форматирования.

Выберите Для
Число Форматирование поля как десятичного числа.
Целое Форматирование поля только как целого числа; округление в большую или меньшую сторону.
Дата/Время Форматирование даты/времени.
Процент Форматирование поля как процентного отношения. Может быть использовано только для полей с переменной или двойной точностью.
Валюта Форматирование поля как количества денежных единиц.
Имя Пользователя Преобразование поля, содержащего USERID, в имя пользователя.
Телефонный Номер Форматирование значения в поле в стандартный формат телефонного номера.
Доступ Преобразование SECCODE в имя пользователя или название команды.
Логический Преобразует 0, 1, T и F в Истина/Ложь.
Положительное Целое Форматирует поле как положительное целое число.
Элемент Справочника Форматирует поле как элемент справочника Infor CRM.
Часовой Пояс Форматирование поля как часовой пояс.
