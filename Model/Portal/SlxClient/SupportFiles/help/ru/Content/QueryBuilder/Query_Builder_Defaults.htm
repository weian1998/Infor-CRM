Закладка Стандартного Конструктора запросов

Используйте закладку Стандартного Конструктора запросов для выбора только неповторящихся записей.

Неповторяющиеся записи - это записи, которые не совпадают или не дублирующиеся записи.

Построитель Запросов поддерживает много функций. Видимые закладки зависят от функции, которую вы используете.

Для настройки

  1. Откройте Конструктор Запросов.
  2. Выберите закладку Стандартно.
  3. Для того, чтобы устранить дублирующие записи, выберите Выводить только различные строки.
