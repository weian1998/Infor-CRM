Назначение Условий Запроса

При добавлении поля на закладке Конструктора Запросов Условия, для этого поля появляется диалог Назначить Условия.

Когда вы перетащите поле на закладку Конструктора Запросов Условия, для этого поля появляется диалог Назначить Условия.

Чтобы назначить

  1. Откройте Конструктор Запросов.
  2. Выберите закладку Условия.
  3. Добавьте условие.

Откроется диалог Назначить Условие.

  1. Выберите Оператор.
  2. В поле Значение введите значение или нажмите Просмотр для выбора значения.

Если вы выбрали в качестве оператора "В" значения должны быть заключены в кавычки для каждой записи.

  1. Выберите Сравнение с учетом регистра для использования заглавных букв в качестве условия.

Если вы используете MS SQL Server, Сравнение с учетом регистра невозможно. Все запросы нечувствительны к регистру.

  1. При необходимости, выберите Использовать значение в качестве константы.
  2. Нажмите ОК.
