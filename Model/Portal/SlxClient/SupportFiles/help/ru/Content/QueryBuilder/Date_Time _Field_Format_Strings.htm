Строки Форматирования Поля Дата/Время

Приведенная ниже таблица описывает параметры строки форматирования для полей дата/время, но не разделители даты/времени.

Параметр Описание
c Показывает дату используя формат, заданный глобальной переменной ShortDateFormat, и, соответственно, время используя формат заданный глобальной переменной LongTimeFormat. Время не отображается, если дробная часть значения DateTime равна нулю.
d Показывает день как число без начального нуля (1-31).
dd Показывает день как число с начальным нулем (01-31).
ddd Показывает сокращенное название дня (Пн-Вс) используя заданные в глобальной переменной ShortDayNames строки.
dddd Показывает полное название дня (Понедельник-Воскресение) используя заданные в глобальной переменной LongDayNames строки.
ddddd Показывает дату используя формат, определенный в глобальной переменной ShortDateFormat.
m Показывает месяц как число без начального нуля (1-12). Если параметр m следует сразу за параметром h или hh, то показываются минуты, а не месяцы.
mm Показывает месяц как число с начальным нулем (01-12). Если параметр mm следует сразу за параметром h или hh, то показываются минуты, а не месяцы.
mmm Показывает сокращенное название месяца (Янв-Дек) используя заданные в глобальной переменной ShortMonthNames строки.
mmmm Показывает полное название месяца (Январь-Декабрь) используя заданные в глобальной переменной LongMonthNames строки.
yy Показывает год в виде числа из двух цифр (00-99).
yyyy Показывает год в виде числа из четырех цифр (0000-9999).
h Показывает час без начального нуля (0-23).
hh Показывает час с начальным нулем (00-23).
n Показывает минуты без начального нуля (0-59).
nn Показывает минуты с начальным нулем (00-59).
s Показывает секунды без начального нуля (0-59).
ss Показывает секунды с начальным нулем (00-59).
t Показывает время используя формат, определенный в глобальной переменной ShortTimeFormat.
tt Показывает время используя формат, определенный в глобальной переменной LongTimeFormat.
am/pm Используется 12-часовой режим для предыдущих параметров h и hh, и отображает 'am' для любого часа до полудня, и 'pm' для любого часа после полудня. Параметр am/pm может использовать нижний, верхний или смешанный регистр, и результат будет отображаться соответственно.
a/p Используется 12-часовой режим для предыдущих параметров h и hh, и отображает 'a' для любого часа до полудня, и 'p' для любого часа после полудня. Параметр a/p может использовать нижний, верхний или смешанный регистр, и результат будет отображаться соответственно.
ampm Используется 12-часовой режим для предыдущих параметров h и hh, и отображается содержимое глобальной переменной TimeAMString для любого часа до полудня, и содержимое глобальной переменной TimePMString для любого часа после полудня.
