Что такое Конструктор Запросов?

Конструктор запросов позволяет вам выбирать данные из базы на основании одного или более условий. Например, вам необходимо посмотреть список основных контактов в банковской индустрии. Конструктор Запросов связан с различными областями Infor CRM. Для эффективного использования Конструктора запросов, вы должны понимать структуру базы данных Infor CRM и концепцию реляционных баз данных, например, отношение данных и объединения.
