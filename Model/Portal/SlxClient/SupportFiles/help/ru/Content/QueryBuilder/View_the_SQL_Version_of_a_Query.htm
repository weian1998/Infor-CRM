Просмотр SQL запроса

Когда вы создаете запрос, он преобразуется в SQL (Structured Query Language). Этот оператор выполняется в базе данных. Можно посмотреть SQL запрос, но вы не можете его изменить.

Просмотр SQL операторов для запроса может быть полезен для понимания, как работает запрос. Возможность просмотра SQL запроса может быть полезной также в случае, если вы захотите написать запрос для другой части программы и хотите видеть пример кода SQL запроса.

Для просмотра SQL запроса

  1. Откройте Конструктор Запросов.
  2. Нажмите Просмотр SQL.
  3. После просмотра SQL кода закройте диалоговое окно.
