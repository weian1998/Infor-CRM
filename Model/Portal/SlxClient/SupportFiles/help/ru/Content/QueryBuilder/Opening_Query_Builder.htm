Открытие Конструктора запросов

Вы можете использовать Конструктор Запросов для добавления или изменения ваших групп.

Чтобы открыть Конструктор запросов

  • Выполните одно из следующих действий:
  • В соответствующем списке в меню Группы выберите Показать Группы.
  • Правой кнопкой мыши кликните по группе, которую вы хотите изменить, затем нажмите Редактировать.
