Редактирование компонентов дела по заявке

Используйте это представление для управления компонентами дела по заявке.

Чтобы изменить

  1. Выполните одно из следующих действий:
    • Откройте закладку компоненты дела по заявке и нажмите Изменить напротив нужной записи.
    • Откройте представление дела по заявке, в области Компоненты дела по заявке, нажмите Изменить напротив нужной записи.
      • Откроется диалог редактирования компонента дела по заявке.
  2. В поле Серийный номер укажите серийный номер, идентифицирующий компонент.
  3. В поле Цена укажите цену компонента.
  4. В поле Описание введите краткое описание компонента.
  5. Нажмите ОК.
