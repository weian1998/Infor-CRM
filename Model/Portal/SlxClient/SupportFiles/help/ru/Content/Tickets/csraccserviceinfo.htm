Информация по обслуживанию субъекта

Диалог Информация по обслуживанию субъекта предоставляет информацию о субъекте, заявке, контакте, продукте и контракте для того, чтобы помочь вам разобраться в отношениях с субъектом, сделавшем запрос. Вы также можете использовать этот диалог для определения типа помощи, которую вы можете оказать в соответствии с контрактом на обслуживание субъекта.

Чтобы открыть диалог Информация по обслуживанию субъекта

  • На детальном представлении Заявки нажмите кнопку Информация по обслуживанию субъекта.

Чтобы использовать Информацию по обслуживанию субъекта при добавлении новой заявки

  1. В меню Создать выберите Заявка.
  2. В представлении Добавить новую заявку, на панели инструментов нажмите кнопку Информация по обслуживанию субъекта, для открытия представления служебная информация по субъекту.
  3. Если в представлении служебная информация по субъекту в поле Субъект не появилось название субъекта, используйте поиск для нахождения нужного субъекта.

После выбора субъекта вся информация об обслуживании появится на карточке.

  1. В разделе Контакт, выберите контакта уполномоченного сообщать о проблеме.

Если контакт имеет право обращаться в сервисную службу, 'Да' появляется в колонке Уполномочен.

  1. В разделе Контракты выберите контракт, на который будет подана заявка.

Если вы не уверены, какой контракт выбрать

  • Убедитесь, что в колонке Активный, стоит 'Да'.
  • Выберите контракт из списка, комментарии по контракту отобразятся в поле Комментарии.

Если контракт не отображается, вам нужно добавить контракт к субъекту.
  1. Нажмите OK для сохранения вашего выбора и закрытия представления Информация по обслуживанию субъекта.
  2. Следуйте инструкции по добавлению заявки. Информация о контакте и контракте, выбранная вами в представлении Информация по обслуживанию субъекта, появится на карточке Добавить Новую Заявку.
