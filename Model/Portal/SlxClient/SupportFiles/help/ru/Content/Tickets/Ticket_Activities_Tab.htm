Использование закладки дела заявки

Закладка дела заявки отображает все дела связанные с работой и решением заявки. Используйте закладку дела заявки для добавления нового дела по проделанной работе по заявке или для изменения/обновления существующих дел. Вы можете назначать будущие (последующие) дела для себя или другого пользователя. Вы можете нажать на заголовок столбца для сортировки списка по выбранному столбцу. По умолчанию, самое последнее дело (по дате и времени) отображается в начале списка.

Как выполнить?

Открыть закладку дела заявки
  1. Откройте детальное представление заявки.
  2. Откройте закладку Дела заявки. Если закладка не отображается, то откройте закладку Все закладки.

Добавить дело по заявке

Просмотреть комментарии дел по заявке
  • Выберите запись в таблице. Комментарии отобразятся в поле таблицы.
