Добавление и редактирование дел по заявке

Используйте данное представление для добавления дел по проделанной работе по заявке или редактирования/обновления существующих дел. Дела заявки позволяют вам отслеживать время, типы задач и другие затраты связанные с решением заявки. Дела учитываются по типу дел и тарифу и используются для выставления счета клиенту за их контракты по обслуживанию.Вы можете назначать будущие (последующие) дела для себя или другого пользователя.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Как выполнить?

Открыть представление дела заявки
  1. Откройте детальное представление заявки.
  2. Откройте закладку Дела заявки. Если закладка не отображается, то откройте закладку Все закладки.
Добавить комментарий по делу к заявке
  1. Используйте поле комментарии, для добавления дополнительной информации.
  2. Нажмите Сохранить.
Добавить подробности по делу заявки и компонентам используя расширенное представление
  1. На закладке Дела заявки, нажмите Расширенное.
    Откроется представление создания дела по заявке.
  2. Заполните информационные поля дела по заявке.
  3. Чтобы добавить компонент дела, нажмите кнопку Добавить компонент в области Компоненты дела по заявке.
  4. Найдите компонент, используя поиск и нажмите Добавить выбранные.
  5. Нажмите Сохранить.

Изменить детали дела по заявке

  1. В списке дел, нажмите Изменить напротив дела, которое необходимо изменить.
  2. Измените информационные поля дела по заявке.

    Некоторые информационные поля не могут быть изменены пользователем.

  3. Нажмите Сохранить.
