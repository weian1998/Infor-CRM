Отправка E-mail из заявки

Вы можете отправить сообщение электронной клиенту или другому пользователю Infor CRM в процессе создания заявки или из детального представления заявки. Электронное письмо может содержать информацию о заявке, для информирования других пользователей, включая клиента о том, что вы открыли заявку или для того, чтобы задать вопрос для помощи в решении проблемы.

Чтобы отправить e-mail из заявки

  1. На навигационной панели нажмите Заявки.
  1. В списочном представлении Заявок, воспользуйтесь поиском для нахождения необходимой заявки, затем откройте детальное представление заявки.
  2. В списочном представлении Заявки, на панели инструментов нажмите кнопку Детали E-mail.
  3. В диалоге Отправить E-mail о заявке, в поле Кому, выберите одного или нескольких адресатов.
  4. В поле Тип E-mail, выберите один из типов данных для отправки. Доступные опции:
    5. Опция Действие
    Информация о заявке Сообщение включает в себя дату и время получения, описание, решение (если имеется), любые внутренние комментарии и идентификационный номер заявки.
    Нет Идентификационный номер заявки заполняет строку тема сообщения, никакой другой информации о заявке в сообщение не добавляется.

Внутренние комментарии доступны только другим пользователям Infor CRM. Если один из получателей письма является контактом, тогда Внутренние Комментарии не будут включаться в e-mail.
  1. Нажмите OK.
  1. Отправьте сообщение используя выбранную вами систему отправки e-mail.
  • Нажмите Отправить. Сообщение не будет записано в историю.
  • При использовании Microsoft Outlook нажмите Отправить в SLX. IЕсли вы прикрепили файл и хотите сохранить вложение, нажмите Да. e-mail сообщение будет записано в историю и файл отобразится на закладке Вложения.
