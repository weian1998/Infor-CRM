Информационные поля заявки

В таблице ниже описаны основные поля.

Поле Описание/Действие
Субъект Субъект, к которому относится заявка. Чтобы сменить субъекта, нажмите кнопку Найти, найдите соответствующего субъекта и нажмите OK.
Контакт Контакт, к которому относится заявка. Чтобы сменить контакта, нажмите кнопку Найти, найдите соответствующего контакта и нажмите OK.
Телефон Номер телефона контакта.
E-mail Адрес электронной почты контакта.
Контракт Выбранный при создании заявки контракт. Чтобы сменить контракт, нажмите кнопку Найти, найдите соответствующий контракт и нажмите OK.
Область - Категория - Проблема

Эти три поля кратко описывают суть проблемы. Нажмите на одну из кнопок Найти, расположенных в каждом из полей, выберите соответствующие значения из списков и нажмите OK.

Набор элементов списка Категория зависит от элементов, выбранных для Области, а набор элементов списка Проблема зависит от элементов, выбранных для Категории.
Источник Содержит список возможных способов отправки заявки. Нажмите и выберите элемент из списка, затем нажмите OK.
Статус По умолчанию указывается статус Открыта. Для выбора другого статуса нажмите и выберите элемент из списка, затем нажмите OK.
Срочность Показывает важность проблемы. Для изменения срочности, из выпадающего списка выберите значение, наилучшим образом описывающее срочность заявки клиента.
Срок исполнения Дата, к которой заявка должна быть разрешена. Дата по умолчанию зависит от выбранной срочности. Для изменения нажмите кнопку Календарь для выбора даты.  
Дата поручения Время и дата назначения заявки для текущего назначенного пользователя. Для изменения нажмите кнопку Календарь для выбора даты.  
Поручено Пользователь службы поддержки, назначенный на область, выбранную для этой заявки. Для переназначения заявки, нажмите кнопку Найти , найдите соответствующего пользователя и нажмите OK
Решено при первом обращении? Установите этот флаг, если вы можете сразу же решить проблему обратившегося, не перезванивая ему или не заставляя его перезванивать вам.
Передать для быстрого поиска

Установите этот флаг, если вы хотите чтобы информация о заявки была доступна для поиска другими пользователями.

В зависимости от того, как ваш администратор настроил быстрый поиск, выбор этой опции может автоматически позволить информации стать доступной, или же информация требует подтверждения для использования в быстром поиске другим пользователем.
Утверждено для быстрого поиска Установите этот флаг для подтверждения того, что заявка была отправлена в быстрый поиск. Эта опция доступна если ваш администратор назначил вас лицом, уполномоченным подтверждать заявки для быстрого поиска.
Общий доступ Нажмите кнопку для выбора опции общего доступа для этой заявки. По умолчанию общий доступ устанавливается вашим администратором. Настройки общего доступа определяют, какие пользователи (внутренние или внешние) могут просматривать определенные группы заявок (например, только новые заявки, или все заявки и т. д.). Эта опция доступна если ваш администратор назначил вас лицом, уполномоченным подтверждать заявки для быстрого поиска.
