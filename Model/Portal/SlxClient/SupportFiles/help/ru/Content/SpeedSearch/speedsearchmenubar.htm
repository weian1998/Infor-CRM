Использование быстрого поиска из панели меню

Теперь вы можете вводить условия поиска и осуществлять поиск из панели меню.

Чтобы осуществить быстрый поиск из панели меню

  • На панели меню, введите условия поиска в поле Быстрый поиск, затем нажмите кнопку Быстрый поиск.
    Откроется быстрый поиск с результатами поиска, вы также можете использовать расширенные настройки.
