Управление ресурсами

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Можно добавить ресурс, например, конференц-кабинет, к списку доступных ресурсов, который появляется при планировании дела.

Как выполнить?

Добавить ресурс
  1. В меню Администрирование, нажмите Новый ресурс.
  2. Заполните информационные поля ресурса.
  1. Нажмите Сохранить.
Изменить ресурс
  1. На навигационной панели нажмите Администрирование, затем нажмите Ресурсы для открытия списочного представления ресурсов. При необходимости, воспользуйтесь поиском для нахождения нужного ресурса, затем нажмите на имя ресурса.
  2. При необходимости, измените информационные поля ресурса.
  3. Нажмите Сохранить.
Удалить ресурс
  1. На навигационной панели нажмите Администрирование, затем нажмите Ресурсы для открытия списочного представления ресурсов. При необходимости, воспользуйтесь поиском для нахождения нужного ресурса, затем нажмите на имя ресурса.
  2. На детальном представлении Ресурса, нажмите кнопку Удалить.

  3. Нажмите OK.
