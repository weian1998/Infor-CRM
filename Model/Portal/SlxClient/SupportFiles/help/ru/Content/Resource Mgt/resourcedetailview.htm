Использование детального представления Ресурсов

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление используется для просмотра, добавления или редактирования информации по выбранной записи.

Как выполнить?

Открыть детальное представление ресурса
  • На навигационной панели нажмите Администрирование, затем нажмите Ресурсы для открытия списочного представления ресурсов. При необходимости, воспользуйтесь поиском для нахождения нужного ресурса, затем нажмите на имя ресурса.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать разделы Записи группы и Обычное задачи панели задач

Просмотреть группу

Информационные поля

Информационные поля ресурса содержат основную информацию по ресурсу. Пользователи с соответствующими правами могут редактировать эти поля.
