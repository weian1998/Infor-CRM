Добавление Глобальных Связей

 Только для Системного Администратора

Глобальная связь является постоянной и существует для каждого сеанса в структуре дерева данных Конструктора Запросов. Будьте очень осторожны при создании глобальных связей. Вы можете случайно создать глобальную связь, которая создаст значительные проблемы при работе.

Для того чтобы добавить

  1. Откройте Построитель Запросов, а затем нажмите кнопку Глобальные Связи.

Эта кнопка появляется только в том случае, если вы вошли как Администратор.

  1. В диалоге Менеджер Глобальных связей нажмите Добавить.
  2. В диалоге Добавление глобальной связи под меткой Родительская таблица и поле связи, выберите таблицу и поле для глобального объединения.
  3. Под меткой Дочерняя таблица и поле связи, выберите таблицу и поле, чтобы связать с родительской таблицей и полем.
  4. Под меткой Тип объединения, выберите тип объединения.

По умолчанию Левое.

  1. Под меткой Опции объединения:
  1. Выберите параметр Видимость, чтобы указать, как объединение отображается в Конструкторе запросов.

По умолчанию Разрешить.

  1. Выберите Тип Каскада, чтобы определить, как записи будут удаляться через объединение.

По умолчанию Игнорировать (Не использовать в каскаде).

  1. Нажмите ОК.
