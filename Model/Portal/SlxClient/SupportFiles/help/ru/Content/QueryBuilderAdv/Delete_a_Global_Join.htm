Удаление Глобальных Связей

 Только для Системного Администратора

Глобальная связь является постоянной и существует для каждого сеанса в структуре дерева данных Конструктора Запросов. Вы не можете удалить предопределенные глобальные связи (системные связи).

Для удаления глобальной связи

  1. Откройте Построитель Запросов, а затем нажмите кнопку Глобальные Связи.

Эта кнопка появляется только в том случае, если вы вошли как Администратор.

  1. Выберите связь, которую вы хотите удалить.
  2. Нажмите Удалить.
  3. Нажмите ОК.
