Использование Значения как Константы

Выбор параметра Использовать Значение как Константу в диалоге Назначить Условие помещает текст в запрос в точности так, как он отображается в поле Значение. В большинстве случаев вам не следует выбирать этот параметр, потому что Конструктор Запросов может определить правильный SQL запрос. Однако, в некоторых случаях, продвинутые пользователи могут захотеть ввести определенный текст, который следует записать непосредственно в SQL запрос.

Например, можно ввести системную функцию, специальную для базы данных (такую как GETDATE в Microsoft SQL Server database) и выбрать Использовать Значение как Константу. GETDATE включается в SQL запрос без кавычек.
