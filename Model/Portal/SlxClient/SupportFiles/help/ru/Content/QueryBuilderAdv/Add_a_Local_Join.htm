Добавление Локальных Связей

Локальная связь является временной и существует только в Конструкторе Запросов для одного сеанса. Объект, использующий полученный запрос будет сохранять свойства этой локальной связи до тех пор, пока он используется.

Для добавления

  1. Откройте Конструктор Запросов.
  2. Выберите таблицу.
  3. Выберите поле.
  4. Щелкните ссылку Создать Локальную Связь.

В диалоге Локальная Связь, выбранные в Конструкторе Запросов таблица и поле появляются в разделе Родительская Таблица и Поле Связи.

  1. Под меткой Дочерняя таблица и поле связи, выберите таблицу базы данных и поле, с которой вы хотите создать локальную связь.
  2. В Тип Связи, выберите тип связи.

По умолчанию Внутренняя.

  1. Нажмите ОК.
