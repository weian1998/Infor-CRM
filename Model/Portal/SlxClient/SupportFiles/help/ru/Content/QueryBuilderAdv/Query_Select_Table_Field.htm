Выбор Таблиц и Полей

Вы можете добавить поле для отображения в запросе.

Для выбора таблицы

  • Выберите таблицу в левой части Конструктора Запросов. При необходимости, нажмите знак плюс для раскрытия списка таблиц.

Чтобы выбрать поле

  • Выберите поле в правой части Конструктора Запросов. Здесь отображены все поля, относящиеся к выделенной таблице. Дважды щелкните на поле для добавления его в таблицу внизу диалога.
