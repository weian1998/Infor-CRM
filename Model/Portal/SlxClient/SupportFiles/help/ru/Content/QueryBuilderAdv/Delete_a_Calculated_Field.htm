Удаление вычисляемых полей

 Только для Системного Администратора

Вы можете открыть диалоговое окно Управление вычисляемыми полями, если зашли в систему под учетной записью администратора.

Для удаления вычисляемого поля

  1. Откройте Построитель запросов, нажмите на кнопку Вычисления.

Эта кнопка появляется только в том случае, если вы вошли как Администратор.

  1. В диалоговом окне Управление вычисляемыми полямивыберите вычисляемое поле из таблицы.
  2. Нажмите Удалить.
  3. Нажмите ОК для подтверждения удаления.
  4. Нажмите Закрыть.
