Рекомендации по Использованию Объединений

Используйте следующие рекомендации при создании связей таблиц в Infor CRM:

  • На карточке, которая отображает поля нескольких таблиц, используйте объединения Слева. Если вы используете Внутренние объединения, и в одной из таблиц нет данных, то в КАЖДОМ поле таблице отобразится НЕТ ДАННЫХ.
  • В группе используйте Внутренние объединение таблиц, используемых в Условиях. Таким образом, вы получите только те записи, которые удовлетворяют вашим критериям. Однако, вы можете захотеть использовать объединения Слева для таблиц, используемых в Макете, для того чтобы все удовлетворяющие Условиям записи учитывались, даже если в одной из таблиц нет данных..

По умолчанию, таблица АДРЕС соединена с таблицей СУБЪЕКТ через Внутреннее объединение. Не изменяйте параметры этого объединения.
