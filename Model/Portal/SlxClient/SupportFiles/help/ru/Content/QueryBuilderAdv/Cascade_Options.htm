Параметры Каскадной Обработки

Выберите Чтобы
Удалить

Удалить выбранную запись и все зависимые записи.

Например, если пользователь удален из таблицы USERINFO, то связанные записи в таблице UERPROFILE также будут удалены.
Очистить

Очищается ключевое поле в дочерней таблице связывающее с записью в родительской, но запись в дочерней таблице остается.

Например, если вы удалите запись КОНТАКТ, записи в таблице ИСТОРИЯ, связанные с этим контактом не будут удалены, а будет удалена только ссылка на контакт.
Заменить

При удалении записи, очищается ключевое поле связывающее запись в дочерней таблице, но не сама запись. Заменить ключевое поле другим значением.

Например, если пользователь удален из таблицы USERINFO, замена может быть использована для назначения субъектов этого пользователя кому-либо другому.
Удалить (Остановить каскадную обработку)

Удалить выбранную запись, но не удалять зависимые записи.

Например, если пользователь удаляется из таблицы USERINFO, связанные записи HISTORY не изменяются.
Игнорировать (Не использовать каскад) При удалении записи пропустить эту таблицу, будто она не существует.
