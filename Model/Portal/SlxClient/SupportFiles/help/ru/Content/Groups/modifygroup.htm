Изменение групп

Вы можете использовать Конструктор запросов для изменения ваших групп.

Чтобы изменить группу

  1. Откройте списочное или детальное представление.
  2. Щелкните правой кнопкой по закладке группы.
  3. Нажмите Изменить.

Откроется Конструктор запросов.

  1. Используйте закладки Конструктора запросов, для измениня группы.
  • Закладка свойства - Используйте для настройки названия и описания группы.
  • Закладка условия - Используйте для настройки условий, используемых при выборе записей. Данная закладка не доступна для специальных групп.
  • Закладка макет – Используйте для настройки данных, которые отобразятся в списочном представлении.
  • Закладка сортировка – Используйте для настройки сортировки данных.
  1. По завершению, нажмите OK.
