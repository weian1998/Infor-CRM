Экспорт группы в Excel

Вы можете экспортировать группы в Excel. Заголовки столбцов и форматирования также будут экспортированы.

Чтобы экспортировать группу в файл

  1. Откройте группу или используйте результаты поиска, содержащие записи, которые вы хотите экспортировать.
  2. Щелкните правой кнопкой по закладке группы и выберите пункт меню Экспортировать в Excel.,
  3. Когда экспорт завешен, откроется Откроется одно из следующих расположений:

    • Страница уведомлений о заданиях

    • Отчеты закладка списочного представления истории

    • Сообщение о завершении работы

    4. Microsoft Excel с экспортированной группой. Если группа содержит информацию по валюте, Excel также создаст таблицу сводную таблицу. Excel добавит строку суммы для каждой колонки, содержащей валюту.

  • Экспорт группы в Excel перенесет всю группу, несмотря на примененные фильтры.
  • Может возникнуть ошибка, если вы попытаетесь экспортировать группу, которая включает поля с большим объемом данных.
