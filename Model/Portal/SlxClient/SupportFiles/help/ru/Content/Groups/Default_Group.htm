Установка группы по умолчанию

Вы можете установить группу по умолчанию для каждого списочного представления. Группа по умолчанию - это группа, которая будет открыта при первом открытии списочного представления. Группа по умолчанию, также определяет макет для новых групп. Группа по умолчанию может быть только одна для каждого списочного представления.

Чтобы установить группу по умолчанию

  • Выполните одно из следующих действий:

    • В списочном представлении, щелкните правой кнопкой по закладке группы, которую вы хотите сделать группой по умолчанию и выберите пункт меню По умолчанию.
    • В меню Инструменты, нажмите Параметры, затем выберите закладку Группы. Используйте списки Основной вид и Группа по умолчанию, чтобы установить группу по умолчанию для каждого основного представления.
