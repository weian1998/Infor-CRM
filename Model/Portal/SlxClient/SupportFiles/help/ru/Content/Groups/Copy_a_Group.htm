Копирование группы

Вы можете выбрать существующую группу и скопировать ее. Вы можете сохранить существующие условия группы или изменить условия в соответствии с вашими потребностями. Вы не можете копировать специальные группы.

Чтобы скопировать существующую группу

  1. В списочном представлении, щелкните правой кнопкой по закладке группы.
  2. Нажмите Копировать.
    Откроется Конструктор запросов.
  3. Используйте закладки Конструктора запросов, для изменения группы.
    • Закладка свойства - Используйте для настройки названия и описания группы.
    • Закладка – Используйте для настройки условий, используемых при выборе записей. Данная закладка недоступна для специальных групп.
    • Закладка макет – Используйте для настройки данных, которые отобразятся в списочном представлении.

      Макет по умолчанию определяется группой, которую вы выбрали группой по умолчанию в настойках групп.

    • Закладка сортировка – Используйте для настройки сортировки данных.
  4. По завершению, нажмите OK
