Добавление группы на основе расширенного запроса

Используйте Конструктор запросов для создания групп. Расширенные запросы позволяют вам создавать условия для группы. Каждый раз, когда вы открываете группу, отрабатывается запрос к базе данных, для определения того, какие записи необходимо отобразить. Если вы хотите, чтобы группа осталась неизменной, создайте специальную группу.

 

Чтобы добавить группу на основе расширенного запроса

  1. Откройте списочное или детальное представление.
  2. Нажмите кнопку Добавить группу.
    Откроется Конструктор запросов Infor.
  3. Используйте закладку условия для настройки условий, используемых при выборе записей.
  4. При необходимости, используйте следующие закладки Конструктора запросов:
    • Закладка свойства - Используйте для настройки названия и описания группы.
    • Закладка макет – Используйте для настройки данных, которые отобразятся в списочном представлении.

      Макет по умолчанию определяется группой, которую вы выбрали группой по умолчанию в настойках групп.

    • Закладка сортировка – Используйте для настройки сортировки данных.
  5. По завершению, нажмите OK.
