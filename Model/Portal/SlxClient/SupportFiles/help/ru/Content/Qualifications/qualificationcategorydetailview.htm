Использование детального представления Квалификации

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте детальное представление для просмотра, добавления или редактирования информации по выбранной записи.

Как выполнить?

Открыть детальное представление квалификации
  • На навигационной панели нажмите Администрирование, затем нажмите Квалификации для открытия списочного представления квалификаций. При необходимости, воспользуйтесь поиском для нахождения необходимой квалификации, затем нажмите на название квалификации.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать разделы Записи группы и Обычное задачи панели задач

Просмотреть группу

Информационные поля

Поле описание предоставляет краткое описание квалификации.

Закладки Детального представления

Закладка Элементы отображает все элементы, связанные с квалификацией. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

  • Элементы

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
