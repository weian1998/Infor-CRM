Управление квалификациями

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Квалификации - это набор элементов квалификации, которые используются для проверки информации о наводках или определения, когда наводка готова перейти в сделку.

Как выполнить?

Добавить квалификацию
  1. В меню Администрирование, нажмите Новая квалификация.
  2. В поле Квалификация для введите описание.
  3. Нажмите Сохранить.
  4. На детальном представлении Квалификации используйте закладку Элементы для добавления элементов квалификации.
Редактировать квалификацию
  1. Откройте детальное представление квалификации.
  2. В поле Квалификация для введите описание.
  3. Нажмите Сохранить.
Удалить квалификацию
  1. На навигационной панели нажмите Администрирование, затем нажмите Квалификации для открытия списочного представления квалификаций. При необходимости, воспользуйтесь поиском для нахождения необходимой квалификации, затем нажмите на название квалификации.
  2. На детальном представлении Квалификации, нажмите кнопку Удалить.

  3. Нажмите OK.

При удалении квалификации вы также удаляете все элементы квалификации.
