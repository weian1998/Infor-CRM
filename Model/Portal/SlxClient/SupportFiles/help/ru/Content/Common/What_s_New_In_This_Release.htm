Что нового в этой версии

Infor CRM Веб-клиент включает в себя следующие обновления:

Версия 8.1.0.06

  • Внесены изменения, чтобы отразить ребрендинг продукта.

  • Теперь для Дел разрешено добавлять несколько Контактов или Наводок, используя закладку Участники.
  • Добавлена возможность связать Контакт с пользователем Infor.
  • Синхронизация с Outlook заменила Intellisync для синхронизации контактов и дел с Microsoft Outlook.
  • Новое списочное представление Истории Синхронизации позволяет пользователям просматривать историю синхронизации, в том числе историю синхронизации с Outlook.
  • Добавлена возможность составить сообщение электронной почты в Outlook и выбрать контакты из адресной книги Infor.
  • Новые виджеты для веб-сайтов, лент новостей, и SData.
  • Отчетность теперь использует менеджер проектов.
  • Экспорт в Excel теперь использует менеджер проектов.
  • Для администраторов и ролей, обладающих соответствующим уровнем доступа, стали доступны:
    • Менеджер Проектов;
    • Интеграция;
    • Управление синхронизацией SData;
    • Возможность связать пользователя с контактом.
