Передача личного сообщения

При передаче личного шаблона другому пользователю, рекомендуется сначала скопировать шаблон, затем передать скопированный шаблон другому пользователю. Таким образом, личный шаблон останется в вашем списке.
  1. В меню Написать выберите Шаблоны.
  2. В списке Личные шаблоны выберите папку Письмо, Факс или E-mail.
  3. Правой кнопкой мыши щелкните по шаблону, который вы хотите передать другому пользователю, и из меню выберите Свойства.
  4. В диалоговом окне Свойства, в поле Доступ, нажмите поиск для выбора пользователя из списка. Кликните два раза по имени пользователя.
  5. Нажмите ОК. Шаблон появится в папке личных шаблонов пользователя, выбранного в шаге 3, и будет удален из списка личных шаблонов текущего пользователя.
