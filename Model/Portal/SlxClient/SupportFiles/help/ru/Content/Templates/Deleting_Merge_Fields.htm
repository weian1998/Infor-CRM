Удаление полей для рассылки

Можно удалить поля для рассылки из шаблонов из Редактора шаблонов E-mail, письма или факса.

Чтобы удалить поле для рассылки

  1. Откройте Редактор шаблонов.
  2. Нажмите Изменить поля на панели инструментов.
    Открывается диалог Редактор полей для почтовой рассылки Infor CRM.
  3. На панели инструментов, нажмите Удалить из шаблона.
  4. Нажмите Да для удаления поля.
  5. Нажмите OK чтобы закрыть Редактор полей для почтовой рассылки Infor CRM.
