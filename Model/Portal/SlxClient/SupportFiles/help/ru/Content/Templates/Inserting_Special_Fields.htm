Вставка специальных полей

Вставка специальных полей доступна только для опытных пользователей

Можно нажать кнопку Вставить специальное поле в Редакторе E-mail, Факса, или Письма, чтобы вставить новую таблицу продуктов по сделке, SQL запрос или поле почтовой рассылки.

Чтобы вставить специальное поле

  1. Откройте Редактор шаблонов.
  2. На панели инструментов нажмите Вставить специальное поле.
  3. Из меню, выберите тип поля, который вы хотите добавить в шаблон.
    • Таблица продуктов по сделке

    Добавляет таблицу с продуктами по сделкам и информацией о ценах.

    • SQL

      • Вы можете вставить следующие поля SQL:

      • Текст
      • Изображение
      • Таблица

      • Список

      • Нумерованный список
    • Пользовательские

      • Если пользовательское поле добавляется к шаблону и скрипты не были корректно модифицированы, почтовая рассылка не начнется до тех пор пока скрипты не будут исправлены, или кастомное поле не будет удалено.

    Перед тем, как вставить пользовательское поле, оно должно быть проработано с помощью кода в SLX Mail Merge OnCustomFieldName script (LAN) или class function Sage.MailMergeService.prototype.HandleOnCustomFieldName, находящемся в jscript\sage-mailmerge\sage-mailmerge-service.js.

    Открывается диалог Редактор полей для почтовой рассылки Infor CRM.

  4. При необходимости, используйте поле изменить, чтобы произвести изменения, затем нажмите Принять.
  5. Нажмите ОК.
