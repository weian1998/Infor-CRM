Удаление шаблона

Вы можете удалить только личные шаблоны, принадлежащие вам. Если шаблон был передан другим и им пользуются, удалить его может администратор Infor CRM.

Чтобы удалить шаблон

  1. В меню Написать выберите Шаблоны.
  2. Откройте Управление шаблонами или диалоговое окно Выбор шаблона .
    • В меню Написать выберите Шаблоны.
  3. В диалоговом окне Управление шаблонамивыберите шаблон из списка Личные шаблоны, а затем нажмите Удалить.
  4. Нажмите OK, чтобы закрыть окно.
  5. В диалоговом окне Управление шаблонами выберите одно из следующих действий:
    • Выберите другой личный шаблон для удаления.
    • Нажмите Закрыть для закрытия диалогового окна.
