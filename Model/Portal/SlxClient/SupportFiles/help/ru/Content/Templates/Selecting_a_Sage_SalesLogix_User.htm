Выбор пользователя Infor CRM

Диалог Выбор пользователя используется при:

  • Создании или копировании шаблона, когда вам нужно установить доступ к шаблону

  • Установке параметров для почтовой рассылки, когда вам необходимо выбрать пользователя для последующей активности.

Как выполнить?

Установить доступ к шаблону
  1. Создайте или скопируйте шаблон.
  2. В диалоге свойства шаблона, в поле Доступ нажмите Поиск.
  3. В диалоге Выбрать пользователя, выполните одно из следующих действий для установки доступа:
    • Выберите пользователя из списка, нажмите OK.
    • Кликните два раза по имени пользователя.
Выбрать пользователя для последующей активности
  1. В меню Написать выберите Почтовая рассылка.
  2. Выберите шаблон для рассылки.
  3. На закладке Параметры истории, в области Планировать последующую активность, заполните информацию для организации последующей активности.
  4. В поле Ведущий нажмите Поиск.
  5. В диалоге Выбрать пользователя, выполните одно из следующих действий для установки доступа:
    • Выберите пользователя из списка, нажмите OK.
    • Кликните два раза по имени пользователя.
