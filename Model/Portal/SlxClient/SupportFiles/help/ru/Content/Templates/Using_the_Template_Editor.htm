Использование Редактора Шаблонов

С помощью Редактора шаблонов можно создавать новый шаблон или редактировать существующий шаблон. Редактор содержит Панель инструментов Infor CRMr, которая дает доступ к специальным функциям наряду с функциями Microsoft Word. Доступные функции зависят от типа шаблона: Письмо, E-mail или Факс.

Чтобы помочь распознать кнопки на панели инструментов, Infor CRM имеет подсказки для определения каждого инструмента. Чтобы просмотреть подсказку, наведите курсор на инструмент.

Поскольку редактор имеет встроенный Microsoft Word, панели инструментов Word также видны. Нажмите кнопку Показать меню Word, чтобы увидеть дополнительные инструменты Microsoft Word, которые не видны в настоящий момент.

Чтобы открыть редактор шаблонов

  1. В меню Написать выберите Шаблоны.

  2. Выполните одно из следующих действий:

    • Выберите личный шаблон, нажмите кнопкуИзменить.
    • Нажмите кнопку Новыйи установите свойства шаблона и затем нажмите OK.

Чтобы вставить поля для рассылки в шаблон

  1. Поместите курсор туда, где вы хотите, чтобы находились поля для рассылки.
  2. Нажмите Вставить поле SLX.
    Откроется меню с наиболее часто используемыми полями для рассылки.
  3. Выберите поле.
  4. Нажмите Все поля SLX чтобы получить доступ к дополнительным полям для рассылки.
  5. Чтобы просмотреть шаблон, на панели инструментов нажмите Предварительный просмотр.
    Это отправит информацию из текущего контакта в шаблон.
    Нельзя редактировать или сохранить шаблон из окна предварительного просмотра. Закройте окно просмотра после окончания просмотра.
  6. На панели инструментов, нажмите Сохранить или Сохранить и закрыть.
    Шаблон сохраняется в базе данных. Если выбрано Сохранить и закрыть, редактор закрывается. Диалог Почтовая рассылка открывается с текущим именем шаблона в поле Шаблон.
  • Поля для рассылки не могут быть добавлены к полю Тема E-mail.
  • Поля для рассылки составляются на основе XML, учитывающего регистр. Если регистр в поле для рассылки меняется, он не будет отослан.
  • Нельзя поместить поля для рассылки в текстовое поле при создании или редактировании шаблона.
  • Шаблон должен содержать поле для рассылки из таблицы Адресов, чтобы отображать диалог Выбрать Тип адреса во время Рассылки. Смотрите выбор типа адреса для получения дополнительной информации.
