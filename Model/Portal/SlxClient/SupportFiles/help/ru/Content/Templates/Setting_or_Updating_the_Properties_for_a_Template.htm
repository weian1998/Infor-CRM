Установить или обновить свойства для шаблона

Можно установить или обновить свойства для личного или общего шаблона при создании, редактировании или копировании шаблона. Если вы редактируете или копируете шаблон, некоторые свойства будут соответствовать свойствам выбранного шаблона, при желании, их можно изменить.

Чтобы открыть диалог Свойства для шаблона, выполните одно из следующих действий:
  • В окне Управление шаблонами нажмите Свойства.
  • В окне Управление шаблонами нажмите на кнопку Новый.
  • В окне Управление шаблонами выберите шаблон, затем нажмите Изменить или Копировать.

Установить или обновить свойства для шаблона

  1. В области Тип шаблона выберите один из типов шаблона. Типы по умолчанию - E-mail, Факс, и письмо, но ваша компания может предложить другие типы
  2. В поле Название введите название нового шаблона.

    Название не должно включать апостроф(‘), слеш (\) или тире (-).

  3. В поле Семейство выберите семейство для шаблона. Установки по умолчанию - E-mail, Факс, и письмо, но ваша компания может предложить другие типы Семейств.
  4. В поле Основная таблица выберите Контакт или Наводка. При создании шаблона для использования вместе с Контактами, выберите Контакт. При создании шаблона для использования вместе с Наводками, выберите Наводка.
  5. В поле Доступвыберите пользователя Infor CRM в качестве того, у кого есть доступ к шаблону.

    Если вы передадите личный шаблон другому пользователю, необходимо сначала скопировать шаблон, затем передать скопированный шаблон другому пользователю. Таким образом, личный шаблон останется в вашем списке.

  6. Нажмите ОК.
