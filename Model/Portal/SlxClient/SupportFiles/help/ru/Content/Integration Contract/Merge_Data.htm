Объединение Данных

Экран Объединения данных отображает различия в данных при связывании записей между Infor и системой учета. Используйте этот экран для выбора данных, которые вы хотите использовать при слиянии записей. Данные, которые вы храните, переписывает данные другой системы. Узнать больше...

  • Если Интеграция учета не включена и вы объединяете два субъекта, заказы, связанные с удаляемым субъектом, добавляются к объединяемому субъекту.
  • Если интеграция учета включена, и вы сливаете два субъекта, субъект, у которого есть связанные заказы или котировки, которые были подтверждены в учетной системе, не могут быть удалены (субъект, который содержит подтвержденные записи, не может быть субъектом-источником). При необходимости, можно контролировать субъект-источник с помощью выбора этого субъекта первым при объединении данных. Это позволяет вам сохранить субъект с связанными заказами.

Чтобы выбрать

  1. Откройте Мастер связи и пройдите предыдущие экраны.
  1. Заполните экран Объединить данные
    ПолеОписание/Действие
    Свойство

    Перечисляет поля данных, которые существуют для записи в двух экземплярах в Infor.

    Колонка [Источник]

    Используйте колонку слева от Колонки источника, чтобы выбрать, какие данные сохранить из Infor при объединении записей.

    Выберите параметр в заголовке колонки, чтобы сохранить все данные Infor (Источник).

    Колонка [Цель]

    Используйте колонку слева от целевой Колонки, чтобы выбрать, какие данные сохранить из учетной системы при объединении записей.

    Выберите параметр в заголовке колонки, чтобы сохранить все данные учетной системы.

    Скрыть Дубликаты или Показать всеНажмите, чтобы показать или скрыть информацию о свойствах, одинаковую для обеих систем. По умолчанию, в списке отображаются только поля с различными значениями.
  2. Нажмите Следующий, чтобы перейти к следующему шагу.
    В зависимости от вашего выбора, вам может понадобиться объединить контакты.
