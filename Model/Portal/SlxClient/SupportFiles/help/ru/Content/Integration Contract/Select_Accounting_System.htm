Выбор Системы учета

Экран Выбрать систему учета в мастер ссылок на учет отображает системы учета, доступные для связи с выбранным субъектом.

Чтобы выбрать

  1. Откройте ссылку на Мастер учета.
  1. Заполните экран Выбрать систему учета.
    ПолеОписание/Действие
    СистемаВыберите систему учета, с которой вы хотите связать субъект.
  2. Нажмите Далее для перехода к следующему шагу.
