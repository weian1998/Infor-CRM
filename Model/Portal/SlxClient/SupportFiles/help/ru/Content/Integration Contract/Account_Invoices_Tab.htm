Закладка Счета

(Только для Интеграции учета) Закладка Счета на детальном представлении Субъекта отображает счета субъекта из учетной системы. Можно использовать эту закладку для просмотра такой информации, как статус счета, баланс, сумма и так далее. Эта закладка только для чтения.

Для того чтобы просмотреть

  1. Открыть Детальное представление Субъекта.
  2. Выберите закладку Счета. Если закладка не отображается, то откройте закладку Все закладки.
  3. Чтобы просмотреть дополнительные детали счета, нажмите номер счета в колонке Счет. Детальное представление счета отображает дополнительную информацию (только чтение).

Представление содержит следующие колонки:

Колонка Описание
Счет # Номер счета.
Дата счета Дата создания счета.
Номер заказа Номер заказа на покупку.
Срок платежа Срок оплаты счета.
Чистый итог Чистый итог счета.
Налог Налог по счету.
Сумма счета Общая сумма счета.
