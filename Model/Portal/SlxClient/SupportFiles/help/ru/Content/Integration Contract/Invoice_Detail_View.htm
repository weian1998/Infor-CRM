Детальное представление счета

Детальное представление счета отображает информацию из вашей учетной системы (только для чтения).

Для того чтобы просмотреть

  1. На лицевой карточке Субъекта, перейдите к закладке Счета.
  2. Нажмите номер счета для счета, который вы хотите просмотреть. Следующая информация отображает:
    • Счет - Номер счета.
    • Субъект - субъект, с которым связан счет.
    • Статус - статус счета.
    • Сумма счета - Общая сумма счета.
  3. Нажмите одну из следующих закладок для просмотра дополнительной информации.
    • Закладка Детали
    ПолеОписание
    Дата счетаДата создания счета.
