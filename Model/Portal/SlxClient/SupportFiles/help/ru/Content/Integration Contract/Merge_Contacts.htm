Объединение контактов

Экран Объединение контактов отображает контакты для субъекта Infor, который вы связываете с клиентом, выбранным в учетной системе. Используйте этот экран для создания связей и объединения записей контакта Infor или между Infor и записями учетной системы.

Чтобы выбрать

  1. Откройте Мастер связи и пройдите предыдущие экраны.
  1. Заполните экран Объединить контакты.
    ПолеОписание/Действие
    Несвязанные контакты из Infor CRM

    Отображает контакт, которые доступны для связывания с контактом учетной системы.

    Нажмите Выбрать для выбора контакта.

    Несвязанные контакты из [конечная точка вашей учетной системы]

    Отображает контакты в системе учета, которые доступны для связывания с контактом.

    Нажмите Выбрать для выбора контакта.

    Кнопка Связать

    Выберите несвязанный контакт из списка и нажмите Связать для создания связи.

    Связанные контакты

    Отображает совпадения, которые были идентифицированы в качестве потенциальных связей, используя определенные критерии совпадения.

    • Нажмите Удалить Связь, чтобы удалить предложенную связь и создать новую связь.

    • Нажмите Выбрать для отображения полей контакта в источнике и целевой системе.

  2. Нажмите Далее.
