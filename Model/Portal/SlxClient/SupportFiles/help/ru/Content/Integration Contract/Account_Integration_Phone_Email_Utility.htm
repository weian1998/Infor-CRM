Запуск Утилиты SLXConversion

Перед настройкой конечных точек Учета интеграции, необходимо запустить утилиту SLXConversion обновить номера телефона и адреса электронной почты записей. Эта утилита поддерживает интеграцию путем добавления столбцов в базу данных Infor, чтобы дать предоставить возможность реализовать отношение один ко многим телефонных номеров и адресов электронной почты в учетной записи и контактах.

Данная утилита создает запись в таблице ERPPHONENUMBER или ERPEMAILADDRESS для каждого телефонного номера и столбца email в таблицах СУБЪЕКТ и КОНТАКТ. Данные записи соответствуют оригинальному значению столбца сущности, в которой они инициировались.

Проверка Подлинности Windows должна настраиваться для пользователя с правами администратора до запуска утилиты обновления E-mail и Телефона Субъекта. Для получения более подробных действий смотрите тему "Использование Проверки Подлинности Windows" в Справке Администратора.

Для запуска утилиты обновления записей E-mail и телефона субъекта.

  1. На вашей рабочей станции администратора, найдите файлы SLXConversionUtility.exe и SLXConversionUtility.config.
    Конфигурационные файлы по умолчанию находятся в ...\Program Files\Saleslogix
  2. Дважды щелкните на SLXConversionUtility.config , чтобы открыть файл в текстовом редакторе по вашему выбору.
  3. Прокрутите до раздела <connectionStrings> и измените значение Initial Catalog с SALESLOGIX_EVAL на псевдоним вашей базы данных.
    Например:
    <connectionStrings> <clear/> <add name="Default" connectionString="Provider=SLXOLEDB.1;Persist Security Info=True;Initial Catalog=MyDatabaseAlias;Data Source=localhost;Extended Properties=&quot;PORT=1706;LOG=ON;CASEINSENSITIVEFIND=ON;AUTOINCBATCHSIZE=1;SVRCERT=12345;ACTIVITYSECURITY=OFF;TIMEZONE=NONE&quot;"/> </connectionStrings>
  4. Дважды щелкните на SLXConversionUtility.exe для запуска утилиты.
  5. Установите все флажки.
  6. Щелкните Обновить.
  7. Нажмите кнопку Закрыть.
