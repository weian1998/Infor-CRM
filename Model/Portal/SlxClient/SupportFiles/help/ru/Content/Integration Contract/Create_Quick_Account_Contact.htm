Создание Нового субъекта из Заказа

Можно создать заказ перед созданием субъекта или контакта. Если потенциальный клиент решит работать с вашей компанией, вы можете создать новый субъект и контакт из заказа с ограниченной информацией о компании и контакте.

Чтобы создать

  1. Создать новый заказ.
  2. Нажмите кнопку Добавить новый субъект.
  3. В диалоге Добавить новый Субъект Контакт добавьте следующую информацию:
    1. В поле Субъект введите имя субъекта.
    2. В поле Тип введите тип субъекта.
    3. В поле Основной тел. введите основной номер телефона субъекта.
    4. В поле Адрес введите адрес субъекта.
    5. В поле Контакт введите имя контакта.
    6. В поле Должность введите или выберите должность контакта.
    7. В поле Основной тел. введите номер телефона контакта.
    8. В поле Мобильный введите номер мобильного телефона контакта.
    9. В поле E-Mail введите адрес электронной почты контакта.
    10. Нажмите ОК.
  4. Нажмите Сохранить.
