Настройка Службы синхронизации Интеграции учета

Эта тема больше не действительна. Дополнительные сведения о настройке службы синхронизации SData см. в разделе "Служба настройки Integration Synchronization".

Служба синхронизации Интеграции учета (Infor системный сервис) делает возможным обмен данными между Infor и вашей системой учета. Служба должна начать работу автоматически.

Синхронизации информации в режиме реального времени требуется подключение к интернету. При использовании Интеграции учета на автономном веб-клиенте, при просмотре данных в режиме реального времени, данные будут по состоянию на момент последней связи клиента с базой данных хоста.

Для настройки

  1. Убедитесь, что вы настроили и подключили конечные точки перед началом работы службы.
  2. Просмотрите файл SLXSDataSyncServer.exe.config, убедитесь, что соединение определено верным образом в конце файла. Информация о соединении должна соответствовать connection.config на вашем Веб Сайте.
    По умолчанию файл конфигурации расположен в ...\Program Files\Saleslogix\SLXSDataSyncServer.exe.config и файл с информацией по соединению в ...\inetpub\wwwroot\slxclient.
  3. На административной рабочей станции откройте Службы.
  4. Правой кнопкой мыши кликните Infor системный сервис, затем нажмите Свойства.
  5. В поле Тип старта выберите Автоматически.
  6. Запустите службу под пользователем WebDLL.
  7. Убедитесь, что вы включили проверку подлинности Windows для пользователя WebDLL.
  8. Нажмите Начать.
  9. Нажмите ОК.
