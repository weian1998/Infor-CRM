Детальное представление Интеграции Учета

Эта тема больше не применяется. Сведения о настройке SData синхронизации см. в разделе "Использование SData синхронизации".

Используйте детальное представление Интеграции Учета для настройки связей между Infor и вашими учетными системами.

Интеграция Учета возможна, когда возможны и настроены точки интеграции на стороне Infor и на стороне учетной системы. Если ничего не указано на закладке Системы учета, тогда интеграция не возможна.

Когда вы включаете или отключаете Интеграцию учета, изменения вступят в силу только после повторного входа в систему.

Установите свойства подаваемой информации для Infor, затем настройте свойства информации для каждой интегрируемой учетной системы.

Чтобы установить

  1. В меню Администрирование выберите Установка интеграции
  2. В поле Infor Канал введите адрес портала SData, который используется Infor для обмена информацией между системами.
    •  Выполните одно из следующих действий:
      • Если ваша реализация не использует порт по умолчанию (порт 80), используйте формат http://server:port/sdatavirtualdir/application/contract/dataset/
      • Если ваша реализация использует порт 80, опустите номер порта в адресе. Сервис Sync автоматически удаляет порт по умолчанию из адреса, что вызывает проблемы и ошибки. Избежать этого можно, используя формат http://server/sdatavirutaldir/application/contract/dataset/
    • Замените значения представленными Вашим партнером по интеграции:
      •  Sdatavirtualdir – например, sdata.
      • application – наименование вашей системы учета. Например, accounts50.
      • contract – наименование контракта. Например, GCRM.
      • dataset – различает набор данных или рабочей компании. Например, / - /
  3. В поле Имя пользователя, введите имя пользователя для базы данных Infor, к которой подсоединяется учетная система.
  4. В поле Пароль введите пароль для имени пользователя.
  5. В поле Название введите отображаемое название для потока.
  6. При необходимости, выберите Ограничить выдвижение Субъекта в единственную систему учета, чтобы ограничить интеграцию одной учетной системой.
  7. При необходимости, установите флажок Разрешить системе учета управлять Заказами для удаления возможности создавать и редактировать заказы в Infor. Когда вы выставите этот флажок, ваша система учета должна быть использована для управления заказами.

    8. Эта опция необходима для некоторых интеграции, например, с X3.
  8. Нажмите Сохранить
  9. Используйте закладку Системы учета для настройки деталей снабжения для вашей учетной системы.
  10. Используйте закладку Синхронизация для настройки интервалов синхронизации.
  11. Используйте закладку Совпадения, чтобы установить глобальные стандартные критерии совпадений для пользователей, создающих связи.
