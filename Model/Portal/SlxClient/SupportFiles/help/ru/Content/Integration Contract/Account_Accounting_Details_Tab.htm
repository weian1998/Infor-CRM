Закладка Детали учета детального представления Субъекта

(Только для Интеграции учета) Закладка Детали учета отображает дополнительную информацию о компании из учетного приложения. Эта информация синхронизируется со связанным субъектом в учетной системе.

Чтобы установить

  1. Откройте Детальное представление Субъекта для связанного субъекта.
  2. Выберите закладку Детали учета.Если закладка не отображается, то откройте закладку Все закладки
  3. Добавьте или измените следующую информацию:
    • Субъект - имя компании.
    • Дата открытия - дата, когда субъект был создан в системе учета.
    • Статус - статус субъекта в учетной системе.
    • Способ доставки - выбранный способ доставки.
    • Финансовый статус - финансовый статус компании.
    • Баланс учета - баланс учета компании.
    • Дата баланса - дата подсчета баланса учета.
    • Финансовый лимит - финансовый лимит компании.
    • Срок платежа - срок платежа компании.
  4. Нажмите Сохранить.
