Поиск Совпадений

Экран Поиск совпадений позволяет вам выбрать критерии, которые будут использоваться при поиске в системе учета клиентов, которые являются потенциальными совпадениями с субъектом, который вы связываете из Infor.

Чтобы выбрать

  1. Откройте мастер ссылок на учет и пройдите предыдущие экраны.
  1. Заполните экран Поиск совпадений.
    ПолеОписание/Действие
    СвойствоВыберите поле для поиска совпадающих данных. Доступные поля - поля, связанные между Infor и системой учета.
    ОператорВыберите критерии совпадения, используемые системой для выбранного свойства. Доступные операторы - Равно, Начинается с, Содержит.
    Искать значениеВведите значение, по которому хотите искать.
    Добавить или удалить критерии поискаЧтобы добавить дополнительные критерии соответствия, нажмите . Чтобы удалить критерии соответствия, нажмите .
  2. Нажмите Далее для перехода к следующему шагу.
