Детальное представление Заказа (Интеграция учета)

(Только для Интеграции учета) Детальное представление платежа отображает информацию из вашей учетной системы (только для чтения).

Если у вас не настроена интеграция учета, используйте Детальное представление заказа для просмотра информации о заказе.

Для того чтобы просмотреть

  1. На детальном представлении Субъекта выберите закладку Заказы учета.
  2. Нажмите номер счета для заказа, который вы хотите просмотреть. Следующая информация отображает:
  • Заказ - Номер заказа.
  • Статус - Статус заказа.
  • Сумма счета - Общая сумма заказа.
  1. Нажмите одну из следующих закладок для просмотра дополнительной информации.
