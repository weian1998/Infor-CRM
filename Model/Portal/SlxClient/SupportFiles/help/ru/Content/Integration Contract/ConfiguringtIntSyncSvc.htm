Служба настройки синхронизации источников интеграции

Служба синхронизации источников интеграции(Служба синхронизации SData) позволяет передавать данные между Вашими интегрированными источниками. Служба должна быть запущена и настроена для работы в автоматическом режиме.

Синхронизация информации в режиме реального времени требует подключения к интернету. При использовании учета интеграции на Автономный веб-клиент, данные в представлениях данных в режиме реального времени будет по состоянию на последнее подключение пользователя к базе данных.

Настройка

  1. Войдите в систему как пользователь WebDLL.

  2. Убедитесь, что вы настроили и включили синхронизацию SData и Вашу интеграцию перед запуском службы.

  3. Просмотрите файл Sage.Saleslogix.SLXSDataSyncServer.exe.config, чтобы обеспечить корректное соединение. Информация о соединении должна соответствовать connection.config на вашем веб-сайте.
    По умолчанию конфигурационный файл находится в ...\Program Files\Saleslogix\Sage.Saleslogix.SLXSDataSyncServer.exe.config и файл подключения будет развернут в ...\inetpub\wwwroot\slxclient
  4. Убедитесь, что Вы разрешили проверку подлинности Windows для пользователя WebDLL.
  5. На рабочей станции администратора, в диалоговом окне служба Windows в Управлении компьютером.
  6. Дважды нажмите правой кнопкой мыши на Infor сервере синхронизации SData и затем нажмите Свойства.
    1. В поле Тип запуска выберите Автоматически.
    2. Установите службу авторизации и запустите под пользователем WebDLL.
    3. Нажмите Старт.
  7.  Нажмите OK.
