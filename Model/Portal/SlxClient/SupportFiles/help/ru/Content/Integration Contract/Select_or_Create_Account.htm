Выбор или Создание субъекта

Экран Выбор или Создание субъекта отображает клиентов в учетной системе, которые являются потенциальными совпадениями для субъекта, которого вы связываете в Infor. Можно выбрать существующего клиента для связи, создать новую запись по клиенту или переопределить критерии поиска и провести поиск еще раз.

Чтобы выбрать

  1. Откройте мастер ссылок на учет и пройдите предыдущие экраны.
  1. Заполните экран Выбрать или создать Субъект и выберите одно из следующих действий:
    2. Поле Описание/Действие Следующий шаг
    Связь [имя субъекта] Отображает любые субъекты в учетной системе, которые являются потенциальными совпадениями для связываемого субъекта. Выберите эту функцию и клиента, с которым вы хотите связать субъект. Если информация в по субъекту и записи по клиенту не совпадают в точности, необходимо провести слияние данных. Слияние данных
    Создать новый Субъект для [имя субъекта] Выберите эту функцию, чтобы использовать запись субъекта в Infor для создания записи клиента в учетной системе.

    Новая учетная запись клиента создается в системе учета.

    Нажмите Закрыть, чтобы закрыть диалоговое окно.
    Определить или расширить поиск для существующего субъекта

    Выберите эту функцию, если вы не можете найти существующую запись по клиенту в учетной системе и хотите изменить критерии поиска и провести поиск еще раз.

    Эта опция доступна только в том случае, если в системе учета была реализована фильтрация.

    		 Поиск совпадений
  2. Нажмите Далее для перехода к следующему шагу.
