Закладка Платежи клиента

(Только для Интеграции учета) Закладка Платежи Клиента на детальном представлении Субъекта отображает статус платежей по счетам субъекта. Эта закладка только для чтения.

Для того чтобы просмотреть

  1. Открыть Детальное представление Субъекта.
  2. Выберите закладку Платежи клиента. Если закладка не отображается, то откройте закладку Все закладки.
  3. Чтобы просмотреть дополнительные детали, нажмите номер квитанции в колонке Номер квитанции. Детальное представление Платежа отображает дополнительную информацию (только чтение).

Представление содержит следующие колонки:

Колонка Описание
Номер квитанции Номер квитанции.
Дата платежа Дата платежа.
Тип Тип квитанции. Например, предоплата, платеж и т.д.
Статус Статус квитанции. Например, в ожидании, авторизованная и т.д.
Сумма платежа Значение квитанции, включая налог и скидки. Например, фактически полученная сумма.
Валюта Валюта квитанции.
