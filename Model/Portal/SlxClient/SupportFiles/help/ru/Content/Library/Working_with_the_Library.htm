Работа с Библиотекой

Библиотека - это основное хранилище информации компании (в виде документов, ссылок и так далее), которые обеспечивают легкий доступ к информации удаленным пользователям. Файлы хранятся в папках и сортируются по названию.Нажмите заголовки столбцов для сортировки списка файлов библиотеки.

При помощи Infor CRM Веб-клиент вы можете запросить и загрузить файл Библиотеки, когда он вам понадобится. В Библиотеке отображается имя файла, размер, даты создания, пересмотра и истечения срока действия, а также краткое описание файла.

Ваша компания может включать в Библиотеку информацию о продукте, правила и инструкции, презентации, веб-адреса и т. д.

Только системный администратор или пользователи с правами администратора могут добавлять файлы в Библиотеку.

Как выполнить?

Открыть Библиотеку
  • На навигационной панели нажмите Библиотека.
Загрузить файл из Библиотеки
  1. В иерархическом представлении, нажмите символ "+" чтобы раскрыть нужную папку.
  2. В списке выберите Файл имя файла, который вы хотите загрузить.

  3. Выполните одно из следующих действий:

    • Откройте файл
    • Сохраните файл

Если ваша программа просмотра открывает вложение вместо того, чтобы сохранить его на вашем компьютере, измените настройки по умолчанию вашей программы просмотра для автоматического сохранения файлов вместо их открытия. В противном случае, зажмите [Shift], а затем щелкните ссылку для загрузки без открытия всех файлов, кроме имеющих расширение HTML. Файлы HTML открываются в другом окне..

Добавить документ в Библиотеку
  1. Выберите папку, в которую вы хотите добавить документ.
  2. Нажмите Добавить файлы.
  3. Выберите файлы, которые вы хотите добавить.
  4. Нажмите Открыть.
Добавить документ в Библиотеку, используя функцию "перетаскивания"

  • У вас может быть недостаточно прав доступа для этой функции. Для большей информации, обратитесь к администратору.
  • Если вы используете Internet Explorer,эта функция требует установки Модуля Интеграции рабочего стола.
  1. Найдите файл, который вы хотите добавить.
  2. На навигационной панели нажмите Библиотека.
  3. В иерархическом представлении, нажмите символ "+" чтобы раскрыть нужную папку.
  4. Нажмите файл и перетащите его в представление Библиотека.
  5. В поле Описание введите описание файла, если описание по умолчанию не понятно.
  6. Нажмите ОК.

Нельзя перетаскивать папку вместе с содержимым в Библиотеку. Перетаскивать можно только отдельные файлы.
