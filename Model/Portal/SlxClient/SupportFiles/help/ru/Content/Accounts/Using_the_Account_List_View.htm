Использование Списочного представления Субъектов

В Списочном представлении Субъектов отображаются все Субъекты, к которым вы имеете доступ. Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей..

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Как выполнить?

Открыть Списочное представление Субъектов
  • На Навигационной панели нажмите Субъекты.

Работа с записями в таблице

Используйте сводное представление

Обновить список
  • Нажмите кнопку Обновить.

Отфильтровать список

Используйте Поиск для нахождения записей

Открыть Детальное представление Субъекта
  • Нажмите на название субъекта, чтобы открыть его Детальное представление.

Просмотреть группы

 

Использование меню Групп

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы
  • Сохранить записи как группу
  • Выгрузить в файл
  • Перевести на Интерактивную панель
