Детальное представление Взаимосвязи Субъектов

Используйте взаимосвязи для создания отношений между субъектами.

Например, если вы работаете с компанией, с несколькими подразделениями, вы можете создать субъект для каждого подразделения, и создать субъект для представления компании в целом (родительский субъект). Затем можно связать каждый субъект подразделения с головным субъектом. Если бы были контакты в различных подразделениях родительской компании, вы могли бы связать контакты.

Для одного субъекта в системе можно создать несколько взаимосвязей. Подробную информацию о взаимосвязи одного субъекта с другим можно просмотреть на Форме для редактирования связи для Субъекта. На данной форме отображаются названия субъектов, а также краткое описание и комментарии.

Для того чтобы связать субъекты

  1. В детальном представлении Субъектов выберите закладку Взаимосвязи. Если закладка не отображается, то откройте закладку Все закладки.
  2. Нажмите на кнопку Добавить связь.
  3. Нажмите кнопку Поиск и найдите субъект для связи с первым субъектом.
  4. Нажмите под первым субъектом и выберите пункт из списка, который лучше всего описывает отношение первого субъекта.
  5. Нажмите под вторым субъектом и выберите пункт из списка, который лучше всего описывает отношение второго субъекта.
  6. В поле Описание введите описание субъекта.

  7. Нажмите ОК.

Для того чтобы изменить

  1. На Детальном представлении Субъекта перейдите к закладке Взаимосвязи.
  2. Нажмите ссылку Изменить для взаимосвязи, которую вы хотите просмотреть или изменить.
  3. Внесите изменения.

  4. Нажмите ОК.
