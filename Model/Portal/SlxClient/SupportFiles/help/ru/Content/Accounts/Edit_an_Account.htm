Редактирование Субъекта

Используйте Детальное представление Субъекта для изменения информации по Субъекту. Чтобы отменить изменения, нажмите кнопку Сброс перед нажатием кнопки Сохранить.

Для того чтобы изменить

  1. На Навигационной панели нажмите Субъекты.
  2. Найдите субъект, который вы хотите изменить и нажмите на имя субъекта.
  3. На Детальном представлении Субъекта внесите изменения в информацию по субъекту.
  4. Нажмите Сохранить.

Вы можете запросить обновление соответсвующей информации по контактам. Проставьте соответствующие флажки и нажмите ОК для обновления информации по всем связанным контактам.
