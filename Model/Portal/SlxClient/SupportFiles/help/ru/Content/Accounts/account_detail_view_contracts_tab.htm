Закладка Контракты Детального представления Субъекта

Закладка Контракты в Детальном представлении Субъекта позволяет просматривать контрактную информацию, связанную с субъектом. Вы можете использовать эту закладку для проверки контрактной информации, поскольку это удобнее, чем открывать каждый контракт в отдельности.

Помимо проверки контрактной информации, можно также вносить изменения в контракты и добавлять новые контракты по субъекту.

Использование закладки Контракты

  1. Откройте детальное представление Субъекта.
  2. Выберите закладку Контракты. Если закладка не отображается, то откройте закладку Все закладки.
  3. Выполните любое из следующих действий:
    • Чтобы увидеть заметки контракта, выберите контракт, который вы хотите просмотреть.
    • Для того чтобы открыть контракт щелкните по ссылке Номер.
      Откроется детальное представление Контракта.
    • Для того чтобы создать новый контракт, нажмите Создать Контракт.
