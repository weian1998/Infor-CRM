Добавление нового Субъекта

Вы можете использовать диалоговое окно Добавить новый Субъект, чтобы добавить новый субъект.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Для того чтобы добавить

  1. В меню Создать выберите Субъект.
  2. Заполните остальные информационные поля субъекта в соответствии с требованиями вашей компании.
  3. Нажмите Сохранить.
