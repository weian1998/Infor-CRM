Информационные поля элемента инф. материалов

В таблице ниже описаны основные поля.

 

Поле Описание/Действие
Номер элемента

Идентификационный номер документа, приписанный менеджером литературы.

Вам не нужно вводить данные в поле Номер элемента. Используйте поле Номер элемента, если в вашей компании назначены номера документов для каждого типа литературы.
Название Название документа.
Автор Автор документа.
Семейство Нажмите и выберите элемент из списка.
Цена Стоимость документа.
В наличии

Количество, в данный момент имеющееся в наличии.

Это число уменьшается каждый раз, когда исполняется запрос. Если поле В наличии не используется, количество не уменьшается. Если вы введете ноль в поле В наличии, количество будет уменьшаться, в сторону отрицательных чисел.
Переупорядоченное кол-во

Минимальное количество, показывающее, что пришло время для пополнения запасов.

Если значение В наличии меньше или равно Переупорядоченному количеству, отображается сообщение, что пришло время для пополнения запасов.
Файл Если документ является файлом, введите путь к файлу в поле Файл.
Истекает

Нажмите кнопку Календарь, выберите дату и нажмите OK.

При достижении даты, документ становится красным в Менеджере Литературы, при необходимости, он может быть обновлен.
