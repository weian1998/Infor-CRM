Детальное представление элемента инф. материалов

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление используется для просмотра, добавления или изменения информации по выбранной записи.

Как выполнить?

Открыть детальное представление элемента инф. материалов
  • На навигационной панели, нажмите Элементы инф. материалов для открытия списочного представления элементов инф. материалов. При необходимости, воспользуйтесь поиском для нахождения необходимого элемента инф. материалов, затем нажмите на номер элемента.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать разделы Записи группы и Обычные задачи панели задач

Просмотреть группу

Использовать обычные задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы
