Закладка комментарии/история

Закладка комментарии/история отображается на детальных представлениях субъекта, контакта, сделки и наводки. Закладка содержит фильтры, которые позволяют быстро получить доступ к необходимой информации. Вы также можете использовать данную закладку, чтобы отправить запись истории по e-mail, создать документ Microsoft Word, просмотреть вложение или отредактировать заметки.

Закладка комментарии/история содержит все взаимодействия, включая полный текст заметок и предоставляют фильтры для просмотра, отправки электронной почты и распечатки записей истории и заметок.

Как выполнить?

Открыть закладку комментарии/история
  • В детальном представлении Контакта, Субъекта, Сделки или Наводки, откройте закладку Комментарии/История. Если закладка не отображается, то откройте закладку Все закладки.
Наложить фильтры
  1. Нажмите Фильтры, чтобы отобразить панель фильтров.

Вы можете выбрать несколько фильтров. Например, вы можете использовать фильтр Моя история и фильтр по дате, чтобы просмотреть все свои записи истории, попадающие в заданный интервал.

  1. Выберите один или несколько фильтров из предложенных ниже:
    • Чтобы отфильтровать список по типу записи, в поле Тип, выберите тип записей, которые хотите отобразить.
    • Изменения базы данных скрыты по умолчанию. Чтобы просмотреть их, выберите параметр Показать изменения базы данных.
    • Используйте поля Диапазон дат, чтобы выбрать диапазон дат в течении которого, записи истории были последний раз обновлены.
    • В поле Пользователь, нажмите на указывающую вниз стрелку и выберите значение из списка.
    • В поле Тема, введите один или несколько символов, содержащихся в теме. Чем больше символов вы введете, тем короче будет список результатов.
  2. Нажмите Применить.
  3. Чтобы сбросить фильтры, нажмите Сброс.
    Снова нажмите на кнопку фильтры, чтобы спрятать панель фильтров.
Просмотреть детали заметки или истории
  • Когда вы выбираете запись из списка, детали записи отображаются на панели, справа от списка.
Открыть заметку или запись истории
  • Нажмите на ссылку Тип записи, которую хотите просмотреть.
Отправить по e-mail запись истории
  1. Выберите запись(и), которые вы хотите отправить по e-mail.
  2. Нажмите Отправить по E-mail.
    Откроется e-mail сообщение, включающее информацию о контакте, теме и информацию о записи истории. Если были выбраны несколько записей, будет включена информация о каждой записи.
  3. Нажмите Отправить.
Отправить записи истории в Microsoft Word
  1. Выберите запись(и), которые вы хотите скопировать в документ Word.
  2. Нажмите Отправить в Word.
    Откроется Microsoft Word и отобразит документ с заметкой или информацией о записи истории.
  3. Вы можете распечатать документ или сохранить в файл.
Добавить заметку
  • На закладке Комментарии/История, нажмите Добавить заметку и следуйте инструкции по добавлению заметки.
