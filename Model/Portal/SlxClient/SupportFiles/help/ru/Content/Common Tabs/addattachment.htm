Добавление вложений

Вложения можно добавлять для основных объектов системы: Субъекта, Контакта, Сделки, Дела и проч. Все файлы сохраняются в общедоступной директории и синхронизируются с удаленными базами данных.

По умолчанию, нельзя добавлять файлы больше 4MB. Если вы добавляете файл или группу файлов больше 4MB, вы получите предупреждение или ошибку. Для получения подробной информации обратитесь к своему администратору.

Как выполнить?

Прикрепить файл
  1. Откройте закладку вложения.
  2. Нажмите кнопку Добавить файл.
  3. Если открылся диалог Прикрепить файл, нажмите Выбрать файл.
  4. Найдите нужный файл и нажмите Открыть.
  5. В поле Описание введите описание для прикрепленного файла.
  6. Нажмите ОК.
Прикрепить файл с помощью перетаскивания

  • Эта функция может потребовать установки модуля Интеграции рабочего стола.
  1. Найдите файл, который вы хотите добавить.
  2. Нажмите на файл и перетащите его в нужное детальное представление.

Откроется диалог Добавить вложение.

    Если вам предложено установить Интеграцию рабочего столаn и вы хотите использовать эту возможность, нажмите Да и следуйте инструкции для установки Infor CRM Интеграции рабочего стола. Чтобы завершить установку, вам будет предложено закрыть все браузеры и Microsoft Outlook.
  1. В поле Описание введите описание для прикрепленного файла.

  2. Нажмите ОК.

Прикрепить URL
  1. Откройте закладку вложения.
  2. Нажмите кнопку Добавить URL.
  3. В поле URL введите URL адрес.
  4. В поле Описание введите описание.
  5. Нажмите OK.
    Описание и URL обязательны для заполнения.
