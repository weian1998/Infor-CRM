Добавление заметки

Используйте заметки, чтобы задокументировать результат дела и для записи взаимодействий с субъектом, контактом, наводкой или сделкой. Добавленные вами заметки отображаются в закладке комментарии/история на детальных представлениях субъекта, контакта, наводки и сделки.

Если вы создали заметку по сделке, а затем удалили связь с заметкой из контакта и субъекта, заметка не будет отправлена удаленным пользователям.

Чтобы добавить заметку

  1. В меню Создать, нажмите Заметка.
  2. В поле Тема, нажмите и выберите элемент из списка. В поле Тема, введите Заметки о встречи.
  3. При необходимости, используйте поле Размещение для ввода расположения.
  4. При необходимости, измените даты Запланировано и Завершено. Чтобы изменить даты, нажмите кнопку Календарь для выбора даты, а затем нажмите OK.  
  5. Выполните одно из следующих действий:
    • Выберите Без времени, если вам не нужно указывать продолжительность заметки.
    • В поле Длительность, нажмите на указывающую вниз стрелку и выберите значение из списка.

    		6.

    Чтобы удалить наводку, контакт, субъект, сделку или заявку, нажмите кнопку Удалить в соответствующем поле.
  6. В поле Приоритет, нажмите и выберите элемент из списка.
  7. При необходимости, нажмите кнопку Найти и с помощью поиска выберите Руководителя.
  8. В поле Заметки, введите необходимую информацию.
  9. При необходимости, нажмите на закладку Вложения, чтобы добавить документ или URL.
  10. (Не обязательно) В поле Следующее, нажмите на и выберите тип следующего дела. Запланируйте следующее дело.
  1. Используйте флаги Перенести заметки или Перенести вложения, чтобы скопировать заметки или вложения из данной заметки в следующее дело.
  2. Нажмите OK.
