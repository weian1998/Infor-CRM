Добавление или редактирование отклика

Используйте данный диалог для обновления информации об отклике. Отклики отслеживают взаимодействия с контактом или наводкой. Отклики отображаются на закладке Отклики детального представления.

Как выполнить?

Добавить отклик
  1. Выполните одно из следующих действий:
    • В детальном представлении Контакта, Наводки или Субъекта, откройте закладку Отклики илиМаркетинг. Если закладка не отображается, то откройте закладку Все закладки.
    • В детальном представлении Маркетинговой программы, откройте закладку Отклики. Если закладка не отображается, то откройте закладку Все закладки.
  2. Щелкните правой кнопкой мыши на закладке Отклики, и нажмите Добавить отклик.
  3. Нажмите кнопку Добавить отклик или Добавить отклик на маркетинговую программу.
  4. Выберите Контакт или Наводка.
  5. В зависимости от вашего выбора на предыдущем шаге, нажмите кнопку Найти напротив Контакта или Наводки и воспользуйтесь поиском для нахождения соответствующей записи.
  6. Метод отклика нажмите и выберите элемент из списка.
  7. Маркетинговая программа при необходимости, нажмите кнопку Найти для поиска маркетинговой программы.
  8. Стадия если вы указали маркетинговую программу, нажмите на указывающую вниз стрелку и выберите значение из списка.
  9. Источник наводки нажмите кнопку Найти для нахождения источника наводки.
  10. Статус нажмите и выберите элемент из списка.
  11. В поле Дата отклика нажмите кнопку Календарь для выбора даты, а затем нажмите OK.  
  12. Метод отклика нажмите и выберите элемент из списка.
  13. Интерес нажмите для выбора описания того, как контакт или наводка отреагировали.
  14. Уровень интереса нажмите для определения того, насколько заинтересован контакт или наводка.
  15. В поле Комментарии, введите любые заметки.
  16. Используйте закладку Продукты для просмотра продуктов, связанных с откликом или нажмите Добавить продукт для поиска и добавления продукта.
  17. Нажмите OK.
Изменить отклик
  1. Выполните одно из следующих действий:
    • В детальном представлении Контакта, Наводки или Субъекта, откройте закладку Отклики или Маркетинг. Если закладка не отображается, то откройте закладку Все закладки.
    • В детальном представлении Маркетинговой программы, откройте закладку Отклики. Если закладка не отображается, то откройте закладку Все закладки.
  2. Нажмите на ссылку Изменить lнапротив записи, которую необходимо отредактировать.
  3. Обновите информацию об отклике.
  4. Используйте закладку Продукты для просмотра продуктов, связанных с откликом или нажмите Добавить продукт для поиска и добавления продукта.
  5. Нажмите OK.
