Использование диалогового окна Запуск /Планирование работ

Диалог Запуск / Планирование работ позволяет Вам установить метки и параметры для типа заданий, которые Вы планируете. Доступные опции могут изменяться в зависимости от выбранного задания и расположения откуда Вы открываете диалоговое окно.

 

Имя Действие
Имя работы Автоматически заполняет имена работ. Не редактируется.
Описание Автоматически заполняется описанием текущей работы. Нажмите на поле, чтобы внести изменения.
Расписание

Показывает тип расписания. Не доступно для редактирования, когда открыто из закладки Менеджер Планирования заданий.

  • Запуск задания немедленно.

  • Запланированное Выполнение выводит на экран опции для выполнения задания в более позднее время, и для повторения

 
Создать расписание работы для запуска

Назначьте время и повторение для Вашей работы

  • Используйте выпадающие списки, чтобы выбрать, когда задание будет выполняться.
  • Используйте Запуск с / по для определения даты, запуска и завершения повторения задания
Параметры:

Не все задания требуют параметров. Если параметры были присвоены, Вы можете отредактировать значения параметров, выбирая строку в сетке и вводя в Столбце значений.

Некоторые типы задания требуют обязательных параметров для успешной работы. Например, Экспорт в Excel требует ID Группы.

Примечание: параметры - усовершенствованная функция разработчика.
