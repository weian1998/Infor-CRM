Списочное представление Менеджера заданий

Менеджер заданий обеспечивает пользовательский интерфейс для создания и управления заданиями, отправленными на Infor Сервер заданий. Сервер заданий- механизм для однократной и \ или повторяющейся обработки задач немедленного или отложенного выполнения. Сервер заданий- совместно используемая служба, которая может быть использована любым клиентом Infor, включая Веб, Windows и Mobile. 9 потоков Сервер заданий позволяют выполнять одновременно несколько задач.

Пользователи с Административными полномочиями могут получить доступ к интерфейсу Менеджера заданий, однако все пользователи отправляют задания, которые будут обработаны Сервером заданий, когда они выполняют любой из следующих предопределенных типов задания:

  • Выполнение или планирование отчетов
  • Экспорт файлов к Excel
  • Перепланирование незавершенного дела
  • Обновление оставшихся дней контрактов
  • Обновления Сделок
  • Обновление или удаление наводок на панели задач

 

Менеджер заданий предоставляет следующие функции администратору:

  • Нажмите Исполнения: закладка для просмотра состояния происходящих заданий и / или отмены происходящих заданий.
  • Нажмите Определения: закладка для просмотра списка предопределенных типов задания. Выберите задание и нажмите правой кнопкой, чтобы открыть Менеджер заданий, где Вы можете изменить или добавить новое расписание для задания.
  • Нажмите Расписание: закладка для просмотра списка заданий, которые настроены запускаться по расписанию. Выберите существующее задание и нажмите правой кнопкой, чтобы открыть Менеджер заданий, где Вы можете изменить расписание задания. Вы можете также принять решение удалить существующее задание.
