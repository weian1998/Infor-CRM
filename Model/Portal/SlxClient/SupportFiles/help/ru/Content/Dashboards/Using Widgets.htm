Использование виджетов

Infor CRM предоставляет несколько стандартных виджетов. Вы можете добавлять, редактировать и удалять виджеты из интерактивной панели, также вы можете изменять порядок виджетов на странице путем перетаскивания.

Используя предопределенные фильтры, вы можете отображать показатели данных через диапазоны или различные значения и отобразить эти данные в одном из графических виджетов. Например, вы можете настроить столбчатую диаграмму на отображение всех закрытых и выигранных сделок по доступу, общему количеству продаж или же настроить круговую диаграмму отражающую выигранные продажи по командам.

Список доступных виджетов определяется вашей реализацией. Вы можете не иметь доступ ко всем видам виджетов.

Как выполнить?

Добавить виджет
  1. Щелкните правой кнопкой по закладке Моя интерактивная панель, затем выберите пункт меню Добавить содержимое.
  2. В диалоге Добавить содержимое, нажмите кнопку Добавить на виджете, который хотите добавить.
    Например, чтобы добавить виджет круговой диаграммы, нажмите Добавить под Круговой диаграммой.
    Пустой виджет отобразится на интерактивной панели и откроется редактор виджета.
  3. Настройте виджет, затем нажмите OK.
Изменить существующий виджет
  1. На панели инструментов виджета, нажмите кнопку Настройки .
  2. Произведите необходимые изменения.
  3. Нажмите ОК.
    Виджет отобразится с изменениями.

      • Заголовок - Заголовок, которые отобразится на панели заголовка виджета.

      • Подпись - Текст, который отобразится ниже заголовка, для предоставления дополнительной информации.

      • Сущность - Сущность из которой вы хотите получить данные. Например, Сделки. Все сущности, имеющие списочное представление доступны для просмотра в виде графиков и списков.

      • Группа - Группа, из которой вы хотите просмотреть данные. Например, Все сделки. Все группы, доступные пользователю, могут быть использованы в графиках и списках.

      • Показатель - Данные, которые вы хотите просмотреть. Например, Вероятность. Все диапазонные фильтры и фильтры различных значений доступны в качестве показателей. 

        Доступные показатели зависят от доступных фильтров сущности и других настроек. Для более подробной информации, обратитесь к вашему системному администратору.

Настроить виджет списка группы
  1. В диалоге Настройка списка группы, введите заголовок виджета.
  2. Выберите Сущность и Группу из выпадающих списков. Например, Сделки и Закрытые и выигранные.
  3. Введите Количество записей , которое будет отображаться в списке. Например, 10, 25, or 50.
  4. Введите количество записей, которые вы хотите видеть в поле Видимые записи; 10, 25 или 50.
    Например, если вы ввели 50 записей на шаге 2, но хотите видеть только десять записей одновременно, введите 10. Виджет изменит размеры, под количество видимых записей.

    		5.

    Вы можете использовать любое число от 1 и до настроенного максимального количества записей для отображения. Максимум по умолчанию - 50.
  5. Нажмите ОК.
    Виджет отобразит список.

Чтобы показать или скрыть столбцы в списке групп, щелкните правой кнопкой мыши заголовок любого столбца и выберите столбцы, которые вы хотите добавить или удалить.
Настроить виджет списка ссылок
  1. В диалоге Настройка ссылок, введите заголовок виджета.
  2. Нажмите Добавить ссылку.
  3. Дважды щелкните на поле URL иконки и введите путь до иконки, которая будет отображаться для ссылки. Например, /google.jpg.

    		4.

    Этот URL должен быть относительным путем к изображению на Веб-сервере.
  4. Дважды щелкните на поле URL ссылки и введите URL для ссылки, которую хотите отобразить. Например, http://localhost/SlxClient/Account.aspx.

    5. Этот путь может быть относительным путем к порталу SLXClient или абсолютной ссылкой на внешний сайт. URL для внешних сайтов, должен начинаться с http://. Например, для ссылки на www.google.com вы должны ввести значение URL http://www.google.com. Иначе, браузер будет искать http://<SLXClient site>/www.google.com.
  5. Дважды щелкните на поле Текст ссылки и введите текст, который будет отображаться в виде ссылки. Например, Страница поиска Google.
  6. Дважды щелкните на поле Цель и введите идентификатор цели. Например, если вы хотите, чтобы ссылка открылась в новом окне браузера, введите Новое окно. Когда вы нажмете на ссылку, откроется новое окно/закладка браузера.

    		7.

    Если вы оставите это поле пустым, при нажатии на ссылку, она откроется в активном окне/закладке браузера.
  7. Нажмите Добавить ссылку, чтобы добавить другую ссылку или нажмите OK, чтобы закрыть редактор.
    Когда вы нажмете OK, виджет обновится и отобразит ссылки.
Настройка виджета SData
  1. В диалоговом окне Изменить настройки виджета введите название виджета.
  2. Выберите Поток SData из выпадающего списка. Настройки включают:
    • Infor Динамическая сущность - предоставляет доступ к стандартным ресурс ам сущности таким, как контакты, субъекты и заявки.
    • Infor система - предоставляет доступ к нестандартным ресурсам таким, как дела и вложения.
    • Любые каналы интеграции учета доступные Вам.
  3. Нажмите Далее.

  4. Если вы получили сообщение о том, что канал не содержит никаких ресурсов, необходимо выбрать другой канал. Если канал не содержит по крайней мере один ресурс, то для Вас отсутствуют данные для просмотра.

  5. Нажмите на выпадающий список Ресурсы и выберите тип информации, которую Вы хотите просматривать. Например, Субъекты и Дела.

    Содержание списка ресурсов зависит от канала, который вы выбрали.

  6. В поле Where Clause введите "где" положение, что было вам предоставлено. В случае, если положение предоставлено не было свяжитесь с администратором.

    Если положение не определено, все данные, к которым у вас есть доступ будут возвращены в ресурса SData. Это может привести к очень большим наборам данных.

    Создание положение где рекомендуется только для продвинутых пользователей. Для получения дополнительной информации о создании положения где, см. в разделе Создание положения где.

  7. Нажмите Далее.
  8. Выберите столбцы из ресурса SData, которые Вы хотите отобразить в виджете.
    • Выберите один или больше столбцов из первого списка и нажмите кнопку >. Для добавления всех столбцов необходимо выбрать каждый и нажать на кнопку >>.
    • Воспользуйтесь кнопками < и << для удаления столбцов .
  9. В поле Видимые записи введите значение в диапазоне от 1 до 100 для определения максимального количества отображаемых записей.
  10. Нажмите Завершить.
    После того, как настройки виджета сохранены, записи отобразились в виджете канала. Информация канала SData подвергается проверке на безопасность, таким образом, Вы можете видеть только те записи, к которым у Вас есть доступ к просмотру. Если поступило записей больше чем заданное максимальное число видимых записей, то воспользуйтесь полосой прокрутки.
Настройки виджета веб-сайт
  1. В диалоговом окне Изменить настройки виджета введите наименование виджета.
  2. В поле Адрес веб-сайтавведите или вставьте URL веб-сайта.

    URL должен быть следующего формата http://website.com

  3. Выполните одно из следующих действий:

    • Примите Высоту как есть по умолчанию .

      Она аналогично высоте списка групп с 10-ю видимыми записями.
    • Измените Высоту, задав значение в диапазоне от 100 до 1000.

      Например, изменение высоты до 500 pixels с делает виджет в два раза длиннее, а изменение высоты до 125 пикселей сделает виджет в два раза короче.
  4. Нажмите ОК.

    Веб-сайт добавлен на панель. Если веб-сайт не отображается, то веб-адрес либо ошибочен, либо недоступен.
Настройка виджета веб-новости
  1. В диалоговом окне Изменить настройки виджетавведите название виджета.
  2. В поле Адрес потока введите или вставьте URL потока.

  3. Нажмите на выпадающий список Отобразить и выберите один из следующих элементов:
    • Только заголовок - отображает только заголовок каждой записи канала.
    • Заголовок и сводная информация - отображает заголовок и краткое описание для каждой записи канала.
  4. В поле Максимум число номер в диапазоне между 1 и 25 для определения максимального количества отображаемых записей потока.

    В зависимости от канала, может отображаться меньше элементов на панели. По умолчанию установлено 10 записей.

  5. Выполните одно из следующих действий:

    • Примите Высоту как есть по умолчанию .

      Она аналогично высоте списка групп с 10-ю видимыми записями.
    • Измените Высоту, задав значение в диапазоне от 100 до 1000.

      Например, изменение высоты до 500 пикселей с делает виджет в два раза длиннее, а изменение высоты до 125 pixels сделает виджет в два раза короче.
  6. Нажмите ОК.

    Веб-новости добавлены на панель.
Минимизировать виджет
  • На панели инструментов виджета, нажмите кнопку Минимизировать .
    Содержание виджета будет свернуто. Нажмите кнопку снова, чтобы развернуть виджет.
Удалить виджет
  • На панели инструментов виджета, нажмите кнопку Закрыть .
    Виджет будет удален из интерактивной панели.
