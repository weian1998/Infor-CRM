Поиск Записей

Вы можете найти запись при помощи выбора условий поиска из справочника и ввода текста.

Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартных настроек.

Для поиска записей

  1. В поле Поиск по выберите условие поиска из выпадающего списка и введите критерии поиска.
  2. Чтобы добавить еще один критерий поиска, нажмите кнопку Добавить условие и повторите шаги 1 и 2 для каждого критерия поиска.
  3. Когда вы ввели все необходимые для поиска условия нажмите Поиск.
    Ниже отобразятся записи, удовлетворяющие заданными вами условиям.
  4. Выберите запись, которую вы искали, и нажмите Добавить выбранные или ОК.

    Некоторые поиски позволяют выбрать несколько записей.

Поиск осуществляется по записям, к которым у вас есть доступ. В зависимости от параметров, выбранных вашим администратором, вы можете также найти записи, основанные на группе, к которой они принадлежат.
