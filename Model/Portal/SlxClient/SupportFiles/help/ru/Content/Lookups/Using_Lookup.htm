Использование Поиска

Функция поиска позволяет выбирать критерии поиска из справочника и вводить текстовые данные. Необходим минимум один параметр поиска для запроса, чтобы отобразить результат. Вы можете сузить область поиска, включив несколько условий. Вы можете также сохранить результаты поиска как группу.

Чтобы выполнить поиск

  1. Выполните одно из следующих действий:
    • Откройте Детальное или Списочное представление для сущности, записи который вы хотите найти, и нажмите Поиск. (Например, Контакт)
    • В Навигационной панели, щелкните правой кнопкой мыши по сущности, записи которой вы хотите найти и выберите Поиск.... Например, Поиск Субъектов...
  2. В поле Поиск по выберите условие поиска из выпадающего списка.

    		3.

    Поиск по параметрам зависит от столбцов, входящих в шаблон поиска, которые определяются группой, выбранной вами в качестве Шаблона поиска по умолчанию в параметрах групп. Чтобы включить еще один пункт поиска в список, нужно либо обновить группу шаблона поиска по умолчанию, включив столбец, по которому вы хотите искать, либо изменить Шаблон поиска по умолчанию на группу, которая включает этот столбец.
  3. Выберите оператор условия поиска из выпадающего списка. Например, Содержит или Начинается с.
  4. В следующем окне введите критерии поиска в зависимости от выбранных условий поиска.
    Например, если вы ищете субъект с именем Abbott Ltd, нужно ввести в этом поле "ab".
  5. Чтобы добавить другие критерии поиска, нажмите на кнопку Добавить условие и повторите шаги 2 и 4 для каждого критерия, который необходимо добавить.
  6. Когда вы ввели все необходимые для поиска условия нажмите Поиск.
    Записи, удовлетворяющие критериям поиска отобразятся на закладке Результаты Поиска

Поиск осуществляется по записям, к которым у вас есть доступ. В зависимости от параметров, выбранных вашим администратором, вы можете также найти записи, основанные на группе, к которой они принадлежат.
