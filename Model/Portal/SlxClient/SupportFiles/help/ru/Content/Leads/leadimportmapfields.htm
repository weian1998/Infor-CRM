Сопоставление полей для импорта

Используйте это представление для сопоставления полей в списке импорта с полями в Infor. Вы можете сохранить информацию по импорту, такую как соответствие полей, доступ по умолчанию и источник наводки по умолчанию и использовать ее для будущих импортов.

Для того чтобы сопоставить поля

  1. Во время импорта наводок, после определения разделителя, нажмите Далее.
  2. Для того чтобы использовать шаблон импорта, который был создан ранее, нажмите стрелку вниз в поле Шаблон сопоставления полей и выберите значение из списка.

    В шаблоне сохраняются следующие параметры: доступ по умолчанию, источник наводки по умолчанию, разделитель и сопоставление полей. При выборе шаблона, выбранные вами доступ по умолчанию и источник наводки будут заменены на значения из шаблона. Если разделитель из шаблона отличается от разделителя, который вы указали, то вам будет предложено выбрать другой шаблон или нажать кнопку Назад и выбрать другой разделитель.

  3. Для отображения всех полей в обоих списках, установите флажок Показать все поля.
  4. Выберите поле из списка и поле из Infor для сопоставления. Нажмите Соответствует.
    Сопоставленные поля из Infor отображаются в столбце Совпадающие Infor поля. Нажмите Не соответствует, чтобы отменить сопоставление между двумя полями.
  5. Повторите шаги 3 и 4 для каждого поля, для которого необходимо настроить соответствие.
  6. Если вы хотите сохранить параметры импорта как шаблон, нажмите Сохранить как шаблон импорта.

    Для того чтобы обновить существующий шаблон импорта, выберите пункт Сохранить шаблон импорта.

  7. Нажмите Далее для перехода к следующему шагу.
