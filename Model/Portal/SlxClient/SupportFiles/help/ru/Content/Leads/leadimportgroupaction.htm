Выбор групповых действий после Импорта

Используйте это представление, чтобы указать, какие действие должны быть проведены с импортированными наводками. Вы можете добавить заметку, добавить записи к маркетинговой программе, создать отклик или назначить дело.

Чтобы выбрать

  1. При импорте наводок, после настроек для дублирующихся записей, нажмите Далее.
  2. Для добавления заметки, рядом с полем Добавить заметку, нажмите Определить и введите нужные комментарии. Установите флажок в поле Активный.
  3. Для добавления отклика, не связанного с программой, рядом с полем Добавить отклик, нажмите кнопку Определить и заполните информационные поля отклика. Установите флажок в поле Активный.
  4. Для добавления отклика к программе, рядом с полем Добавить цель программы нажмите кнопку Определить и выберите программу. Установите флажок в поле Активный.
  5. Для назначения телефонного звонка, нажмите кнопку Планировать звонок и заполните информационные поля звонка. Установите флажок в поле Активный.
  6. Для назначения встречи, нажмите кнопку Планировать встречу и заполните информационные поля встречи. Установите флажок в поле Активный.
  7. Для назначения действия, нажмите кнопку Планировать действие и заполните информационные поля действия. Установите флажок в поле Активный.
  8. Нажмите Далее для перехода к следующему шагу.
