Импорт Наводок - проверка на дубликаты

Используйте это представление для просмотра числа потенциальных дубликатов и ошибок, прежде чем импортировать список наводок. Если у вас есть большое количество потенциальных дубликатов, настройте фильтры совпадений так, чтобы были найдены только повторяющиеся записи.

При импорте большого списка, только первые сто записей могут быть проверены на дубликаты.

Для поиска

  1. В процессе импорта наводок на шаге Обработка дублей, нажмите Запустить Тест.
  2. Проверьте следующую информацию:
  • Найденные наводки
  • Потенциальные дубликаты
  • Доля дубликатов
  1. После завершения обработки, нажмите ОК.
