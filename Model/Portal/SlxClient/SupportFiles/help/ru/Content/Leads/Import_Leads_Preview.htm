Определение разделителя

Используйте это представление, чтобы выбрать разделитель записей в вашем списке и просмотреть записи перед импортом в Infor. Разделитель отделяет поля в списке. Infor анализирует список чтобы определить, какой разделитель используется в настоящее время, но вы можете выбрать другой разделитель в случае необходимости.

Для определения разделителя и предварительного просмотра

  1. Во время импорта наводок, после настройки полей, нажмите Далее.

  2. Если нужно изменить Разделитель, выберите один из следующих значений:

    • Табуляция
    • Точка с запятой
    • Запятая
    • Пробел
    • Другой
  3. Если вы выбрали Другой, необходимо указать разделитель, используемый для разделения полей в списке. Введите символ-разделитель в поле Другой.
  4. В поле Квалификатор текста, нажмите на стрелку вниз и выберите квалификатор из списка.
    Квалификатор текста определяет текст, который должен быть импортирован в точности так, как есть.
  5. Если список содержит наименования полей отметьте флажок Первая строка содержит наименования полей для их отображения в поле Предпросмотр.

    Если в импортируемом файле есть символы, некорректно отображающиеся в поле Предпросмотр, нужно пересохранить импортируемый файл и определить тип кодировки UTF-8, Unicode, или ANSI. После пересохранения файла, нажмите кнопку Назад, чтобы вернуться к шагу Выбрать файл и выбрать заново обновленный файл перед тем, как продолжать импорт.

  6. Нажмите Далее для перехода к следующему шагу.
