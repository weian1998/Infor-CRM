Информационные поля Дел, назначаемых при импорте Наводок

Используйте это представление для назначения дел каждой импортированной наводке. Полный текст дела, дата и время будут отображаться на закладке Дела на Детальном представлении.

Для того чтобы создать дело

  1. При импортировании наводок создайте групповое дело и выберите Назначить телефонный звонок, Назначить встречуили Назначить действие.
  2. Для Время начала нажмите кнопку Календарь для выбора даты, а затем нажмите OK.  
  3. Если дело не относится к какому-либо моменту времени, то вместо этого отметьте флажок В течение дня.
  4. В поле Длительность введите число и выберите Минуты, Часы или Дни из выпадающего списка для определения необходимого для выполнения дела времени.
  5. Установите флажок Автоматически перенести на следующий день если вы хотите переносить дело на следующий день, если оно не завершено в назначенное время. Опция активна, если установлен флажок В течение дня.
  6. Если вы хотите чтобы вам напомнили о деле перед его началом, установите флажок Напоминание, введите значение и выберите Минут(ы), Час(ы) или Дни из выпадающего списка для определения, за какое время до начала дела должно появиться напоминание.
  7. В поле Тема нажмите и выберите элемент из списка.
  8. В поле Приоритет нажмите и выберите элемент из списка.
  9. В поле Категория нажмите и выберите элемент из списка.
  10. В поле Заметки ведите дополнительную информацию о данном деле.
  11. Нажмите Сохранить.

Пользователь, для которого будет назначено дело, определяется отдельно для каждой импортированной наводки.
