Информационные поля Наводки

В таблице ниже описаны основные поля.

 

Поле Описание/Действие
Имя Полное имя Наводки. Нажмите на кнопку Редактировать , для того чтобы открыть диалог Изменить имя и внесите изменения с помощью этого диалогового окна.
Компания Название компании Наводки.
Веб Веб-адрес наводки. Щелкните на поле и внесите изменения. Для того чтобы открыть Веб-сайт, нажмите кнопку WWW.
Должность Должность Наводки. Нажмите и выберите элемент из списка.
Адрес

Адрес местоположения Наводки. Нажмите на кнопку Редактировать для того чтобы открыть диалог Редактирования Адреса.

Для просмотра адреса на карте нажмите кнопку MapQuest..
Отрасль Отрасль, в которой работает Наводка. Нажмите и выберите элемент из списка.
Код SIC Код отрасли согласно классификации.
Комментарии Дополнительная информация по Наводке.
Эл. почта Адрес электронной почты Наводки.Щелкните в поле и введите любые изменения. Для того чтобы отправить Наводке сообщение электронной почты, нажмите на кнопку Эл. почта.
Телефон Телефонный номер Наводки. Щелкните в поле и введите любые изменения.
Справочный тел. Справочный телефонный номер Наводки. Щелкните в поле и введите любые изменения.
Описание бизнеса Описание бизнеса Наводки.
Интересы в Интересы или хобби Наводки.
Источник наводки Как была создана запись по Наводке.
Доступ Пользователь или команда пользователей, которым предоставляется доступ к наводке. Нажмите кнопку Поиск для того, чтобы найти пользователя или команду внутри своей организации, которые обладают правами на доступ к информации о наводке.
