Добавление заметок по импортируемым наводкам

Используйте это представление для добавления заметки каждой импортированной наводке. Полный текст заметки, дата и время появляются на закладке Заметки на Детальном представлении.

Чтобы добавить

  1. В процессе импорта наводок создайте групповое действие и нажмите кнопку Определить рядом с полем Добавить заметку.
  2. При необходимости, измените значение в поле Дата. Для изменения даты нажмите кнопку Календарь для выбора даты, а затем нажмите OK.  
  3. Если заметка не относится к какому-либо моменту времени, то вместо этого отметьте флажок В течение дня.
  4. Для ввода времени в поле Длительность введите число и выберите Минуты, Часы или Дни из выпадающего списка для определения времени, необходимого для выполнения дела.
  5. В полях Тема и Категория нажмите и выберите элемент из списка .
  6. В поле Заметки введите соответствующую информацию.
  7. Нажмите Сохранить.
