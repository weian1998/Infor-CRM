Использование Детального представления Истории импорта Наводок

Используйте Детальное представление Истории импорта для просмотра детальной информации о процессе импорта наводок.

Как выполнить?

Открыть Детальное представление Истории импорта Наводок
  • Выполните одно из следующих действий:
    • После импорта наводок, в представлении Выполнение запроса нажмите Просмотреть детали.
    • В Списочном представлении Истории импорта наводок нажмите на Номер импорта, который вы хотите просмотреть.
Открыть Списочное представление Истории импорта Наводок из Детального представления.
  • В Детальном представлении Истории импорта Наводок нажмите Представление в виде списка.
Остановить процесс импорта
  • Нажмите Прервать.

Информационные поля

Информационные поля содержат основную информацию по импорту наводок.

  • Номер импорта - Уникальный номер, идентифицирующий импорт наводок.
  • Описание - Название списка импортируемых записей.
  • Всего записей - Количество наводок в списке импорта.
  • Количество обработанных записей - Количество наводок, которое обработано. Если импорт еще продолжается, в поле показывается сколько записей было обработано на текущий момент.
  • Всего импортировано - Количество импортированных наводок.
  • Всего ошибок - Количество наводок, которое не было импортировано из-за возникновения ошибок.
  • Всего предупреждений - Количество предупреждений во время импорта.
  • Всего дубликатов - Количество дубликатов в импорте.
  • Всего объединено - Количество записей, объединенных во время импорта.
  • Начал - Пользователь, запустивший импорт.
  • Статус - Статус импорта. Например, Завершенный.
  • Дата начала - Дата, когда импорт был запущен.
  • Дата завершения - Дата, когда импорт был завершен.
  • Шаблон импорта - Шаблон, использованный при импорте.

Закладки Детального представления

Каждая закладка содержит определенную информацию, которую вы можете использовать для связанных с субъектом дел. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Дубликаты История

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
