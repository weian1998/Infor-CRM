Что такое отчеты?

Представление Отчеты предоставляет доступ к набору предопределенных запросов к базе данных, которые представляют собой наиболее часто используемые бизнес отчеты. Доступность отчетов определяется вашим системным администратором и, как правило, соотносится с потребностями вашего бизнеса.

Infor CRM Веб-отчетность позволяет вам:

  • Выбор из списка заранее определенных образцов отчетов и применение фильтров для сокращения списка результатов
  • Планирование запуска отчет на более позднее время, прямо сейчас или по расписанию
  • Просмотр списка отчетов и времени запланированного запуска
  • Просмотр, печать, хранение отчетов

Для открытия представления Отчеты

  • На навигационной панели нажмите Отчеты.

Для запуска веб-отчетов в удаленном режиме, Автономный веб-клиент должен быть запущен на платформе Window 2003 Server с установленным IIS. Infor CRM сервер веб-отчетов и Crystal сервер отчетов также должны быть установлены и настроены. Если эти условия не выполняются, то пользователи отсоединенных Веб-клиентов смогут запустить Веб-отчетность только, если у них есть подключение к хост-машине на которой установлен сервер веб-отчетности. Такие пользователи будут видеть данные в отчетах из базы данных хост-машины, а не из своей удаленной базы данных.
