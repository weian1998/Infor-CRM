Мастер планирования Отчета

Мастер Планирование отчетов позволяет изменить параметры отчета, чтобы регулировать содержание выбранного отчета для удовлетворения ваших потребностей.

Изменение параметров

  1. В диалоговом окне Выбор записей в поле Отобразить записи, которые соответствуют, выберите одно из следующего:
  • Все: включает в себя все записи для выбранного типа отчета.
  • Особые условия: позволяет добавлять параметры фильтра в отчет.
  • Admin:
  1. Если доступно в списке Выбор Соответствия, определитесь, хотите ли Вы фильтровать по всем параметрам или Вы выбираете по любому из параметров..
  • Все Возвращает все записи, которые точно соответствуют всем параметрам

  • Некоторые Возвращает записи, которые соответствуют любому из параметров фильтра.

  1. Если доступно в заголовке сетки нажмите Плюс чтобы добавить параметры фильтра, используя диалоговое окно Добавить / Изменить условия, и затем нажать OK.

  2. Нажмите Далее.

  3. Если доступно в диалоговом окне Введите значения выберите из доступных параметров для удовлетворения Вашим потребностям.

  4. В диалоговом окне Параметры экспорта / Параметры настройки и планирования экспорта выполните следующие действия:

    1. В поле Описание, выберите текст по умолчанию или введите свой собственный.

    2. Этот текст появится на закладке Планирование Отчетов в столбце Описание.

    3. Выберите формат файла для отчета. Опции варьироваться в зависимости от типа задачи, но могут включать PDF, Word, Excel, CSV и XML.

    4. (Только для администратора) Используйте поле Запуска от имени для запуска отчетов другого пользователя. Используйте поиск, чтобы выбрать пользователя.

    5. Если вы выбрали расписание на шаге 1, можно создать расписание задания для запуска отчета. Выберите одну из доступных функций планирования.

    6. Нажмите Закончить.
