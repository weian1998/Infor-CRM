Запуск отчета

Отчеты позволяют вам просмотреть информация о контактах, субъектах, сделках, заявках и другую информацию.

Чтобы запустить отчет

  1. На навигационной панели, нажмите Отчеты.
  2. В Списочном представлении отчетов, закладка Отчетывыберите отчет, нажмите правой кнопкой мыши и выберите один из следующих пунктов меню:
    • Удалить: удаление выбранного отчета из списка.
    • Запуск: Выберите этот вариант, для отчетов, которые вы хотите запустить только один раз, или отчетов, которые хотите запустить немедленно.
    • Планировать: Выберите этот вариант, чтобы настроить день, дату и время, чтобы запуск отчета и повторения. Используйте эту опцию для отчетов, которые должны регулярно запускаться или для больших отчетов.

  3. Сделайте выбор с помощью мастера планирования отчетов, а затем нажмите на кнопку Готово.

  4. Когда появится окно с сообщением о ходе выполните одно из следующих действий:
    • Подождите, пока задание завершиться и откройте отчет по ссылке, которая отобразится.
    • Нажмите Закрыть чтобы закрыть окно. Отчет продолжит обрабатываться в фоновом режиме. Когда отчет станет доступен, появится предупреждение в строке меню. Вы можете либо просмотреть отчет из поля Уведомления о работах или на закладке История отчетов.
