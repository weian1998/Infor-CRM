Фильтрация данных в Отчетах

Когда вы запускаете отчет, вы можете отфильтровать входящие в него данные. Фильтры состоят из одного или нескольких условий. Например, вы можете задать условия для включения только определенных субъектов или групп в отчет. Вы можете сохранить фильтры для того, чтобы затем можно было быстро применять их при каждом составлении отчета. При необходимости, Администраторы могут просмотреть формулы условий отчетов для разрешения возникающих проблем.

Чтобы создать фильтр

  1. Выберите отчет, который вы хотите составить, и нажмите Изменить.

Условия фильтрации появятся в правой части экрана.

  1. Из списка Условия выполняются выберите одно из следующих значений:
  • Все - для соответствия всем указанным условиям.
  • Любое - для соответствия одному и более указанным условиям.
  1. Выберите условие в списке Поиск по (как описано ниже) и введите всю остальную необходимую информацию для условия. Нажмите кнопку плюс (+) для добавления условия в фильтр. Вы можете добавить столько условий, сколько вам нужно.

    		2.

    Администратор выбирает поля, используемые для вычисления данных и условий фильтра пользователя. Если вы используете эти фильтры и ваш отчет не содержит необходимой вам информации, это может быть связано с полями, используемыми для вычислений. Для получения подробной информации обратитесь к своему администратору.
    Поиск по Комментарий
    Группа

    Фильтрация записей с помощью предопределенной группировки. Результат может отличаться, в зависимости от выбранного типа отчета. Некоторые отчеты не допускают фильтрации по Группе.
    Временной диапазон

    Фильтрация отчета на основании заданного периода или промежутка времени.

    Пользователь Фильтрация записей на основании типа пользователя или имени пользователя.
    Запрос Фильтрация записей на основании выбранных или введенных Таблицы, Поля, Оператора и Значения.
  1. (Дополнительно) Введите название в поле Сохранить условие как, а затем нажмите Сохранить.

Для удаления условий из фильтра

  • Выполните одно из следующих действий:
    • Нажмите удалить условие (красный X) напротив условия, которое хотите удалить.
    • Чтобы удалить все условия, нажмите Очистить все.
