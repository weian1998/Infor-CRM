Добавление и изменение условий отчета

Условия отчета доступны для изменения для изменения содержания отчета перед созданием, чтобы сгенерировать более полезные данные. Настройки доступны для Вас в зависимости от выделений, которые Вы сделаете.

Соответствия  
Диапазон дат

Из выпадающего списка выберите один или несколько периодов.

Группа

Группы это коллекции записей. Вы можете использовать группы, чтобы работать с подмножествами записей.

Воспользуйтесь поиском,чтобы выбрать из списка групп.

Запрос

Из выпадающих списков выберите следующее:

  • Таблица,
  • Поле,
  • Оператор,
  • Значение: введите значение или выберите дату.

Например.

Таблица: Сделки

Поле: CREATEDATE

Оператор: Is After

Дата:  October 1.
Пользователь

Воспользуйтесь поиском,чтобы выбрать из списка пользователей.
