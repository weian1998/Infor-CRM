Изменение Плана продаж

План продаж - это общий доход, которые будет получен по сделке. По умолчанию, план продаж - это сумма дохода от всех связанных продуктов.

Можно отключить автоматическое вычисление плана продаж по сделке и вручную добавить нужную сумму. Возможно, это необходимо при интеграции с учетной системой, которая управляет продуктами для вашей компании.

Когда вы отключаете автоматическое вычисление и вручную вводите значение, все вычисления, использующие план продаж, не обновляются.

Для того чтобы изменить

  1. В Информации по сделке, нажмите ссылку План.

  2. Выполните одно из следующих действий:

  • Выберите флаг Отменить расчетный план продаж и введите значение в поле План.
  • Снимите флаг Отменить расчетный план продаж.

Снятие этого параметра обновит все вычисления в соответствии с бизнес правилами сделки.
