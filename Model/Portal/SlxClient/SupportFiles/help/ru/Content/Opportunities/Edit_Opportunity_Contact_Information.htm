Изменение Контакта Сделки

После добавления контакта к сделке, вы можете изменять контакт, добавляя необходимую информацию. Однако, вы не можете изменить контакт или должность контакта.

Для того чтобы изменить

  1. Откройте диалог Редактирование Контакта Сделки.
  • На закладке Контакты Детального представления Сделки, щелкните по ссылке Изменить напротив контакта, который необходимо изменить.
  • В диалоге Добавить Сделку щелкните ссылку Изменить рядом с контактом, который вы хотите изменить.
  1. Установите флажок Основной Контакт для пометки этого контакта как основного контакта для сделки..

Если более одного контакта помечены как основные, то последний выбранный контакт станет единственным основным контактом.
  1. В поле Роль нажмите и выберите элемент из списка.
  2. Рядом с полем Предпочтительный конкурент нажмите кнопкуНайти выберите конкурента, которого вы хотите добавить, и нажмите OK.
  3. В поле Оценка предложения нажмите и выберите элемент из списка.

  4. В поле Стратегия введите краткое описание стратегии конкурента (например, стоимость или простота).

  5. В поле Личные достоинства, введите любые заметки, относящиеся к этому контакту.t.

  6. В поле Проблемы введите любую другую информацию, которую вам нужно указать для этого контакта.

  7. Нажмите ОК.
