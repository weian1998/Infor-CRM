Закладка Маркетинг Сделки

Используйте закладку Маркетинг на Детальном представлении Сделки для просмотра и добавления программ, связанных со сделкой.

Закладка включает в себя следующие колонки: статус, код программы, название программы, стадия, дата начала, дата окончания.

Для просмотра программ, связанных с текущей записью

  • На детальном представлении Сделки, перейдите к закладке Маркетинг. Если закладка не отображается, то откройте закладку Все закладки.

Для связывания программы со сделкой

  • На закладке Маркетинг нажмите кнопку Связать Программу, а затем выберите программу для связывания со сделкой.

Для разрыва связи программы со сделкой

  • На закладке Маркетинг нажмите кнопку Удалить рядом с программой, связь с которой вы хотите удалить.
