Продукты Сделки

Используйте это представление для добавления, редактирования и удаления продуктов из сделки. Вы также можете изменить порядок продуктов в списке.

Как выполнить?

Открыть закладку
  • Закладка отображается на форме Создания сделки и в детальном представлении Сделки. Если закладка не отображается, то откройте закладку Все закладки.

Добавить продукт к сделке

Удалить продукт из Сделки

Редактировать информацию о Продукте

Редактировать информацию о продукте в таблице
  • Выберите запись, которую необходимо изменить, внесите изменения в таблицу, а затем нажмите Сохранить. Чтобы отменить изменения, нажмите кнопку Сброс до нажатия кнопки Сохранить.

    Не все поля могут быть отредактированы в таблице. Чтобы определить, может ли столбец быть отредактирован, щелкните таблицу. Если поле белое, то столбец может быть отредактирован.

Представление содержит следующие колонки:

  • Строка № - отображает номер строки продукта. Это полезно при печати заказа, чтобы убедиться, что продукты следуют в правильном порядке.(Только для Интеграции учета)
  • Продукт - отображается название продукта.
  • Семейство - отображает семейство продуктов.
  • Уровень Цены - отображает используемый для этого продукта уровень цен. Щелкните в таблице для редактирования.
  • Цена (Основная Валюта) - отображает цену продукта. Эта колонка основана на типе валюты по умолчанию, определенному вашими администратором.
  • Скидка - позволяет ввести скидку в процентах. Щелкните в таблице для редактирования.
  • Догов. Цена - отображает базовую цену продукта, минус скидка. Щелкните в таблице для редактирования.  Позволяет ввести новую уточненную цену. Если введена скорретированная цена, то скидка будет расчитана заново в соответствии со скорректированной ценой. Эта колонка основана на типе валюты по умолчанию, определенному вашиим администратором.
  • Цена (Валюта Сделки) - отображает цену продукта в пересчете на валюту сделки. (только для режима поддержки Нескольких Валют).
  • Количество - отображает количество продукта, добавленного в сделку. Щелкните в таблице для редактирования. Позволяет ввести новое количество.
  • Единица - Отображает единицы, используемые для измерения количества продукта. Например, "Штуки" или "Коробка из 10 шт". (Только для Интеграции учета)
  • Расширенный (Основная Валюта) - отображает произведение цены продукта на его количество без учета скидок. Эта колонка основана на типе валюты по умолчанию, определенному вашиим администратором.
  • Расширенный (Валюта Сделки) - отображает произведение цены продукта на его количество без учета скидок, в пересчете на валюту сделки. (только для режима поддержки Нескольких Валют).
