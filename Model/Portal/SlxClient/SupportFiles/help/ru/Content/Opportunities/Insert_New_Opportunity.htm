Добавление новой Сделки

Используйте view Создать Сделку для ввода информации по сделке и добавления, редактирования и удаления продуктов, конкурентов или контактов. Вы можете также обновить валюту сделки и курс при включенной поддержке нескольких валют.

Добавить новую сделку

  1. В меню Создать выберите пункт Сделку.
  2. Субъект - Нажмите кнопку Найти, чтобы найти субъект, связанный с сделкой.
  3. Введите информацию в полях Сделки.
    В зависимости от ваших настроек сделок, некоторые поля могут быть уже заполнены.
    • Описание - Введите название для сделки.
      Чтобы отслеживать сделки, используйте имя субъекта в названии сделки
    • Субъект - Нажмите кнопку Поиск для поиска субъекта, чтобы связать его со сделкой.
    • Менеджер Субъекта - Нажмите кнопку Поиск для нахождения пользователя или команды внутри своей организации, отвечающего за взаимоотношение с субъектом.
    • Партнер - Нажмите кнопку Поиск, для поиска посредника, связанного со сделкой.
    • Дата закрытия (план) - Нажмите кнопку Календарь , выберите дату ожидаемого закрытия сделки, а затем нажмите ОК.
    • Процесс продаж - выберите утвержденный процесс продаж, который используется для сделки.

      Поле Процесс Продаж по Умолчанию будет недоступно, если не установлен бандл Расширения Infor.

    • Статус - Ваша компания определяет список статусов сделки. Нажмите , для выбора значения из списка.
    • Тип - Нажмите , и выберите тип сделки из предлагаемого списка.
    • Источник наводки - Нажмите кнопку Поиск, для выбора основного источника по сделке.
    • Вероятность Закрытия - Нажмите , и выберите из предлагаемого списка вероятность закрытия сделки.
  4. Добавьте продукты, конкурентов, или контакты.
  5. Нажмите Сохранить, чтобы сохранить сделку и закрыть окно, или нажмите Сохранить и Новая, чтобы сохранить текущую сделку, а затем открыть добавление новой сделки.

    		6.

    Используйте блок Информация по сделке на Детальном представлении Сделки для добавления плана продажи.
