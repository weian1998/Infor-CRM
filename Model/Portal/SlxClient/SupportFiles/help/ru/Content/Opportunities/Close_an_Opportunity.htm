Закрытие сделки

Когда вы будете готовы, чтобы закрыть сделку, вы можете изменить статус сделки в детальном представлении сделки на Закрыта и выиграна или Закрыта и проиграна. Рекомендуется, закрывать сделку, даже если он не привела к продаже. В целях последующего контроля и отчетности, вы должны указать причину для закрытия сделки. Используйте поле Комментарии по сделке, чтобы предоставить более подробную информацию о причинах проигрыша или выигрыша продажи.

Чтобы закрыть сделку, которая была выиграна
  1. Откройте детальное представление Сделки, которую вы хотите закрыть.
  2. В поле Статус нажмите , и выберите Закрыта и выиграна из списка.
    Откроется диалоговое окно Завершение сделки.
  3. При необходимости, измените Фактическую сумму.
  4. Если необходимо, измените дату Закрыта (факт). Чтобы изменить дату, нажмитекнопку Календарь , выберите новую дату, а затем нажмите кнопку OK.
  5. В поле Причина выигрыша, нажмите кнопку с символом многоточия, выберите один или несколько элементов из списка, а затем нажмите ОК.
  6. Если необходимо, в поле Конкурент устранен нажмите кнопку Поиск для выбора конкурента, который потерял сделку. Для выбора конкурента, выберите конкурента из окна поиска конкурентов, а затем нажмите кнопку OK. Повторите этот процесс, пока не добавите всех конкурентов.
  7. В поле Комментарии, введите любые дополнительные замечания.
  8. Нажмите ОК.
Чтобы закрыть сделку, которая была проиграна
  1. Откройте детальное представление Сделки, которую вы хотите закрыть.
  2. В поле Статус нажмите , и выберите Закрыта и проиграна из списка.
  3. При необходимости, измените Фактическую сумму.
  4. Если необходимо, измените дату Закрыта (факт). Чтобы изменить дату, нажмите кнопку Календарь , выберите новую дату, а затем нажмите кнопку OK.
  5. В поле Причина поражения, нажмите кнопку с символом многоточия, выберите один или несколько элементов из списка, а затем нажмите ОК.
  6. Если необходимо, в поле Конкурент нажмите кнопку Поиск, чтобы выбрать конкурента, который проиграл сделку. Для выбора конкурента, выберите конкурента из диалога Поиск Конкурента, а затем нажмите кнопку ОК.
  7. В поле Комментарии, введите любые дополнительные замечания.
  8. Нажмите Сохранить.
