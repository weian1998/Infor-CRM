Использование Списочного представления информации о Сделках

Списочное представление содержит список Сделок, к которым Вы имеете доступ.Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Как выполнить?

Открыт детальное представление Сделки
  • На навигационной панели нажмите Сделки.
Обновить список
  • Нажмите кнопку Обновить.
