Для обновления Сделок

  1. На навигационной панели нажмите на кнопку Сделки.
  2. Откройте группу содержащую Сделки, которые Вы хотите обновить.

  3. Выберите записи для обновления и выполните одно из следующих действий:

    • Нажмите правой кнопкой мыши и выберите Обновить Сделки.
    • На Панели задач в Общие задачи, выберите Обновить Сделки.
  4. Воспользуйтесь диалогом Обновление сделок для обновления следующей информации:

    • Менеджер Субъекта

      1. В списке Обновления выберите Менеджер Субъекта.
      2. Нажмите To: кнопку Найти, sвыберите нового Менеджера Субъекта из списка и нажмите OK.
    • Добавить в Прогноз
      1. Из выпадающего списка Обновить выберите Добавить прогноз?.
      2. Выберите Да, чтобы включать возможность прогнозировать отчеты или выберите Нет, чтобы исключить возможность прогнозировать отчеты.
    • Вероятность %
      1. Из выпадающего списка ОбновитьвыберитеВероятность %.
      2. Нажмите , и выберите из списка вероятность.

    • Примерная дата закрытия

      1. Из выпадающего списка Обновитьвыберите,Примерная дата закрытия .
      2. Выберите как Вы бы хотели обновить дату:
        • Чтобы переместить предполагаемую на определенную дату, выберите To: нажмите на кнопку Calendar выберите дату и нажмите OK.
        • Чтобы переместить предполагаемую дату назад на определенное число дней, выберите опцию Перенести выберите из выпадающего списка Назад число дней, на которое Вы хотите переместить дату назад.
        • Чтобы переместить предполагаемую дату вперед на определенное число дней, выберите опцию Перенести выберите из выпадающего списка Вперед число дней, на которое Вы хотите переместить дату вперед..
    • Комментарии
      1. Из выпадающего списка Обновить выберите Комментарии.
      2. Введите Ваши заметки в поле To:.
  5. Нажмите ОК.
