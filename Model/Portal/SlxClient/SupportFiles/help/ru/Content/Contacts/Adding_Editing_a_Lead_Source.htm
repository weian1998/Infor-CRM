Детальное представление источника наводки

Каждый контакт имеет источник наводки, такие как: телемаркетинг, выставки и т. д., что позволяет формировать потенциальных покупателей. Это диалоговое окно отображает описание, дату создания, аббревиатуру, дату источника, тип и статус источника наводки, связанного с текущим контактом.

Как выполнить?

Добавить или изменить источник наводки
  1. На закладке Источники наводки детального представления Контакта,выполните одно из следующих действий:
    • Для добавления источника наводки нажмите Добавить источник наводки.
    • Нажмите на ссылку Изменить lнапротив записи, которую необходимо отредактировать.
  2. (Только добавление) По необходимости, нажмите в Описаниикнопку Поиск и выберите источник наводки.
    Ваш выбор отобразится в диалоговом окне. Поля аббревиатура, тип, дата источника и статус будут заполнены из выбранного источника наводки.
  3. По необходимости, нажмите в поле Дата источникакнопку Календарь для выбора даты, а затем нажмите OK.
  4. Нажмите ОК.
