Потенциальные дубликаты

При добавлении новых контакта или субъекта можно сделать запрос в базу данных на наличие этих записей. Если в базе данных присутствует запись с такой же информацией, это поможет избежать дублирования информации всистеме. Если добавляемый субъект все таки присутствует в системе, то вновь введенные данные добавятся к уже существующему субъекту. Если дубликатов контакта или субъекта не найдено, то можно продолжать ввод информации.

Для поиска потенциальных дубликатов

  1. Введите информацию по контакту и субъекту в диалоговое окно Контакт/Субъект d а затем нажмите кнопку Найти похожую запись.
  2. В поле Тип, выберите по какому типу записей будет осуществлен поиск.
  3. В разделе Соответствие фильтров, установите флаги напротив тех фильтров, которые вы хотите применить.
  4. Выберите один из следующих параметров:
    • Дубликаты всех выбранных фильтров - Запись является дубликатом, только если она соответствует всем выбранным фильтрам.
    • Дубликаты любых выбранных фильтров - Запись является дубликатом, только если она соответствует любому выбранному фильтру.
  5. Для запуска Расширенного поиска нажмите Расширенные параметры.
  6. Нажмите Обновить потенциальные дубликаты.
  7. Выполните одно из следующих действий:
    • В гриде Дубликаты контакта/наводки нажмите Открыть рядом с потенциальным дубликатом, чтобы просмотреть дополнительную информацию о записи.
    • В гриде Дубликаты субъекта нажмите Использовать субъект для добавления контакта к существующему субъекту.
    • Нажмите Отмена для возврата к информации о Контакте/Субъекте и продолжите создавать контакты и субъекты.
