Добавить/Удалить взаимосвязь с пользователем

Если Ваша реализация Infor включает в себя создание Контакта для каждого пользователя Infor Вы можете легко связать пользователя с контактов в списочном представлении контактов.

Как выполнить?

Связать пользователя с Контактом
  1. На панели навигации нажмите Контакты.
  2. Выберите Контакт, который требуется связать с пользователем..
  3. На Панели задач под задачами Контактов нажмите Связать с пользователем.
  4. В диалоговом окне Установить связь Контакта с Пользователем нажмите Пользователь кнопку Найти.
  5. Воспользуйтесь Поиском чтобы найти пользователя, с которым хотите установить связь, затем установите OK.
  6. Нажмите ОК.
Удалить взаимосвязь с Контактом
  1. На навигационной панели нажмите Контакты.

  2. Выберите один или несколько Контактов взаимосвязи, с которыми Вы хотите удалить взаимосвязь с любыми пользователями.

  3. На Панели задач, под Задачи контакта, нажмитеУдалить связь с Пользователем.
  4. Нажмите ОК.
