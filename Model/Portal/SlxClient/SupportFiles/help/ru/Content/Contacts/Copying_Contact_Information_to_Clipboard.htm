Копирование Контактной информации в буфер обмена

Информация помещается в буфер обмена чтобы ее можно было использовать в других приложениях. Это успешно прменяется, если вам нужно скопировать информацию в сообщение электронной почты или документы Microsoft Word.

Эта функция не доступна, если вы используете в качестве браузера Firefox.

Для копирования данных контакта из представления Контакта

  1. Откройте Детальное представление контакта.
  2. При возможности, нажмите на кнопку Копировать информацию в буфер обмена.
  3. В открывшемся диалоговом окне Копировать текст в буфер обмена выберите текст, который вы хотите скопировать, и нажмите клавиши Ctrl + C, а затем нажмте Закрыть.

Для копирования данных контакта из представления Субъекта

  1. Откройте Детальное представление субъекта.
  2. При возможности, нажмите на кнопку Копировать информацию в буфер обмена.
  3. В открывшемся диалоговом окне Копировать текст в буфер обмена выберите текст, который вы хотите скопировать, и нажмите клавиши Ctrl + C, а затем нажмите Закрыть.
