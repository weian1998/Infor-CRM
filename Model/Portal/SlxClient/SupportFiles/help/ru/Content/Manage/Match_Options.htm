Параметры совпадения

Используйте диалог Параметры совпадения, чтобы установить расширенные параметры фильтра для Мастера Дубликатов. Параметры, установленные здесь, применяются каждый раз при проверке данных на дубликаты.

Параметр Определение
Дубликат Введите число, определяющее набор результатов, которые быстрый поиск помечает как дубликаты. Например, если набор между 50 и 100, любые записи, соответствующие 50% значений, будут возвращены как дубликаты.
Возможный дубликат Введите число, определяющее набор результатов, которые быстрый поиск помечает как возможные дубликаты.
Не дубликат Отображает набор результатов, который быстрый поиск отмечает как не дубликаты. Это число определяется значениями Дубликат и Возможный дубликат.
Использовать морфологический поиск Выберите этот параметр, чтобы другие грамматические формы слова рассматривались как дубликаты.
Использовать звуковой поиск Выберите этот параметр, чтобы слова, похожие по звучанию, рассматривались как дубликаты.
Использовать синоним Выберите этот параметр, чтобы синонимы слова рассматривались как дубликаты.
Использовать нечеткий поиск Выберите этот параметр, чтобы слова, написанные по-разному, рассматривались как дубликаты. После выбора этого параметра, установите Уровень нечеткости, чтобы определить, сколько отличий в символах может иметь слово и рассматриваться при этом как дубликат.
