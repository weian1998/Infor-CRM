Проверить на дубликаты

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Мастер дубликатов сравнивает записи в Infor для определения, содержат ли данные дубликаты. Мастер создает задачу, содержащую возможные дубликаты. Можно использовать результаты задачи для объединения или удаления дубликатов.

Быстрый поиск может быть настроен для использования Проверки на дубликаты.

Чтобы запустить

  1. В меню Инструменты выберите пункт Проверить на дубликаты.

  2. На экране Выбрать источник:

    1. В выпадающем списке Выберите тип задания выберите тип сущности, которую вы хотите проверить на дубликаты.
    2. В списке Группа выберите группу записей в выбранном типе сущности.
    3. Нажмите Далее..
  1. На экране Параметры поиска выберите фильтры, которые должны использоваться при поиске соответствий.
    Если вы выбираете значение, данные в обеих записях должны в точности соответствовать, чтобы считаться дубликатом.
  2. Нажмите Расширенные параметры соответствия для настройки дополнительных параметров соответствия.
  3. Нажмите Далее
  4. На экране Обзор убедитесь в правильности вашего выбора для этой задачи.
  5. Нажмите Передать.
  6. Чтобы отменить запущенную задачу, нажмите Прервать.
  7. При необходимости, нажмите ссылку Номер задачи чтобы открыть Детальное представление Проверить на историю дубликатов и просмотреть потенциальные дубликаты.
