Добавление источника наводки в список

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Вы можете добавлять основные источники, такие, как показы и объявления, в список доступных основных источников.

Для того чтобы добавить

  1. В меню Администрирование, нажмите Создать источник наводки.
  2. Заполните поля основного источника:
  • Описание - заполните название основного источника.
  • Сокр. Описание - заполните название основного источника.
  • Тип - нажмите кнопку и выберите элемент из списка.
  • Статус - нажмите кнопку и выберите элемент из списка.
  • Стоимость - введите цену, связанную с производством основного источника.
  • Дата - Для изменения Даты нажмите кнопку календарь, далее выберите дату и нажмите OK.
  1. Нажмите Сохранить.
