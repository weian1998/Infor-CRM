Использование детального представления продукта

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

При включении интеграции учета продукты доступны только на чтение. Необходимо управлять продуктами в учетной системе и синхронизировать изменения с Infor.

Как выполнить?

Открыть Детальное представление Продукта
  • На навигационной панели нажмите Породажи, Поддержка или Администрирование, затем нажмите Продукты чтобы открыть Список Продуктов. При необходимости используйте Поиск, чтобы найти продукт и затем выберите имя нужного продукта.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать список групп Панели Задач или общие задачи

Просмотреть группы

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы

Информационные поля

  • Имя - отображается название продукта.
  • Описание - Описание продукта.
  • Единица учета - единица учета продукта.
  • Семейство - отображает семейство продуктов. Нажмите кнопку и выберите элемент из списка.
  • Статус - статус продукта. Нажмите кнопку и выберите элемент из списка.
  • Стоимость - цена продукта.
  • (Только для Интеграции учета) Единица - Отображает единицы, используемые для измерения количества продукта. Например, "Штуки" или "Коробка из 10 шт".

Закладки

Каждая закладка содержит определенную информацию, которую вы можете использовать для связанных с продуктом дел.

Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Пакеты Цены Уровни Вложения

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
