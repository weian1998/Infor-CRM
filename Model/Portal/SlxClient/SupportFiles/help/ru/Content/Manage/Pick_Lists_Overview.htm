Что такое Справочники?

Справочник - список значений, которые вы можете выбрать при вводе данных в определенное поле. Например, при вводе информации в поле Город пользователь может выбрать значение из списка городов. Узнать больше...

Справочники часто используются, т.к. они исключают случайное нажатие клавиш и обеспечивают корректный, цельный ввод данных. При синхронизации справочников, они предоставляют согласованную по всем базам данных информацию

В зависимости от атрибутов справочника, пользователи могут:

  • Выбрать один или более пунктов из списка.
  • Добавить элемент в справочник.
  • Изменить элемент справочника.
