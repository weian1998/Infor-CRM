Использование списочного представления Справочников

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Список сравочников отображает справочники в виде таблицы. Таблица предоставляет информацию о каждом справочнике.

  • Чтобы сортировать таблицу по конкретному полю, нажмите на один из заголовков столбца.
  • Чтобы отобразить справочники, отвечающие специфическим критериям, настройте фильтры.

Как выполнить?

Открыть списочное представление Справочников
  • На навигационной панели нажмите Администрирование, а затем нажмите Справочник.
Открыть детальное представление Справочника
  • Для того чтобы открыть карточку Справочника, щелкните мышкой по его названию..
Обновить список
  • Нажмите кнопку Обновить.
