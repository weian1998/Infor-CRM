Добавление справочника

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Чтобы добавить

  1. На навигационной панели нажмите Администрирование, а затем нажмите Справочник.
  2. На Панели Общие задачи, нажмите Добавить справочник.
  3. В поле Название справочника введите название справочника, затем нажмите OK
  4. В детальном представлении Справочника установите Атрибуты справочника.
  5. Добавьте пункты к справочнику с помощью закладки Элементы.
  6. Если необходимо, нажмите кнопку Тестировать список для просмотра справочника.
  7. Нажмите Сохранить.
