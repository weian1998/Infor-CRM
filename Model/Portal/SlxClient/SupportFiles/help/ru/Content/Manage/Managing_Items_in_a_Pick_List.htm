Управление Элементами в Справочнике

Можно редактировать справочник, добавляя новый элемент, редактируя существующий элемент, или удаляя элемент из справочника.

Если вы добавляете элемент к справочнику Статус сделки, и пользователь устанавливает статус на этот элемент, Снимок Детального представления Сделки ведет себя так, как если бы статус сделки был "Открыта".

Для управление

  1. Откройте список просмотра Справочника.
  2. На закладке Элементы выберите один из вариантов:
  • НажмитеДобавить чтобы добавить новый пункт.
  • Щелкните ссылку Изменить для пункта, который вы хотите изменить.
  • Щелкните ссылку Удалить для пункта, который вы хотите удалить.

Если вы используете другие приложения, использующие информацию из базы данных вместе с Infor, нужно удалить тот же пункт справочника в обоих приложениях.

  1. В диалоге Добавить/Редактировать Элемент:
  • В поле Текст введите текст, который вы хотите видеть в справочнике.
  • (Не обязательно) В Поле Код введите код для пункта.

Если вы используете другие приложения, использующие информацию из базы данных вместе с Infor, нужно использовать одни и те же пункты справочника в обоих приложениях.

  • В поле Порядок определите порядок, в котором пункт должен появляться в списке. Узнать больше...

Например, если вы введете 0, пункт появится первым в списке. Если у вас будут несколько пунктов с одинаковым порядковым номером, они будут сгруппированы по этому номеру, и вся группа появится в списке перед следующим номером. Заметьте, что если выбран атрибут Отсортировано по алфавиту, порядковые номера не учитываются.

  • Отметьте флажок по умолчанию для того чтобы сделать элемент выбором по умолчанию.
  • Нажмите Сохранить и Новый для продолжения добавления пунктов, или нажмите OK если вы закончили.
  1. Нажмите Сохранить.

    После обновления элементов справочника, вам, возможно, придется проинструктировать пользователей о том, что им будет необходимо зайти в их настройки и обновить данные справочника, чтобы увидеть последние изменения.
