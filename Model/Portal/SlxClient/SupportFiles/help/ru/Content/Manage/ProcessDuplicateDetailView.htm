Детальное представление Обработки записей-дубликатов

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление Обработка записей-дубликатов отображает информацию о задаче по проверке дублей и любые неразрешенные записи-дубликаты из этой задачи.

 

Как выполнить?

Открыть Детальное представление Обработка записей-дубликатов
  • В меню Инструменты нажмите Обработка записей-дубликатов, чтобы открыть список обработок записей-дубликатов. Для того чтобы перейти к задаче, можно использовать Поиск, после чего выберите номер задачи.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Информационные поля

  • Номер задачи - номер задачи.
  • Источник - объект в Infor, являющийся потенциальным дубликатом.
  • Тип источника - сущность, которую обрабатывала задача.
  • Статус решения - статус результата. Статус Возможный дубликат отображает, что запись не была обработана. Статус Не дубликат или Объединено отображает, что дубликат был обработан и больше не является дубликатом или был объединен с соответствующей записью.
  • Кем решено - Пользователь, который разрешил потенциальный дубль.
  • Дата решения - дата обработки записи дубликата.
  • Не дубликат - нажмите эту кнопку, чтобы пометить возможное совпадение как не дубликат.

Закладки

  • Фильтры поиска - Отображает фильтры, которые использовались при выполнении задачи.
  • Неразрешенные дубликаты - отображает неразрешенные дубликаты в задаче.
