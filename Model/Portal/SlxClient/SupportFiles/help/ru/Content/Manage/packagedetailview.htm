Использование детального представления Пакета

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартных настроек.

Как выполнить?

Открыть Детальное представление Пакета
  • На Навигационной Панели нажмите Поддержка, затем Пакеты, чтобы открыть Список Пакетов. При необходимости используйте Поиск, чтобы найти пакет продуктов и затем выберите имя нужного пакета.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать список групп Панели Задач

Просмотреть группы

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы

Информационные поля

  • Имя - отображается название пакета продуктов.
  • Описание - введите описание пакета продуктов.
  • Статус - статус пакета. Нажмите кнопку и выберите элемент из списка.
  • Дата создания - Отображает дату создания пакета продуктов.
  • Создано - Отображает имя пользователя, создавшего пакет продуктов.
  • Дата изменения - Отображает последнюю дату изменения информации о пакете продуктов.
  • Изменено - Отображает имя пользователя, создавшего пакет продуктов.

Закладки

Каждая закладка содержит определенную информацию, которую вы можете использовать для связанных с пакетом дел. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Продукты

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
