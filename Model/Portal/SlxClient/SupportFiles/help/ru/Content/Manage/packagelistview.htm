Использование списочного представления Пакетов

В Списке просмотра Пакетов отображаются все Пакеты, к которым вы имеете доступ. Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей.

Как выполнить?

Открыть списочное представление Пакетов
  •  На навигационной панели нажмите Продажи, Поддержка, или Администрирование, затем нажмите Пакеты.
Обновить список
  • Нажмите кнопку Обновить.

Открыть детальное представление Пакета
  • Для того чтобы открыть карточку Пакета,.щелкните мышкой по его названию.

Использовать список групп Панели Задач

Просмотреть группы

Использовать обычные задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы
  • Сохранить записи как группу
  • Выгрузить в файл
