Добавление продукта в список

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Можно добавить продукты в список продуктов. Этот список доступен при добавлении или редактировании продуктов сделки или продуктов заказа.

При включении интеграции учета продукты доступны только на чтение. Необходимо управлять продуктами в учетной системе и синхронизировать изменения с Infor.

Для того чтобы добавить

  1. В меню Администрирование, выберите Новый Продукт.

Откроется диалог Добавить Продукт.

  1. Заполните поля по продукту:
  • Продукт - отображается название продукта.
  • Описание - введите описание пакета продуктов.
  • Единица учета - Введите или измените единицу учета продукта.
  • Семейство - нажмите кнопку и выберите элемент из списка.
  • Статус - нажмите кнопку и выберите элемент из списка.
  • Цена - Введите или измените цену продукта.
  • Единица - Отображает единицы, используемые для измерения количества продукта. Например, "Штуки" или "Коробка из 10 шт".

  1. Нажмите Сохранить

  2. Чтобы добавить уровень цены, сделайте следующее:
  3. На детальном представлении Продукта откройте закладку Уровни Цены и нажмите Добавить цену.
  1. Введите информацию о цене продукта и нажмите OK.
  2. Повторите шаги 4а и 4b для каждого уровня цены.
