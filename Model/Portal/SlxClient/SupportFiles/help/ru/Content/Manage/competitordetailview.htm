Использование детального представления конкурента

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление используется для просмотра, добавления или изменения информации по выбранной записи.

Ваша рабочая область может выглядеть по-разному, в зависимости от прав доступа или использования нестандартных настроек.

Как выполнить?

Открыть лицевую карточку Конкурента
  • На Навигационной Панели нажмите Администрирование, затем Конкуренты, чтобы открыть Список Конкурентов. При необходимости используйте Поиск, чтобы найти конкурента и затем выберите имя нужного конкурента.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать список групп Панели Задач или общие задачи

Просмотреть группы

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы

Информационные поля

  • Конкурент - отображает название конкурента.
  • Субъект - Нажмите кнопку Поиск и выберите субъект с помощью поиска.
  • Контакт -Нажмите кнопку Поиск
  • Телефон - отображает телефонный номер конкурента.
  • Веб-Адрес - отображает Веб-Адрес конкурента.
  • Рейтинг - Введите или редактируйте рейтинг конкурента.
  • Слабые стороны - Введите или редактируйте слабые стороны конкурента
  • Достоинства - Введите или редактируйте сильные стороны конкурента
  • Стратегия - Введите или редактируйте используемую конкурентом стратегию.
  • Заметки - Введите или редактируйте заметки о конкуренте.
