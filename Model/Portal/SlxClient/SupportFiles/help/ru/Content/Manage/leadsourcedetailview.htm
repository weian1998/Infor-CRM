Использование детального представления Источника наводки

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление используется для просмотра, добавления или изменения информации по выбранной записи.

Как выполнить?

Открыть Детальное представление Источника наводки
  • На навигационной панели нажмите Администрирование, затем Источники наводки, чтобы открыть Список Источников наводки. При необходимости используйте Поиск, чтобы найти основной источник и затем выберите имя основного источника.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать список групп Панели Задач или общие задачи

Просмотреть группы

Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы

Информационные поля

  • Описание - заполните название основного источника.
  • Сокр. Описание - заполните название основного источника.
  • Тип - нажмите кнопку и выберите элемент из списка.
  • Статус - нажмите кнопку и выберите элемент из списка.
  • Стоимость - введите цену, связанную с производством основного источника.
  • Дата - Для изменения Даты нажмите кнопку Календарь/Время, далее выберите дату и нажмите OK.
