Детальное представление Справочника

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Как выполнить?

Открыть детальное представление Справочника
  1. На Навигационной Панели нажмите Администрирование, затем Справочники, чтобы открыть Список Справочников.
  2. Нажмите на справочник в списке, или используйте Поиск, чтобы найти справочник.

Информация о справочнике откроется в Детальном представлении.

Используйте Поиск для поиска записей

Открыть группу

Переключиться в режим списочного представления
  • На инструментальной панели нажмите кнопку Представление в виде списка.
Использовать общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы
  • Добавить справочник

Информационные поля

Информационные поля справочника содержат основную информацию по Справочнику.

Закладки Детального представления

Закладка Элементы используется для управления пунктами в списке.

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
