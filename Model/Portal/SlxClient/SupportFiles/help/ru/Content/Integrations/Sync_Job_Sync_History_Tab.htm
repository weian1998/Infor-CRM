Закладка История синхронизации работ

Закладка История синхронизации в списочном представлении Синхронизации работ каждая запись истории синхронизации связана с работой.

Как выполнить?

Открыть закладку История синхронизация
  • Откройте Детальное представлении Синхронизации работ и нажмите История синхронизации..
Фильтрация списка Истории синхронизации
  1. Воспользуйтесь Фильтром по имени пользователя, нажмите на кнопку Найти и выберите пользователей, записи которых Вы хотите просмотреть.

  2. В фильтре списка по виду ресурса выберите сущность для журнала.

Закладка История синхронизации работ содержит следующие столбцы::

  • Журнал истории синхронизации - открывает детальное представление Истории Синхронизации.

  • Пользователь - имя пользователя, который вносило изменение.

  • Ресурс - тип ресурса, который синхронизировался.

  • Запись - открывает детально представление записи, которая была обновлена.

  • Отправленный От - приложение, в котором было осуществлено изменение.

  • Полученный - приложение, которое получило изменение.

  • Уровень важности - тип информации зарегистрированного для синхронизации элемента истории.

  • Журнал Создает Дату - когда элемент журнала истории синхронизации был создан.

  • Ссылка Ссылка - ссылка используется для синхронизации данных между Infor и внешним источником интеграции.

  • Интеграция - какая интеграция была включена в синхронизацию информации.

  • Состояние - результат синхронизации. Возможные состояния включают:

    • Создаваемый - запись создана в приложении, которое получило изменение.
    • Конфликт-Победил - запись была обновлена ​​в в приложении, которое получило изменение в результате правил по разрешению конфликтов.
    • Конфликт-Потерял - запись была обновлена в приложении, которое получило изменение в результате правил по разрешению конфликтов.
    • Удаленные- запись была удалена приложении, которое получило изменение.
    • Никаких изменений не сделано- никакое изменение не было внесено в приложении, которое должно было получить изменение из-за ошибок.
