Информационные поля интеграции

Информационные поля интеграции содержат основную информацию об интеграции. В зависимости от интеграции, вы можете просмотреть или изменить или информацию по мере необходимости. Некоторая информация не может быть отредактирована.

Следующая информация отображается на Детальном представлении Интеграции.

Поле Описание действие
Интеграция Название интеграции.

Тип аутентификации

 

Способ аутентификации данных.

Изменение типа аутентификации может привести к ошибке интеграции.

Внешний служба должна быть разрешена одним служебным пользователем или нескольким, если метод OAuth 1 или OAuth2, интеграция включена и ID клиента и секрет указаны.
Метод синхронизации Определяет как информация синхронизируется между интегрированной системой и Infor CRM. Настройки включают Не Синхронизирует, Клиентская Синхронизация и Синхронизация Сервера. Синхронизация сервера указывает, что Служба Синхронизации Интеграции используется.
Тип Ссылки Тип ссылки, используемый для передачи данных между интегрированной системой и Infor CRM. Если Администратор или Система выбраны, то данными управляют через единственную ссылку. Если Пользователь выбран, то данные переданы для каждого пользователя через отдельную ссылку.

Секрет

OAuth1 или OAuth2 только типы Аутентификации.

Присвоенный провайдером интеграции во время процесса регистрации.

Используйте эту опцию включить или выключить синхронизацию для этой интеграции.

Если эта опция не будет выбрана, то никакие данные не будут переданы между интегрированной системой и. Это также сделает все следующие опции синхронизации недоступными.
Обработчик Синхронизации

Информация только для чтения.

Определяет обработчик синхронизации non-SData для интеграции, которая не представляет ее собственные конечные точки синхронизации SData.

Обработчик ответственен за отображение данных между контрактом и интеграцией.
Синхронизация Каждый Определяет число минут между циклами синхронизации.
Запустите синхронизацию между Определяет часы, между которыми синхронизируется интеграция.
