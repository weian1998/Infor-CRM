Использование списочного представления Интеграции

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Списочное представление интеграции содержит все источники интеграции, к которым Вы имеете доступ. Этот список включает в себя все интеграции, опубликованные, а также любые пользовательские интеграции.Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей.

Как выполнить?

Откройте списочное представление Интеграции
  • На навигационной панели нажмите Интеграция.

Работа с записями в таблице

Обновить список
  • Нажмите кнопку Обновить.

Отфильтровать список

Используйте Поиск для нахождения записей

Откройте детальное представление Интеграции
  • Нажмите на название интеграции, чтобы открыть Детальное представление Интеграции.

Просмотреть группы

Использование контекстного меню в группах

Используйте общие задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы
  • Сохранить записи как группу
  • Экспорт в файл
