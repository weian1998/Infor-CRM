Списочное представление Истории синхронизации

Списочное представление Истории синхронизации отображает информацию о циклах синхронизации между Infor и внешними источниками интеграции.

  • Чтобы сортировать сетку по определенному полю, нажмите на одном из заголовков столбцов.

  • Чтобы только вывести на экран журналы, которые соответствуют определенным критериям, используйте фильтры.

  • Чтобы вывести на экран журналы за заданный периода времени, выберите одну из предоставленных групп.

Как выполнить?

Открыть Списочное представление Истории синхронизации
  • На навигационной панели нажмите Администрирование и затем нажмитеИстория Синхронизации.
Открыть Детальное представление Истории синхронизации
  • Нажмите на имя журнала с историей синхронизации, чтобы открыть Детальное представление Истории синхронизации.

Воспользуйтесь Поиском, чтобы найти записи.
