Использование представления настройки Синхронизации SData

Используйте представление настройки Синхронизации SData, чтобы настроить канал, используемый для синхронизации с интегрированными системами.

Настройка SData

  1. На навигационной панели нажмите Конфигурация Синхронизации SData.

  2. Настройте канал Infor. Воспользуйтесь следующими полями для настройки канала. Канал будет автоматически обновляться по мере ввода данных.

    1. Если ваш URL SData использует Secure Socket Layer, выберите в поле Использовать SSL (Secure Socket Layer).
    2. В поле Сервер введите имя сервера для Вашего SData URL.
    3. В поле Порт введите номер порта для вашего SData URL.
  3. В поле Имя пользователя, оставьте "Admin" или введите логин для базы данных.

    Проверка подлинности Windows необходима в соответствии с настройками для пользователя с правами администратора.

  4. В поле Пароль введите пароль для имени пользователя.

  5. В поле Отображаемое имя, оставьте "" или введите отображаемое имя.

  6. Нажмите кнопку Проверить ссылку и просмотрите информацию в поле Статус.

    7. Если проверка канала прошла не успешно, проверьте информацию в полях и протестируйте канал снова.

  7. Нажмите Сохранить.

Закладки

Нижняя секция Конфигурации Синхронизации SData содержит следующие закладки: Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Интеграции Совпадения

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
