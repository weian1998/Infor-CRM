Управление схемой

Используйте диалог Управление схемой, чтобы сравнить схему базы данных Infor со своей схемой источника интеграции для обеспечения целостности данных. Вы можете сравнить данные для различных ресурсов, чтобы определить, какие интеграции различия в длине строк между системами.

Рассказать больше об управлении различиями схем в Infor...

MaxLength - если данные синхронизируются с Infor, при превышении максимальной длины поля, вся строка будет отклонена.

FractionalDigits - если данные синхронизируются с Infor и у данных количество знаков после запятой превышает заданное, то данные округляются, например, если поле валюта определяется как 17,4, то значение 123,12348 сохраняется как 123,1235, а значение 123,12343 сохраняется как 123,1234.

Как выполнить?

Откройте диалоговое окно Управление схемой
  • В детальном представлении Интеграции на закладке Ресурсы нажмите на кнопку Управление схемой.
Сравните схему для ресурсов
  1. В диалоговом окне Управление схемой воспользуйтесь фильтром и выберите тип ресурса.
  2. Нажмите Выполнить.
  3. Просмотрите различия в сетке
    Обратите внимание на любую информацию, которая может вызвать проблемы с целостностью данных. Например, поле системы учета может принять только три символа, когда такое же поле принимает свободный текст.

  4. Повторите эти действия для всех типов ресурсов.

  5. Нажмите Закрыть.
