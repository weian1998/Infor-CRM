Добавление или редактирование основной ссылки интеграции

Используйте диалоговое окно Добавить и Изменить ссылки, чтобы добавить или отредактировать ссылки интеграции. Если тип авторизации интеграции Основной, используйте это диалоговое окно, чтобы добавить ссылки.

Как выполнить?

Добавьте ссылку
  1. В детальном представлении интеграции нажмите на закладку Ссылки.

    		2.

    Закладка Ссылки доступна только при интеграции методом синхронизации на стороне клиента синхронизации или сервера синхронизации.
  2. Нажмите Добавить ссылку.
    Эта кнопка доступна только при интеграции Эта кнопка только доступна, если тип ссылки интеграции - Система или Администратор.

  3. В поле URL введите URL SData, который использует, чтобы обмениваться информацией между системами.

    • Воспользуйтесь одним из следующих форматов:
      • Если ваша реализация не использует порт по умолчанию (порт 80), используйте формат: http://server:port/sdatavirtualdir/application/contract/dataset/
      • Если ваша реализация использует порт 80, не указывать номер порта из URL.
        Синхронизация служба автоматически удаляет порт по умолчанию из URL, который вызывает соответствующие вопросы и ошибки. Избежать этого можно, используя формат: http://server/sdatavirtualdir/application/contract/dataset/
    • Замените значения, предоставляемые партнером по интеграции:
      •  Sdatavirtualdir - Например, sdata.

      •  application - наименование системы учета. Например, accounts50.

      •  contract - имя контакта. Например, GCRM.
      •  dataset - различает набор данных или рабочей компании. Например, / - /
  4. В поле Отображаемое имя введите имя, которое Вы хотите выводить на экран, когда происходит соединение.
  5. В поле Имя пользователя, введите имя, чтобы войти в систему интеграции.
  6. В поле Пароль введите соответствующий пароль, используемый пользователем, чтобы войти в систему интеграции.
  7. Нажмите кнопку Тестировать ссылку. Имя кнопки зависит от типа авторизации и типа ссылки.
  8. Нажмите ОК.
Изменить ссылку
  1. В детальном представлении интеграции нажмите на закладку Ссылки.

    2. Закладка Ссылки доступна только при интеграции методом синхронизации на стороне клиента синхронизации или сервера синхронизации.
  2. Нажмите на гиперссылку Изменить.
    Эта кнопка доступна, если тип ссылки интеграции - Система или Администратор.
  3. Внесите любые изменения.
  4. Нажмите кнопку Проверить ссылку.
  5. Нажмите ОК.
