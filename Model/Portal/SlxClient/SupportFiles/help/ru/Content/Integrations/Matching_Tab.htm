Закладка Совпадения

Используйте вкладку Совпадения, чтобы сконфигурировать опции для нахождения соответствий между Infor и любыми другими интегрированными системами. Опции, которые Вы выбираете здесь, определяют, как данные, введенные пользователями в Infor, будут соответствовать к существующим данным в Вашей системе учета.

Эти параметры настройки определяют критерии соответствия значения по умолчанию всех пользователей, соединяющих данные системы учета. Настройка уточняющих критериев поиска могут не возвращать достаточно результатов пользователям, и стать причиной создания копий.

Эта закладка относится только к интегрированным системам учета.

Для настройки критериев

  1. Откройте Представление конфигурации Синхронизации SData.

  2. Нажмите на закладку Совпадения.

  3. Нажмите на ссылку Изменить для ресурса, для которого необходимо определить или изменить критерии поиска совпадений.

  4. В диалоге Изменить Критерии, выполните следующие действия:

    1. В поле Свойство выберите свойство, для которого необходимо сравнивать значения.
      • В Вашем браузере введите адрес URL, созданный при настройке системы учета, следующий /$schema и убедитесь, что в свойстве атрибута canFilter значение true.

        URL пример:

        http://server:port/sdatavirtualdir/application/contract/dataset/$schema

    2. В поле Операторвыберите оператор для завершения ввода условий совпадения.

      • Равно - равно, значение свойства в каждой системе должно быть равно числу совпадений. Например, Abbott не соответствует Abbott Inc
      • Начинается с - начинается с, значение свойства в каждой системе должно начинаться с того же значения.
      • Cодержит- содержит, значение свойства в каждой системе должно содержать такое же значение. Например, Abbott будет найдено в Abbott Inc и Abbott Worldwide.
    3. Для добавления критерия поиска совпадений нажмите .
    4. Для удаления критерия поиска совпадений нажмите .
    5. Нажмите OK.
