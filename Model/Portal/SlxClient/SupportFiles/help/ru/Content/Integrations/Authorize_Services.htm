Разрешение Сервисов

Используйте закладку Параметры сервисов авторизации для разрешения соединения с внешней службой. Перед тем, как разрешить обслуживание, администратор должен настроить интеграцию и предоставить доступ для пользователей. Данные параметры применяются только тогда, когда текущий пользователь выполнил вход в Веб-клиент Infor CRM.
  • Эта функция может быть недоступна для всех пользователей. Внешние интеграции должны быть реализованы администратором.
  • Вы должны разрешить всплывающие окна, прежде чем Вы можете разрешить сервисы. Просмотрите в справке вашего браузера инструкции о том, как разрешить всплывающие окна.

Разрешить сервис

  1. В меню Сервис, выберите Параметры.
  2. Перейдите на закладкуРазрешить сервисы.
  3. Нажмите Разрешить рядом со службой, которую необходимо разрешить.

  4. Выполните несколько рекомендаций для авторизации.
