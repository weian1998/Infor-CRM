Автономный веб-клиент Infor CRM

Только для удаленных пользователей

Автономный веб-клиент работает также как и удаленный клиент Infor CRM, за исключением того, что пользователи подключаются к своим локальным базам данных используя Веб-клиент. Автономный веб-клиент позволяет вам работать отсоединенно без использования подключения к интернету.

Чтобы перенести изменения между пользовательской базой данных отсоединенного Веб-клиента и основной базой данных, используется синхронизация. Чтобы синхронизировать изменения между вашей базой данных и основной базой данных, нужно использовать клиента синхронизации. Для получения дополнительной информации смотрите онлайн справку клиента синхронизации.

Для запуска веб-отчетов в удаленном режиме, the Автономный веб-клиент должен быть запущен на платформе Window 2003 Server с установленным IIS. Infor CRM сервер веб-отчетов и Crystal сервер отчетов также должны быть установлены и настроены. Если эти требования не выполнены, то отсоединенные Веб-клиенты смогут запускать веб-отчетность только в том случае, когда они подключены к основному серверу веб-отчетности. Такие пользователи будут видеть данные в отчетах из основной базы данных, а не из их удаленной базы данных.

Каковы особенности отсоединенного Веб-клиента?

  • Представление Что нового? позволяет пользователям отсоединенного Веб-клиента подписаться и отписаться от субъектов.
  • (Только для Интеграции учета) Синхронизация между Infor CRM и поддерживаемой системой учета осуществляется на основной базе данных. Таким образом, любые изменение на отсоединенном Веб-клиенте или в системе учета должны быть синхронизованны через основную базу данных.
