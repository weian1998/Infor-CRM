Что нового?

Используйте представление "Что Нового?" для просмотра списка всех последних изменений информации, к которой у вас есть доступ. Это представление отображает список всех новых или измененных субъектов, контактов, наводок, сделок, дел, заметок, записей истории, контрактов, дефектов и возвратов. Вы можете просматривать только ту информацию, к которой у вас есть доступа.

Представление Что Нового? состоит из двух частей: группа закладок сверху, при помощи которых можно перейти к определенным типам измененных записей, таким как субъекты и область списка, в котором отображены новые или измененные элементы.

Являетесь ли вы отсоединенным пользователем?

Для того чтобы открыть представление Что нового?

  1. На навигационной панели, нажмите Что нового?.
  2. Чтобы отфильтровать записи представления Что нового? по дате, нажмите Изменения в базе данных с кнопку Календарь , выберите дату и нажмите кнопку OK, затем нажмите Поиск.

Для перехода к определенному списку новых или измененных записей

  • Выберите закладку с типом записей, которые вы хотите просмотреть. Доступные варианты: субъекты, контакты, наводки, сделки, дела, заметки, история, контракты, дефекты или возвраты.

Для просмотра подробной информации

  1. Выберите соответствующую типу записей закладку, например субъекты.
  2. Из списка, выберите запись для просмотра.
  3. Щелкните по названию записи.
