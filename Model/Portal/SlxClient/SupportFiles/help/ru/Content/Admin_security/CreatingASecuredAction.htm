Создание действия безопасности

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.Только системный администратор и пользователи с ролью администратора могут иметь доступ к этим функциям.

Вы можете создавать новые действия безопасности для защиты элементов управления в Веб-клиенте.

При создании нового действия безопасности, убедитесь, что действие имеет уникальное имя, ранее не использованное (за исключением случая, когда вы восстанавливаете случайно удаленное действие безопасности) Если вы удалили действие безопасности без ручного удаления данного действия из элементов управления в Архитекторе приложений, новое действие может предоставить доступ к не предвиденным вами элементам. Например, вы создали действие безопасности с именем Печать/Отчеты и добавили это действие к множеству кнопок Печать (используя свойство Applied Security в Архитекторе приложений) Затем, вы удалили действие безопасности Печать/Отчеты, но вручную не поменяли свойство Applied Security для кнопок Печать. Если вы создадите новое действие безопасности с именем Печать/Отчеты, все кнопки Печать, ранее связанные с действием Печать/Отчеты, будут связаны с новым действием Печать/Отчеты.

Infor рекомендует использовать описательные названия для действий безопасности, указывающие на приписанные им цели. Например, действие безопасности Сущность/Субъект/Удалить используется для контроля элементов, позволяющих пользователям удалять субъекты. Эта структура названия рекомендуется, но не навязывается к использованию. Имя действия безопасности - это строка и вы можете использовать любые значения.

Для создания действия безопасности

  1. В меня Администрирование, нажмите Новое действие безопасности.
  2. В поле Действие безопасности, введите название действия безопасности.
  3. В поле Описание, введите описание действия безопасности.
  4. Нажмите Сохранить для того, чтобы сохранить действие безопасности или Сохранить и Новый для того, чтобы сохранить действие безопасности и создать новое.
  5. Используйте закладку Роли чтобы добавить действия безопасности к роли.
  6. Используйте закладку Размещение чтобы просмотреть, где используется данное действие безопасности.
