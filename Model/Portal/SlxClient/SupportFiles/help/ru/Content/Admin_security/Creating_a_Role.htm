Создание роли

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Доступ к функционалу в Веб-клиенте разграничивается ролями. Вы можете создавать любое количество ролей и назначать их любому количеству пользователей. Кроме того, вы можете использовать роль по умолчанию.

Пользователю могут быть назначены одна или несколько ролей. Если пользователю назначены более одной роли, он/она будет иметь доступ ко всем действиям связанным с ролями, даже если каждая роль содержит различные действия. Нельзя назначить роль подразделению или команде.

Для создания роли

  1. В меню Администрирование, нажмите Новая роль.
  2. В поле Название, введите название роли.
  3. В поле Описание, введите описание роли.
  4. Нажмите Сохранить.
  5. Воспользуйтесь закладкой Действия для добавление действий.
  6. Воспользуйтесь закладкой Пользователи для добавления или удаленный пользователей.
