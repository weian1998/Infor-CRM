Детальное представление Роли

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Как выполнить?

Создать роль

Откройте детальное представление роли

  1. На навигационной панели, нажмите Администрирование, затем нажмите Роли для открытия списочного представления Ролей.
  2. Выберите роль из списка или воспользуйтесь поиском для нахождения необходимой роли.

Откроется детальное представление роли.

Используйте Поиск для нахождения необходимых записей

Открыть группу

Переключитесь на списочное представление

  • На панели инструментов, нажмите кнопку Список.
Использовать обычные задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Добавить в группу
  • Удалить из группы

Информационные поля

Информационные поля роли содержат основную информацию о роли.

Закладки Детального представления

Каждая закладка показывает определенную информацию, касающуюся роли. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
