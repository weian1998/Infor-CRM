Использование детального представления Действий безопасности

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.Только системный администратор и пользователи с ролью администратора могут иметь доступ к этим функциям.

Используйте детальное представление для просмотра, добавления или редактирования информации о выбранной записи. Детальное представление состоит из информационных полей, закладок и определяемой пользователем, средней панели.

Как выполнить?

Создать действие безопасности

Откройте детальное представление действия безопасности

  1. На навигационной панели, нажмите Администрирование, затем нажмите Действия безопасности для открытия списочного представления действий безопасности.
  2. Выберите действие безопасности из списка или воспользуйтесь поиском для нахождения необходимого действия безопасности.

Откроется детальное представление действия безопасности.

Используйте Поиск для нахождения необходимых записей

Открыть группу

Переключитесь на списочное представление

  • На панели инструментов, нажмите кнопку Список.

Информационные поля

Информационные поля действия безопасности содержат основную информацию о действии безопасности.

  • Действие безопасности - Название действия безопасности.
  • Описание - Описание действия безопасности.

Закладки Детального представления

Каждая закладка показывает определенную информацию, касающуюся действия безопасности. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
