Использование списочного представления Действий безопасности

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.Только системный администратор и пользователи с ролью администратора могут иметь доступ к этим функциям.

Списочное представление Действий безопасности отображает действия безопасности в виде таблицы. Таблица содержит информацию о каждом действии безопасности, например, название действия безопасности и описание.

  • Чтобы отсортировать таблицу по конкретному полю, нажмите на заголовк столбца, по которому необходимо произвести сортировку.
  • Чтобы отобразить все действия безопасности определенного типа, выбреите необходимую группу.
