Закладка Роли

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.Только системный администратор и пользователи с ролью администратора могут иметь доступ к этим функциям.

Используйте закладку Роли в детальном представлении сущности действия безопасности для управления действиями, назначенными ролям. Роль определяет, к какому функционалу пользователь может иметь доступ в Веб-клиенте. Добавление действий безопасности к роли определяет, к какому функционалу роль будет предоставлять доступ. Добавление пользователей к роли определяет, кто будет иметь доступ к функционалу..

На данной закладке, а также на закладке Действия детального представления сущности Роли отображается взаимосвязь между ролями и действиями безопасности.

  • Закладка Роли отображает действия безопасности, связанные с ролью.
  • Закладка Действия отображает роли, которые содержат действия безопасности.

Когда вы добавляете или удаляете действие безопасности из роли, изменения отображаются на обеих закладках.

Как выполнить?

Добавить действие безопасности к роли

  1. Откройте Детальное представление действия безопасности.
  2. Перейдите на закладку Роли.
  3. Нажмите кнопку Добавить роль.
  4. Воспользуйтесь поиском для того чтобы найти роль, к которой вы бы хотели добавить действие безопасности.
  5. Повторите шаги 3 - 4 для других ролей.
  6. Нажмите ОК.

Удалить действие безопасности из роли

  1. Откройте Детальное представление действия безопасности.
  2. Перейдите на закладку Роли.
  3. Нажмите на ссылку Удалить в записи роли, которую вы хотите удалить.

  4. Нажмите ОК.
