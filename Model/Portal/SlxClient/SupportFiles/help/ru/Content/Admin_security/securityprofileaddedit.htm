Добавление или редактирование профиля безопасности

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте данный диалог для добавления или редактирования существующего профиля безопасности.

Для добавления или редактирования профиля безопасности

  1. Откройте представление менеджер безопасности.
  2. На панели задач, в разделе Обычные задачи, нажмите Добавить профиль или Редактировать профиль.
  3. В поле Название введите название профиля безопасности.
  4. Из списка Тип профиля выберите Пользовательский или Системный.
    Эти два типа предназначены только для организационных целей; они не имеют никаких функциональных различий. Системные профили предназначены для использования в командах, а пользовательские профили предназначены для использования отдельными пользователями.

  5. Нажмите ОК.

Инструкция по обновлению прав доступа профиля безопасности находится в разделе использование менеджера безопасности.
