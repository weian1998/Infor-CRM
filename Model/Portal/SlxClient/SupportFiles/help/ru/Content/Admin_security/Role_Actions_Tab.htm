Закладка Действия

Используйте закладку Действия, детального представления сущности Роли для управления действиями безопасности, назначенными роли. Роль определяет, к какому функционалу пользователь может иметь доступ в Веб-клиенте. Добавление действий безопасности к роли определяет, к какому функционалу роль будет предоставлять доступ. Добавление пользователей к роли определяет, кто будет иметь доступ к функционалу.

Узнать больше...

Безопасность элементов управления в Архитекторе приложений обеспечивается путем назначения им действий безопасности. Когда назначено действие безопасности, пользователи имеют доступ к элементу управления только в том случае, если им назначена роль, включающая в себя соответствующее действие безопасности. Например, выберите Entities/Account/Save в качестве значения свойства Applied Security для кнопки Сохранить на детальном представлении Субъекта. Во время работы кнопка будет видима или не видима в зависимости от того имеет ли пользователь разрешения на выполнение действия.

Для более подробной информации смотрите раздел "Понимание действий безопасности" в руководстве Архитектора приложений.

Добавление или удаление действия безопасности из роли с помощью данной закладки также добавляет или удаляет действие безопасности из закладки Роли детального представления действий безопасности.

Как выполнить?

Добавить действие безопасности к роли

  1. Откройте Детальное представление роли.
  2. Перейдите на закладку Действия.
  3. Нажмите кнопку Добавить действие.
  4. Воспользуйтесь поиском для того чтобы найти действия, которые вы бы хотели добавить к роли.
  5. Нажмите Добавить выбранные.

Удалить действие безопасности из роли

  1. Откройте Детальное представление Роли.
  2. Перейдите на закладку Действия.

  3. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.

  4. Нажмите Да.
