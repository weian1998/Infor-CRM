Использование детального представления Стандартной проблемы

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Детальное представление используется для просмотра, добавления или изменения информации по выбранной записи.

Как выполнить?

Открыть детальное представление Стандартной проблемы
  • На навигационной панели выберите Сервис, затем нажмите Стандартные проблемы для открытия списочного представления стандартных проблем. При необходимости, воспользуйтесь поиском для нахождения проблемы и нажмите на ID стандартной проблемы.

Используйте Поиск для нахождения записей

Переключиться в режим списочного представления
  • На панели инструментов нажмите кнопку Список.

Использовать разделы Записи группы и Обычные задачи панели задач

Просмотреть группу

Информационные поля

Информационные поля стандартной проблемы содержат основную информацию по стандартной проблеме. Пользователи с соответствующими правами могут редактировать эти поля.

Закладки Детального представления

Каждая закладка показывает определенную информацию, связанную со стандартной проблемой. Нажмите на ссылку, чтобы посмотреть подробную информацию о каждой закладке.

  • Стандартные решения

Определяемая пользователем область

Необходимую вам закладку вы можете перетащить мышкой в область, которая находится посередине (область, настраиваемую пользователем). Выбранная закладка будет отображаться в средней части экранной формы детального представления до тех пор, пока не будет заменена на новую или удалена.
