Использование списочного представления стандартных решений

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Списочное представление стандартных решений отображает все стандартные решения, к которым у вас есть доступ. Вы можете использовать группы или поиск, чтобы сократить список отображаемых записей.

Как выполнить?

Открыть списочное представление стандартных решений
  • На навигационной панели выберите Сервис, затем нажмите Стандартные решения.
Обновить список
  • Нажмите кнопку Обновить.

Работа с записями в таблице

Отфильтровать список

Используйте Поиск для нахождения записей

Открыть детальное представление стандартного решения
  • Нажмите на ID стандартного решения для открытия детального представления стандартного решения.

Просмотреть группы

Используйте обычные задачи

На панели задач, в разделе Обычные задачи, вы можете:

  • Сохранить записи как группу
  • Экспортировать в файл
