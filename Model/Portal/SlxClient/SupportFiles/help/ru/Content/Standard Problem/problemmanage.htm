Управление стандартными проблемами

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Используйте это представление, чтобы ввести информацию, которая будет доступна всем пользователям при добавлении заявок.

Как выполнить?

Добавить стандартную проблему
  1. В меню Создать, нажмите Стандартная проблема.
  2. Заполните информационные поля стандартной проблемы.
  1. Нажмите Сохранить.
Изменить стандартную проблему
  1. На навигационной панели выберите Сервис, затем нажмите Стандартные проблемы для открытия списочного представления стандартных проблем. При необходимости, воспользуйтесь поиском для нахождения проблемы и нажмите на ID стандартной проблемы.
  2. При необходимости, внести изменения в информационные поля стандартной проблемы.
  3. Нажмите Сохранить.
Удалить стандартную проблему
  1. На навигационной панели выберите Сервис, затем нажмите Стандартные проблемы для открытия списочного представления стандартных проблем. При необходимости, воспользуйтесь поиском для нахождения проблемы и нажмите на ID стандартной проблемы.
  2. На детальном представлении Стандартной проблемы нажмите кнопку Удалить.
  3. Нажмите OK.
