Установка напоминаний

Напоминание может предупредить вас о делах, которые должны быть завершены. Вы можете установить оповещения для существующего дело или при планировании дела. Вы должны настроить Напоминания для получения уведомлений.

Для настройки оповещения

  1. Выполните одно из следующих действий:
  • Планировать новое дело.
  • Найти дело, для которого вы хотите установить оповещение и откройте детальное представление Дела.
  1. Установите флажок Напоминание, а затем установите интервал времени, за который вы хотите получать напоминание о деле.
  2. По завершению нажмите OK.
