Удаление события

Вы можете удалить созданные вами события из списка Дел или из представления .

Для удаления события

  1. Откройте списочное представление Дел.
  2. Откройте События.
  3. При необходимости, используйте фильтры, чтобы сузить список.
  4. Щелкните правой кнопкой мыши по событию, которое вы хотите удалить, и нажмите Удалить событие.
  5. Нажмите ОК.
