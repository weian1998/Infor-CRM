Дубликаты контрактов

Номер Контракта должен быть уникальным. Если для Контракта указан номер, который уже используется в системе, то появится предупреждающее сообщение.

Для того чтобы ввести правильный номер Контракта:

  • Выберите пункт Ввести вновь номер контракта и нажмите кнопку OK. После чего откроется форма для создания нового Контракта, при помощи которой можно указать другой номер. Вы можете ввести новый номер контракта.
  • Для того чтобы создать Контракт с существующим номером, выберите пункт Создать дублирующий номер контракта и нажмите кнопку OK.
