Добавление контракта

Используйте представление добавления нового контракта для создания контракта. Используйте кнопку копировать для быстрого создания новой записи с похожей информацией.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Чтобы добавить контракт

  1. Откройте представление создание нового контракта.

    • В меню Создать, нажмите Контракт.
    • На детальном представлении Субъекта, на закладке Контракты, нажмите Создать контракт.
  2. Заполните информационные поля контракта.
  3. Нажмите Сохранить.
  4. При необходимости, воспользуйтесь закладкой Охваченные активы для связывания активов с контрактом.
