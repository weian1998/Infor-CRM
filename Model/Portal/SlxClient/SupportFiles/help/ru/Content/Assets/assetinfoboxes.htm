Параметры Актива

В таблице ниже описаны основные поля.

Поле Описание/Действие
Название продукта Название Актива. Нажмите кнопку Поиск и введите значение при помощи поиска.
Единица учета Единица учета продукции.
Количество Количество приобретенных Субъектом продуктов.
Серийный номер Уникальный серийный номер продукта.
Контакт Имя контакта. Нажмите кнопку Поиск и введите значение при помощи поиска.
Телефон Отображает телефонный номер контакта.
E-mail Отображает e-mail адрес контакта.

Размещение

Код продукта

Окружение

Система

 Используйте эти параметры для того чтобы зафиксировать информацию, характерную для вашего бизнеса.
Версия Версия Приобретенного продукта.
Последнее изменение

Дата добавления актива или последнего изменение.
Изменил

Пользователь, который добавил или последний редактировал актив.
Описание Описание Приобретенного продукта.
Возвращено Введите это значение, если продукт был возвращен. Данное поле не является обязательным для заполнения.
Оценка Введите это значение, если продукт передан для оценки (тестирования) клиенту.
Сделка Отображается связанная сделка (если указана).
Заказ Отображается связанный заказ (если указан).
Комментарии Указывается дополнительная информация по продукту.
