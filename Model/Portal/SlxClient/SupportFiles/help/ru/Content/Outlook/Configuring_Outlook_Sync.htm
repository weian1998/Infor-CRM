Настройка Синхронизация с Outlook и Интеграция с Outlook

Настройки синхронизации и интеграции следует производить в Microsoft Outlook.

 Подключение и Отправка в SLX записей в историю может быть настроено для пользователей веб-клиента, которые установили свои варианты в Менеджер рабочего стола.

Вы можете настроить различные параметры для различных типов записей. Например, вы можете включить одностороннюю синхронизацию контактов и двустороннюю синхронизацию для календаря. Следующие шаги описывают рекомендуемые настройки синхронизации между Microsoft Outlook или и Infor.

Перед настройкой Outlook:

  • Microsoft Outlook должен быть установлен и запущен на локальной машине.

    • Web - Интеграция рабочего стола не будет установлена, если не установлен Microsoft Office, который включает Outlook.

    • Windows - Если Microsoft Outlook не был установлен прежде чем установлен Infor CRM клиент, Вам необходимо выполнить дополнительные действия по настройке.

  • Вы должна знать сервер для подключения, номер порта, имя пользователя и пароль. Свяжитесь со своим системным администратором для получения дополнительной информации.

  • Вы должны создать группу синхронизации Контактов, которые Вы хотите синхронизировать между Outlook и Infor.

Настройка Синхронизации с Outlook и Интеграции с Outlook

  1. Откройте Microsoft Outlook.
  2. Если кнопка Опции не отображается, то установка синхронизация Outlookпроизведена не верно.

  3. Воспользуйтесь закладками для настройки следующих опций:

      • Подключение - соединение с сервером Infor. Используется для синхронизации контактов и календаря и интеграции почты.
      • Синхронизация - опции синхронизации для контактов, назначений (телефонные вызовы, встречи и персональные действия) и задачи (действий). Используется для синхронизации контактов и календарей.
      • Отправка в SLX/Запись в Историю - опции для подсказок, вывода на экран информации и поведения вложений при использовании Отправить в SLX или Запись в Историю. Используется для почтовой интеграции.
      • Адресная книга - настройки отображения Адресной книги Infor. Используется для интеграции почты.
      • Журналирование - возможность включить или выключить фиксацию информации о неполадках. Используется для синхронизации контактов и календаря.
      • Все Опции - возможность просмотреть опции как пары значений. Рекомендуется для продвинутых пользователей.

     

  4. Нажмите ОК.
