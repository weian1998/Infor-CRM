Настройки опций подключения синхронизации

Опции Sync определяют, какая информация и каким образом будет синхронизироваться между Infor и Outlook.

Эта закладка используется только в Синхронизации с Outlook.

Настроить опции

  1. Нажмите Опции и выберите Синхронизировать.

  2. Если Вы хотите синхронизировать контакты, выполните следующее:

    1. Установите флажок Синхронизировать Контакты.

    2. Нажмите на выпадающий список Специальные группы и выберите контактную группу для синхронизации.

      3. Если группа, которую Вы создали, не появляется в списке, нажмите кнопку Обновить рядом с выпадающим списком, чтобы обновить список.

    3. Нажмите на выпадающий список Папка и выберите папку в Outlook, которая содержит Контакты для синхронизации.

    4. Нажмите на выпадающий список Направление синхронизации и выберите как Вы хотите, чтобы осуществлялась синхронизация между Infor и Outlook.

  3. Если Вы хотите синхронизировать назначения (телефонные вызовы, встречи и персональные действия), выполните следующее:
    1. Установите флажок в поле Назначение Синхронизации.
    2. Нажмите кнопку Папка и выберите папку в Outlook, которая будет содержать назначения для синхронизации. По умолчанию используется Календарь.
    3. Нажмите на выпадающий список Направление синхронизации и выберите, как вы хотите, чтобы осуществлялась синхронизация между Infor и Outlook.
    4. Чтобы ограничить назначения, подлежащие синхронизации, выберите На первой синхронизации, нажмите на стрелку раскрывающегося списка и выберите дату в календаре.
      Только задачи, которые планируется на или после выбранной даты будут синхронизированы.
  4. Вы можете синхронизировать задачи, выполните следующее:
    1. Установите флажок в поле Задачи Синхронизации.
    2. Нажмите кнопку рядом с полем Папка и выберите папку в Outlook, которая будет содержать задачи синхронизации. Значение по умолчанию - Задачи.
    3. Нажмите на выпадающий список Направление синхронизации и выберите, как вы хотите, чтобы осуществлялась синхронизация между Infor и Outlook.
    4. Чтобы ограничить назначения, подлежащие синхронизации, выберите На первой синхронизации, нажмите на стрелку раскрывающегося списка и выберите дату в календаре.
      Только задачи, которые планируется на или после выбранной даты будут синхронизированы.

  5. В поле Интервал синхронизации выберите количество минут между синхронизациями Infor и Outlook.

  6. Нажмите OK, если Вы закончили или выберите на другой закладке.
