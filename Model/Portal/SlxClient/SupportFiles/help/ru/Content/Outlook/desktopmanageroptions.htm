Установка Параметров истории E-mail для Менеджера рабочего стола

Используйте Infor CRM менеджер рабочего стола, чтобы ввести информацию о подключении и установить параметры истории электронной почты.

Для настройки параметров

  1. В разделе Системный трей на панели задач Windows, дважды щелкните по значку Infor CRM Менеджер рабочего стола.
  2. Выполните одно из следующих действий:
  • Пользователи веб-клиента - В поле Портал URL введите URL для подключения. Используйте следующий формат:

http://servername:port

  • Автономный веб-клиентпользователи - В поле Портал URL введите URL для подключения. Используйте следующий формат:

http://localhost

Если вы используете полное имя, вам может понадобиться добавить домен domain name.com после имени сервера.

  1. Выполните одно из следующих действий:
  • Если вы используете для входа в Infor CRM информацию об учетной записи Windows, выберите вариант Использовать проверку подлинности Windows .
  • Введите Логин и Пароль.
  1. В поле Пользователь введите имя пользователя, под которым Вы входите в систему Infor CRM веб-клиент.
  2. В поле Пароль введите пароль для входа в Infor CRM.
  1. Нажмите Проверка подключения.
  2. Выберите История параметров, чтобы при входе в электронную почту Outlook шла запись в историю Infor CRM. Выберите один из следующих пунктов:
  • Запрос на дубликаты контактов - запрос появляется при возникновении дубликата контакта или наводки.
  • Запрос для контактов - предложит вам выбрать, какие контакты или наводки должны быть связаны.
  • Запрос на нерешенные Контакты - запрос появляется, когда контакт или наводка не найдены в базе данных.
  • Отображение истории деталей перед сохранением - запрос появляется перед диалоговым окном Закрыть E-mail перед записью в историю e-mail сообщений.
  1. Выберите флажок Отображать уведомления в трее, если вы хотите видеть всплывающие сообщения, которые описывают рабочую деятельность менеджера.

Эти параметры не применимы к "перетаскиваемым" e-mail. Используйте параметр Запись в Историю в Параметрах для "перетаскиваемых" e-mail.

Для получения дополнительной информации о Менеджере рабочего стола, обратитесь к справочной системе Менеджер рабочего стола .
