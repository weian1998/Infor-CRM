Настройка параметров подключения Infor

Используйте параметры подключения, чтобы настроить соединение с сервером.
  • Данная закладка требует Синхронизации с Outlook и Интеграции с Outlook.
  • Эти параметры могут быть уже настроен для пользователей веб-клиента, которые устанавливают свои опции в Менеджере рабочего стола.

Настроить опции

  1. Нажмите Опции и если необходимо нажмите Подключение.
  2. В поле Сервер введите Имя сервера.

    • InforКлиент и пользователи веб-клиента - Введите URL, который предоставил Вам администратор, чтобы соединиться с Infor. Используйте следующий формат:

      http://servername:порт или http://servername:порт

    • Пользователи офлайн веб-клиент - Введите URL чтобы соединиться с локальным сервером Infor. Используйте следующий формат:
      http://localhost:порт
      Порт обычно 8088.
  3. Выполните одно из следующих действий:
    • Если Вы используете свои сведения об аккаунте Windows, чтобы войти в систему Infor, выберите опцию Использовать Windows аутентификацию.

    • Введите свое Имя пользователя и пароль.

  4. Нажмите Проверить.

    Если проверка прошла неудачно, проверьте введенную информация. Если информация введена корректно, Вы не можете подключиться все равно, обратитесь к Администратору.

  5. Нажмите OK, если Вы закончили или выберите на другой закладке.
