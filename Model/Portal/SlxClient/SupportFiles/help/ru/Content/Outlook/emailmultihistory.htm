Завершение Записи Истории для нескольких E-mail

Диалог Завершить Быстро открывается при выборе нескольких e-mail сразу для записи в историю. Используйте диалог Завершить быстро для ввода информации истории для нескольких e-mail сообщений, или можно выбрать записать информацию для каждого e-mail в отдельности.

Как выполнить?

Запись истории для нескольких e-mail сразу
  1. Нажмите Результат и выберите элемент из списка.
  2. В поле Заметки добавьте любую информацию, которую хотите добавить к каждой записи истории.
  3. Нажмите Как запланировано чтобы создать записи истории, используя дату и время отправления e-mail. Или нажмите Сейчас, чтобы создать записи истории, используя текущее число и время.
Запись истории для нескольких e-mail по отдельности
  1. Нажмите Отдельночтобы завершить запись каждого e-mail в отдельности.
  2. Используйте диалог История - E-mail для записи информации о завершенном электронном письме.
