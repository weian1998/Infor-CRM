Ссылка на адреса Контакта

Ссылки на адреса Контакта позволяет связать контакты, которые Вы связаны в предыдущем окне с адресной информации при слиянии записей.

Чтобы установить связь адрес для объединенных записей

  1. Откройте Мастер ссылок или Мастер объединения записей и пройдите через предыдущие окна.
  2. Заполните экранную форму Адреса контакта.
    3. Поле Описание / Действие
    Связанные Контакты

    Отображает связанные контакты из предыдущего экрана.

    Нажмите кнопку Выбрать, чтобы выбрать контактные данные.
    Несвязанные адреса источника

    Отображает несвязанные адреса для выбранного источника контакта.

    Нажмите кнопку Выбрать, чтобы выбрать адрес.
    Несвязанные адреса цели

    Отображает несвязанными адреса для выбранного целевого контакта.

    Нажмите кнопку Выбрать, чтобы выбрать адрес.
    Кнопа ссылки

    Выберите несвязанные адреса из каждого списка, а затем выберите команду Связать с созданием соединения.
    Связанные Адреса Контакта

    Отображает любые связанные контактные адреса.

    Нажмите Разорвать связь, для удаления очистить предлагаемой ссылку и создать новую ссылку.
  3. Нажмите Объединить.
