Установление связи между адресами

Используйте связывание адресов для определения по какому адресу будут доступны объединенные записи.

Только системный администратор и пользователи с соответствующей ролью могут иметь доступ к этим функциям.

Соединить адрес для объединенных записей

  1. При слиянии записей в списочном представлении нажмите Далее.
  2. Заполните экранную форму Ссылка адресов.
    ПолеОписание / Действие
    Несвязанные адреса источника

    Отображает адреса в источнике, которые будут заменены в объединенной записи, если адрес не связан с адресом назначения.

    Нажмите кнопку Выбрать, чтобы выбрать адрес.

    Несвязанные адреса цели

    Отображение адреса в цели, которые будут оставаться связанной с объединенной записи, если адрес не связан с целевым адресом.

    Нажмите кнопку Выбрать, чтобы выбрать адрес.

    Кнопа ссылки

    Выберите несвязанный адреса в несвязанных источниках Адресов и несвязанных целевых адресов из списков, а затем выберите команду Связать с созданием соединения.

    Связанные Адреса

    Отображает список связанных адресов.

    Нажмите Разорвать связь для удаления предлагаемый ссылки и создайте новую ссылку.

    Нажмите кнопку Выбрать, чтобы отобразить поля данных контакта в исходной и целевой системах.

  3. Нажмите Далее для перехода к следующему шагу.
