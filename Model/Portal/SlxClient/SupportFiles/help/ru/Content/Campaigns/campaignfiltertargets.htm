Фильтрация целей

Используйте настройки фильтра Включить, для того чтобы ограничить целевые контакты и наводки, отображаемые в списке. Вы можете применить фильтр с помощью установки флага. Фильтры сохранят ваш последний выбор в качестве значений по умолчанию.

Чтобы применить фильтры

  1. Откройте диалог поиск целей.
  2. Выберите тип целей, которые вы хотите включить:
    • Наводки (неквалиф.)
    • Субъекты (только основные контакты)
    • Субъекты (все контакты)
    • Контакты (личные)
  3. Установите флаги на фильтрах, которые вы хотите применить.
    • Компания - Фильтр по названию компании.
    • Отрасль - Фильтр по отрасли.
    • Доступ - Фильтр по доступу. Можно ввести много значений, разделенных запятыми.
    • Должность - Фильтр по должности контакта или наводки.
    • Продукты принадлеж. - Фильтр по продуктам, принадлежащим контакту или наводке.
    • Статус - Фильтр по статусу контакта или наводки.
    • Город - Фильтр по городу. Можно ввести много значений, разделенных запятыми.
    • Регион/Обл - Фильтр по региону. Можно ввести много значений, разделенных запятыми.
    • Индекс - Фильтр по почтовому индексу. Можно ввести много значений, разделенных запятыми.
    • Источник наводки - Фильтр по источнику наводки контакта или наводки.
    • Источник импорта - Фильтр по источнику импорта наводки.
    • Дата создания -Фильтр по дате создания контакта или наводки. Можно выбрать диапазон дат.

    В зависимости от выбранного типа включаемой цели, некоторые фильтры могут быть недоступны.

  4. Выберите выражение для условия из выпадающего списка. Возможный выбор: Начинается с, Содержит, Равно, Не равно, Меньше или равно, Больше или равно, Меньше чем, Больше чем.
  5. Введите или выберите информацию о соответствующих критериях.
  6. Если вы хотите включить в кампанию контакты или наводки, помеченные флагом Не оповещать, установите флаг Не оповещать или один или несколько следующих флагов:
    • Включить Не по E-mail
    • Включить Не звонить
    • Включить Не по почте
    • Включить Не по факсу
  7. Чтобы узнать, сколько целей соответствуют условиям фильтров до поиска целей, нажмите кнопку Сколько?.
  8. Нажмите Найти

Чтобы очистить фильтры

  • Нажмите Очистить все.
