Добавление/Редактирование/Завершение стадий маркетинговой программы

Стадии маркетинговой программы - это группы задач, которые могут помочь пользователям Infor управлять их маркетинговыми программами. Вы можете добавлять и редактировать информацию о стадии или завершить стадию по ее окончанию.

Как выполнить?

Добавить стадию
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, затем нажмите Добавить стадию.
  2. Введите Описание для стадии.
  3. В поле Статус нажмите и выберите пункт из списка
  4. В поле Источник Наводки нажмите кнопку Найти для выбора источника наводки.
    Ваш выбор добавлен в диалоговое окно.
  5. При необходимости, измените маркетинговой программы. Нажмите OK в диалоговом окне.

    Изменение кода кампании означает, что код маркетинговой программы для данной стадии не будет больше совпадать с маркетинговой программой и другими стадиями маркетинговой программы.

  6. В поле Комментарии введите любую дополнительную информацию.
  7. При необходимости, в поле Дата начала, нажмите на кнопку Календарь для выбора даты, а затем нажмите OK.  
  8. В поле Дата окончания, нажмите на кнопку Календарь для выбора даты, а затем нажмите OK.  
  9. Добавьте задачи в таблицу Задачи.
  10. Нажмите ОК.
Редактировать стадию маркетинговой программы
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, затем нажмите Редактировать стадию.
    • Из диалогов Создание маркетинговой программы или Копирование маркетинговой программы выберите стадию, которую вы хотите изменить, затем нажмите Изменить.
    • В Детальном представлении маркетинговой программы, откройте закладку Стадии/Задачи, щелкните правой кнопкой мыши на стадию, которую хотите изменить, затем выберите пункт меню Изменить стадию.
  2. При необходимости, произведите изменения, затем нажмите OK.
Завершить стадию кампании
  1. Из диалогов Создание маркетинговой программы или Копирование маркетинговой программы выберите стадию, которую вы хотите завершить, затем нажмите Завершить стадию.

Откроется диалог Завершение стадии со значением статуса - "Завершенная".

  1. При необходимости, произведите изменения, затем нажмите OK.
