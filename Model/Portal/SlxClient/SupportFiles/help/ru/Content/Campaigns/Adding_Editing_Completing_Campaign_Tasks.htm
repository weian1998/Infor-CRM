Добавление/Редактирование/Завершение задач маркетинговой программы

Задачи маркетинговой программы могу помочь пользователям-маркетологам Infor CRM управлять их маркетинговыми программами. Вы можете добавлять и редактировать информацию о задачах или завершить задачу как только она закончилось.

Как выполнить?

Добавить задачи маркетинговой программы
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, найдите стадию, в которую хотите добавить задачу, затем нажмите Добавить задачу.
  2. Введите Описание задачи.
  3. В поле Статус, нажмите на и выберите элемент из списка.
  4. В поле Приоритет, нажмите на и выберите элемент из списка.
  5. В поле % Завершения, введите процент от 0-100.
  6. В поле Требуемая дата нажмите на кнопку Календарь для выбора даты, а затем нажмите OK.  .
  7. Если задача уже завершена, в поле Дата завершения нажмите на кнопку Календарь для выбора даты, а затем нажмите OK.  
  8. В поле Комментарии, введите любую дополнительную информацию.
  9. В поле Ожидаемый расход, введите ожидаемый расход на задачу.
  10. Если вам известны фактические затраты на задачу, введите сумму в поле Фактический расход.
  11. В поле Ожидаемые часы, введите ожидаемые часы, которые понадобятся для выполнения задачи.
  12. Если вам известны фактические часы для задачи, то введите количество в поле Фактические часы.
  13. Если настроена мультивалютность, выберите Отобразить, используя "Мою валюту" для просмотра затрат на задачи, используя тип валюты, который вы выбрали в св>их параметрах пользователя.
  14. Выберите соответствующий параметр Назначить.
  15. В поле Назначить, выполните одно из следующих действий:
    • Если вы выбрали Существующий Пользователь/Команда, нажмите кнопку Найти , затем выберите пользователя или команду, которой вы хотите назначить задачу.
    • Если вы выбрали Существующее подразделение, нажмите на указывающую вниз стрелку и выберите значение из списка.
    • Существующий контакт, нажмите кнопку Найти, затем выберите человека, которому вы хотите назначить задачу.
    • Если вы выбрали Другой человек, введите имя человека,которому вы хотите назначить задачу.
    • Если вы выбрали Нет, перейдите к следующему шагу.
  16. В поле Дата назначения, нажмите на кнопку Календарь для выбора даты, а затем нажмите OK.  
  1. Нажмите OK.
Редактировать задачи маркетинговой программы
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, и нажмите Изменить.
    • Из диалога Создание маркетинговой программы или Копирование маркетинговой программы,выберите задачу, которую хотите изменить и нажмите Изменить.
    • Из диалога Добавления, Редактирования или Завершения стадии, нажмите Изменить задачу.
    • В Детальном представлении маркетинговой программы, откройте закладку Стадии/Задачи, щелкните правой кнопкой в таблице, затем выберите пункт меню Изменить задачу.
  2. При необходимости, произведите изменения, затем нажмите OK.

Если вы изменили требуемую дату, вам будет предложено изменить требуемую дату для всех незавершенных задач, на то же количество дней. Нажмите Да, чтобы изменить требуемую дату для других задач.
Завершить задачи маркетинговой программы
  1. В детальном представлении Маркетинговой программы, откройте закладку Стадии/Задачи, нажмите Завершить.
    Откроется диалог завершения задачи, в поле статус будет установлено значение "Заврешена", % Завершения - 100 и дата завершения - текущая дата.
  2. При необходимости, произведите изменения, затем нажмите OK.

Если вы изменили требуемую дату, вам будет предложено изменить требуемую дату для всех незавершенных задач, на то же количество дней. Нажмите Да, чтобы изменить требуемую дату для других задач.
