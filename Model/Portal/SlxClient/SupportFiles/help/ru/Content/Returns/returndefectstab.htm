Использование закладки дефекты

Закладка дефекты в детальных представлениях заявки и возврата используется для просмотра дефектов. Эта закладка используется для установки или удаления связи с дефектом.

Закладка отображает доступную только для чтения информацию о идентификационном номере дефекта, дате сообщения, приоритете, важности, статусе, решении и описании.

Чтобы открыть

  1. Откройте детальное представление заявки или возврата.
  2. Откройте закладку Дефекты. Если закладка не отображается, то откройте закладку Все закладки.

Чтобы связать с дефектом

  1. Нажмите Связать дефект.
  2. Найдите и выберите элемент используя поиск.
  3. Нажмите ОК.

Чтобы удалить связь с дефектом

  1. Выберите запись, которую вы хотите удалить и нажмите кнопку Удалить.

  2. Нажмите Да.

Чтобы добавить новый дефект

  1. На детальном представлении Заявки нажмите кнопку Создать дефект.
  2. Следуйте инструкции по добавлению дефекта.
