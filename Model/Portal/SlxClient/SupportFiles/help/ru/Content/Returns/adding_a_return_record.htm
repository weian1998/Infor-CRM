Добавление возврата

Испольуйте представление создание возврата для создания возврата.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Чтобы добавить возврат

  1. Откройте представление создания возврата.
    • В меню Создать, нажмите Возврат.
  2. Заполните необходимые информационные поля возврата.
  3. Нажмите Сохранить.
    Откроется детальное представление возврата.
  4. Используйте закладку детали доставки для добавления информации о заменяющем продукте, который был доставлен клиенту.
  5. Используйте закладку детали возврата для добавления возвращенных активов.
