Использование закладки Детали возврата

Используйте данную закладку для просмотра информации о контакте, осуществляющем возврат или для связывания или отвязывания активов.

Информация о контакте добавляется из связанной заявки. Вы не можете редактировать поля Контакт, Телефон или Email. Используте поле заметки для внесения дополнительной информации.

Как выполнить?

Открыть закладку детали возврата
  1. Откройте детальное представления возврата.
  2. Откройте закладку Детали. Если закладка не отображается, то откройте закладку Все закладки.
Привязать актив
  1. В области Активы к возврату, нажмите кнопку Привязать актив.
  2. Найдите и выберите актив используя поиск.
  3. Нажмите ОК.
Отвязать актив
  1. Нажмите на ссылку Отвязать напротив записи, которую вы хотите отвязать.
  2. Щелкните правой кнопкой и нажмите Отвязать.
  3. Нажмите Да.
