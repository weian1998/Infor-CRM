Информационные поля возврата

В таблице ниже описаны основные поля.

Поле Описание/Действие

Идентификационный номер возврата

 Автоматически отображает следующее доступное значение идентификационного номера. Значение идентификационного номера недоступно для редактирования. Не доступен при добавлении возврата.
Идентификационный номер заявки Отображает идентификационный номер заявки, из которой был сформирован текущий возврат. Нажмите кнопку Найти и введите значение при помощи поиска.
Субъект Отображает название субъекта, связанного с заявкой.
Контакт Отображает имя контакта, по которому был сформирован данный возврат. Нажмите кнопку Найти и введите значение при помощи поиска.
Причина Причина возврата.
Номер заказа клиента Номер заказа, по которому был сделан возврат.
Статус Отображает статус возврата. В данном списке содержатся следующие значения: Открыт, Закрыт, Отправлен, В процессе.
Тип Тип возврата.
Приоритет Приоритет, присвоенный возврату.
Ожидается Дата, к которой ожидается возврат продукта. Для изменения нажмите кнопку Календарь для выбора даты.  
Поручено Пользователь, которому поручена работа с данным возвратом. Нажмите кнопку Найти и введите значение при помощи поиска.
Создал

Отображается имя пользователя, который создал возврат. Нажмите кнопку Найти и введите значение при помощи поиска.
Закрыто

Отображается имя пользователя, который закрыл возврат. Нажмите кнопку Найти и введите значение при помощи поиска.

Дата и время закрытия возврата. Для изменения нажмите кнопку Календарь для выбора даты.
