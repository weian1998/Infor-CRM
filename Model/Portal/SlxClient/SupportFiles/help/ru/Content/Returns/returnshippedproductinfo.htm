Информационные доставленного продукта

Используйте эти поля для ввода информации о доставленном продукте клиенту.

Поле Описание/Действие
Полученный продукт Продукт, возвращенный клиентом. Нажмите кнопку Найти и введите значение при помощи поиска.
Единица учета Идентификатор товарной позиции возвращенного продукта.
Серийный номер Отображает серийный номер возвращенного продукта.
Вернуть существующий Указывает был ли отправленный клиенту продукт, тем же продуктом, что вернул клиент.
Продукт Продукт, отправленный клиенту. Нажмите кнопку Найти и введите значение при помощи поиска.
Единица учета Идентификатор товарной позиции для продукта, который доставляется.
Серийный номер Введите серийный номер доставляемого клиенту продукта, если это не тот же продукт, что был возвращен.
Доставлен После того, как продукт был доставлен клиенту, вы можете выставить данный флаг.
Дата доставки Дата, когда новый или замененный продукт был доставлен клиенту. Для изменения нажмите кнопку Календарь для выбора даты.
