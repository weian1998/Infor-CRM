Использование закладки детали доставки

Используйте закладку для ввода информации о замещающих продуктах, отправленных клиенту. Информация о клиенте добавляется из информации по возврату, но контакт, который получит доставленный продукт, может быть изменен.

Как выполнить?

Открыть закладку детали доставки
  1. Откройте детальное представление возврата.
  2. Откройте закладку Детали доставки.Если закладка не отображается, то откройте закладку Все закладки.
Изменить контактную информацию доставки
  1. В поле Контакт, нажмите кнопку Найти и воспользуйтесь поиском для выбора контакта, который получит замененный продукт.
  2. В поля Доставка и Сопров. № введите дополнительную информацию.
  3. В поле Обращение, введите имя клиента или другую информацию, которая может понадобиться при отправке продукта(ов).
  4. Нажмите на кнопку Изменить поля Адрес, чтобы добавить или изменить адрес доставки.
  5. В поле Спец. инструкции введите необходимые инструкции по доставке.
  6. Нажмите Сохранить.
Добавить доставленный клиенту продукт
  1. В блоке Продукты к доставке, нажмите Добавить доставленный продукт.
  2. Заполните информационные поля доставленного продукта.
  3. Нажмите ОК.
Изменить доставленный клиенту продукт
  1. В блоке Продукты к доставке, нажмите Изменить напротив продукта, который хотите изменить.
  2. Измените информационные поля доставленного продукта.
  3. Нажмите ОК.
Удалить доставленный клиенту продукт
  1. В блоке Продукты к доставке, нажмите Удалить напротив продукта, который хотите удалить.
  2. Нажмите OK.
