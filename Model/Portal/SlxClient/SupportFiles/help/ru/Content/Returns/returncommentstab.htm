Использование закладки комментарии по возврату

Закладка комментарии по возврату представляет собой текстовое поле для всех служебных (внутренних) заметок, связанных с возвратом.

Чтобы добавить комментарий

  1. Откройте детальное представление возврата.
  2. Откройте закладку Комментарии. Если закладка не отображается, то откройте закладку Все закладки.
  3. Введите информацию в поле комментарии.
