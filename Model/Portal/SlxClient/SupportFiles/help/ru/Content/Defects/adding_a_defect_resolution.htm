Добавление решения по Дефекту

После того как дефект разрешен, необходимо добавить описание его решения.

Для того чтобы добавить решение

  1. Открыть детальное представление Дефекта.

  2. В поле Статус нажмите и выберите элемент из списка.
  3. (Необязательно) В поле Исправлено в версии нажмите и выберите элемент из списка.

  1. На закладке Детали, используйте поле Решение для ввода информации о ремонте или изменениях процесса.

  2. (Необязательно) Если необходимо изменить какую-либо задачу по дефекту, откройте закладку Задачи дефекта и измените задачи по Дефекту.

  3. Нажмите Сохранить.
