Добавление Дефекта

Для того чтобы добавить новый Дефект, откройте форму Создать Дефект. Дефект можно создать на основе существующей записи (скопировать его), для этого нажмите кнопку скопировать.

От вашего уровня доступа зависит, какой функционал системы будет вам доступен. Для изменения прав доступа обратитесь к вашему администратору. Для пользователей с типом WebViewer могут быть доступны не все функции системы.

Добавить новый дефект

  1. Открыть диалог Добавить новый дефект.
  • В меню Создать выберите Дефект.
  • В детальном представлении Заявки нажмите на закладку Дефекты затем нажмите Создать дефект .
  1. Заполните необходимые информационные поля.
  2. Нажмите Сохранить.
  3. При необходимости, при помощи закладок Детального представления Дефекта добавьте необходимые задачи по дефекту, а также связанные продукты, возвраты или заявки.
