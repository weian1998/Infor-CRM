Использование закладки Возвраты Дефекта

Эта закладка используется для установки или удаления связи возвратов и дефекта. Пользователи могут также использовать данную закладку для определения серьезности дефекта и действий, необходимых для его устранения.

Для того чтобы открыть

  1. Открыть детальное представление Дефекта.

    Откройте закладку Возвраты.

Если закладка не отображается, то откройте закладку Все закладки.

Чтобы связать

  1. Выберите пунктСвязать.
  2. Для того чтобы выбрать нужный возврат используйте поиск.
  3. Нажмите ОК.

Удаление связи

  1. Выберите запись, которую вы хотите удалить, и нажмите кнопку Удалить.

  2. Нажмите ОК.
