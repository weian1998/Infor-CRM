Отслеживание Дефекта

Отслеживание дефекта означает работы по его разрешению. Используйте следующие вопросы или ссылки на бизнес-правила вашей компании по отслеживанию дефектов.

  • Необходимо ли связывать заявку с дефектом?

Если необходимо, создайте Заявку для каждой выполняемой работы.

  • Нужно ли отслеживать время, затраченное на работы по устранению дефекта?

Если необходимо, создайте Задачу для каждой выполняемой работы.

  • Нужно ли изменять статус Дефекта (например с Открыт на В процессе)?

Если необходимо, измените значение поля Статус на карточке детальной информации по Дефекту.

  • Нужно ли добавлять решение по Дефекту?

Если нужно, то добавьте описание решения на карточке детальной информации по Дефекту.
