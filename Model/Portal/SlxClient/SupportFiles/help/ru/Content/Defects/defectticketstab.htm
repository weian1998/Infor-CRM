Использование закладки Заявки Дефекта

Эта закладка используется для установки или удаления связи Заявки и Дефекта. Пользователи могут также использовать данную закладку для определения серьезности дефекта и действий, необходимых для его устранения.

Как выполнить?

Открыть закладку Заявки
  1. Открыть детальное представление Дефекта.
  2. Откройте закладку Заявки. Если закладка не отображается, то откройте закладку Все закладки.
Открыть детали Заявки
  • Выполните одно из следующих действий:
    • Нажмите на ИД заявки для перехода к Детальному представлению Заявки.

    • При выборе записи в списке, детали этой записи отображаются на панели предварительного просмотра справа от списка.

Для добавления заявки
  1. Нажмите Связать с Заявкой.
  2. Для того чтобы выбрать нужную заявку используйте поиск.
  3. Нажмите ОК.
Для удаления связи
  1. Нажмите Удалить рядом с заявкой, которую вы хотите отвязать от Дефекта.

  2. Нажмите ОК.
