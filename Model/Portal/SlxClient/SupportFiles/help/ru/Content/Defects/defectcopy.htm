Копирование Дефекта

Новый дефект можно создать на основе существующего, скопировав информацию, введенную ранее. При нажатии кнопки Копировать Infor копирует информацию из оригинала.

Для того чтобы скопировать

  1. Открыть детальное представление Дефекта.
  2. Нажмите Скопировать Дефект.

После чего откроется форма для добавления Дефекта, в которой будут указаны данные из дефекта-оригинала.
