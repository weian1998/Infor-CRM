Filter Manager Metrics Tab

Metrics determine how Dashboard widget data is calculated. The Metrics tab on the Filter Manager view only appears when the Detail pane is active. The Metrics tab lists all metrics associated with a selected entity. From this tab, you can add, edit, or delete metrics.

The system administrator and users with the appropriate role can access these features.

How Do I?

Open the Filter Manager Metrics tab
  1. On the Navigation Bar, click Filter Manager.
  2. Select a record in the Entities list.
  3. Click Detail.
    The Details pane displays at the bottom of the list view with detailed information about the selected record.
  4. Click Metrics and view the information in the lower pane.

Add a metric

Edit a metric

Delete a metric
  1. Select the appropriate entity from the Entities list.
  2. In the Metrics tab in the lower pane, select the metric you want to delete.
  3. Click Remove Metric.

  4. Click OK.

Add or hide a column

  • Click the and select the check box of the columns you want to appear in the grid and clear the check boxes of the columns you want to hide in the grid. When finished, move your mouse cursor off of the menu and click to close the menu.
