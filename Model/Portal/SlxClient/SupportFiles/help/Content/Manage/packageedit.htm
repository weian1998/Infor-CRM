Editing a Product Package

Use the Package Detail view to edit package information.

To edit

  1. On the Navigation Bar, click Sales, Support, or Administration, and then click Packages to open the Packages List view. If necessary, use the Lookup to locate a package, and then click on the package name.
  2. If necessary, change the package information.
  3. Click Save.
