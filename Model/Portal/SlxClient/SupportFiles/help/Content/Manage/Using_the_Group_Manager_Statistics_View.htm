Using the Group Manager - Usage Tab

The Group Manager Usage tab provides a list of all groups and the statistics and tasks needed to manage those groups. Use the filters to narrow the list of groups you want to view and the group tasks to manage groups.

The system administrator and users with the appropriate role can access these features.

How Do I?

Open the Group Manager View Usage tab
  • On the Navigation Bar, click Group Manager, and then Usage.

Refresh list information
  • Click the Refresh button.

Filter my list

View group statistics

  1. Select a record.
  2. Click Detail.
    The Details pane displays at the bottom of the list view with detailed information about the selected record.

Hide group details

  • To hide the Details pane, click Hide Detail.

Reset Statistics

  • To purge existing group statistics, select a group and, under Group Manager Tasks, click Reset Statistics.
    The statistics for any selected groups will be cleared of any previous data and will begin capturing new statistics from that point forward.

Disable Statistics

  • To stop collecting group statistics, select a group and, under Group Manager Tasks, click Disable Statistics.
