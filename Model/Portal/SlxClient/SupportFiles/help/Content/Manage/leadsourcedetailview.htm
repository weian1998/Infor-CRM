Using the Lead Source Detail View

The system administrator and users with the appropriate role can access these features.

Use the Detail view to see, add or edit information for a selected record.

How Do I?

Open the Lead Source Detail view
  • On the Navigation Bar, click Administration, and then click Lead Sources to open the Lead Sources List view. If necessary, use the Lookup to locate a lead source, and then click on the lead source name.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button, or click the current group tab.

Use the Task Pane group list or common tasks

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group

Information Boxes

  • Description - Enter a description for the lead source.
  • Abbrev. Desc.- Enter a short description for the lead source.
  • Type - Click the drop-down arrow and select an item from the list.
  • Status - Click the drop-down arrow and select an item from the list.
  • Cost - Enter the cost associated with producing the lead source.
  • Date - To edit, click the Calendar button, select the date, and then click OK.
