Using the Filter Manager

Filters are used to group records and can be used to filter records in list views and to configure Dashboard widgets. The Filter Manager provides a list of entities . Use the filters to narrow the list of entities you want to view and the details pane tabs to manage filters and view metrics.

The system administrator and users with the appropriate role can access these features.

How Do I?

Open the Filter Manager View
  • On the Navigation Bar, click Filter Manager.

Work with records in a grid

Refresh list information
  • Click the Refresh button.

Filter my list

View filter details
  1. Select a record in the Entities list.
  2. Click Detail.
    The Details pane displays at the bottom of the list view with detailed information about the selected record.
  3. Select and view the information in the detail pane tabs..
    • Filters
    • Metrics
Hide group details
  • To hide the Details pane, click Hide Detail.
