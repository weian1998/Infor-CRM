Using the Group Manager - Shares Tab

The Group Manager Shares tab provides a list of all groups that have been shared with users and the tasks needed to manage those groups. Use the filters to narrow the list of groups you want to view and the group tasks to manage groups.

The system administrator and users with the appropriate role can access these features.

How Do I?

Open the Group Manager View Shares tab
  • On the Navigation Bar, click Group Manager, and then click Shares.

Work with records in a grid

Refresh list information
  • Click the Refresh button.

Filter my list

View group details

  1. Select a record.
  2. Click Detail.
    The Details pane displays at the bottom of the list view with detailed information about the selected record.
  3. View the information in the lower pane.

Hide group details

  • To hide the Details pane, click Hide Detail.

Delete a group
  1. Select one or more groups in the list.
  2. Under group tasks, click Delete.
  3. Click OK.
