Using the Competitor List View

The system administrator and users with the appropriate role can access these features.

The Competitor List view displays all of the competitors that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Competitor List View
  • On the Navigation Bar, click Administration, and then click Competitors.
Refresh list information
  • Click the Refresh button.

Use the Summary View

Work with records in a grid

Filter my list

Use the Lookup to search for records

Open the Competitor Detail View
  • Click the competitor's name to open the Competitor Detail view.

View a group

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
