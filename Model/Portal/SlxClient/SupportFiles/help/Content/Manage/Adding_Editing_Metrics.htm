Adding or Editing Metrics

Use the Add Metric or Edit Metric view to add a new metric or edit an existing metric. Metrics determine how Dashboard widget data is calculated.

The system administrator and users with the appropriate role can access these features.

How Do I?

Add a metric
  1. On the Navigation Bar, click Filter Manager.
  2. From the Entities list, select the entity of the filter you want to add .
  3. In the Metrics tab in the lower pane, click Add Metric.
  4. In the Metric Name box, type a unique and descriptive name for the metric.
    There can be no spaces or special characters in the Metric Name.

    		A metric and filter cannot have the same name.
  5. In the Display Name box, type the metric name you want to be displayed in the Infor CRM Web Client.
  6. Click the Type drop-down arrow and select from the list. Options are:
    • Metric - a calculation based on an aggregation only .
    • Date Difference Metric - a calculation based on an aggregation between two date fields.
  7. Click the Fields drop-down arrow and select the entity property you want to use for the metric calculation. If necessary, click the plus sign to expand the list of fields.
    The available fields depend on the entity selected in step 2.
  8. Click the Aggregationsdrop-down arrow and select the type of calculation from the list. Options are:
    • Total (sum)
    • Count
    • Average
    • Minimum Value
    • Maximum Value
  9. If the metric type is Date Difference Metric, then click the From and To drop-down lists and select the date fields to be calculated.
  10. Click Save.
Edit a metric
  1. Select the appropriate entity from the Entities list.
  2. In the Metrics tab in the lower pane, click Edit Metric.
  3. Make your changes.
  4. Click Save.
