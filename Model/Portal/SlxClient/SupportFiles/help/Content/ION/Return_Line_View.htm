Viewing Return Product Details (Back Office Extension)

Infor CRM Back Office Extension only. A Back Office Extension is an ERP system that is configured to send information to display in Infor CRM. Back Office Extension is managed by an administrator. Use the Return Product Details dialog box to view return product information. This information is for display only and must be added or edited in the integrated application. For specific questions please refer to the documentation for the integrated application or contact your administrator.

How Do I?

View return product information
  1. On the Return Detail view, click the Products tab.
  2. Click the View link for the return line you want to view.
  3. View the return product information.
  4. Click the Close button when finished.
Open the Product Detail view
  • Click the Product link.
