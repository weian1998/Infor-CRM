Using the ERP Persons List View

Infor CRM Back Office Extension only. A Back Office Extension is an ERP system that is configured to send information to display in Infor CRM. Back Office Extension is managed by an administrator. The List view displays all of the ERP persons that you have access to. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Persons List view
  • On the Navigation Bar, click ERP Persons.
Refresh list information
  • Click the Refresh button.

Work with records in a grid

Filter my list

Use the Lookup to search for records

Open the ERP Person Detail view
  • Click the person's name to open the ERP Person Detail view.
