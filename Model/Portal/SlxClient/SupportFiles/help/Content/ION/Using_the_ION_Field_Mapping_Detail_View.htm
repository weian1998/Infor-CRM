Using the ION Field Mapping Detail View

Infor CRM Back Office Extension only. A Back Office Extension is an ERP system that is configured to send information to display in Infor CRM. Back Office Extension is managed by an administrator. Use the Detail view to see information for a selected record. 

The system administrator and users with the appropriate role can access these features.

How Do I?

Open the ION BOD Mapping Detail view.
  • On the Navigation Bar, click ION Field Mappings to open the ION Field Mappings List view. If necessary, use the Lookup to locate an ION field mapping, and then click on the ION field mapping Name.

Clone a field mapping

Use the Lookup to search for records

View a group

Switch to the List view
  • On the toolbar click the List View button, or click the current group tab.

information boxes

The information boxes contain the primary information about the ION field mapping. Some information can be edited, however other information cannot be added after the field mapping is created or is for display only and must be added or edited in the integrated application. For specific questions please refer to the documentation for the integrated application or contact your administrator.
