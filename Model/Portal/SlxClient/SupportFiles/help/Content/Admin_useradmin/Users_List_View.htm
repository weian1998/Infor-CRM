Using the Users List View

The system administrator and users with the appropriate role can access these features.

The Users List view displays Infor CRM users in a grid. The grid provides information about each user, such as the user's title and the user type.

  • To sort the grid by a particular field, click one of the column headings.
  • To display all users of a particular type, click a tab in the grid.
