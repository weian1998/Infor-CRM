Managing Calendar Access Rights

Set calendar access using the following options:

  • Add Activities - allows a user to schedule an activity for another user.
  • Delete Activities- allows a user to delete another user's activities.
  • Edit Activities- allows a user to edit another user's activities
  • Sync Activities - allows a user to receive the activities of another user.

This can only be set up for a Remote user. However, Remote users can receive activities from both Network and Remote users.
