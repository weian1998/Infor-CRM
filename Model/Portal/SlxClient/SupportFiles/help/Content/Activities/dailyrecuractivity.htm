Recurring Activities, Daily

You can choose to schedule an activity to occur once, daily, weekly, monthly or yearly.

To schedule an activity to recur daily

  1. Schedule or edit an activity.
  2. Click the Recurring tab.
  3. Under This activity occurs, select Daily.
  4. Select Every X Day(s), or if you want the activity to recur only after the previous occurrences have been completed, select Every X Day(s) after completion.
  5. Enter the number of days between occurrences.
  6. In the Start Recurring box, click the Calendar button, select a date, and then click OK.

  7. Do one of the following:

    • In the End After box, enter the number of times the activity should occur.
    • Click the End Recurring Calendar button, select a different date, and then click OK.
  8. When finished making all activity changes, click OK.
