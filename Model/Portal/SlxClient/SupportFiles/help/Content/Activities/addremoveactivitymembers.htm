Availability Tab

The Availability tab shows you when the leader, other Infor CRM activity members, or any required resources are free or busy to help you schedule your activity. You can only manage members or resources required for a phone call or meeting.

If an Infor CRM user associated with a contact is added to the Availability tab, then the contact is added to the Participants tab. If you add a contact who is associated with an Infor CRM user to the Participants tab, then the associated user is automatically added to the Availability tab, so you can see if the user is free or busy.

To open the Availability tab
  • Click the Availability tab from the Schedule Activity or Edit Activity view.
To view leader, member, and resource availability
  1. Use the Availability tab grid to determine if the leader, activity members and resources are free or busy for the proposed activity time.
  2. If necessary, use the arrow buttons to find a time when all members and resources are available, and then change the Start Time.
To add members and resources (Phone Call and Meeting only)
  1. Click the Add Members and Resources button.
  2. Use the lookup to find members or resources you want to add to the activity.
    Use the Lookup by and Search button to narrow the list.
  3. Select each member or resource and click Add Selected.

To select more than one record, hold down [Ctrl] or [Shift], and highlight the records.
To view member responses
  • Use the Availability tab grid to determine if activity members have accepted, declined or not yet responded to the activity confirmation.
    The user has accepted the activity confirmation.
    The user has declined the activity confirmation.
    No iconThe user has not accepted or declined the activity confirmation.
To remove members or resources
  • Click the Remove button next to the member or resource you want to remove from the activity.
