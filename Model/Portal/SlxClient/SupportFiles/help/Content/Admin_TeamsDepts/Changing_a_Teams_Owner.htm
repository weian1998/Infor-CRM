Changing a Team's Owner

You can use the Teams view or Users view to change a team's owner. Teams may have multiple team owners.

How Do I?

Use the Teams view to manage a team's owner

  1. Open the Team Detail view.
  2. On the Members tab, click the Security Profile link for the user you want to add or remove as the team owner.
  3. In the Security Profile box, select a new security profile for the user.
  4. Click OK.
  5. Click Save.

Use the Users view to manage a team's owner

  1. Open the User Detail view for the user you want to add or remove as team owner.
  2. Click the Team Membership tab.
  3. Click the Security Profile link for the appropriate team.
  4. In the Security Profile box, select a new security profile for the user.
  5. Click OK.
  6. Click Save.
