Troubleshooting Desktop Integration - Mail Merge

You may encounter the following issues with Desktop Integration and Mail Merge.

Problem:

“The mail merge service could not be loaded. This error may occur if Infor CRM Desktop Integration is not installed or is not installed properly.”

Possible Causes and Solutions:

  • The Desktop Integration Module is not installed.
  • You must install the Desktop Integration Module.
  • The Desktop Integration Module was installed before your browser was installed.
  • If you installed your current browser after you installed the Desktop Integration Module, you must reinstall the Desktop Integration Module using your current browser.

    		•

    This can occur if you installed Firefox after installing the Desktop Integration Module using Internet Explorer.

Problem:

“There was an error in GetMailMergeService(). This error may occur if Infor CRM Desktop Integration is not installed or is not installed properly. The specified class name cannot be mapped to a COM object. Details: name=Error; line=257; source=sage-mailmerge-loader-debug.js.”

Possible Causes and Solutions:

  • One of the Mail Merge .dlls did not install properly
  • You must reinstall the Desktop Integration Module.

Problem:

“There was an error in Sage.MailMergeService.PopulateWriteMenu(). This error may occur if Infor CRM Desktop Integration is not installed or is not installed properly. The specified class name cannot be mapped to a COM object. Details: name=SyntaxError; number=-2146827287.”

Possible Causes and Solutions:

  • One of the Mail Merge .dlls did not install properly
  • You must reinstall the Desktop Integration Module.

Problem:

"There was an error in ExecuteWriteAction(). Object doesn't support this property or method."

Possible Causes and Solutions:

  • Your Infor CRM Web Client site is not a trusted site.
  • Add the Infor CRM Web Client site to your Internet browser's list of trusted sites. For more information see your Internet browser's help system.
  • Your browser is running in Protected mode.
  • If you are using Internet Explorer v. 8.0, you must disable Protected mode. For more information see your Internet browser's help system.

Problem:

"An error has occurred in the script on this page. Error: 'Ext.ux.data.Buffered.Json.Reader' is null or not an object."

Possible Causes and Solutions:

  • Your Infor CRM Web Client site is not a trusted site.
  • Add the Infor CRM Web Client site to your Internet browser's list of trusted sites. For more information see your Internet browser's help system.
  • Your browser is running in Protected mode.
  • If you are using Internet Explorer v. 8.0, you must disable Protected mode. For more information see your Internet browser's help system.
