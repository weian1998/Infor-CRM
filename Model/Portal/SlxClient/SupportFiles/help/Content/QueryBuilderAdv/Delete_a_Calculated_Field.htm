Deleting Calculated Fields

 System Administrator Only

You can only open the Manage Calculated Fields dialog box if you are logged on as an administrator.

To delete a calculated field

  1. Open the Query Builder, and then click the Calculations button.

This button only appears if you are logged on as Admin.

  1. In the Manage Calculated Fieldsdialog box, select a calculated field from the grid.
  2. Click Delete.
  3. Click OK to confirm deletion.
  4. Click Close.
