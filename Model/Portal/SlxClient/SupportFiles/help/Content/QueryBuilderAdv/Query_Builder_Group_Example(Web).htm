Example: Using the Query Builder to Create a Group

The following example explains how to create a group of all primary contacts who work in the Sales department of their companies.

  1. Log on to Infor CRM Web Client.
  2. On the Navigation Bar, click Contacts.
  3. On the Groups menu, click Add Group.
  4. In Query Builder, complete the following on the Properties tab:
  5. In the Name box, type Primary Sales Contacts.
  6. In the Description box, type Primary contacts who work in Sales.
  7. On the Conditions tab, complete the following:
  1. In the upper left side of the Query Builder, select CONTACT.
  2. In the upper right side of the Query Builder, double-click Isprimary.

The Assign Condition dialog box appears, with Isprimary in the Field Name box, and String as the Field Type.

  1. In the Operator box, select (=) equal to.
  2. Next to the Value is box, click Browse, select T (for True), and then click OK.

Leave Case Sensitive Compare selected and do not select Use Value as a Literal.

  1. Click OK.
  2. In the upper right side of the Query Builder, double-click Department.

The Assign Condition dialog box appears, with Department in the Field Name box, and String as the Field Type.

  1. In the Operator box, select contains.
  2. Next to the Value is box, click Browse, select Sales, and click OK.
  3. Clear the Case Sensitive Compare check box, and do not select Use Value as a Literal.
  4. Click OK.
  1. Click the Layout tab. The Layout tab shows which fields will appear when the query is saved as a group. Complete the following:
  2. On the right side of the Query Builder, select the Department field and double-click to add it to the grid below.
  1. Use the scroll bar to scroll to the far right, and click Department.
  2. Click the Move Left button repeatedly until Department appears between Account and City
  1. Click the Sorting tab. By default, the contacts are sorted by last name in ascending (A-Z) order. If necessary, you can add additional sorting criteria or change the order.
  2. On the Defaults tab, leave the options set as is.
  3. Click OK.

The new Primary Sales Contacts group appears in the Contact Group list and you can select it to display all contacts that meet the criteria you defined.
