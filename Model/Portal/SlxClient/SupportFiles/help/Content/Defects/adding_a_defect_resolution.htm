Adding a Defect Resolution

When you determine how a defect is going to be resolved, you need to add the information to the defect.

To add

  1. Open the Defect Detail view.
  2. In the Status box, click the drop-down arrow and select an item from the list.
  3. (Optional) In the Fixed in Version box, click the drop-down arrow and select an item from the list.
  1. On the Details tab, use the Resolution box to type the information regarding your repair or process change.
  2. (Optional) If you need to update any tasks for the defect, click the Defect Tasks tab and edit the Defect Tasks.
  3. Click Save.
