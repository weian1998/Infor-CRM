Using the Import History List View

Use the Import History List view to see a list of record imports and imported lead lists.

How Do I?

Open the Import History List View
  • On the Tools menu, click Import History.

Work with a List view

Work with records in a grid

Refresh list information
  • Click the Refresh button.

Filter my list

Use the Lookup to search for records

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
  • Promote to Dashboard
Open the Import History Detail view from the List view
  • From the Import History List view, click the Import Number of the import you want to view.

 

This view contains the following columns:

  • Import Number - Unique number that identifies the lead import.
  • Description - Name of the list that was imported.
  • Create Date - Date the list was imported.
  • Entity Name - Type of records that were imported.
  • Status - Status of the import. For example, Completed.
  • Imported Count - Number of leads that were imported.
  • Duplicate Count - Number of duplicates in the import.
  • Merge Count - Number of records that were merged during the import. (Lead import only)
  • Error Count - Number of errors that occurred during the import.
