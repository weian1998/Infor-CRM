Qualification Items Tab

The system administrator and users with the appropriate role can access these features.

Use this view to add, edit, or remove qualification items.

How Do I?

Open this view
  • On the Qualification Detail view, click the Items tab.

Add a Qualification Item

Edit a Qualification Item

Remove a qualification item
  1. Click Remove next to the qualification item you want remove.
  2. Click OK.

This view contains the following columns:

  • Visible - Indicates whether or not the item displays on the Lead Detail view.
  • Sort Position - Indicates the order that the item displays on the Lead Detail view.
  • Description - Indicates the description that displays on the Lead Detail view.
  • Show Notes - Indicates whether or not a user can type information in a notes box on the Lead Detail view.
