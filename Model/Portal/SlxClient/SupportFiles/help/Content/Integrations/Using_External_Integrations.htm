Using Integrations

Integrations allow communication between Infor CRM and external applications and services. Each integration may communicate different information, or have different communication behavior. Tell me more For example, the Accounting Integration with X3 synchronizes specified accounting information between Infor CRM and X3. Other integrations might have single-direction communication. For example, an integration with a social media service might configure Infor CRM to get information from the service to display it.

After the administrator implements Outlook, Infor CRM communicates with Outlook automatically.
