Using the Integrations List View

The system administrator and users with the appropriate role can access these features.

The Integrations List view displays all of the integrations that you have access to. This list includes all of the integrations released by Infor CRM as well as any custom integrations. You can use groups and lookups to narrow the list of records that display.

How Do I?

Open the Integrations List View
  • On the Navigation Bar, click Integrations.

Work with records in a grid

Refresh list information
  • Click the Refresh button.

Filter my list

Use the Lookup to search for records

Open the Integration Detail View
  • Click the integration's name to open the Integration Detail view.

View a group

Using the Group Shortcut Menu

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
