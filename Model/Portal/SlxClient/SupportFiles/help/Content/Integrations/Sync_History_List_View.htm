Sync History List View

The Sync History List View displays information about synchronization cycles run between Infor CRM and external integrations.

  • To sort the grid by a particular field, click one of the column headings.
  • To only display logs that meet specific criteria, use filters.
  • To only display logs for a certain time-frame, select one of the provided groups.

How Do I?

Open the Sync History List View
  • On the Navigation Bar, click Administration, and then click Sync History.
Open the Sync History Detail View
  • Click the sync history log name to open the Sync History Detail view.

Use the Lookup to search for records
