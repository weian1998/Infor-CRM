Integration Information Boxes

The integration information boxes contain information about the selected integration. Depending on the integration, you can view or edit the information as needed. Some information may not be edited.

The following information displays on the Integration Detail view.

Box Description/Action
Integration The name of the integration.

Authentication Type

 

The method for authenticating data.

Changing the authentication type may cause the integration to fail.

The external service must be authorized by a single service user or multiple users if the method is OAuth 1 or OAuth2, the integration is enabled, and Client ID and Secret are specified.
Sync Method Defines how or if information is synchronized between the integrated system and Infor CRM. Options include Does Not Sync, Client-side Sync and Server Sync. Server Sync indicates the Infor CRM Integration Synchronization Service is used.
Link Type The type of link used to communicate data between the integrated system and Infor CRM. If Admin or System is selected, then the data is managed through a single link. If User is selected, then data is communicated for each user through a separate link.
Client ID

OAuth1 or OAuth2 Authentication Types only.

Assigned by the integration provider during the registration process.
Secret

OAuth1 or OAuth2 Authentication Types only.

Assigned by the integration provider during the registration process.

Enabled

Use this option to turn on or turn off synchronization for this integration.

If this option is not selected, no data will be exchanged between the integrated system and Infor CRM. It will also make all of the following synchronization options unavailable.
Sync Handler

This information is read-only.

Defines a non-SData synchronization handler for an integration that does not expose its own SData sync endpoints.

The handler is responsible for mapping data between the contract and the integration.

Sync Every Defines the number of minutes that should pass between synchronization cycles.
Run sync between Defines the hours between which the integration synchronizes.
