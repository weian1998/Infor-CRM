Sync Jobs List View

The Sync Jobs List View displays information about synchronization jobs run between Infor CRM and external integrations.

  • To sort the grid by a particular field, click one of the column headings.
  • To only display jobs that meet specific criteria, configure filters.

How Do I?

Open the Sync Jobs List View
  • On the Navigation Bar, click Administration, and then click Sync Jobs.
Open the Sync Job Detail View
  • Click the job name to open the Sync Job Detail view.

Use the Lookup to search for records

View a Group

Set synchronization options for an integration
  • Open the Integration Detail view for the integration you want to edit.
