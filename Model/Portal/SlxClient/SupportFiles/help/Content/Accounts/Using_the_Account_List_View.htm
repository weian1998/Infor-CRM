Using the Account List View

The Account List view displays all of the accounts that you have access to. You can use groups and lookups to narrow the list of records that display..

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

How Do I?

Open the Account List View
  • On the Navigation Bar, click Accounts.

Work with records in a grid

Use the Summary View

Refresh list information
  • Click the Refresh button.

Filter my list

Use the Lookup to search for records

Open the Account Detail View
  • Click the account's name to open the Account Detail view.

View a group

 

Using the Group Shortcut Menu

Use common tasks

On the Task Pane, under Common Tasks, you can:

  • Add to Group
  • Remove from Group
  • Save Records as a Group
  • Export to File
  • Promote to Dashboard
