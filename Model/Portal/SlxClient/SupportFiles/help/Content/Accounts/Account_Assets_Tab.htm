Using the Account Assets Tab

Use this tab to view a list of assets associated with won opportunities or added to an account.

How Do I?

Open the Account Assets tab
  1. Open the Account Detail view.
  2. Click the Assets tab. If the tab is not visible, click the More Tabs tab

Add an Asset

Edit an Asset

Delete an Asset

This view contains the following columns:

  • Product Name - Displays the product name.
  • SKU - Displays the SKU of the asset.
  • Serial # - Displays a unique serial number.
  • Qty - Displays the number of this asset owned by the account.
  • Contact - Displays the contact for the account that owns the asset.
  • Location - Displays the location of the asset.
  • Asset # - Displays the asset code.
