Viewing a Map for an Address

This function is only available for the first line of the primary address of an account, contact or lead. The second address line is not used. If the pound (#) sign appears in an address, place it on the second address line to allow MapQuest to better locate the address.

MapQuest may not be able to locate all addresses, or you may need to focus the MapQuest results to find the exact location of an address.

To view the map to an address

  • On a Contact, Account or Lead Detail view, click the MapQuest button.

Another instance of the web browser opens. The MapQuest site displays a map of the area around the address listed. The address is automatically entered for you.
