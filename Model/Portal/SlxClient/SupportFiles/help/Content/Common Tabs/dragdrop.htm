Dragging and Dropping

Use the drag and drop feature to rearrange portions of the Infor CRM interface. If you have installed Desktop Integration, you can also add attachments or Microsoft Outlook e-mail messages to Infor CRM.

How Do I?

Rearrange items in the interface
  1. Move your mouse over the item you want to move. If the tab can be moved, the cursor changes.
  2. Press and hold your mouse button on the item and drag it to the desired position.

For example, another position in a grid, on the tab bar, in the More Tabs area, or in the user defined middle section.

  1. Release your mouse button.
Attach a file using drag and drop

  • If you are using Internet Explorer, this feature requires the installation of the Desktop Integration Module.
  • Drag and drop is not available on the history or activity Attachments tabs.
  1. Browse to the file you want to add.
  2. Drag the file and drop it onto the appropriate record detail view.

The Add Attachment(s) dialog box opens.

    If prompted to install Infor CRM Desktop Integration and you want to use this feature, click Yes and follow the steps to install Infor CRM Desktop Integration. In order to complete the install you will be prompted to close all browser windows and Microsoft Outlook.
  1. In the Description box, enter a description for the file if the default description is not clear.

  2. Click OK.

Record an e-mail message to history
  1. Open the detail view of the appropriate contact, account, lead, opportunity or ticket.
  2. In Microsoft Outlook select the e-mail message you want to add to Infor CRM.
  3. Click the message and press and hold your mouse button on the item and drag it to the open detail view in Infor CRM.
    Depending upon your options, you may be prompted to enter history information.
