Using the Opportunity Detail View

Use the Detail view to see, add, or edit information for a selected record. The Detail view consists of information boxes, tabs, and a user-defined middle section.

Your security access determines what functions are available. Contact your administrator for any access rights changes. WebViewer users may not access full functionality.

How Do I?

Open the Opportunity Detail view
  • On the Navigation Bar, click Opportunities to open the Opportunity List view. If necessary, use the Lookup to locate an opportunity, and then click on the opportunity name.

Use the Lookup to search for records

View a group

Switch to the List view
  • On the toolbar click the List View button, or click the current group tab.
Use common tasks

On the Task Pane, under Common Tasks, you can:

  • E-mail
  • Mail Merge
  • Add Note
  • New Meeting
  • New Phone Call
  • New To-Do
  • Add Sales Order

information boxes

The opportunity information boxes contain the primary information about the opportunity. Users with appropriate permissions can edit these boxes. The opportunity snapshot appears in the upper section of the detail view as well.

detail view tabs

Each tab shows specific information you can use for opportunity related activities. Click a link to see more information about each tab.

Contacts Notes/History More Tabs
Activities Attachments Products
Competitors Sales Processes Sales Orders

user defined section

You can drag and drop tabs into the user defined middle section. The tab will remain there until it is moved back to the lower section.
