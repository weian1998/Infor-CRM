Updating Multiple Opportunities

You can update multiple opportunities at once using the Update Opportunities dialog box.

To update multiple opportunities with different information

  • Use the Opportunity grid to view and modify opportunity information, and then click OK to save the changes and close the Update Opportunities dialog box.

To update multiple opportunities

  1. On the Navigation Bar, click the Opportunities button.
  2. Open the group that contains the opportunities that you want to update.
  3. Select the records you want to update, and do one of the following:
    • Right-click and select Update Opportunities.
    • In the Task Pane, under Common Tasks, click Update Opportunities.
  4. Use the Update Opportunities dialog box to update any of the following information:

    • Account Manager

      1. From the Update drop-down list, select Account Manager.
      2. Click the To: Find button, select a new account manager from the list, and then click OK.
    • Add to Forecast
      1. From the Update drop-down list, select Add to Forecast?.
      2. Select Yes to include opportunities in forecasting reports, or select No to exclude opportunities from forecasting reports.
    • Close Probability %
      1. From the Update drop-down list, select Close Probability %.
      2. Click the drop-down arrow, and select a new close probability from the list.

    • Estimated Close

      1. From the Update drop-down list, select Estimated Close, .
      2. Select how you would like to update the estimated close date:
        • To move the estimated close date to a specific date, select the To: option, click the Calendar button, select a new date, and then click OK.
        • To move the estimated close date backwards a certain number of days, select the Move option, select Backward from the drop-down list, and then enter the number of days you want to move the date backwards.
        • To move the estimated close date forward a certain number of days, click the Move option, select Forward from the drop-down list, and then enter the number of days you want to move the date forward.
    • Comments
      1. From the Update drop-down list, select Comments.
      2. Type your notes in the To: box next to the drop-down box.
  5. Click OK.
