Literature Requests Tab

The Literature Requests tab displays on the Account and Contact Detail Views, and lists all literature requests for the account or contact. The Literature Requests Tab contains the request date, send date, and what the request is regarding.

How Do I?

View a literature request
  1. From the Account or Contact Detail view, click the Literature Requests tab. If the tab is not visible, click the More Tabs tab.
  2. Click theDescription to view more information about the selected literature request.
    The Literature Request Detail View opens.
Schedule a literature request
  1. Click the Schedule Literature Request button.
  2. Follow the steps for scheduling a literature request.
