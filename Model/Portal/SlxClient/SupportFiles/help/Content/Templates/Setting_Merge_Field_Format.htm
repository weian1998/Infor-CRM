Setting Merge Field Format

Use the Format Infor CRM Field dialog box to set the format type and format string for an Infor CRM merge field.

The default Field Format for most text fields is None. However, if the field contains numeric information, a phone number, currency, a true/false value, or other data, change the Field Format to match the type of information. Look at the options in the Field Format list to select a match for the type of data you are using.

Show me examples
  • Select Date as a format type, then you would type the format string as mm/dd/yyyy.
  • The User format type will format a USERID as “LastName, FirstName” by default, and when you use the FormatString “FL” it will format the USERID as “FirstName LastName”.

To set the merge field format

  1. In the Format Infor CRM Field dialog box, select a type from the Format Type list.
  2. In the Format String box, type the string needed for the format type.
  3. Click OK.
