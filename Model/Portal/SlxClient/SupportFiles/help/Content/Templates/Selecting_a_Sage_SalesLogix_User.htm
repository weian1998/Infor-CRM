Selecting an Infor CRM User

The Select User dialog box is used when you are:

  • Creating or copying a template, and you need to select an owner for a template
  • Setting history options for Mail Merge, and you need to select a leader for a follow-up activity.

How Do I?

Select an owner for a template
  1. Create or copy a template.
  2. In the template properties dialog box, at the Owner box, click Lookup.
  3. In the Select User dialog box, do one of the following to select an owner:
    • Select a user from the list, and then click OK.
    • Double-click a user's name.
Select a leader for a follow-up activity
  1. On the Write menu, click Mail Merge.
  2. Select a template for the merge.
  3. On the History Options tab, in the Schedule a Follow-up area, complete the information to schedule a follow-up activity.
  4. At the Leader box, click Lookup.
  5. In the Select User dialog box, do one of the following to select an owner:
    • Select a user from the list, and then click OK.
    • Double-click a user's name
