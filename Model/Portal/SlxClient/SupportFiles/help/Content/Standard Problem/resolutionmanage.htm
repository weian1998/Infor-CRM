Adding or Editing a Standard Problem Resolution

The system administrator and users with the appropriate role can access these features.

Standard problem resolutions provide information for resolving a commonly occurring problem.

How Do I?

Add a resolution
  1. On the New menu, click Standard Resolution.
  2. Add the following information:
  • Description - Type a description for the resolution.
  • Notes - Type any notes to describe how to resolve the problem.

  1. Click Save

Edit a resolution
  1. On the Navigation Bar, click Service, and then click Standard Resolutions to open the Standard Resolutions List view. If necessary, use the Lookup to locate a resolution, and then click the standard resolution ID.
  2. Edit any of the following information:
  • Description - Type a description for the resolution.
  • Notes - Type any notes to describe how to resolve the problem.

  1. Click Save.

Delete a Standard Resolution
  1. On the Navigation Bar, click Service, and then click Standard Resolutions to open the Standard Resolution List view. If necessary, use the Lookup to locate a resolution, and then click on the standard resolution ID.
  2. On the Standard Resolution Detail view, click the Delete button.
  3. Click OK.
