Editing Product Information

Use the Edit Product dialog box to edit a sales order's product information.

To edit

  1. Open the Sales Order Detail view, and click the Products tab.
  2. Click the Edit link for the product you want to change.
    To edit the product in the grid, select the product you want to edit and select or enter information in the appropriate column.
  3. In the Edit Product dialog box, edit the information as appropriate.
    • In the Description box, edit the product description.
    • In the Family box, select a product family.
    • Click the Price Level drop-down arrow and select the price level to be used.
    • In the Discount box, enter a discount percentage.
      The number is automatically changed to a percent. For example, "50" will be "50%". If you enter ".5", then it will be ".5%". The Adjusted Price is automatically calculated.
    • In the Quantity box, enter the number of products.

    Additional information may be available if your implementation includes a Back Office Extension. A Back Office Extension is an ERP system that is configured to send information to display in Infor CRM. Back Office Extension is managed by an administrator. This information is for display only and must be added or edited in the integrated application. For specific questions please refer to the documentation for the integrated application or contact your administrator.

  4. Click OK.
