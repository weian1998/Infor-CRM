Using the Resource Detail view

The system administrator and users with the appropriate role can access these features.

Use the Detail view to see, add, or edit information for a selected record.

How Do I?

Open the Resource Detail view
  • On the Navigation Bar, click Administration, and then click Resources to open the Resources List view. If necessary, use the Lookup to locate a resource, and then click on the resource name.

Use the Lookup to search for records

Switch to the List view
  • On the toolbar click the List View button, or click the current group tab.

Use the Task Pane group list or common tasks

View a group

Information Boxes

The resource information boxes contain the primary information about the resource. Users with appropriate permissions can edit these boxes.
