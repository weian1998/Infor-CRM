Asset Information Boxes

The following table describes the default information boxes.

Box Description/Action
Name The name of the asset. Click the Find button and edit using thelookup.
SKU The Stock Keeping Unit (SKU).
Quantity The units of this asset owned by the account.

Serial Number Prefix

Serial Number Start

Serial Number Increment

 Use these boxes to create a unique serial number for the asset.
Contact The name of the contact. Click the Find button and edit using the lookup.
Phone Displays the contact's phone number.
E-mail Displays the contact's e-mail address.

Location

Asset Code

Environment

System

 Use these boxes to enter additional information specific to your business.
Version The version of the asset.
Last Updated

The day on which the asset was added or most recently edited.
Updated By

The user who added or most recently edited the asset.
Description Description of the asset.
Returned Select this box if the asset has been returned. This box may not be available.
Evaluation Select this box if the asset is an evaluation copy.
Opportunity Displays the related opportunity if applicable.
Sales Order Displays the related sales order if applicable.
Comments Additional information regarding the asset can be added in this box.
