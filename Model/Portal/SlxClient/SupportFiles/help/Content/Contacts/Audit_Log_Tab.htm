Contact Detail View Audit Log Tab

Use the Audit Log tab on the Contact Detail view to see contact-specific usage of the Infor CRM Customer Portal. This tab will only contain data if the contact is set up with web access to the Infor CRM Customer Portal.

Each time an event occurs, a record is created that also captures helpful information related to the event. A separate record is created for each of the following events:

  • Forgot User name
  • Forgot Password
  • Change Password
  • Login
  • Login Failure
  • Logout

To open the Audit Log tab

  1. Open the Contact Detail view for the contact you want to view.
  2. Click the Audit Log tab. If the tab is not visible, click the More Tabs tab.
  3. View the following:
    • Event Name - the type of event that occurred.
    • Event Date - the date that the event occurred.
    • Event Source - the client on which the event occurred. For example, Customer Portal.
    • Browser - the browser name and version used to accessInfor CRM Customer Portal.
    • IP Address - the IP address of the client used to access Infor CRM Customer Portal.

      		An IP Address of :1 indicates that the contact accessed the localhost address.
    • Language - the language setting specified by the contact.
    • Time Zone - the time zone of the client used to access Infor CRM Customer Portal.
    • Additional Info - additional information about the event. The contents of this column depend upon the type of event.
