Search for Matches

(Accounting Integration Only) The Search for Matches screen allows you to select the criteria to use when searching the accounting system for customers that are potential matches for the account you are linking from Infor CRM.

To select

  1. Open the Link to Accounting wizard and proceed through the previous screens.
  1. Complete the Search for Matches screen.
    BoxDescription/Action
    PropertySelect a field to use when searching for matching data. The available fields are fields mapped between Infor CRM and the account system.
    OperatorSelect the matching criteria used by the system for the selected property. Available operators are Equal To, Starts With, and Contains.
    Search ValueType the value you want to search on.
    Add or remove search criteriaTo add additional matching criteria, click . To remove matching criteria, click .
  2. Click Next to go to the next step.
