Exporting Records to a File

You can export records to a file. The group column headings and formatting will also be exported.

If the Export to File option is unavailable, you must download Infor CRM Desktop Integration. You can install Infor CRM Desktop Integration from the Web Client login screen. 

To export records to a file

  1. View the group or lookup results that contain the records you want to export.
  2. Do one of the following:
    • Select one or more records by pressing the CTRL or Shift key as you click each item.From the Task Pane, under Common Tasks, select Export to File.

      If you do not select any records, you will be asked if all records in the group should be used. If you click OK, all group records will be selected.

    • To export the entire group, right-click the group tab and click Export to File.

    		3.

    If you do not select any records, you will be asked if all records in the group should be used. If you click OK, all group records will be selected.
  3. Select which file format you would like to use.
    • Standard CSV - individual records are separated by a comma
    • Tab Delimited- individual records are separated by a tab

    If you want Infor CRM to remember your selection and you do not want to be prompted again, select Remember my selection and do not ask again.

  4. Click OK.

You may receive an error if you try to export a group that includes a data field with a large amount of data.
