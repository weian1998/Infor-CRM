Adding or Removing Filter Items

Filters allow you to narrow the list of records that display in a list view. You can customize your filters to only include the filter items you want to use for each group.

Some filters cannot be customized.

To add or remove filter items

  1. From a list view, in the Task Pane, under Filters, click a filter name. For example, click Owner.
  2. Click Edit Items.
    The Edit Filter Items dialog box opens.
  3. If necessary, type the first couple of letters in the Find Item box and click Find to search for a filter item.
  4. Select the check boxes of the filter items you want to display in the Filters list and clear the check boxes of the items you want to hide. Click Select All to select all filter items.
  5. Click OK.
    The filter includes only the selected items.
