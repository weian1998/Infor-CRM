Customizing Your List View Groups

You can customize your list view tabs by adding or removing columns, changing the column order and adjusting the column widths. Infor CRM will remember these changes each time you log in using the same browser and computer. You can also hide or show the list view check boxes.

These changes will not be available from another computer and will be lost if you clear your browser cache. To make permanent changes to a group, use the Query Builder Layout tab.

How Do I?

Add or remove columns
  1. On a list view, open the group you want to edit.
  2. Right-click any column header and select the columns you want to add and clear the columns you want to remove.
  3. When finished click outside of the box.
Change the column order
  1. On a list view, open the group you want to edit.
  2. Click the column header of the column you want to move and drag it to the desired location.
Change the column width
  1. On a list view, open the group you want to edit.
  2. Place your mouse over the edge of the column until the cursor changes and then click and drag the edge of the column to the desired width.
Hide or show the check box column
  1. On a list view, open the group you want to edit.
  2. Right-click any column header and select the first option to show the check boxes or clear the first option to hide the check boxes.
  3. When finished click outside of the box.
Hide or show check boxes on all groups
  1. On the Tools menu, click Options.
  2. On the Groups tab
Undo my customizations
  • On a list view, right-click the group tab you want to undo your changes for and click Revert Layout.
    Undoes any customizations made to the group and returns the group appearance to layout determined by the Query Builder Layout tab.
