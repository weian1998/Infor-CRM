Linking Contact Addresses

The Link Contact Addresses step allows you to associate the contacts you linked in the previous screen with address information while merging records.

To link address for merged records

  1. Open the Linking wizard or Merge Records wizard and proceed through the previous screens.
  2. Complete the Linked Contact Addresses screen.
    BoxDescription/Action
    Linked Contacts

    Displays the linked contacts from the previous screen.

    Click Select to select contact details.

    Unlinked Source Addresses from

    Displays unlinked addresses for the selected source contact.

    Click Select to select address details.

    Unlinked Target Addresses from

    Displays unlinked addresses for the selected target contact.

    Click Select to select address details.

    Link button

    Select an unlinked address from each list, and then click Link to create a connection..

    Linked Contact Addresses

    Displays any linked contact addresses..

    • Click Unlink to clear the proposed link and create a new link.
  3. Click Merge.
