Linking Addresses

Use the Link Addresses step to determine what address information will be included when merging records.

The system administrator and users with the appropriate role can access these features.

To link address for merged records

  1. When merging records from a list view, click Next.
  2. Complete the Link Addresses screen.
    BoxDescription/Action
    Unlinked Source Addresses from

    Displays the addresses in source that will be replaced in the merged record unless the address is linked to a target address.

    Click Select to select address details.

    Unlinked Target Addresses from

    Displays the addresses in target that will be remain associated with the merged record unless the address is linked to a target address.

    Click Select to select address details.

    Link button

    Select an unlinked address in the Unlinked Source Addresses and Unlinked Target Addresses from the lists, and then click Link to create a connection.

    Linked Addresses

    Displays the list of linked addresses.

    Click Unlink to clear the proposed link.

  3. Click Next to go to the next step.
