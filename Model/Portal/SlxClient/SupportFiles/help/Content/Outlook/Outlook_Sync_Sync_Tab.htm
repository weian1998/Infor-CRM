Setting Infor CRM Connector Sync Options

The Sync options determine what and how information will synchronize between Infor CRM and Outlook.

This tab is required for Outlook sync only.

To set options

  1. Click the Infor CRM Options button and click Sync.
  2. If you want to synchronize contacts, complete the following:
    1. Select the Synchronize Contacts check box.
    2. Click the Ad hoc Group drop-down arrow and select the contact group you created in Infor CRM that will contain the contacts to synchronize.
      If the group you created does not appear in the list, click the Refresh button next to the drop down to refresh the list.
    3. Click the Folder drop-down arrow and select the folder in Outlook that will contain the contacts to synchronize.
    4. Click the Sync Direction drop-down arrow and select how you want contacts to be synchronized between Outlook and Infor CRM.
  3. If you want to synchronize appointments (Infor CRM phone calls, meetings and personal activities), complete the following:
    1. Select the Synchronize Appointments check box.
    2. Click the Folderellipsis button and select the folder in Outlook that will contain the appointments to synchronize. The default is Calendar.
    3. Click the Sync direction drop-down arrow and select how you want contacts to be synchronized between Outlook and Infor CRM.
    4. To limit the appointments to be synchronized, select the On first sync, link appointments newer than check box, click the drop-down arrow and select a date from the calendar.
      Only appointments (or phone calls, meetings and personal activities) scheduled on or after the selected date will be synchronized. However, activities that are edited (including snoozing or dismissing an alarm) after the selected date and any past occurrences of a current recurring activity will synchronize.
  4. If you want to synchronize tasks (Infor CRM to-dos), complete the following:
    1. Select the Synchronize Tasks check box.
    2. Click the Folder ellipsis button and select the folder in Outlook that will contain the tasks to synchronize. The default is Tasks.
    3. Click the Sync Direction drop-down arrow and select how you want contacts to be synchronized between Outlook and Infor CRM.
    4. To limit the tasks to be synchronized, select the On first sync, link tasks newer than check box, click the drop-down arrow and select a date from the calendar.
      Only tasks (or to-dos) scheduled on or after the selected date will be synchronized. However, activities that are edited (including snoozing or dismissing an alarm) after the selected date and any past occurrences of a current recurring activity will synchronize.
  5. In the Sync interval box, select the number of minutes between synchronizing between Outlook and Infor CRM.
  6. Click OK if you are finished setting all of your options, or select another options tab.
