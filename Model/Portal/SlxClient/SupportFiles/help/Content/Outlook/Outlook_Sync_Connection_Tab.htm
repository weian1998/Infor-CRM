Setting Infor CRM Connector Connection Options

Use the Connection options to set up your connection to the Infor CRM server.

  • This tab is required for both Outlook sync and Outlook Integration.
  • These options may already be configured for Web Client users who set their options in the Desktop Manager.

To set options

  1. Click the Infor CRM Options button and, if necessary, click Connection.
  2. In the Server box, type your server name.
    • Infor CRM Client and Infor CRM Web Client users - Type the URL that your administrator provided in order to connect to Infor CRM. Use the following format:
      http://servername:port or https://servername:port
    • Offline Web Client users - Type the URL to connect to your local Infor CRM server. Use the following format:
      http://localhost:port
      The port is usually 8088.

  3. Do one of the following:

    • If you use your Windows account information to log in to Infor CRM, select the Use Windows Authentication option.
    • Enter your Infor CRM User Name and Password.
  4. Click Test.
    If the test fails, review your connection information. If the information is correct and you still cannot connect, contact your administrator.
  5. Under Connect the Go To CRM button to, select the Infor CRM client you use.
    1. If you use the Infor CRM Windows Client, select the Infor CRM Windows Client option.
      This option will be unavailable if the Infor Windows Client is not installed.
    2. If you use the Infor CRM Web Client, select the Infor CRM Web Client option and then if necessary, in the Url box type the URL that your administrator provided in order to connect to Infor CRM. Use the following format:
      http://servername:port/slxclient or https://servername:port/slxclient
  6. Click OK if you are finished setting all of your options, or select another options tab.
