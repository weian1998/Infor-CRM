Changing Your Infor CRM Connector Server

If you change your server on the Connection tab, your data will be synchronized with the new Infor CRM server. However this should not result in duplicate records.

To change

  1. Click the Infor CRM Options button and click Connection.
  2. In the Server box, type the URL for the new Infor CRM server.
  3. If necessary, change your user name and password.
  4. Click the Test button to test the new connection.
  5. On the Sync tab, click the Ad Hoc Group dropdown arrow and select the contact sync group.
    You must re-select the Ad Hoc Group even if the group on the new server has the same name.
  6. Click OK.
