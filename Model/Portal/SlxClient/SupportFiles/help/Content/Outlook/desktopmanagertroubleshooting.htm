Troubleshooting Desktop Integration Module

You may encounter the following issues with the Desktop Integration module.

Problem:

The Infor CRM Desktop Manager icon does not appear in my SysTray.

Possible Causes and Solutions:

  • The Desktop Integration Module is not installed.
  • You must install the Desktop Integration Module.
  • The Desktop Integration Module was installed before your browser was installed.
  • If you installed your current browser after you installed the Desktop Integration Module, you must reinstall the Desktop Integration Module using your current browser.

    		•

    This can occur if you installed Firefox after installing the Desktop Integration Module using Internet Explorer.
  • Browse to the SLXDesktopManager.exe and open it manually. The location of this file depends on your operating system:

The default location is:

\\Documents and Settings\[username]\Local Settings\Application Data\ (Windows XP/2000/2003) or

\\Users\<user.domain>\appdata\local (Windows Vista/7/2008).

Problem:

How do I turn off the Desktop Manager?

Solution:

  • In the System Tray section of the Windows Taskbar, right-click the Infor CRM Desktop Manager icon and click Exit.

You must first exit Infor CRM Desktop Manager before you can uninstall the Infor CRM Desktop integration Module.

Problem:

How do I uninstall the Desktop Integration Module?

Solution:

  1. Open Windows Add or Remove Programs.
  2. Select Infor CRM Desktop Integration Module from the list and click Remove.
  3. Click Yes.

    		4.

    If you use Firefox as your browser, you must first uninstall the Infor CRM Desktop Integration Module Add-on in Firefox.

    To uninstall the Desktop Integration Module add-on, open Firefox. On the Tools menu, click Add-ons, select Infor CRM Desktop Integration Module and click Uninstall. Click Uninstall to confirm. Click Restart Firefox. Click Restart.
