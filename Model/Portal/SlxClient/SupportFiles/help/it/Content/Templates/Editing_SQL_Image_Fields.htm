Modificare Campi Immagine SQL

E' possibile usare l'Editor dei Campi Stampa Unione per modificare campi unione nuovi e Immagine SQL esistenti. Quando si seleziona un campo Immagine SQL dalla vista elenco, le opzioni di modifica verranno visualizzate nel pannello di modifica.

  • Identificatore Campo - Consente di modificare il testo che appare nel modello.
  • SQL - Consente di inserire un'istruzione SELECT per richiedere dati dall'OLE DB provider. Soltanto il primo record della prima colonna sarà visualizzato.
    Fare clic sul pulsante Modifica per aprire la finestra di dialogo Modifica SQL.
  • Formatta Come - Visualizza il tipo campo unione.
  • Risultati Query - Consente di selezionare come viene archiviata l'immagine. E' possibile scegliere tra:
  • Dati Immagine blob

    L'immagine è archiviata come campo blob nel database. E' necessario anche selezionare un tipo immagine dall'elenco. Il tipo immagine è usato per creare un nome file temporaneo con l'estensione del tipo file.

  • Percorso completamente qualificato e nome file di un'immagine

    Usare questa opzione se il record restituito dalla query SQL contiene un percorso completamente qualificato e il nome di un'immagine.

    Se l'immagine comincia con HTTP, sarà inserito come un collegamento ipertestuale a quel file; altrimenti verrà archiviata una copia dell'immagine col documento.

  • Indirizzo Collegamento ipertestuale - Consente di inserire un URL che può essere aperta dal campo unione.
  • Esempi di Indirizzo Collegamento Ipertestuale
  • http://www.infor.com/
  • mailto:jabbott@abbott.demo?Subject=Offerta Speciale&Body=Sì, inviatemi informazioni su questa offerta speciale!
  • mailto::SLXUserName <:SLXUserEmail>?Subject=Special Offer&Body=Sì, inviatemi informazioni su questa offerta speciale!
  • ftp://ftp.winzip.com/winzip81.exe
  • news://msnews.microsoft.com/microsoft.public.word.mailmerge.fields
  • Target Collegamento Ipertestuale - Consente di scegliere come sarà aperto l'URL. Dall'elenco selezionare un Target.
    • Opzione Target Descrizione
    Pagina Predefinita (nessuna) Apre il collegamento alla stessa finestra.
    Stesso Frame

    Apre il collegamento allo stesso frame.

    Corrisponde al target HTML di ''_self''.
    Pagina Intera

    Apre il collegamento nel corpo completo della finestra corrente.

    Corrisponde al target HTML di ''_top''.
    Finestra Nuova

    Apre il collegameto in una finestra nuova senza nome.

    Corrisponde al target HTML di ''_blank''.
    Frame Principale

    Apre il collegamento alla finestra principale del documento col collegamento.

    Corrisponde al target HTML di ''_parent''.
  • Consiglio su Schermo Collegamento Ipertestuale - consente di digitare un testo che apparirà quando il puntatore del mouse è posizionato sul collegamento. I
    Parametri possono essere inclusi nell'Indirizzo Ipertestuale e nel Consiglio su Schermo e possono essere utili per creare un mailto: collegamento ipertestuale.
    • Infor CRM Parametri Descrizione
    SLXAccountManagerEmail Il parametro :SLXAccountManagerEmail sarà sostituito con l'indirizzo email del responsabile per il record contatto con cui viene unito.
    SLXAccountManagerName Il parametro :SLXAccountManagerName sarà sostituito con il nome del responsabile per il record contatto con cui viene unito.
    SLXAccountManagerTitle Il parametro :SLXAccountManagerTitle sarà sostituito con il titolo del responsabile per il record contatto con cui viene unito.
    SLXAccountManagerPhone Il parametro :SLXAccountManagerPhone sarà sostituito con il numero di telefono del responsabile per il record contatto con cui viene unito.
    SLXAccountManagerFax Il parametro :SLXAccountManagerFax sarà sostituito con il numero fax del responsabile per il record contatto con cui viene unito.
    SLXUserEmail Il parametro :SLXUserEmail sarà sostituito con l'indirizzo email dell'utente stampa unione.
    SLXUserName Il parametro :SLXUserName sarà sostituito con il nome dell'utente stampa unione.
    SLXUserTitle Il parametro :SLXUserTitle sarà sostituito con il titolo dell'utente stampa unione.
    SLXUserPhone Il parametro :SLXUserPhone sarà sostituito con il numero di telefono dell'utente stampa unione.
    SLXUserFax Il parametro SLXUserFax sarà sostituito con il numero di fax dell'utente stampa unione.
