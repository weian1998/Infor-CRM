Allegare o Rimuovere un File da un Modello

E' possibile allegare ad un modello file di Windows o della Libreria di Infor CRM. Quando il modello viene utilizzato in un'unione stampa i file allegati vengono automaticamente inviati al contatto o al nominativo. Durante la copia di un modello vengono copiati anche i suoi allegati. In aggiunta, Ã¨ possibile rimuovere un allegato da un modello.

Per allegare un file ad un modello

  1. Aprire l'Editor dei Modelli.
  2. A seconda del tipo di file da allegare, eseguire uno dei seguenti passaggi:
    • File di Windows: Fare clic su Inserisci File. Localizzare il file, quindi fare clic su Inserisci File.
    • File della Libreria: Fare clic su Inserisci File LibreriaInfor CRM. Nella finestra di dialogo Libreria Infor CRM trovare i file e fare clic su Inserisci.

      Il nome del file viene visualizzato nel campo Allegati.
  3. E' possibile allegare altri file, oppure fare clic su Salva e Chiudi.

Per rimuovere un allegato da un modello

  1. Aprire l'Editor dei Modelli.
  2. Nella casella Allegati, selezionare l'allegato da eliminare.
  3. Per rimuovere l'allegato, eseguire uno dei passi seguenti:
    • Clic destro e poi fare clic su Rimuovi.
    • Premere Elimina.
  4. E' possibile rimuovere un altro file, oppure fare clic su Salva e Chiudi per chiudere l'Editor Modelli.
