Usare l'Editor del Fax

Con l'editor del Fax è possibile creare un nuovo modello o modificarne di esistenti. Così come gli editor Lettera ed Email, quello del fax contiene una barra degli strumenti Infor CRM che permette di accedere a speciali funzioni sui modelli così come molte funzioni di Microsoft Word.

L'editor Fax visualizza le opzioni seguenti quando si modifica o crea un modello:

Campo Descrizione/Azione
Inserisci Campo SLX Fare clic per visualizzare un menu coi campi unione Infor CRM usati comunemente.
Inserisci File Questa opzione è disponibile solo per il modello tipo email.
Inserisci file della Libreria Questa opzione è disponibile solo per il modello tipo email.
Salva e Chiudi Fare clic per salvare le modifiche effettuate al modello e chiudere l'editor del modello.
Salva Fare clic per salvare le modifiche effettuate al modello ma lasciare l'editor del modello aperto.
Mostra Questa opzione è disponibile solo per il modello tipo email.
Formato Messaggio Questa opzione è disponibile solo per il modello tipo email.
Anteprima Fare clic per visualizzare il modello così come sarebbe se fosse unito al record corrente.
Stampa Unione SLX Fare clic per lanciare la Stampa Unione.
Oggetto Fax Fare clic per inserire un oggetto per il modello fax. Quando si esegue una Stampa Unione con questo modello e si seleziona Fax come Output, la riga Oggetto della Stampa Unione si compila in automatico con il testo dalla linea oggetto del modello.
Mostra Menù Word Fare clic per visualizzare il menù di Microsoft Word.
Modifica Campi Fare clic per aprire la finestra Infor CRMEditor dei Campi Stampa Unione Infor CRM.
Inserire Campi Speciali Fare clic per inserire una nuova tabella prodotto opportunità, istruzione SQL, o campi stampa unione personalizzati nel modello.
Opzioni Email Questa opzione è disponibile solo per il modello tipo email.
