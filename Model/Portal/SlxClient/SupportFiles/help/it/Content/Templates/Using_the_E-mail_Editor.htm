Utilizzare l'Editor Email

Con la finestra Editor Email è possibile creare un nuovo modello di email oppure modificarne uno esistente. Così come per gli Editor Lettera e Fax, l'Editor Email contiene una barra di strumenti Infor CRM che danno accesso alle funzioni speciali per modelli nonchè alle funzionalità di Microsoft Word.

L'editor Email visualizza le opzioni seguenti quando si modifica o crea un modello:

Campo Descrizione/Azione
Inserisci Campo SLX Fare clic per visualizzare un menu coi campi unione Infor CRM usati comunemente.
Inserisci File Fare clic per inserire un file Windows come allegato all'email. Questa opzione è disponibile solo per il modello tipo email.
Inserisci file della Libreria Fare clic per inserire file dalla Libreria come allegati all'email. Questa opzione è disponibile solo per il modello tipo email.
Salva e Chiudi Fare clic per salvare le modifiche effettuate al modello e chiudere l'editor del modello.
Salva Fare clic per salvare le modifiche effettuate al modello ma lasciare l'editor del modello aperto.
Mostra Consente di selezionare quale dei seguenti campi email saranno visualizzati nell'editor. Da, A, Oggetto, e Allegati. Questa opzione è disponibile solo per il modello tipo email.
Formato Messaggio Fare clic sulle frecce per selezionare un formato messaggio dall'elenco. Questa opzione è disponibile solo per il modello tipo email.
Anteprima Fare clic per visualizzare il modello così come sarebbe se fosse unito al record corrente.
Stampa Unione SLX Fare clic per lanciare la Stampa Unione.
A: Questo campo è popolato automaticamente con l'indirizzo email del contatto e non può essere modificato da questa finestra.
Oggetto Fare clic per inserire un oggetto per il modello email. Quando si esegue una Stampa Unione con questo modello e si seleziona Email come Output, la riga Oggetto della Stampa Unione si compila in automatico con il testo dalla linea oggetto del modello.
Mostra Menù Word Fare clic per visualizzare il menù di Microsoft Word.
Modifica Campi Fare clic per aprire la finestra Infor CRMEditor dei Campi Stampa Unione Infor CRM.
Inserire Campi Speciali Fare clic per inserire una nuova tabella prodotto opportunità, istruzione SQL, o campi stampa unione personalizzati nel modello.
Opzioni Email Fare clic per cambiare opzioni quali Rispondi a, Importanza e Sensibilità.

Quando si seleziona un formato messaggio per un modello, fare attenzione che Microsoft Word non visualizzi intestazione e piè di pagina o grafici in pagina HTML, archivia i dati.
