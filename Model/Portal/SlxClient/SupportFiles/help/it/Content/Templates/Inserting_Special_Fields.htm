Inserire Campi Speciali

L'opzione è disponibile solo per utenti esperti.

E' possibile fare clic sul pulsante Inserisci Campi Speciali dell'Editor dei Modelli Email, Fax o Lettera per inserire una nuova tabella prodotto opportunità, istruzione SQL o campo stampa unione personalizzato.

Per inserire un campo speciale

  1. Aprire l'Editor dei Modelli.
  2. Dalla barra degli strumenti, fare clic su Inserisci Campo Speciale.
  3. Dal menù, selezionare il tipo di campo che si desidera aggiungere al modello.
    • Tabella Prodotto Opportunità

    Aggiunge una tabella con le informazioni prodotto opportunità e prezzo.

    • SQL

      • E' possibile inserire i seguenti campi SQL:

      • Testo
      • Immagine
      • Tabella
      • Elenco Puntato
      • Elenco Numerato
    • Personalizzato

      • Se viene aggiunto un campo personalizzato al modello e gli script non sono stati modeificati correttamente, la stampa unione non funzionerà finchè non vengono corretti gli script oppure rimosso il campo personalizzato.

    Prima di inserire un campo personalizzato, il campo deve essere trattato nel codice della stampa unione SLX sullo script OnCustomFieldName (LAN) oppure nella funzione di classe Sage.MailMergeService.prototype.HandleOnCustomFieldname localizzata in jscript\sage-mailmerge\sage-mailmerge-service.js.

    Si apre la finestra di dialogo Editor campo Stampa Unione Infor CRM.

  4. Se necessario, usare il campo modifica per fare le modifiche e poi fare clic su Applica.
  5. Fare clic su OK.
