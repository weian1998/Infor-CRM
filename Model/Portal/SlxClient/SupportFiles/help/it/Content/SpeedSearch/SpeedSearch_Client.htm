SpeedSearch

SpeedSearch è un potente strumento che consente di trovare informazioni archiviate nel database Infor CRM o su file esterni. Ad esempio, è possibile cercare nei ticket esistenti o procedure per aiutare a risolvere un problema del cliente, oppure cercare un documento specifico, come una presentazione di vendita. Il tipo di informazione che è possibile cercare dipende dagli indici impostati sull'Administrator dal proprio amministratore di sistema.

Per eseguire una ricerca con SpeedSearch

  • Effettuare una delle seguenti operazioni:

    • Inserire le informazioni di ricerca nella barra dei menù.
    • Fare clic sul pulsante SpeedSearchnella scheda Dettagli Ticket o nella vista Inserisci Ticket.
    • Nel menu Strumenti, clic su SpeedSearch Avanzato.

Se si manifestano problemi durante l'apertura o l'utilizzo di SpeedSearch, contattare il proprio amministratore Infor CRM. Le informazioni di configurazione e di risoluzione di problemi di SpeedSearch sono disponibili nella Guida di Administrator.
