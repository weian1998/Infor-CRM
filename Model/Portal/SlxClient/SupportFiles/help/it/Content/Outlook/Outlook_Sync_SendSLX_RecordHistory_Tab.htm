Impostare le Opzioni del Connettore Infor Invia a SLX/Registra in Cronologia

Utilizzare Invia a SLX/Registra in Cronologia per impostare le opzioni di cronologia e-mail e allegati.
  • Questa scheda è richiesta solo per l'integrazione Outlook.
  • Queste opzioni possono già essere configurate per gli utenti del client Web che impostare le opzioni nel Desktop Manager.

Per configurare le opzioni

  1. Cliccare il pulsante Infor CRMOpzionie cliccare Invio SLX/Record di Cronologia.
  2. Selezionare quando si desidera essere avvisati durante la registrazione di un'email Outlook all'interno della cronologia Infor CRM.
    1. Avvisa in caso di Contatti Duplicati - Avvisa in caso di contatti o nominativi duplicati.
    2. Avvisa per i Contatti - Avvisa per selezionare il contatto o il nominativo a cui dovrebbe essere associata la cronologia.
    3. Avvisa in caso di Contatti Non Risolti - Avvisa quando non bviene trovato un contatto o nominativo nel database.
    4. Visualizza i Dettagli della Cronologia prima di salvare - Avvisa con la finestra di dialogo Completa Email prima di registrare un messaggio email in cronologia.
    5. Notifiche di Visualizzazione - Visualizza i messaggi pop-up che descrivono le attività di Gestione Desktop come occorre.
  3. Impostare le opzioni degli allegati.
    1. Selezionare la minima taglia dell'immagine dell'allegato da ignorare.
      Utilizzare questa opzione per ignorare i piccoli file allegati, quali firme di posta elettronica, in modo che non vengono salvate come allegato Infor CRM.
    2. Nella finestra Ignorare allegati con questi nomi di file, elencare le estensioni di file o qualsiasi tipo di file che non devono essere salvati come allegati Infor CRM.
  4. Cliccare OK se si sta finendo l'impostazione di tutte le opzioni, o selezionare un'altra scheda opzioni.
