Cambiare il proprio Connettore Server Infor

Se si cambia il proprio server nella scheda Connessione, i dati saranno sincronizzati col nuovo servizio Infor CRM. Tuttavia questo non dovrebbe comportare record duplicati.

Per modificare

  1. Cliccare il pulsante Infor CRMOpzioni e cliccare Connessione.
  2. Nella casella Server digitare l'URL per il nuovo server Infor CRM.
  3. Se necessario, cambiare il proprio nome utente e la password.
  4. Cliccare il pulsante Verifica per verificare la nuova connessione.
  5. Sulla scheda Sincronizzazione, cliccare la freccia a discesa Gruppo Ad Hoc e selezionare il gruppo di sincronizzazione contatto.
    E' necessario selezionare nuovamente il gruppo ad hoc, anche se il gruppo sul nuovo server ha lo stesso nome.
  6. Fare clic su OK.
