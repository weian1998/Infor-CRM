Impostare le Opzioni di Connessione del Connettore Infor

Utilizzare le opzioni Connessione per impostare la propria connessione per il server Infor.

  • Questa scheda è richiesta per entrambe Outlook sync e l'integrazione Outlook.
  • Queste opzioni possono già essere configurate per gli utenti del client Web che impostare le opzioni nel Desktop Manager.

Per configurare le opzioni

  1. Cliccare il pulsante Infor CRMOpzioni e cliccare Connessione.
  2. Nella casella Server, inserire il nome del proprio server.
    • InforClienti e Utenti Client Web - Digitare l'URL che l'amministratore ha fornito, al fine di connettersi a Infor. Utilizzare il seguente formato:
      http://servername:port or https://servername:port
    • Gli utenti del client Web offline - Digitare l'URL per connettersi al server locale Infor CRM. Utilizzare il seguente formato:
      http://localhost:port
      La porta è solitamente 8088.

  3. Effettuare una delle seguenti operazioni:

    • Se si utilizza l'autenticazione di Windows per connettersi a Infor CRM, selezionare l'opzione Usa l'Autenticazione di Windows.
    • Inserire il proprio InforNome Utente e Password.
  4. Cliccare Verifica.
    Se il test fallisce, rivedere le informazioni di connessione. Se le informazioni sono corrette e ancora non riesce a connettersi, contattare l'amministratore.
  5. Cliccare OK se si sta finendo l'impostazione di tutte le opzioni, o selezionare un'altra scheda opzioni.
