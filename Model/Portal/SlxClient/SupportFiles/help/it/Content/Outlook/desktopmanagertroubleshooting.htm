Risoluzione di Problemi del Modulo Desktop Integration

E' possibile riscontrare i seguenti problemi durante l'utilizzo del modulo Desktop Integration.

Problema:

L'icona Infor CRM Desktop Manager non appare nella mia barra di sistema.

Cause e Soluzioni possibili:

  • Il modulo Desktop Integration non è installato.
  • E' necessario installare il modulo Desktop Integration.
  • Il modulo Desktop Integration è stato installato prima del browser.
  • Se si è installato il browser corrente dopo il modulo Desktop Integration, è necessario reinstallare il modulo Desktop integration utilizzando il browser corrente.

    		•

    Questo può accadere se si è installato Firefox dopo aver installato il modulo Desktop Integration utilizzando Internet Explorer.
  • Sfogliare le cartelle in cerca di SLXDesktopManager.exe e aprirlo manualmente. Il percorso di questo file varia in base al proprio sistema operativo:

Di norma è:

\\Documents and Settings\[nome utente]\Impostazioni Locali\Dati Applicazioni (XP/2000/2003) oppure

\\Utenti\<utente.dominio>\appdata\local (Vista/7/2008).

Problema:

Come disattivo Desktop Manager?

Soluzione:

  • Nella barra di sistema della Barra applicazioni di Windows fare doppio clic sull'Infor CRMicona Gestione Desktop e fare clic su Esci.

E' necessario uscire dalla Gestione Desktop Infor CRM prima di poter disinstallare il modulo Desktop Integration Infor CRM.

Problema:

Come disinstallo il Modulo Desktop Integration?

Soluzione:

  1. Aprire Aggiungi/Rimuovi Programmi di Windows.
  2. Selezionare Infor CRMModulo Integrazione Desktop nell'elenco e fare clic su Rimuovi.
  3. Fare clic su Sì.

    		4.

    Se si utilizza Firefox come browser, è necessario prima disinstallare il modulo aggiuntivo Desktop Integration da Firefox Infor CRM.

    Per disinstallare il modulo aggiuntivo Desktop Integration, aprire Firefox. Sul menù Strumenti, fare clic su Componenti Aggiuntivi, selezionare Desktop Integration Module Infor CRM e poi fare clic su Disinstalla. Fare clic su Disinstalla per confermare. Fare clic su Riavvia Firefox. Fare clic su Riavvia.
