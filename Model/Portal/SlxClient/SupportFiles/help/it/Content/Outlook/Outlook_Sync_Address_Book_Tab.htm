Impostare le Opzioni della Rubrica Infor

Utilizzare le opzioni Rubrica per impostare l'apparenza della propria rubrica Infor CRM.

Questa scheda Ã¨ richiesta solo per l'integrazione Outlook.

Per configurare le opzioni

  1. Cliccare il pulsante Infor CRMOpzioni, e se necessario, cliccare Rubrica.
  2. Sotto Mostra nome contatto come selezionare il modo in cui si desidera che i nomi appaiono nella rubrica.
  3. Sotto Quando viene mostrata la Rubrica, selezionare o cancellare le opzioni per personalizzare la propria rubrica.
  4. Cliccare OK se si sta finendo l'impostazione di tutte le opzioni, o selezionare un'altra scheda opzioni.
