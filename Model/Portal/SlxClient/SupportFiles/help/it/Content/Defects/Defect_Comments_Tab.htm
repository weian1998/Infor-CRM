Usare la scheda Commenti

Questa scheda può essere utilizzata per aggiungere tutte le note interne relative a un difetto o ad un contratto.

Per aggiungere

  1. Effettuare una delle seguenti operazioni:

    • Aprire la vista Dettaglio Difetto.
    • Aprire la vista Dettaglio Contratto.
  2. Fare clic sulla scheda Commenti. Se la scheda non è visibile, fare clic sulla scheda Altre Schede..
  3. Digitare le informazioni nel campo Commenti.
