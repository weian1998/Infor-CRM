Scheda Dettagli Difetto

La scheda Dettagli appare nella vista Dettaglio Difetto. Questa scheda contiene informazioni specifiche sul difetto.

Per modificare la scheda dettagli

  1. Dalla vista Dettaglio Difetto, fare clic sulla scheda Dettagli.Se la scheda non Ã¨ visibile, fare clic sulla scheda Altre Schede..
  2. Nel campo Soggetto, digitare una breve descrizione del problema.
  3. Utilizzare il campo Descrizione per digitare una descrizione dettagliata del problema.
  4. Utilizzare il campo Soluzione per digitare i passaggi effettuati per risolvere il difetto.
  5. Fare clic su Salva..
