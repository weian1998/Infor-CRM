Risoluzione dei Problemi di Desktop Integration - Stampa Unione

E' possibile riscontrare i seguenti problemi con Desktop Integration e Unione Stampa.

Problema:

''Il servizio di unione stampa non può essere avviato. Questo errore può avere luogo se Desktop Integration Infor CRM non è installato oppure non è installato correttamente.''

Cause e Soluzioni possibili:

  • Il modulo Desktop Integration non è installato.
  • E' necessario installare il modulo Desktop Integration.
  • Il modulo Desktop Integration è stato installato prima del browser.
  • Se si è installato il browser corrente dopo il modulo Desktop Integration, è necessario reinstallare il modulo Desktop integration utilizzando il browser corrente.

    		•

    Questo può accadere se si è installato Firefox dopo aver installato il modulo Desktop Integration utilizzando Internet Explorer.

Problema:

Errore in GetMailMergeService(). Questo errore può avere luogo se Desktop Integration Infor CRM non è installato oppure non è installato correttamente. Il nome della classe specificata non può essere associato a un oggetto COM. Dettagli: nome=Error; riga=257; origine=sage-mailmerge-loader-debug.js.''

Cause e Soluzioni possibili:

  • Una delle .dll di Unione Stampa non è stata installata correttamente.
  • E' necessario installare il modulo Desktop Integration.

Problema:

Errore in Sage.MailMergeService.PopulateWriteMenu(). Questo errore può avere luogo se Desktop Integration Infor CRM non è installato oppure non è installato correttamente. Il nome della classe specificata non può essere associato a un oggetto COM. Dettagli: nome=SyntaxError; numero=-2146827287.''

Cause e Soluzioni possibili:

  • Una delle .dll di Unione Stampa non è stata installata correttamente.
  • E' necessario installare il modulo Desktop Integration.

Problema:

''Errore in ExecuteWriteAction(). L'oggetto non supporta questa proprietà o metodo.''

Cause e Soluzioni possibili:

  • Il proprio sito Client Web Infor CRM non è attendibile.
  • Aggiungere il sito Client Web Infor CRM all'elenco dei siti affidabili del proprio browser. Per maggiori informazioni consultare il sistema di guida del proprio browser.
  • Il proprio browser è eseguito in modalità Protetta.
  • Se si utilizza Internet Explorer v. 8.0, è necessario disattivare la modalità Protetta. Per maggiori informazioni consultare il sistema di guida del proprio browser.

Problema:

Un errore è occorso nello script di questa pagina. Errore: 'Ext.ux.data.Buffered.Json.Reader' è null or non è un oggetto.''

Cause e Soluzioni possibili:

  • Il proprio sito Client Web Infor CRM non è attendibile.
  • Aggiungere il sito Client Web Infor CRM all'elenco dei siti affidabili del proprio browser. Per maggiori informazioni consultare il sistema di guida del proprio browser.
  • Il proprio browser è eseguito in modalità Protetta.
  • Se si utilizza Internet Explorer v. 8.0, è necessario disattivare la modalità Protetta. Per maggiori informazioni consultare il sistema di guida del proprio browser.
