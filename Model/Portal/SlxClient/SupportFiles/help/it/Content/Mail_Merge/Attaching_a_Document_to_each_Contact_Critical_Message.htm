Messaggio Critico Allegare un Documento a Ogni Contatto o Nominativo

Per gli utenti Infor CRM Web, questo messaggio viene visualizzato se si effettua l'unione di un gran numero di contatti.

Se si fa clic su Sì, verrà allegato un documento ad ogni contatto; se il numero dei contatti è elevato, il processo di unione sarà insolitamente lento. Questo perchè ogni documento deve essere trasferito via HTTP al server Web, quindi copiato nel percorso allegati e registrato nel percorso log. Questo processo potrebbe richiedere molto tempo e rallentare il server Web. Si raccomanda, per effettuare l'unione di un gran numero di contatti, la disponibilità di una connessione veloce. Idealmente, si raccomanda inoltre di non unire più di 100 contatti se il percorso degli allegati e il percorso di log sono anche sul server Web (condizioni ottimali).

Se si fa clic su No, l'opzione rimarrà deselezionata e non verrà allegato alcun documento.
