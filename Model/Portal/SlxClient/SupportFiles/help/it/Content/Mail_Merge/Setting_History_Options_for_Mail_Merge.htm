Impostare le Opzioni Cronologia di Stampa Unione

La scheda Opzioni Cronologia permette di creare una voce cronologia per la stampa unione e di creare un'attività seguito..

Per impostare le Opzioni Cronologia

  1. Nella scheda Opzioni Cronologia, nell'area Cronologia, effettuare le seguenti operazioni:
    1. Per allegare la corrispondenza a ciascuna scheda Allegati del record selezionare Allega <email/fax/lettera> a ciascun <Contatto/Nominativo>.
      Il file di unione è allegato a ciascun nominativo o contatto, azienda ed opportunità in base all'opzione Unisci con selezionata nelle Opzioni di Unione.
      Per gli utenti Infor CRM Web questa opzione causerà la visualizzazione di un messaggio di avviso durante l'unione.
    2. Per aggiungere una voce alla cronologia del record, selezionare Aggiungi cronologia a ciascun <Contatto/Nominativo>.
    3. Nella casella Risultato, fare clic sul pulsante con i puntini di sospensione per selezionare un tipo di risultato dall'elenco Codici Risultato Impegni.
    4. Nella casella Argomento, fare clic sul pulsante con i puntini di sospensione per selezionare un tipo di argomento dall'elenco.
    5. Nella casella Categoria, fare clic sul pulsante con i puntini di sospensione per selezionare un tipo di categoria dall'elenco.
    6. Nella casella Note, digitare note o elementi importanti che si desidera includere nella cronologia.
  2. Per pianificare un'attività seguito selezionare Pianifica un Seguito ed effettuare le seguenti operazioni:
    1. Se si riceve un messaggio di conferma attività, fare clic su OK.
    2. Nell'elenco selezionare il tipo di attività seguito.
    3. Per riportare le note nell'attività seguito, selezionare Riporta note.
    4. Nella casella Orario di Inizio fare clic sul pulsante calendario per selezionare la data in cui avrà luogo l'attività.
    5. Effettuare una delle seguenti operazioni:
      • Spuntare la casella Senza orario per creare un'attività che abbia luogo in una specifica data ma non ad un'ora o con una durata prestabilite.
      • Fare clic su DurataFare clic sulla freccia e selezionare dall'elenco..
    6. Per impostare un allarme d'attività selezionare Promemoria. Utilizzare le caselle a scorrimento per selezionare il numero di minuti, ore o giorni. E' necessario inserire un incremento compreso tra 1 e 240.
    7. Nella casella Argomento, fare clic sul pulsante con i puntini di sospensione per selezionare un tipo di argomento dall'elenco.
    8. Nell'area Note, eseguire uno dei seguenti passi:
      • Digitare le note nella casella.
      • Fare clic su Note per aprire una casella più grande, digitare le note, quindi fare clic su OK.
    9. Nella casella Priorità selezionare l'importanza dell'attività.
    10. Nella casella Categoria, fare clic sul pulsante con i puntini di sospensione per selezionare un tipo di categoria.
    11. Nella casella Leader, utilizzare una Ricerca per selezionare un leader d'attività.
  3. Una volta terminata la configurazione delle opzioni Unione e Cronologia, fare clic su Unisci.
