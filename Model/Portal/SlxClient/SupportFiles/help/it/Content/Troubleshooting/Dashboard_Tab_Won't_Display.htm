Una scheda su My Dashboard non visualizza o apre un sito Web diverso

Problema:

Quando cerco di visualizzare una scheda sul mio Dashboard, accade una delle seguenti cose:

  • La scheda non si apre.
  • Sono subito portato in un altro sito web.

Soluzione:

La scheda dispone di un widget sito web che accede a un sito web che non è ottimizzato per l'uso di widget. Si deve cancellare la scheda e ricrearla.

Si dovrebbe prima creare una scheda ''Verifica'' e aggiungere il widget sito web lì per assicurarsi che il sito web si comporta come previsto prima di aggiungere un widget sito web di una scheda esistente.
