Raggruppare Espressioni Condizionali

In una query database avanzata, potrebbe essere necessario non solo utilizzare condizioni multiple, ma anche raggruppare queste condizioni per poter eseguire ricerche più complesse.

Per raggruppare espressioni condizionali

  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Condizioni ed aggiungere tutte le espressioni condizionali.
  3. Per la prima condizione del gruppo, fare clic sul pulsante Aggiungi (+) nella colonna col simbolo delle parentesi aperte ''(''.
  4. Per l'ultima condizione del gruppo, fare clic sul pulsante Aggiungi (+) nella colonna col simbolo delle parentesi chiuse '')''.
  5. Ripetere fino al completamento dei raggruppamenti desiderati. Sotto-livelli sono possibili, ma potrebbero creare confusione. Ogni condizione ulteriore verrà applicata come sotto-livello della condizione precedente.

Per rimuovere il gruppo dalle espressioni condizionali

  1. Per la prima condizione del gruppo, fare clic sul pulsante Rimuovi (-) nella colonna col simbolo delle parentesi aperte ''(''.
  2. Per l'ultima condizione del gruppo, fare clic sul pulsante Rimuovi (-) nella colonna col simbolo delle parentesi chiuse '')''.
  3. Modificare i connettori come necessario.
