Stringhe Formato per Campi Interi

La tabella seguente elenca le opzioni stringhe formato per i campi interi. Formati Stringa validi dipendono dal tipo formato selezionato.

Un formato stringa deve cominciare con il segno % e terminare con un carattere.

[%] [tipo]
Opzione Descrizione
d Decimale. Il valore deve essere un intero. Il valore viene visualizzato come decimale.
x Esadecimale. Il formato del campo deve essere un valore intero. Il valore viene convertito in una stringa di cifre esadecimali.
