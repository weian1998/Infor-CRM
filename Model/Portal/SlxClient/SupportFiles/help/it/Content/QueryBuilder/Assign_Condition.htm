Assegnare Condizioni alla Query

Quando si aggiunge un campo nella scheda Condizioni del Query Builder, apparirà la finestra di dialogo Assegna Condizione relativa al campo.

Per assegnare

  1. Aprire il Query Builder
  2. Fare clic sulla scheda Condizioni.
  3. Aggiungere un'istruzione condizionale.

Si aprirà la finestra di dialogo Assegna Condizione.

  1. Selezionare un Operatore.
  2. Nel campo Il Valore è inserire un valore oppure fare clic su Sfoglia per selezionare un valore.

Se si seleziona l'operatore ''In'', sarà necessario delimitare tutti i valori con virgolette.

  1. Selezionare Confronta maiuscole/minuscole per usare le maiuscole come parte delle condizione.

Se si utilizza MS SQL Server, l'opzione Confronta Maiuscole/Minuscole non è applicabile. Tutte le query sono insensibili ai caratteri maiuscoli/minuscoli.

  1. Se necessario, selezionare Utilizza Valore come Testo Letterale.
  2. Fare clic su OK.
