Cos'è il Query Builder?

Il Query Builder consente di selezionare dati dal database sulla base di una o più condizioni. Ad esempio, si potrebbe aver necessità si vedere un elenco di contatti primari nel settore bancario. Il Query Builder è connesso a diverse aree dei variInfor CRM. Per utilizzare al meglio il Query Builder, occorre comprendere la struttura del database Infor CRM ed i concetti dei database relazionali, come le relazioni tra i dati e le join.
