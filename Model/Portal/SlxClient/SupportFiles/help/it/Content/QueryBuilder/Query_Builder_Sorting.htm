Ordinamento nel Query Builder

Utilizzare la scheda Ordinamento del Query Builder per specificare quali campi utilizzare per l'ordinamento dei dati e se i dati devono apparire in ordine crescente o decrescente. Il criteri di ordinamento sono applicati nell'ordine in cui appaiono nella scheda.

  • Il Query Builder supporta molte funzioni. Determinate schede diventano visibili in base alla funzionalità che si sta utilizzando.

  • Alcuni report includono criteri di ordinamento predefiniti che potrebbero essere eliminati, se necessario.
  • I campi blob non possono essere utilizzati per ordinare i dati.

(Binary Large OBject) Un tipo di dati che contiene una grande quantità variabile di dati. I blob possono avere dati binari o testo. Utilizzare il Blob Viewer nella finestra di dialogo SQL Result Set per vedere i dati blob di testo.

  • I campi calcolati numericamente non possono essere utilizzati per ordinare i dati perchè i calcoli matematici non vengono completati prima dell'applicazione dell'ordinamento.
Per aggiungere un criterio di ordinamento
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Ordinamento.
  3. Selezionare una tabella dal lato sinistro del Query Builder. Se necessario, fare clic sul segno più per espandere l'elenco delle tabelle.
  4. Selezionare un campo dal lato destro del Query Builder. Questo campo visualizza tutti i campi che appartengono alla tabella selezionata. Fare doppio clic sul campo per aggiungerlo alla griglia nella parte bassa della finestra di dialogo.
  5. fare clic sulla freccia Direzione e poi selezionare Ascendente (ordina dalla A alla Z) oppure Discendente (ordina dalla Z alla A).
  6. Per aggiungere un campo di ordinamento secondario, ripetere i passi 3-5. Ogni ulteriore ordinamento verrà applicato come un sotto-livello dell'ordinamento precedente.

I campi calcolati numericamente non possono essere ordinati.

Per riordinare i criteri di ordinamento
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Ordinamento.
  3. Selezionare un campo nella griglia.
  4. Fare clic su Sposta in Alto o Sposta in Basso.
Per eliminare un criterio di ordinamento
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Ordinamento.
  3. Selezionare il campo da eliminare nella griglia.
  4. Fare clic su Elimina.
