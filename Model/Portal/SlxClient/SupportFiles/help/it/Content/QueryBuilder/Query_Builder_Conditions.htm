Condizioni del Query Builder

Utilizzare la scheda Condizioni del Query Builder per aggiungere, modificare o eliminare espressioni condizionali che Infor CRM applicherà sul database per selezionare i dati. Le condizioni sono applicate nell'ordine con cui appaiono nella griglia; comunque, è possibile utilizzare i pulsanti Sposta in Alto e Sposta in Basso per cambiare l'ordine. E' inoltre possibile raggruppare le espressioni condizionali.
  • Il Query Builder supporta molte funzioni. Determinate schede diventano visibili in base alla funzionalità che si sta utilizzando.
  • Tutti i gruppi creati dovrebbero contenere almeno una condizione. I gruppi che contengono tutti i record potrebbero rallentare le prestazioni del sistema, ed appare un messaggio che avvisa l'utente all'apertura di tali gruppi.
Per aggiungere un'espressione condizionale
  1. Aprire il Query Builder.
  2. Nel Query Builder, fare clic sulla scheda Condizioni.
  3. Selezionare una tabella dal lato superiore sinistro del Query Builder. Se necessario, fare clic sul segno più per espandere l'elenco delle tabelle.
  4. Selezionare un campo dal lato superiore destro del Query Builder. Questo campo visualizza tutti i campi che appartengono alla tabella selezionata. Fare doppio clic sul campo per aggiungerlo alla griglia nella parte bassa della finestra di dialogo.

Si aprirà automaticamente la finestra di dialogo Assegna Condizioni. E' possibile aprire questa finestra di dialogo in ogni momento facendo doppio clic su di un'espressione nella griglia.

  1. Selezionare un Operatore.
  2. Nella casella Valore, digitare un valore o fare clic su Sfoglia per selezionare un valore dall'elenco delle voci esistenti per il campo.

Se si seleziona l'operatore ''In'', sarà necessario delimitare tutti i valori con virgolette.

  1. Selezionare Confronta maiuscole/minuscole se si desidera utilizzare le informazioni relative all'uso di maiuscole e minuscole del valore utilizzato per la condizione. Per impostazione predefinita, questa opzioni è selezionata.

Se si utilizza MS SQL Server, l'opzione Confronta Maiuscole/Minuscole non è applicabile. Tutte le query sono insensibili ai caratteri maiuscoli/minuscoli.

  1. Se necessario, selezionare Utilizza Valore come Testo Letterale.
  2. Fare clic su OK.
  3. Per aggiungere un'altra espressione, ripetere i passi 1-9. Quindi, selezionare il connettore appropriato.
Per modificare un'espressione condizionale
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Condizioni.
  3. Selezionare un'espressione nella griglia (parte inferiore della finestra di dialogo Query Builder)
  4. Fare clic su Modifica.
  5. Nella finestra di dialogo Assegna Condizioni, effettuare le modifiche.
  6. Fare clic su OK.
Per eliminare un'espressione condizionale
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Condizioni.
  3. Selezionare un'istruzione nella griglia.
  4. Fare clic su Elimina.
