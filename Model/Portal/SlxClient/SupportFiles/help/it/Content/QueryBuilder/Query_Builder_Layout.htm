Layout del Query Builder

Utilizzare la scheda Layout del Query Builder per specificare quanti campi verranno visualizzati ed il modo con cui i dati appariranno. E' possibile aggiungere, modificare, eliminare o riordinare i campi nel layout. E' inoltre possibile impostare il nome, la larghezza ed il formato delle colonne di ciascun campo. La scheda Layout deve includere almeno un campo oppure non verrà visualizzato alcun dato.

Il Query Builder supporta molte funzioni. Determinate schede diventano visibili in base alla funzionalità che si sta utilizzando.
Per aggiungere un campo in un layout
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Layout.

    L'aspetto predefinito per i nuovi gruppi è determinato dal gruppo selezionato come gruppo predefinito.

  3. Selezionare una tabella.
  4. Selezionare un campo.

E' possibile scorrere l'elenco layout per vedere come è stato aggiunto il campo al layout.

  1. Selezionare il campo e fare clic su Modifica.
  2. Completare la finestra di dialogo Assegna Layout alla Query.
Per modificare un campo in un layout
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Layout.
  3. Selezionare un campo dalla griglia.
  4. Fare clic su Modifica.
  5. Fare le modifiche nella finestra di dialogo Assegna Layout alla Query.
Per eliminare un campo in un layout
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Layout.
  3. Selezionare un campo nella griglia.
  4. Fare clic su Elimina.
Per riordinare i campi di un layout
  1. Aprire il Query Builder.
  2. Fare clic sulla scheda Layout.
  3. Selezionare un campo. Effettuare una delle seguenti operazioni:
  • Trascinare il campo nella nuova posizione.
  • Fare clic su Sposta a Sinistra oppure Sposta a Destra finchè il campo è nella posizione corretta.
