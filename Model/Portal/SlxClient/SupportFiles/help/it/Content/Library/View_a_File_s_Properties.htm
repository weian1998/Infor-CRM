Proprietà dei Documenti della Libreria

Utilizzare la finestra di dialogo Proprietà dei Documenti per aggiungere o modificare le proprietà dei file presenti nella Libreria. In aggiunta ai campi modificabili, questa finestra fornisce le seguenti informazioni di sola lettura:

  • Directory: La posizione in cui il documento è salvato.
  • Stato: Si applica soltanto per gli utenti remoti. Può essere Disponibile, Ordinato, Spedito, Non letto, Letto, Revisionato.
  • Creato il: La data in cui il file è stato aggiunto alla libreria.
  • Dimensioni: Le dimensioni del file.

Possono accedere a queste funzioni l'amministratore di sistema e gli utenti con il ruolo appropriato.

Per visualizzare o configurare le proprietà di un documento della Libreria

  1. Sulla barra di navigazione fare clic su Libreria.
  2. Nell'elenco selezionare il documento da modificare, quindi fare clic sul pulsante Proprietà del File.
  3. Utilizzare il campo Revisionatoper modificare la data e l'ora dell'ultima modifica del file, consentendo così l'invio di questo file ai Client remoti.
    Questo campo non si aggiorna automaticamente.
  4. Deselezionare la casella Nessuna Scadenza per attivare il campo Scade.
    Questa casella è spuntata per impostazione predefinita.
  5. Inserire una data nella casella di testo Scade. Questa opzione può essere utilizzata per documenti urgenti come i listini prezzi.
  6. Utilizzare la casella di testo per includere un commento lungo fino a 255 caratteri riguardante il file.
    La descrizione del file apparirà nella vista dettagli della Libreria.
  7. Utilizzare il campo Estratto per includere un commento lungo fino a 1024 caratteri riguardante il file.
    L'estratto può essere visualizzato soltanto in questa finestra di dialogo.
  8. Spuntare l'opzione Forza la distribuzione di questo file per inviare automaticamente i file e le loro revisioni a tutti gli utenti remoti.
  9. Fare clic su OK.
