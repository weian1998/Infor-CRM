Aggiungere una Query Avanzata al Gruppo

Utilizzare il Query Builder per Creare un Gruppo. Le query avanzate consentono di impostare le condizioni per il gruppo. Ogni volta che un gruppo è aperto, la query esamina il database per rintracciare i record che dovranno essere visualizzati. Se si desidera accertarsi che il contenuto del gruppo non cambi, si consiglia di creare un gruppo ad hoc.

 

Creare una query avanzata nel gruppo

  1. Aprire una vista elenco o di dettaglio.
  2. Fare clic sul pulsante Aggiungi Gruppo.
    Si apre il Query Builder Infor CRM.
  3. Servirsi della scheda Condizioni per definire le espressioni condizionali usate per selezionare i record.
  4. Se necessario, utilizzare le seguenti schede del Query Builder:
    • Scheda Proprietà - Consente di definire il nome e la descrizione del gruppo.
    • Scheda Layout - Consente di definire quali dati sono mostrati nella vista Elenco.

      Il layout predefinito è determinato dal gruppo selezionato come Gruppo Predefinito nelle opzioni di Gruppo.

    • Scheda Ordinamento - Consente di definire la modalità di ordinamento dei dati.
  5. Al termine delle modifiche, fare clic su OK.
