Modificare i Gruppi

Per modificare i propri gruppi, è possibile utilizzare il Query Builder.

Per modificare un gruppo

  1. Aprire una vista elenco o di dettaglio.
  2. Fare clic destro sulla scheda di un gruppo.
  3. Fare clic su Modifica.

Si apre il Query Builder.

  1. Per modificare il gruppo, utilizzare le schede del Query Builder.
  • Scheda Proprietà - Consente di definire il nome e la descrizione del gruppo.
  • Scheda Condizioni - Consente di impostare le istruzioni condizionali utilizzate per selezionare i record. Questa scheda non è disponibile per i gruppi ad hoc.
  • Scheda Layout - Consente di definire i dati visualizzati nella vista Elenco.
  • Scheda Ordinamento - Consente di definire la modalità di ordinamento dei record.
  1. Al termine delle modifiche, fare clic su OK.
