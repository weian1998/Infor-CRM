Cosa sono i Report?

La visualizzazione elenco Reports fornisce un elenco di report di esempio predefiniti per consentire di visualizzare e analizzare le informazioni nel database Saleslogix. E' possibile eseguire report relativi alle viste principali, quali contatti, azienda, opportunità e così via. Se si preferisce, è possibile modificare il report per soddisfare le tue esigenze di business. Le relazioni specifiche disponibili sono stabiliti dall'amministratore di sistema.

Infor CRM WebLa segnalazione consente di:

  • Selezionare da un elenco di report di esempio predefiniti e applicare i filtri per perfezionare i risultati
  • Pianificare un report per l'esecuzione in un secondo momento, una volta o su un programma
  • Consulta l'elenco dei rapporti e delle volte sono programmati per l'esecuzione.
  • Rapporti per visualizzare, stampare e memorizzare.

Per aprire la vista Report

  • Sulla barra di navigazione fare clic su Report.

Per eseguire report Web su uno stato disconnesso, il Client Web Offline bisogna eseguire su Windows 2003 con IIS.Devono essere installati e configurati anche il Server del Web Reporting Infor CRM Web e il . Se non vengono soddisfatti questi requisiti, gli utenti del Client Web disconnesso possono eseguire Web Reporting soltanto se dispongono di una connessione a Host Web Reporting Server. Questi utenti vedranno i report che rappresentano i dati del database host e non del loro database remoto.
