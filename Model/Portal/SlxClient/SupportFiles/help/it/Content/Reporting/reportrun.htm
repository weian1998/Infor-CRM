Eseguire un Report

I report consentono di vedere le informazioni sui contatti, aziende, opportunità, ticket e altri dati correlati.

Per eseguire un report

  1. Sulla barra di navigazione fare clic su Report.
  2. Nella vista Elenco record, scheda Report, selezionare un record, fare clic col tasto destro, e poi cliccare una delle seguenti:
    • Elimina Rimuove i record selezionati dal proprio elenco.
    • Esegui: Selezionare questa opzione per i report che si vogliono eseguire una sola volta, o report richiesti immediatamente.
    • Pianifica: Selezionare questa opzione per impostare un giorno, la data e l'ora di esecuzione di un report e di una ricorrenza. Utilizzare questa opzione per i report che si devono eseguire regolarmente, o report di grandi dimensioni che si desidera eseguire in orari non di punta.
  3. Fare le proprie selezioni utilizzando Report Job Scheduler wizard, e poi cliccare Fine.
  4. Nel messaggio di conferma, eseguire una delle azioni seguenti:
    • Attendere la fine del job e aprire il report dal link che viene visualizzato.
    • Fare clic su Chiudi per chiudere la finestra di dialogo.Il report continua l'elaborazione in background. Quando il report è disponibile, compare un avviso di Notifica Job nella barra dei menù. E' possibile visualizzare il report dal campo Notifiche Job o dalla scheda Cronologia Report.
