Scheda Posizioni

La scheda Posizioni della vista Dettaglio Azione Protetta visualizza il punto in cui l'azione protetta è utilizzata. Sono visualizzati il nome del file e il percorso server del sito Web di ciascun elemento. Le informazioni sono di sola lettura.

Nel momento in cui un'azione protetta è eliminata o rinominata viene disaccoppiata, in Application Architect, dall'elemento. Prima di modificare l'azione protetta, Infor CRM raccomanda consiglia di salvare le informazioni presenti su questa scheda come record cronologia. E' possibile copiare e incollare le informazioni in un file di testo.

Se necessario, si possono utilizzare le informazioni presenti in questa scheda per ricreare un'azione protetta o rimuoverla dagli elementi in Application Architect.
  • Quando si accede per la prima volta a questa scheda, il caricamento delle informazioni può richiedere diverso tempo.
  • Se vengono apportate modifiche alle azioni protette, le informazioni nella scheda verranno aggiornate nel momento in cui il sito Web è rimodellato e pubblicato.
