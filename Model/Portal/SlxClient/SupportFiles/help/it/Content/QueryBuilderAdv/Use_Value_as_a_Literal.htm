Utilizzare un Valore come Testo Letterale

La selezione di Utilizzare un Valore come Testo Letterale, nella finestra di dialogo Assegna Condizione, posiziona il testo nella query esattamente come compare nel campo Valore. In molti casi, non si dovrebbe selezionare questa opzione in modo che il Query Builder possa determinare l'istruzione SQL corretta. Comunque, in alcuni casi, utenti avanzati potrebbero voler digitare un testo specifico che dovrebbe essere essere scritto direttamente nell'istruzione SQL.

Ad esempio, si potrebbe voler digitare una funzione di sistema specifica per il database (come ad esempio la GETDATE() su un database MS SQL Server) e selezionare Utilizza Valore come Testo Letterale. Questo assicura che la GETDATE() venga inclusa nell'istruzione SQL senza apici.
