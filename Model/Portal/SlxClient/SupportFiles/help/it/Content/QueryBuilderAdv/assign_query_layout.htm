Assegnare il Layout di una Query

Utilizzare la finestra di dialogo Assegna il Layout di una Query per specificare l'aspetto di ciascuna colonna di dati (o campo) nella query.

Per impostare le opzioni layout

  1. Aprire il Query Builder e fare clic sulla scheda Layout.
  2. Aggiungere o modificare un campo layout.
  3. Nel campo Didascalia, digitare il nome da visualizzare nell'intestazione di colonna.
  4. Nella casella Larghezza selezionare la dimensione della larghezza colonna. Il valore predefinito è 100.

Alcune modifiche alla larghezza gruppo effettuate rimangono attive per la sessione corrente di connessione.

  1. Nella casella Allineam. Didascalia selezionare l'allineamento dell'intestazione di colonna.
  2. Nella casella Allineam. Testo selezionare l'allineamento dei dati.
  3. Nel campo Tipo Formato selezionare il formato appropriato per il tipo di dati visualizzato in questo campo.

    Il tipo formato Telefono formatta il valore del campo come un numero di telefono standard. Se la propria società non utilizza il formato standard USA a 10 caratteri, si dovrebbe prendere in considerazione l'ipotesi di impostare il tipo di formato per un numero telefonico a Nessuno per far in modo che non venga applicata alcun tipo di maschera telefonica.

  4. Se necessario, digitare un formato stringa nel campo Formato per modificare il Tipo Formato Predefinito selezionato.
  5. Per collegare l'informazione nella colonna ad un'entità della vista portale, selezionare Collegato a Entità.
  6. Per applicare la formattazione da un foglio di stile CSS, inserire la classe che si desidera applicare nel campo Classe Css.
