Visualizzare una Join Globale

In Infor CRM, molte join globali sono già impostate per comodità. Comunque, non tutte sono visibili alla apertura del Query Builder per limiti di spazio. Quando si utilizza del Query Builder per selezionare dati, è preferibile utilizzare, quando possibile le join già esistenti, piuttosto che crearne di nuove.

Per aggiungere una join locale

  1. Aprire il Query Builder.
  2. Se necessario, utilizzare i segni più per espandere l'elenco delle tabelle nella parte superiore sinistra del Query Builder. Selezionare una tabella.
  3. Fare clic su Join Globale.

Compare il Gestore Global Join, che visualizza le join globali esistenti.

Se una join globale non è disponibile, è possibile aggiungere una join locale.
