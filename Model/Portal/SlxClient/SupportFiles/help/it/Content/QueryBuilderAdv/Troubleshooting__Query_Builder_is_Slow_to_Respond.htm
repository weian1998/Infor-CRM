Risoluzione dei Problemi: Query Builder Impiega molto tempo per Rispondere

 Solo per l'amministratore di sistema

Se il Query Builder impiega molto tempo per rispondere, potrebbero esistere join ricorsive che è possibile rimuovere. Un esempio di join ricorsiva è:

Join 1: Azienda->Azienda_Ext

Join 2: Azienda_Ext->Azienda

Per trovare le join ricorsive

  • Eseguire la seguente istruzione:

    SELECT j1.tabellada tabellada1, j1.campoda campoda1, j1.tabellaa tabellaa1, j1.campoa campoa1

    FROM sysdba.JOINDATA j1, sysdba.joindata j2 where j1.fromtable = j2.totable

    AND j1.tabellaa = j2.tabellada

    AND j1.campoda = j2.campoa

    AND j1.campoa = j2.campoda

Non rimuovere le seguenti join di sistema:

  • Opportunity_Salesstep.Salesstepid -> Salesstep.Salesstepid
  • Salesstep.Salesstepid -> Opportunity_Salesstep.Salesstepid

Per ulteriori informazioni sull'esecuzione di istruzioni SQL, consultare la Guida di Administrator.
