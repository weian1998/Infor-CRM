Cambiare il Tipo di un Utente

E' possibile utilizzare la vista Dettaglio Utente per convertire il tipo di utente. Si può, ad esempio, cambiare un utente Web Only in utente Concorrente. Se lo si desidera è possibile inoltre ritirare un utente oppure riattivare un utente ritirato.

E' indispensabile disporre di una licenza adatta per il nuovo tipo di utente, tranne quando si assegna il tipo Ritirato. Il ritiro di un utente rilascia una licenza del tipo in precedenza assegnato a quell'utente.

Regole

Prima di cambiare un utente in Ritirato, effettuare le seguenti operazioni:

  • Se l'utente è proprietario di almeno un record, effettuare il riallineamento territoriale in Administrator.
  • Se l'utente è un responsabile di una o più aziende, effettuare un riallineamento territoriale in Administrator e assegnare un nuovo responsabile di azienda.
  • Se l'utente è un responsabile, riassegnare ciascun utente che fa capo a lui al nuovo responsabile oppure all'amministratore di sistema.

Prima di modificare il tipo di utente in Web Only, assicurarsi di aver effettuato le seguenti operazioni:

  • Se l'utente è un responsabile, disattivare questa impostazione nella vista Dettaglio Utente.
  • Se altri sono elencati come parte del Team Utente dell'utente, rimuoverli.

Per modificare

  1. Aprire la vista Dettaglio Utente dell'utente da convertire.
  2. Nel campo Tipo Utente, selezionare un nuovo tipo di utente.
  3. Fare clic su Salva..
