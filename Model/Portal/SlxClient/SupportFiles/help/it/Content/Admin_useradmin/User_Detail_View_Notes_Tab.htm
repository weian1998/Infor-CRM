Scheda Note

La Scheda Note della vista Dettaglio Utente può essere utilizzata per aggiungere annotazioni riguardanti l'utente. Consente di aggiungere informazioni che non possono essere inserite in altre schede.

Per gestire

  1. Dalla vista Dettaglio Utente, fare clic sulla scheda Note.
  2. Per aggiungere un timbro con data e ora clicare su .
  3. Digitare le annotazioni sull'utente.
  4. Fare clic su Salva..
