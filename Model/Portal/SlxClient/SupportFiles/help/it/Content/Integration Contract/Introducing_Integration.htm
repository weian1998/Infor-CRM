L'Integrazione Contabilità

L'Integrazione Contabilità consente la condivisione di informazioni tra Infor e qualsiasi sistema di contabilità che supporti il Contratto SData CRM ERP. InforGli utenti del Client Web possono visualizzare ordini di vendita, fatture e pagamenti dei clienti, fatture aperte e qualsiasi dettaglio corrispondente. Gli utenti potrebbero anche creare ordini di vendita nel sistema di contabilità integrato all'interno di Infor. Questi ordini di vendita non sono integrati con Infor ordini di vendita.

Infor funzionalità di sincronizzazione delle informazioni tra Infor e le applicazioni di contabilità supportate. Tuttavia, in base al proprio sistema di contabilità, alcune di queste informazioni potrebbero non essere disponibili. Ciascun sistema di contabilità determina i tipi di record che saranno sincronizzati con Infor. In base alla propria integrazione è possibile sincronizzare una parte o tutte le seguenti informazioni:

  • Aziende/Imprese Commerciali
  • Indirizzi
  • Contatti

Comprendere L'Integrazione Contabilità e i Remoti

Alcune funzionalità dell'Integrazione Contabilità non sono disponibili per gli utenti remoti e per gli utenti di Uffici Remoti. Gli utenti remoti che si connettono al sito Web Host hanno accesso a tutte le funzionalità.

  • Gli utenti Client Web Offline non possono:
    • promuovere aziende.
    • visualizzare informazioni dal sistema di contabilità, mostrate in tempo reale in una vista di dati (ad esempio, Fatture e Pagamenti). Questa funzione è disponibile se l'utente Client Web Offline è connesso.
  • Gli utenti di Ufficio Remoto non possono
    • promuovere aziende.
