Gestione Schema

Utilizzare il dialogo Gestione Schema per confrontare lo schema del database in Infor con il proprio schema di integrazione per garantire l'integrità dei dati. E' possibile confrontare i dati delle varie risorse per determinare se esistano, tra i sistemi, delle differenze nelle lunghezze di stringa.

Per saperne di più sul modo in cui Infor gestisce le differenze di schema...

MaxLength - se i dati sincronizzati in Infor superano il valore MaxLength di un campo, l'intera riga non sarà accettata.

FractionalDigits - se i dati sincronizzati in Infor superano il numero di cifre dopo la virgola decimale, il valore sarà arrotondato al numero massimo di cifre dopo la virgola. Ad esempio se un campo valuta è definito come 17,4, un valore di 123,12348 sarà salvato come 123,1235 e un valore di 123,12343 sarà salvato come 123,1234.

Come fare a?

Aprire il dialogo Gestione Schema
  • Sulla vista Dettaglio Integrazione scheda Risorse, cliccare il pulsante Gestione Schema.
Confrontare lo schema per le risorse
  1. Nella casella Gestione SchemaFiltrare per Risorsa, selezionare il tipo di risorsa.
  2. Fare clic su Esegui.
  3. Esaminare le differenze nella griglia.
    Annotare qualsiasi informazione che possa causare problemi di integrità dei dati. Ad esempio, un campo del sistema di contabilità può accettare soltanto tre caratteri quando invece lo stesso campo in Infor CRM può accettare un qualsiasi testo.
  4. Ripetere per tutti i tipi di risorsa.
  5. Fare clic su Chiudi.
