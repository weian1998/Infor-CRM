Definire un Delimitatore

Utilizzare questa vista per selezionare il valore di delimitatore appropriato per il proprio elenco e per l'anteprima dei record prima di importarli in Infor CRM. Un delimitatore analizza i campi in un elenco. Infor CRM analizza l'elenco per determinare quale delimitatore è attualmente utilizzato, ma è possibile selezionare un delimitatore diverso, se necessario.

Per definire un delimitatore e un'anteprima

  1. Quando si importano i nominativi fare clic su Avanti dopo aver selezionato un file.
  2. Se si ha necessità di cambiare il delimitatore, fare clic su una delle opzioni seguenti:
    • Scheda
    • Punto e virgola
    • Virgola
    • Spazio
    • Altro
  3. Se si seleziona Altro bisogna specificare il delimitatore da usare per separare i campi dell'elenco. Digitare il carattere di delimitazione nel campo Altro.
  4. Nell'elenco Qualificatore di Testo fare clic sulla freccia e selezionare un qualificatore.
    Il qualificatore identifica il testo che va importato così come appare.
  5. Se l'elenco contiene nomi di campo, spuntare la casella La Prima Riga contiene Nomi di Campo per visualizzare i nomi nell'area di Anteprima.

    Se sono presenti caratteri estesi nel file importato che non sono visualizzati correttamente nell'area Anteprima, è necessario salvare nuovamente il file stabilendo esplicitamente il tipo di codifica (UTF-8, Unicode o ANSI). Una volta salvato il file fare clic sul pulsante Indietro per tornare al passaggio Seleziona un File e riselezionare il file aggiornato prima di proseguire nell'importazione.

  6. Fare clic su Avanti per continuare con il passaggio successivo.
