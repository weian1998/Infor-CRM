Trascinamento

La funzionalità di trascinamento consente di riorganizzare parti dell'interfaccia del Infor CRM. Se è stato installato il desktop Integration è possibile anche aggiungere allegati o messaggi email di Microsoft Outlook al Infor.

Come fare a?

Riorganizzare gli elementi dell'interfaccia
  1. Muovere il mouse sull'oggetto da spostare. Se la scheda può essere spostata il puntatore del mouse cambierà aspetto.
  2. Tenere premuto il pulsante del mouse sull'elemento e trascinarlo sulla posizione desiderata.

Ad esempio in un'altra posizione di una griglia, sulla barra delle schede, nell'area Altre Schede oppure sulla sezione centrale definita dall'utente.

  1. Rilasciare il pulsante del mouse.
Allegare un file usando il trascinamento

  • Se si utilizza Internet Explorer, questa funzionalità richiede l'installazione del Modulo di Integrazione Desktop.
  • Il trascinamento dei file non è disponibile nella cronologia o nelle schede Allegati dell'attività.
  1. Cercare fra le cartelle il file da aggiungere.
  2. Trascinare il file e rilasciarlo nell'apposita vista di dettaglio del record.

Si apre la finestra di dialogo Aggiungi Allegati.

    Se si viene avvisati di installare il Desktop Integration Infor CRM e si desidera utilizzare questa funzione, fare clic su Sì e seguire i passi per installare Infor CRM Desktop Integration. Per completare l'installazione si verrà avvisati di chiudere tutti i browser di windows e Microsoft Outlook.
  1. Nel campo Descrizione inserire una descrizione per il file se la descrizione predefinita non è chiara.

  2. Fare clic su OK.

Registrare la posta elettronica nella Cronologia
  1. Aprire la vista di dettaglio del contatto, azienda, nominativo, opportunità o ticket appropriati.
  2. In Microsoft outlook selezionare il messaggio email che si desidera aggiungere a Infor.
  3. Fare clic sul messaggio e tenendo premuto il pulsante del mouse trascinare l'elemento nella vista di dettaglio aperta nel Infor.
    A seconda delle opzioni, è possibile che venga richiesto di inserire informazioni di cronologia.
