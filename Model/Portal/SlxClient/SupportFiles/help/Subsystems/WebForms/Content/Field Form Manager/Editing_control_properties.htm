Editing Control Properties

The Form Designer shows many of the property values for the currently selected control and allows you to edit them.

To edit

  1. Open a form in the Form Designer.
  2. Click a control in the layout.
    The properties of the selected control display on the tabs below the control layout pane.
    Additional control properties may be accessible with the Application Architect.
  3. Click a tab and edit the settings for the control you are configuring. Click a link below to see more information about the properties of a control:
