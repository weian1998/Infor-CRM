Bibliotheksdokumenteigenschaften

Im Dialogfeld "Dokumenteigenschaften" können Sie den Dateien in der Bibliothek neue Eigenschaften hinzufügen oder die vorhandenen Eigenschaften ändern. Zusätzlich zu den bearbeitbaren Feldern bietet dieses Dialogfeld die folgenden schreibgeschützten Informationen:

  • Verzeichnis: Der Speicherort, an dem das Dokument gespeichert ist.
  • Status: Gilt nur für Remote-Benutzer. Der Status kann folgende Werte annehmen: Verfügbar, Angefordert, Zugesendet, Nicht gelesen, Gelesen oder Neue Version.
  • Erstellt: Das Datum, an dem die Datei der Bibliothek hinzugefügt wurde.
  • Größe: Die Größe der Datei.

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

So zeigen Sie Bibliotheksdokumenteigenschaften an bzw. legen sie fest

  1. Klicken Sie auf der Navigationsleiste auf Bibliothek.
  2. Wählen Sie in der Liste das Bibliotheksdokument aus, das Sie bearbeiten möchten, und klicken Sie auf die Schaltfläche Dateieigenschaften.
  3. Ändern Sie im Feld Neue Version das Datum und die Uhrzeit der letzten Bearbeitung der Datei, sodass sie bei der Synchronisierung an die Remote-Clients gesendet wird.
    Dieses Feld wird nicht automatisch aktualisiert.
  4. Deaktivieren Sie das Kontrollkästchen Gültig belassen , um das Kontrollkästchen Läuft ab zu aktivieren.
    Dieses Kontrollkästchen ist standardmäßig aktiviert.
  5. Legen Sie im Textfeld Läuft ab ein Validierungsdatum fest. Diese Option kann für zeitabhängige Dokumente wie Preislisten verwendet werden.
  6. Im Textfeld Beschreibung können Sie einen bis zu 255 Zeichen umfassenden Kommentar zur Datei eingeben.
    Die Dateibeschreibung wird in der Detailansicht der Bibliothek angezeigt.
  7. Im Textfeld Zusammenfassung können Sie einen bis zu 1024 Zeichen umfassenden Kommentar zur Datei eingeben.
    Die Zusammenfassung kann nur in diesem Dialogfeld angezeigt werden.
  8. Aktivieren Sie die Option Verteilung der Datei erzwingen, um Dateien und Dateiänderungen automatisch an alle Remote-Benutzer zu senden.
  9. Klicken Sie auf OK.
