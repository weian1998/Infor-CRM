Hinzufügen, Umbenennen oder Löschen eines Bibliotheksordners

Zur besseren Organisation können Sie die Bibliotheksordner zum Gruppieren von Dokumenten nutzen.

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Vorgehensweise

Hinzufügen eines Bibliotheksordners
  1. Klicken Sie auf der Navigationsleiste auf Bibliothek.
  2. Wählen Sie den Ordner aus, dem Sie das Element hinzufügen möchten, und klicken Sie auf Ordner hinzufügen.
  3. Geben Sie in das Dialogfeld Neuer Ordner einen Namen für den neuen Ordner ein.

  4. Klicken Sie auf OK.

Umbenennen eines Ordners
  1. Klicken Sie auf der Navigationsleiste auf Bibliothek.
  2. Wählen Sie den Ordner aus, den Sie umbenennen möchten, und klicken Sie auf Ordnername bearbeiten.
  3. Geben Sie in das Dialogfeld einen neuen Namen für den Ordner ein.

  4. Klicken Sie auf OK.

Löschen eines Ordners
  1. Klicken Sie auf der Navigationsleiste auf Bibliothek.
  2. Wählen Sie den zu löschenden Ordner aus, und klicken Sie auf Ordner löschen.
  3. Klicken Sie auf Ja, um den Löschvorgang zu bestätigen.

Alle Ordner und Dateien im Ordner werden ebenfalls gelöscht.
