Hinzufügen eines Vertrags

In der Ansicht "Neuen Vertrag einfügen" können Sie einen Vertragsdatensatz erstellen. Nutzen Sie die Schaltfläche Kopieren, um schnell einen neuen Datensatz mit ähnlichen Informationen zu erstellen.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

So fügen Sie einen Vertrag hinzu

  1. Öffnen Sie die Ansicht "Neuen Vertrag einfügen".

    • Klicken Sie im Menü Neu auf Vertrag.
    • Klicken Sie in der Detailansicht Firma zuerst auf die Registerkarte Verträge und anschließend auf Vertrag erstellen.
  2. Füllen Sie die Felder für Vertragsinformationen aus.
  3. Klicken Sie auf Speichern.
  4. Klicken Sie, falls erforderlich, auf die Registerkarte Abgedeckte verkaufte Artikel, um verkaufte Artikel mit dem Vertrag zu verknüpfen.
