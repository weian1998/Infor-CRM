Deaktivieren eines Vertrags

Sie können einen Vertrag deaktivieren, wenn keine Tickets mit ihm verknüpft sind. Inaktive Verträge werden weiterhin im Dialog "Firmenserviceinformationen" angezeigt.

So deaktivieren Sie einen Vertrag

  1. Öffnen Sie die Vertragsdetailansicht.
  2. Klicken Sie auf die Registerkarte Tickets.
  3. Stellen Sie sicher, dass keine mit dem Vertrag verknüpften Tickets mehr vorhanden sind.
  4. Deaktivieren Sie im oberen Teil der Detailansicht Vertrag das Kontrollkästchen Aktiv.
  5. Klicken Sie auf Speichern.
