Vertragsreferenznummer - Duplikat

Vertragsreferenznummern müssen eindeutig sein. Ist eine Referenznummer bereits vorhanden, wird das Dialogfeld "Vertragsreferenznr. - Duplikat" geöffnet.

Um den durch das Duplikat entstehenden Konflikt zu lösen, führen Sie einen der folgenden Schritte durch:

  • Referenznummer des Vertrags erneut eingeben: Wählen Sie diese Option aus und klicken Sie dann auf OK. Das Meldungsfeld wird geschlossen, und der Cursor kehrt zum Feld Referenznummer des Dialogs "Vertrag hinzufügen" zurück. Sie können anschließend eine neue Referenznummer eingeben.
  • Eine doppelte Vertragsreferenznummer erstellen - Wenn Sie diese Option wählen und auf OK klicken, wird die Referenznummer dupliziert, und Sie können das Hinzufügen neuer Vertragsinformationen fortsetzen.
