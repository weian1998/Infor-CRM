Exportieren einer Gruppe nach Excel

Sie können Gruppen nach Excel exportieren. Die Überschriften und die Formatierung der Gruppenspalten werden ebenfalls exportiert.

So exportieren Sie eine Gruppe in eine Datei

  1. Zeigen Sie die Gruppe oder Suchergebnisse an, die die zu exportierenden Datensätze enthalten.
  2. Klicken Sie mit der rechten Maustaste auf die Gruppenregisterkarte und anschließend auf In Excel exportieren.
  3. Öffnen Sie die Gruppe nach Beendigung des Exports von einem der folgenden Speicherorte:
  • Seite "Mitteilungen"
  • Registerkarte Berichtslistenansicht "Historie"
  • Meldungsfeld "Auftrag abgeschlossen"
    Die exportierte Gruppe wird in Microsoft Excel geöffnet. Wenn die Gruppe Währungsinformationen enthält, wird automatisch eine Excel Pivot-Tabelle erstellt. Zu jeder Spalte wird eine Summenzeile mit Währungsinformationen hinzugefügt, damit Sie die Gruppengesamtwerte sehen können.

Eventuell erhalten Sie eine Fehlermeldung, wenn Sie versuchen, eine Gruppe zu exportieren, die ein Datenfeld mit einer großen Datenmenge enthält.
