Festlegen einer Standardgruppe

Sie können eine Standardgruppe für jede Listenansicht festlegen. Die Standardgruppe wird als erste Gruppe angezeigt, wenn Sie die Listenansicht öffnen. Zudem bestimmt die Standardgruppe auch das Layout neuer Gruppen. Pro Listenansicht kann nur eine Standardgruppe festgelegt werden.

So legen Sie den Standard fest

  • Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie in einer Listenansicht mit der rechten Maustaste auf die Gruppenregisterkarte, die Sie zur Standardgruppe machen möchten, und klicken Sie auf Als Standardgruppe festlegen.
    • Klicken Sie im Menü Extras auf Optionen, und klicken Sie dann auf die Registerkarte Gruppen. Verwenden Sie die Listen Hauptansicht und Standardgruppe, um die Standardgruppe für die einzelnen Hauptansichten festzulegen.
