Anzeigen von ausgeblendeten Gruppen

Sie können eine Gruppe anzeigen, die ausgeblendet wurde.

So zeigen Sie eine ausgeblendete Gruppe an

  1. Klicken Sie in der geeigneten Listenansicht auf das Menü Gruppe.
  2. Stellen Sie sicher, dass die Option Ausgeblendete einblenden ausgewählt ist.
  3. Aktivieren Sie die Kontrollkästchen Sichtbar der Gruppen, die eingeblendet werden sollen.
