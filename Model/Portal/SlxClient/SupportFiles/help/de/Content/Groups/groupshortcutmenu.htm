Verwenden des Kontextmenüs für Gruppen

Mit dem Kontextmenü für Gruppen können Sie Gruppen erstellen und verwalten.

Nicht alle Menüoptionen sind in allen Listenansichten verfügbar.

So öffnen Sie das Kontextmenü für Gruppen

  1. Klicken Sie in einer Listen- oder Detailansicht mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Treffen Sie Ihre Auswahl unter den folgenden Optionen:
    • Kopieren
    • Bearbeiter
    • Ausblenden
    • Löschen
    • Freigeben
    • In Excel exportieren
    • In Datei exportieren
    • Als Standard festlegen
    • Layout zurücksetzen

    Macht Anpassungen an der Gruppe rückgängig und setzt die Darstellung der Gruppe auf das Layout zurück, das von der Registerkarte "Layout" im Abfrage-Assistenten bestimmt wird.

    • Gruppe hinzufügen

So öffnen Sie das Kontextmenü "Suchergebnisse"

  1. Wenn Sie eine Suche durchgeführt haben, klicken Sie mit der rechten Maustaste auf die Registerkarte Suchergebnisse.
  2. Treffen Sie Ihre Auswahl unter den folgenden Optionen:
    • Suche als Gruppe speichern
    • In Excel exportieren
    • In Datei exportieren
    • Layout zurücksetzen

    Macht Anpassungen an der Gruppe rückgängig und setzt die Darstellung der Gruppe auf das Layout zurück, das von der Registerkarte "Layout" im Abfrage-Assistenten bestimmt wird.

    • Gruppe hinzufügen

So öffnen Sie das Kontextmenü für die Gruppenlistenansicht

  1. Wählen Sie einen oder mehrere Datensätze aus.
  2. Klicken Sie mit der rechten Maustaste in die Gruppenlistenansicht.
  3. Treffen Sie Ihre Auswahl unter den folgenden Optionen:
    • Datensätze als Gruppe speichern
    • Aus Gruppe entfernen
    • Zu vorhandener Gruppe hinzufügen

    • Interessenten aktualisieren (nur Interessentenlistenansicht)

    Ermöglicht das Zuordnen eines Zugriffsberechtigten zu ausgewählten Datensätzen.

    • Interessenten löschen (nur Interessentenlistenansicht)
    • Verkaufschancen aktualisieren (nur Verkaufschancen-Listenansicht)
    • Verkaufschancenstatistiken (Verkaufschancen-Listenansicht)
    • Datensätze zusammenführen (nur Kontakt- und Firmenlistenansichten)
    • E-Mail (nur Kontakt- und Interessentenlistenansichten)
