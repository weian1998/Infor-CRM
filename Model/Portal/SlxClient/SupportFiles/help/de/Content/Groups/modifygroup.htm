Ändern von Gruppen

Mit dem Abfrage-Assistenten können Sie Gruppen ändern.

So ändern Sie eine Gruppe

  1. Öffnen Sie eine Listen- oder Detailansicht. 
  2. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  3. Klicken Sie auf Bearbeiten.

Der Abfrage-Assistent wird geöffnet.

  1. Ändern Sie die Gruppe mithilfe der Registerkarten des Abfrage-Assistenten.
  • Eigenschaften – Auf dieser Registerkarte legen Sie den Namen und die Beschreibung der Gruppe fest.
  • Bedingungen - Auf dieser Registerkarte definieren Sie die für die ausgewählten Datensätze zu verwendenden Bedingungsanweisungen. Diese Registerkarte ist für Ad-hoc-Gruppen nicht verfügbar.
  • Layout – Auf dieser Registerkarte legen Sie fest, welche Daten in der Listenansicht angezeigt werden.
  • Sortieren – Auf dieser Registerkarte geben Sie an, wie die Datensätze sortiert werden.
  1. Wenn Sie die gewünschten Änderungen vorgenommen haben, klicken Sie auf OK.
