Kopieren einer Gruppe

Sie können eine vorhandene Gruppe markieren und kopieren. Sie können die vorhandenen Bedingungen für die Gruppe beibehalten oder ändern, um diese Ihren Wünschen anzupassen. Ad-hoc-Gruppen können Sie nicht kopieren.

So kopieren Sie eine vorhandene Gruppe

  1. Klicken Sie in einer Listenansicht mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Kopieren. 
    Der Abfrage-Assistent wird geöffnet.
  3. Ändern Sie die Gruppe mithilfe der Registerkarten des Abfrage-Assistenten.
    • Eigenschaften – Auf dieser Registerkarte legen Sie den Namen und die Beschreibung der Gruppe fest.
    • Bedingungen – Auf dieser diese Registerkarte legen Sie die Auswahlbedingung(en) für die Datenauswahl fest. Diese Registerkarte ist für Ad-hoc-Gruppen nicht verfügbar.
    • Layout - Auf dieser Registerkarte legen Sie fest, welche Daten in der Listenansicht angezeigt werden.

      Das Standardlayout wird von der Gruppe bestimmt, die Sie in den Gruppenoptionen als Standardgruppe ausgewählt haben.

    • Sortieren - Verwenden Sie diese Registerkarte, um die Sortierung der Daten festzulegen.
  4. Wenn Sie die gewünschten Änderungen vorgenommen haben, klicken Sie auf OK.
