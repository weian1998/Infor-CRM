Hinzufügen von Gruppen (Erweiterte Abfrage)

Erstellen Sie mit dem Abfrage-Assistenten eine Gruppe. Mit erweiterten Abfragen können Sie die Bedingungen für die Gruppe festlegen. Bei jedem Öffnen der Gruppe wertet die Abfrage die Datenbank aus, um zu ermitteln, welche Datensätze angezeigt werden sollen. Wenn Sie sicherstellen möchten, dass sich die Gruppe nicht ändert, erstellen Sie stattdessen eine Ad-hoc-Gruppe.

 

So erstellen Sie eine Gruppe (Erweiterte Abfrage)

  1. Öffnen Sie eine Listen- oder Detailansicht.
  2. Klicken Sie auf die Schaltfläche Gruppe hinzufügen.
    Der Infor CRM Abfrage-Assistent wird geöffnet.
  3. Verwenden Sie die Registerkarte Bedingungen, um die Bedingungsanweisungen für die Datensatzauswahl zu definieren.
  4. Verwenden Sie bei Bedarf die folgenden Registerkarten des Abfrage-Assistenten:
    • Eigenschaften – auf diesen Registerkarten legen Sie Ihren Namen und die Beschreibung der Gruppe fest.
    • Layout - Auf dieser Registerkarte legen Sie fest, welche Daten in der Listenansicht angezeigt werden.

      Das Standardlayout wird von der Gruppe bestimmt, die Sie in den Gruppenoptionen als Standardgruppe ausgewählt haben.

    • Sortieren - Verwenden Sie diese Registerkarte, um die Sortierung der Daten festzulegen.
  5. Wenn Sie die gewünschten Änderungen vorgenommen haben, klicken Sie auf OK.
