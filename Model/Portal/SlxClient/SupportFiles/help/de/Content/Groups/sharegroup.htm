Freigeben von Gruppen für andere Benutzer

Sie können Ihre Gruppen für andere Infor CRM-Benutzer, -Teams oder -Abteilungen freigeben.

  • Wenn Sie eine Gruppe für andere Benutzer freigeben und anschließend die Gruppe löschen, ist die Gruppe nicht mehr für die anderen Benutzer verfügbar.
  • Sie können keine Gruppen freigeben, die ein anderer Benutzer für Sie freigegeben hat.
  • Wenn Sie Änderungen an einer für andere Benutzer freigegebenen Gruppe vornehmen, müssen Sie die Gruppe aktualisieren, damit die anderen Benutzer die neue Version erhalten.
  • Sie können Zugriffsinformationen für vorhandene Gruppen anzeigen, um den Zugriffsberechtigten und die Benutzer zu ermitteln, die die Gruppe anzeigen können.
  • Benutzer können nur die Firmen, Kontakte, Verkaufschancen und Tickets anzeigen, für die sie Zugriffsrechte haben.
  • Wenn eine Ad-hoc-Gruppe für andere Benutzer freigegeben ist, können diese Benutzer Gruppendatensätze hinzufügen und entfernen.
  • Wenn Sie eine Registerkarte "Cockpit" freigeben, die eine Gruppe enthält, muss die Gruppe ebenfalls für diese Benutzer freigegeben werden, damit die Gruppe auf der Registerkarte "Cockpit" angezeigt wird.
Eine Gruppe gemeinsam mit anderen Benutzern verwenden
  1. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Freigeben.
     Wenn Sie nicht zum Freigeben der Gruppe berechtigt sind, ist diese Option nicht verfügbar.
  3. Klicken Sie auf die Schaltfläche Hinzufügen.
  4. Klicken Sie im Dialogfeld Zugriffsberechtigten zuweisen auf die Registerkarte Benutzer.
  5. Wenn Sie die aufgelisteten Benutzer begrenzen möchten, wählen Sie aus der Dropdown-Liste Benutzertyp einen Benutzertyp aus (z. B. "Remote").
  6. Wählen Sie die Benutzer aus, für die Sie Ihre Gruppe freigeben möchten, und klicken Sie auf OK.
So geben Sie eine Gruppe für andere Teams frei
  1. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Freigeben.
     Wenn Sie nicht zum Freigeben der Gruppe berechtigt sind, ist diese Option nicht verfügbar.
  3. Klicken Sie auf die Schaltfläche Hinzufügen.
  4. Klicken Sie im Dialog Zugriffsberechtigten zuweisen auf die Registerkarte Teams.
So geben Sie Gruppen für andere Abteilungen frei
  1. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Freigeben.
     Wenn Sie nicht zum Freigeben der Gruppe berechtigt sind, ist diese Option nicht verfügbar.
  3. Klicken Sie auf die Schaltfläche Hinzufügen.
  4. Klicken Sie im Dialog Zugriffsberechtigten zuweisen auf die Registerkarte Abteilungen.
  5. Wählen Sie die Abteilungen aus, für die Sie Ihre Gruppe freigeben möchten, und klicken Sie auf OK.
So entfernen Sie den Zugriff auf eine freigegebene Gruppe
  1. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Freigeben.
     Wenn Sie nicht zum Freigeben der Gruppe berechtigt sind, ist diese Option nicht verfügbar.
  3. Wählen Sie Benutzer, Abteilungen oder Teams aus, für die Ihre Gruppe nicht mehr freigegeben sein soll.
  4. Klicken Sie auf die Schaltfläche Entfernen und danach auf OK.
So aktualisieren Sie eine freigegebene Gruppe

Wenn Sie Änderungen an einer Gruppe vornehmen und diese Änderungen freigeben möchten, müssen Sie die Gruppe aktualisieren.

  1. Klicken Sie mit der rechten Maustaste auf eine Gruppenregisterkarte.
  2. Klicken Sie auf Freigeben.
     Wenn Sie nicht zum Freigeben der Gruppe berechtigt sind, ist diese Option nicht verfügbar.
  3. Klicken Sie im Dialogfeld Gruppe freigeben auf OK, um die Änderungen für die aufgelisteten Benutzer freizugeben.
