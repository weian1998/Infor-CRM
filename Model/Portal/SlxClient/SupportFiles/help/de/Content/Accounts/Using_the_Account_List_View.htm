Verwenden der Firmenlistenansicht

In der Firmenlistenansicht werden alle Firmen angezeigt, auf die Sie Zugriff haben.  Mithilfe von Gruppen und Suchen können Sie die Liste der angezeigten Datensätze begrenzen..

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

Vorgehensweise

Öffnen der Firmenlistenansicht
  • Klicken Sie auf der Navigationsleiste auf Firmen.

Verwenden von Datensätzen in einem Raster

Verwenden der Ansicht "Zusammenfassung"

Aktualisieren von Listeninformationen
  • Klicken Sie auf die Schaltfläche Aktualisieren.

Meine Liste filtern

 Verwenden Sie die Suchfunktion, um nach Datensätzen zu suchen. 

Öffnen der Firmendetailansicht
  • Klicken Sie auf den Namen der Firma, um die Ansicht Firmendetail zu öffnen.

Anzeigen einer Gruppe

 

Verwenden des Kontextmenüs für Gruppen

Allgemeine Aufgaben

Im Aufgabenbereich unter Allgemeine Aufgaben können Sie Folgendes ausführen:

  • Zu Gruppe hinzufügen
  • Aus Gruppe entfernen
  • Datensätze als Gruppe speichern
  • In Datei exportieren
  • In Cockpit hochstufen
