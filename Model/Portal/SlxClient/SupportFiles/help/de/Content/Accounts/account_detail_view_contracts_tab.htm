Firmendetailansicht - Registerkarte "Verträge"

Auf der Registerkarte "Verträge" können Sie bestimmte Vertragsinformationen zur Firma zentral anzeigen. Diese Registerkarte dient zum Überprüfen von Vertragsinformationen und nicht zum Öffnen einzelner Verträge.

Zusätzlich zur Anzeige von Vertragsinformationen können Sie auch einen Vertragsdatensatz öffnen, um Änderungen vorzunehmen oder einen neuen Vertrag für die Firma hinzufügen.

So verwenden Sie die Registerkarte "Verträge"

  1. Öffnen Sie die Firmendetailansicht. 
  2. Klicken Sie auf die Registerkarte Verträge. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  3. Führen Sie einen der folgenden Schritte durch:
    • Wenn Sie Anmerkungen zu dem Vertrag anzeigen möchten, wählen Sie den als Vorschau anzuzeigenden Vertrag aus.
    • Um den Vertrag zu öffnen, klicken Sie auf die Referenznummer des gewünschten Vertrags. 
      Die Vertragsdetailansicht wird geöffnet.
    • Zum Erstellen eines neuen Vertrags klicken Sie auf Vertrag erstellen.
