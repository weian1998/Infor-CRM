Anzeigen von SpeedSearch-Ergebnissen

Auf der Registerkarte "Ergebnisse" wird eine Liste der Einträge, die Ihren Suchkriterien entsprechen, angezeigt. Wenn Sie einen Eintrag auswählen, wird dieser in der Registerkarte "Vorschau" angezeigt. Wenn Sie auf der Registerkarte "Ergebnisse" auf einen Hyperlink klicken, gelangen Sie entweder zur Detailansicht des Datensatzes oder in das Dialogfeld mit Aktivitäts- oder Historiendetails.

Wenn mehr als nur eine Seite mit Suchergebnissen angezeigt wird, klicken Sie auf die Schaltflächen Vorherige oder Nächste, um weitere Ergebnisse anzuzeigen.

So führen Sie eine Suche aus und fügen eine Lösung in das aktive Ticket ein

  1. Klicken Sie in der Detailansicht Ticket auf der Registerkarte Details auf die Schaltfläche SpeedSearch.
  2. Führen Sie eine Suche aus.
  3. Wählen Sie auf der Registerkarte Ergebnisse den Eintrag aus, der die Lösung enthält, die Sie verwenden möchten.
  4. Wenn Sie nur einen Teil der Lösung einfügen möchten, wählen Sie den entsprechenden Text im Bereich Vorschau aus. Anderenfalls wird die gesamte Lösung in das aktive Ticket kopiert.
  5. Klicken Sie auf die Schaltfläche Ausgewählten Text kopieren und in den Datensatz einfügen.
    SpeedSearch wird geschlossen und Sie kehren zu dem aktiven Ticket zurück. Die Lösung wurde in das Feld Lösung eingefügt.
