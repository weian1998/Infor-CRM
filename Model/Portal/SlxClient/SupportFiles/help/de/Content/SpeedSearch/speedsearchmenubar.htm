Verwenden von SpeedSearch von der Menüleiste

Sie können jetzt von der Menüleiste aus Suchkriterien eingeben und SpeedSearch-Suchen durchführen.

So führen Sie eine SpeedSearch-Suche von der Menüleiste aus durch

  • Geben Sie über die Menüleiste Ihre Suchkriterien in das Feld SpeedSearch ein, und klicken Sie anschließend auf die Schaltfläche SpeedSearch.
     SpeedSearch wird mit den Suchergebnissen geöffnet oder ermöglicht Ihnen, erweiterte Optionen auszuwählen.
