Anzeigen von SpeedSearch-Suchergebnissen in der Vorschau

Nachdem Sie eine Suche durchgeführt und einen Eintrag ausgewählt haben, können Sie mit der Registerkarte "Vorschau" detaillierte Informationen zu diesem Eintrag anzeigen.  

So finden Sie schnell relevante Informationen

  1. Führen Sie eine SpeedSearch-Suche durch und klicken Sie auf die Schaltfläche Suchen.
  2. Klicken Sie bei Bedarf auf die Schaltflächen Erste Seite, Vorherige Seite, Nächste Seite und Letzte Seite, um die Ergebnisdatensätze anzuzeigen und nach dem Datensatz zu suchen, zu dem Sie die Vorschau anzeigen möchten.
  3. Klicken Sie auf Vorschau (roter Hyperlink unter dem Datensatz).

  4. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie auf Ersten Treffer suchen, um das erste Vorkommen eines Schlüsselwortes anzuzeigen.
    • Klicken Sie links und rechts des Schlüsselwortes auf die Schaltflächen Zurück bzw. Weiter, um jeweils die vorherige bzw. nächste Fundstelle des Schlüsselwortes anzuzeigen.
    • Verwenden Sie die innere Bildlaufleiste, um den Datensatz zu durchsuchen.

Wenn im Bereich "Vorschau" die Meldung "Zugriff verweigert" angezeigt wird, ist Ihnen der Zugriff auf das Ergebnis aus Sicherheitsgründen untersagt. Wenden Sie sich zwecks Zugriffsberechtigung an den Systemadministrator.

So kennzeichnen Sie ein Ergebnis als verwendet

Wenn Sie einen Eintrag als hilfreich erachten, die Ergebnisse aber nicht einfügen, gehen Sie wie folgt vor:

  • Wählen Sie den Eintrag aus und klicken Sie auf Diesen Eintrag als "hilfreich" kennzeichnen. Das verschafft dem System einen Überblick darüber, welche Einträge häufig verwendet werden.
