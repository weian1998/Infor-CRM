Kopieren einer Kampagne

Über den Dialog "Kampagne kopieren" können Sie mit den Informationen einer vorhandenen Kampagne eine neue Kampagne erstellen. Sie können Kampagneninformationen kopieren, z. B. Informationen zu Phasen, Aufgaben und Budget.

So kopieren Sie eine Kampagne

  1. Öffnen Sie die Kampagnendetailansicht für die Kampagne, die Sie kopieren möchten.
  2. Klicken Sie auf Kopieren.

Der Dialog "Kampagne einfügen" wird angezeigt und mehrere Felder sind bereits mit Standardinformationen ausgefüllt. Die Informationen zu Kampagnenzielen, -phasen und -aufgaben werden kopiert und auf den Registerkarten "Ziele" sowie "Phasen und Aufgaben" der Kampagnendetailansicht angezeigt, wenn Sie die Kampagne speichern.

  • Kampagnenname
  • Status
  • Anfangsdatum
  • Manager
  • Zugriffsberechtigter
  1. Ändern Sie die kopierten Informationen bei Bedarf und füllen Sie dann die verbleibenden Informationsfelder aus.
  2. Klicken Sie auf Speichern.
