Hinzufügen einer Kampagne

In der Ansicht "Kampagne einfügen" können Sie Kampagneninformationen hinzufügen. Nachdem Sie eine Kampagne hinzugefügt haben, können Sie Produkte, Phasen und Aufgaben sowie Ziele hinzufügen.

So fügen Sie eine Kampagne hinzu

  1. Klicken Sie im Menü Neu auf Kampagne.
    Das Dialogfeld "Kampagne einfügen" wird angezeigt.
  2. Füllen Sie die Kampagneninformationsfelder aus.
  3. Klicken Sie auf Speichern .  
      Verwenden Sie die Registerkarten in der Kampagnendetailansicht, um Produkte, Phasen, Aufgaben und Ziele hinzuzufügen.
