Registerkarte "Kampagnenziele"

Auf der Registerkarte "Kampagnenziele" werden alle Ziele für eine Kampagne angezeigt. Sie können kampagnenbezogene Aufgaben ausführen, beispielsweise die Zielliste verwalten, Zielinformationen aktualisieren, Ziele in Excel exportieren oder eine Gruppe erstellen. Diese Funktionen stehen nicht zur Verfügung, wenn Sie eine externe Liste mit Zielen verwenden, Sie können jedoch Informationen über die externe Liste angeben.

Vorgehensweise

Öffnen der Registerkarte Kampagnenziele
  • Klicken Sie in der Ansicht Kampagnendetail auf die Registerkarte Ziele. Werden keine Datensätze angezeigt, klicken Sie auf Aktualisieren.
Verwenden von Zielen aus einer externen Liste
  1. Aktivieren Sie das Kontrollkästchen Externe Liste verwenden.

Wenn Sie eine externe Liste verwenden, werden alle anderen verknüpften Ziele aus der Kampagne entfernt.

  1. Klicken Sie auf OK, um fortzufahren.
  2. Verwenden Sie die folgenden Felder, um Informationen über die Liste anzugeben:
  • Beschreibung: Geben Sie eine Beschreibung oder den Namen der Liste mit Zielen ein, die Sie für die Kampagne verwenden.
  • Speicherort: Geben Sie den Speicherort ein, an dem sich die Liste befindet. Zum Beispiel: C:\Eigene Dokumente\Kampagnenzielliste
  • Anzahl der Ziele: Geben Sie die Anzahl der Ziele ein.
  1. Klicken Sie auf Speichern.
Anzeigen von Datensätzen
  • Aktivieren Sie das Kontrollkästchen Ergebnisse beim Öffnen anzeigen.

Entfernen eines Ziels
  1. Klicken Sie in der Ansicht Kampagnendetail auf die Registerkarte Ziele und aktivieren Sie die Kontrollkästchen der Datensätze, die Sie aus der Kampagne entfernen möchten.
  2. Klicken Sie auf Ausgewählte Ziele entfernen.
  3. Klicken Sie auf OK.
