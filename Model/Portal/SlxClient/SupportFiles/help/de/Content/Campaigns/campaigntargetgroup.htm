Erstellen von Kampagnenzielgruppen

Zur Verwaltung Ihrer Kampagnen und Kampagnenziele können Sie Kampagnenzielgruppen erstellen.

Es ist nicht möglich, eine Gruppe mit sowohl Kontakt- als auch Interessentendatensätzen zu erstellen. Welche Datensätze in die Gruppe einbezogen werden, hängt ab vom Gruppentyp, den Sie im Dialog "Kampagnengruppe erstellen" auswählen.

So erstellen Sie eine Kampagnengruppe mit allen Kontakten oder Interessenten

  1. Öffnen Sie die Registerkarte "Ziele" der Ansicht "Kampagnendetail".
  2. Führen Sie einen der folgenden Schritte durch:
  • Fahren Sie mit Schritt 3 fort, um eine Gruppe mit allen aufgeführten Kontakten bzw. Interessenten zu erstellen.
  • Um eine Gruppe mit ausgewählten Kontakten oder Interessenten zu erstellen, aktivieren Sie das Kontrollkästchen neben jedem Datensatz, der zu der Gruppe hinzugefügt werden soll.
  1. Klicken Sie auf Als Gruppe speichern.

Der Dialog "Kampagnengruppe erstellen" wird angezeigt.

  1. Klicken Sie im Feld Gruppentyp auf den Dropdown-Pfeil, und wählen Sie Kontakt oder Interessent aus.
  2. Geben Sie im Feld Gruppenname einen eindeutigen Namen für die Gruppe ein.
  3. Führen Sie einen der folgenden Schritte durch:
  • Um eine Gruppe mit allen Kontakten oder Interessenten zu erstellen, wählen Sie Alle Kampagnenziele aus.
  • Zum Erstellen der Gruppe mit den in Schritt 2 ausgewählten Datensätzen wählen Sie Ausgewählte Kampagnenziele aus.
  1. Klicken Sie auf OK.

Die Kontakt- bzw. Interessentenlistenansicht wird geöffnet, und die neue Gruppe ist ausgewählt.
