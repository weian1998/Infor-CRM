Was sind Verkaufsprozesse?

Diese Funktion ist nur verfügbar, wenn die entsprechende Lizenz installiert ist. Ausführliche Informationen zur Lizenzierung finden Sie im SalesLogix-Planungshandbuch.

Zum Verwenden aller Vertriebsprozessfunktionen Sie müssen die Infor CRM-Desktop-Integration herunterladen. Sie können die Installation der Infor CRM-Desktop-Integration vom Web Client-Anmeldebildschirm aus durchführen . 

Verkaufsprozesse erleichtern die Verwaltung der Verkaufs-Pipeline. Ein Verkaufsprozess zeigt Ihnen die Schritte, die bis zum Abschluss des Verkaufs auszuführen sind. Prozesse können Sie auch dazu auffordern, bestimmte Aktivitäten durchzuführen, wie das Tätigen von Anrufen oder Schreiben von Briefen. Sie können einen Verkaufsprozess an eine geöffnete Verkaufschance anhängen, indem Sie einen Prozess in einer Liste verfügbarer Prozesse auswählen. Sie können beispielsweise einen Verkaufsprozess namens "Großaufträge" für eine Verkaufschance mit großem Umsatzpotenzial auswählen.

Sie finden Verkaufsprozesse in der Verkaufschancen-Detailansicht auf der Registerkarte Verkaufsprozess.

Verkaufsprozesse sind in Phasen und Schritte unterteilt. Schritte werden zu Phasen zusammengefasst. Die Schritt in der einer Phase mit dem Namen "Demonstration" können beispielsweise "Vorbereiten", "Produktdemo" und "Demo - Dank-E-Mail" heißen. Für jede Phase wird der Phasenname und die Wahrscheinlichkeit für einen erfolgreichen Verkaufschancenabschluss nach dem Ende der Phase angezeigt. Die Schritte führen Sie durch die Aktionen, die für einen erfolgreichen Geschäftsabschluss durchzuführen sind.

Verkaufsprozesse werden in Architect erstellt und verwaltet, normalerweise von einem Vertriebsmanager oder Systemadministrator. Diese Person kann festlegen, wer Zugriff auf die einzelnen Verkaufsprozesse hat.

Wenn Sie eine Phase überspringen, wird die Wahrscheinlichkeit der Verkaufschance nicht aktualisiert. Damit die Wahrscheinlichkeit der Verkaufschance automatisch aktualisiert wird, müssen Sie die Phasen der Reihe nach abschließen. Wenn eine Phase fehlt, können Sie die Phase manuell abschließen. Die Wahrscheinlichkeit wird dann ordnungsgemäß aktualisiert.
