Registerkarte "Verkaufsprozess"

Diese Funktion ist nur verfügbar, wenn die entsprechende Lizenz installiert ist. Ausführliche Informationen zur Lizenzierung finden Sie im Infor CRM Planungshandbuch.

Auf der Registerkarte "Verkaufsprozesse" können Sie einen Verkaufsprozess an eine Verkaufschance anhängen, den aktuell an eine Verkaufschance angehängten Verkaufsprozess ändern oder an den Schritten in einem Verkaufsprozess arbeiten.

Innerhalb des Verkaufsprozesses können Sie Schritte planen, abschließen oder überspringen. Normalerweise empfiehlt es sich, die Schritte der Reihe nach auszuführen. Sie haben jedoch die Möglichkeit, zu einer anderen Phase zu wechseln und an den Schritten dieser Phase zu arbeiten.

Sie schließen einen Schritt ab, indem Sie das Kontrollkästchen in der Spalte neben dem Schrittnamen aktivieren. Wenn alle Schritte einer Phase abgeschlossen sind, wird die prozentuale Wahrscheinlichkeit (dargestellt im Feld Momentaufnahme) aktualisiert.
  • Damit der Prozentsatz der Wahrscheinlichkeit automatisch aktualisiert wird, müssen die Phasen der Reihe nach abgeschlossen werden. Darüber hinaus umfassen einige Verkaufsprozess erforderliche Schritte, die verhindern, dass Schritte nicht nacheinander abgeschlossen werden.
  • Die prozentuale Wahrscheinlichkeit wird nicht aktualisiert, wenn die Phase offene Schritte enthält oder die vorherige Phase noch geöffnet ist.
  • Wenn eine Phase fehlt, können Sie die Phase manuell abschließen. Die Wahrscheinlichkeit wird dann ordnungsgemäß aktualisiert.
  • Wenn Sie einen Schritt wieder öffnen (durch Deaktivieren des Kontrollkästchens), wird die Wahrscheinlichkeit nicht reduziert.

Vorgehensweise

Anhängen von Verkaufsprozessen an eine Verkaufschance und Ändern der angehängten Prozesse

Sie können einen Verkaufsprozess an eine offene Verkaufschance anhängen oder den zugeordneten Verkaufsprozess durch einen anderen ersetzen, solange die Verkaufschance noch offen ist.

So hängen Sie einen Verkaufsprozess an eine Verkaufschance an

  1. Klicken Sie in der Verkaufschancen-Detailansicht auf die Registerkarte Verkaufsprozess.
  2. Klicken Sie im Feld Verkaufsprozess auf den Dropdown-Pfeil und wählen Sie den gewünschten Verkaufsprozess aus.

Die Phasen und Schritte für den ausgewählten Prozess werden im Raster angezeigt.

Ändern oder Entfernen eines Verkaufsprozesses, der an eine Verkaufschance angehängt ist
  1. Klicken Sie in der Verkaufschancen-Detailansicht auf die Registerkarte Verkaufsprozess.

  2. Führen Sie einen der folgenden Schritte durch:

    • Wählen Sie in der Liste Verkaufsprozess die Option Kein aus, um einen abgehängten Prozess zu entfernen.
    • Wählen Sie in der Liste Verkaufsprozess einen anderen Prozess aus, um Prozesse zu ändern.
      Sie können den an eine Verkaufschance angehängten Verkaufsprozess ändern, auch wenn einige Schritte im Prozess bereits durchgeführt wurden.
Arbeiten an Phasen und Schritten in einem Verkaufsprozess

Im Raster der Registerkarte "Verkaufsprozesse" werden Phasen und deren Schritte angezeigt. Schritte werden zu Phasen zusammengefasst. Die Qualifizierungs-Phase kann beispielsweise aus den Schritten "Anruf", "Informationen senden" und "Follow-Up" bestehen. Für jede Phase wird der Phasenname und die Wahrscheinlichkeit für einen erfolgreichen Verkaufschancenabschluss nach dem Ende der Phase angezeigt. Die Phase wird als beendet gekennzeichnet, wenn der Benutzer das Kontrollkästchen neben dem letzten Schritt der Phase aktiviert.

So führen Sie einen Schritt aus

  1. Suchen Sie auf der Registerkarte Verkaufsprozess die Phase, an der Sie arbeiten möchten.

In der Spalte "Abschließen" wird angezeigt, ob der Schritt abgeschlossen ist, in der Spalte "Schritt" wird der Name des Schritts angezeigt, und in der Spalte "Beschreibung" wird die Aktion angezeigt, die in dem Schritt enthalten ist. Wenn die Schritte nicht sichtbar sind, klicken Sie auf die Schaltfläche mit Doppelpfeil (Chevron) rechts neben dem Namen der Phase..

  1. Klicken Sie in der Spalte Schritt auf den Namen des Schritts.

Die Schrittnamen sind wie Hyperlinks formatiert (blau und unterstrichen).

  1. Führen Sie einen der folgenden Schritte durch:
  • Wenn in der Spalte Beschreibung der Wert Keine angezeigt wird, führen Sie die Anweisungen in der Spalte Schritt durch.
  • Wenn die Spalte Beschreibung eine Aktion enthält (beispielsweise Meeting oder Seriendruck), klicken Sie auf den Hyperlink und geben Sie Informationen in die Aktivitätsdialoge ein. (Falls verfügbar).
  1. Wenn Sie die in dem Schritt beschriebene Aktion ausgeführt haben, aktivieren Sie auf das Kontrollkästchen in der benachbarten Spalte.

Das Abschlussdatum wird automatisch in die Spalte "Abgeschlossen am" eingetragen.
