Bearbeiten von Details in der Verkaufsauftrag-Momentaufnahme

In diesem Dialogfeld können Sie den Steuersatz, Lieferungsdetails oder Rabatte für einen Verkaufsauftrag bearbeiten.

So gehen Sie zum Bearbeiten vor

  1. Öffnen Sie die Verkaufsauftrag-Detailansicht.
  2. Klicken Sie in der Verkaufsauftrag-Momentaufnahme auf Rabatt, Lieferung oder Steuer.
  3. Klicken Sie in das Feld Rabattsatz und geben Sie einen Zahlenwert für den Rabattsatz ein. Die Zahl wird automatisch in eine Prozentangabe geändert. Beispiel: "50" wird zu "50 %". Wenn Sie ",5" eingeben, wird dies zu "0,5 %".

Der Betrag wird automatisch berechnet.

  1. Klicken Sie in das Feld Lieferung und geben Sie die Kosten für die Lieferung des Verkaufsauftrags ein.
  2. Klicken Sie in das Feld Steuersatz und geben Sie einen Zahlenwert für den Steuersatz ein. Die Zahl wird automatisch in eine Prozentangabe geändert. Beispiel: "50" wird zu "50 %". Wenn Sie ",5" eingeben, wird dies zu "0,5 %".

Der Betrag wird automatisch berechnet.

  1. Klicken Sie auf OK.
