Planen einer Literaturanfrage

 Sie können eine Literaturanfrage planen, um den Versand von Marketinginformationen oder anderen Dokumenten an einen oder mehrere Kontakte zu veranlassen.  Nachdem Sie eine Anfrage geplant haben, kann die für Literaturanfragen zuständige Person oder Gruppe in Ihrer Firma die Anfrage erfüllen und den Auftrag abschließen.

Wenn Sie eine Anfrage für einen einzelnen Kontakt planen, wählen Sie diesen Kontakt aus, bevor Sie die folgenden Schritte befolgen:

So planen Sie

  1. Klicken Sie im Menü Planen auf Literaturanfrage.
  2. Geben Sie in das Feld Beschreibung eine kurze Beschreibung der Literaturanfrage ein.
  3. Führen Sie im Feld Anforderung für einen der folgenden Schritte aus:
    • Wenn Sie eine Anfrage für einen einzelnen Kontakt erstellen möchten, wählen Sie Kontakt, klicken auf die Schaltfläche Kontakt suchen  und wählen einen Kontakt aus.
      Wenn Sie die Planung von der Kontaktdetailansicht aus vornehmen, kann diese Feld unter Umständen schon einen Wert enthalten.
    • Zum Erstellen einer Anfrage für eine Gruppe von Kontakten wählen Sie Gruppe aus und klicken auf den Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.
    • Wenn Sie die Anfrage für eine Verkaufschance erstellen, wählen Sie Verkaufschance aus, klicken auf die Schaltfläche Verkaufschance suchen, wählen mithilfe der Suchfunktion eine Verkaufschance aus und klicken dann auf Schließen.
  4. Klicken Sie auf die Senden bis, klicken Sie auf Schaltfläche  Kalender  , wählen Sie ein Datum aus, und klicken Sie auf OK.
  5. Aktivieren Sie das Kontrollkästchen Lokal abschließen und drucken, um die Literaturanfrage an den Standarddrucker zu senden (der Druckvorgang erfolgt, nachdem Sie die Literaturanfrage geplant haben).
  6. Klicken Sie in den Feldern Versandart und Priorität auf die Schaltfläche Dropdown-Pfeil , und wählen Sie einen Eintrag in der Liste aus.
  7. Klicken Sie im Feld Deckblatt auf die Ordnerschaltfläche, wählen Sie in der Liste den gewünschten Begleitbrief oder ein Memo als Deckblatt aus und klicken Sie dann auf OK.
  8.  Unter Literaturliste drucken wählen Sie die Druckoptionen für die Liste aus: Liste mit Begleitbrief, auf separater Seite (nicht auf der Seite des Begleitbriefs), nur Anlagenliste oder nur Begleitbrief (keine Liste).
  9. Um eine Vorlage als Begleitbrief zu verwenden, klicken Sie bei Deckblatt auf auf Suchen und klicken dann auf die Vorlage, die Sie verwenden möchten.

    Das Auswählen einer Vorlage ist nur in Internet Explorer möglich, da das Deckblatt den Seriendruck erfordert, der nur in Internet Explorer, Versionen 9 und später, unterstützt wird. Weitere Informationen finden Sie unter Unterstützte Desktop-Integrationsfunktionen nach Browser.

    Um eine Vorlage in Internet Explorer zu verwenden, Sie müssen die Infor CRM-Desktop-Integration herunterladen. Sie können die Installation der Infor CRM-Desktop-Integration vom Web Client-Anmeldebildschirm aus durchführen . 
  10. Wählen Sie unter Verfügbare Einträge  die Elemente aus, die Sie in Ihre Anfrage einbeziehen möchten.
     
    1. Klicken Sie ggf. auf Verfügbare Einträge filterndie Schaltfläche Verfügbare Einträge filtern und geben Sie Zeichen ein, die in Familie, Nummer oder Name des hinzuzufügenden Eintrags enthalten sind.
    2. Doppelklicken Sie in die Spalte Menge und geben Sie die Anzahl von Kopien dieses Eintrags an, die Sie anfordern möchten.
  11. Klicken Sie auf Literaturanfrage planen.
  12. Wenn Sie das Kontrollkästchen Lokal abschließen und drucken aktiviert haben und zur Auswahl eines Druckers aufgefordert werden, wählen Sie einen Drucker in der Liste aus und klicken auf OK.

Die Literaturanfrage wird in der Ansicht "Aktivitäten" auf der Registerkarte Literatur sowie in den Kontakt- und Firmendetailansichten auf den Registerkarten "Literaturanfragen" aufgezeichnet.

Der/die vom Administrator bestimmte Verantwortliche für die Überwachung von Literaturanfragen wird Ihre Anfrage erhalten und zum angegebenen Datum erfüllen.
  • Die Gesamtkosten ergeben sich aus den Kosten, die mit einem Dokument verknüpft sind, und der Anzahl der gewählten Dokumente.
  • Wenn Sie über die entsprechenden Zugriffsrechte verfügen, können Sie der Literaturliste Dokumente hinzufügen, sie bearbeiten oder aus der Liste löschen.
