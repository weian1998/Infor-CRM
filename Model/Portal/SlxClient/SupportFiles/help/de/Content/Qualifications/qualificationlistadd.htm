Hinzufügen und Bearbeiten eines Qualifizierungsmerkmals

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Qualifizierungsmerkmale sind die Kriterien, mit denen Informationen zu Interessenten überprüft werden können, und dienen zum Bestimmen, ob und wann ein Interessent zu einer Verkaufschance werden kann.

Vorgehensweise

Hinzufügen eines Qualifizierungsmerkmals
  1. Öffnen Sie in der Qualifizierungsmerkmal-Detailansicht die Registerkarte Elemente und klicken Sie auf Qualifizierungsmerkmalelement hinzufügen.
  2. Fügen Sie die folgenden Informationen hinzu:
    • Sichtbar - Aktivieren Sie das Kontrollkästchen, wenn das Element in der "Interessentendetailansicht" angezeigt werden soll.
    • Sortierungsposition - Geben Sie die Nummer ein, die die Position des Elements darstellt. Wenn der Eintrag beispielsweise an zweiter Stelle in der Liste angezeigt werden soll, geben Sie 2 ein.
    • Beschreibung - Geben Sie für das Qualifizierungsmerkmal eine Beschreibung ein, die in der Interessentendetailansicht angezeigt werden soll.
    • Notizen anzeigen - Aktivieren Sie das Kontrollkästchen, wenn Sie Benutzern die Möglichkeit geben möchten, in der Interessentendetailansicht in einem Notizenfeld Informationen einzugeben.

  3. Klicken Sie auf OK.

Bearbeiten eines Qualifizierungsmerkmals
  1. Öffnen Sie in der Qualifizierungsmerkmal-Detailansicht die Registerkarte Elemente und klicken Sie auf neben dem Qualifizierungsmerkmal, das Sie bearbeiten möchten, auf Bearbeiten.
  2. Bearbeiten Sie beliebige der folgenden Informationen:
    • Sichtbar - Aktivieren Sie das Kontrollkästchen, wenn das Element in der "Interessentendetailansicht" angezeigt werden soll.
    • Sortierungsposition - Geben Sie die Nummer ein, die die Position des Elements darstellt. Wenn der Eintrag beispielsweise an zweiter Stelle in der Liste angezeigt werden soll, geben Sie 2 ein.
    • Beschreibung - Geben Sie für das Qualifizierungsmerkmal eine Beschreibung ein, die in der Interessentendetailansicht angezeigt werden soll.
    • Notizen anzeigen - Aktivieren Sie das Kontrollkästchen, wenn Sie Benutzern die Möglichkeit geben möchten, in der Interessentendetailansicht in einem Notizenfeld Informationen einzugeben.

  3. Klicken Sie auf OK.
