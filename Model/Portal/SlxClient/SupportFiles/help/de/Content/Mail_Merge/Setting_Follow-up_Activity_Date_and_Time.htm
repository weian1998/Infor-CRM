Festlegen von Datum und Uhrzeit für eine Folgeaktivität

Bei Verwendung der Infor CRM-Seriendruckfunktion können Sie auf der Registerkarte "Historienoptionen" eine Folgeaktivität planen. Für diese Aktivität muss ein Datum eingegeben werden, und Sie können eine Uhrzeit und eine Dauer angeben.

So wählen Sie ein Datum aus:

  1. Doppelklicken Sie im Bereich Aktivitätszeiten auf das Feld Datum, oder klicken Sie auf die Kalenderschaltfläche.
    Nun wird die aktuelle Monatskalenderansicht angezeigt, wobei das aktuelle Datum in Blau hervorgehoben wird. Bei Bedarf können Sie mit den nach links bzw. nach rechts weisenden Pfeilen den Monat und das Jahr ändern.
  2. Zum Auswählen eines Datums führen Sie einen der folgenden Schritte aus:
    • Setzen Sie den Mauszeiger auf ein Datum, und doppelklicken Sie.
    • Setzen Sie den Mauszeiger auf ein Datum, klicken Sie darauf und klicken Sie dann auf OK.

Das ausgewählte Datum wird im Feld Datum angezeigt.

So geben Sie eine Uhrzeit und eine Dauer an:

  1. Deaktivieren Sie ggf. die Option Ohne Zeitangabe. Das Häkchen wird entfernt.
    Auf diese Weise werden die Felder unter Zeit aktiviert.
  2. Doppelklicken Sie in das Feld Zeit, um das Dialogfeld Zeit und Dauer angeben zu öffnen.
  3. Setzen Sie den Mauszeiger auf die gewünschte Startzeit der Aktivität, und klicken Sie, um die Zeit auszuwählen.
    Die Dauer wird in 15-Minuten-Intervallen anzeigt.
  4. Zum Ändern der Dauer setzen Sie den Mauszeiger auf die Startzeit der Aktivität, klicken, halten die Maustaste gedrückt und ziehen den Cursor. Lassen Sie die Maustaste los, wenn die richtige Dauer angezeigt wird.
  5. Klicken Sie auf OK, um die Uhrzeit und Dauer zu speichern.
