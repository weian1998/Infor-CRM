Anhängen eines Dokuments an jeden Kontakt oder Interessenten - Wichtige Meldung

Dieser wichtige Hinweis gilt für Infor CRM Web-Benutzer, wenn ein Seriendruck an eine große Anzahl von Kontakten erstellt wird.

Wenn Sie auf Ja klicken, wird für jeden Kontakt ein Dokument angehängt, und bei einer großen Anzahl von Kontakten verläuft die Verarbeitung des Seriendrucks ungewöhnlich langsam. Der Grund hierfür ist, dass jedes Dokument via HTTP an den Webserver übertragen werden muss. Anschließend wird es in den für Anlagen vorgegebenen Pfad kopiert und im Protokollierungspfad protokolliert. Dies kann recht viel Zeit in Anspruch nehmen und zudem die Leistung des Webservers beeinträchtigen. Bei einem Seriendruck an eine große Anzahl Kontakte sollten Sie also über eine Hochgeschwindigkeitsverbindung verfügen. Ein Seriendruck sollte idealerweise nicht mehr als 100 Kontakte umfassen, wenn sich Anlagenpfad und Protokollierungspfad ebenfalls auf dem Webserver befinden (optimale Bedingungen).

Wenn Sie auf Nein klicken, wird die Option nicht aktiviert und es werden keine Dokumente angehängt.

Verwandte Themen

Festlegen von Historienoptionen für den Seriendruck
