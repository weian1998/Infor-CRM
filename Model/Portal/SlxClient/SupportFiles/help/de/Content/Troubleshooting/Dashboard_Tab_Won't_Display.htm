Eine Registerkarte in "Eigenes Cockpit" erscheint nicht oder öffnet eine andere Website

Problem:

Wenn ich versuche, eine Registerkarte in "Eigenes Cockpit" anzuzeigen, tritt eines der folgenden Ereignisse auf:

  • Die Registerkarte wird nicht geöffnet.
  • Ich werde sofort zu einer anderen Website weitergeleitet.

Lösung:

Die Registerkarte enthält ein Website-Widget, das auf eine Website zugreift, die nicht für die Verwendung von Widgets optimiert ist. Sie müssen die Registerkarte löschen und neu erstellen.

Um dies in Zukunft zu vermeiden, sollten Sie zuerst eine "Test"-Registerkarte erstellen und das Website-Widget dort hinzufügen, um sicherzustellen, dass das Verhalten der Website den Erwartungen entspricht, bevor ein Website-Widget zu einer vorhandenen Registerkarte hinzugefügt wird.
