Auswählen von Datensatz/-sätze, für die Historieneinträge aufgezeichnet werden sollen

Wenn Sie Microsoft Outlook als E-Mail-Anwendung einsetzen, können Sie Outlook-E-Mail-Nachrichten als Infor CRM-Historieneinträge speichern. Wenn Sie eine E-Mail an mehrere Kontakte oder Interessenten senden, können Sie wählen, für welche Datensätze ein dazugehöriger Historieneintrag erstellt werden soll.

So wählen Sie Datensätze aus

  1. Wählen Sie im Dialogfeld Auswählen Datensatz/-sätze die Kontakte oder Interessenten aus, für die ein zugeordneter Historieneintrag erstellt werden soll.
  1. Falls weitere Informationen zum Bestimmen des ordnungsgemäßen Kontakts oder Interessenten benötigt werden, wählen Sie einen Datensatz aus, und klicken Sie auf Kontakt-/Interessentendetails.
  1. Die Kontakt- oder Interessentendetailansicht enthält zusätzliche Informationen.
  2. Klicken Sie zum Abschluss auf Schließen.
  1. Klicken Sie auf OK.
