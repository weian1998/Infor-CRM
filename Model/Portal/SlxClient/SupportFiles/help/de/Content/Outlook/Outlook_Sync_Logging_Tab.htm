Festlegen der Protokollierungsoptionen für Infor CRM Connector

Verwenden Sie die Registerkarte "Protokollierung", um die Protokollierung zu aktivieren oder die Protokollierungsinformationen anzuzeigen, wenn Sie Probleme mit der Synchronisierung haben, oder wenn Sie von Ihrem Administrator entsprechend aufgefordert wurden.

Hat ein Benutzer Synchronisierungsprobleme, kann die Aufzeichnung von Microsoft Outlook aus aktiviert werden, um die Fehlersuche zu erleichtern.

Diese Registerkarte gilt nur für Outlook-Synchronisierung.

So legen Sie Optionen fest

  1. Klicken Sie auf die Schaltfläche Infor CRM Optionen, und klicken Sie ggf. auf Protokollierung.
  2. Wählen Sie Protokollierung aktivieren aus.
  3. Anzeigen von Protokollierungsinformationen im Feld Letzte Fehler.
  4. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
