Verwenden der Outlook-Integration

 Manche Funktionen von Outlook Integration sind nicht bei allen Browsern verfügbar. Für weitere Informationen siehe Unterstützte Funktionen nach Browser.

Wenn Ihr Unternehmen Outlook Integration verwendet, können Sie und Outlook-E-Mail-Nachrichten als Infor CRM-Historieneinträge speichern. Beispielsweise sind folgende Aktionen möglich:

  • Verfassen Sie eine E-Mail-Nachricht in Outlook, wählen Sie im Infor CRM-Adressbuch "Kontakte" aus, und klicken Sie dann auf "Senden an CRM", um den Nachrichtentext und etwaige Anlagen im dazugehörigen Datensatz aufzuzeichnen. Nachrichten können auch mit Anlagen versehen werden.Weitere Informationen finden Sie unter Grundlegendes zur E-Mail-Integration.
  • Wählen Sie eine E-Mail-Nachricht in Outlook aus, und klicken Sie dann auf "In Historie aufzeichnen", um den Nachrichtentext und etwaige Anlagen im dazugehörigen Datensatz aufzuzeichnen.
  • Senden der Informationen zu einem ein Infor CRM-Kontakt im vCard-Dateiformat per E-Mail.
  • Anhängen von Dokumenten in der Infor CRM-Bibliothek an eine Nachricht in Outlook.
  • Sie können E-Mail-Nachrichten aus Outlook per Ziehen und Ablegen auf die Registerkarte Infor CRM-Historie verschieben.
  • Klicken Sie in der Kontakt- oder Interessentendetailansicht auf die Schaltfläche "E-Mail", um einen neue E-Mail-Nachricht zu öffnen, die an diesen Kontakt oder Interessenten adressiert ist. Weitere Informationen finden Sie unter Senden von E-Mails an Kontakte oder Interessenten.
  • Klicken Sie in der Ticket-Detailansicht auf die Schaltfläche E-Mail, um Ticket-Informationen in eine neue E-Mail-Nachricht zu kopieren. Klicken Sie danach in Microsoft Outlook auf Senden an CRM, um die Nachricht zu senden und sie in Infor CRM zu speichern. Weitere Informationen finden Sie unter Senden einer E-Mail-Nachricht aus einem Ticket.

Wichtige Informationen zur Outlook E-Mail-Integration

Wenn Sie die E-Mail-Integrationsfunktionen verwenden, sollten Sie Folgendes beachten:

  • Aufzeichnen von Notizen/Historie-Daten aus einer von Ihnen gesendeten Nachricht: In jeder E-Mail-Nachricht, die Sie in Outlook verfassen, wird die Schaltfläche "Senden an CRM" angezeigt. Klicken Sie auf diese Schaltfläche, um Ihre Nachricht zu senden und Notiz-/Historiendaten aufzuzeichnen. Die Nachricht wird für jeden Infor CRM-Kontakt oder -Interessenten und die dazugehörige Firma auf der Registerkarte "Notizen/Historie" aufgezeichnet. Infor CRM ermittelt den Kontakt oder Interessenten anhand einer übereinstimmenden E-Mail-Adresse. Wenn Sie beispielsweise in Outlook eine Nachricht an jschmidt@schmidtgmbh.de verfassen und auf "Senden an CRM" klicken, sucht Infor CRM nach einem Kontakt oder Interessenten mit derselben E-Mail-Adresse. Wenn ein solcher gefunden wird, dann wird die Nachricht in den Notiz-/Historiendaten aufgezeichnet. Wenn keine übereinstimmende E-Mail-Adresse gefunden wird, dann wird in einem Dialog gemeldet, dass der Kontakt nicht gefunden wurde, und es werden keine Notiz-/Historiendaten aufgezeichnet. (Sie können die Notiz-/Historiendaten jedoch aufzeichnen, indem Sie die Nachricht aus Outlook zum Infor CRM Client ziehen.) Wenn mehrere Kontakte gefunden werden, werden Sie aufgefordert, einen Kontakt auszuwählen, der die Notiz-/Historiendaten erhalten soll.
    Wenn die Nachricht eine oder mehrere Anlagen hat, werden Sie gefragt, ob eine Kopie der Anlagen in Infor CRM beibehalten werden soll. Wenn Sie auf "Ja" klicken, wird für jeden Kontakt oder Interessenten ein Hyperlink zu dieser Anlage zur Registerkarte "Anlagen" hinzugefügt.
  • Aufzeichnen von Notizen/Historiendaten aus einer von Ihnen empfangenen Nachricht: Im Hauptfenster von Outlook wird die Schaltfläche "In Historie aufzeichnen" angezeigt. Wählen Sie eine Nachricht aus, und klicken Sie auf "In Historie aufzeichnen" (im Menü Extras), oder ziehen Sie die Nachricht aus Outlook in den Infor CRM-Kontaktdatensatz. Anlagen werden wie oben im Zusammenhang mit der Schaltfläche "Senden an CRM" beschrieben aufgezeichnet.
  • Senden von Nachrichten bei geschlossenem Infor CRM: Wenn der Infor CRM Client beim Klicken auf die Schaltfläche "Senden an CRM" oder "In Historie aufzeichnen" geschlossen ist, wird eine temporäre Datei in ein temporäres Verzeichnis geschrieben. Wenn Sie Infor CRM Client öffnen, werden die Notizen-/Historien-Informationen aufgezeichnet.

Erweiterte Outlook-Integration

  • Ausschließen von Kontakten aus Ihrer Organisation: Ihr Systemadministrator kann Regeln für einen Domänennamenausschluss einrichten, um für einige Kontakte (z. B. für Mitarbeiter Ihres Unternehmens) die Aufzeichnung von Notizen/Historiendaten zu verhindern. Dieselben Regeln gelten auch, wenn Nachrichten automatisch für Folgeaktivitäten markiert werden. Weitere Information hierzu erhalten Sie bei Ihrem Systemadministrator oder im Thema "Hinzufügen eines Domänenausschlusses" in der Hilfe zu Administrator.
  • Senden von Nachrichten ohne Verbindung zum Server mit Exchange: Das Infor CRM-Adressbuch in Outlook ist nicht von einer Verbindung zum Host-Server mit Exchange, sondern von der Verbindung zur Datenbank (Netzwerk oder remote) abhängig. Wenn Sie auf die Schaltfläche "Senden an CRM" klicken, wird die Nachricht in Infor CRM in den Notiz-/Historiendaten aufgezeichnet, verbleibt jedoch in Ihrem Outlook-Postausgang, bis eine Verbindung zum Server mit Exchange hergestellt wird.
