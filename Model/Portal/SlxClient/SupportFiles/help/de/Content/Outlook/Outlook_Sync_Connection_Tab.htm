Festlegen der Verbindungsoptionen für Infor CRM Connector

Verwenden Sie die Verbindungsoptionen, um Ihre Verbindung mit dem Infor CRM-Server einzurichten.
  • Diese Registerkarte ist sowohl für Outlook-Synchronisierung als auch für Outlook Integration erforderlich.
  • Für Web Client-Benutzer, die ihre Optionen im Desktop Manager festlegen, sind diese Optionen evtl. bereits konfiguriert.

So legen Sie Optionen fest

  1. Klicken Sie auf die Schaltfläche Infor CRM Optionen, und klicken Sie ggf. auf Verbindung.
  2. Geben Sie im Feld Server Ihren Servernamen ein.
    • Infor CRM Client und Infor CRM Web Client-Benutzer – Geben Sie die URL ein, die Ihr Administrator für die Verbindung mit Infor CRM bereitgestellt hat. Wählen Sie folgendes Format:
      http://servername:port or https://servername:port
    • Offline Web Client-Benutzer – Geben Sie den URL ein, um eine Verbindung mit Ihrem lokalen Infor CRM-Server herzustellen. Wählen Sie folgendes Format:
      http://localhost:port
      Der Port ist gewöhnlich 8088.

  3. Führen Sie einen der folgenden Schritte durch:

    • Wenn Sie sich mithilfe Ihrer Windows-Anmeldeinformationen bei Infor CRM anmelden, wählen Sie die Option Windows-Authentifizierung verwenden.
    • Geben Sie Ihren Infor CRM-Benutzernamen und Ihr Passwort ein.
  4. Klicken Sie auf Testen.
    Wenn der Test fehlschlägt, überprüfen Sie Ihre Verbindungsinformationen. Wenn die Informationen richtig sind und Sie dennoch keine Verbindung herstellen können, wenden Sie sich an Ihren Administrator.
  5. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
