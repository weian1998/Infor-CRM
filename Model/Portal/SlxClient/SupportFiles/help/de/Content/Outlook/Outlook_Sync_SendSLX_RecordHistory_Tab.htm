Festlegen der Infor CRM Connector-Optionen für "Senden an CRM/In Historie aufzeichnen"

Verwenden Sie die Option "Senden an CRM/In Historie aufzeichnen", um Optionen für die E-Mail-Historie und Anlagen festzulegen.
  • Diese Registerkarte ist nur für Outlook Integration erforderlich.
  • Für Web Client-Benutzer, die ihre Optionen im Desktop Manager festlegen, sind diese Optionen evtl. bereits konfiguriert.

So legen Sie Optionen fest

  1. Klicken Sie auf die Schaltfläche Infor CRM Optionen und klicken Sie auf Senden an CRM/In Historie aufzeichnen.
  2. Wählen Sie aus, wann Sie zu einer Aktion aufgefordert werden möchten, wenn eine Outlook-E-Mail als ein Infor CRM-Historiendatensatz protokolliert wird.
    1. Bei doppelten Kontakten nachfragen - Fragt nach, wenn ein doppelter Kontakt oder Interessent gefunden wird.
    2.  Bei Kontakt nachfragen - Fordert Sie zur Auswahl auf, mit welchen Kontakten oder Interessenten der Historieneintrag verknüpft werden soll.
    3. Bei nicht gefundenem Kontakt nachfragen - Fragt nach, wenn ein Kontakt oder Interessent nicht in der Datenbank gefunden wird.
    4. Historiendetails vor dem Speichern anzeigen - Blendet das Dialogfeld "E-Mail abschließen" ein, bevor eine E-Mail-Nachricht in der Historie aufgezeichnet wird.
    5. Benachrichtigungen auf Taskleiste anzeigen – Zeigt Einblendmeldungen an, in denen die jeweiligen Desktop Manager-Aktivitäten beschrieben werden.
  3. Legen Sie die Anlagenoptionen fest.
    1. Wählen Sie die Mindestgröße für zu ignorierende Bildanlagen aus.
      Verwenden Sie diese Option, um kleine Dateianlagen, z. B. E-Mail-Signaturen, zu ignorieren, damit sie nicht als ein Infor CRM-Anlage gespeichert werden.
    2. Listen Sie im Feld Anlagen mit diesen Dateinamen ignorieren die Dateierweiterungen oder Dateitypen auf, die nicht als Infor CRM-Anlagen gespeichert werden sollen.
  4. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
