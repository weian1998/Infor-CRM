Festlegen der Adressbuch-Optionen für Infor CRM Connector

Verwenden Sie die Adressbuch-Optionen, um das Aussehen Ihres Infor CRM-Adressbuches einzustellen.

Diese Registerkarte ist nur für Outlook Integration erforderlich.

So legen Sie Optionen fest

  1. Klicken Sie auf die Schaltfläche Infor CRM Optionen, und klicken Sie bei Bedarf auf Adressbuch.
  2. Wählen Sie unter Kontaktnamen anzeigen als aus, wie die Namen in Ihrem Adressbuch erscheinen sollen.
  3. Wählen Sie unter Beim Anzeigen des Adressbuchs die Optionen zur benutzerspezifischen Anpassung Ihres Adressbuchs aus, oder heben Sie die Markierungen dieser Optionen auf.
  4. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
