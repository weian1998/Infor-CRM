Ändern Ihres Infor CRM Connector-Servers

Wenn Sie Ihren Server auf der Registerkarte "Verbindung" ändern, werden Ihre Daten mit dem neuen Infor CRM-Server synchronisiert. Dies sollte jedoch nicht zu doppelten Datensätzen führen.

So nehmen Sie die Änderung vor

  1. Klicken Sie auf die Schaltfläche Infor CRM Optionen, und klicken Sie auf Verbindung.
  2. Geben Sie im Feld Server die URL für den neuen Infor CRM-Server ein.
  3. Ändern Sie ggf. Ihren Benutzernamen und das Passwort.
  4. Klicken Sie auf die Schaltfläche Test, um die neue Verbindung zu testen.
  5. Klicken Sie auf der Registerkarte Sync auf den Dropdown-Pfeil Ad-hoc-Gruppe, und wählen Sie die Kontaktsynchronisierungsgruppe.
    Sie müssen die Ad-hoc-Gruppe auch dann erneut auswählen, wenn die Gruppe auf dem neuen Server denselben Namen aufweist.
  6. Klicken Sie auf OK.
