Registerkarte "Kalender des Benutzers"

Auf der Registerkarte "Kalender des Benutzers" in der Benutzerdetailansicht können Sie Berechtigungen für andere Benutzer zum Anzeigen und Ändern des Kalenders dieses Benutzers festlegen. Standardmäßig können Benutzer nicht auf die Kalender anderer Benutzer zugreifen. Wenn der Kalender eines Benutzers zum Profil eines anderen Benutzers hinzugefügt wird, werden automatisch Zugriffsrechte zum Anzeigen aktiviert.

Beachten Sie beim Bestimmen des Kalenderzugriffs die folgenden Regeln:

  • Ein Benutzer wird in seiner eigenen Kalenderliste angezeigt. Entfernen Sie einen Benutzer nicht aus seinem eigenen Kalender. Durch Entfernen des Benutzers wird der Zugriff auf seinem eigenen Kalender aufgehoben.
  • Der Benutzer "Administrator" wird in der Kalenderliste eines Benutzers angezeigt und kann nicht entfernt werden.
  • Wenn ein Benutzer nicht in der Kalenderliste angezeigt wird, hat er keinen Zugriff auf den Kalender des ausgewählten Benutzers.
  • Wenn ein Benutzer in der Liste angezeigt wird, kann er den Kalender des ausgewählten Benutzers anzeigen (wenn Hinzufügen, Bearbeiten, Löschen und Synchronisieren nicht ausgewählt sind, hat der Benutzer lediglich schreibgeschützten Zugriff).
  • Wenn ein Benutzer eine Aktivität plant und diese Aktivität von einem anderen Benutzer, der Zugriff auf seinen Kalender hat, geändert wird, wird die Person, die diese Aktivität geplant hat, automatisch über die Änderung benachrichtigt.

Vorgehensweise

Öffnen der Registerkarte Kalender des Benutzers
  • Klicken Sie in der Benutzerdetailansicht auf die Registerkarte Andere Kalender. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
Berechtigen anderer Benutzer zum Zugriff auf den Kalender dieses Benutzers
  1. Klicken Sie auf Benutzer hinzufügen, um Benutzer hinzuzufügen.
  2. Wählen Sie den gewünschten Benutzer mithilfe der Suchfunktion aus.

  3. Klicken Sie auf OK.

  4. Verwenden Sie das Dialogfeld Kalenderzugriff, um Zugriffsrechte festzulegen.

  5. Klicken Sie auf OK.

Ändern des Zugriffs auf den Kalender dieses Benutzers
  1. Klicken Sie neben dem zu bearbeitenden Benutzer auf Bearbeiten.
  2. Use the to set access rights .
  3. Verwenden Sie das Dialogfeld Kalenderzugriff, um Zugriffsrechte festzulegen.
Entfernen des Zugriffs auf den Kalender dieses Benutzers
  1. Klicken Sie neben dem zu entfernenden Benutzer auf Entfernen.
    Wenn Sie den Benutzer entfernen, der als anzuzeigender Standardkalender für diesen Benutzer im Web Client ausgewählt ist, ist auf den Kalender dieses Benutzers weiter ein Zugriff möglich, bis ein neuer Standardbenutzer ausgewählt wird.
  2. Klicken Sie auf OK.
