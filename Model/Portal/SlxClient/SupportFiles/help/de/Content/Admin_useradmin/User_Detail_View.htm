Benutzerdetailansicht

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In der Detailansicht können Sie Informationen für einen ausgewählten Datensatz anzeigen, hinzufügen oder bearbeiten. Die Detailansicht umfasst Informationsfelder, Registerkarten und einen benutzerdefinierten mittleren Bereich. 

Vorgehensweise

Öffnen der Benutzerdetailansicht
  1. Klicken Sie auf der Navigationsleiste auf Administration und dann auf Benutzer, um die Benutzerdetailansicht zu öffnen.
  2. Klicken Sie auf einen Benutzer in der Liste, oder nutzen Sie die Suchfunktion, um einen Benutzer auszuwählen.

Die Benutzerinformationen werden in der Detailansicht geöffnet.

Verwenden Sie die Suchfunktion, um nach Benutzerdatensätzen zu suchen.

Öffnen einer Gruppe

Umschalten zur Listenansicht
  • Klicken Sie auf der Symbolleiste auf die Schaltfläche Listenansicht.

Allgemeine Aufgaben

Im Aufgabenbereich unter Allgemeine Aufgaben können Sie Folgendes ausführen:

  • Zu Gruppe hinzufügen
  • Aus Gruppe entfernen
  • Benutzer kopieren
  • Profil kopieren
  • In allen Teams ersetzen

Informationsfelder

Die Benutzerinformationsfelder enthalten die Hauptinformationen über den Benutzer.

Detailansicht-Registerkarten

Auf jeder Registerkarte werden für den betreffenden Benutzer bestimmte Informationen angezeigt. Klicken Sie auf einen Hyperlink, um weitere Informationen zu jeder Registerkarte anzuzeigen.
