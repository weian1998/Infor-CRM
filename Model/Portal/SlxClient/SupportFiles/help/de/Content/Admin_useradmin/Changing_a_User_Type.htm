Ändern von Benutzertypen

Wählen Sie die Funktion "Benutzertyp ändern", um einen Benutzer in einen anderen Benutzertyp umzuwandeln. Sie können beispielsweise einen Nur Web-Benutzer in einen Gleichzeitigen Benutzer ändern. Sie können auch einen Benutzer deaktivieren oder einen inaktiven Benutzer reaktivieren.

Für den neuen Benutzertyp muss eine Lizenz vorhanden sein, es sei denn, Sie wandeln einen Benutzer in einen inaktiven Benutzer um. Durch die Deaktivierung eines Benutzers wird eine Instanz des Lizenztyps frei, der diesem Benutzer zugewiesen war.

Regeln

Bevor Sie einen Benutzer in einen inaktiven Benutzer umwandeln, müssen Sie wie folgt vorgehen:

  • Falls der Benutzer für Datensätze zuständig ist, führen Sie (in Administrator) ein Gebietsneuausrichtung durch.
  • Falls der Benutzer für einige Firmen ein Account Manager ist, dann führen Sie eine Gebietsneuausrichtung durch, um einen neuen Account Manager zuzuweisen.
  • Falls der Benutzer ein Manager bzw. Vorgesetzter ist, dann teilen Sie alle Benutzer, die ihm direkt unterstellt sind, ihrem neuen Vorgesetzten oder dem Administrator zu.

Bevor Sie einen Benutzer in einen Nur Web-Benutzer umwandeln, müssen Sie Folgendes tun:

  • Falls der Benutzer ein Manager ist, deaktivieren Sie diese Einstellung in der Benutzerdetailansicht.
  • Wenn andere Benutzer als Mitglieder des Benutzerteams dieses Benutzers aufgeführt sind, entfernen Sie diese Benutzer.

So nehmen Sie die Änderung vor

  1. Öffnen Sie die Benutzerdetailansicht des Benutzers, den Sie umwandeln möchten.
  2. Wählen Sie im Feld Benutzertyp einen neuen Benutzertyp aus.
  3. Klicken Sie auf Speichern.
