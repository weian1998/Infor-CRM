Hinzufügen von Benutzern

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

 Lesen Sie vor dem Hinzufügen eines oder mehrerer Benutzer das Thema Ein neuer Mitarbeiter benötigt Zugriff auf Infor CRM, und gehen Sie die ersten vier Optionen mit Kontrollkästchen durch.

Sie können keine Infor CRM-Administratoren (Benutzer mit dem Benutzernamen „admin“) hinzufügen.

So gehen Sie zum Hinzufügen vor

  1. Klicken Sie im Menü Administration auf Benutzer hinzufügen.
  2. Wählen Sie im Feld Hinzufügen den Benutzertyp aus, den Sie hinzufügen möchten.
  3. Klicken Sie auf Lizenzen aktualisieren, um die verfügbare Anzahl von Lizenzen zu aktualisieren.
  4. Wählen Sie im Feld Menge die Anzahl der Benutzer aus, die Sie hinzufügen möchten.

Neben dem Feld Menge wird die Anzahl der Benutzer angezeigt, die für diesen Lizenztyp verfügbar sind.

  1. Falls Sie zum Konfigurieren der Einstellungen jede Registerkarte im Profil des Benutzers manuell bearbeiten möchten, fahren Sie mit Schritt 8 fort.
  2. Um den Benutzer anhand einer Vorlage oder des Profils eines vorhandenen Benutzers zu erstellen, aktivieren Sie das Kontrollkästchen Profil erstellen von. Wählen Sie danach eine der folgenden Optionen aus:

    		3.

    Wird die Vorlage oder der Benutzer nicht der Rolle Standardbenutzer zugewiesen, können alle neuen Benutzer, die aus dem Profil erstellt werden, nicht auf manche der Web Client-Funktionen zugreifen.
  • Vorlage - Dient zum Hinzufügen eines Benutzers anhand einer vorhandenen Benutzervorlage. Wählen Sie im Feld Vorlagen den Namen des Vorlagenbenutzers aus.
  • Vorhandener Benutzer -Dient zum Hinzufügen eines neuen Benutzers anhand des Profils eines vorhandenen Benutzers. Wählen Sie im Feld Benutzer den Namen des vorhandenen Benutzers aus.
  1. Wählen Sie die Registerkarten aus, die aus der Vorlage oder dem Benutzerprofil kopiert werden sollen. Weitere Informationen...
  • Allgemein - Allgemeine und Abteilungseinstellungen werden kopiert. Die Felder Benutzername, Name, Titel und E-Mail werden beim Kopieren von Informationen nicht übernommen.
  • Mitarbeiter - Die Felder Fax, Mobil, Pager, Durchwahl, PIN und Buchhaltung sowie die persönlichen Felder werden beim Kopieren von Informationen nicht übernommen.
  • Sicherheit - Alle Sicherheitseinstellungen werden kopiert.
  • Teams - Alle Teameinstellungen werden kopiert.
  • Kalender - Alle Kalendereinstellungen werden kopiert.
  • Service/Support - Alle Service- und Support-Einstellungen werden kopiert.
  • Client-Optionen (Client-System) - Der Standardzugriffsberechtigte der Firma und Basisvorlageneinstellungen werden kopiert.
  1. Klicken Sie auf OK.

Wenn Sie neue Benutzer hinzufügen, wird jedem einzelnen Benutzer automatisch ein Site-Code zugewiesen.

  1. Nach dem Erstellen eines neuen Benutzers können Sie in der Benutzerdetailansicht das Profil des Benutzers ändern.

    Sie müssen den Benutzernamen und das Passwort für neue Benutzer festlegen. Falls Sie das Standardpasswort nicht mitteilen möchten, ändern Sie das Passwort in ein generisches Passwort und weisen Sie die Benutzer an, ihr Passwort bei der ersten Anmeldung bei Infor CRM zu ändern.

  2. Klicken Sie auf Speichern.
