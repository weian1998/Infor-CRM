Registerkarte "Benutzerteammitglieder"

Benutzerteams ermöglichen den Benutzern, auf Firmen zuzugreifen, deren Zugriffsberechtigter ein anderer Benutzer ist. Wenn Sie beispielsweise Herrn Meier Zugriff auf alle Firmen von Herrn Müller geben möchten, können Sie dies erreichen, indem Sie Herrn Meier dem Benutzerteam von Herrn Müller hinzufügen.

Benutzer auf dieser Registerkarte können auf die Firmen zugreifen, deren Zugriffsberechtigter der Hauptbenutzer (der Benutzer, dessen Profil Sie gerade bearbeiten) ist.

Vorgehensweise

Öffnen der Registerkarte Benutzerteammitglieder
  • Klicken Sie in der Benutzerdetailansicht auf die Registerkarte Benutzerteammitglieder. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
Berechtigen anderer Benutzer zum Zugriff auf Firmen dieses Benutzers
  1. Klicken Sie auf die Schaltfläche Anderen Benutzer zum Benutzerteam dieses Benutzers hinzufügen.
  2. Wählen Sie den gewünschten Benutzer mithilfe der Suchfunktion aus.

  3. Klicken Sie auf OK.

Bearbeiten des Sicherheitsprofils eines Benutzers
  1. Klicken Sie auf den Link Sicherheitsprofil, und legen Sie im Dialogfeld Berechtigungen fest.
  2. Klicken Sie auf OK.
    Dieses Sicherheitsprofil wird nur dann wirksam, wenn der ausgewählte Benutzer auf Firmen zugreift, deren Zugriffsberechtigter der Hauptbenutzer (der Benutzer, dessen Profil Sie gerade bearbeiten) ist. Dies trifft nicht zu, wenn der ausgewählte Benutzer auf Firmen zugreift, für die er Zugriffsberechtigter ist.
Wenn Sie einen Benutzer aus dem Team des Benutzers entfernen möchten,
  1. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Entfernen.

  2. Klicken Sie auf OK.
