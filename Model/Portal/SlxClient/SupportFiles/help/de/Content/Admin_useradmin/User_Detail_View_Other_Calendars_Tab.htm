Registerkarte "Andere Kalender"

Auf der Registerkarte "Andere Kalender" in der Benutzerdetailansicht können Sie Berechtigungen für einen Benutzer zum Anzeigen und Ändern der Kalender anderer Benutzer festlegen. Standardmäßig können Benutzer nicht auf die Kalender anderer Benutzer zugreifen. Wenn der Kalender eines Benutzers zum Profil eines anderen Benutzers hinzugefügt wird, werden automatisch Zugriffsrechte zum Anzeigen aktiviert.

Beachten Sie beim Bestimmen des Kalenderzugriffs die folgenden Regeln:

  • Ein Benutzer wird in seiner eigenen Kalenderliste angezeigt. Entfernen Sie einen Benutzer nicht aus seinem eigenen Kalender. Durch Entfernen des Benutzers wird der Zugriff auf seinem eigenen Kalender aufgehoben.
  • Wenn ein Benutzer nicht in der Kalenderliste angezeigt wird, hat der Benutzer, dessen Profil Sie bearbeiten, keinen Zugriff auf den Kalender dieses Benutzers.
  • Wenn ein Benutzer in der Liste angezeigt wird, hat der Benutzer, dessen Profil Sie bearbeiten, Anzeigezugriff auf den Kalender des ausgewählten Benutzers (wenn Hinzufügen, Bearbeiten, Löschen und Synchronisieren nicht ausgewählt sind, hat der Benutzer lediglich schreibgeschützten Zugriff).
  • Wenn ein Benutzer eine Aktivität plant und diese Aktivität von einem anderen Benutzer, der Zugriff auf seinen Kalender hat, geändert wird, wird die Person, die diese Aktivität geplant hat, automatisch über die Änderung benachrichtigt.

Vorgehensweise

Öffnen der Registerkarte Andere Kalender
  • Klicken Sie in der Benutzerdetailansicht auf die Registerkarte Andere Kalender. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
Erlauben des Zugriffs auf den Kalender eines anderen Benutzers
  1. Klicken Sie auf Benutzer hinzufügen, um Benutzer hinzuzufügen.
  2. Wählen Sie den gewünschten Benutzer mithilfe der Suchfunktion aus.
  3. Klicken Sie auf OK.
  4. Verwenden Sie das Dialogfeld Kalenderzugriff, um Zugriffsrechte festzulegen.
  5. Klicken Sie auf OK.
Ändern des Zugriffs auf den Kalender eines anderen Benutzers
  1. Klicken Sie neben dem Benutzer, dessen Kalender Sie ändern möchten, auf Bearbeiten.
  2. Verwenden Sie das Dialogfeld Kalenderzugriff, um Zugriffsrechte festzulegen.

  3. Klicken Sie auf OK.

Entfernen des Zugriffs auf den Kalender eines anderen Benutzers
  1. Klicken Sie neben dem zu entfernenden Benutzer auf Entfernen.
    Wenn Sie den Benutzer entfernen, der als anzuzeigender Standardkalender für diesen Benutzer im Web Client ausgewählt ist, ist auf den Kalender dieses Benutzers weiter ein Zugriff möglich, bis ein neuer Standardbenutzer ausgewählt wird.
  2. Klicken Sie auf OK.
