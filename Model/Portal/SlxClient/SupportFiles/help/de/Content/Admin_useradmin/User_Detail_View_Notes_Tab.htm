Registerkarte "Notizen"

Auf der Registerkarte "Notizen" können Sie Notizen zum Benutzer hinzufügen. Diese Registerkarte ermöglicht das Hinzufügen von Informationen, die auf einer anderen Registerkarte ggf. nicht erfasst wurden.

So erfolgt die Verwaltung

  1. Klicken Sie in der Benutzerdetailansicht auf die Registerkarte Notizen.
  2. Klicken Sie zum Hinzufügen eines Datums-/Uhrzeitstempels auf .
  3. Geben Sie Notizen zum Benutzer ein.
  4. Klicken Sie auf Speichern.
