Einführung in Widgets

  Widgets sind Plug-Ins, die Sie zu einem Cockpit hinzufügen können. Es gibt verschiedene Typen von Widgets, mit deren Hilfe Sie Ihre Daten anzeigen können. Bewegen Sie den Mauszeiger über dem Diagramm in einem Widget, um die für dieses Widget ausgewählte Dimension und Kennzahl anzuzeigen.

 Dimensionen sind die einer Entität, die Sie anzeigen möchten, zugeordneten Daten.

Für die Entität "Firma" können Sie beispielsweise die folgenden Dimensionen anzeigen:

  • Account Manager
  • Stadt
  • Branche
  • Zugriffsberechtigter
  • BDI
  • Bundesland/Kanton
  • Status
  • Untertyp
  • Type

Kennzahlen bieten eine Möglichkeit der Berechnung der Daten, die Sie anzeigen möchten.

Für die Entität "Firma" können Sie beispielsweise folgende Kennzahlen verwenden:

  • Durchschnittliche Anzahl Tage seit letzter Aktivität
  • Durchschnittlicher Umsatz
  • Durchschnittliche Zeit als Kunde
  • Firmenanzahl ermitteln
  • Gesamtumsatzerlös

Klicken Sie unten in jedem Widget auf den Hyperlink, um detaillierte Informationen zu diesen Daten anzuzeigen. Wenn Sie beispielsweise das Widget "Gruppenliste" so konfiguriert haben, dass alle offenen Verkaufschancen angezeigt werden, wird über den Hyperlink Gruppe anzeigen die Verkaufschancen-Listenansicht geöffnet. Klicken Sie auf der Navigationsleiste auf Willkommen, um zum Cockpit zurückzukehren.

Es gibt mehrere Typen von Widgets:

Die Liste der verfügbaren Widgets ist von Ihrer Implementierung abhängig. Möglicherweise haben Sie nicht auf alle Widget-Typen Zugriff.

 Diagramm-Widgets

  •  Balken: Zeigt die Daten als horizontale Balken an. BeispielWenn Sie die Daten für "Durchschnittliche Auftragsgröße nach Zugriffsberechtigtem" auswählen, werden die Teams auf der Y-Achse und die Anzahl der Aufträge auf der X-Achse angezeigt. Jeder Balken erstreckt sich waagerecht bis zu dem Wert auf der X-Achse, der der Gesamtanzahl am entspricht.
  •  Spalte: Zeigt die Daten als vertikale Säulen an. BeispielWenn Sie die Daten für "Durchschnittliche Auftragsgröße nach Zugriffsberechtigtem" auswählen, werden die Teams auf der X-Achse und die Anzahl der Aufträge auf der Y-Achse angezeigt. Jeder Balken erstreckt sich senkrecht bis zu dem Wert auf der Y-Achse, der der Gesamtanzahl am entspricht. Sie können der Konfiguration einen Zielwert hinzufügen, der im Diagramm als waagerechte Linie angezeigt wird und den Stand der Säulen gegenüber dem Ziel veranschaulicht.
  •  Trichter: Zeigt die Daten in progressiven Anteilen in einer farblich kodierten Trichterdarstellung an. Verwenden Sie Trichterdiagramme, um Informationen in einem mehrstufigen Prozess darzustellen, wie etwa dem potenziellen Umsatz oder der Anzahl der Verkaufschancen in Ihrer Verkaufschancen-Pipeline. Beispiele

Offene Verkaufschancen nach Verkaufspotenzial

Verwenden Sie ein Trichterdiagramm zum Aufzeigen des potenziellen Umsatzes von Verkaufschancen in Ihrer Pipeline für jede Umsatzebene.

  1. Klicken Sie mit der rechten Maustaste auf eine Cockpit-Registerkarte, und klicken Sie auf Inhalt hinzufügen.
  2. Klicken Sie unter Trichterdiagramm auf die Schaltfläche Hinzufügen.
  3. Geben Sie in Titel einen Namen für das Trichterdiagramm ein, z. B. Offene Verkaufschancen nach Verkaufspotenzial.
  4. Wählen Sie in der Liste Entität die Option Verkaufschance aus.
  5. Wählen Sie in der Liste Gruppe die Option Alle offenen aus.
  6. Wählen Sie in der Liste Dimension die Option Phase aus.
  7. Wählen Sie in der Liste Metrik die Option Verkaufspotenzial gesamt aus.
  8. Legen Sie über Beschriftungen anzeigen, Beschriftungen abschneiden und Beschriftungslänge die gewünschten Einstellungen fest.
  9. Klicken Sie auf OK. 
    Ein Bereich von Verkaufspotenzialbeträgen wird angezeigt, wobei jeder Verkaufspotenzialbereich als eine andersfarbige Ebene dargestellt wird. Die Größe jeder Ebene stellt die Anzahl der Verkaufschancen in einem bestimmten Verkaufspotenzialbereich dar.

Offne Verkaufschancen nach Phase

Veranschaulichen Sie mit einem Trichterdiagramm, wie viele Verkaufschancen sich in den einzelnen Phasen der Verkaufschancen-Pipeline befinden.

  1. Klicken Sie mit der rechten Maustaste auf eine Cockpit-Registerkarte, und klicken Sie auf Inhalt hinzufügen.
  2. Klicken Sie unter Trichterdiagramm auf die Schaltfläche Hinzufügen.
  3. Geben Sie in Titel einen Namen für das Trichterdiagramm ein, z. B. Meine Verkaufschancen nach Phase.
  4. Wählen Sie in der Liste Entität die Option Verkaufschance aus.
  5. Wählen Sie in der Liste Gruppe die Option Alle Verkaufschancen aus.
  6. Wählen Sie in der Liste Dimension die Option Phase aus.
  7. Wählen Sie in der Liste Metrik die Option Verkaufschancenanzahl ermitteln aus.
  8. Legen Sie über Beschriftungen anzeigen, Beschriftungen abschneiden und Beschriftungslänge die gewünschten Einstellungen fest.
  9. Klicken Sie auf OK. 
    Jede Phase wird in der Ablauffolge des Prozesses als andersfarbige Ebene dargestellt. Die Größe jeder Ebene stellt die Anzahl der Verkaufschancen in einer bestimmten Phase dar.
  •  Linie: Zeigt die Daten als Abfolge von Datenpunkten an, die mit einer Linie verbunden sind. Beispiel Wenn Sie die Daten für Aktive Marketingkampagnen nach Region auswählen, werden die Teams auf der X-Achse, die Anzahl der Kampagnen auf der Y-Achse und die Anzahl der Datenpunkte im Diagramm angezeigt.
  •  Kreis: Zeigt die Daten in einem Kreis mit Segmenten in unterschiedlichen Farben und einer Legende mit Zusatzangaben an. Beispiel Wenn Sie Firmen nach Region mit fünf Segmenten anzeigen, zeigt das Diagramm einen fünffarbigen Kreis mit einer Legende, die die Teams sowie die Gesamtanzahl der Firmen pro Team enthält. Sie können das Kreisdiagramm mit fünf oder zehn Segmenten konfigurieren und festlegen, ob ein Segment Sonstige mit allen nicht dargestellten Daten angezeigt werden soll.

Listen-Widgets

  • Gruppe- Zeigt die angegebenen Gruppeninformationen an. Beispiel Wenn Sie "Eigene abgeschlossene Verkaufschancen" auswählen. Sie können die Spalten in der Liste sortieren und über den Hyperlink "Gruppe anzeigen" zu der Firma oder Entität navigieren, um detaillierte Informationen anzuzeigen.

    		•
    • Wenn Sie eine Registerkarte "Cockpit" freigeben, die eine Gruppe enthält, muss die Gruppe ebenfalls für diese Benutzer freigegeben werden, damit die Gruppe auf der Registerkarte "Cockpit" angezeigt wird.
    • Der Hyperlink "Gruppe anzeigen" steht für die Historie nicht zur Verfügung.

  • Links: Zeigt die angegebenen Links an. BeispielSie können Hyperlinks hinzufügen, die Sie in Infor CRM oder im Internet häufig nutzen
  • Zuletzt angezeigt: Zeigt die zuletzt angezeigten Details als Links an. Wenn Sie die Details einer Firma oder eines Kontakts angezeigt haben, enthält die Liste Hyperlinks zu diesen Detailansichten.

Weitere Widgets

  • Aktivitäten von heute - Zeigt eine Liste aller Ihrer für den heutigen Tag geplanten Aktivitäten an. Wenn Sie weitere Informationen einblenden möchten, klicken Sie auf den Hyperlink, um die Detailansicht der Aktivität aufzurufen.
  • Willkommen - Zeigt eine Einführung in den Infor CRM Web Client mit einem Link zur Hilfe zu Infor CRM Web Client an. Eine kurze Beschreibung aller Funktionen, die Infor CRM Ihnen zum Verwalten Ihrer Arbeit bereitstellt.
  •  SData – Zeigt einen Web-Feed mithilfe von SData (Sage Data)-Feeds an. Sie können aus einer Liste der verfügbaren Feeds in Ihrer Infor CRM-Datenbank oder Buchhaltungsintegration auswählen.
  •  Website – Zeigt eine Website innerhalb eines Widgets auf dem Cockpit an. Sie können eine URL einfügen oder eingeben und die Höhe des Widgets festlegen, um zu bestimmten, wie viel Platz es im Cockpit einnehmen wird.

    		Manche Websites sind evtl. nicht dafür optimiert, als Widget hinzugefügt zu werden. Sie sollten zuerst eine "Test"-Registerkarte erstellen und das Website-Widget dort hinzufügen, um sicherzustellen, dass das Verhalten der Website den Erwartungen entspricht, bevor ein Website-Widget zu einer vorhandenen Registerkarte hinzugefügt wird.
  •  Web-Feed - Zeigt ein RSS- oder Atom-Web-Feed innerhalb eines Widgets auf dem Cockpit an. Sie können eine Web-Feed-Adresse einfügen oder eingeben, um zu bestimmen, wie die Feed-Stories erscheinen sollen, und die Höhe des Widgets festlegen, um zu bestimmen, wie viel Platz es im Cockpit einnehmen wird.
