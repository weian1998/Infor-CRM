Verwenden von Widgets

Infor CRM stellte mehrere standardmäßige Widgets zur Verfügung. In einem Cockpit können Sie Widgets hinzufügen, bearbeiten und entfernen sowie per Ziehen und Ablegen auf der Seite neu anordnen.

Mithilfe vordefinierter Dimensionen, einer Art von Filter, können Sie Kennzahlen für Ihre Daten wertebereichs- oder einzelwertbezogen in einem der Diagramm-Widgets anzeigen. Sie können beispielsweise ein Spaltendiagramm-Widget konfigurieren, das alle geschlossenen, erfolgreichen Verkaufschancen nach Zugriffsberechtigtem und Gesamtsumme des Verkaufsauftrags anzeigt, oder ein Kreisdiagramm zur Darstellung von Verkaufsaufträgen, die ein Team generiert hat.

Die Liste der verfügbaren Widgets ist von Ihrer Implementierung abhängig. Möglicherweise haben Sie nicht auf alle Widget-Typen Zugriff.

Vorgehensweise

Hinzufügen eines Widgets
  1. Klicken Sie mit der rechten Maustaste auf die Registerkarte Mein Cockpit, und klicken Sie auf Inhalt hinzufügen.
  2. Klicken Sie im Dialogfeld Inhalt hinzufügen für das Widget, das Sie hinzufügen möchten, auf die Schaltfläche Hinzufügen.
    Um beispielsweise ein Kreisdiagramm-Widget hinzuzufügen, klicken Sie unter "Kreisdiagramm" auf "Hinzufügen".
    Das leere Widget wird im Cockpit angezeigt und der Widget-Editor geöffnet.
  3. Konfigurieren Sie die Widget-Einstellungen, und klicken Sie auf OK.
Bearbeiten eines vorhandenen Widgets
  1. Klicken Sie in der Symbolleiste des Widgets auf die Schaltfläche Einstellungen .
  2. Nehmen Sie die erforderlichen Änderungen vor.
  3. Klicken Sie auf OK.
    Das geänderte Widget wird angezeigt.

      • Titel - Der Titel, der auf der Titelleiste des Widgets angezeigt wird.

      • Beschriftung - Der unter dem Titel angezeigte Text mit weiteren Informationen.

      • Entität - Die Entität, aus der Sie Daten abrufen möchten. Beispiel: Verkaufschancen. Alle Entitäten mit Listenansichten können in Diagrammen und Listen angezeigt werden.

      • Gruppe - Die Gruppe, für die Sie Daten anzeigen möchten. Beispiel: Alle Verkaufschancen. Alle einem Benutzer zur Verfügung stehenden Gruppen können in Diagrammen und Listen verwendet werden.

      • Dimension - Die Daten, die Sie anzeigen möchten. Beispiel: Wahrscheinlichkeit. Alle aktuellen Filter vom Typ Bereich und Eindeutig stehen als Dimensionen zur Verfügung.
         

        Die verfügbaren Dimensionen hängen von den verfügbaren Entitätsfiltern und anderen Konfigurationseinstellungen ab. Weitere Informationen hierzu erhalten Sie von Ihrem Systemadministrator.

      • Kennzahl - Die Berechnungsweise der Daten. Beispiel: Tatsächlicher Gesamtbetrag. Alle Entitäten mit Listenansichten verfügen über eine dazugehörige Zählungskennzahl, z. B. CountLeads und CountAccounts. Für Verkaufschancen und Historie gibt es zusätzliche Kennzahlen.

      • X-Achse - Die Beschriftung mit der Beschreibung der Daten auf der X-Achse des Diagramms.

      • Y-Achse - Die Beschriftung mit der Beschreibung der Daten auf der Y-Achse des Diagramms.

      • Limit - Geben Sie die Höchstzahl von Elementen an, die im Diagramm angezeigt werden soll. Abhängig von der gewählten Dimension wird die Höchstzahl möglicherweise ignoriert.

      • Beschriftungen abschneiden - Wählen Sie Ja oder Nein aus, um Beschriftungen abzuschneiden, damit sie in das Diagramm passen.

      • Beschriftungslänge - Die maximale Länge einer Beschriftung, bevor sie abgeschnitten wird.

Konfigurieren des Gruppenlisten-Widgets
  1. Geben Sie im Dialogfeld Gruppenliste - Einstellungen einen Titel für das Widget ein.
  2. Wählen Sie eine Entität und eine Gruppe aus den Dropdown-Listen aus. Beispiel: Verkaufschancen und Abgeschlossen - Erfolg.
  3. Geben Sie in Anzahl der Datensätze die in der Liste gewünschte Anzahl von Datensätzen ein. Beispiel: 10, 25 oder 50.
  4. Geben Sie in das Feld Sichtbare Datensätze die gewünschte Anzahl sichtbarer Datensätze ein: 10, 25 oder 50.
    Wenn Sie z. B. im 3. Schritt  50 Datensätze eingegeben haben, aber zugleich nur 10 in der Liste anzeigen möchten, geben Sie 10 ein. Das Widget wird entsprechend der gewünschten Anzahl der sichtbaren Datensätze angepasst.

    Sie können einen beliebigen Wert von 1 bis zur maximal zugelassenen Anzahl von Datensätzen wählen. Der Standardhöchstwert ist 50.

  5. Klicken Sie auf OK.
    Das geänderte Widget wird angezeigt.

    Zum Ein- oder Ausblenden von Spalten in einer Gruppenliste, klicken Sie mit der rechten Maustaste auf eine Spaltenüberschrift und wählen die hinzuzufügenden Spalten aus bzw. heben Sie die Auswahl der zu entfernenden Spalten auf.

Konfigurieren des Verknüpfungslisten-Widgets
  1. Geben Sie im Dialogfeld Verknüpfungen - Einstellungen einen Titel für das Widget ein.
  2. Klicken Sie auf Verknüpfung hinzufügen.
  3. Doppelklicken Sie auf das Feld Symbol-URL, und geben Sie den Pfad des Symbols ein, das Sie für die Verknüpfung anzeigen möchten, Beispiel: /google.jpg.

    		4.

    Dieser URL muss ein relativer Pfad zu einem Bild auf dem Webserver sein.
  4. Doppelklicken Sie auf das Feld Verknüpfungs-URL, und geben Sie den URL der Verknüpfung ein, den Sie anzeigen möchten. Beispiel: http://localhost/SlxClient/Account.aspx.

    		5.

    Dieser Pfad kann ein relativer Pfad im SLXClient-Portal oder ein absoluter Hyperlink zu einer externen Website sein. Die URL für eine externe Website muss die Angabe "http://" enthalten. Um beispielsweise auf www.google.com zuzugreifen, muss der URL http://www.google.com lauten. Andernfalls sucht der Browser nach http://<SLXClient-Website>/www.google.com
  5. Doppelklicken Sie auf das Feld Verknüpfungstext, und geben Sie den Text ein, den Sie als Verknüpfung anzeigen möchten. Beispiel: Google-Suchseite.
  6. Doppelklicken Sie auf das Feld Ziel, und geben Sie eine ID für das Ziel ein. Wenn der Hyperlink z. B. in einem neuen Browserfenster geöffnet werden soll, geben Sie Neues Fenster ein. Wenn Sie auf den Hyperlink klicken, wird ein neues Browserfenster bzw. eine neue Registerkarte geöffnet.

    7. Wenn Sie dieses Feld leer lassen, wird der Hyperlink nach dem Klicken im aktiven Browserfenster bzw. in der aktiven Registerkarte geöffnet. Wenn mehrere Hyperlinks dieselbe Zielangabe haben, wird jeder Hyperlink beim Klicken auf den nächsten Hyperlink ersetzt.
  7. Klicken Sie auf Verknüpfung hinzufügen, um einen weiteren Hyperlink hinzuzufügen, oder auf OK, um den Editor zu schließen.
    Nach Klicken auf OK wird das Widget mit den Hyperlinks angezeigt.
Konfigurieren des SData-Widgets
  1. Geben Sie im Dialogfeld Widget-Einstellungen bearbeiten einen Titel für das Widget ein.
  2. Klicken Sie auf den Dropdown-Pfeil neben SData-Feed, und wählen Sie einen Feed aus der Liste aus. Folgende Optionen sind möglich:
    • Infor CRMDynamische Entität - ermöglicht den Zugriff auf standardmäßige Entitätsressourcentypen wie Kontakte, Firmen und Tickets.
    • Infor CRM System - ermöglicht den Zugriff auf nicht standardmäßige Ressourcentypen wie Aktivitäten und Anlangen.
    • Alle Buchhaltungsintegrations-Feeds, die Ihnen zur Verfügung stehen.
  3. Klicken Sie auf Weiter.
  4. Wenn Sie eine Meldung erhalten, die darauf hinweist, dass der Feed keine Ressourcen enthält, müssen Sie einen anderen Feed auswählen. Wenn ein Feed nicht mindestens eine Ressource enthält, können keine Daten angezeigt werden.
  5. Klicken Sie auf den Dropdown-Pfeil neben Ressource, und wählen Sie den Typ der Informationen aus, die Sie in der Liste anzeigen möchten. Beispiel: Firma oder Aktivität.

    Die Inhalte der Ressourcenliste basieren auf dem Feed, den Sie ausgewählt haben.

  6. Geben Sie im Feld WHERE-Klausel die WHERE-Anweisung ein, die Ihnen bereitgestellt wurde. Wenn keine WHERE-Anweisung bereitgestellt wurde, wenden Sie sich an Ihren Administrator.
    Falls eine WHERE-Klausel definiert ist, werden alle Daten, auf die Sie Zugriff haben, für die SData-Ressource ausgegeben. Dies kann zu einem sehr großen Datensatz führen.

    Das Erstellen einer WHERE-Klausel wird nur fortgeschrittenen Benutzern empfohlen. Weitere Informationen zum Erstellen einer WHERE-Klausel finden Sie unter Erstellen einer WHERE-Klausel.

  7. Klicken Sie auf Weiter.
  8. Wählen Sie die Spalten aus Ihrer SData-Ressource, die Sie im Widget anzeigen möchten.
    • Wählen Sie eine oder mehrere Spalten aus der ersten Liste, und klicken Sie auf die Schaltfläche >. Zum Auswählen aller Spalten muss nicht jede Spalte einzeln ausgewählt werden; klicken Sie einfach auf die Schaltfläche >>.
    • Verwenden Sie die Schaltflächen < und <<, um alle Spalten zu entfernen, die Sie nicht sehen möchten.
  9. Geben Sie im Feld Sichtbare Datensätze einen Wert zwischen 1 und 100 ein, um die maximale Anzahl der Datensätze zu bestimmen, die in Ihrem Cockpit angezeigt werden sollen.
  10. Klicken Sie auf Fertig stellen.
    Nachdem die Widget-Einstellungen gespeichert wurden, zeigt das Widget die Datensätze aus dem Feed an. Informationen aus einem SData-Feed sind geschützt, damit Sie nur die Datensätze einsehen können, für die Sie Zugriffsberechtigungen haben. Wenn die Anzahl der ausgegebenen Datensätze zu groß für "Sichtbare Datensätze" ist, verwenden Sie den Abrollbalken.
Konfigurieren des Website-Widgets
  1. Geben Sie im Dialogfeld Widget-Einstellungen bearbeiten einen Titel für das Widget ein.
  2. Geben oder fügen Sie im Feld Website-Adresse die URL der Website ein, die Sie zum Cockpit hinzufügen möchten.
    Die URL sollte das Format http://website.com aufweisen.

  3. Führen Sie einen der folgenden Schritte durch:

    • Akzeptieren Sie für die Höhe den Standardwert von 250 Pixeln.
      Diese Größe entspricht in etwa der Größe eines Gruppenlisten-Widgets mit 10 sichtbaren Datensätzen.
    • Ändern Sie die Höhe auf einen beliebigen Wert zwischen 100 und 1000.
      Wird beispielsweise die Höhe zu 500 Pixel geändert, wird das Widget doppelt so groß; wird die Höhe zu 125 Pixel geändert, wird das Widget halb so groß.
  4. Klicken Sie auf OK.
    Die Website wird dem Cockpit hinzugefügt. Wenn die Website leer erscheint, ist die Website-Adresse entweder ungültig oder die Website ist nicht verfügbar.
Konfigurieren des Web-Feed-Widgets
  1. Geben Sie im Dialogfeld Widget-Einstellungen bearbeiten einen Titel für das Widget ein.
  2. Geben oder fügen Sie im Feld Feed-Adresse die URL des Feeds ein, den Sie zum Cockpit hinzufügen möchten.
  3. Klicken Sie auf den Dropdown-Pfeil neben Einblenden, und wählen Sie eine der folgenden Optionen aus:
    • Nur Titelzeile - Zeigt nur die Titelzeile der jeweiligen Feed-Story an.
    • Titelzeile und Zusammenfassung - zeigt die Titelzeile und eine Zusammenfassung der jeweiligen Feed-Story an.
  4. Geben Sie im Feld Maximal eine Zahl zwischen 1 und 25 ein, um die maximale Anzahl der Feed-Einträge zu bestimmen, die im Cockpit angezeigt werden sollen.
    Je nach Web-Feed können auf dem Cockpit weniger Einträge erscheinen. Die Standardanzahl ist 10; dies ist eine für viele Web-Feeds typische Grenze.

  5. Führen Sie einen der folgenden Schritte durch:

    • Akzeptieren Sie für die Höhe den Standardwert von 250 Pixeln.
      Diese Größe entspricht in etwa der Größe eines Gruppenlisten-Widgets mit 10 sichtbaren Datensätzen.
    • Ändern Sie die Höhe auf einen beliebigen Wert zwischen 100 und 1000.
      Wird beispielsweise die Höhe zu 500 Pixel geändert, wird das Widget doppelt so groß; wird die Höhe zu 125 Pixel geändert, wird das Widget halb so groß.
  6. Klicken Sie auf OK.
    Der Web-Feed wird zum Cockpit hinzugefügt.
Minimieren eines Widgets
  • Klicken Sie in der Symbolleiste des Widgets auf die Schaltfläche Minimieren.
    Der Widget-Inhalt wird im Cockpit minimiert. Zum Maximieren des Widgets klicken Sie erneut auf die Schaltfläche.
Entfernen eines Widgets
  • Klicken Sie in der Symbolleiste des Widgets auf die Schaltfläche Schließen.
    Das Widget wird dauerhaft aus dem Cockpit entfernt.
