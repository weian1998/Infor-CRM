Ändern von Abteilungsmitgliedern

Zum Ändern von Abteilungsmitgliedern können Sie die Abteilung bearbeiten oder ein Benutzerprofil ändern.

Vorgehensweise

Bearbeiten von Abteilungsmitgliedern in der Benutzerdetailansicht

  1. Öffnen Sie die Ansicht Benutzerdetail des Benutzers, den Sie als Abteilungsmitglied hinzufügen oder entfernen möchten.
  2. Klicken Sie auf die Registerkarte Abteilungsmitgliedschaft, und führen Sie einen der folgenden Schritte durch:
  • Klicken Sie auf Hinzufügen, um den Benutzer einer Abteilung hinzuzufügen. Wählen Sie die Abteilung mithilfe der Suchfunktion aus.
  • Klicken Sie für die Abteilung, aus der Sie den Benutzer entfernen möchten, auf den Hyperlink Entfernen.
  1. Klicken Sie auf Speichern.

Bearbeiten von Abteilungsmitgliedern in der Abteilungsdetailansicht

  1. Öffnen Sie die Abteilungsdetailansicht der Abteilung, die Sie ändern möchten.
  2. Klicken Sie auf die Registerkarte Mitgliedern, und führen Sie einen der folgenden Schritte durch:
  • Klicken Sie auf Hinzufügen, um einen Benutzer der Abteilung hinzuzufügen. Wählen Sie den Benutzer mithilfe der Suchfunktion aus.
  • Klicken Sie für das Abteilungsmitglied, das Sie löschen möchten, auf den Hyperlink Entfernen.
  1. Klicken Sie auf Speichern.
