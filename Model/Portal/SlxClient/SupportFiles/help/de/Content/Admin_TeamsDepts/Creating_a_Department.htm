Erstellen von Abteilungen

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Eine Abteilung ist ein spezialisiertes Team. Benutzer werden standardmäßig nicht einer Abteilung zugewiesen.

Abteilungsregeln

  • Abteilungen können Benutzer, jedoch keine Teams enthalten. Teams können jedoch Abteilungen enthalten.
  • Ein Benutzer kann zu mehr als einer Abteilung gehören.
  • Abteilungen können nicht Zugriffsberechtigte von Firmen sein.

So gehen Sie zum Erstellen vor

  1. Klicken Sie im Menü Administration auf Neue Abteilung.
  2. Geben Sie in das Feld Name einen Namen für die Abteilung ein.
  3. Wählen Sie im Feld Standardsicherheitsprofil ein neues Standardprofil für die Abteilung aus. Dieses Profil gilt, sobald die Abteilung einem Team hinzugefügt wurde.
  4. Klicken Sie auf Speichern.
  5. Auf der Registerkarte Mitglieder können Sie der Abteilung Benutzer hinzufügen.
  6. Auf der Registerkarte Teams können Sie die Abteilung Teams hinzufügen.
