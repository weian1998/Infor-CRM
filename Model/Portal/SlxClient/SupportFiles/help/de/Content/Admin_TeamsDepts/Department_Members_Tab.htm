Registerkarte "Mitglieder"

Auf der Registerkarte "Mitglieder" werden die Benutzer in einer Abteilung angezeigt. Auf dieser Registerkarte können Sie die Abteilungsmitglieder verwalten.

So erfolgt die Verwaltung

  1. Klicken Sie in der Abteilungsdetailansicht auf die Registerkarte Mitglieder.
  2. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf Mitglied zur Abteilung hinzufügen, um einen Benutzer zu der Abteilung hinzuzufügen. Wählen Sie das Abteilungsmitglied mithilfe der Suchfunktion aus.
    • Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Entfernen.

  3. Klicken Sie auf OK.
