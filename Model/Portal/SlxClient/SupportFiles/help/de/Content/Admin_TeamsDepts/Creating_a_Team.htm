Erstellen von Teams

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Ein Team kann aus Benutzern, Teams und Abteilungen bestehen. Beim Erstellen eines Teams müssen Sie eine Person als Team-Zugriffsberechtigten auswählen.

Beachten Sie beim Festlegen der Rechte eines Benutzers zum Erstellen und Verwalten von Teams die folgenden Regeln.

Ein Benutzer kann niemals

  • sich selbst zu einem bestehenden Team hinzufügen
  • sich selbst aus einem Team löschen
  • den Systemadministrator aus einem Team löschen
  • ein Teammitglied löschen

Weitere Informationen zur Teamsicherheit...

Wenn ein Benutzer einem Team hinzugefügt wird, werden seine Sicherheitseinstellungen standardmäßig auf das Profil festgelegt, das auf der Registerkarte Sicherheit festgelegt ist. Das Sicherheitsprofil kann für jedes Team geändert werden, zu dem der Benutzer gehört. Wenn Sie das Sicherheitsprofil eines Benutzers in einem Team ändern, wirkt sich dies nicht auf das Standardprofil des Benutzers oder auf sein Profil in einem der anderen Teams aus, denen der Benutzer angehört.

Wenn ein Benutzer mehrere Sicherheitsprofile hat (z. B. wenn der Benutzer einem Team als geschachteltes Team oder geschachtelte Abteilung hinzufügt wurde und auch ein direktes Mitglied des übergeordneten Teams ist) haben die Sicherheitsprofile denselben SECCODE (Zugriffsberechtigter). Beim Bestimmen des Sicherheitszugriffs für ein Feld können mehrere Profile zurückgegeben werden. Im Web Client wird der Zugriff durch Kombinieren des maximalen Zugriffs auf alle Profile bestimmt, die dem jeweiligen Benutzer und Zugriffsberechtigten zugewiesen sind.

So gehen Sie zum Erstellen vor

  1. Klicken Sie im Menü Administration auf Neues Team.
  2. Geben Sie in das Feld Name einen Namen für das Team ein.
  3. Geben Sie im Feld Zugriffsberechtigter einen Zugriffsberechtigten für das Team ein. Teams können mehrere Teamzugriffsberechtigte haben.
  4. Wählen Sie im Feld Standardsicherheitsprofil ein neues Standardprofil für das Team aus. Dieses Profil gilt, sobald das Team anderen einem Team hinzugefügt wurde.
  5. Aktivieren Sie das Kontrollkästchen Manager des Benutzers automatisch zu Team hinzufügen, wenn Sie beim Hinzufügen eines neuen Teammitglieds den Manager des Benutzers dem Team hinzufügen möchten.
  6. Klicken Sie auf Speichern.
  7. Fügen Sie Teammitglieder hinzu.
  8. Bearbeiten Sie ggf. die Sicherheitsrechte der einzelnen Teammitglieder.
  9. Klicken Sie auf Speichern.
