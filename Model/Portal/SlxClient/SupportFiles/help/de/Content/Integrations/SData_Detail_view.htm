Verwenden der Konfigurationsansicht der SData-Synchronisierung

Verwenden Sie die SData-Sync-Konfigurationsansicht, um den Infor CRM-Feed einzurichten, der für die Synchronisierung von Infor CRM mit integrierten Systemen verwendet wird.

So konfigurieren Sie SData

  1. Stoppen Sie auf dem Web-Host den Infor CRM SData-Synchronisierungsdienst (Dienst).
  2. Klicken Sie in der Navigationsleiste auf SData-Sync-Konfiguration
  3. Richten Sie den Infor CRM Feed ein, indem Sie die folgenden Felder konfigurieren. Der Feed wird während der Dateneingabe automatisch aktualisiert.
    1. Wenn Ihr SData-URL "Secure Socket Layer" verwendet, markieren Sie das Kontrollkästchen SSL (Secure Socket Layer) verwenden.
    2. Geben Sie im Feld Server den Namen des Servers für Ihren SData-URL ein.
    3. Geben Sie im Feld Port die Portnummer für Ihren SData-URL ein.
  4. Lassen Sie "ADMIN" im Feld Benutzername unverändert, oder geben Sie den Benutzernamen für die Infor CRM-Datenbank ein.

    Für den Benutzer mit Administratorrechten muss die Windows-Authentifizierung konfiguriert werden. Einzelheiten dazu finden Sie unter dem Thema "Verwenden der Windows-Authentifizierung" in der Administrator-Hilfe.

  5. Geben Sie im Feld Passwort das Passwort für den Benutzernamen ein
  6. Lassen Sie den Eintrag "Infor CRM" im Feld Anzeigename unverändert, oder geben Sie den Anzeigenamen an, der für den SData-Feed angezeigt werden soll.
  7. Klicken Sie auf die Schaltfläche Link testen, und zeigen Sie die Informationen im Feld Status an.
    Wenn der Feed nicht erfolgreich getestet wurde, prüfen Sie die Informationsfelder, nehmen Sie die erforderlichen Änderungen vor, und testen Sie den Link erneut.
  8. Klicken Sie auf Speichern.

Detailansicht-Registerkarten

Der untere Abschnitt enthält die Registerkarten der SData Sync-Konfigurationsansicht. Klicken Sie auf einen Hyperlink, um weitere Informationen zu jeder Registerkarte anzuzeigen.

Benutzerdefinierter Abschnitt

Sie können Registerkarten per Ziehen und Ablegen in den benutzerdefinierten mittleren Bereich verschieben. Die Registerkarten bleiben dort, bis sie wieder in den unteren Bereich verschoben werden.
