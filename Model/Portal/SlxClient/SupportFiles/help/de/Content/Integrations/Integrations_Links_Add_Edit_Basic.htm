Hinzufügen oder Bearbeiten eines Basic-Integrations-Links

Verwenden Sie das Dialogfeld "Link hinzufügen oder bearbeiten", um Integrationslinks hinzuzufügen oder zu bearbeiten. Wenn der Autorisierungstyp einer Integration "Basic" ist, verwenden Sie dieses Dialogfeld, um Links hinzuzufügen.

Vorgehensweise

Hinzufügen einer Verknüpfung
  1. Klicken Sie in der Detailansicht Integrationen auf die Registerkarte Links.

    Die Registerkarte "Links" ist nur für Integrationen mit einer Synchronisierungsmethode von Client-seitigem Sync oder Server Sync verfügbar.

  2. Klicken Sie auf die Schaltfläche Verknüpfung hinzufügen.
    Diese Schaltfläche ist nur verfügbar, wenn der Verknüpfungstyp der Integration "System" oder "Admin" ist.
  3. Geben Sie im Feld URL die SData-URL ein, die Infor CRM für die Freigabe von Informationen zwischen den Systemen verwendet.
    • Verwenden Sie eines der folgenden Formate:
      • Falls Ihre Implementierung nicht den standardmäßigen Port (Port 80) nutzt, verwenden Sie das folgende Format http://server:port/sdatavirtualdir/Anwendung/Vertrag/Datensatz/
      • Verwendet Ihre Implementierung jedoch Port 80, geben Sie die Port-Nummer nicht im URL an.
        Der Synchronisierungsdienst entfernt den Standard-Port aus dem URL, was zu Zuordnungsproblemen und Fehlern führt. Sie vermeiden dies durch Verwendung des folgenden Formats: http://server/sdatavirtualdir/application/contract/dataset/
    • Von Ihrem Integrationspartner bereitgestellte Ersatzwerte:
      •  Sdatavirtualdir - Beispiel: sdata.

      •  Anwendung - der Name Ihrer Buchhaltungsanwendung. Beispiel: Firma50.

      •  Vertrag - Der Name des Vertrags. Beispiel: GCRM.
      •  Datensatz - unterscheidet einen Datensatz oder ein aktives Unternehmen. Beispiel: /-/
  4. Geben Sie im Feld Anzeigename den Namen ein, den Sie anzeigen möchten, wenn Benutzer Datensätze zwischen Infor CRM und dem integrierten System verknüpfen.
  5. Geben Sie im Feld Benutzername den Namen ein, der für die Protokollierung im Integrationssystem verwendet wird.
  6. Geben Sie im Feld Passwort das entsprechende Passwort ein, das der Benutzer verwendet, um sich beim Integrationssystem anzumelden.
  7. Klicken Sie auf die Schaltfläche Link testen.
    Der Name der Schaltfläche ist abhängig vom Autorisierungs- und Verknüpfungstyp.
  8. Klicken Sie auf OK.
Bearbeiten einer Verknüpfung
  1. Klicken Sie in der Detailansicht Integrationen auf die Registerkarte Links.

    		2.

    Die Registerkarte "Links" ist nur für Integrationen mit einer Synchronisierungsmethode von Client-seitigem Sync oder Server Sync verfügbar.
  2. Klicken Sie auf den Hyperlink Bearbeiten für die Verknüpfung, die Sie bearbeiten möchten.
    Diese Verknüpfung ist nur verfügbar, wenn der Verknüpfungstyp der Integration "System" oder "Admin" ist.
  3. Nehmen Sie die erforderlichen Änderungen vor.
  4. Klicken Sie auf die Schaltfläche Link testen.
  5. Klicken Sie auf OK.
