Erstellen einer Kontaktsynchronisierungsgruppe

Zum Synchronisieren der Kontaktinformationen zwischen Infor CRM und Outlook muss die Gruppe als Kontaktsynchronisierungsgruppe festlegen. Falls die Gruppe nicht bereits vorhanden ist, müssen Sie zunächst die Kontaktsynchronisierungsgruppe erstellen.

So erstellen Sie eine Kontaktsynchronisierungsgruppe

  1. Öffnen Sie die Kontaktlistenansicht.
  2. Wählen Sie einen oder mehrere Datensätze aus, indem Sie beim Klicken auf die einzelnen Einträge die STRG- oder UMSCHALTTASTE drücken.
  3. Klicken Sie mit der rechten Maustaste in das Raster der Listenansicht und wählen Sie Datensätze als Gruppe speichern aus. 

    		4.

    Wenn Sie keine Datensätze auswählen, wird angefragt, ob alle Datensätze in der Gruppe verwendet werden sollen. Wenn Sie auf OK klicken, werden alle Gruppendatensätze ausgewählt.
  4. Geben Sie der Gruppe einen beschreibenden Namen, Beispiel SyncSalesLogix:

  5. Klicken Sie auf OK.

Verwandte Themen

Auswählen von Infor CRM-Kontakten zur Synchronisierung mit Outlook
