Registerkarte "Integrationen" in der Ansicht SData-Sync-Konfiguration

Verwenden Sie die Registerkarte "Integrationen", um eine Liste der Integrationen anzuzeigen, die Daten mithilfe von SData synchronisieren. Außerdem können Sie die Synchronisierung für eine Integration aktivieren oder deaktivieren oder alle Integrationen manuell gleichzeitig synchronisieren.

Vorgehensweise

Öffnen der Registerkarte "Integrationen"
  • Klicken Sie in der SData Sync-Konfigurationsansicht auf die Registerkarte Integrationen.
Synchronisierung für eine Integration aktivieren oder deaktivieren
  1. Aktivieren Sie das Kontrollkästchen Aktiviert neben einer Integration, um die Synchronisierung ein- oder auszuschalten.
    Markieren Sie das Kontrollkästchen, um die Synchronisierung zu aktivieren, oder entfernen Sie die Markierung im Kontrollkästchen, um die Synchronisierung einer Integration zu deaktivieren.
  2. Klicken Sie auf Speichern.
Alle Integrationen gleichzeitig synchronisieren
  • Klicken Sie auf die Schaltfläche Alle synchronisieren.
