Verwalten eines Schemas

Verwenden Sie das Dialogfeld "Schema verwalten", um das Datenbankschema in Infor CRM mit Ihrem Integrationsschema zu vergleichen und die Datenintegrität sicherzustellen. Sie können Daten verschiedener Ressourcen vergleichen, um festzustellen, ob Unterschiede bei der Länge von Zeichenfolgen zwischen den Systemen vorhanden sind.

Weitere Informationen dazu, wie sich Infor CRM bei Schemaunterschieden verhält...

Max. Länge – Wenn Daten mit Infor CRM synchronisiert werden, die die maximale Länge eines Felds überschreiten, wird die gesamte Zeile zurückgewiesen.

Nachkommastellen – Wenn Daten mit Infor CRM synchronisiert werden, die die Anzahl der Nachkommastellen überschreiten, wird der Wert auf die maximale Anzahl von Nachkommastellen gerundet. Wenn beispielsweise ein Währungsfeld als 17,4 definiert ist, wird der Wert 123,12348 als 123,1235 und der Wert 123,12343 als 123,1234 gespeichert.

Vorgehensweise

Öffnen des Dialogfeldes "Schema verwalten"
  • Klicken Sie in der Detailansicht Integration auf der Registerkarte Ressourcen auf die Schaltfläche Schema verwalten.
Schema für Ressourcen verwalten
  1. Wählen Sie im Dialogfeld Schema verwalten im Feld Filtern nach Ressource den Ressourcentyp aus.
  2. Klicken Sie auf Ausführen.
  3. Überprüfen Sie im Raster die Unterschiede.
    Prüfen Sie sämtliche Daten, die Integritätsprobleme verursachen könnten. Ein Feld im Buchhaltungssystem akzeptiert beispielsweise nur drei Zeichen, während dasselbe Feld in Infor CRM die freie Texteingabe unterstützt.
  4. Wiederholen Sie den Schritt für alle Ressourcentypen.
  5. Klicken Sie auf Schließen.
