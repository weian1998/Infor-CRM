Autorisieren von Services

Verwenden Sie die Optionsregisterkarte "Services autorisieren", um eine Verbindung mit einem externen Service herzustellen. Bevor Sie einen Service autorisieren können, muss Ihr Administrator die Integration konfigurieren und den Benutzern eine Zugriffsberechtigung erteilen. Diese Optionen gelten nur, wenn der Benutzer bei Infor CRM Web Client angemeldet ist.
  • Diese Funktion steht evtl. nicht allen Benutzern zur Verfügung. Externe Integrationen müssen von Ihrem Administrator durchgeführt werden.
  • Sie müssen Popups zulassen, um Services autorisieren zu können. Spezielle Informationen zum Zulassen von Popupfenstern finden Sie in der Browserhilfe.

So autorisieren Sie einen Service

  1. Klicken Sie im Menü Extras auf Optionen.
  2. Klicken Sie auf die Registerkarte Services autorisieren.
  3. Klicken Sie auf Autorisieren neben dem Service, den Sie autorisieren möchten.

  4. Befolgen Sie alle Eingabeaufforderungen zur Durchführung der Autorisierung.
