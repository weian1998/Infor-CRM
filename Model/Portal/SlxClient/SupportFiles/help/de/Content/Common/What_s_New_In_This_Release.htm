Neues in dieser Version

Infor CRM Web Client enthält folgende Neuerungen:

Version 8.1.0,06

  • Es wurden Änderungen vorgenommen, um das Rebranding des Produkts zu reflektieren.

Version 8.1

  • Aktivitäten lassen nun zu, dass mehrere Kontakte oder Interessenten mithilfe der Registerkarte Teilnehmer hinzugefügt werden können.
  • Die Möglichkeit, einen Kontakt einem ein Infor CRM-Benutzer zuzuordnen.
  •  Outlook-Synchronisierung  ersetzt Intellisync zur Synchronisierung von Kontakten und Aktivitäten mit Microsoft Outlook.
  • Die neue Ansicht "Sync-Historien-Liste" gibt Benutzern die Möglichkeit, den Verlauf der Integrations-Synchronisation zu prüfen, einschließlich des Verlaufs für Outlook Sync.
  • Die Möglichkeit, eine E-Mail-Nachricht in Outlook zu verfassen und mehrere Kontakte aus dem Infor CRM-Adressbuch auszuwählen.
  • Neue Cockpit-Widgets für Website, Web Feed und SData.
  • Reporting verwendet jetzt den Auftragsmanager.
  • In Excel exportieren verwendet jetzt den Auftrags-Manager.
  • Funktionen für Administratoren und Benutzer mit Administratorrechten:
    • Auftrags-Manager
    • -Integrationen
      •  
    • SData Sync-Management
    • Die Möglichkeit, einen Benutzer einem Kontakt zuzuordnen
