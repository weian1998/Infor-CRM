Hinzufügen und Bearbeiten von Berichtsbedingungen

Berichtsbedingungen geben Ihnen die Möglichkeit, den Inhalt eines vordefinierten Berichts zu modifizieren, um die nützlichsten Daten zu generieren. Die Ihnen zur Verfügung stehenden Optionen sind abhängig von der von Ihnen getroffenen Auswahl.

Abgleichen anhand von  
Datumsbereich

Wählen Sie einen der standardmäßigen Zeiträume aus der Auswahlliste aus, oder verwenden Sie die Option "Bestimmte Termine", um einen benutzerdefinierten Zeitraum auszuwählen.

Gruppe

Gruppen sind eine Zusammenstellung von Datensätzen. Sie können Gruppen verwenden, um mit einer Untergruppe von Datensätzen zu arbeiten.

Verwenden Sie die Suchfunktion, um eine Gruppe aus einer Liste von Gruppen auszuwählen.

Abfrage

Wählen Sie Folgendes aus der Auswahlliste aus:

  • Tabelle
  • Feld
  • Operator
  • Wert: Geben Sie entweder einen Wert ein, oder wählen Sie ein Datum aus.

Beispiel:

Tabelle: Verkaufschance

Feld: ERSTELLUNGSDATUM

Operator: Kommt nach

Datum: 1. Oktober.
Benutzer

Verwenden Sie die Suchfunktion, um einen Benutzer aus einer Liste von Benutzern auszuwählen.
