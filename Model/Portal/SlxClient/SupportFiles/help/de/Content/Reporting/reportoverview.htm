Was sind Berichte?

Die Listenansicht für "Berichte" enthält eine Liste der vordefinierten Beispielberichte, um Ihnen die Möglichkeit zu geben, Informationen in der Infor CRM-Datenbank anzuzeigen und zu analysieren. Sie können Berichte zu den Hauptansichten anzeigen, z. B. Kontakt, Firma, Verkaufschance usw. Auf Wunsch können Sie den Bericht an Ihre jeweiligen geschäftlichen Anforderungen anpassen. Die Ihnen jeweils zur Verfügung stehenden Berichte werden von Ihrem Systemadministrator festgelegt.

Infor CRM Web Reporting bietet folgende Funktionen:

  • Aus einer Liste Berichte auswählen. Sie können Filter anwenden, um die Ergebnisse einzugrenzen.
  • Planen einer einmaligen Ausführung eines Berichts zu einem späteren Zeitpunkt oder der Ausführung eines Berichts nach einem bestimmten Zeitplan
  • Zeigen Sie eine Liste der Berichte und den Zeitplan für ihre Ausführung an.
  • Anzeigen, Drucken und Speichern von Berichten.

So öffnen Sie die Ansicht "Berichte"

  • Klicken Sie auf der Navigationsleiste auf Berichte.

Zum Ausführen von Web Reporting im nicht verbundenen Zustand gilt: der Offline Web Client muss unter Window Server 2003 mit IIS ausgeführt werden.. Außerdem muss der Infor CRM Web Client Reporting-Server installiert und konfiguriert sein. Sind diese Voraussetzungen nicht erfüllt, können Benutzer nicht verbundener Web-Clients das Web Reporting nur ausführen, wenn sie mit dem Web Reporting-Serverhost verbunden sind. Die Berichte, die diese Benutzer angezeigt bekommen, stellen Daten aus der Hostdatenbank und nicht aus ihrer Remote-Datenbank dar.
