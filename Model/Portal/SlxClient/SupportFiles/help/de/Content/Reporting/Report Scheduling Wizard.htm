Berichtsplanungsassistent

Mit dem Berichtsplanungsassistenten können Sie Berichtsparameter ändern, um die Inhalte des ausgewählten Berichts an Ihre Anforderungen anzupassen.

So ändern Sie Parameter

  1. Markieren Sie im Dialogfeld Datensätze auswählen das Feld Datensätze anzeigen, die übereinstimmen, und wählen Sie eine der folgenden Möglichkeiten aus.
  • Alle: Umfasst sämtliche Datensätze für den ausgewählten Berichtstyp.
  • Spezifische Bedingungen: Damit können Sie Filterparameter zum Bericht hinzufügen.
  • Aktueller Benutzer: Alle Datensätze für den angemeldeten Benutzer werden aufgenommen.
  1. Entscheiden Sie in der Auswahlliste Abgleichen, falls verfügbar, ob Sie nach allen gewählten Parametern oder nach einem beliebigen der gewählten Parameter filtern möchten.
  • Alle bedeutet, dass nur die Datensätze zurückgegeben werden, die mit allen Parametern exakt übereinstimmen.
  • Beliebig bedeutet, dass alle Datensätze zurückgegeben werden, die mit irgendeinem Ihrer Filterparameter übereinstimmen.
  1. Klicken Sie in der Rasterüberschrift, falls verfügbar, auf das Plus-Symbol, um mit dem Dialogfeld Bedingungen hinzufügen/bearbeiten Filterparameter hinzuzufügen, und klicken Sie auf OK.
  2. Klicken Sie auf Weiter.
  3. Wählen Sie, falls verfügbar, im Dialogfeld Werte eingeben eine der verfügbaren Optionen aus, um den Auftragsbericht Ihren Anforderungen anzupassen.
  4. Im Dialog Exportoptionen/Export- und Planungsoptionen gehen Sie wie folgt vor:
    1. Wählen Sie im Feld Beschreibung den Standardtext oder geben Sie Ihren eigenen ein.
      Dieser Text erscheint auf der Registerkarte "Berichtsplanung" in der Spalte "Beschreibung".
    2. Wählen Sie das Dateiformat für Ihren Bericht aus. Die Optionen sind je nach Typ des Auftrags, den Sie ausführen, unterschiedlich. PDF, Word, Excel, CSV und XML können zur Auswahl stehen.
    3. (Nur Administratoren) Verwenden Sie das Feld Ausführen als, um die Berichte eines anderen Benutzers auszuführen. Verwenden Sie die Suchfunktion, um den Benutzer auszuwählen.
    4. Wenn Sie in Schritt 1 "Zeitplan" ausgewählt haben, können Sie einen Auftragszeitplan zum Ausführen Ihres Berichts erstellen. Treffen Sie Ihre Auswahl aus den verfügbaren Planungsoptionen.
    5. Klicken Sie auf Fertig stellen.
