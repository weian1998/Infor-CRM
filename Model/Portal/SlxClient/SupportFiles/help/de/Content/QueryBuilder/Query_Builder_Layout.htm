Layout des Abfrage-Assistenten

Auf der Registerkarte "Layout" des Abfrage-Assistenten können Sie angeben, welche Felder angezeigt werden und wie die Daten dargestellt werden. Sie können Felder im Layout hinzufügen, bearbeiten, löschen oder neu anordnen. Sie können auch einen Spaltennamen, die Spaltenbreite und ein Format für jedes Feld festlegen. Die Registerkarte "Layout" muss mindestens ein Feld enthalten, damit Daten angezeigt werden.

Der Abfrage-Assistent unterstützt eine Vielzahl von Funktionen. Die angezeigten Registerkarten sind von der verwendeten Funktion abhängig.
So fügen Sie ein Feld zum Layout hinzu
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Layout.

    Das Standardlayout für neue Gruppen wird von der Gruppe bestimmt, die Sie als Standardgruppe ausgewählt haben.

  3. Wählen Sie eine Tabelle aus.
  4. Wählen Sie ein Feld aus.

Sie können einen Bildlauf in der Layoutliste durchführen, um das Feld so anzuzeigen, wie es dem Layout hinzugefügt wird.

  1. Wählen Sie das Feld aus, und klicken Sie auf Bearbeiten.
  2. Füllen Sie das Dialogfeld Abfragelayout zuweisen aus.
So bearbeiten Sie ein Feld im Layout
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Layout.
  3. Wählen Sie ein Feld aus dem Raster aus.
  4. Klicken Sie auf Bearbeiten.
  5. Nehmen Sie im Dialog Abfragelayout zuweisen die notwendigen Änderungen vor.
So löschen Sie ein Feld aus dem Layout
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Layout.
  3. Wählen Sie ein Feld aus dem Raster aus.
  4. Klicken Sie auf Löschen.
So ordnen Sie Felder im Layout neu an
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Layout.
  3. Wählen Sie ein Feld aus. Führen Sie einen der folgenden Schritte durch:
  • Markieren Sie das Feld, und ziehen Sie es an die gewünschte Position.
  • Klicken Sie auf Nach links oder Nach rechts, bis sich das Feld an der richtigen Position befindet.
