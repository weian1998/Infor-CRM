Eigenschaften des Abfrage-Assistenten

Auf der Registerkarte "Eigenschaften" des Abfrage-Assistenten können Sie den und den einer Abfrage eingeben.

Der Abfrage-Assistent unterstützt eine Vielzahl von Funktionen. Die angezeigten Registerkarten sind von der verwendeten Funktion abhängig.

So legen Sie die Eigenschaften des Abfrage-Assistenten fest

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Eigenschaften.
  3. Geben Sie im Feld Name einen Namen für die Abfrage ein.

Wenn Sie eine Gruppe erstellen, erscheint der Name, mit dem Sie die Abfrage benennen, im Gruppen-Manager. 

  1. Geben Sie im Feld Anzeigename den Namen so ein, wie er angezeigt werden soll.

Wenn Sie eine Gruppe erstellen, wird der Name der neuen Abfrage auf der Registerkarte "Gruppe" in der Listenansicht angezeigt. Wenn Sie keinen Anzeigenamen eingeben, wird auf der Registerkarte der im Feld "Name" eingegebene Text angezeigt.

  1. Geben Sie im Feld Beschreibung eine Beschreibung ein, um den Zweck der Abfrage zu kennzeichnen.
