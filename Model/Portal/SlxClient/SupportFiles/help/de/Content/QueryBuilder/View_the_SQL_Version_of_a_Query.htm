Anzeigen der SQL-Version einer Abfrage

Wenn Sie eine Abfrage erstellen, wird diese in eine SQL-Anweisung (Structured Query Language) konvertiert. Diese Anweisung wird auf die Datenbank angewendet. Sie können die SQL-Version einer Abfrage anzeigen, die SQL-Anweisung jedoch nicht ändern.

Das Anzeigen der SQL-Anweisung für eine Abfrage kann Ihnen helfen zu verstehen, wie die Abfrage funktioniert. Mit dieser Option können Sie zudem einen funktionierenden SQL-Code anzeigen, wenn Sie SQL-Anweisungen für andere Teile des Programms schreiben.

So zeigen Sie die SQL-Version einer Abfrage an:

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf SQL-Code anzeigen.
  3. Schließen Sie das Dialogfeld, nachdem Sie die SQL-Anweisung angezeigt haben.
