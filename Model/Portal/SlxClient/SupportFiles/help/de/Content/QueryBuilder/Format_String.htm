Formatzeichenfolge

Mit einer Formatzeichenfolge können Sie die Daten in einer Abfrage formatieren. Wählen Sie zuerst den Formattyp aus. Geben Sie dann eine Formatzeichenfolge ein, um den ausgewählten Formattyp zu ändern. Welche Optionen gültig sind, hängt vom ausgewählten Formattyp ab. Eine Formatzeichenfolge muss mit einem %-Zeichen beginnen und mit einem Formattypzeichen enden:

[%] [Typ]

Bei Bedarf kann er ferner Folgendes beinhalten:

  • einen Argumentindex-Bezeichner
  • einen Linksbündig-Indikator
  • eine Breitenangabe
  • eine Genauigkeitsangabe

[%] [Index:] [-] [Breite] [.Genauigkeit] [Typ]

Zum Festlegen der Anzahl von Dezimalzahlen für eine Formatzeichenfolge setzen Sie die Formatzeichenfolge auf das Format, das Sie normalerweise verwenden. Fügen Sie anschließend einen Genauigkeitsanzeiger - ein Dezimalkomma gefolgt von der gewünschten Anzahl von Dezimalstellen - hinzu. Dieser muss vor dem Formatzeichenfolgenzeichen, aber nach dem %-Zeichen eingefügt werden. Beispiel:

  • Feldtyp: Gleiten
  • Formattyp: ftCurrency.
  • Formatzeichenfolge: %.4f
  • Daten (unformatiert): 3.1415926&ldots;
  • Ergebnis: 3.1416

Sie können die folgenden Formatzeichenfolgen verwenden:

  • Formatzeichenfolgen boolesche Felder
  • Formatzeichenfolgen Währungs- und Telefonnummernfeld
  • Formatzeichenfolgen Datum/Uhrzeit-Feld
  • Formatzeichenfolgen Festes Feld
  • Ganzzahl-Feldformatzeichenfolgen
