Was ist der Abfrage-Assistent?  

Mit dem Abfrage-Assistenten können Sie Daten anhand einer oder mehrerer Bedingungen in der Datenbank auswählen. Sie möchten beispielsweise eine Liste mit Hauptkontakten aus dem Bankwesen anzeigen. Der Abfrage-Assistent ist mit verschiedenen Bereichen der Infor CRM-Clients verbunden. Um den Abfrage-Assistenten auf effiziente Weise einsetzen zu können, müssen Sie die Infor CRM-Datenbankstruktur sowie die Konzepte relationaler Datenbanken wie Datenbeziehungen und Joins kennen.
