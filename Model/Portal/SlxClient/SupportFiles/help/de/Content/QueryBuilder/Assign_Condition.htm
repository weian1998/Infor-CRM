Zuweisen von Bedingungsanweisungen

Wenn Sie ein Feld in die Registerkarte "Bedingungen" des Abfrage-Assistenten aufnehmen, wird der Dialog "Bedingung zuweisen" für dieses Feld angezeigt.

So gehen Sie zum Zuweisen vor

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Bedingungen.
  3. Fügen Sie eine Bedingungsanweisung hinzu.

Das Dialogfeld "Bedingung zuweisen" wird angezeigt.

  1. Wählen Sie einen Operator aus.
  2. Geben Sie in das Feld Wert einen Wert ein oder klicken Sie auf Durchsuchen, um einen Wert auszuwählen.

Wenn Sie "In" als Operator wählen, müssen die Werte in Anführungszeichen eingeschlossen werden.

  1. Wählen Sie Groß-/Kleinschreibung berücksichtigender Vergleich aus, um die Großschreibung als Teil der Bedingung zu verwenden.

Wenn Sie Microsoft SQL Server verwenden, ist die Option Groß-/Kleinschreibung berücksichtigender Vergleich nicht verfügbar. Die Groß/Kleinschreibung spielt bei diesen Abfragen keine Rolle.

  1. Wählen Sie im Bedarfsfall Wert nicht für SQL formatieren.
  2. Klicken Sie auf OK.
