Bedingungen des Abfrage-Assistenten

Mit der Registerkarte "Bedingungen" des Abfrage-Assistenten können Sie Bedingungsanweisungen, die zum Auswählen von Daten verwendet werden, hinzufügen, bearbeiten oder löschen. Bedingungen werden in der Reihenfolge angewendet, in der sie in der Tabelle aufgeführt sind. Sie können die Reihenfolge jedoch mithilfe der Schaltflächen  Bild auf und Bild ab ändern. Bedingungsanweisungen können auch gruppiert angeordnet werden.
  • Der Abfrage-Assistent unterstützt eine Vielzahl von Funktionen. Die angezeigten Registerkarten sind von der verwendeten Funktion abhängig.
  • Alle Gruppen, die Sie erstellen, sollten mindestens eine Bedingung enthalten. Gruppen, die alle Datensätze enthalten, können evtl. die Systemleistung beeinträchtigen, und wenn Benutzer versuchen, diese Gruppen anzuzeigen, wird unter Umständen eine entsprechende Warnmeldung ausgegeben.
So fügen Sie eine Bedingungsanweisungen hinzu
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie im Abfrage-Assistenten auf die Registerkarte Bedingungen.
  3. Wählen Sie auf der linken Seite des Abfrage-Assistenten eine Tabelle aus. Klicken Sie bei Bedarf auf das Pluszeichen, um die Liste der Tabellen zu erweitern.
  4. Wählen Sie oben rechts im Abfrage-Assistenten ein Feld aus. Hier werden alle Felder der ausgewählten Tabelle angezeigt. Doppelklicken Sie auf das Feld, um es in die Tabelle unten im Dialog aufzunehmen.

Das Dialogfeld "Zuweisungsbedingung" wird automatisch geöffnet. Sie können diesen Dialog jederzeit öffnen, indem Sie auf eine der Anweisungen im Raster doppelklicken.

  1. Wählen Sie einen Operator aus.
  2. Geben Sie in das Feld Wert einen Wert ein oder klicken Sie auf Durchsuchen, um aus den vorhandenen Einträgen für dieses Feld einen Wert auszuwählen.

Wenn Sie "In" als Operator wählen, müssen die Werte in Anführungszeichen eingeschlossen werden.

  1. Wählen Sie Groß-/Kleinschreibung berücksichtigender Vergleich aus, wenn die Großbuchstaben des Wertes einen Teil der Bedingung darstellen sollen. Diese Option ist standardmäßig ausgewählt.

Wenn Sie MS SQL Server verwenden, ist die Option Groß-/Kleinschreibung berücksichtigender Vergleich nicht verfügbar. Die Groß/Kleinschreibung spielt bei diesen Abfragen keine Rolle.

  1. Wählen Sie im Bedarfsfall Wert nicht für SQL formatieren.
  2. Klicken Sie auf OK. 
  3. Um eine weitere Anweisung hinzuzufügen, wiederholen Sie die Schritte 1 bis 9 und wählen Sie anschließend die passende Verbindungsstelle aus.
So bearbeiten Sie eine Bedingungsanweisung
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Bedingungen.
  3. Wählen Sie eine Anweisung auf dem Raster aus (unten im Dialog "Abfrage-Assistent").
  4. Klicken Sie auf Bearbeiten.
  5. Nehmen Sie im Dialogfeld Bedingung zuweisen die notwendigen Änderungen vor.
  6. Klicken Sie auf OK. 
So löschen Sie eine Bedingungsanweisung
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Bedingungen.
  3. Wählen Sie eine Anweisung auf dem Raster aus.
  4. Klicken Sie auf Löschen.
