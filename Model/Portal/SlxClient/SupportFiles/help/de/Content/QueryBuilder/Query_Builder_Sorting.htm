Sortierung des Abfrage-Assistenten

Auf der Registerkarte "Sortieren" des Abfrage-Assistenten können Sie angeben, nach welchen Feldern und in welcher Reihenfolge (auf- oder absteigend) die Daten sortiert werden. Die Sortierkriterien werden in der Reihenfolge angewendet, die auf der Registerkarte angezeigt wird.

  • Der Abfrage-Assistent unterstützt eine Vielzahl von Funktionen. Die angezeigten Registerkarten sind von der verwendeten Funktion abhängig.

  • Einige Berichte enthalten vordefinierte Sortierkriterien, die bei Bedarf gelöscht werden können.
  • Felder vom Typ Blob können nicht zum Sortieren von Daten verwendet werden.

(Binary Large OBject) Ein Datentyp, der eine große, variable Menge von Daten enthält. Blobs können binäre Daten oder Textdaten enthalten. Mit dem Blob Viewer können Sie im Dialog "SQL-Ergebnismenge" Text-Blob-Daten anzeigen.

  • Numerische Formelfelder können ebenfalls nicht zum Sortieren von Daten verwendet werden, da die mathematische Berechnung erst abgeschlossen wird, nachdem die Sortierung erfolgt ist.
So fügen Sie Sortierkriterien hinzu
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Sortieren.
  3. Wählen Sie auf der linken Seite des Abfrage-Assistenten eine Tabelle aus. Klicken Sie bei Bedarf auf das Pluszeichen, um die Liste der Tabellen zu erweitern.
  4. Wählen Sie auf der rechten Seite des Abfrage-Assistenten ein Feld aus. Hier werden alle Felder der ausgewählten Tabelle angezeigt. Doppelklicken Sie auf das Feld, um es in die Tabelle unten im Dialog aufzunehmen.
  5. Klicken Sie auf den Dropdown-Pfeil neben Richtung und wählen Sie anschließend Aufsteigend (zur Sortierung von A bis Z) bzw. Absteigend (zur Sortierung von Z bis A).
  6. Um ein zweites Sortierfeld hinzuzufügen, wiederholen Sie die Schritte 3 bis 5. Jede nachfolgende Sortierung wird als untergeordnete Sortierung der vorherigen angewendet.

Numerische Formelfelder können nicht sortiert werden.

 

So ordnen Sie Sortierkriterien neu an
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Sortieren.
  3. Wählen Sie ein Feld aus dem Raster aus.
  4. Klicken Sie auf Nach oben oder auf Nach unten.
So löschen Sie Sortierkriterien
  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Sortieren.
  3. Wählen Sie im Raster das zu löschende Feld aus.
  4. Klicken Sie auf Löschen.
