Verwenden von "Not" in einer Bedingungsanweisung

Sie können "Nicht" (NOT) vor einer Bedingungsanweisung oder einer Gruppe von Bedingungsanweisungen hinzufügen. Wenn "Nicht" (NOT) vor einer Gruppe von Bedingungsanweisungen eingefügt wird, gilt der Operator für alle Bedingungsanweisungen in der Gruppe.

So fügen Sie "Nicht" (NOT) vor einer Bedingungsanweisung oder einer Gruppe von Bedingungsanweisungen ein:

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie auf die Registerkarte Bedingungen und fügen Sie alle Bedingungsanweisungen hinzu.
  3. Bei Bedarf gruppieren Sie die Bedingungsanweisungen.
  4. Aktivieren Sie das Kontrollkästchen Not für eine Bedingungsanweisung oder die erste Anweisung in einer Gruppe.
  5. Klicken Sie auf OK, um die Abfrage auszuführen.

Beispiel

Der Abfrage-Assistent würde alle Kontakte auswählen, die an anderen Orten in Kalifornien als Los Angeles oder San Francisco beheimatet sind.

Das Raster auf der Registerkarte "Bedingungen" wird in diesem Beispiel wie folgt angezeigt:

Not ( Feld Operator Wert Groß-/Kleinschr. ) And/Or
NOT ( Kontakt.Adresse.Stadt gleich Los Angeles Wahr   OR
    Kontakt.Adresse.Stadt gleich San Francisco Wahr ) AND
    Kontakt.Adresse.Bundesland/Kanton gleich CA Wahr   END

Eine Abfrage kann innerhalb einer Gruppe untergeordnete Gruppen enthalten. Wenn NOT vor eine Gruppe eingefügt wird, gilt dies auch für alle Untergruppen in der Gruppe.
