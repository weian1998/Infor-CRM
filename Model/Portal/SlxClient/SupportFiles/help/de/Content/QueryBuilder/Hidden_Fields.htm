Ausgeblendete Felder

Einige Felder werden im Datenbank-Manager als ausgeblendete Felder festgelegt. Ausgeblendete Felder können zwar angezeigt werden, sind aber standardmäßig im Abfrage-Assistenten nicht sichtbar.

So blenden Sie ausgeblendete Felder aus:

  • Klicken Sie im Abfrage-Assistenten auf Ausgeblendete Felder verbergen.

So blenden Sie ausgeblendete Felder ein:

  • Klicken Sie im Abfrage-Assistenten auf Ausgeblendete Felder einblenden.
    Diese Option ist nur verfügbar, nachdem ausgeblendete Felder verborgen wurden.
