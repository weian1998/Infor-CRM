Nachverfolgen eines Fehlers

Beim Nachverfolgen eines Fehlers werden die Schritte zur Behebung des Fehlers überprüft. Verwenden Sie die folgende Checkliste oder befolgen Sie die Unternehmensrichtlinien für das Nachverfolgen von Fehlern.

  • Muss ein Ticket mit diesem Fehler verknüpft werden?

Wenn ja, erstellen Sie eine Aufgabe für jeden durchgeführten Arbeitsschritt.

  • Muss die Zeit protokolliert werden, die für die Bearbeitung dieses Fehlers aufgewendet wird?

Wenn ja, erstellen Sie eine Aufgabe für jeden durchgeführten Arbeitsschritt.

  • Muss der Status des Tickets geändert werden (beispielsweise von "Offen" in "Wird ausgeführt")?

Wenn ja, ändern Sie das Feld Status in der Fehlerdetailansicht.

  • Muss für den Fehler eine Lösung hinzugefügt werden?

Wenn ja, können Sie auf der Registerkarte "Fehlerdetail" eine Lösung hinzufügen.
