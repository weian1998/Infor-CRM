Hinzufügen eines Fehlers

In der Ansicht "Neuen Fehler hinzufügen" können Sie einen Fehlerdatensatz erstellen. Nutzen Sie die Schaltfläche Kopieren, um schnell einen neuen Datensatz mit ähnlichen Informationen zu erstellen.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

So fügen Sie einen Fehler hinzu

  1. Öffnen Sie das Dialogfeld Neuen Fehler hinzufügen.
    • Klicken Sie im Menü Neu auf Fehler.
    • Klicken Sie in der Detailansicht Ticket zuerst auf die Registerkarte Fehler und anschließend auf die Schaltfläche Fehler anlegen.
  2. Füllen Sie die erforderlichen Fehlerinformationsfelder aus.
  3. Klicken Sie auf Speichern.
  4. Fügen Sie bei Bedarf über die Registerkarten der Fehlerdetailansicht Aufgaben für Fehler und beliebige dazugehörige Produkte, Retouren oder Tickets hinzu.
