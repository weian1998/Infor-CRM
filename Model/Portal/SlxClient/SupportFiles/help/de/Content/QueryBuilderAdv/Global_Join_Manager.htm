Global Join-Manager

 Nur für Systemadministratoren

  • Hierbei sollte jedoch sehr sorgfältig vorgegangen werden, da das Erstellen eines Global Join die Leistung des Systems beeinträchtigen kann.
  • Standardmäßig ist die Tabelle ADRESSE mit der Tabelle FIRMA über einen Inner Join verbunden. Die Eigenschaften dieses Join dürfen nicht geändert werden.

Der Global Join Manager zeigt alle Global Joins in der Infor CRM-Datenbank an und ermöglicht dem Systemadministrator oder einem Benutzer mit Administratorrechten, bei Bedarf Global Joins hinzuzufügen, zu bearbeiten oder zu löschen. Ein Global Join wird in der Verzeichnisstruktur des Abfrage-Assistenten von einer Session zur nächsten beibehalten.

Infor CRM enthält eine Reihe von bereits definierten Global Joins (so genannten System-Joins), die nicht gelöscht werden können. Diese vordefinierten Global Joins können von allen Benutzern verwendet werden, indem sie im Abfrage-Assistenten ein Feld auswählen und auf die Schaltfläche "Global Join" klicken. Wenn für dieses Feld bereits ein Join vorhanden ist, wird im Kontextmenü die Option "Global Join mit" mit dem angegebenen Tabellennamen zusammen angezeigt.

Der Global Join-Manager steht nur zur Verfügung, wenn Sie als Administrator angemeldet sind.

Vorgehensweise

Öffnen des Global Join-Managers
  1. Öffnen Sie den Abfrage-Assistenten.
  1. Öffnen Sie den Abfrage-Assistenten und klicken Sie anschließend auf die Schaltfläche Global Joins.

Diese Schaltfläche wird nur angezeigt, wenn Sie als Administrator angemeldet sind.

Der Global Join-Manager zeigt die folgenden Felder an:

Feld Beschreibung
Sys Ein System-Join, der nicht gelöscht werden kann.
1:1 Gibt eine 1:1-Beziehung an.
Übergeordnete Tabelle, übergeordnetes Feld Die Haupttabelle und das Feld eines Joins.
Untergeordnete Tabelle, untergeordnetes Feld Die mit der übergeordneten Tabelle und dem übergeordneten Feld verknüpfte Tabelle und das Feld.
Typ Der Join-Typ (Inner, Left, Right)
Kaskadieren Gibt an, wie Datensätze durch einen Join gelöscht werden.
Sichtbarkeit Gibt an, wie der Join im Abfrage-Assistenten angezeigt wird.
