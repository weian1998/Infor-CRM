Bearbeiten eines Global Join

 Nur für Systemadministratoren

Ein Global Join wird in der Verzeichnisstruktur des Abfrage-Assistenten von einer Session zur nächsten beibehalten. Mit vordefinierten Global Joins (System-Joins) können Sie nur die Optionen "Join-Typ", "Sichtbarkeit" und "Typ für abhängiges Löschen" bearbeiten.

So gehen Sie zum Bearbeiten vor

  1. Öffnen Sie den Abfrage-Assistenten und klicken Sie anschließend auf die Schaltfläche Global Joins.

Diese Schaltfläche wird nur angezeigt, wenn Sie als Administrator angemeldet sind.

  1. Wählen Sie im Dialog Global Join-Manager den zu bearbeitenden Join aus, und klicken Sie auf Bearbeiten.
  2. Ändern Sie die Parameter "Join-Typ", "Sichtbarkeit" oder "Kaskadierungstyp".
  3. Klicken Sie auf OK..
