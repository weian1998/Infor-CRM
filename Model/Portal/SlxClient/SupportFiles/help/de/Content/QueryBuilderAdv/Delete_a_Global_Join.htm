Löschen eines Global Join

 Nur für Systemadministratoren

Ein Global Join wird in der Verzeichnisstruktur des Abfrage-Assistenten von einer Session zur nächsten beibehalten. Sie können vordefinierte Global Joins (System-Joins) nicht löschen.

So löschen Sie einen Global Join

  1. Öffnen Sie den Abfrage-Assistenten und klicken Sie anschließend auf die Schaltfläche Global Joins.

Diese Schaltfläche wird nur angezeigt, wenn Sie als Administrator angemeldet sind.

  1. Wählen Sie den zu löschenden Join aus.
  2. Klicken Sie auf Löschen.
  3. Klicken Sie auf OK.
