Hinzufügen eines Local Join

Ein Local Join ist im Abfrage-Assistenten nur für die Dauer einer Session verfügbar. Das Objekt, das die resultierende Abfrage verwendet, speichert die Eigenschaften dieses Local Join nur für die Dauer der Verwendung.

So gehen Sie zum Hinzufügen vor

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Wählen Sie eine Tabelle aus.
  3. Wählen Sie ein Feld aus.
  4. Klicken Sie auf den Hyperlink Local Join erstellen.

Im Dialogfeld "Local Join" werden die im Abfrage-Assistenten gewählte Tabelle und das Feld im Abschnitt "Übergeordnete Tabelle" und "Join-Feld" angezeigt.

  1. Wählen Sie unter Untergeordnete Tabelle und untergeordnetes Join-Feld die Datenbanktabelle und das Feld aus, zu denen Sie den Local Join herstellen möchten.
  2. Wählen Sie unter Join-Typ den Typ des Joins aus.

Standardmäßig ist ein "Inner Join" ausgewählt.

  1. Klicken Sie auf OK.
