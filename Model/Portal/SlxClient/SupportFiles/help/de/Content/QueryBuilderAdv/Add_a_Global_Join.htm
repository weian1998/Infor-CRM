Hinzufügen eines Global Join

 Nur für Systemadministratoren

Ein Global Join wird in der Verzeichnisstruktur des Abfrage-Assistenten von einer Session zur nächsten beibehalten. Beim Erstellen von Global Joins muss sehr sorgfältig vorgegangen werden. Sie könnten sonst versehentlich einen Global Join erstellen, der die Leistung des Systems erheblich beeinträchtigt.

So gehen Sie zum Hinzufügen vor

  1. Öffnen Sie den Abfrage-Assistenten und klicken Sie anschließend auf die Schaltfläche Global Joins.

Diese Schaltfläche wird nur angezeigt, wenn Sie als Administrator angemeldet sind.

  1. Klicken Sie im Dialog Global Join-Manager auf Hinzufügen.
  2. Wählen Sie im Dialog Global Join hinzufügen unter Übergeordnete Tabelle und Join-Feld die Tabelle und das Feld für den Global Join aus.
  3. Wählen Sie unter Untergeordnete Tabelle und untergeordnetes Join-Feld die Tabelle und das Feld für die Verknüpfung mit der übergeordneten Tabelle und dem übergeordneten Feld aus.
  4. Wählen Sie unter Join-Typ den Typ des Joins aus.

Die Standardausrichtung ist links.

  1. Führen Sie im Bereich Join-Optionen folgende Schritte durch:
  1. Wählen Sie eine Sichtbarkeitsoption aus, um die Anzeige des Join im Abfrage-Assistenten zu definieren.

Der Standardeinstellung ist "Zulassen".

  1. Wählen Sie einen Kaskadierungstyp aus, um das Löschen der Datensätze durch den Join festzulegen.

Die Standardeinstellung ist "Ignore" (keine Kaskadierung).

  1. Klicken Sie auf OK.
