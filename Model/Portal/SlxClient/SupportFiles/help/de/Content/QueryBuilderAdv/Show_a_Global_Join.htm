Anzeigen eines Global Join

In Infor CRM sind bereits eine Reihe von Global Joins eingerichtet. Allerdings werden beim Öffnen des Abfrage-Assistenten aus Platzgründen nicht alle angezeigt. Wenn Sie den Abfrage-Assistenten zum Auswählen von Daten verwenden, und einer dieser Joins ist bereits vorhanden, verwenden Sie diesen und erstellen keinen neuen.

So fügen Sie einen Local Join hinzu

  1. Öffnen Sie den Abfrage-Assistenten.
  2. Klicken Sie bei Bedarf auf das Pluszeichen, um die Liste der Tabellen oben links im Abfrage-Assistenten zu erweitern. Wählen Sie eine Tabelle aus.
  3. Klicken Sie auf Global Join.

Das Dialogfeld "Global Join-Manager" wird geöffnet und zeigt die vorhandenen Global Joins an.

Wenn kein Global Join vorhanden ist, können Sie einen Local Join hinzufügen.
