Formel-Alias

Geben Sie einen Namen ein, der in der Berechnung verwendet wird. Dieser Name darf keine Leerzeichen oder Symbole enthalten. Wenn ein Formelfeld beim Erstellen eines Plug-Ins oder einer Abfrage verwendet wird, wird dieser Name in der Liste der verfügbaren Felder angezeigt. Sie müssen einen Wert in dieses Feld eingeben. Sie können das Erstellen von Formelfeldern nicht beenden, wenn dieses Feld keinen Wert enthält.
