Löschen einer Notiz

Sie können nur Notizen löschen, die Sie selbst erstellt haben. Sie müssen den gesamten Historieneintrag löschen, der mit einer Notiz verknüpft ist.

So löschen Sie eine Notiz

  1. Öffnen Sie die Detailansicht Kontakt, Interessent, Firma oder Interessenten- oder Verkaufschancen-Detail mit der entsprechenden Notiz.
  2. Klicken Sie auf die Registerkarte Notizen/Historie, und anschließend auf den Hyperlink Notiz zu der Notiz, die Sie öffnen möchten. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  3. Klicken Sie in der Ansicht Notizendetail auf die Schaltfläche Löschen.
  4. Klicken Sie auf OK, um den Datensatz unwiderruflich löschen, oder klicken Sie auf Abbrechen, um den Löschvorgang abzubrechen.
