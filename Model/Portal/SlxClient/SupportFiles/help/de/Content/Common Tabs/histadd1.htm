Hinzufügen von Historieneinträgen

Historieneinträge sind abgeschlossene Aktivitäten, E-Mails, Fragen und Datenbankänderungen.

So gehen Sie zum Hinzufügen vor

  1. Klicken Sie in einer der Detailansichten Kontakt, Firma, Interessent oder Verkaufschance auf die Notizen/Historie, Registerkarte und anschließend auf die Schaltfläche Aktivität abschließen.
  2. Sie können eine geplante Aktivität oder eine ungeplante Aktivität abschließen.
  3. Führen Sie die Schritte zum Abschließen einer Aktivität aus.
    Wenn Sie eine Folgeaktivität geplant haben, wird das Dialogfeld Aktivität planen angezeigt.
