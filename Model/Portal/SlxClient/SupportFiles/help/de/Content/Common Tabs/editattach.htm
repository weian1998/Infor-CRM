Bearbeiten von Anlageninformationen

In dieser Ansicht können Sie die Anlagenbeschreibung ändern und eine neue Anlagenversion hochladen. Anlagen werden synchronisiert.

Standardmäßig dürfen als Anlagen hinzugefügte Dateien maximal 4 MB groß sein. Sie erhalten eine Fehlermeldung, es sei denn, der Administrator hat Ihre Einstellungen angepasst.

So bearbeiten Sie Anlageninformationen

  1. Öffnen Sie die Registerkarte Anlagen.
  2. Klicken Sie auf die zu bearbeitende Anlage und anschließend auf Bearbeiten.
  3. Aktualisieren Sie bei Bedarf die Beschreibung.
  4. Führen Sie einen der folgenden Schritte aus:
    • Um eine andere Datei hochzuladen, klicken Sie auf Andere Datei auswählen, suchen nach der neuen Datei, die Sie hochladen möchten, und klicken dann auf Öffnen.

      Wenn Sie eine Anlage ändern, müssen Sie die neue Datei hochladen, damit die Änderungen in Infor CRM verfügbar sind. Änderungen, die Sie vornehmen, erfolgen an der lokalen Kopie der Datei und nicht an der angehängten Datei.

    • Zur Verknüpfung mit einem anderen URL geben Sie im Feld URL den neuen URL ein.

  5. Klicken Sie auf OK.
