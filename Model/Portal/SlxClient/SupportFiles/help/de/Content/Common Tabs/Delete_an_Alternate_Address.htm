Löschen einer alternativen Adresse

Adressen, die nicht mehr gültig sind, können gelöscht werden, solange es sich nicht um eine Liefer- oder Hauptadresse handelt.

Wenn Sie eine als Haupt- oder Lieferadresse gekennzeichnete vorhandene Adresse löschen möchten, müssen Sie zunächst eine andere Adresse erstellen (sofern keine vorhanden ist) und diese als neue Haupt- oder Lieferadresse kennzeichnen. Anschließend können Sie die vorherige Adresse löschen.

So gehen Sie beim Löschen vor

  1. Öffnen Sie die Kontakt- oder Firmendetailansicht.
  2. Klicken Sie auf die Registerkarte Adresse.
  3. Klicken Sie neben der Adresse, die Sie löschen möchten, auf Löschen.
  4. Klicken Sie auf OK.
