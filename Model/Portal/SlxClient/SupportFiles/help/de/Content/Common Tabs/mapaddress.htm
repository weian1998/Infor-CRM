Anzeigen von Lageplänen

Diese Funktion ist nur für die erste Zeile der Hauptadresse einer Firma, eines Kontakts oder eines Interessenten verfügbar. Die zweite Adresszeile wird nicht verwendet. Wenn # (Raute) in einer Adresse angezeigt wird, platzieren Sie das Zeichen in der zweiten Adresszeile, um die Ermittlung der Adresse durch MapQuest zu vereinfachen.

MapQuest kann u. U. nicht alle Adressen lokalisieren, oder Sie müssen ggf. die MapQuest-Ergebnisse näher betrachten, um die exakte Lage einer Adresse zu ermitteln.

So zeigen Sie einen Lageplan an

  • Klicken Sie in der Detailansicht Kontakt, Interessent oder Firma auf die Schaltfläche MapQuest.

Eine weitere Instanz des Web-Browsers wird geöffnet. Auf der MapQuest-Site wird ein Lageplan zur aufgelisteten Adresse angezeigt. Die Adresse wird automatisch eingegeben.
