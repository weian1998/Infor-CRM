Löschen von Anlagen

Sie können Anlagen für einen Datensatz löschen.

Wenn Sie eine Anlage aus einem einzelnen Vorkommen einer sich wiederholenden Aktivität löschen, wird die Anlage aus allen Instanzen der Aktivität entfernt.

So gehen Sie beim Löschen vor

  1. Öffnen Sie die Registerkarte Anlagen.
  2. Wählen Sie den Datensatz aus, den Sie entfernen möchten, und klicken Sie auf die Schaltfläche Löschen.

  3. Klicken Sie auf OK.
