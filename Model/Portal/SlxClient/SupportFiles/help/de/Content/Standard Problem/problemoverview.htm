Was sind häufig auftretende Probleme und Standardlösungen?

Ein häufig auftretendes Problem ist ein Fehler in Ihrem Produkt, der von mehreren Kunden gemeldet wird. Ihr Callcenter erhält ggf. Anrufe zu Problemen, die ähnlich gelagert sind. Über die Funktion "Häufig auftretende Probleme" können Sie Beschreibungen dieser Probleme aufzeichnen. Über die Funktion "Standardlösungen" können Sie bewährte Methoden zur Problembehebung speichern. Nachdem Informationen zu häufig auftretenden Problemen und dazugehörige Lösungen eingerichtet wurden, können Sie diese zur Ergänzung einer Ticket-Beschreibung oder als Ticket-Lösung verwenden.

Ein häufig auftretendes Problem kann mit einer Standardlösung oder mehreren Lösungen verknüpft werden. Darüber hinaus kann ein Standardlösung mit mehreren Problemen verknüpft werden. Für ein Druckproblem kann es z. B. drei mögliche Lösungen geben. Ebenso kann eine Lösung eines Fehlers für mehrere Probleme in Frage kommen.
