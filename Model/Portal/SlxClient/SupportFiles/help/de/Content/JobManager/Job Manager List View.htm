Listenansicht für Auftrags-Manager

Der Auftrags-Manager liefert eine Benutzerschnittstelle zum Erstellen und Verwalten der Aufträge, die zum Infor CRM-Auftragsserver gesendet werden. Der Auftragsserver ist eine Engine zum Bearbeiten einzelner und/oder wiederkehrender Aufgaben zur sofortigen oder verzögerten Ausführung. Der Auftragsserver ist ein freigegebener Dienst, der von jedem adaptiven Infor CRM-Client eingesetzt werden kann (Web, Windows und Mobile eingeschlossen). Es gibt 9 Threads zum Auftragsserver, sodass mehrere Aufträge gleichzeitig ausgeführt werden können.

Benutzer mit Administratorberechtigungen können auf die Schnittstelle des Auftragsmanagers zugreifen, jedoch senden alle Benutzer Aufträge, die vom Auftragsserver bearbeitet werden, wenn sie eine der folgenden vordefinierten Auftragstypen durchführen:

  • Ausführen oder Planen von Berichten
  • Dateien in Excel exportieren
  • Ausdehnen nicht abgeschlossener Aktivitäten
  • Aktualisieren der Restdauer in Tagen für aktive Verträge
  • Aktualisieren von Verkaufschancen
  • Aktualisieren oder Löschen von Interessenten mithilfe des Aufgabenbereichs.

 

Administratoren verwenden den Auftragsmanager für die folgenden Hauptfunktionen:

  • Klicken Sie auf die Registerkarte Ausführungen, um den Status von derzeitig in Bearbeitung befindlichen Aufträgen einzusehen und/oder abzubrechen.
  • Klicken Sie auf die Registerkarte Definitionen, um eine Liste der vordefinierten Auftragstypen einzusehen. Wählen Sie einen Auftrag aus, und klicken Sie mit der rechten Maustaste auf den Auftragsmanager-Assistenten, wo Sie einen Zeitplan ändern oder einen neuen Zeitplan für den Auftrag hinzufügen können.
  • Klicken Sie auf die Registerkarte Zeitplan, um eine Liste der Aufträge einzusehen, die für die Ausführung nach einem bestimmten Zeitplan festgelegt wurden. Wählen Sie einen vorhandenen Auftrag aus, und klicken Sie mit der rechten Maustaste auf den Auftragsmanager-Assistenten, wo Sie einen Zeitplan ändern können. Außerdem können Sie einen vorhandenen Auftrag löschen.

 

Verwandte Themen.

Verwenden der Registerkarte "Zeitpläne" zum Modifizieren und/oder Löschen von Auftragszeitplänen

Verwenden der Registerkarte Definitionen zum Auswählen aus vordefinierten Aufträgen

Verwenden der Registerkarte "Ausführungen" zum Anzeigen und Abbrechen von derzeit in Bearbeitung befindlichen Aufträgen.
