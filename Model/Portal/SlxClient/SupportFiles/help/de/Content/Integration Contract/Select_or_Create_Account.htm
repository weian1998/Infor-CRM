Auswählen oder Erstellen einer Firma

(Nur Buchhaltungsintegration) Auf dem Bildschirm "Firma auswählen oder erstellen" werden die Kunden im Buchhaltungssystem angezeigt, die eine mögliche Übereinstimmung mit der Firma darstellen, mit der Sie in Infor CRM eine Verknüpfung herstellen. Sie können für die Verknüpfung einen vorhandenen Kunden auswählen, einen neuen Kundendatensatz erstellen oder Ihre Suchkriterien verbessern und eine neue Suche starten.

So gehen Sie beim Auswählen vor

  1. Öffnen Sie den Assistent "Verknüpfung zur Buchhaltung" und durchlaufen Sie die vorherigen Bildschirme.
  1. Wählen Sie auf dem Bildschirm Konto auswählen oder erstellen eine der folgenden Optionen aus:
    FeldBeschreibung/AktionNächster Schritt
    Mit [Firmenname] verknüpfenZeigt Kundendatensätze im Buchhaltungssystem, die eine mögliche Übereinstimmung mit der Firma darstellen, mit der Sie eine Verknüpfung herstellen. Wählen Sie diese Option und das Kundenkonto aus, mit dem eine Verknüpfung hergestellt werden soll. Wenn die Informationen im Firmen- und Kundendatensatz nicht exakt übereinstimmen, müssen Sie die Daten zusammenführen. Daten zusammenführen
    Für [Firmenname] einen neuen Kundendatensatz erstellenWählen Sie diese Option, um mithilfe des Infor CRM-Firmendatensatzes einen Kundendatensatz im Buchhaltungssystem zu erstellen.

    Im Buchhaltungssystem wird ein neues Kundenkonto angelegt.

    Klicken Sie auf Schließen, um den Dialog zu schließen.

    Verfeinern oder erweitern Sie die Suche nach einem bestehenden Konto

    Wählen Sie diese Option aus, wenn Sie einen vorhandenen Kundendatensatz im Buchhaltungssystem nicht finden und Sie die Suchkriterien ändern möchten, um anschließend eine neue Suche zu starten.

    Diese Option steht nur zur Verfügung, wenn die Filterfunktion für das Buchhaltungssystem implementiert ist.

    		Suchen nach Übereinstimmungen
  2. Klicken Sie auf Weiter, um zum nächsten Schritt zu gelangen.
