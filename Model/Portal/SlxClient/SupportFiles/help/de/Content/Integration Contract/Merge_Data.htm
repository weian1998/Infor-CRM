Zusammenführen von Daten

 (Nur Buchhaltungsintegration) Auf dem Bildschirm "Daten zusammenführen" werden Unterschiede zwischen den Daten angezeigt, wenn Sie Datensätze zwischen Infor CRM und einem Buchhaltungssystem verknüpfen. Auf diesem Bildschirm können Sie die Daten auswählen, die Sie beim Zusammenführen der Datensätze verwenden möchten. Die Daten, die Sie beibehalten, überschreiben die Daten im anderen System. Weitere Informationen...

  • Wenn die Buchhaltungsintegration nicht aktiviert ist und Sie zwei Firmen zusammenführen, werden der gelöschten Firma zugeordnete Verkaufsaufträge dem zusammengeführten Konto hinzugefügt.
  • Wenn die Buchhaltungsintegration jedoch aktiviert ist und Sie zwei Firmen zusammenführen, kann eine Firma, der Verkaufsaufträge oder Angebote zugeordnet sind, die an das Buchhaltungssystem übermittelt wurden, nicht gelöscht werden (die Firma, die die übermittelten Datensätze enthält, kann nicht die Quellfirma sein). Bei Bedarf können Sie die Quellfirma bestimmen, indem Sie diese Firma beim Zusammenführen von Daten zuerst auswählen. Dadurch können Sie die Firma mit zugeordneten Verkaufsaufträgen beibehalten, indem die Zusammenführung mit ihr erfolgt.

So gehen Sie beim Auswählen vor

  1. Öffnen Sie den Verknüpfungsassistenten und durchlaufen Sie die vorherigen Bildschirme.
  1. Füllen Sie den Bildschirm Daten zusammenführen aus.
    FeldBeschreibung/Aktion
    Eigenschaft

    Listet die Datenfelder auf, die für die Firma in beiden Datensätzen in Infor CRM vorhanden sind.

    [Quelle] Spalte

    Verwenden Sie die Spalte links neben der Spalte "Quelle" zum Bestimmen auf Feldbasis, welche Daten im Infor CRM-Firmendatensatz (Quelle) beibehalten werden sollen, wenn die Datensätze zusammengeführt werden.

    Wählen Sie in der Spaltenüberschrift die Option "Alle" aus, um alle Infor CRM-Daten (Quelle) beizubehalten.

    [Ziel] Spalte

    Verwenden Sie die Spalte links neben der Spalte "Ziel" zum Bestimmen auf Feldbasis, welche Daten im Buchhaltungssystem (Ziel) beibehalten werden sollen, wenn die Datensätze zusammengeführt werden.

    Wählen Sie in der Spaltenüberschrift die Option "Alle" aus, um alle Daten im Buchhaltungssystem beizubehalten.

    Duplikate ausblenden oder Alle anzeigenKlicken Sie, um die Eigenschaftsinformationen ein- oder auszublenden, die in beiden Systemen gleich sind. Standardmäßig enthält die Liste nur Felder mit unterschiedlichen Werten.
  2. Klicken Sie auf Weiter, um zum nächsten Schritt zu gelangen.
    Abhängig von Ihrer Auswahl müssen Sie möglicherweise Kontakte zusammenführen.
