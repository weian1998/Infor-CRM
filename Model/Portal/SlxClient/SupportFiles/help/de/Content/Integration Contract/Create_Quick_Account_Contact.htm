Erstellen einer neuen Firma anhand eines Verkaufsauftrags

Sie können einen Verkaufsauftrag erstellen, bevor Sie eine Firma oder einen Kontakt erstellen. Wenn der Interessent in Geschäftsbeziehungen mit Ihrem Unternehmen treten möchte, können Sie anhand des Verkaufsauftrags eine neue Firma und einen neuen Kontakt mit eingeschränkten Unternehmens- und Kontaktinformationen anlegen.

So gehen Sie zum Erstellen vor

  1. Erstellen Sie einen neuen Verkaufsauftrag.
  2. Klicken Sie auf die Schaltfläche Neue Firma hinzufügen.
  3. Vervollständigen Sie im Dialogfeld Firmenkontakt schnell einfügen folgende Aufgaben:
    1. Geben Sie in das Feld Firma einen Firmennamen ein.
    2. Wählen Sie im Feld Typ den Firmentyp aus.
    3. Geben Sie in das Feld Telefon die Telefonnummer der Hauptniederlassung ein.
    4. Geben Sie in das Feld Adresse die Geschäftsadresse der Firma ein.
    5. Geben Sie in das Feld Kontakt einen Kontaktnamen ein.
    6. Geben Sie in das Titel den Titel des Kontakts ein.
    7. Geben Sie im Feld Telefon die Telefonnummer des Kontakts ein.
    8. Geben Sie in das Feld Mobil die Mobiltelefonnummer des Kontakts ein.
    9. Geben Sie in das Feld E-Mail die E-Mail-Adresse des Kontakts ein.
    10. Klicken Sie auf OK.
  4. Klicken Sie auf Speichern.
