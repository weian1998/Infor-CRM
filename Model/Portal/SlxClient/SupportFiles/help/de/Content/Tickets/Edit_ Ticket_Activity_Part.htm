Bearbeiten von Ticket-Aktivitätenteilen

In dieser Ansicht können Sie Teile in Relation zur Ticket-Aktivität verwalten.

So gehen Sie zum Bearbeiten vor

  1. Führen Sie einen der folgenden Schritte durch:
    • Öffnen Sie die Registerkarte Ticket-Aktivität (Teile), und klicken Sie neben dem zu bearbeitenden Teil auf Teil bearbeiten.
      •  
    •  Öffnen Sie die Ticket-Aktivitätsansicht, und klicken Sie im Bereich Ticket-Aktivität (Teile) neben dem zu bearbeitenden Teil auf Bearbeiten.
      • Das Dialogfeld "Ticket-Aktivität (Teile)" wird geöffnet.
  2. Geben Sie in das Feld Seriennummer die Seriennummer des Teils ein.
  3. Geben Sie in das Feld Preis den Preis des Teils ein.
  4. Geben Sie in das Feld Beschreibung eine kurze Beschreibung des Teils ein.
  5. Klicken Sie auf OK.
