Senden einer E-Mail-Nachricht aus einem Ticket

Sie können eine E-Mail-Nachricht an einen Kunden oder anderen Infor CRM-Benutzer senden, während Sie ein neues Ticket hinzufügen oder in der Ticket-Detailansicht arbeiten. Die E-Mail-Nachricht kann Ticket-Informationen zur Verfügung stellen, Dritte einschließlich dem Kunden über das Öffnen des Tickets benachrichtigen oder Fragen zur Problemlösung stellen.

So senden Sie eine E-Mail-Nachricht aus einem Ticket

  1. Klicken Sie auf der Navigationsleiste auf Tickets.
  2. Verwenden Sie die Suchfunktion in der Listenansicht Tickets, suchen Sie mit Suche nach einem Ticket, und öffnen Sie die Detailansicht des Tickets.
  3. Klicken Sie in der Symbolleiste der Ansicht Ticket-Detail auf die Schaltfläche Details als E-Mail senden.
  4. Klicken Sie im Dialog Ticket-E-Mail senden unter An auf eine oder mehrere Empfängeroptionen.
  5. Wählen Sie unter E-Mail-Typ die Art der zu übertragenden Informationen aus. Mögliche Optionen:
    OptionAktion
    TicketinformationenDie Nachricht enthält das Empfangsdatum, die Beschreibung, die Lösung (falls verfügbar), ggf. vorhandene interne Kommentare und die Ticket-ID.
    KeineDie Ticket-ID wird in die Betreffzeile der E-Mail-Nachricht eingegeben. Die Nachricht enthält sonst keine weiteren Ticket-Informationen.

    Interne Kommentare stehen nur anderen Infor CRM-Benutzern zur Verfügung. Ist einer der E-Mail-Empfänger ein Kontakt, werden die internen Kommentare nicht in die E-Mail eingeschlossen.

  6. Klicken Sie auf OK.
  7. Zum Versenden Ihrer E-Mail führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf die Schaltfläche Senden. Die Nachricht wird nicht in der Historie aufgezeichnet.
    • Wenn Sie Microsoft Outlook verwenden, klicken Sie auf die Schaltfläche Senden an CRM. Wenn Sie eine Datei angehängt haben und diese als Anlage speichern wollen, klicken Sie in der Eingabeaufforderung auf Ja. Die E-Mail-Nachricht wird in der Historie aufgezeichnet und die Datei wird auf der Registerkarte Anlagen angezeigt.
