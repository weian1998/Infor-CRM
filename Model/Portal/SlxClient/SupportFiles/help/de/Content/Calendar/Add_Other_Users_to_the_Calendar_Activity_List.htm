Bearbeiten der Kalenderbenutzerliste

Bei Verwendung des Infor CRM-Kalenders und abhängig von den vom Systemadministrator festgelegten Zugriffsrechten können Sie die Aktivitäten und Ereignisse anderer Benutzer anzeigen und/oder ändern.

So fügen Sie Kalenderbenutzer aus der Liste hinzu oder entfernen sie daraus

  1. Klicken Sie in der Kalender-Ansicht im Aufgabenbereich unter Kalenderbenutzer auf Kalenderbenutzer bearbeiten.
    Das Dialogfeld "Kalenderbenutzer bearbeiten" wird geöffnet.
  2. Geben Sie ggf. die ersten Buchstaben eines Benutzernamens in das Feld Benutzer suchen ein, und klicken Sie auf Suchen, um einen Benutzer zu suchen.
    Ihr Kalendersicherheitsgrad bestimmt, wessen Kalender Sie anzeigen können.
  3. Aktivieren Sie die Kontrollkästchen der Benutzer, die Sie in der Liste Kalenderbenutzer  anzeigen möchten, und entfernen Sie die Markierungen in den Kontrollkästchen der Benutzer, die Sie ausblenden möchten. Ihr Name wird standardmäßig ausgewählt und kann nicht gelöscht werden.
    Sie können bis zu 25 Benutzer (einschließlich Ihrer eigenen Person) für Ihre Liste auswählen. Sie dürfen jedoch nur bis zu vier Kalender gleichzeitig anzeigen.
  4. Klicken Sie auf OK. 
    Die Liste "Kalenderbenutzer" enthält nur die ausgewählten Benutzer.
