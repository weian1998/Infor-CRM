Hinzufügen oder Bearbeiten eines verkauften Artikels

Sie können einer Firma einen neuen verkauften Artikel hinzufügen oder die Liste der mit einer Firma verknüpften verkauften Artikel bearbeiten.

Durch Ihren Sicherheitszugriff wird bestimmt, welche Funktionen verfügbar sind. Wenden Sie sich an den Administrator, wenn Sie Ihre Zugriffsrechte ändern möchten. Für WebViewer-Benutzer kann die Funktionalität eingeschränkt sein.

So gehen Sie zum Hinzufügen vor

  1. Öffnen Sie die Firmendetailansicht.
  2. Klicken Sie auf die Registerkarte Verkaufte Artikel. Falls die Registerkarte nicht angezeigt wird, klicken Sie auf die Registerkarte Weitere Registerkarten.
  3. Klicken Sie auf die Schaltfläche Artikel hinzufügen.  
     
  4. Füllen Sie die erforderlichen Informationsfelder zu verkauften Artikeln aus.
  5. Klicken Sie auf OK. 
     Der Artikel wird in der Produktliste dieser Firma angezeigt.

So gehen Sie zum Bearbeiten vor

  1. Klicken Sie in der Ansicht Firmendetail auf die Registerkarte Verkaufte Artikel.
  2. Führen Sie auf der Registerkarte Verkaufte Artikel einen der folgenden Schritte aus:
    • Klicken Sie auf Bearbeiten, bearbeiten Sie die Informationsfelder zu verkauften Artikeln, und klicken Sie auf OK.

    • Wählen Sie den Datensatz aus, den Sie bearbeiten möchten, nehmen Sie Ihre Änderungen im Raster vor, und klicken Sie auf Speichern. Um Ihre Änderungen rückgängig zu machen, klicken Sie auf Zurücksetzen, bevor Sie auf Speichern klicken.

      Es können nicht alle Spalten im Raster bearbeitet werden. Zum Bestimmen, ob eine Spalte bearbeitet werden kann, klicken Sie auf das Raster. Wenn ein weißes Feld angezeigt wird, kann die Spalte bearbeitet werden.
