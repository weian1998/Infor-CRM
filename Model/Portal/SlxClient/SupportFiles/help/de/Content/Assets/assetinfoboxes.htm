Informationsfelder zu verkauften Artikeln

In der nachfolgenden Tabelle werden die Standardinformationsfelder beschrieben.

Feld Beschreibung/Aktion
Name Der Name des verkauften Artikels. Klicken Sie auf die Schaltfläche Suchen, und nehmen Sie die Bearbeitung mithilfe der Suchfunktion vor.
SKU Die Lagerhaltungseinheit (SKU = Stock Keeping Unit).
Menge Die Einheiten an diesem verkauften Artikel, die im Besitz der Firma sind.
Seriennummer Eine eindeutige Seriennummer für den verkauften Artikel.
Kontakt Der Name des Kontakts. Klicken Sie auf die Schaltfläche Suchen, und nehmen Sie die Bearbeitung mithilfe der Suchfunktion vor.
Telefon Zeigt die Telefonnummer des Kontakts an.
E-Mail Zeigt die E-Mail-Adresse des Kontakts an.

Position

Verkaufter Artikel - Code

Umgebung

System

 In diese Felder können Sie zusätzliche Informationen speziell für Ihr Unternehmen eingeben.
Version Die Version des verkauften Artikels.
Letzte Aktualisierung

Der Tag, an dem der verkaufte Artikel hinzugefügt oder zuletzt bearbeitet wurde.
Aktualisiert von

Der Benutzer, der den verkauften Artikel hinzugefügt oder zuletzt bearbeitet hat.
Beschreibung Beschreibung des verkauften Artikels.
Zurückgegeben Aktivieren Sie dieses Kontrollkästchen, wenn der verkaufte Artikel zurückgegeben wurde. Eventuell steht dieses Kontrollkästchen nicht zur Verfügung.
Bewertung Aktivieren Sie dieses Kontrollkästchen, wenn der verkaufte Artikel ein Bewertungsartikel ist.
Verkaufschance Zeigt die zugehörige Verkaufschance an (sofern zutreffend).
Verkaufsauftrag Zeigt den zugehörigen Verkaufsauftrag an (sofern zutreffend).
Kommentare In dieses Feld können weitere Informationen zum verkauften Artikel eingegeben werden.

Verwandte Themen

Firma - Hinzufügen eines verkauften Artikels

Firma - Bearbeiten eines verkauften Artikels
