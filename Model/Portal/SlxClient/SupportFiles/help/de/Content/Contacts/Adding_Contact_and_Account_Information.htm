Hinzufügen von Kontakt- und Firmeninformationen

In der Ansicht "Kontakt-/Firmeninformationen" können Sie einen neuen Kontakt oder eine neue Firma hinzufügen oder einer bestehenden Firma einen neuen Kontakt hinzufügen.

So öffnen Sie die Ansicht "Kontakt-/Firmeninformationen"

  • Klicken Sie im Menü Neu auf Firma/Kontakt.

So fügen Sie einer vorhandenen Firma einen Kontakt hinzu

So fügen Sie einen neuen Kontakt und eine neue Firma hinzu

So fügen Sie eine neue Firma hinzu
