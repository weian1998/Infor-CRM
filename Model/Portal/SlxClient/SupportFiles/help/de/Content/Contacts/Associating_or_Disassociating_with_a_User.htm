Zuordnen zu einem Benutzer oder Aufheben der Zuordnung

Wenn Ihre Infor CRM-Implementierung das Erstellen eines Kontakts für jeden Infor CRM-Benutzer beinhaltet, können Sie den Benutzer und Kontakt problemlos aus der Listenansicht "Kontakte" zuordnen.

Vorgehensweise

Zuordnen eines Benutzers zu einem Kontakt
  1. Klicken Sie auf der Navigationsleiste auf Kontakte.
  2. Wählen Sie den Kontakt aus, den Sie einem Benutzer zuordnen möchten.
  3. Klicken Sie im Aufgabenbereich unter Benutzeraufgaben auf Zu Benutzer zuordnen.
  4. Klicken Sie im Dialogfeld Kontakt zu Benutzer zuordnen auf Benutzer Schaltfläche  Suchen  .
  5. Verwenden Sie die Suchfunktion, um den Benutzer zu suchen, den Sie dem Kontakt zuordnen möchten, und klicken Sie dann auf OK.
  6. Klicken Sie auf OK.
So heben Sie die Zuordnung eines Benutzers zu einem Kontakt auf
  1. Klicken Sie auf der Navigationsleiste auf Kontakte.
  2. Wählen Sie einen oder mehrere Kontakte aus, deren Zuordnung zu Benutzern Sie aufheben möchten.
  3. Klicken Sie im Aufgabenbereich unter Benutzeraufgaben auf Zuordnung zu Benutzer aufheben.
  4. Klicken Sie auf OK.
