Auswählen einer alternativen Adresse

Über den Dialog "Adresse auswählen" können Sie eine vorhandene Adresse für einen neuen Kontakt auswählen. Die Schaltfläche Weitere Adresse auswählen ist nur verfügbar, wenn vorhandene Adressen existieren. Wenn Sie beispielsweise einen Kontakt zu einer Firma mit mehreren Adressen hinzufügen, sind die Adressen der Firma verfügbar.

So wählen Sie eine alternative Adresse aus

  1. Klicken Sie im Dialog Kontakt-/Firmeninformationen auf die Schaltfläche Weitere Adresse auswählen, falls verfügbar.

Der Dialog "Adresse auswählen" wird geöffnet.

  1. Klicken Sie auf die Adresse, die Sie verwenden möchten.
  2. Klicken Sie auf Speichern.

Verwandte Themen

Verwalten alternativer Adressen für einen Kontakt oder eine Firma

Hinzufügen von Kontakt- und Firmeninformationen
