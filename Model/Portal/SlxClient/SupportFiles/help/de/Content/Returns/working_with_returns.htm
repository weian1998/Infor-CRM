Arbeiten mit Retouren

Als Retoure wird ein vom Kunden zurückgesendetes Produkt bezeichnet, das repariert oder umgetauscht werden oder für das eine Rückerstattung oder Gutschrift erfolgen soll. In der Retourendetailansicht können Sie mit einer einzelnen Retoure und in der Retourenlistenansicht mit Retouren im Tabellenformat arbeiten.

Im Folgenden finden Sie einige Funktionen, die Ihnen für das Verwalten von Retouren zur Verfügung stehen:

  • In der Retourendetailansicht können Sie Informationen in einem Retourendatensatz hinzufügen oder bearbeiten.
  • In der Retourenlistenansicht können Sie Retourengruppen anzeigen und verwalten.
