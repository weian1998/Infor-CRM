Ändern eines Passwortes (Administrator)

Sie können das Passwort eines Benutzers in der Ansicht "Benutzer" oder über Benutzeroptionen ändern.

Weitere Informationen...

Neuen Benutzern wird beim Erstellen das Standardpasswort password zugeteilt. Dieses Passwort kann jederzeit vom Systemadministrator, vom Verwaltungsbenutzer oder vom Infor CRM-Benutzer geändert werden.
  • Passwörter müssen mit einem numerischen Zeichen (0-9) oder einem Buchstaben von A bis Z bzw. a-z beginnen und enden.
  • Ein Semikolon (;) ist kein gültiges Passwortzeichen.

So ändern Sie ein Passwort in "Benutzeroptionen"

  1. Klicken Sie im Menü Extras auf Optionen.
  2. Klicken Sie auf die Registerkarte Passwort und wählen Sie den gewünschten Benutzer aus der vorgegebenen Liste aus.
  3. Geben Sie das neue Passwort ein.
  4. Geben Sie in das Feld Passwort bestätigen das neue Passwort erneut ein.
  5. Klicken Sie auf Speichern. Es wird eine Meldung zur Bestätigung der Passwortänderung angezeigt.
