Auswählen einer Zeitzone

Infor CRM Web Client zeigt Ihre Zeitzone in der Statusleiste am unteren Rand des Browserfensters an. Die Standardzeitzone basiert auf der Zeitzoneneinstellung Ihres lokalen Rechners. Ist die Zeitzone falsch, können Sie zu einem anderen Gebietsschema innerhalb derselben Zeitzone wechseln.

So ändern Sie die Zeitzone

  1. Doppelklicken Sie auf die Zeitzone, die in der Statusleiste angezeigt wird.
  2. Klicken Sie im Dialogfeld Zeitzoneneinstellungen auf den Dropdown-Pfeil, und wählen Sie einen Eintrag in der Liste aus.

  3. Klicken Sie auf OK.
