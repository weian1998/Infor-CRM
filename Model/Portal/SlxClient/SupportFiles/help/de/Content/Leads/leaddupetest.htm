Interessenten importieren – Prüfen auf doppelte Datensätze

In dieser Ansicht können Sie die Anzahl möglicher Duplikate und Fehler vorab anzeigen, bevor Sie eine Liste von Interessenten importieren. Werden viele mögliche Duplikate aufgelistet, sollten Sie Ihre Abgleichfilter so anpassen, dass nur doppelte Datensätze gefunden werden.

Beim Importieren umfangreicher Listen werden nur die ersten einhundert Datensätze geprüft.

So gehen Sie bei der Suche vor

  1. Beim Importieren von Interessenten klicken Sie während des Schritts Duplikate verwalten auf Test ausführen.
  2. Überprüfen Sie die folgenden Informationen:
  • Gesuchte Interessenten
  • Mögliche Duplikate
  • Voraussichtliche Duplikatrate
  1. Klicken Sie auf OK, wenn Sie fertig sind.
