Qualifizieren eines Interessenten

Die Angaben zu einem Interessenten können unqualifizierte Informationen enthalten. Sie können Kriterien zur Interessentenqualifizierung heranziehen, um Informationen zu überprüfen und um zu bestimmen, ob der Interessent eine Verkaufschance darstellt. Nach der Qualifizierung können Sie einen Interessenten in einen Kontakt und eine Firma konvertieren und eine neue Verkaufschance erstellen.

Wenn der Systemadministrator die Qualifizierungskriterien angepasst hat, sehen die in Ihrem Arbeitsbereich angezeigten Einträge möglicherweise anders aus.

So qualifizieren Sie einen Interessenten

  1. Öffnen Sie die Lead-Detailansicht für den Lead, den Sie qualifizieren möchten. 
  2. Klicken Sie im Feld Qualifizierungsmerkmal auf den Dropdown-Pfeil, wählen Sie die Qualifizierungsmerkmalkategorie in der Liste aus.
  3. Klicken Sie auf Ist Interessent bereits Kunde?, um zu bestimmen, ob der Interessent in Infor CRM bereits als Kontakt oder Firma vorhanden ist.
    1. Verwenden Sie das Dialogfeld Interessent, um nach doppelten Datensätzen zu suchen und sie zusammenzuführen.
    2. Wenn keine Übereinstimmungen gefunden werden, klicken Sie auf Abbrechen und fahren mit dem nächsten Schritt fort.
  4. Wählen Sie, wenn die Kriterien erfüllt sind, für jedes der Qualifizierungskriterien das Feld aus, und geben Sie dann bei Bedarf Anmerkungen in das Textfeld ein.
  5. Führen Sie unter Qualifizierungsmerkmal Folgendes durch:
    1. Aktivieren Sie das Kontrollkästchen Gültiger Kontakt.
    2. Aktivieren Sie das Kontrollkästchen Budget, und geben Sie dann in das entsprechende Textfeld 5000 ein.
    3. Aktivieren Sie das Kontrollkästchen Entscheidungszeitraum, und geben Sie dann in das entsprechende Textfeld 3 Monate ein.
    4. Aktivieren Sie das Kontrollkästchen Geschäftsanforderung identifiziert und geben Sie in das entsprechende Textfeld Buchhaltungs-Software ein.
    5. Aktivieren Sie das Kontrollkästchen Entscheider identifiziert, und geben Sie in das entsprechende Textfeld Werner Schulze ein.
  6. Wenn der Interessent qualifiziert wurde und in einen Kontakt und eine Firma konvertiert werden kann, klicken Sie auf die Schaltfläche Qualifizierten Interessenten konvertieren.
     Der Dialog Interessenten konvertieren wird geöffnet.
