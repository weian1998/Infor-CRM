Definieren eines Trennzeichens

In dieser Ansicht wählen Sie den geeigneten Trennzeichenwert für Ihre Liste aus und sehen sich eine Vorschau der Datensätze an, bevor Sie sie in Infor CRM importieren. Ein Begrenzer analysiert Felder in einer Liste. Infor CRM analysiert die Liste, um zu bestimmen, welcher Begrenzer momentan verwendet wird. Sie können bei Bedarf jedoch einen anderen Begrenzer auswählen.

So definieren Sie ein Trennzeichen und zeigen die Vorschau an

  1. Beim Importieren von Interessenten klicken Sie nach dem Auswählen einer Datei auf Weiter.
  2. Wenn Sie einen anderen Begrenzer auswählen möchten, klicken Sie auf eine der folgenden Optionen:
    • Registerkarte
    • Semikolon
    • Komma
    • Leerzeichen
    • Sonstige
  3. Wenn Sie Sonstige ausgewählt haben, müssen Sie das Trennzeichen angeben, das zum Trennen von Feldern in der Liste verwendet wird. Geben Sie das Trennzeichen in das Feld Sonstige ein.
  4. Klicken Sie im Feld Texttrennzeichen auf den Dropdown-Pfeil, und wählen Sie ein Trennzeichen in der Liste.
    Ein Texttrennzeichen bestimmt Text, der entsprechend der Anzeige importiert werden soll.
  5. Wenn die Liste Feldnamen enthält, aktivieren Sie das Kontrollkästchen Erste Zeile enthält Feldnamen, um die Feldnamen im Bereich Vorschau anzuzeigen.

    Wenn die importierte Datei erweiterte Zeichen enthält, die in der Vorschau nicht korrekt angezeigt werden, müssen Sie die Importdatei erneut speichern und den Kodierungstyp ausdrücklich als UTF-8, Unicode oder ANSI definieren. Wenn Sie die Datei erneut gespeichert haben, klicken Sie auf die Schaltfläche Zurück, um zum Schritt Auswählen einer Datei zurückzukehren, und wählen die aktualisierte Datei aus, bevor Sie den Import fortsetzen.

  6. Klicken Sie auf Weiter, um zum nächsten Schritt zu gelangen.
