Verwalten doppelter Datensätze

Beim Importieren von Interessenten können Sie nach anderen Interessenten oder Kontakten suchen, die in der Datenbank vorhanden sind. Wenn in der Datenbank ein Datensatz mit identischen Informationen enthalten ist, können Sie dadurch doppelte Datensätze vermeiden.

Infor CRM kann nur nach doppelten Datensätzen suchen, zu deren Anzeige Sie berechtigt sind. Es können dann trotzdem noch doppelte Datensätze vorhanden sein, auf die Sie nicht zugreifen dürfen.

So suchen Sie nach übereinstimmenden Datensätzen

  1. Beim Importieren von Interessenten klicken Sie nach dem Zuordnen von Feldern auf Weiter.
  2. Wählen Sie Nach doppelten Interessenten suchen aus.
  3. Wählen Sie neben Typ die zu suchenden Datensatztypen aus.
  4. Aktivieren Sie unter Mit Filtern abgleichen die Kontrollkästchen der Filter, die Sie anwenden möchten.

Die verfügbaren Optionen hängen von den Werten ab, die Sie beim Zuordnen von Feldern ausgewählt haben.

  1. Um eine erweiterte Suche durchzuführen, klicken Sie auf Erweiterte Übereinstimmungsoptionen.

  2. Um doppelte Datensätze beim Import automatisch zusammenzuführen, wählen Sie Doppelte Datensätze automatisch zusammenführen aus.
  3. Wählen Sie im Feld Konflikte eine der folgenden Optionen aus:
  4. Vorhandener Datensatz gewinnt: Beim Zusammenführen von Datensätzen werden vorhandene Datensatzinformationen verwendet.
  • Importierter Datensatz gewinnt: Beim Zusammenführen von Datensätzen werden die vorhandenen Datensatzinformationen durch die importierten Datensatzinformationen ersetzt.

    Weitere Informationen dazu finden Sie unter dem Thema Regeln für die automatische Zusammenführung von Datensätzen beim Import.
  1. Klicken Sie auf Test ausführen, um den Test der Duplikatsuche für die ersten 100 Datensätze des Imports auszuführen.
  2. Wenn Sie die Importquelldatei auf doppelte Datensätze überprüfen möchten, aktivieren Sie die Option Importquelldatei auf Duplikate prüfen.

    		3.

    Durch die Auswahl dieser Option verlängert sich eventuell die Dauer des Importvorgangs.
  3. Klicken Sie auf Weiter, um zum nächsten Schritt zu gelangen.

Doppelte Datensätze, die gefunden, aber nicht zusammengeführt wurden, werden nach Abschluss des Imports in der Detailansicht Interessentenimporthistorie in der Registerkarte Duplikate angezeigt. Dort können Sie die doppelten Datensätze verwalten.
