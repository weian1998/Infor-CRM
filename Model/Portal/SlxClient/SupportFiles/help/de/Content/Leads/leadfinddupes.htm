Auflösen doppelter Interessenten beim Import

Beim Importieren von Interessenten können Sie nach anderen Interessenten oder Kontakten suchen, die in der Datenbank vorhanden sind. Mithilfe dieser Ansicht können Sie die doppelten Datensätze manuell importieren oder zusammenführen.

Infor CRM kann nur nach doppelten Datensätzen suchen, zu deren Anzeige Sie berechtigt sind. Es können dann trotzdem noch doppelte Datensätze vorhanden sein, auf die Sie nicht zugreifen dürfen.

So suchen Sie nach übereinstimmenden Datensätzen

  1. Öffnen Sie die Detailansicht Interessentenimporthistorie und klicken Sie auf die Registerkarte Duplikate.
  2. Klicken Sie neben dem Interessenten, der erneut importiert oder zusammengeführt werden soll, auf Auflösen.
  3. Wählen Sie neben Typ die zu suchenden Datensatztypen aus.
  4. Aktivieren Sie unter Mit Filtern abgleichen die Kontrollkästchen der Filter, die Sie anwenden möchten.
  5. Wählen Sie eine der folgenden Optionen aus:
  • Mit allen ausgewählten Filtern abgleichen: Ein Datensatz ist nur ein Duplikat, wenn er allen ausgewählten Filtern entspricht.
  • Mit ausgewählten Filtern abgleichen: Ein Datensatz ist ein Duplikat, wenn er einem der ausgewählten Filter entspricht.
  1. Um eine erweiterte Suche durchzuführen, klicken Sie auf Erweiterte Optionen.
  2. Klicken Sie auf Potenzielle Übereinstimmungen aktualisieren.
  3. Führen Sie einen der folgenden Schritte durch:
  4. Führen Sie den Interessenten mit einem übereinstimmenden Kontakt zusammen.
  1. Klicken Sie im Raster Übereinstimmende Kontakt-/Interessentendatensätze neben dem Datensatz, mit dem eine Zusammenführung erfolgen soll, auf Zusammenführen.
  2. Führen Sie die Schritte für das Zusammenführen von Datensätzen aus.
  1. Importieren Sie den Interessenten manuell.
  • Klicken Sie auf Interessenten einfügen.
  1. Klicken Sie auf Abbrechen, um den Interessenten als Duplikat beizubehalten.

Nach der Auflösung eines Duplikats wird es aus der Registerkarte "Duplikate" entfernt.
