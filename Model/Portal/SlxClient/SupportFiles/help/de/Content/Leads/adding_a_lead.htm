Hinzufügen eines Interessenten

Verwenden Sie den Dialog "Interessenten einfügen" zum Hinzufügen eines neuen Interessenten zu Infor CRM Web Client.

So gehen Sie zum Hinzufügen vor

  1. Klicken Sie im Menü Neu auf Interessent. 
  2.  Klicken Sie auf die Schaltfläche Name bearbeiten. 
     Das Dialogfeld Name bearbeiten wird geöffnet.
  3. Füllen Sie die verbleibenden Felder mit Informationen zum Interessenten aus.
  4. Klicken Sie auf Nach übereinstimmenden Datensätzen suchen, um in der Datenbank nach Interessenten, Kontakten und Firmen mit übereinstimmenden Informationen zu suchen.
     
     
    1. Wenn keine potenziellen Übereinstimmungen vorhanden sind, klicken Sie auf OK, um den Dialog zu schließen. Fahren Sie dann mit dem Eingeben von Informationen zum Interessenten fort.
    2. Wenn potenzielle Übereinstimmungen vorhanden sind, wird das Dialogfeld "Duplikate suchen" geöffnet.
    3. Wenn das System zukünftig nach potenziellen Übereinstimmungen suchen soll, ohne dass Sie auf die Schaltfläche Nach übereinstimmenden Datensätzen suchen klicken müssen, aktivieren Sie das Kontrollkästchen Automatische Suche, nachdem "Telefon" ausgefüllt wurde.
      4. Der Dialog "Übereinstimmende Interessenten" wird angezeigt. Wenn in diesem Dialog ein übereinstimmender Datensatz angezeigt wird, können Sie diesen aufrufen, um nachzusehen, ob der Interessent bzw. der Kontakt oder die Firma mit dem eingegebenen Interessenten übereinstimmen.
  5. Führen Sie einen der folgenden Schritte durch:
    • Um den Interessenten zu speichern und die Ansicht zu schließen, klicken Sie auf Speichern
    • Um den Interessenten zu speichern und einen neuen Interessenten einzugeben, klicken Sie auf Speichern und neu.
