Konvertieren von Interessenten

Nach der Qualifizierung von Interessenteninformationen kann ein Interessent in einen Kontakt und eine Firma konvertiert werden. Im Dialog "Interessent konvertieren" werden Kontakte oder Firmen mit übereinstimmenden Informationen angezeigt. Interessenten mit übereinstimmenden Informationen lassen sich mit einem vorhandenen Kontakt zusammenführen oder einer vorhandenen Firma hinzufügen. Alternativ können die Interessenteninformationen auch als neuer Kontakt bzw. neue Firma hinzugefügt werden. Sie können die verwendeten Filterkriterien ändern, um übereinstimmende Datensätze zu finden.

Infor CRM kann nur nach doppelten Datensätzen suchen, zu deren Anzeige Sie berechtigt sind. Es können dann trotzdem noch doppelte Datensätze vorhanden sein, auf die Sie nicht zugreifen dürfen.

So konvertieren Sie einen qualifizierten Interessenten

  1. Öffnen Sie die Interessentendetailansicht für den zu konvertierenden Interessenten, und klicken Sie dann auf Qualifizierten Interessenten konvertieren.
    Die Ansicht "Interessenten konvertieren" wird geöffnet. Hier sind im oberen Bereich Interessenteninformationen aufgeführt, darunter die übereinstimmenden Kontakt- und Firmeninformationen (sofern vorhanden).
  2. Aktivieren Sie unter Mit Filtern abgleichen die Kontrollkästchen der Filter, die Sie anwenden möchten.
  3. Wählen Sie eine der folgenden Optionen:
    • Mit allen ausgewählten Filtern abgleichen – Ein Datensatz ist nur ein Duplikat, wenn er allen ausgewählten Filtern entspricht.
    • Mit ausgewählten Filtern abgleichen: Ein Datensatz ist ein Duplikat, wenn er einem der ausgewählten Filter entspricht.
  4. Um eine erweiterte Suche durchzuführen, klicken Sie auf Erweiterte Optionen.
  5. Klicken Sie auf Potenzielle Übereinstimmungen aktualisieren.
  6. Klicken Sie auf den Dropdown-Pfeil Konflikte, um zu ermitteln, wie Sie Konflikte zwischen Informationen beheben können.
    • Wählen Sie Firma gewinnt aus, wenn vorhandene Kontaktinformationen Vorrang haben sollen.
    • Wählen Sie Interessent gewinnt aus, wenn im Konflikt stehende Informationen durch Interessenteninformationen ersetzt werden sollen.
  7. Wenn Sie eine Verkaufschance erstellen möchten, aktivieren Sie das Kontrollkästchen Verkaufschance nach Interessentenkonvertierung oder -zusammenführung erstellen.
    Alle Interessentenaktivitäten, Historieneinträge und Notizen werden mit der neuen Verkaufschance verknüpft.
  8. Führen Sie einen der folgenden Schritte durch:
    • Führen Sie den Interessenten mit einem übereinstimmenden Kontakt zusammen.
      1. Klicken Sie im Raster Übereinstimmende Kontakt-/Interessentendatensätze neben dem Datensatz, mit dem eine Zusammenführung erfolgen soll, auf Zusammenführen.
      2. Führen Sie die Schritte für das Zusammenführen von Datensätzen aus.
    • Fügen Sie einen Kontakt einer übereinstimmenden Firma hinzu.
      • Klicken Sie im Raster Überstimmende Firmendatensätze dem der Firma, der Sie einen Kontakt hinzufügen möchten, auf Kontakt hinzufügen.
        Der Interessent wird als Kontakt zur ausgewählten Firma hinzugefügt und die Detailansicht "Kontakt" wird geöffnet.
    • Fahren Sie mit dem Konvertieren des Interessenten in einen neuen Kontakt bzw. in eine neue Firma fort.
      • Klicken Sie auf Konvertieren.
