Importieren von Interessenten aus einer Liste

Über die Ansicht "Interessenten importieren" können Sie Listen mit Interessenten in Infor CRM importieren. Sie können Interessenten nur aus einer Liste mit durch Kommas getrennten Werten (CSV) importieren. Gültige Dateitypen sind Microsoft Excel, TXT-Dateien und andere Datenbankdateitypen, die Dateien müssen jedoch als CSV-Datei gespeichert sein.

Die Listen müssen als trennzeichenbegrenzte Textdateien (ANSI oder ASCII) vorliegen.

So importieren Sie Interessenten

  1. Klicken Sie im Menü Extras auf Interessenten importieren.
  2. Machen Sie die entsprechenden Angaben unter Wählen Sie eine Datei aus und klicken Sie auf Weiter.
  3. Geben Sie die entsprechenden Informationen unter Begrenzer definieren ein, und klicken Sie auf Weiter.
  4. Machen Sie die entsprechenden Angaben unter Felder zuordnen, und klicken Sie auf Weiter.
  1. Machen Sie die entsprechenden Angaben unter Duplikate verwalten und klicken Sie auf Weiter.
  2. Machen Sie die entsprechenden Angaben unter Importaktionen auswählen und klicken Sie auf Weiter.
     
  3. Prüfen Sie die Importeinstellungen, und klicken Sie auf Übergeben.
     
  4. Überwachen Sie den angeforderten Importvorgang.
  5. Nach Beendigung des Imports klicken Sie auf den Hyperlink Importnummer, um die Ergebnisse in der Importhistorie in der Importhistorienansicht anzuzeigen.
