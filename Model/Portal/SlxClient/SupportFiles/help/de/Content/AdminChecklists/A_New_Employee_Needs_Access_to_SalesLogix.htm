Ein neuer Mitarbeiter benötigt Zugriff auf Infor CRM

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Stellen Sie, wenn eine Person Zugriff auf Infor CRM benötigt, anhand dieser Checkliste sicher, dass Sie alle Schritte zum Hinzufügen neuer Benutzer ausführen. (Meist ist es hilfreich, eine Kopie dieser Checkliste sowie einige der verknüpften Themen mit zusätzlichen Informationen auszudrucken.)

  • Stellen Sie anhand der Windows-Ordnerfreigabeberechtigungen für die Ordner "Sync Logs", "Library" und "Documents" sicher, dass der neue Benutzer Zugriff auf diese Ordner hat.
  • Legen Sie den Benutzertyp fest, den Sie hinzufügen möchten.
  • Vergewissern Sie sich, dass für diesen Benutzertyp eine Lizenz vorhanden ist.
  • Legen Sie fest, wie Sie das Benutzerprofil konfigurieren möchten:
  • Möchten Sie mehrere Benutzer mit identischen Profilen hinzufügen? Falls ja, legen Sie fest, ob Sie eine Benutzervorlage verwenden möchten.
  • Enthält ein vorhandenes Benutzerprofil die Einstellungen, die für den neuen Benutzer gelten sollen? Falls ja, notieren Sie den Namen des vorhandenen Benutzers, sodass Sie beim Hinzufügen des neuen Benutzers das richtige Profil auswählen können, aus dem Sie kopieren möchten. Wenn bestimmte Einstellungen nicht kopiert werden sollen, notieren Sie die Registerkarten, die beim Kopieren des Profils nicht einbezogen werden sollen.
  • Benötigt dieser Benutzer ein bestimmtes Profil mit Einstellungen, die erheblich von Benutzervorlagen oder vorhandenen Benutzern abweichen? Falls ja, müssen Sie jede Registerkarte in der Benutzerdetailansicht manuell ändern, um die Einstellungen zu konfigurieren.
  • Fügen Sie den Benutzer hinzu.
  • Ändern Sie ggf. das Profil des Benutzers. Vergewissern Sie sich anschließend, dass auf der Registerkarte "Allgemein" das Kontrollkästchen "Anmeldung aktiviert" aktiviert ist.
  • Wenn in Ihrem Unternehmen Teams Firmenzugriffsberechtigte sind (empfohlen), fügen Sie den Benutzer dem entsprechenden Team bzw. mehreren Teams hinzu. Vergewissern Sie sich, dass Sie die Arbeitsweise von Teams in Infor CRM verstanden haben, da die Teams, denen ein Benutzer angehört, darüber entscheiden, auf welche Firmen zugegriffen werden kann.
  • Wenn Ihr Unternehmen Outlook-Synchronisierung zum Synchronisieren von Kontakt- und Kalenderinformationen zwischen Infor CRM und Outlook verwendet, konfigurieren Sie die Sync Integration.
