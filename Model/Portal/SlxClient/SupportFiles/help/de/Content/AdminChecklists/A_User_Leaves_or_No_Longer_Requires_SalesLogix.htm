Ein Benutzer scheidet aus oder benötigt Infor CRM nicht mehr

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Wenn ein Benutzer aus Ihrem Unternehmen ausscheidet, sollte sein Profil am besten in das eines inaktiven Benutzers umgewandelt und nicht gelöscht werden. Inaktive Benutzer nutzen keine Lizenz und können sich nicht anmelden, ihre Namen werden jedoch in der Datenbank beibehalten. Damit wird die Anzeige "Unbekannter Benutzer" in Historiendatensätzen vermieden. Verwenden Sie die folgende Checkliste, wenn ein Benutzer ausscheidet:

  • Wenn in Ihrem Unternehmen Firmen in der Zuständigkeit einzelner Personen sind, und der Benutzer, der ausscheidet, für Firmen zuständig ist, müssen Sie die Firmen neu zuweisen. Über die Gebietsneuausrichtung in Administrator können Sie die Firmen des Benutzers einem anderen Benutzer zuweisen.
  • Öffnen Sie die Benutzerdetailansicht.
  • Bestimmen Sie für den ausscheidenden Benutzer auf der Registerkarte Teammitgliedschaft, ob der Benutzer ein Teamzugriffsberechtigter für Teams ist. Falls ja, ändern Sie das Sicherheitsprofil des Benutzer in ein andere Profil als das des Teamzugriffsberechtigten.
  • Sollte das Feld Benutzer ist ein Manager aktiviert sein, deaktivieren Sie es. (Sie können Benutzer, die Manager sind, nicht entfernen.)
  • Geben Sie im Feld Name vor dem Benutzernamen INAKT oder eine ähnliche Zeichenfolge ein. Dieser Schritt ist zwar nicht unbedingt notwendig, Sie können aber auf diese Weise Personen leichter als ausgeschiedene Benutzer erkennen.
  • Wählen Sie in der Liste Benutzertyp die Option Inaktiver Benutzer aus.
