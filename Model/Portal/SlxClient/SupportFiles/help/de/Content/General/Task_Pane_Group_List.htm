Verwenden der Detailansicht "Gruppenliste"

Die Detailansicht "Gruppenliste" enthält eine Liste aller Datensätze in einer Gruppe oder Suche innerhalb der Detailansicht. Sie können auf einen Blick andere Datensätze in der Gruppe oder Suche erkennen, oder auf einen Datensatz in der Liste klicken, um zur Detailansicht dieses Datensatzes zu navigieren. Für die Gruppenliste gibt es zwei unterschiedliche Modi: Im normalen Modus wird nur der Datensatzname angezeigt, im erweiterten Modus werden weitere Spalten mit zusätzlichen Informationen angezeigt. Sie können Optionen festlegen, um das Standardverhalten der Gruppenliste zu bestimmen.

So rufen Sie Details auf

  • Klicken Sie auf die Schaltfläche Gruppenliste erweitern.

So blenden Sie Details aus

  • Klicken Sie auf die Schaltfläche Gruppenliste minimieren.

So legen Sie Optionen für Gruppenlisten fest

  1. Klicken Sie auf die Schaltfläche Optionen für erweiterte Gruppenlisten.
  2. In das Feld Spalten tippen Sie die Anzahl der Spalten, die im erweiterten Modus angezeigt werden sollen. Wenn alle Spalten angezeigt werden sollen, die in der Listenansicht angezeigt werden, tippen Sie 0.
    Es werden die Spalten angezeigt, die in der Listenansicht angezeigt werden.
  3. Aktivieren oder deaktivieren Sie Erweiterte Gruppenliste nach Datensatzauswahl ausblenden.
    • Aktivieren Sie diese Option, um die erweiterte Liste auszublenden, wenn Sie einen Datensatz aus der Liste ausgewählt haben.
    • Deaktivieren Sie diese Option, wenn die Gruppenliste nach der Auswahl eines Datensatzes aus der Liste im aktuellen Status verbleiben soll.
  4. Aktivieren oder deaktivieren Sie Nach Suche in Detailansicht bleiben.
    • Aktivieren Sie diese Option, wenn Sie die Detailansicht bei der Navigation durch die Datensätze in den Suchergebnissen beibehalten möchten.
    • Deaktivieren Sie diese Option, um die Suchergebnisse immer in der Listenansicht anzuzeigen.
  5. Aktivieren oder deaktivieren Sie Erweiterte Gruppenliste nach Suche anzeigen.
    • Aktivieren Sie diese Option, wenn die erweiterte Liste beim Anzeigen der Detailansicht der Datensätze in den Sucherergebnissen angezeigt werden soll.
    • Deaktivieren Sie diese Option, wenn die erweiterte Liste beim Anzeigen der Detailansicht der Datensätze in den Sucherergebnissen automatisch ausgeblendet werden soll.

  6. Klicken Sie auf OK.
