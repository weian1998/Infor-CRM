Unterstützte Desktop-Integrationsfunktionen nach Browser

Das Infor CRM-Desktopintegrationsmodul ist eine Zusammenstellung von Funktionen zur Optimierung der Web-Umgebung von Infor CRM Web. Einige Browser unterstützen Funktionen, aber benötigen dazu die Desktopintegration, und einige Funktionen werden nur mit bestimmten Browsern unterstützt.
