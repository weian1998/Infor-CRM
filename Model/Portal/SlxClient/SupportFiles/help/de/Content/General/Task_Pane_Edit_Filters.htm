Hinzufügen und Entfernen von Filtern

Mit Filtern können Sie die Liste der Datensätze eingrenzen, die in einer Listenansicht angezeigt wird. Die verfügbaren Filter basieren auf den Spalten in der ausgewählten Listenansicht und den in den Spalten enthaltenen Informationen. Sie können die Liste der Filter jedoch so anpassen, daß sie nur die Filter enthält, die Sie für die einzelnen Gruppen verwenden möchten.

So fügen Sie Filter hinzu, die im Aufgabenbereich angezeigt werden, bzw. entfernen sie

  1. Klicken Sie in einer Listenansicht im Aufgabenbereich auf Filter bearbeiten.
    Das Dialogfeld "Filter bearbeiten" wird geöffnet.
  2. Aktivieren Sie die Kontrollkästchen der Filter, die in der Liste Filter angezeigt werden sollen, und deaktivieren Sie die Kontrollkästchen der Filter, die ausgeblendet werden sollen. Über das Kontrollkästchen Alle auswählen können Sie alle Kontrollkästchen gleichzeitig aktivieren bzw. deaktivieren.

  3. Klicken Sie auf OK.
