Senden von E-Mails an Kontakte oder Interessenten

Diese Funktion ist nur verfügbar, wenn Ihr E-Mail-Programm mit Ihrem Browser kompatibel ist und für einen Kontakt oder Interessenten eine E-Mail-Adresse in der Datenbank eingegeben wurde. Wenn Sie eine E-Mail-Adresse für einen Kontakt oder Interessenten eingeben, aktualisieren Sie den Datensatz, um die Adresse in die Datenbank zu übernehmen, bevor Sie eine E-Mail senden.

So senden Sie eine E-Mail an einzelne Kontakte oder Interessenten

  1. Gehen Sie nach einer der folgenden Methoden vor:
    • Klicken Sie in der Detailansicht Kontakt oder Interessent auf die Schaltfläche E-Mail neben der E-Mail-Adresse, oder klicken Sie im Menü Verfassen auf E-Mail.
    • Klicken Sie in der Listenansicht Kontakt oder Interessent auf einen E-Mail-Hyperlink, oder klicken Sie mit der rechten Maustaste auf einen Datensatz, und klicken Sie dann auf E-Mail.
    • Klicken Sie im Aufgabenbereich, auf E-Mail.
    •  
  2. Wenn E-Mail aktiviert ist, Sie sich aber noch nicht angemeldet haben, klicken Sie im Meldungsfeld auf Ja, um sich anzumelden.
  3. Füllen Sie das Feld Betreff und das Nachrichtenfeld des Fensters für die neue E-Mail aus.
  4. (Optional) Soll gemeinsam mit der E-Mail-Nachricht eine Anlage versendet werden, klicken Sie auf Einfügen und dann auf eine der Menüoptionen.
  5. Zum Versenden Ihrer E-Mail führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf die Schaltfläche Senden. Die Nachricht wird nicht in der Historie aufgezeichnet.
    • Wenn Sie Microsoft Outlook verwenden, klicken Sie auf die Schaltfläche Senden an CRM. Wenn Sie eine Datei angehängt haben und diese als Anlage speichern wollen, klicken Sie in der Eingabeaufforderung auf Ja. Die E-Mail-Nachricht wird in der Historie aufgezeichnet und die Datei wird auf der Registerkarte Anlagen angezeigt.

Grafikdateien, die kleiner als die vom Administrator festgelegte Mindestgröße sind, werden nicht als separate Anlagen gespeichert.

So senden Sie eine E-Mail an mehrere Kontakte oder Interessenten

  1. Öffnen Sie die Listenansicht Kontakte oder Interessenten.
  2. Öffnen Sie eine Gruppe oder führen Sie eine Suche durch, um die Datensätze zu suchen, an die Sie die E-Mail senden möchten.
  3.  Wählen Sie die Datensätze aus, an die Sie eine E-Mail senden möchten. 
     Wenn Sie mehrere Datensätze markieren möchten, dann halten Sie [STRG] oder [Umschalt] gedrückt und markieren die Datensätze.
  4. Klicken Sie im Aufgabenbereich unter Allgemeine Aufgaben auf E-Mail.
  5. Wenn das Dialogfeld Namen auswählen geöffnet wird, wählen Sie einen oder mehrere Namen aus, und klicken Sie auf die Schaltfläche An, Cc oder Bcc, um festzulegen, wie die E-Mail gesendet werden soll. Klicken Sie dann auf OK.
    Die herausgefilterten Informationen im unteren Teil des Dialogfelds zeigen an, warum Datensätze nicht in die E-Mail aufgenommen werden konnten und wie viele Datensätze herausgefiltert wurden. Mögliche Ursachen bestehen darin, dass die Datensätze mit "Keine Informationen erwünscht" gekennzeichnet sind, keine gültige E-Mail-Adresse bekannt ist oder der Datensatz doppelt vorhanden ist.
  6. Wenn E-Mail aktiviert ist, Sie sich aber noch nicht angemeldet haben, klicken Sie im Meldungsfeld auf Ja, um sich anzumelden.
  7.  Füllen Sie das Feld Betreff und das Nachrichtenfeld des Fensters für die neue E-Mail aus.
  8. (Optional) Soll gemeinsam mit der E-Mail-Nachricht eine Anlage versendet werden, klicken Sie auf Einfügen und dann auf eine der Menüoptionen.
  9. Zum Versenden Ihrer E-Mail führen Sie einen der folgenden Schritte durch:
    • Klicken Sie auf die Schaltfläche Senden. Die Nachricht wird nicht in der Historie aufgezeichnet.
    • Klicken Sie auf die Schaltfläche Senden an CRM. Wenn Sie eine Datei angehängt haben und diese als Anlage speichern wollen, klicken Sie in der Eingabeaufforderung auf Ja. Die E-Mail-Nachricht wird in der Historie aufgezeichnet und die Datei wird auf der Registerkarte Anlagen angezeigt.
