Hinzufügen oder Entfernen von Filterelementen

Mit Filtern können Sie die Liste der Datensätze eingrenzen, die in einer Listenansicht angezeigt wird. Sie können Ihre Filter so anpassen, dass sie nur die Filterelemente beinhalten, die Sie für die einzelnen Gruppen verwenden möchten.

Manche Filter können nicht angepasst werden.

So fügen Sie Filterelemente hinzu bzw. entfernen sie

  1. Klicken Sie von einer Listenansicht aus im Aufgabenbereich unter Filter auf den Namen eines Filters. Klicken Sie beispielsweise auf "Zugriffsberechtigter".
  2. Klicken Sie auf Elemente bearbeiten.
    Das Dialogfeld "Filterelemente bearbeiten" wird angezeigt.
  3. Geben Sie bei Bedarf die ersten Buchstaben in das Feld Element suchen ein und klicken Sie auf Suchen, um nach einem Filterelement zu suchen.
  4. Aktivieren Sie die Kontrollkästchen der Filterelemente, die in der Liste Filter angezeigt werden sollen, und deaktivieren Sie die Kontrollkästchen der Elemente, die ausgeblendet werden sollen. Durch Klicken auf Alle auswählen wählen Sie alle Filterelemente aus.
  5. Klicken Sie auf OK. 
    Der Filter enthält nur die ausgewählten Elemente.
