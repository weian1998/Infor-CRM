Verwenden der Ansicht "Zusammenfassung"

In der Ansicht "Zusammenfassung" können Sie gängige Informationen für jeden Datensatztyp anzeigen. Die Informationen und verfügbaren Funktionen sind von der Art des Datensatzes abhängig.

Vorgehensweise

Öffnen der Zusammenfassungsansicht
  • Klicken Sie in einer Listenansicht auf Zusammenfassung.

In der Ansicht "Zusammenfassung" werden Informationen in drei Spalten angezeigt.

Spalte Beschreibung/Aktion
Links

Zeigt Informationen für den Hauptdatensatz an. Abhängig vom Datensatztyp können Sie Folgendes tun:

  • Auf den Datensatznamen klicken, um die Detailansicht zu öffnen
  • Auf eine E-Mail-Adresse klicken, um eine E-Mail zu senden
  • Auf eine Web-Adresse klicken, um eine Website in einem neuen Browserfenster zu öffnen
  • Die Aktivität durch Klicken auf Abschließen abschließen, sofern Sie Aktivitäten anzeigen.
Mitte Zeigt zusätzliche Informationen über den Datensatz an, die Links zu dazugehörigen Datensätzen oder schreibgeschützte Informationen umfassen können.
Rechts

Zeigt dazugehörige Datensatztypen an. Abhängig vom Datensatztyp können Sie Folgendes tun:

  • Auf Hyperlinks klicken, um die Detailansicht auf der entsprechenden Registerkarte zu öffnen
  • Klicken Sie auf Zahlenwerte, um eine Liste der dazugehörigen Datensätze anzuzeigen. Dies könnten beispielsweise Kontakte oder Tickets sein. Die angezeigte Liste kann Hyperlinks zum Öffnen der Detailansicht für den dazugehörigen Datensatz enthalten.
