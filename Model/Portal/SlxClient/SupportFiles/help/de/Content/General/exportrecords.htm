Exportieren von Datensätzen in eine Datei

Sie können Datensätze in eine Datei exportieren. Die Überschriften und die Formatierung der Gruppenspalten werden ebenfalls exportiert.

Wenn die Option In Datei exportieren nicht zur Verfügung steht,

So exportieren Sie Datensätze in eine Datei

  1. Zeigen Sie die Gruppe oder Suchergebnisse an, die die zu exportierenden Datensätze enthalten.
  2. Führen Sie einen der folgenden Schritte durch:
    • Wählen Sie einen oder mehrere Datensätze aus, indem Sie beim Klicken auf die Einträge die STRG- oder UMSCHALTTASTE gedrückt halten. Wählen Sie im Aufgabenbereich unter Allgemeine Aufgaben den Eintrag In Datei exportieren aus.

      Wenn Sie keine Datensätze auswählen, wird angefragt, ob alle Datensätze in der Gruppe verwendet werden sollen. Wenn Sie auf OK klicken, werden alle Gruppendatensätze ausgewählt.

    • Um die gesamte Gruppe zu exportieren, klicken Sie mit der rechten Maustaste auf die Gruppenregisterkarte und klicken dann auf In Datei exportieren.

    		3.

    Wenn Sie keine Datensätze auswählen, wird angefragt, ob alle Datensätze in der Gruppe verwendet werden sollen. Wenn Sie auf OK klicken, werden alle Gruppendatensätze ausgewählt.
  3. Wählen Sie das gewünschte Dateiformat aus.
    • Standardmäßiges CSV-Format: Die einzelnen Datensätze werden durch ein Komma getrennt.
    • Tabstoppgetrenntes Format: Die einzelnen Datensätze werden durch einen Tabstopp getrennt.

    Wenn Infor CRM die Auswahl speichern und nicht mehr nachfragen soll, wählen Sie Auswahl speichern und nicht mehr anfragen aus.

  4. Klicken Sie auf OK.

Eventuell erhalten Sie eine Fehlermeldung, wenn Sie versuchen, eine Gruppe zu exportieren, die ein Datenfeld mit einer großen Datenmenge enthält.
