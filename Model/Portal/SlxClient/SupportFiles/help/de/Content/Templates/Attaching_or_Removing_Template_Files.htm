Anhängen oder Entfernen von Vorlagendateien

Sie können Windows-Dateien oder Library-Dateien an eine Vorlage anhängen. Wenn die Vorlage für einen Seriendruck verwendet wird, werden die angehängten Dateien automatisch an den Kontakt oder Interessenten gesendet. Wenn die die Anlage enthaltende Vorlage kopiert wird, wird die Anlage ebenfalls kopiert. Eine Anlage kann auch von der Vorlage entfernt werden.

So hängen Sie eine Datei an eine Vorlage an

  1. Öffnen Sie den Vorlagen-Editor.
  2. Führen Sie abhängig von dem Dateityp, den Sie anhängen möchten, einen der folgenden Schritte aus:
     
    • Windows-Datei: Klicken Sie auf Datei einfügen. Suchen Sie die Speicherposition der Datei, und klicken Sie dann auf Datei einfügen.
    • Bibliotheksdatei: Klicken Sie auf im Menü Einfügen oder in der Symbolleiste auf Bibliothek. Suchen Sie die Dateien im Dialogfeld Bibliothek, und klicken Sie dann auf Einfügen.
      Der Dateiname wird im Feld Anlagen angezeigt.
  3. Sie können eine weitere Datei anhängen, oder Sie können auf Speichern und schließen klicken.

So entfernen Sie eine Anlage von einer Vorlage

  1. Öffnen Sie den Vorlagen-Editor.
  2. Wählen Sie die Anlage, die Sie entfernen möchten, im Feld Anlagen aus.
  3. Führen Sie einen der folgenden Schritte aus, um eine Anlage zu entfernen:
    • Klicken Sie mit der rechten Maustaste, und wählen Sie dann Entfernen.
    • Drücken Sie auf Entf.
  4. Sie können eine weitere Datei entfernen, oder Sie können auf Speichern und schließen klicken, um den Vorlagen-Editor zu schließen.
