Erstellen einer Vorlage

Mithilfe des Vorlagen-Editors können Sie private Vorlagen erstellen.

So erstellen Sie eine neue Vorlage

  1. Klicken Sie im Menü Verfassen auf Vorlagen verwalten.
  2. Klicken Sie im Dialog Vorlagen verwalten auf Neu.
  3. Im Dialogfeld "Eigenschaften" können Sie die Eigenschaften der Vorlage festlegen. Klicken Sie anschließend auf OK.
    Im Editor wird als Beispiel eine Basisvorlage angezeigt. Wollen Sie beispielsweise eine Briefvorlage erstellen, wird im Editor eine Basisbriefvorlage als Beispiel angezeigt.
    In diese Vorlage können Sie Inhalte einfügen oder hieraus entfernen oder auch Text aus einem anderen Dokument einfügen.
  4. Wenn Sie der Vorlage ein Infor CRM-Seriendruckfeld hinzufügen möchten, setzen Sie den Cursor in der Vorlage an die Position, an der das Seriendruckfeld erscheinen soll, klicken dann auf SLX-Feld einfügen und wählen danach aus der Liste der Seriendruckfelder ein Feld aus.
    Das neue Feld wird in der Vorlage angezeigt.
  5. Fügen Sie nach Bedarf weitere Seriendruckfelder hinzu oder entfernen Sie überflüssige Felder.
  6. Um die Vorlage zu speichern, ohne den Editor zu schließen, klicken Sie auf Speichern.
  7. Verwenden Sie im Editor die Infor CRM-Symbolleiste, um eine Vorschau der zusammengeführten Vorlage zu sehen und Dateien anzuhängen oder um andere Funktionen auszuführen.
  8. Wenn Sie fertig sind, klicken Sie auf Speichern und schließen, um die Vorlage zu speichern und den Editor zu schließen.
