Festlegen oder Aktualisieren der Eigenschaften einer Vorlage

Beim Erstellen, Bearbeiten oder Kopieren einer Vorlage können Sie die Eigenschaften für eine private oder öffentliche Vorlage festlegen oder aktualisieren. Wenn Sie eine Vorlage bearbeiten oder kopieren, weisen einige Eigenschaften die Standardeinstellungen des gewählten Vorlagentyps auf. Diese Einstellungen können jedoch im Bedarfsfall geändert werden.

Um das Dialogfeld "Eigenschaften" einer Vorlage zu öffnen, führen Sie einen der folgenden Schritte aus:
  • Klicken Sie im Dialogfeld Vorlagen verwalten auf Eigenschaften.
  • Klicken Sie im Dialogfeld Vorlagen verwalten auf Neu.
  • Wählen Sie im Dialogfeld Vorlagen verwalten eine Vorlage aus und klicken Sie dann auf Bearbeiten oder auf Kopieren.

So legen Sie die Eigenschaften einer Vorlage fest bzw. aktualisieren diese:

  1. Klicken Sie im Bereich Vorlagentyp auf einen der Vorlagentypen. Die Standardtypen sind "E-Mail", "Fax" und "Brief", Ihr Unternehmen kann jedoch auch andere Typen bereitstellen.
  2. Geben Sie in das Feld Name einen Namen für die neue Vorlage ein.

    Der Name darf keinen Apostroph ('), keinen umgekehrten Schrägstrich (\) und keinen Bindestrich (-) enthalten.

  3. Wählen Sie im Feld Familie die Vorlagenfamilie aus. Die Standardeinstellungen sind "Brief", "Fax" und "E-Mail", Ihr Unternehmen kann jedoch auch andere Familientypen bereitstellen.
  4. Wählen Sie im Feld Haupttabelle entweder Kontakt oder Interessent aus. Wenn Sie eine Vorlage erstellen, die in Verbindung mit Kontakten verwendet werden soll, wählen Sie Kontakt aus. Wenn Sie eine Vorlage erstellen, die in Verbindung mit Interessenten verwendet werden soll, wählen Sie Interessent aus.
  5. Im Feld Zugriffsberechtigter wählen Sie einen ein Infor CRM-Benutzer als Zugriffsberechtigten der Vorlage aus.

    Wenn Sie eine private Vorlage für andere Benutzer freigeben, empfiehlt es sich, die Vorlage zunächst zu kopieren und dann die Kopie freizugeben. Auf diese Weise befindet sich die private Vorlage weiterhin in Ihrer Liste.

  6. Klicken Sie auf OK.
