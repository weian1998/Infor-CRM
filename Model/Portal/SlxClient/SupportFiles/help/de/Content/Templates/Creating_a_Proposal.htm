Erstellen eines Verkaufsangebots

Sie können mit der Seriendruckfunktion neue Verkaufsangebote erstellen.

So erstellen Sie einen Verkaufsangebot:

  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Klicken Sie im Dialog Vorlagen verwalten auf Neu.
  3. Im Dialogfeld "Eigenschaften" können Sie die Eigenschaften der Vorlage festlegen. Klicken Sie anschließend auf OK.
    Im Editor wird als Beispiel eine Basisvorlage angezeigt, die Ihrer Auswahl entspricht. Wollen Sie beispielsweise eine Briefvorlage erstellen, wird im Editor eine Basisbriefvorlage als Beispiel angezeigt.
    In dieser Vorlage können Sie Inhalte einfügen oder entfernen oder auch Text einfügen, den Sie in einem anderen Dokument kopiert haben.
  4. Wenn Sie der Vorlage ein Infor CRM-Seriendruckfeld hinzufügen möchten, setzen Sie den Cursor in der Vorlage an die Position, an der das Seriendruckfeld erscheinen soll, klicken dann auf SLX-Feld einfügen und wählen danach aus der Liste der Seriendruckfelder ein Feld aus.
    Das neue Feld wird in der Vorlage angezeigt.
  5. Fügen Sie nach Bedarf weitere Seriendruckfelder hinzu oder entfernen Sie überflüssige Felder.
  6. Platzieren Sie zum Hinzufügen einer Verkaufschancen-Produkttabelle den Cursor in der Vorlage an der Stelle, an der die Tabelle eingefügt werden soll, und klicken Sie danach in der Symbolleiste auf Spezialfeld einfügen.
    1. Wählen Sie im Menü die Option Verkaufschancen-Produkttabelle aus.
      Das Dialogfeld des Feld-Editors für die Infor CRM-Seriendruckfunktion wird geöffnet.
    2. Markieren Sie in der Liste das Tabellenfeld, das Sie bearbeiten möchten.
    3. Nehmen Sie mithilfe von Felder bearbeiten Ihre Änderungen vor, und klicken Sie danach auf Übernehmen.
      • Feld-ID: Ermöglicht Ihnen, den Text zu ändern, der in der Vorlage angezeigt wird
      • Tabellenformat: Ermöglicht Ihnen, Formatierung auf ein Tabellenfeld anzuwenden
      • Optionen: Ermöglicht Ihnen, Eigenschaften auf ein Tabellenformat anzuwenden
      • Automatisch anpassen: Ermöglicht Ihnen, Eigenschaften für die automatische Anpassung für die Tabelle festzulegen
      • Text zusammenfassen: Ermöglicht Ihnen, Text einzugeben, der als Beschriftung für die erweiterte Preissumme von Produkten in der Tabelle verwendet werden soll
      • Tabellenspalten bearbeiten – Ermöglicht Ihnen, Beschriftungen, Breite und Ausrichtung von Tabellenspalten zu bearbeiten

  7. Klicken Sie auf OK.
