Kopieren einer Vorlage

Sie können private und öffentliche Vorlagen kopieren. Wird eine öffentliche Vorlage kopiert, wird diese allerdings als private Vorlage gespeichert.

Wenn Sie eine private Vorlage für andere Benutzer freigeben, empfiehlt es sich, die Vorlage zunächst zu kopieren und dann die Kopie freizugeben. Auf diese Weise befindet sich die private Vorlage weiterhin in Ihrer Liste.

So kopieren Sie eine Vorlage

  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Wählen Sie im Dialogfeld Vorlagen verwalten eine Vorlage aus der Liste aus, und klicken Sie dann auf Kopieren.
  3. Im Dialogfeld "Eigenschaften" können Sie die Eigenschaften der Vorlage festlegen. Klicken Sie anschließend auf OK.
    Die Vorlage wird im Editor geöffnet.
  4. Führen Sie einen der folgenden Schritte durch:
    • Wenn keine Bearbeitung erforderlich ist, klicken Sie auf der Infor CRM-Symbolleiste auf Speichern und schließen.
    • Ist eine Bearbeitung erforderlich, finden Sie auf der Infor CRM-Symbolleiste die erforderlichen Schaltflächen zum Hinzufügen von Feldern, zur Vorschau oder für andere Funktionen.
