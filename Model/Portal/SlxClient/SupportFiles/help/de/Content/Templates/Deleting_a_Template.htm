Löschen einer Vorlage

Sie können nur Ihre eigenen privaten Vorlagen löschen. Eine Vorlage, die für andere Benutzer freigegeben wurde, sollte vom Infor CRM-Administrator gelöscht werden.

So löschen Sie eine Vorlage:

  1. Klicken Sie im Menü Verfassen auf Vorlagen.
  2. Öffnen Sie das Dialogfeld Vorlagen verwalten oder Vorlage auswählen.
    • Wählen Sie im Menü Verfassen die Option Vorlagen.
  3. Wählen Sie im Dialogfeld Vorlagen verwalten eine Vorlage in der Liste Private Vorlagen aus, und klicken Sie dann auf Löschen.
  4. Klicken Sie auf OK, um die Meldung zu schließen.
  5. Führen Sie im Dialogfeld Vorlagen verwalten einen der folgenden Schritte aus:
    • Wählen Sie eine weitere private Vorlage zum Löschen aus.
    • Klicken Sie auf Schließen, um den Dialog zu schließen.
