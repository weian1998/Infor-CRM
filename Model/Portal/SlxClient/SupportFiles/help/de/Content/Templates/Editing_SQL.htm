Bearbeiten von SQL-Anweisungen

SQL-Anweisungen sollten nur von erfahrenen Benutzern bearbeitet werden.

Im Dialogfeld "SQL bearbeiten" können Sie SQL-Anweisungen für Seriendruckfelder erstellen und bearbeiten. Über das Menü "Bearbeiten" oder die Symbolleiste können Sie Text ausschneiden, kopieren, einfügen oder löschen. Sie können auch Parameter in eine SQL-Anweisung einfügen.

So bearbeiten Sie eine SQL-Anweisung:

  1. Geben Sie die SELECT-Anweisung ein, die Sie zum Seriendruckfeld hinzufügen möchten, oder bearbeiten Sie sie.
  2. Klicken Sie in der Ansicht SQL bearbeiten an die Stelle, an der Sie den Parameter einfügen möchten.
  3. Klicken Sie auf die Schaltfläche Parameter einfügen oder wählen Sie im Menü Bearbeiten die Option Parameter einfügen aus.
    Ein Dropdown-Menü mit einer Liste der Parameter wird angezeigt, die hinzugefügt werden können.
    • AccountID: Fügt den SQL-Parameter "AccountID" ein. Dieser Parameter wird durch die Firmen-ID des Kontaktdatensatzes ersetzt, der für den Seriendruck verwendet wird.
    • AccountManagerID: Fügt den SQL-Parameter "AccountManagerID" ein. Dieser Parameter wird durch die ID des Firmen-Verantwortlichen des Kontaktdatensatzes ersetzt, der für den Seriendruck verwendet wird.
    • AddressID: Fügt den SQL-Parameter "AddressID" ein. Dieser Parameter wird durch die Adressen-ID des Kontaktdatensatzes ersetzt, der für den Seriendruck verwendet wird.
    • ContactID: Fügt den SQL-Parameter "ContactID" ein. Dieser Parameter wird durch die Kontakt-ID des Kontaktdatensatzes ersetzt, der für den Seriendruck verwendet wird.
    • OpportunityID: Fügt den SQL-Parameter "OpportunityID" ein. Dieser Parameter wird durch die Verkaufschancen-ID des Kontaktdatensatzes ersetzt, der für den Seriendruck verwendet wird.
    • UserID: Fügt den SQL-Parameter "UserID" ein. Dieser Parameter wird durch die Benutzer-ID des Seriendruckbenutzers ersetzt.
  4. Wählen Sie den Parameter aus, den Sie hinzufügen möchten.
  5. Klicken Sie auf OK.
