Bearbeiten einer Verkaufschance

In der Ansicht "Verkaufschancendetail" können Sie Verkaufschanceninformationen bearbeiten.

So bearbeiten Sie eine Verkaufschance

  1. Klicken Sie auf der Navigationsleiste auf Verkaufschancen.
  2.  Suchen Sie die Verkaufschance, die Sie bearbeiten möchten, und klicken Sie dann auf die Beschreibung der Verkaufschance.
  3. In der Verkaufschancen-Detailansicht können Sie die Verkaufschanceninformationen bearbeiten.
  4. Klicken Sie auf Speichern
