Auswahllisten-Informationsfelder

In der nachfolgenden Tabelle werden die Standardinformationsfelder beschrieben. Weitere Informationen zu Auswahllistenoptionen...

Machen Sie sich vor dem Festlegen von Auswahllistenoptionen mit den Eigenschaften des Steuerelements "Auswahlliste" und der Speicherung von Auswahllistendaten in der Datenbank vertraut. Sie müssen wissen, wie Daten in der Auswahlliste an die Datenbank gebunden sind, um zu bestimmen, ob eine Option unterstützt wird. Beispiel: Wenn Sie die Auswahlliste "Einstellungen" bearbeiten und "Mehrfachauswahl zulassen" aktivieren, funktioniert diese Eigenschaft nicht richtig, da das Datenbankfeld für die Speicherung großer Textmengen zu klein ist.

Feld Beschreibung/Aktion Gilt für Web Gilt für Network
Name Der Name der Auswahlliste.
Erforderlicher Eintrag Erfordert eine Benutzereingabe, bevor der Vorgang im nächsten Feld fortgesetzt werden kann. Falls ausgewählt, gilt diese Option automatisch für Netzwerk-Auswahllisten und für Web-Auswahllisten dann, wenn die Option "Optionen auf Web-Auswahlliste anwenden" ebenfalls ausgewählt ist. *
Mehrfachauswahl zulassen Erlaubt Benutzern, mehr als einen Eintrag in einer Auswahlliste auszuwählen. Falls ausgewählt, gilt diese Option automatisch für Netzwerk-Auswahllisten und für Web-Auswahllisten dann, wenn die Option "Optionen auf Web-Auswahlliste anwenden" ebenfalls ausgewählt ist. *
Text muss mit einem Listeneintrag übereinstimmen Stellt sicher, dass der eingegebene Wert mit einem Eintrag in der Auswahlliste übereinstimmt. Falls ausgewählt, gilt diese Option automatisch für Netzwerk-Auswahllisten und für Web-Auswahllisten dann, wenn die Option "Optionen auf Web-Auswahlliste anwenden" ebenfalls ausgewählt ist. *
Alphabetisch sortiert Zeigt Auswahllisteneinträge in alphabetischer Reihenfolge an. Ist diese Option ausgewählt, werden Werte für "Reihenfolge" ignoriert. Falls ausgewählt, gilt diese Option automatisch für Netzwerk-Auswahllisten und für Web-Auswahllisten dann, wenn die Option "Optionen auf Web-Auswahlliste anwenden" ebenfalls ausgewählt ist. *
Benutzer können Einträge bearbeiten Lässt zu, dass Benutzer Auswahllisteneinträge ergänzen oder ändern dürfen. Diese Option gilt nur für Network-Auswahllisten. Im Web Client können Benutzer keine Auswahllisteneinträge bearbeiten.  
Optionen auf Web-Auswahlliste anwenden Aktivieren Sie dieses Kontrollkästchen, um die in dieser Ansicht für Web-Auswahllisten ausgewählten Optionen zu verwenden. Ist diese Option deaktiviert, wird die Web-Auswahlliste von den Attributen gesteuert, die in Application Architect für das Steuerelement festlegt sind.  
Aktueller Standardwert Zeigt den Standardwert der Auswahlliste.

* Die Option "Optionen auf Web-Auswahlliste anwenden" muss ebenfalls ausgewählt werden.
