Hinzufügen einer Interessentenquelle zur Liste

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Sie können Interessentenquellen, z. B. Messen und Zeitungsanzeigen, der Liste der vorhandenen Interessentenquellen hinzufügen.

So gehen Sie zum Hinzufügen vor

  1. Klicken Sie im Im Menü Administration auf Neue Interessentenquelle.
  2. Füllen Sie die Informationsfelder für die Interessentenquelle aus.
  • Beschreibung - Geben Sie eine Beschreibung der Interessentenquelle ein.
  • Abk. Beschr. - Geben Sie eine Kurzbeschreibung der Interessentenquelle ein.
  • Typ - Klicken Sie auf die Schaltfläche Dropdown-Pfeil , und wählen Sie einen Eintrag in der Liste aus.
  • Status - Klicken Sie auf die Schaltfläche Dropdown-Pfeil , und wählen Sie einen Eintrag in der Liste aus.
  • Kosten - Geben Sie die mit dem Generieren der Interessentenquelle verbundenen Kosten ein.
  • Datum - Zum Bearbeiten des Datums klicken Sie auf die Schaltfläche Kalender, wählen das gewünschte Datum aus und klicken auf OK.
  1. Klicken Sie auf Speichern.
