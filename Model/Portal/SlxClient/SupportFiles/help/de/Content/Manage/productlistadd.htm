Hinzufügen eines Produkts zur Liste

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Sie können Produkte der Produktliste hinzufügen. Diese Liste steht zur Verfügung, wenn Verkaufschancen- oder Verkaufsauftragsprodukte hinzugefügt oder bearbeitet werden.

Produkte sind schreibgeschützt, wenn die Buchhaltungsintegration aktiviert ist. Sie müssen Produkte in Ihrem Buchhaltungssystem verwalten und Änderungen mit Infor CRM synchronisieren.

So gehen Sie zum Hinzufügen vor

  1. Klicken Sie im Im Menü Administration auf Neues Produkt.

Der Dialog "Produkt einfügen" wird angezeigt.

  1. Füllen Sie die Produktfelder aus:
  • Name - Geben Sie den Namen des Produkts ein bzw. bearbeiten Sie diesen.
  • Beschreibung - Geben Sie eine Beschreibung für das Produkt ein bzw. bearbeiten Sie diese.
  • SKU - Geben Sie die Produkt-SKU ein oder bearbeiten Sie sie.
  • Familie – Klicken Sie auf den Schaltfläche Dropdown-Pfeil , und wählen Sie einen Eintrag in der Liste aus.
  • Status - Klicken Sie auf die Schaltfläche Dropdown-Pfeil , und wählen Sie einen Eintrag in der Liste aus.
  • Kosten - Geben Sie die Produktkosten ein oder bearbeiten Sie diese.
  •  Maßeinheit - Der Maßstab zum Bemessen der Menge des Produkts. Beispiel: "Jede" oder "10-er Paket".

  2. Führen Sie die folgenden Schritte durch, um eine Preisstufe hinzuzufügen:
  3. Öffnen Sie in der Produktdetailansicht die Registerkarte Preisstufen, und klicken Sie auf Preis hinzufügen.
