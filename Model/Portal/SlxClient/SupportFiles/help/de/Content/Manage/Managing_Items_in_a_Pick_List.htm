Verwalten von Einträgen in einer Auswahlliste

Sie können eine Auswahlliste bearbeiten, indem Sie sie umbenennen, einen vorhandenen Eintrag bearbeiten oder einen Eintrag aus der Auswahlliste löschen.

Wenn Sie der Auswahlliste "Status der Verkaufschance" einen benutzerdefinierten Eintrag hinzufügen und ein Benutzer den Status der Verkaufschance auf diesen Eintrag festlegt, verhält sich die Momentaufnahme der Verkaufschancen-Detailansicht so, als wäre der Status der Verkaufschance auf "Offen" festgelegt worden.

So erfolgt die Verwaltung

  1. Öffnen Sie die Auswahllisten-Detailansicht.
  2. Führen Sie auf der RegisterkarteEinträge einen der folgenden Schritte aus:
  • Klicken Sie auf Hinzufügen, um einen neuen Eintrag hinzuzufügen.
  • Klicken Sie für den Eintrag, das Sie ändern möchten, auf den Hyperlink Bearbeiten.
  • Klicken Sie für den Eintrag, den Sie entfernen möchten, auf den Hyperlink Löschen.

Wenn Sie andere Softwareanwendungen verwenden, die gemeinsam mit Infor CRM auf Datenbankinformationen zugreifen, müssen Sie in beiden Anwendungen die gleichen Auswahllisteneinträge löschen.

  1. Im Dialog Eintrag hinzufügen/bearbeiten:
  • Geben Sie im Feld Text den Text ein, der in der Auswahlliste enthalten sein soll.
  • (Optional) Geben Sie in das Feld Code einen Code für den Eintrag ein.

Wenn Sie andere Softwareanwendungen verwenden, die gemeinsam mit Infor CRM auf Datenbankinformationen zugreifen, müssen Sie in beiden Anwendungen die gleichen Auswahllistencodes verwenden.

  • Geben Sie im Feld Reihenfolge die Reihenfolge an, in der der Eintrag in der Auswahlliste enthalten sein soll. Weitere Informationen...

Wenn der Eintrag beispielsweise an erster Stelle in der Liste angezeigt werden soll, geben Sie 0 ein. Wenn es mehrere Einträge mit derselben Reihenfolgennummer gibt, werden diese gemäß dieser Nummer gruppiert, und die gesamte Gruppe wird in der Liste vor der nächsthöheren Reihenfolgennummer angezeigt. Bei Auswahl der Option Alphabetisch sortiert werden Reihenfolgenwerte ignoriert.

  • Aktivieren Sie das Kontrollkästchen Ist Standardeintrag, um den Eintrag als Standardlisteneintrag festzulegen.
  • Klicken Sie auf Speichern und neu, um weitere Einträge hinzuzufügen, oder auf OK, wenn Sie fertig sind.
  1. Klicken Sie auf Speichern.

    Nach dem Ändern von Einträgen in einer Auswahlliste müssen Sie die Benutzer möglicherweise auffordern, zu den Optionen zu navigieren und die Auswahllistendaten zu aktualisieren, damit ihnen die neuesten Änderungen angezeigt werden.
