Hinzufügen und Bearbeiten von Produktpreisen

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Sie können Produktpreisinformationen hinzufügen oder bearbeiten.

Vorgehensweise

Hinzufügen einer Preisstufe

  1. Führen Sie einen der folgenden Schritte durch:

    • Öffnen Sie in der Ansicht Produktdetail die Registerkarte Preisstufen.
    • Öffnen Sie die Ansicht Produkt einfügen.
  2. Klicken Sie auf die Schaltfläche Preis hinzufügen.
  3. Fügen Sie die folgenden Informationen hinzu:
    • Beschreibung – Geben Sie eine Preisbeschreibung ein.
    • Preis - Geben Sie den Preis des Produkts ein bzw. bearbeiten Sie diesen.
    • Standard - Geben Sie den Standardpreis des Produkts an. Sie können nur jeweils einen Standardpreis wählen. Wenn Sie einen weiteren auswählen, ist der erste Preis nicht mehr ausgewählt.

  4. Klicken Sie auf OK.

Bearbeiten einer Preisstufe

  1. Führen Sie einen der folgenden Schritte durch:

    • Öffnen Sie in der Produktdetailansicht die Registerkarte Preisstufen.
    • Öffnen Sie die Ansicht Produkt einfügen.
  2. Klicken Sie neben dem zu bearbeitenden Preis auf Bearbeiten.
  3. Bearbeiten Sie beliebige der folgenden Informationen:
    • Beschreibung – Bearbeiten Sie den Preis des Produkts.
    • Preis - Bearbeiten Sie den Preis des Produkts.
    • Standard – Geben Sie den Standardpreis des Produkts an. Sie können nur jeweils einen Standardpreis wählen. Wenn Sie einen weiteren auswählen, ist der erste Preis nicht mehr ausgewählt.

  4. Klicken Sie auf OK.

Verwandte Themen

Verwenden der Registerkarte "Produktpreise"
