Verwenden der Suchfunktion

Im Rahmen der Suchfunktion können Sie Kriterien aus einer Auswahlliste auswählen oder einen Text eingeben. Zum Anzeigen von Ergebnissen müssen Sie für die Abfrage mindestens einen Suchparameter angeben. Sie können Ihre Suche einschränken, indem Sie weitere Bedingungen hinzufügen.  Sie können eine Suche auch als Gruppe speichern.

So führen Sie eine Suche durch

  1. Führen Sie einen der folgenden Schritte durch:
    • Öffnen Sie die Detail- oder Listenansicht für den gesuchten Datensatztyp, und klicken Sie auf die Registerkarte Suchen.  (Z. B. Kontakt).
    • Klicken Sie in der Navigationsleiste mit der rechten Maustaste auf den zu suchenden Datensatz, und klicken Sie auf Suchen.... (Zum Beispiel Firmen suchen....)
  2. Wählen Sie im Feld Suchen nach eine Suchbedingung in der Dropdown-Liste aus.

    		3.

    Die Optionen unter Suchen nach hängen von den Spalten ab, die im Layout der Suche enthalten sind. Diese Spalten werden von der Gruppe bestimmt, die Sie in Ihren Gruppenoptionen als Standard-Suchlayout festgelegt haben. Um weitere Sucheinträge in die Liste aufzunehmen, aktualisieren Sie entweder die Gruppe mit dem Standard-Suchlayout, damit diese die bei der Suche zu verwendende Spalte enthält, oder legen für das Standard-Suchlayout eine andere Gruppe fest, die die Spalte im Layout enthält.
  3. Wählen Sie in der Dropdown-Liste einen Suchbedingungsoperator aus. Beispiel: "Enthält" oder "Beginnend mit".
  4. Geben Sie im nächsten Feld Ihre Suchkriterien ein. Diese müssen mit der in Schritt 2 ausgewählten Suchbedingung übereinstimmen.
     Wenn Sie beispielsweise nach der Firma Aberle Druckguss GmbH suchen, geben Sie "ab" in dieses Feld ein.
  5. Um ein weiteres Suchkriterium hinzuzufügen, klicken Sie auf die Schaltfläche Bedingung hinzufügen und wiederholen zu jedem hinzugefügten Suchkriterium die Schritte 2 bis 4.
  6. Wenn Sie all Ihre Suchanforderungen eingegeben haben, klicken Sie auf Suchen.
    Die Datensätze, die Ihren Kriterien entsprechen, werden in der Registerkarte "Suchergebnisse" angezeigt.

Alle Suchvorgänge werden für die Datensätze durchgeführt, auf die Sie zugreifen können. Abhängig von den vom Administrator festgelegten Optionen können Sie auch Datensätze auf der Grundlage der Gruppe suchen, der sie angehören.
