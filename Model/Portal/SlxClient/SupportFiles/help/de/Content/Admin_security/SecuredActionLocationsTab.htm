Registerkarte "Standorte"

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In der Detailansicht "Gesicherte Aktion" können Sie auf der Registerkarte "Standorte" erkennen, wo die gesicherte Aktion verwendet wird. Der Dateiname und Serverpfad der Website wird für jedes Element angezeigt. Diese Informationen sind schreibgeschützt.

Wenn Sie eine gesicherte Aktion löschen oder umbenennen, wird die gesicherte Aktion in Application Architect vom Element getrennt. Infor CRM empfiehlt empfiehlt, vor dem Ändern der gesicherten Aktion die Informationen auf dieser Registerkarte als Historiendatensatz zu speichern. Sie können die Informationen kopieren und in eine Textdatei einfügen.

Bei Bedarf können Sie anhand der Informationen auf dieser Registerkarte eine gesicherte Aktion wiederherstellen oder die Aktion aus Elementen in Application Architect entfernen.
  • Beim ersten Zugriff auf diese Registerkarte kann das Laden der Informationen lange dauern.
  • Wenn Sie gespeicherte Aktionen ändern, werden die Informationen auf dieser Registerkarte aktualisiert, sobald Sie Ihre Website neu erstellen und bereitstellen.
