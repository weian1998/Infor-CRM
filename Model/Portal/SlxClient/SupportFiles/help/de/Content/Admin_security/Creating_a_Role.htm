Erstellen von Rollen

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

Berechtigungen für Features und Funktionen im Web Client werden von Rollen bestimmt. Sie können beliebig viele Rollen erstellen und diese einer beliebigen Anzahl von Benutzern zuweisen. Alternativ können Sie mit einer Standardrolle arbeiten.

Einem Benutzer können mehrere Rollen zugewiesen werden. Wenn ein Benutzer mehreren Rollen zugewiesen ist, hat er Zugriff auf alle gesicherten Aktionen in diesen Rollen, auch wenn die einzelnen Rollen nicht dieselben Aktionen enthalten. Eine Abteilung oder ein Team kann keiner Rolle zugewiesen werden.

So gehen Sie zum Erstellen vor

  1. Klicken Sie im Menü Administration auf Neue Rolle.
  2. Geben Sie in das Feld Name einen Namen für die Rolle ein.
  3. Geben Sie im Feld Beschreibung beschreibende Informationen zur Rolle ein.
  4. Klicken Sie auf Speichern.
  5. Auf der Registerkarte Aktionen können Sie der Rolle Aktionen hinzufügen.
  6. Auf der Registerkarte Benutzer können Sie Benutzer der Rolle hinzufügen oder aus ihr entfernen.
