Verwenden der Detailansicht "Gesicherte Aktion"

Der Systemadministrator und Benutzer mit der entsprechenden Rolle können auf diese Funktionen zugreifen.

In der Detailansicht können Sie Informationen für einen ausgewählten Datensatz anzeigen, hinzufügen oder bearbeiten. Die Detailansicht umfasst Informationsfelder, Registerkarten und einen benutzerdefinierten mittleren Bereich.

Vorgehensweise

Erstellen gesicherter Aktionen

Öffnen der Detailansicht "Gesicherte Aktion"

  1. Klicken Sie auf der Navigationsleiste auf Administration und dann auf Gesicherte Aktionen, um die Listenansicht Gesicherte Aktionen zu öffnen.
  2. Klicken Sie auf eine gesicherte Aktion in der Liste oder nutzen Sie die Suchfunktion, um eine gesicherte Aktion auszuwählen.

Die Informationen zur gesicherten Aktion werden in der Detailansicht geöffnet.

Verwenden Sie die Suchfunktion, um nach Datensätzen zu suchen.

Öffnen einer Gruppe

Umschalten zur Listenansicht

  • Klicken Sie auf der Symbolleiste auf die Schaltfläche Listenansicht.

Informationsfelder

Die Informationsfelder zur gesicherten Aktion enthalten die relevanten dazugehörigen Informationen.

  • Gesicherte Aktion - Der Name der gesicherten Aktion.
  • Beschreibung - Beschreibende Informationen zur gesicherten Aktion.

Detailansicht-Registerkarten

Auf jeder Registerkarte werden für die betreffende gesicherte Aktion bestimmte Informationen angezeigt. Klicken Sie auf einen Hyperlink, um weitere Informationen zu jeder Registerkarte anzuzeigen.

Benutzerdefinierter Abschnitt

Sie können Registerkarten per Ziehen und Ablegen in den benutzerdefinierten mittleren Bereich verschieben. Die Registerkarten bleiben dort, bis sie wieder in den unteren Bereich verschoben werden.
