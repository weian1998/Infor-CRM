Zuweisen von Benutzern zur Rolle "Standardbenutzer"

Standardmäßig wird im Web Client erstellten Benutzern die Rolle "Standardbenutzer" zugewiesen. Diese Rolle erlaubt das Hinzufügen, Bearbeiten und Löschen von Berechtigungen für nicht administrative Web Client-Entitäten wie Firmen, Kontakte und Verkaufschancen.

In Administrator erstellten Benutzern wird keine Rolle zugewiesen, und die Rollensicherheitseinstellungen können nicht in Administrator festgelegt werden. Wenn Benutzer vorhanden sind oder Sie neue Benutzer in Administrator erstellen, müssen Sie die Benutzer der Rolle Standardbenutzer zuweisen.

So gehen Sie zum Zuweisen vor

  1. Öffnen Sie die Benutzerlistenansicht.
  2. Wählen Sie die Benutzer aus, die über Hinzufüge-, Bearbeitungs- und Löschberechtigungen für nicht administrative Entitäten im Web Client verfügen sollen.
  3. Klicken Sie im Bereich Benutzeraufgaben auf Zu Rolle hinzufügen.
  4. Wählen Sie im Dialogfeld Rolle auswählen die Rolle Standardbenutzer aus, und klicken Sie auf OK.

  5. Klicken Sie auf OK.
