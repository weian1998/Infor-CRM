Sich wiederholende Aktivitäten - Täglich

Sie können festlegen, ob eine Aktivität Einmal, Täglich, Wöchentlich, Monatlich oder Jährlich stattfinden soll.

So planen Sie eine tägliche periodische Aktivität

  1. Planen oder bearbeiten Sie eine Aktivität.
  2. Klicken Sie auf die Registerkarte Periodisch.
  3. Wählen Sie unter Diese Aktivität findet statt... die Option Täglich aus.
  4. Wählen Sie Alle X Tage aus. Sie können auch Alle X Tage nach Abschluss auswählen, wenn die Aktivität nur nach Abschluss der vorherigen Aktivitäten wiederholt werden soll.
  5. Geben Sie die Anzahl der Tage zwischen den Wiederholungen ein.
  6. Klicken Sie im Feld Sich wiederholende Aktivität starten auf die Schaltfläche  Kalender  , wählen Sie ein Datum aus, und klicken Sie dann auf OK.

  7. Führen Sie einen der folgenden Schritte durch:

    • Geben Sie in das Feld Ende nach ein, wie oft die Aktivität sich wiederholen soll.
    • Klicken Sie unter Wiederholung beenden auf die Schaltfläche  Kalender  , wählen Sie ein anderes Datum aus, und klicken Sie dann auf OK. 
  8. Wenn Sie alle Aktivitätsänderungen vorgenommen haben, klicken Sie auf OK.
