Registerkarte "Alarme"

Auf der Registerkarte "Alarme" in der Ansicht "Aktivitäten" werden Aktivitäten mit aktiven Alarmen aufgelistet. In dieser Registerkarte können Sie Alarme zur Wiedervorlage auswählen bzw. verwerfen.

Wenn Sie für jede geplante Aktivität einen automatischen Alarm festlegen möchten, wählen Sie im Menü "Extras" den Befehl Optionen und klicken dann auf die Registerkarte Registerkarte Aktivitäten. Wählen Sie unter "Standardeinstellungen pro Aktivitätstyp" Standard-Alarmeinstellungen für jeden Aktivitätstyp aus.

Vorgehensweise

Anzeigen eigener Alarme
  1. Klicken Sie in der Ansicht Aktivitäten auf Alarme.
  2. Grenzen Sie die Liste ggf. mithilfe der Filter ein.
Ein- und Ausblenden des Detailfensters
  1. Klicken Sie auf Detail.
  2. Wählen Sie einen Datensatz aus.
    Das Detailfenster wird im unteren Teil der Listenansicht angezeigt und enthält detaillierte Informationen über den ausgewählten Datensatz.
  1. Zum Ausblenden des Detailfensters klicken Sie auf Details ausblenden.
Anzeigen der Zusammenfassungsansicht
  1. Klicken Sie auf Zusammenfassung.
  2. Um zur Listenansicht zurückzukehren, klicken Sie auf Liste.
Wiedervorlegen eines oder mehrerer Alarme
  1. Wählen Sie den bzw. die Alarme aus, die Sie wiedervorlegen möchten. Wenn Sie mehrere Datensätze markieren möchten, dann halten Sie [STRG] oder [Umschalt] gedrückt und markieren die Datensätze..
  2. Klicken Sie auf die Schaltfläche Erinnerungsalarm.
  3. (Optional) Wenn Sie das Wiedervorlageintervall ändern möchten, klicken Sie auf den Dropdown-Pfeil neben Wiedervorlegen in und wählen einen Wert aus der Liste. Der Standardwert für Wiedervorlegen in basiert auf der Standardwiedervorlageoption für Warnungen.
  4. Klicken Sie auf Wiedervorlegen.
Ablehnen eines oder mehrerer Alarme
  1. Wählen Sie den bzw. die Alarme aus, die Sie ablehnen möchten. Wenn Sie mehrere Datensätze markieren möchten, dann halten Sie [STRG] oder [Umschalt] gedrückt und markieren die Datensätze.
  2. Klicken Sie auf die Schaltfläche Alarm ablehnen.

Verwenden des Kontextmenüs

Aktivität abschließen
  1. Führen Sie einen der folgenden Schritte durch:
    • Klicken Sie in der Ansicht Zusammenfassung oder im Detailfenster auf die Schaltfläche Abschließen der Aktivität, die Sie abschließen möchten.
    • Wählen Sie die Aktivität, die Sie abschließen möchten, und klicken Sie im Aufgabenbereich Allgemeine Aufgaben auf Aktivität abschließen.
    • Klicken Sie mit der rechten Maustaste auf die gewünschte Aktivität, und klicken Sie auf Aktivität abschließen.
  2. Führen Sie die Schritte zum Abschließen einer Aktivität aus.

Abschließen mehrerer Aktivitäten

Löschen einer Aktivität
  1. Wählen Sie die Aktivität, die Sie löschen möchten, und klicken Sie im Aufgabenbereich Allgemeine Aufgaben auf Aktivität löschen.
  2. Klicken Sie auf OK.
Löschen mehrerer Aktivitäten
  1. Wählen Sie die Aktivitäten aus, den Sie löschen möchten.
  2. Klicken Sie im Aufgabenbereich Allgemeine Aufgaben auf Aktivität löschen.
  3. Klicken Sie auf OK.
Anruf planen

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Anruf.
    • Klicken Sie im Raster mit der rechten Maustaste Planen, und klicken Sie dann auf Anruf.
  2. Befolgen Sie die Anweisungen zum Planen eines Anrufs.
Meeting planen

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Meeting.
    • Klicken Sie im Raster mit der rechten Maustaste auf Planen, und klicken Sie dann auf Meeting.
  2. Befolgen Sie die Anweisungen zum Planen eines Meetings.
Aufgabe planen

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Aufgabe.
    • Klicken Sie im Raster mit der rechten Maustaste auf Planen, und klicken Sie dann auf Aufgabe.
  2. Befolgen Sie die Anweisungen zum Planen einer Aufgabe.
Planen einer persönlichen Aktivität

  1. Führen Sie einen der folgenden Schritte durch:

    • Klicken Sie im Aufgabenbereich Allgemeine Aufgaben unter Planen auf Persönliche Aktivität.
    • Klicken Sie im Raster mit der rechten Maustaste auf Planen, und klicken Sie dann auf Persönliche Aktivität.
  2. Befolgen Sie die Anweisungen zum Planen einer persönlichen Aktivität.
