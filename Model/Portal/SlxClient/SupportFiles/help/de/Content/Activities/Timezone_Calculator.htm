Verwenden der Zeitzonenberechnung

Mit der Zeitzonenberechnung können Sie eine Aktivität basierend auf den Zeitzonen der Beteiligten planen. Hierzu ein Beispiel: Sie haben einen Telefonanruf bei einem Kunden für 10:00 Uhr Ihrer Ortszeit geplant und möchten herausfinden, welcher Uhrzeit dies bei Ihrem Kunden entspricht, der sich in einer anderen Zeitzone befindet.

Sie können auch eine Aktivität so planen, dass sie zu einer bestimmten Uhrzeit in einer anderen Zeitzone erfolgt. Beispiel: Sie möchten einen Telefonanruf bei einem Kunden, der sich in einer anderen Zeitzone befindet, für 9:00 Uhr morgens seiner Ortszeit planen und herausfinden, wann Sie diesen Anruf tätigen müssen.

So öffnen Sie die Zeitzonenberechnung

  1. Öffnen Sie eines der Dialogfelder zum Planen (Anruf, Meeting, Aufgabe oder persönliche Aktivität) oder die Ansicht Aktivität bearbeiten.
  2. Klicken Sie auf die Registerkarte Zeitzonenberechnung.

So vergleichen Sie das Anfangsdatum und die Anfangszeit einer Aktivität mit einer anderen Zeitzone

  1. Ändern Sie ggf. die Aktuelle Zeitzone.
  2. Klicken Sie auf den Dropdown-Pfeil von Zeitzonenvergleich, und wählen Sie danach die Zeitzone aus, die Sie vergleichen möchten.
  3. Klicken Sie auf Vergleichen.

Datum und Uhrzeit der Aktivität werden in den Vergleichszeitzoneanzeigen angezeigt.
