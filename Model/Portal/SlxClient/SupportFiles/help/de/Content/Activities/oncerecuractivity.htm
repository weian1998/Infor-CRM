Sich wiederholende Aktivitäten - Einmal

Sie können festlegen, ob eine Aktivität Einmal, Täglich, Wöchentlich, Monatlich oder Jährlich stattfinden soll.

So planen Sie eine sich einmal wiederholende Aktivität

  1. Planen oder bearbeiten Sie eine Aktivität.
  2. Klicken Sie auf die Registerkarte Periodisch.
  3. Wählen Sie unter Aktivität findet statt... die Option Einmal aus. Diese Option ist standardmäßig ausgewählt.
  4. Wenn Sie alle Aktivitätsänderungen vorgenommen haben, klicken Sie auf OK
