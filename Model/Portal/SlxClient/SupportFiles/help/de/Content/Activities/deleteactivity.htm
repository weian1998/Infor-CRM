Löschen einer Aktivität

Sie können alle Aktivitäten löschen, die Sie selbst angelegt haben.

So löschen Sie eine einzelne Aktivität

  1. Führen Sie einen der folgenden Schritte durch:

    • Öffnen Sie die zu löschende Aktivität, und klicken Sie dann auf Löschen.
    • Klicken Sie in einer der Registerkarten zu Aktivitäten auf Abschließen.
    • Klicken Sie in der Listenansicht Aktivitäten mit der rechten Maustaste auf eine Aktivität, und wählen Sie Aktivität löschen aus.
  2. Wenn Sie eine sich wiederholende Aktivität ausgewählt haben und eine entsprechende Aufforderung erhalten, wählen Sie Dieses Vorkommen löschen oder Alle Vorkommen löschen aus und klicken dann auf Fortfahren.
  3. Klicken Sie auf OK, um den Datensatz unwiderruflich löschen, oder klicken Sie auf Abbrechen, um den Löschvorgang abzubrechen.

Wenn Sie eine Aktivität löschen, der Sie als Mitglied zugewiesen sind, wird der Organisator für die Aktivität darüber informiert, dass Sie die Aktivität abgelehnt haben.

So löschen Sie mehrere Aktivitäten

  1. Wählen Sie in der Ansicht Aktivitäten die Aktivitäten aus, die Sie löschen möchten, und klicken Sie im Aufgabenbereich im Bereich Allgemeine Aufgaben auf Aktivität löschen oder klicken Sie mit der rechten Maustaste auf das Raster, und wählen Sie Aktivität löschen aus.

    Wenn es sich bei Aktivitäten in Ihrer Auswahl um sich wiederholende Aktivitäten handelt, wird nur das bzw. die ausgewählten Vorkommen der sich wiederholenden Aktivität gelöscht.
  2. Klicken Sie auf OK, um den Datensatz unwiderruflich löschen, oder klicken Sie auf Abbrechen, um den Löschvorgang abzubrechen.
