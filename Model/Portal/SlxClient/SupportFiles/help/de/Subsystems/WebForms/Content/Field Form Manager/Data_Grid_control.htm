Eigenschaften des Steuerelements "Datenraster"

Sie können die folgenden Eigenschaften dieses Steuerelements anzeigen und deren Werte ändern:

Standardeigenschaft Beschreibung

Beschriftung

Die Bezeichnung, die auf dem Formular dieses Steuerelements angezeigt wird.

Beschriftungsausrichtung

Ausrichtung des Bezeichnungstexts. Folgende Optionen sind verfügbar:

  • Links
  • Rechts
  • Zentriert

Spaltengröße änderbar

Erlaubt dem Benutzer das Ändern der Breite der Rasterspalten.
Erweiterte Eigenschaft Beschreibung

Vertikal wiedergeben

Gibt an, ob die Daten in Rasterzeilen als Spalten angezeigt werden sollen.

Sichtbar

Dient zum Ein- oder Ausblenden dieses Steuerelements auf dem Formular.

Zeilentext in leerer Tabelle

Der Text, der angezeigt werden soll, wenn eine leere Zeile im Datenraster zurückgegeben wird. Beispiel: "Den Auswahlkriterien entsprechen keine Datensätze."

Erweiterbare Zeilen

Stellen Sie ein Symbol zum Ein-/Ausblenden zur Verfügung, damit der Benutzer entweder nur die erste Zeile oder den gesamten Text anzeigen kann.

Datenquelle

Die Quelle der Daten dieses Steuerelements. Dies kann ein anderes Steuerelement oder die Hauptentität der Ansicht sein. Diese Informationen sind im Formular-Designer schreibgeschützt.

Steuerelement-ID

ID dieses Steuerelements. Diese Informationen sind im Formular-Designer schreibgeschützt.

Steuerelementtyp

Infor CRM-Steuerelementtyp, z. B. Textfeld. Diese Informationen sind im Formular-Designer schreibgeschützt.
Eigenschaft "Layout" Beschreibung

Zeile

Zeilennummer des Steuerelements.

Spalte

Spaltennummer des Steuerelements.

Zeilenbreite

Anzahl der Zellen, die das Steuerelement vertikal einnimmt.

Spaltenweite

Anzahl der Zellen, die das Steuerelement horizontal einnimmt.
