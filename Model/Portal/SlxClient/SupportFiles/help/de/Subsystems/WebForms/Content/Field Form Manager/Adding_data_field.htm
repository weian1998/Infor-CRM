Hinzufügen eines Datenfelds zu einem Formular (Hinzufügen eines Steuerelements)

Im Bereich Feld hinzufügen im Formular-Designer können Sie bestehende Datenfelder zu einem Formular hinzufügen. Angezeigt wird eine Struktur aus Datenfeldern, die dem aktuellen Formular hinzugefügt werden können. Sie können Elemente mit einem Pluszeichen in der Struktur erweitern, um ihre Felder zu verwenden. Beispiel: Adresse.Stadt.

Falls das Feld, das Sie Ihrem Formular hinzufügen möchten, in der Datenbank nicht vorhanden ist, muss es hinzugefügt werden. Weitere Informationen finden Sie unter dem Thema "Hinzufügen von Feldern zu Tabellen" in der Administrator-Hilfe oder unter "Ändern einer Entität" und "Einfügen einer neuen Entitätseigenschaft" in der Application Architect-Hilfe.

So fügen Sie Informationen hinzu

  1. Klicken Sie optional zuerst auf Speichern , falls Sie die nachfolgenden Schritte rückgängig machen müssen.
  2. Wählen Sie ein Element im Bereich Feld hinzufügen aus, und ziehen Sie es.

  3. Legen Sie das Element an einer Position ab, die hervorgehoben wird, wenn Sie den Mauszeiger darüber bewegen.

    Ein Steuerelement wird erstellt und an das Datenfeld in der Datenbank gebunden. Der Steuerelementtyp wird basierend auf dem Datentyp für Sie festgelegt. Durch Ablegen einer Telefonnummer auf dem Formular wird beispielsweise das Steuerelement Telefon erstellt.

  4. Optional können Sie Eigenschaftswerte des neuen Steuerelements bearbeiten.
  5. Wenn Sie sicher sind, dass Sie das hinzugefügte Feld behalten möchten, klicken Sie auf Speichern. Andernfalls drücken Sie F5, um zur letzten gespeicherten Version des Formulars zurückzukehren.
