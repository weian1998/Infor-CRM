Utilisation de la vue Liste des tickets

Vous pouvez utiliser la vue Tickets pour rechercher un ticket, afficher les tickets par groupe, ajouter un nouveau ticket ou effectuer des recherches dans les index.

Pour rechercher un ticket

  1. Dans la barre de navigation, cliquez sur Tickets.
  2. Utilisez la fonction Rechercher pour rechercher des enregistrements de tickets.

Les recherches sont effectuées sur les tickets auxquels vous avez accès.

  1. Cliquez sur le lien hypertexte du N° de Ticket pour ouvrir la vue Détails du ticket.

Pour afficher les tickets par groupe

  • Dans la vue Liste de tickets, dans la liste de Groupe de tickets, sélectionnez Tous les tickets du portail Web, Tickets fermés du portail Web ou Tickets ouverts du portail Web.

La liste des tickets du groupe sélectionné s'affiche dans la partie inférieure de la vue.

Pour ajouter un nouveau ticket

  1. Dans la barre d'outils, cliquez sur Ajouter un ticket.
  2. Suivez les étapes d'ajout d'un ticket.

Pour utiliser SpeedSearch

  1. Dans la barre de navigation, cliquez sur SpeedSearch.
  2. Suivez les étapes de recherche d'informations.
