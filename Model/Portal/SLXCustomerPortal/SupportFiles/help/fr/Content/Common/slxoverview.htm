Présentation de Portail client Infor CRM

Portail client Infor CRM est une solution en libre service qui permet d'ajouter et de modifier les tickets. Vous pouvez cliquer sur Ajouter un ticket dans la barre de menus pour ajouter un nouveau ticket ou utiliser la vue Liste des tickets pour rechercher des tickets existants. Dans la vue Détails du ticket, vous pouvez afficher les informations sur les tickets, ajouter ou modifier une pièce jointe ou ajouter des commentaires.

SpeedSearch permet de rechercher des informations stockées dans Infor CRM.

Ce système d'aide explique comment utiliser Portail client Infor CRM.
