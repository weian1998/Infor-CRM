Dépannage des problèmes courants

Vous trouverez ci-après une liste des problèmes courants :

Problème :

Mes requêtes renvoient aux mêmes données et j'obtiens toujours les mêmes pages.

Je dois effacer mon cache

Solution :

Il est possible que votre navigateur ne traite pas correctement certaines balises et récupère les pages dans le cache (dossier où sont stockées les pages Web) au lieu de les récupérer sur le serveur, lors de votre utilisation du client Web. Vous devez paramétrer les options d'affichage de votre navigateur de telle sorte qu'il récupère les données sur le serveur et non pas dans le cache.

Pour des étapes détaillées, veuillez consulter l'aide de votre navigateur.

Problème :

Le fonctionnement de l'aide Web Infor CRM est différent dans Firefox

Si vous utilisez Firefox en tant que navigateur, l'aide Web Infor CRM s'ouvre dans un nouvel onglet dans la même fenêtre du navigateur et redimensionne la fenêtre. Toutefois, vous pouvez définir des options qui vont empêcher la fenêtre de se redimensionner ou ouvrir de nouvelles pages dans une fenêtre distincte.

La modification de ces options va avoir des conséquences sur le comportement des onglets et des fenêtres pour tous les sites Web et pas seulement pour les pages concernant Infor CRM.

Solutions :

Pour ouvrir toutes les nouvelles pages dans une fenêtre distincte

  1. Dans le menu Outils de Firefox, cliquez sur Options.
  2. Cliquez sur Onglets.
  3. Sélectionnez Ouvrir de nouvelles fenêtres dans un nouvel onglet à la place.
  4. Cliquez sur OK.

Pour empêcher la fenêtre de se redimensionner

  1. Dans le menu Outils de Firefox, cliquez sur Options.
  2. Cliquez sur Contenu.
  3. En regard de Activer Javascript cliquez sur Avancé.
  4. Désélectionnez l'option permettant de Déplacer ou redimensionner les fenêtres existantes, puis cliquez sur OK.
  5. Cliquez sur OK.

Problème :

J'ai modifié les paramètres de mon navigateur ou mes paramètres régionaux, mais certaines informations apparaissent dans la mauvaise langue.

Solution :

  • Vos paramètres régionaux et paramètres de navigateur doivent être les mêmes. Vérifiez que vos paramètres régionaux et la langue de votre navigateur sont définis sur la même langue.
