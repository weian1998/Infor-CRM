Aperçu des résultats de SpeedSearch

Après avoir effectué une recherche et sélectionné un élément, utilisez l'onglet Aperçu pour afficher des informations détaillées sur cet élément.  

Pour rechercher rapidement les informations appropriées

  1. Exécutez SpeedSearch et cliquez sur le bouton Rechercher.
  2. Si cela est nécessaire, utilisez les boutons Première page, Page précédente, Page suivante et Dernière page pour visualiser les enregistrements renvoyés et chercher celui que vous voulez prévisualiser.
  3. Cliquez sur Aperçu(lien en rouge sous l'enregistrement).

  4. Effectuez l’une des opérations suivantes :

    • Cliquez sur Rechercher le premier résultat pour accéder à la première instance d'un mot-clé.
    • Cliquez sur les boutons Précédent ou Suivant situés sur le côté d'un mot-clé pour accéder à l'instance précédente ou suivante de ce mot-clé.
    • Utilisez la barre de défilement interne pour passer en revue l'enregistrement ou le fichier.

Si la zone Aperçu affiche un message « Accès refusé », vous n'avez pas les droits d'accès nécessaires pour consulter le résultat. Consultez votre administrateur système pour obtenir l'accès.

Pour marquer un résultat comme étant utilisé

Si vous trouvez un élément utile, mais que vous n'êtes pas en phase d'insertion des résultats, procédez comme suit :

  • Sélectionnez l'élément et cliquez sur Marquer cet élément comme utile. Cela permet au système de garder une trace des éléments fréquemment utilisés.
