Modification des informations de pièce jointe

Utilisez cette vue pour modifier la description de la pièce jointe et télécharger une nouvelle version d'une pièce jointe. Les pièces jointes se synchronisent.

Par défaut, il est impossible de joindre des fichiers dont la taille dépasse 4 Mo. Dans un tel cas, vous recevrez un message d'erreur sauf si votre administrateur a personnalisé vos paramètres.

Pour modifier des informations de pièce jointe

  1. Ouvrez l'onglet Pièces jointes.
  2. Sélectionnez la pièce jointe à modifier puis cliquez sur Modifier.
  3. Le cas échéant, mettez à jour la description.
  4. Effectuez l'une des opérations suivantes :
    • Pour télécharger un autre fichier, cliquez sur Sélectionner un autre fichier, recherchez le nouveau fichier à télécharger puis cliquez sur Ouvrir.

      Si vous modifiez une pièce jointe, vous devez télécharger le nouveau fichier afin que ces modifications soient disponibles dans Infor CRM. Toute modification effectuée se fait sur une copie locale du fichier et non sur le fichier joint.

    • Pour utiliser une URL différente, saisissez la nouvelle URL dans la zone URL.

  5. Cliquez sur OK.
