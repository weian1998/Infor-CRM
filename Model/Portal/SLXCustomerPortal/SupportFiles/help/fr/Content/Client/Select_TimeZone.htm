Sélection d'un fuseau horaire

Le Portail client Infor CRM affiche votre fuseau horaire dans la barre d'état au bas de la fenêtre de votre navigateur. Le fuseau horaire par défaut est basé sur le paramètre Fuseau horaire de votre ordinateur local. Si le fuseau horaire est incorrect, vous pouvez passer à d'autres paramètres locaux avec le même fuseau horaire.

Pour modifier votre fuseau horaire

  1. Double-cliquez sur le fuseau horaire qui apparaît dans la barre d'état.
  2. Dans la boîte de dialogue Paramètres de fuseau horaire, cliquez sur le la flèche du menu déroulant et effectuez votre sélection dans la liste.

  3. Cliquez sur OK.
