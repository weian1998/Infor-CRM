Gestionnaire de groupes

Utilisez la liste Groupes pour afficher et gérer les groupes dans les vues Liste Tickets ou Détails des tickets. Vous pouvez rechercher, afficher les groupes masqués et masquer des groupes.

Procédure

Ouvrir le Gestionnaire de groupes

  • Dans une liste de détails ou vue, cliquez sur le bouton Groupes.
Rechercher un groupe
  • Dans le champ Rechercher, saisissez les premières lettres d'une partie du nom du groupe que vous recherchez, puis cliquez sur Rechercher.
    Pour inclure les groupes masqués à votre recherche, sélectionnez Afficher les groupes masqués.
  • Cliquez sur Effacer pour supprimer le contenu du champ Rechercher et démarrer une nouvelle recherche.
Afficher ou masquer les groupes masqués dans la liste
  • Sélectionnez l'option Afficher les groupes masqués pour afficher les groupes masqués dans la liste ou décochez cette option pour les masquer.
Développer ou trier une colonne
  • Pour développer la colonne, cliquez sur le séparateur de colonne et maintenez le bouton de la souris enfoncé tout en faisant glisser le séparateur jusqu'à la taille voulue.
  • Pour trier par la colonne Groupe, cliquez sur l'en-tête de la colonne. Vous pouvez trier par ordre croissant ou décroissant.
Afficher ou masquer un groupe
  1. Sélectionnez l'option Afficher les groupes masqués.
  2. Dans la colonne Visible, cochez les cases des groupes que vous voulez afficher et décochez les groupes que vous voulez masquer.
