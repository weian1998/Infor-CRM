Beispiel-Hilfethema

Dies ist ein Beispiel-Hilfethema.

Ersetzen Sie den Text in diesem Thema durch die Anweisungen zur Verwendung Ihrer benutzerdefinierten Ansicht und speichern Sie das Thema anschließend unter Verwendung der vorgeschlagen Namenskonventionen.

Weitere Informationen zum Hinzufügen von benutzerdefinierten Web-Hilfethemen finden Sie im Thema "Web-Hilfe - Übersicht" in der Hilfe zur Anwendung Architect.

Richtlinien für Namenskonventionen

Wenn Sie eine benutzerdefinierte Hilfe für Listenansichten erstellen, müssen Sie die folgenden Namenskonventionen verwenden, damit Ihre Hilfe richtig angezeigt wird:

 

entitylistview.htm - Beispiel: accountlistview.htm

Dabei ist "entity" der Name der mit der benutzerdefinierten Listenansicht verbundenen Entität.

Tipps zum Verfassen von Hilfethemen

Wenn Sie ein benutzerdefiniertes Web-Hilfethema für eine Listenansicht erstellen, kann es hilfreich sein, wenn Sie einen zusammenfassenden oder einleitenden Abschnitt einfügen, in dem die Arten von Datensätzen, die in der Listenansicht angezeigt werden, kurz beschrieben sind. Sie können auch mit aufnehmen, wie die Listenansicht geöffnet wird, wie mit Gruppen gearbeitet wird oder wie beliebige andere Aufgaben ausgeführt werden, die in der Listenansicht ausgeführt werden können.
