Voraussetzungen und Konventionen

In diesem Online-Handbuch wird vorausgesetzt, dass Sie mit der Handhabung von Browsern (z. B. Benutzen von Suchprogrammen, Eingeben einer URL) und grundlegenden Betriebssystemkonzepten (z. B. Klicken und Ziehen, Dateisuche) vertraut sind. Sollten Sie damit nicht vertraut sein, ziehen Sie die zur Verfügung stehenden Browser-Hilfedateien zu Rate.

 

Konventionen

In diesem Hilfesystem werden folgende Textkonventionen verwendet:

  • Soweit keine anders lautenden Angaben gemacht werden, bezieht sich der Begriff "Menüleiste" auf die Menüleiste im Browser-Fenster.
  • Der Begriff "Navigationsleiste" bezieht sich auf den Frame links im Browser-Fensters, der unter anderem Schaltflächen für Kontakte, Firmen, Verkaufschancen und Neuigkeiten enthält.
  • "Drücken Sie die Eingabetaste" wird immer dann verwendet, wenn Sie die Eingabetaste (Enter- oder Return-Taste) drücken sollen. Drücken Sie die Eingabetaste nicht, wenn Sie sich in einem Feld befinden. Falls Sie dies tun, erhalten Sie eine Fehlermeldung. Verwenden Sie die Tabulatortaste, um von Feld zu Feld zu navigieren.
  • Eine "Notiz" enthält in der Regel wichtige Informationen oder die Lösung zu einem eventuellen Problem.
  • Sofern nicht anders angegeben, betreffen alle Informationen sowohl den Infor CRM Kundenportal als auch den WebViewer.

Wichtige Informationen werden durch einen Hinweis gekennzeichnet.
