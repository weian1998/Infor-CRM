Verwenden des Hilfesystems

Das Hilfesystem, das im Zusammenhang mit Infor CRM zur Verfügung steht, enthält Erklärungen der einzelnen Funktionen sowie Hyperlinks zu verwandten Themen.

  • Wenn Sie auf eine Hilfe-Schaltfläche klicken, wird das entsprechende Hilfethema geöffnet.
  • Wenn Sie auf blaue Links klicken, wechseln Sie zu einem anderen Thema, in dem zusätzliche Informationen angezeigt werden.
  • Wenn Sie auf blaue Links klicken, werden zusätzliche Informationen eingeblendet.

zusätzliche Informationen

  • Auch die Abschnitte "Vorgehensweise" und "Verwandte Themen", die in vielen Themen enthalten sind, bietet weiterführende Informationen.
  • Im Inhaltsverzeichnis finden Sie Informationen nach der Funktion gruppiert.
  • Über die Funktion Suchen können Sie Informationen durch Eingabe eines Suchbegriffs suchen.
  • Über die Schaltfläche Zurück oben im Hilfefenster können Sie zum vorherigen Thema wechseln.
  • Klicken Sie auf die Registerkarte Favoriten, um die am häufigsten verwendeten Hilfethemen zu verwalten.

Hier können Sie die in diesem Hilfesystem verwendeten Konventionen nachlesen.
