Introducing Infor CRM Customer Portal

Infor CRM Customer Portal is a self-service solution that enables you to add and edit tickets. You can click Add Ticket on the menu bar to add a new ticket, or use the Tickets List view to find an existing ticket. From the Ticket Detail view, you can view ticket information, add or edit an attachment or add comments.

SpeedSearch allows you to find information stored in Infor CRM.

This Help system explains how to use Infor CRM Customer Portal.
