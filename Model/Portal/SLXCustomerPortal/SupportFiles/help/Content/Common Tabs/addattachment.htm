Adding an Attachment

You can add an attachment to a record. All files are stored in the shared attachments directory and synchronize to and from remote databases.
  • By default, you cannot attach files that are larger than 4MB. If you attach a file or group of files that are larger than 4MB you may receive an error or warning. Contact your administrator for more information.
  • Localized implementations that use extended characters require the Desktop Integration Module in order to support attachments with localized file names.

How Do I?

Attach a file
  1. Open the Attachments tab.
  2. Click the Add File button.
  3. If the Attach File dialog box opens, click Browse.
  4. Navigate to the file you want to add, and then click Open.
  5. In the Description box, enter a description for the attachment if the file name is not clear.
  6. Click OK.
Attach a URL
  1. Open the Attachments tab.
  2. Click the Add URL button.
  3. In the URL box, type the complete URL address.
  4. In the Description box, enter a description.
  5. Click OK.
    Both a description and URL are required.
