Предварительный Просмотр Заявки Перед Печатью

Вы можете просмотреть версию для печати Карточки Заявки. Версия для печати содержит основную информацию о заявке, предмет заявки, описание проблемы, решение заявки в случае его существования, какие-либо внутренние комментарии и всю информацию, содержащуюся на связанных с заявками закладках. Используйте полосу прокрутки для просмотра всей информации. Это представление только для чтения.

Для подробного просмотра заявки перед печатью

  1. На Детальном представлении Заявки на панели инструментов нажмите кнопку Просмотр Печати.

Откроется для просмотра версия для печати карточки Заявки.

  1. Используйте для печати кнопку печати вашего браузера.
  2. Для закрытия версии для печати нажмите кнопку Закрыть Просмотр Печати.
