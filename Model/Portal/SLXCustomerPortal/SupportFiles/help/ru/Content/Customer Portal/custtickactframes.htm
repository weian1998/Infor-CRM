Закладка Комментарии

Используйте закладку Комментарии на лицевой карточке заявки для добавления комментариев к заявке. Однажды добавленные вами комментарии появляются в делах для заявки. Представитель Сервисной Службы при работе над решением проблемы может в клиенте Infor редактировать относящиеся к заявке дела.

Для добавления комментариев к заявке

  1. На Панели Навигации нажмите Заявки.
  2. Используйте кнопку Заявки Поиск для поиска заявки.

  3. Выберите заявку.

  4. На Детальном представлении Заявки перейдите к закладке Комментарии.
  5. В текстовом поле Комментарии введите свои комментарии для заявки.
  6. Нажмите Передать.
    Комментарии появятся в области под закладками.
