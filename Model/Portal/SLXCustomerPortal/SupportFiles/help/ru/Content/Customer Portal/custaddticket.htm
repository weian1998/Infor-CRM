Добавление Заявки на Портале Клиента

Используйте диалог Создать Заявку для добавления новой заявки по вашей проблеме. Система автоматически добавит информацию о вашем имени, названии компании, рабочем телефоне и адресе электронной почты.

Для добавления заявки

  1. На панели инструментов нажмите Добавить Заявку.
  2. Нажмите кнопку Найти , расположенную после поля Область, Категория или Проблема , выберите необходимые элементы из списка и нажмите ОК.
    Набор элементов списка Категория зависит от пунктов, выбранных для Области; а набор элементов списка Проблема зависит от пунктов, выбранных для Категории.
  3. В поле Приоритет нажмите на указывающую вниз стрелку и выберите значение из списка.
  4. В поле Тема введите краткое описание проблемы.
  5. В поле Описание введите подробное описание проблемы.
  6. Нажмите Сохранить.
    Откроется детальное представление Заявки.
  7. Вы можете выбрать закладку Комментарии для добавления комментариев к заявке. Комментарии позволяют представителям Службы Поддержки просмотреть подробную информации о проблеме.
  8. Вы можете выбрать закладку Вложения для прикрепления вложений к заявке. Вложение документов к заявкам может помочь представителям Службы Поддержки лучше понять суть проблемы.
  9. После завершения ввода информации, выйдите из системы или нажмите Добавить Заявку для добавления еще одной заявки.
