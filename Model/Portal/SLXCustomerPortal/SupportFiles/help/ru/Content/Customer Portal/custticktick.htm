Использование Списка Заявок

Вы можете использовать Список Заявок для поиска заявок, просмотра заявок по группам, добавления новых заявок или поиска списков информации.

Для поиска заявки

  1. На Панели Навигации нажмите Заявки.
  2. Используйте Поиск для поиска записей о заявках.

Поиск осуществляется по записям, к которым у вас есть доступ.

  1. Нажмите ссылку ИД Заявки для открытия лицевой карточки заявки.

Для просмотра заявок по группам

  • В списочном представлении Заявок, в списке Групп Заявок выберите Веб-портал Все Заявки, Веб-портал Закрытые Заявки, или Веб-портал Открытые Заявки.

Список заявок в выбранной группе отобразится в нижней части карточки.

Для добавления заявки

  1. На панели инструментов нажмите Добавить Заявку.
  2. Следуйте инструкции по добавлению заявки.

Для использования быстрого поиска

  1. На Панели Навигации нажмите Быстрый поиск.
  2. Следуйте инструкции по поиску информации.
