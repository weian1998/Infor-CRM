Быстрый поиск

Быстрый поиск является мощным инструментом, который поможет вам найти информацию, хранящуюся в базе данных Infor CRM или внешние файлы. Например, вы можете осуществлять поиск по заявкам, чтобы помочь в решении проблемы клиенту или использовать его для поиска конкретного документа, например, презентации. Тип информации, по которой возможен поиск, определяется вашим системным администратором.

Чтобы начать быстрый поиск

  • Нажмите кнопку Быстрый поиск.

Если у вас возникли проблемы с открытием быстрого поиска, свяжитесь с вашим системным администратором. Информация по настройке и решению проблем быстрого поиска находится в справке администратора.
