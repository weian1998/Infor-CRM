Пример Раздела Справки

Это пример раздела справки.

Замените текст в этом разделе с инструкциями по использованию вашего персонального вида. Затем сохраните раздел согласно рекомендуемым правилам наименования.

Для получения подробной информации о добавлении собственных разделов в справку через Веб, пожалуйста, обратитесь к разделу Обзор Справки по Программе Веб в Справке Программы Архитектор приложений.

Соглашение о принципах именования

При создании собственной справки для списков просмотра вы должны придерживаться принципов именования в следующем порядке для того, чтобы ваша справка отображалась корректно:

 

entitylistview.htm - Пример: accountlistview.htm

Где entity - это название объекта, ассоциированного с пользовательским списком просмотра

Советы по Написанию Разделов Справки

При создании собственных разделов по Спискам просмотра в Справке по Программе Веб рассмотрите возможность включения обзорного абзаца или введения, в котором были бы кратко описаны типы записей, отображаемых в списке просмотра. Кроме того, можно включить описание последовательности действий для открытия Списка просмотра, работы с группами или любые другие действия, которые могут быть выполнены с данным Списком просмотра.
